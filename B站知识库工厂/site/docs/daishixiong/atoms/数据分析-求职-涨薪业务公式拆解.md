# 涨薪业务公式拆解

## 一句话定义

用数分业务公式拆自己的工资：到手工资 = 为企业赚的钱（基础价值+溢出价值）− 企业投入成本（管理/培养/项目）− 企业收走的劳动剩余；打工人可控的只有「提高溢出价值」和「降低自己的培养成本」两项。

## 详细说明

**公式各项**：

- 为企业赚的钱 = 市场平均水平的基础价值（工时费）+ 超出市场能力的**溢出价值**。别讲「稀缺」，回到「别人能干的我也能干、我产出比别人多」。
- 企业投入成本 = 管理成本 + 培养成本（最高）+ 项目成本。
- 劳动剩余：层级越高吸走的越多，这是晋升薪资高的本质。

**推论**：

1. 可控发力点只有两个：创造高于市场的溢出价值；即插即用、让领导少带（降低培养成本）——「成年人不培养，只筛选」。
2. 每年约 10% 的普调只是跟随社保基数/通胀的全行业动作，**涨幅超过 10% 才算真涨薪**。
3. 配套趋势：做 π 型人才（两条强腿，如数分+产品）优于 T 型；人生三支撑 = 工作、兴趣、投资。
