# 零数据积累分析经验

## 一句话定义

没数据、没权限时，从成交额反推业务链路争取明细，或自建 MySQL+自动化报表+个人 BI 跑通全流程来练分析。

## 详细说明

**思路一 · 反推链路**
- 成交额 → 思考曝光、转化、订单明细、行为、用户画像
- 识别核心成交人群，做活跃度与策略诊断

**思路二 · 自建数据环境**
- 本地 MySQL 8.0+ 导入下载的表
- Excel 连库做自动日报/周报
- Tableau/Power BI 搭个人看板（仅自用）
- 跑通入库→处理→出图，等价于岗位主流工作
