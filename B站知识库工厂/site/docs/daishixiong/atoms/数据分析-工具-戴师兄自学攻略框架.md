# 戴师兄自学攻略框架

## 一句话定义

戴师兄系列自学攻略的共性骨架：**方法论先行（3 个月入门）→ Excel → SQL → BI（可面试突击）→ Python 延后**，配合书单与项目产出。

## 详细说明

**方法阶段**：
- 入门：《深入浅出数据分析》
- 进阶涉猎 + 《精益数据分析》产出报告

**工具阶段（经典版）**：

| 顺序 | 模块 | 目标 |
| --- | --- | --- |
| 1 | Excel | 函数、透视表、PQ |
| 2 | SQL | 七语句 + SQLZOO |
| 取数补充 | 后羿采集器 | 无需编程 |
| 3 | BI | Tableau/PBI；**面试前学目标公司工具** |
| 4 | Python | 工作后/百万级数据再学 |

**新版攻略页**：方法论四阶段 + 五工具模块一站式收藏。
