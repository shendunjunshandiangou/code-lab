# 透视表行列值字段规则

## 一句话定义

行/列放离散维度，值放可聚合度量；离散字段入值区默认计数，ID 类字段需先转文本避免科学计数法。

## 详细说明

- **行/列（维度）**：区域、省份、期数等离散分类，像切蛋糕的横切/竖切。
- **值（度量）**：成交金额等连续数值，默认求和。
- **离散入值区**：默认变为「计数」；销售工号被识别为数值时显示科学计数法，需先转文本。
- **双击单元格**：查看该汇总值背后的明细行数据。
- **扩展冲突**：同 Sheet 多张透视表时，前表加字段需预留空间，否则报「无法覆盖」错误。
