# 策略运营价值上升

## 一句话定义

市场增量见顶后进入精细化运营，策略运营（补贴/增长/产品运营等）背业绩、奖金倾斜大；分析师若不甘只做执行须向策略能力进阶。

## 详细说明

- **背景**：低垂果实被摘完，从 0 到 1 变难，大厂分工更细
- **策略 vs 分析**：策略对业务结果直接负责，背 KPI
- **能力**：定价、补贴、供需匹配、AB 实验、数据建模
- 岗位名可能叫策略运营、补贴运营、增长策略等
