# 系统求职课七讲框架

## 一句话定义

戴师兄 2024 金三银四免费求职直播合集（约 387 分钟）的七讲闭环：就业局势与信息差 → 简历制作 → 项目准备 → 面试准备 → 系统自学 → 投递技巧 → 解题思路。

## 详细说明

| 讲次 | 主题 | 核心方法论 |
| --- | --- | --- |
| 1 | 就业局势 + 信息差 | 5 个信息差（只谈税前、不主动谈涨薪、薪资后开口、领导>自己>同事、有新工作再离职）+ 涨薪业务公式 |
| 2 | 简历制作 | 大厂级标准、PPT 锁格式模板（凑足 5 份经历）、流水账提炼法 |
| 3 | 项目准备 | 报告 + BI 看板两大必备作品；Excel 自动化报表为加分项 |
| 4 | 面试准备（最重要） | 风险转积累解释法、项目介绍主动暴露问题、带着答案去面试、复盘 |
| 5 | 系统自学 | 学习循环；Excel+BI+思路 5～7 天、SQL 一周、总计不超一个月 |
| 6 | 投递技巧 | 渠道认知（脉脉/Boss/官网内推）、简历发图打招呼、分层投递、1% 响应率基准 |
| 7 | 解题思路 | 不会的题四步应对；刷题→临场→经验三境界 |

定位：面向金三银四/金九银十求职季的完整复习清单，各讲有配套笔记与录播。
