# 算法工程边界觉悟

## 一句话定义

算法岗对工程部署须二选一：**要么完全不碰，要么做到底**；半吊子工程是无底洞，会挤占算法产出业务价值的精力。

## 详细说明

- 模型调优+业务沟通已饱和，兼工程常见陷阱：部署→维护平台→对接业务→被迫接手离职同事工作
- 30 岁+须选路线：纯算法（配资源）或算法+工程（次优选，适合建设不完善小团队）
- 工程比算法好学：跑通一条路径即可，文档/GitHub 成熟
