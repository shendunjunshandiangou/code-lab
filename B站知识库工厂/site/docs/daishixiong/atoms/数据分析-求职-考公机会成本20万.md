# 考公机会成本20万

## 一句话定义

考公的直接成本（备考时间、培训、生活费、放弃工作收入）每年至少 20 万起步——失败则叠加私企「负经验」开局。

## 详细说明

**成本构成**：
- 备考时间精力 + 考试花费
- 生活费 + 收入损失
- 考公失败 → 无工作经验 → 私企涨薪速度大受影响

**决策建议**：
- 家境不好：认真评估上岸成功率，成则投入，不成尽早放弃
- 「家贫走他乡」——考公像买彩票，不计长年投入成本则不明智
- 错考公可能是最大成本，比想象大得多
