# 完美主义四维执念

## 一句话定义

完美主义≠高标准，而是对绝对标准的执念，体现在时间、场合、质量、数量四个维度，导致内耗与拖延。

## 详细说明

| 维度 | 表现 |
| --- | --- |
| 时间 | 「今天晚了就不运动了，明天再说」 |
| 场合 | 「刚才说错一句话，一直内耗」 |
| 质量 | 「做不好就先不做，堆到后面」 |
| 数量 | 「想法不够高质量/不够多，干脆不提」 |

**根源**：社会环境「以偏概全」——考试/KPI 只看结果不看过程，培养同辈压力与自我施压。

**危害**：内耗 + 拖延，求职/学习中表现为「准备不够不敢投」「学不会怎么办」的模糊焦虑。
