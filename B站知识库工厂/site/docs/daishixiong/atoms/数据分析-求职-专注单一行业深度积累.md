# 专注单一行业深度积累

## 一句话定义

职业成长不能「既要又要」多行业——工作 2～3 年时应绑定一个行业（如电商/某头部公司）做深度经验积累，否则无法突破。

## 详细说明

- 多行业浅尝辄止 → 理解太少 → 无法突破
- 戴师兄案例：绑定电商数据研究公司，与某节电商高度绑定，甚至进内部了解推荐逻辑
- 跳槽时 HR 看：专业性 + 头部公司经历 + 时间精力凑齐 → 薪资才有明显提升
- 需提前 1～2 年规划；前提是基础工具能力已会
