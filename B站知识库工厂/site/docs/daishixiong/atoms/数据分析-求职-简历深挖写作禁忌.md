# 简历深挖写作禁忌

## 一句话定义
简历每句话必须能讲清背景与做法；禁止吹牛逼（实习生写优化 50% 不成立），写不出经历说明工具没学明白或经历太少。

## 详细说明

### 写作原则

- 简历上每个字都是准备好的，面试官随时深挖
- 写项目整体提升 5%～10%、自己是其中一份子 → 可以
- 实习生写「带领部门优化效能 50%」→ 不成立
- 不要写太多虚的量化（优化 30%/40% 要能解释如何量化）

### 写不出的根因

1. 经历太少，以前工作不动脑子
2. 工具没学明白（如 Excel 只会填数据 vs 玩出自动化流程）

简历是能力最直观的表现——写不出来 = 太菜了。
