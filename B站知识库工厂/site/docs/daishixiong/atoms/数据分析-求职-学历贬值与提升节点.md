# 学历贬值与提升节点

## 一句话定义

学历持续贬值（供给稀释），有效提升只有两个节点：专升本、出国留学；**量变考研**若不能一步到 985 则代价大于收益——同样时间私企工作 1～2 年更有含金量。

## 详细说明

**贬值原因**：985/211 人数逐年增加，同等学历需与更多学弟学妹竞争。

**两个关键节点**：
1. 大专 → 本科（专升本）
2. 出国留学（打开跨境贸易赛道，SHEIN/Temu/亚马逊等偏好留学生）

**考研判断**：
- 不能一步 985 → 改善微弱，失败代价惨重（零实习/零经历天崩）
- **量变考研**（普本升一本）：2 年后本科生凭经历可追平，30 万机会成本不值（见 [数据分析-求职-考研30万机会成本](/daishixiong/atoms/数据分析-求职-考研30万机会成本.html)）
- 天坑专业只为换专业而考 → 不如直接就业 2 年
- 工作后留学/考研：成功率更高，失败有工作经历兜底
- 从小到大成绩未进前 10% → 建议放弃考研

**筛选档位**：留学生/985 第一档 → 211/一本有差距 → 大专/普本淘汰。
