# 面试时长正向信号

## 一句话定义

面试时长越长，通常说明面试官对你越感兴趣；应主动延长有效交流而非匆匆结束。

## 详细说明

- 时长长 = 兴趣高，是正向信号
- 灵魂拷问类问题通常出现在匹配度已经较高时
- 建议多参加面试练手（先面次优公司），面试能力与工具/笔试能力是**独立技能**
