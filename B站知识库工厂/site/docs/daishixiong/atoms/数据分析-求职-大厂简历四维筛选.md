# 大厂简历四维筛选

## 一句话定义

大厂简历关看学历、能力、经历、稳定性四维，优先级为稳定性 > 经历 > 能力 > 学历；90% 以上候选人倒在简历关。

## 详细说明

| 维度 | 校招 | 社招 |
| --- | --- | --- |
| 学历 | 985/211 有优势；高绩点可弥补 | 本科及以上够用；高学历影响薪资职级 |
| 能力 | 与 JD 对得上即可，深度靠笔面 | 同左；可作品集+临时学习补 |
| 经历 | 与大厂/竞品/上下游越匹配越好 | 同左；跨行业优势不大 |
| 稳定性 | 实习≥3 个月；每段工作≥1 年 | 每段≥1 年、最好 3 年 |

能同时满足四点的简历很少；按此标准准备可大幅提升进笔面概率。
