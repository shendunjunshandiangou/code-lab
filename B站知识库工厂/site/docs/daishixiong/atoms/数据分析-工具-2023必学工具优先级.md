# 2023 必学工具优先级

## 一句话定义

2023 版工具地图：**Power BI 升首选 BI**；必学 Prep Builder + 影刀；MySQL+DataGrip；Python 分析师仅 Pandas+项目+PyGWalker，爬虫用后羿勿深钻。

## 详细说明

| 优先级 | 工具 | 要点 |
| --- | --- | --- |
| BI 首选 | Power BI | 微软 Fabric/Copilot 生态 |
| BI 次选 | Tableau | 可视化强，迁移易 |
| 自动化 | Prep Builder | 图形 ETL、定时任务 |
| 自动化 | 影刀 RPA | 重复操作自动化 |
| 办公 | M365 + Excel + 飞书文档 | Excel 函数/透视/PQ |
| 数据库 | MySQL + DataGrip | 查询刷题；增删改备笔试 |
| Python | Jupyter + **Pandas** + **PyGWalker** | 必做**成型项目** |
| 爬虫 | **后羿采集器** | 勿深钻 requests/逆向 |

FineBI/Quick BI 等公司指定再学。Hive SQL 入职补充。
