# 时序分析找参照物

## 一句话定义

同环比和预测的核心目的是在时间上找到有效参照物——没有前后对比就无法判断好坏。

## 详细说明

- 所有基于时间的同比、环比、预测都是时序分析
- 只有一天数据无法分析好坏
- 现在比之前好 → 总结为什么好；差 → 总结为什么差
- 预测后才能说服业务执行策略

**流程**：时间趋势好坏一目了然 → 对比追溯原因。
