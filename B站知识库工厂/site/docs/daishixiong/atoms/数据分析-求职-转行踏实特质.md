# 转行踏实特质

## 一句话定义

成功转行者最共同的特质是踏实——定好目标后一步一个脚印认真学，不想花里胡哨，全力奔目标。

## 详细说明

戴师兄做社群团长见过很多成长故事，成功者身上最共同的一点：

- **踏实**——认认真真学，不去想那些花里胡哨的
- 已经定好自己的目标，就全力奔着那个目标往前冲
- 对外界「年纪大了别折腾」等劝阻：敷衍或不回应，「你怎么看是你的事，我自己怎么看才最重要」

**反面**：指望捷径、建模比赛加分、不实战就觉得自己够了——往往走不通。
