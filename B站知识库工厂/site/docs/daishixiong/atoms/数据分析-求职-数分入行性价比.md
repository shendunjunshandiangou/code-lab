# 数分入行性价比

## 一句话定义

数据分析门槛适中、万金油、性价比超级高——有一定护城河但不至于学太难，岗位一直在，想要年薪 20–30 万就去做。

## 详细说明

**为什么还值得进**：
- 门槛没有那么高
- 万金油属性——多行业都需要
- 性价比超级高
- 有一定门槛和护城河，但不至于学太难
- 薪资也会稍微高一点

**定位**：
- 岗位一直在——不是昙花一现的风口
- 想要年薪 20–30 万就去做，不想要就算了
- 9 年土木转数分案例：杭州上岸约 1.3 万+，较原行业涨薪，「转行穷 3 年」未发生
