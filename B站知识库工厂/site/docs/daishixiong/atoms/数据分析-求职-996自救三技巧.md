# 996 自救三技巧

## 一句话定义

周中批量完成周末分批交付、按自己节奏安排工作、提升工具效率——三法应对无效内卷加班；老板坐背后无限派活则赶紧跑。

## 详细说明

| 技巧 | 做法 |
| --- | --- |
| 一次干完多次交付 | 周中做完整周活，周末分批交，营造饱和假象 |
| 自己节奏 | 固定健身等个人时段，灵活调整工作，不必等同事走 |
| 提升生产力 | Excel/SQL/Python 提效；休息好才能高效 |

**副作用（正向）**：治拖延、预留学习时间、应对突发。

**底线**：老板天天坐背后无限派加班 → 命比工作重要，换环境。
