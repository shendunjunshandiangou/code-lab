# 裁员面试敏感话题禁忌

## 一句话定义
被裁后面试绝不能透露能力短板或无法改变的环境因素（领导差、内斗、加班），应在双向选择时自行判断岗位风险。

## 详细说明

### 两类绝对不能透露的原因

1. **自身能力不足**（压力太大、绩效差、沟通问题）——公司要优中选优，暴露短板极大降低成功率
2. **无法改变的环境因素**（领导太坏、同事太菜、内斗、天天加班）——HR 会认为适应能力差

### 正确心态

- 很多裁员是高层决定裁整个业务线，并非个人能力问题
- 完美工作环境不存在；雷点过多应在面试中收集信息，**双向选择时果断开溜**
- 离裁员可能不是你不行，而是公司不行
