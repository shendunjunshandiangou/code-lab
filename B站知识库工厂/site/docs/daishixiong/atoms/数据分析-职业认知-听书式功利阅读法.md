# 听书式功利阅读法

## 一句话定义
只为高效解决问题而读书：用听书栏目（20~30 分钟/本）利用通勤时间大量筛书，特别感兴趣的再详读——效率与正反馈兼得。

## 详细说明

- **立场**：读书是借鉴书中总结好的经验，比自己摸索高效得多；文艺类和小说少看，阅读目的非常功利。
- **痛点**：通读全文效率极低（自己读《三体》花了近一个学期）。
- **方法**：得到"每天听本书"类栏目把一本书的主要内容压缩到 20~30 分钟；每天 3 小时通勤可听 2~3 本，戴师兄半年听了 230+ 本；特别感兴趣的书再买来详读。
- **收益机制**：听书保证效率，"只详读感兴趣的书"建立正反馈——读了就一定有收获，从而养成持续的读书习惯。读书跟健身一样，是少有的只要投入就一定有回报的事情。
