# 力大砖飞简历策略

## 一句话定义

校招生/零经历者可把在校简历做到与工作两年的人一样漂亮——实力证明与在职者同等，是校招脱颖而出的终极解法。

## 详细说明

- 课程里在校生改出的简历可与工作两年的人相当
- 核心优势：向企业证明在校期间学得很扎实，实力与在职者一样 + 刚毕业薪资低
- 配合 [数据分析-求职-无经历转行唯一解](/daishixiong/atoms/数据分析-求职-无经历转行唯一解.html)：学企业项目融入简历
- 校招地狱难度下，大多数零实习者需构建/重构简历才能进入响应池
- 即将毕业零实习：过年后启动学习 + 投递，真正上岸在毕业后第二个金九银十
