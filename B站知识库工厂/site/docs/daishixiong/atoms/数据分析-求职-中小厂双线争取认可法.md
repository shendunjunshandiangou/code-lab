# 中小厂双线争取认可法

## 一句话定义

二三线/中小厂分析师生存：同时争取业务领导与中台领导认可——前者给业务成果，后者帮推 OPR 最重指标，等待重要场合曝光。

## 详细说明

1. 摸清业务/中台对数据的认知与 KPI
2. 与业务领导 N-1 建关系 → 获取真实业务逻辑 → 做出亮眼成果
3. 选中台领导（与老板有私交、OPR 最重项）帮推项目/写材料
4. 双线「枕边风」→ 季度会/述职等场合给 20 分钟汇报机会

大厂逻辑类似但人事关系更复杂；专业力是关键时刻武器，需软实力铺垫。
