# 2026 最快学习路径

## 一句话定义

2026 求职导向的最短路径：**Excel + SQL + 分析报告**；BI 可作品集截图+话术、入职再学；Python 非必需。

## 详细说明

| 模块 | 2026 定位 |
| --- | --- |
| Excel | 日常用得最多；只学一个就选它 |
| SQL | 硬门槛、必考；写不顺不配称分析师 |
| 分析思维 | 最难回答、最重要 |
| BI | 可不系统学；各公司自研 BI 入职后重学 |
| Python | 少数分析师日常使用；AI 已覆盖大量代码场景 |

**压缩效果**：原 3–6 个月多工具学习 → **数周**完成核心三件套。

**必做项目**：报表 + 看板（可截图）+ **分析报告**（缺报告如缺论文）。
