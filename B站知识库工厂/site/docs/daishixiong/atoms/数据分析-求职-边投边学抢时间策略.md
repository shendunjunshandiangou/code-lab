# 边投边学抢时间策略

## 一句话定义

抢求职时间时停止纯学习先写简历投递，通过边投边学根据响应度迭代——没有「准备好再投」。

## 详细说明

- 当务之急：停止手上学习 → 写简历 → 直接投递 → 看响应
- 没响应 = 马上改简历
- 每天投 100 个但要筛选好岗位、照 JD 写
- 时间通过边投边学抢回来，不存在准备好再投
