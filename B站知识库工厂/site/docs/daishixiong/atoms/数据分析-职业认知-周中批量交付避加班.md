# 周中批量交付避加班

## 一句话定义

周中提前完成整周工作、周末分批交付，营造工作饱和假象，避免领导主动加活——同时治拖延、预留学习时间。

## 详细说明

- 技巧一：一次干完、多次交付
- 技巧二：固定个人节奏（如健身），不必等同事下班
- 技巧三：提升工具效率；老板坐背后无限派活则应考虑换环境
