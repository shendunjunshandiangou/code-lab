# 产品三价值与真需求

## 一句话定义

梁宁《真需求》：商业本源是真需求；产品价值 = 功能 + 情绪 + 资产，三者缺一都有问题。

## 详细说明

- 无人付费 = 伪需求，须快速验证
- 定价权来自长期大规模主动投入
- 评估人：技能点、资源盘、影响力
- 适合有 2–3 年业务基础后深读
