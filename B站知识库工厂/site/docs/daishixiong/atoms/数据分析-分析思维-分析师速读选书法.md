# 分析师速读选书法

## 一句话定义

速读 = 跳过序言直看目录+抽样；好书看再版与作者背景；先读有主心骨的方法论，工具书优先视频课。

## 详细说明

- **选书**：再版≥2 次；作者最好从底层成长（如黄有璨）
- **流程**：目录定类型 → 量子阅读抽样 → 能否一句话复述全书
- **原则**：开卷有益（≥10% 信息差）；一天一本可多读几遍
- 纯工具操作书：B 站教程更直观，书适合应急查图表
