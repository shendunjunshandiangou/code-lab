# 户口本年龄简历处理

## 一句话定义

简历年龄与户口本不一致时，互联网行业较开放——可写真实年龄，HR 问到时实话实说「家里着急写错了」，真诚沟通即可。

## 详细说明

**案例**：实际不到 28 岁，户口本写大 2 岁 → 简历上像 29 岁应届无经验，严重不占优。

**处理方式**：
1. 户口本日期可以重新公证（嫌麻烦可跳过）
2. **直接写真实年龄**——互联网行业比较开放
3. HR 问到时说「那时候家里着急给我写错了」，实话实说
4. 真诚沟通，大家能看出来你是认真的

**不要因此放弃**：年龄不是互联网最硬的门槛，项目和作品集才是。
