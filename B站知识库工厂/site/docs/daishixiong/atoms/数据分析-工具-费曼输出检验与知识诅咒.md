# 费曼输出检验与知识诅咒

## 一句话定义
学没学会的两个检验标准：做得出作品、讲得出来；脑子里清楚但开口讲不出，是陷入了"知识的诅咒"——默认对方也懂你的口诀。

## 详细说明

费曼学习法人人都提，但没人实践。输入（定计划、找资料、快速学）之后，核心在**输出**：

1. **做出作品**：做不出作品 = 没学会。作品带来正反馈（"学完立马能做出来，太爽了"），没有正反馈很难坚持，也不会得出"我喜欢数据分析"的结论。
2. **讲得出来**：讲不出来 = 没融入自己的知识体系。

**知识的诅咒的典型案例**：面试问"指标异动飙升怎么排查"，上过课的学员脑子里只蹦出"业务流程""向前看向下钻"几个词，却无法向没上过课的面试官解释这六个字是什么意思——你默认了对方也懂这套口诀。

**检验方式**：开口讲一遍并录下来听，或直接用面试实战检验（面试是最接近实战的检验场，反问环节还能收获信息）。
