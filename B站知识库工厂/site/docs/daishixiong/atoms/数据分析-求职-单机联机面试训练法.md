# 单机联机面试训练法

## 一句话定义
面试是独立于笔试/工具技能的独立能力，通过「单机」（自我介绍+经历滚瓜烂熟）和「联机」（好友模拟）+ 多面真场练手提升，没有捷径只能多练。

## 详细说明

### 单机入门
- 简短自我介绍：你是谁、来自哪、专业能力
- 详细经历描述：学过什么、掌握什么、取得什么成果——熟到随时能进入状态

### 联机模式
- 与好友角色扮演模拟面试，一题一题考
- 回答不满意就复盘迭代、针对性反复练习

**多面练手策略**（求职全攻略）：先面次优公司找回感觉，积累面试/笔试经历后再面最想去的公司；面试时长越长通常说明兴趣越高。

练到面对提问不慌不忙，再不停参加真实面试复盘，即可独当一面。
