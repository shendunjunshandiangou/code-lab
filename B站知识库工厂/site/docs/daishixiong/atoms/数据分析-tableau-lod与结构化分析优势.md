# Tableau LOD 与结构化分析优势

## 一句话定义

Tableau 用简单 LOD 表达式可在明细表上完成购买频次×客户数等聚合分析，无需预建聚合表，且结果可复用——这是成熟 BI 的结构化分析优势。

## 详细说明

**传统分析**：不同年龄段/地区用户数——需预聚合或复杂函数。

**结构化分析**：不同 **购买频次** 的客户数量（频次与客户数均为聚合特征）——Tableau 一个 **LOD 表达式** 即可，并可保存供他人使用。

**原则**：把复杂问题变简单，而非把简单问题搞复杂（复购率/留存/周转不必堆大量中间表）。

**RFM**：成熟 BI 应能在 **明细表基础** 上直接做 RFM；必须中间表则割裂分析流程。
