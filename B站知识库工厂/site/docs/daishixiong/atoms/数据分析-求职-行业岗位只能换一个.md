# 行业岗位只能换一个

## 一句话定义

转行时行业和岗位只能同时换一个：同行业换岗 OK，同岗位换行业 OK，两个都换难度陡增。

## 详细说明

- 游戏开发 → 游戏分析师/产品/运营：简单
- 游戏开发 → 电商产品经理：难
- 传统行业应去头部互联网化公司（地产→贝壳/链家，出行→顺丰再跳美团/滴滴）
- 先进入行业龙头做数据相关工作，再转分析师；上岸后换行业较容易
