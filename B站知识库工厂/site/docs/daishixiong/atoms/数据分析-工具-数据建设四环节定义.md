# 数据建设四环节定义

## 一句话定义

数据建设 = 数据获取 + 数据处理 + 数据存储 + 数据应用；仅有「下载数据看一眼」的应用不算建设。

## 详细说明

| 环节 | 内容 | 要点 |
| --- | --- | --- |
| 获取 | 埋点、日志、业务录入、爬虫 | 可借外部数据，但须有稳定来源 |
| 处理 | 清洗、ETL、口径统一 | 脏数据不处理无法复用 |
| 存储 | 数据库、数仓分层 | **存下来才有价值**，可复用、可回溯 |
| 应用 | 看板、日报、取数、分析 | 每家公司都会做，但 alone 不算建设 |

**核心判断**：刨去前三环只剩应用 → 不算数据建设。
