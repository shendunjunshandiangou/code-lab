# SQL 是什么：一大能力两大属性

## 一句话定义

SQL 是对数据库发号施令的标准化语言；两大属性是「结构化」（只处理规整数据 + 严格语法顺序）和「标准化」（ANSI 统一语法，跨库通用）。

## 详细说明

### 一大能力

对数据库发号施令：创建库、查询数据、优化查询。

### 属性一：结构化

**数据层面**：只处理行列规整的结构化数据，不能处理合并单元格、图片等；改数据是整行整列操作。

**语法层面**：七大语句顺序固定，有严格运行顺序；SQL 是单向执行语言。

### 属性二：标准化

- ANSI 制定统一语法标准
- 主流数据库均兼容
- 学会 SQL 如同学会数据库界的通用语言（需有权限）

### 类比

数据库是山洞金库，SQL 是「芝麻开门」的咒语。
