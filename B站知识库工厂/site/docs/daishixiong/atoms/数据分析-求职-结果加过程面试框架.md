# 结果加过程面试框架

## 一句话定义
中国式面试不用 STAR/PART（最多 60 分），应一上来讲成果+核心过程，其余细节留给面试官提问——问到没得问即通过。

## 详细说明

### 推荐结构

> 项目输出了 X，业绩增长 Y%。我通过 [三板斧过程] 实现了目标。

示例：「上季度销售业绩提升 10%——第一提升内容质量，第二增加留资，第三增加复购优惠。」

### 主动讲 vs 留给提问

| 自己快速讲 | 留给面试官问 |
| --- | --- |
| 项目结果 | 背景、思考、困难、深度细节 |

- 面试官问得越多 = 越感兴趣
- 80% 准备用于应对提问
- 面试官深挖出来的信息才觉得可信

目标：用此框架可达 80～100 分。
