# 岗位 JD 拆解法

## 一句话定义

把 JD 拆成职能、工具、技能三维：要你做什么事、用什么做、具体怎么做——据此调整简历与学习计划。

## 详细说明

以数据分析为例：

- **职能**：指标优化、支持业务取数、日报周报、专业分析、模型预测
- **工具**：Excel、SQL、Power BI、Tableau、Python 等
- **技能**：指标体系、异动分析、AB 测试、业务诊断等

拆解完成后，才能有的放矢地升级简历内容、补齐工具技能。
