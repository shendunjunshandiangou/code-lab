# 主动学习选工具法

## 一句话定义

在陌生技术领域应**需求驱动主动探索**：定需求 → 搜工具 → 读官网 → 对照 Quick Start 动手，而非被动跟教程。

## 详细说明

**被动学习问题**：投入大无产出、教程版本过时、视野窄（如 Web 只认 Flask/Django）。

**主动四步**：
1. 明确需求（如「像 Tableau 拖拽、少写代码」）
2. 搜索并打开**官网**（比课程准确）
3. 一句话介绍 + 示例图判断是否匹配
4. 按官方安装/Quick Start 实现

**Python 生态**：约 60 万第三方包，必须需求筛选，不可能全学。

**BI 优先原则**：无硬性工具要求时，**BI 拖拽永远比写代码快**（本课前提下的共识）。
