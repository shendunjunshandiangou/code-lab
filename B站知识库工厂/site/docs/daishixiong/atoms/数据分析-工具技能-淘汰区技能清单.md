# 淘汰区技能清单

## 一句话定义

纯执行型、机械操作的数据技能会被 AI 完全替代，包括手写 SQL/Python、死记函数语法、人肉取数做表。

## 详细说明

最容易被 AI 干掉的是这些**纯执行动作**：

- 手写 SQL
- 手写 Python 代码
- 死记函数语法
- 人肉取数做表

**注意**：不是说 SQL 和 Python 不重要，而是**只靠机械执行、不知道为什么要取这个数、不知道口径**的工作会被全部替代——AI 写得比你快多了。
