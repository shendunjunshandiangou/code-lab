# Python 可视化主动学习选工具法

## 一句话定义

在陌生领域按「明确需求 → 搜工具名 → 读官网一句话介绍+示例」主动选型，避免只跟 B 站教程被动学习。

## 详细说明

**被动学习问题**：动力衰退、只会教程内图表、工具版本过时。

**无硬性工具要求时**：BI 拖拽 **比写代码快、成本低**（Tableau/Power BI）。

**主动流程**：
1. 写清需求（如「Python 里 Tableau 式拖拽」）
2. 搜索候选工具 → 打开 **官网**（比任何课程准确）
3. 看一句话定位 + 示例图判断是否匹配
4. 按官网安装/Quick Start 动手

Python 可视化生态工具远多于教程覆盖（Matplotlib、PyGWalker、Shiny 等）。
