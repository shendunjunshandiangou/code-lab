# 大厂业务协作算账流程

## 一句话定义

顶级大厂分析师核心参与「策划→算账→定目标→执行→监控→复盘→迭代」全流程，关键是参与决策层商业算账而非只做看板。

## 详细说明

**八步流程**：
1. **策划**：产品/运营提出业务方向（基于公司战略）
2. **算账**：分析师/战略分析师评估能不能做、成本、预算回收、ROI
3. **方案对比**：A/B/C 方案比投入产出，各方利益博弈
4. **定目标**：预算拆解为各团队量化产出（~10 人配 1 经营分析师）
5. **执行**：业务朝目标推进（可能藏进度）
6. **监控**：线上看板实时追踪
7. **复盘**：定位未完成原因
8. **迭代/专项**：抓增长点、竞品研究

**分析师精力分配**：
- ❌ 只做大盘看板 → 越做越工具人
- ✅ 参与决策层算账、目标拆解、团队产能理解

**分工逻辑**：业务团队（销售/客服）不可能人人学数据分析——学会了就转分析师岗，成本hold不住。
