# 转行项目最低配置

## 一句话定义

转行数分最低需两个项目：有数据逻辑的企业看板 + 从分析到出报告的分析型项目，融入简历即可拿到门票。

## 详细说明

**看板项目**：
- 为什么先看这些指标、再看那些指标
- 看完板能得出什么结论
- 覆盖数据监控和数据处理

**分析型项目**：
- 指标上升/下降如何解读
- 图表如何制作、如何看

大多数 3 年以内分析师做的也就是这些；区别在于 3 年中是否积累了经营分析/增长分析等专项能力。有这两个基本盘，比拼的是学历、稳定性、面试表达。
