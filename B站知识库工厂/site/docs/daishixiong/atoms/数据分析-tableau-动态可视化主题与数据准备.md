# 动态可视化主题与数据准备

## 一句话定义

动态图表/仪表盘需要「时间 + 数值」字段；主题可选业务指标趋势或求职向热点数据，数据来自社区、天池/Kaggle 或手动整理。

## 详细说明

**适用场景**：任何随时间变化的过程（如 Gapminder 200 年各国发展；业务 KPI 趋势）。

**主题选择**：
- 在职：负责指标的时间增长 → 呈现业务贡献
- 求职：疫情、综艺排行等热点 → 有趣、易代入

**数据来源**：天池、Kaggle；Tableau Public 社区（含 Query+SQL）；手动记录（如 UP 主 B 站播放量）。

**字段要求**：必须含 **日期/时间 + 对应数值**。

**字幕缺失**：本视频 P2 动态图表/仪表盘实操步骤未收录于字幕。
