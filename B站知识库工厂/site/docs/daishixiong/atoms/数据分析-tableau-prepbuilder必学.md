# Tableau Prep Builder 必学

## 一句话定义

Prep Builder 是 Tableau 生态图形化 ETL 工具，拖拽整合本地/云端数据并上传 BI，可定时自动跑——分析师效率飞升工具，2023 工具清单标为必学。

## 详细说明

- 图形化流程化数据处理，无需编程即可掌握
- 本地表格、混合云端、上传数据库与 BI 平台
- 一键运行；配置后可 **定时自动处理**（上班前挂任务）
- 适合数据环境混乱、无数据建设的公司

与 Power Query 相比，Prep 可 **输出数据**；Power Query 严格意义上不算完整 ETL（无输出，数据治理难）。
