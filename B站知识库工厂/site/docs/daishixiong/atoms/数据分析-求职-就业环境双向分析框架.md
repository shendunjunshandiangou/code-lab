# 就业环境双向分析框架

## 一句话定义

「好不好找工作」须从供需双侧分析——岗位由企业提供（大环境），个人优势靠能力；勿只听培训机构贩卖焦虑或机会，应用框架+招聘数据自行验证。

## 详细说明

1. 岗位谁提供 → 看行业景气、政策、融资
2. 哪些方向岗位增 → 数字化、跨境、AI 提效等
3. 自身牌怎么打 → 专业、经历、技能组合
