# 豆包模拟面试全流程

## 一句话定义
用豆包搭建模拟面试官智能体的完整工作流：角色设定 → 喂简历和 JD → 语音通话压力面试 → AI 复盘打分 → 生成标准题库答案和一分钟自我介绍。

## 详细说明

**为什么用豆包**：① 可直接语音打电话，模拟真实面试压力；② 语音输入又准又能长时间识别；③ 支持自定义智能体，过程结果可留存复盘。

**完整流程**：

1. **建智能体**：用[数据分析-求职-AI模拟面试身份反转设定](/daishixiong/atoms/数据分析-求职-ai模拟面试身份反转设定.html)让豆包生成角色设定词，精简后在"更多 → AI 智能体 → 创建"里粘入。
2. **喂材料**：把简历 + 岗位 JD 发给 AI（Boss 直聘 JD 不让复制，截图后用微信"提取文字"OCR），要求"评估这场面试需要准备什么，并扮演严厉 HR 给我压力面试"。
3. **语音面试**：在智能体里直接发起通话。面试中答不上来可随时求助："我忘了怎么做，告诉我这题怎么答最好，并给学习建议。"
4. **复盘打分**："站在大厂面试官角度评估这位面试者能打几分，哪些需要优化。"AI 会输出优势/短板的结构化复盘（演示中给出 6.5 分，指出"指标关联业务决策深度不足"是关键扣分点）。
5. **要题库和标准答案**："给我这个岗位场景下的问题清单和每题的标准答案，我拿来和自己的回答对比。"
6. **生成自我介绍**："基于简历给我多版一分钟以内的自我介绍。"

**附带用法**：简历不要自己闷头写，先口述让 AI 总结，顺便训练口语表达。不用现成模拟面试智能体的原因：不好调参；自己搭掌握的是生成各类角色智能体的源头能力。
