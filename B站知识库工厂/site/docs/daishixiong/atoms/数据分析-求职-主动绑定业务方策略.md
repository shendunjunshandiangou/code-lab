# 主动绑定业务方策略

## 一句话定义

被动接报表需求学不到超认知内容——应观察 1 个月后自拉小群，绑定高价值业务伙伴，跳过正式需求机制做最痛问题的专项。

## 详细说明

**操作步骤**：
1. 观察心仪业务方约 1 个月，勿一上来就绑定（没得换）
2. 拉小群：高价值需求直接在群里提，跳过需求机制
3. 做出成绩后向老板汇报「手上有重活」，卸掉低价值需求
4. 边界说明白：研究一个最痛的问题；业务伙伴会滔滔不绝输出细节

**为何重要**：
- 被动分配的需求都在你认知内，用传统方法解决就结束
- 必须触达能力边界才能向上积累
- 无法坐业务工位时，拉群是唯一办法
