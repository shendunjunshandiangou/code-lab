# 先就业再择业原则

## 一句话定义

不必执念先转数分；先找任意工作获得场景，再在业务岗用数据提效，后续再视情况转数分——没工作场景成长会停滞。

## 详细说明

- 有工作场景才能用数据、交流、产出价值
- 可先业务/运营，把数据分析当武器，开心则继续，想转再跳
- 数分不一定适合所有人（天天对着表和代码）
- 转型路径：业务 → 数据运营 → 数据分析，比零经验直转易
