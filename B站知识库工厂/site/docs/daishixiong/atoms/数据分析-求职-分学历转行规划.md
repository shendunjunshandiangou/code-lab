# 分学历转行规划

## 一句话定义

985 堆实习数量、211/一本重质量找中小公司、普本/大专从数据标注/专员做起——各学历层级策略不同。

## 详细说明

| 学历 | 策略 |
| --- | --- |
| 985/留学生 | 堆实习**数量**（质量不如数量）；大三前 1～2 份 → 大三暑假冲腾讯/字节/小红书/B 站/得物 |
| 211/一本 | 找中小公司，岗位放开（数据标注也行）；重实习细节和质量，写清数分链路 |
| 普本 | 数据标注/数据专员做起，练好报表/看板工具链 |
| 大专 | 工作后必须专升本，纯靠工作经历提升 |

211 校招/实习不要碰大厂，拿 211 跟普本/一本竞争中小公司反而有优势。985 要刷数量因心境喜欢频繁跳槽不太积累真东西；211 把一个数分链路写明白反而匹配腰部公司。
