# 直播间主播商品阶段搭配

## 一句话定义

直播分析要把场次拆成引流/留存/变现阶段，分别匹配主播、话术行为与商品顺序，而不是全程同一套动作。

## 详细说明

常见搭配逻辑（多组答辩共识形态）：

| 阶段 | 目标 | 常见选择 |
| --- | --- | --- |
| 前期 | 吸引注意、拉停留 | 介绍商品 + 大屏展示；引流品/高吸引力 SKU |
| 中期 | 抬在线人数 | 强化大屏展示等聚人气行为；留存向主播 |
| 后期 | 促成成交 | 展示购买者评论、感性营销、高人均 GMV 主播与爆品 |

商品顺序：人气不足时先推高吸引/高销量品 → 留存品 → 高变现品；冷门品穿插。输出必须是可执行排班/排品建议。
