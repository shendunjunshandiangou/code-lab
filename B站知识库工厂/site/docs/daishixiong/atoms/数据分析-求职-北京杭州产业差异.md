# 北京杭州产业差异

## 一句话定义

北京适合高薪跳槽、杭州适合入门——北京政治中心、纯技术大厂远离、活下来公司规模都不小；杭州电商产业密集、小企业多、数据分析用人标准相对低。

## 详细说明

| 维度 | 北京 | 杭州 |
| --- | --- | --- |
| 人才竞争 | 硕士太多，周边省份的人都往北京跑 | 相对宽松 |
| 公司规模 | 能在北京活下来的公司规模都不小 | 大量电商/主播小企业 |
| 产业分布 | 滴滴/美团等靠近政治中心有牌照优势；腾讯等纯技术远离 | 电商、互联网产业密集 |
| 数据分析需求 | 要求高、岗位相对少 | 需求多、标准相对低 |
| 适合阶段 | 跳槽涨薪 | 入门上岸 |

**河北等地困境**：第二产业为主 → 第三产业/服务业少 → 数据分析需求少 → 只能往北京投但要求更高。

**白领城市总原则**：做白领只能去大型企业扎堆的城市；GDP 高不等于白领岗位多。
