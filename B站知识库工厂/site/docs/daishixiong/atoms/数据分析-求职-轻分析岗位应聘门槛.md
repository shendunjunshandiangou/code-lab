# 轻分析岗位应聘门槛

## 一句话定义
海外云业务拓展等「需要分析但要求不高」的岗位，掌握基础工具 + 面试前了解业务模式即可应聘，不必系统报课，入职后实战中提升。

## 详细说明

适用场景：JD 要求分析技能但深度有限（如海外云业务拓展）。

应聘准备：

1. 先看 JD 具体写什么
2. **基础工具**：免费课内容足够
3. **分析思维**：面试前了解业务模式，所需分析能力往往比较单一
4. 掌握基本分析流程即可入职

系统报课适合：真正决定系统学习、职场瓶颈需新出路、或想作技能储备/兴趣拓展。在校生可通过实训项目写入简历替代实习经历。
