# 人脉带来跳槽溢价

## 一句话定义
正常应聘很难大幅涨薪，因为工作态度、发展潜力这类软实力无法被陌生公司承认；而合作中被看见的人脉渠道可以直接兑现这部分溢价。

## 详细说明

- **案例**：戴师兄毕业半年月薪 6000 → 1.8 万（3 倍），机会来自公司合作伙伴——新东家老大在此前合作中看重他的表现，直接开出三倍条件。
- **原理**：陌生公司通过面试只能认可你的经历和能力，态度、潜力等软实力没有信任基础；合作过的人则完整见过你的做事方式。**再有眼力的伯乐，也得先知道你的存在才能相中你。**
- **人脉的成本**：定期参加各类活动；在公司间合作中表现得更能干、更积极；抽出相当一部分时间专门约饭聊天。
- **心法**：积累人脉不只为工作机会，更是结识优秀的人；只要优秀和努力被看见，人脉会带来意想不到的收获。
