# 数据专员到分析师路径

## 一句话定义

转型数分最简路径：数据专员（工具岗，能写 SQL）→ 老实干满 1 年 → 再投数据分析师；或先做业务再用数据提效后转。

## 详细说明

**路径一（直接）**：自学攻略工具 → 找数据专员/数据工具人 → 1 年后转分析师。

**路径二（推荐）**：先做运营/销售/客服等业务 → Excel/SQL/Tableau 提效 → 转数据运营/分析师。

数分需求随年限上升，有 1～2 年互联网经验再转比零经验轻松得多。

**两步拆解原理（局长直播补充）**："为什么想做数分"和"为什么想去某行业"两个问题合在一起很难回答，拆成两步每步只答一个问题，路径丝滑：土木转电商数分 = 先在偏传统公司做数分再换电商，或先做电商运营再转数分。**只要能接触到数据，就有操作空间**——数据专员、数据运营都可以去。
