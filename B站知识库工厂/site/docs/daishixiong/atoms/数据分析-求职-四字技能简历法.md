# 四字技能简历法

## 一句话定义
工作经历用「BI 看板」「经营分析」「算法建模」等四字前缀描述，HR 不看后面也能判断对口程度，并可按 JD 灵活改词。

## 详细说明

示例：
- 「算法建模」→ 跟算法相关
- 「BI 看板」→ 数据呈现
- 「经营分析」→ 成本/人效

按 JD 调整：目标招 BI 看板的人，「数据应用」可改为「BI 看板」；目标招 BI 平台，可改为「BI 平台」。

注意：
- 避免烂大街标签（5W1H、sklearn、RFM）——大厂挂简历最常见原因就是「没有亮点」
- 写能看懂又有差异化的内容

**补充（转行直播）**：
- 专业写大类：国民经济管理→经济学；工商管理→管理学 + 经济学课程
- 岗位往数据靠：数据整理→数据专员/数据运营；外贸→外贸数据统计
- 公司写简称不写「XX 省市 XX 信息科技」
