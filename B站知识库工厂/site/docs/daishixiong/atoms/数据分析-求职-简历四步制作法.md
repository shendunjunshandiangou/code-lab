# 简历四步制作法

## 一句话定义

写简历按「你有啥 → 你要啥 → 怎么说 → 说哪个」四步：先盘点素材，再拆 JD，用职能+结果表达，最后按目标岗位取舍展示内容。

## 详细说明

| 步骤 | 动作 |
| --- | --- |
| 你有啥 | 基本信息、教育、工作/实习、项目、技能、作品集 |
| 你要啥 | 拆解目标岗位用人需求 |
| 怎么说 | 职能 + 结果，分点量化 |
| 说哪个 | 校招重学历实习、社招重技能强度，整合删减 |

原则：Less is more，少而精好过堆砌。

**补充（系统求职课）——流水账提炼法**：「你有啥」写不出来时，先按时间线把做过的事写成流水账（任何活动都有策划→组织→执行→分析反馈四阶段），写完才能提炼出「活动策划、组织协调、数据分析、汇报总结」等技能，再归类对齐岗位。在校活动认真复盘=低配实习；工作与数据无关的，自己搭 MySQL 库导数据、连 BI 做看板报告，不必等领导安排。配套模板要求：简历凑足 5 份经历（工作/实习+项目），删个人评价，内容刚好填满一页、第二页放作品。
