# 无经历转行唯一解

## 一句话定义

没有相关经历想转行数分，唯一解是学企业正在做的数据分析项目，融入简历说成过去工作/实习中做过。

## 详细说明

- **次优解**：现岗学会数据分析并产出成果——这叫掌心（自我提升），不算转行
- 每家公司具体做什么外人不清楚，关键是项目能讲清楚
- 伪造简历 + 面试逐字稿：试用期大概率被裁，培训班两三万白交
- 必须在企业实战场景中学习，脱离企业的学院式基础学习对转行求职不够
- 分析师工作中做什么就学什么；学不到对应内容，面试和简历都无从写起
