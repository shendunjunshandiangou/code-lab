# 面试录音AI复盘法

## 一句话定义
每场面试录音，发给 AI 分析下场要迭代什么、逐字稿怎么更新，用过程复盘而非单次结果驱动成长。

## 详细说明

1. 面试前尽可能**打磨逐字稿**
2. 每场面试**必须录音**
3. 录音发给 AI：下场面试迭代什么？逐字稿怎么更新？
4. 注重**过程而非结果**——能力提升来自能把项目讲好、回应各种问题
5. 多投递不想去的岗位，**免费刷经验、免费模拟面试**
