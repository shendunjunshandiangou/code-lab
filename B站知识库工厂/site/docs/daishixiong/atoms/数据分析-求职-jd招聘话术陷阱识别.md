# JD招聘话术陷阱识别

## 一句话定义
招聘 JD 里「工资面议」「年轻有活力」「免费宵夜」等话术，往往掩盖低薪、加班、偏远办公等真实条件，需要翻译成真实含义再决策。

## 详细说明

| JD 写法 | 真实含义 |
| --- | --- |
| 工资面议 | 低到不好意思写，先骗过来再说 |
| 期权激励 | 用期权拴住员工，未来不一定值钱 |
| 免费宵夜 | 廉价宵夜喂到饱，不到凌晨不准走 |
| 团队年轻有活力 | 老员工跑路，在职都是新来的 |
| 可适当加班者优先 | 适当加七八个小时 |
| 回去等消息 | 再看看有没有更便宜的候选人 |
