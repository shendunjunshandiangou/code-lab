# 数据岗位六类归纳

## 一句话定义

繁杂的数据岗位可归纳为六类：算法、数分、数仓、开发、产品、运营——先了解实际工作内容再选岗。

## 详细说明

| 类别 | 核心工作 |
| --- | --- |
| 算法 | 开发算法模型 |
| 数分 | 业务分析、报告、策略 |
| 数仓 | 数据仓库、ETL 基建 |
| 开发 | SQL/报表开发 |
| 产品 | 数据产品需求沟通 |
| 运营 | 接触数据的运营岗 |

- 每家企业岗位名称和定义不同
- 不要对岗位名称有执念，按实际内容选择
- 产品实际是做需求沟通，不是只设计页面
