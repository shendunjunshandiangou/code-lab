# 无权限自建数据库练手

## 一句话定义

没有系统/数据库权限时，自建 MySQL 8.0+ 导入手上数据，用 Excel 连库做自动化报表 + Tableau/PBI 自建看板，跑通入库→处理→看板全流程。

## 详细说明

**前提**：公司通常会给表或报表下载入口。

**步骤**：
1. 本地装 MySQL 8.0+（支持函数）
2. 按规范命名导入所有数据（每天手动更新也行）
3. Excel 连接数据库 → 一键刷新日报/周报
4. Tableau/Power BI 自建看板，**只给自己用**，不对外

**价值**：大多数分析师岗位也就做这些活；有表就能复现，无权限不是借口。
