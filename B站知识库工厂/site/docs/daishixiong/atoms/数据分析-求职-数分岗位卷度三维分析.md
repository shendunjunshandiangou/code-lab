# 数分岗位卷度三维分析

## 一句话定义

判断数分「卷不卷」看三轴：岗位数量、入门门槛、薪资；再区分就业卷与从业卷——数分比业务卷、与产品接近，薪资高于二者。

## 详细说明

| 维度 | 数分位置 |
| --- | --- |
| 岗位数量 | 公司内最少（业务>开发>产品>数分） |
| 入门门槛 | 高于业务/产品，低于开发 |
| 薪资 | 低于开发，高于产品/业务 |
| 从业卷度 | 因公司/团队而异，非岗位固有 |

结论：一分内卷一分收获；除业务外其他岗位都卷，按兴趣与职业形态选择。
