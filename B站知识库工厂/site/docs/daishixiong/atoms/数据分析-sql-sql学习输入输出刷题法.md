# SQL 学习输入输出刷题法

## 一句话定义

学 SQL 的输入（课程/文章/面经）和输出（刷题）都是刷题；用 Excel 操作迁移理解每个关键字，做不出题先判断是思路问题还是语法问题。

## 详细说明

### 学习路径

1. **Excel 迁移**：每个 SQL 关键字对应 Excel 什么操作
2. **输入**：刷题、课程、文章、面经
3. **输出**：继续刷题

### 问题定位

| 症状 | 可能原因 |
| --- | --- |
| 完全没思路 | 数据处理流程不理解（Excel 也不会） |
| 有思路写不出 | 某个 SQL 语法点未掌握 |

有思路后用 Excel 或 SQL 都能复现，才算真正掌握。
