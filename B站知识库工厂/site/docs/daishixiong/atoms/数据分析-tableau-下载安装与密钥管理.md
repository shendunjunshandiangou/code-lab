# Tableau 下载安装与密钥管理

## 一句话定义

Tableau 从官网下载安装后可 14 天试用；正式使用需产品密钥激活，学生可申请一年免费密钥，每个密钥最多绑定两台电脑。

## 详细说明

**下载**：从 Tableau 官网版本列表选最新版；大陆不支持 Win32，需台湾站点或独立老版本链接；下载慢可用迅雷加速。

**安装**：默认安装到系统盘（需 2–3 GB），**不要自定义安装**以避免路径问题。

**账户**：官网注册邮箱账户并激活；首次安装每台电脑可试用 14 天。

**密钥管理**：帮助 → 管理产品密钥 → 使用产品密钥激活。每个有效密钥最多激活 **两台电脑**，多台使用需先停用旧设备。

**学生账户**：在读生凭学生证/学历证明申请 **一年免费** 激活码，信息须与证明文件完全一致；审核 1–3 个工作日，失败可重新上传。
