# 大厂校招数分最低实习配置

## 一句话定义
大厂校招数分最低配置：2 份数分实习 + 其中 1 份大厂背书；零实习找校招非常难，相关实习指行业或岗位至少一个相关。

## 详细说明

| 目标 | 最低配置 |
| --- | --- |
| 大厂 | ≥2 份数分实习 + 1 份大厂任意岗位背书 |
| 中场 | ≥1 份数分实习 + 工具技能和分析思路 |
| 小厂 | 1 份相关行业实习 + 成熟工具技能 |

- 大厂面试重分析思维和软实力，弱考察工具（默认笔试+实习已证明）
- 中场可能继续追问工具技能
- **相关实习**：行业或岗位至少一个相关（游戏运营实习面新能源数分不搭）
- 至少 2 份实习才能进大厂校招面试环节
