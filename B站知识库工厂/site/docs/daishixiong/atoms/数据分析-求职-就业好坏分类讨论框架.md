# 就业好坏分类讨论框架

## 一句话定义

别问「好不好找工作」这种宏观空题，应拆解为「哪些行业/情况好找、哪些不好找」，再把资源投到好找的方向。

## 详细说明

**问题重构**：
- 培训机构必答「有机会」或贩卖焦虑，参考价值低
- 正确问法：明年**哪些行业、哪些情况**好找工作

**双向因素**：

| 侧 | 内容 |
| --- | --- |
| 个人 | 能力、简历、面试 |
| 环境 | 岗位供给（个人无法左右） |

**分析师方法**：
1. 岗位由企业提供 → 看行业景气、政策、融资
2. 抓「好找」的方向集中投入（无法穷尽所有「不好找」行业）
3. 结合自身牌面选赛道

类比电商选品：不是分析所有水杯好坏，而是找卖得最好的品类放大资源。
