# 领域用户定位

## 一句话定义

分析师属于 Python **第四类「领域用户」**：非程序员，用 Python 自动化「手动也能做」的工作以提效；只需基础语法 + 领域包（Pandas），且须先有业务经验。

## 详细说明

**四类人群**：
1. 语言创造者 2. 工具/包创造者 3. 应用开发者 4. **领域用户（分析师）**

**领域用户特点**：
- 手动做不出的，代码也做不出
- 核心价值是**效率**，不是开发产品
- 须先积累岗位/行业经验再学工具（与 Excel/SQL/BI 同理）

**知识深度**：基础语法 + Pandas 等常用 API 即可，不必程序员级深度。

**业务价值**：工具提效；**增长靠人的策略**，工具本身不能带来收入。
