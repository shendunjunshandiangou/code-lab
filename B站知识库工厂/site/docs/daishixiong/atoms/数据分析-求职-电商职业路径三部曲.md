# 电商职业路径三部曲

## 一句话定义

电商理想路径：服务商入门 → **拼多多**（第一站平台）→ 京东等平台 → 最后带信息差去**品牌方**变现；最终归宿是品牌方或创业。

## 详细说明

**第二份去平台——第一站只能拼多多**：
- 淘天/京东因商业间谍问题不要其他平台现成电商人
- 拼多多英雄不问出处；卷出认证后跳其他平台很顺
- 建议再补京东（高客单）掌握全貌

**为何品牌放最后**：
- 平台员工工资不如品牌方，晋升岗也少
- 应带平台决策逻辑与信息差去品牌变现

**收入预期**：平台间跳，分析师/运营年薪四五十万可期；品牌方卷到负责人可达百万。

**完美叙事**：起于市井（服务商）→ 发于平台 → 品牌方指导小公司——涨幅 50%～100% 可能。
