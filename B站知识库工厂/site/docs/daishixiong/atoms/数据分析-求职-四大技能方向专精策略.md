# 四大技能方向专精策略

## 一句话定义

数分有四个方向（数仓、取数、业务分析、算法），专精一个能上岸约 20 万，但需 2～3 个方向叠加才具备稀缺竞争力。

## 详细说明

| 方向 | 定位 |
| --- | --- |
| 数据工程/数仓 | 数据入库与建设 |
| 数据取数 | 清洗、切配 |
| 业务分析 | 包装数据、产出策略（经济价值最高） |
| 算法/机器学习 | 卷学历技术，多数走工程师 |

- 单方向：能上岸，约 20 万年薪，竞争力脆弱
- 取数 + 业务分析：大厂约 40 万级核心人才
- 数仓 + 取数：从脏乱差环境到 BI 看板
- 跨三个方向：市场稀缺性最高
