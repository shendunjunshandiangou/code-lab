# 可以不实习不能不面试

## 一句话定义

大二大三备暑期实习：可没时间去实习，但不能没面试机会——改简历混大厂面试练手，面了不去无背调。

## 详细说明

- 毕业时间写大三、一周五天；构建大厂风格简历专用于面试
- 大一大二就开始混面试 → 大三大四吊打同龄人
- 实习不挑：1～3 个月，不对就走；学生身份容错高
