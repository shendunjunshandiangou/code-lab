# 互联网行业入行门槛

## 一句话定义

基于 Boss/拉勾近 10 万岗位：学历本科够用、3～5 年经验需求最大（45%）、各岗位需 Office + 专属工具并持续学习。

## 详细说明

| 维度 | 数据/结论 |
| --- | --- |
| **学历** | 本科够用，远低于金融/考公；看能力多于学历 |
| **经验分布** | 3～5 年需求 45%；3 年以下 36%；运营 1～3 年需求 45% |
| **薪资（一线 3～5 年）** | 开发/产品/数分月薪 ~1.7 万；非一线也能保证 1 万+ |
| **技能** | Office 三件套 + 岗位工具；需持续学习（如美团要求 SQL 取数） |

**入行路径参考**：戴师兄先做运营（6000/月）→ 转型数据分析。

**建议**：已互联网化行业去头部企业；过于传统行业看眼前薪资。
