# 转行简历模拟升级法

## 一句话定义

找原经历与目标岗位重叠的职能 → 学会 JD 要求的工具技能 → 把新技能用到旧职能里模拟目标岗位 → 输出对口的新经历描述。

## 详细说明

操作顺序：

1. 对照 JD 拆解，找出原工作中相关职能
2. 学习并掌握目标岗位的工具与技能
3. 将新工具/技能应用到原工作场景，模拟目标岗位产出
4. 重写经历，使其读起来像目标岗位的工作

**铁律**：必须真实掌握并实操过才能写进简历；胡编在笔试/面试/背调中会翻车。
