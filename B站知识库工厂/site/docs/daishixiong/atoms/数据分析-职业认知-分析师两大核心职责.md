# 分析师两大核心职责

## 一句话定义

成熟数据分析师就干两件事：给业务数据支持，以及基于分析为业务出谋划策。

## 详细说明

**职责一 · 数据支持（占 60%～80% 时间）**
- 七字概括：**取数、图表、大监控**
- SQL 取数 → Excel/Python 加工 → Tableau/PBI 日常监控看板
- 无现成数据时：爬虫或提需求让开发建表

**职责二 · 分析决策（价值所在）**
- 结合数据与业务给方案、算投入产出比
- 沟通会（日/周/月）收集整合信息；分析会针对痛点专项深度分析
- 用业务公式从数据中提取隐藏信息，需结合场景
