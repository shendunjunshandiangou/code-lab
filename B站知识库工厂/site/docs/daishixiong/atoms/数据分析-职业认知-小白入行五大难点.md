# 小白入行五大难点

## 一句话定义
零基础/转行者入行数据分析必然依次撞上的五个问题：学什么 → 学不会 → 学了忘 → 没人要 → 用不上，每个都有对应解法。

## 详细说明

| 难点 | 解法 |
| --- | --- |
| 1 学什么 | 从公司数据链路推导：必学只有 SQL，路线 Excel → BI → SQL → Python（后两者可换序），见 [数据分析-工具-公司规模数据链路与必学工具](/daishixiong/atoms/数据分析-工具-公司规模数据链路与必学工具.html) |
| 2 学不会 | 费曼学习法，核心在输出：做出作品 + 讲得出来，见 [数据分析-工具-费曼输出检验与知识诅咒](/daishixiong/atoms/数据分析-工具-费曼输出检验与知识诅咒.html)；没时间就用 [数据分析-工具-百关键词快速入行法](/daishixiong/atoms/数据分析-工具-百关键词快速入行法.html) |
| 3 学了忘 | 要有忘的觉悟——大脑像系统会自动清理"回收站"，现在用不上的一定会忘；要用时能快速捡回。不想忘只有两条路：上岸后每天用，或每天复习 |
| 4 没人要 | 简历没响应/写不出/面试答不上，先诊断原因（投递→追问反馈→定位问题），再针对性改简历和面试 |
| 5 用不上 | 幸福的烦恼——学的东西全用上才可怕（所有活都堆给你），用不上先放着，要用时现捡 |
