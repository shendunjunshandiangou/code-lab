# AB测试实验基础

## 一句话定义

策略上线前用随机分组对照实验验证效果，设计主指标+护栏指标，样本量和周期足够，P&lt;0.05 为显著。

## 详细说明

- 不要全量上线（大力出奇迹），先灰度/随机分组
- 主指标：策略直接影响的指标（12 期占比、客单价）
- 护栏指标：避免副作用（成交率、投诉数）
- 样本量足够（几十到上万）、周期够长（避免新奇效应）
- P 值 &lt; 0.05 = 巧合概率 5% 以内
