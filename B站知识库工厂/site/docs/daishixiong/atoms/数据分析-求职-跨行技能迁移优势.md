# 跨行技能迁移优势

## 一句话定义

转行并非纯零经验——从业年限带来的沉稳和跨领域可迁移技能是优势，只是垂直方向上乍一看不对口。

## 详细说明

**9 年土木转数分案例**：
- 面试中从业年限是**优势**——经历多了更沉稳，见的多了技能能迁移
- 并非「纯零经验」——只是垂直方向（数据分析）上一眼看是零经验
- 真正接触业务后发现很多能力可迁移

**与「零经验转行」话术的区别**：
- 不要把自己说成完全空白
- 强调前行业积累的分析思维、项目管理、业务理解等可迁移能力
