# 透视表筛选排序与布局

## 一句话定义

透视表支持维度值筛选、按度量排序、多种报表布局；日期字段自动组合年月日，值字段设置与开始栏格式作用范围不同。

## 详细说明

### 筛选

- **维度值筛选**：在该维度汇总层面筛选（区域 >1000 万筛的是区域汇总，不是省份行）。
- **看到的行上筛数值**：点击透视表空白处再添加筛选器。
- 值字段设置：对整个值字段统一格式；开始栏格式：只对选中单元格。

### 排序

维度默认按文本/拼音排序。按度量排序：右键 → 其他排序选项 → 选度量字段降序。内外层维度分别设置。

### 布局

- 压缩形式 / 大纲形式 / 表格形式；可重复所有项目标签。
- 分类汇总：每组底部/顶部小计。
- 固定默认布局：文件 → 选项 → 数据 → 编辑默认布局 → 导入当前设置。

### 日期特殊性

日期入行区自动组合年/月/日；文本型日期无法组合也无法设数字格式。
