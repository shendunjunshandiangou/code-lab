# 骑驴找马跳槽原则

## 一句话定义

跳槽必须拿到下家 offer 再提离职——骑驴找马，避免裸辞意外与简历硬伤。

## 详细说明

- 待满 1 年 + 找到更高薪下家 + 简历不花 → 可果断跳
- 前 3 年薪资相对次要，但高薪往往意味着更重视你的能力
- **绝不要**未拿到 offer 就裸辞——爽完后面临稳定性质问
- 与 [数据分析-求职-每份工作干满一年原则](/daishixiong/atoms/数据分析-求职-每份工作干满一年原则.html) 配合：未满一年慎跳

**补充（系统求职课）**：
- 口头「offer 稳了」不算数，可能被更好候选人顶掉，必须拿到正式 offer 再提离职
- 在职能随时离职本身就是对企业的威慑；最优策略是拿 3 个 offer 对比后果断走
- 对糟糕领导最有效的反抗：说服同事一起找好下家集体离职（公司用离职率/在职时长考核领导）
