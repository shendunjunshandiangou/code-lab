# Tableau 标记卡标签与工具提示

## 一句话定义

Tableau 的「标记」卡可将度量拖到标签（常显数值）或工具提示（悬停显示）；顶部 T 按钮可快速切换标签显示。

## 详细说明

**标签**：将度量（如新增确诊）拖到「标签」，数值常显在柱顶/扇区旁。

**工具提示**：将多个度量拖到「工具提示」，鼠标悬停时才显示（如新增确诊、新增治愈、新增死亡）。

**快捷操作**：
- 顶部 **T** 按钮：显示/隐藏标签
- **Ctrl+拖拽** 字段：复制到另一架子，避免重复拖放
- 右键标签 → **旋转标签**：竖排改横排
