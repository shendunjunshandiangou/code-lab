# Tableau 喜乐君学习路径

## 一句话定义

学 Tableau 应：基本图形+字段分类 → 数据模型 → 表计算与 LOD；勿走炫酷/复杂报表偏路；补足行业分析能力。

## 详细说明

1. **基本图形** + 理解 **数据结构/字段分类**（分析大厦基础）
2. **数据表与数据模型**（大规模分析前提）
3. **表计算与 LOD 表达式**（高级问题必备，否则停留简单表达）
4. **勿走偏**：炫酷、复杂报表在企业 **无前途**
5. **行业分析师**：航空/金融/制造业等与领导同步、提供决策建议

工具原理限制变现程度；缺的是行业深度而非只会工具。
