# Tableau 国内购买部署与成本

## 一句话定义

Tableau 未退出中国；国内经阿里云/优阅采购，小企业约 1 万元可用，3–5 分析师团队首期 Creator+Viewer 配置约 3–4 万元/年。

## 详细说明

**现状**：未曾退出，无需「归来」；无法官网直购。

**渠道**：阿里云、优阅（树懒）；价格与国际一致 + 换汇服务。

**个人**：Public 免费；Desktop 约 900 美元/年。

**小企业**：约 **1 万元** 获全套，按用户扩展。

**典型团队**（3–5 分析师 + ~10 业务查看）：首期约 3 Creator + 3–5 Viewer，重点创建内容与观看评论；二期再扩账号或嵌入式开发，首年 **3–4 万元**。
