# 求职漏斗模型

## 一句话定义

数据分析求职是典型漏斗：观望→学习→具备实力（最大筛人环节）→投简历→面试→offer→入职；先获得准入资格，后续环节难度大幅下降。

## 详细说明

**漏斗各环节**：
1. 外围观望（「数分挺火」）
2. 学习投入
3. **具备实力的求职者** ← 90%+ 的人在此被淘汰
4. 投简历 → 面试 → offer → 入职（还有被老东家留住、公司反悔等变数）

**关键数据**：完整数据分析课程完课率通常 1%～3%。

**三原则**：
- 先具备准入资格，再优化简历/面试
- 目标先做前 10%，不要一上来就想第一
- 多做少想：感兴趣就先学，少纠结「能不能学会」
