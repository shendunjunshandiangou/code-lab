# 数据是国家战略资产

## 一句话定义

从企业（阿里/瑞幸数字化）到国家（数据主权、TikTok 禁令、国家数据局），数据是信息时代核心生产力；个人应去数字化行业并用数据技能武装自己。

## 详细说明

- **企业**：瑞幸靠全面数字化生产逆境重生——门店/供应链全自动数据记录
- **国家**：数据本土化、算法/算力禁运、地理数据安全（滴滴审查）
- **个人建议**：① 去数字化企业 ② Excel/分析思维/业务监控等实用技能 ③ 兴趣与功利并重，不必人人做专业分析师
