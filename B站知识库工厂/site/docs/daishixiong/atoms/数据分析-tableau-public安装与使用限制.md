# Tableau Public 安装与使用限制

## 一句话定义

Tableau Public 免费安装练习 Tableau 操作，但无法保存工作簿到本地；不推荐破解版，正式需求见 Desktop 安装教程。

## 详细说明

- 官网下载，默认安装；破解版易出现图标不显示等 bug。
- **Public 限制**：不能保存文件到本地，仅适合练习用法。
- 正式版 Tableau Desktop 见 [数据分析-Tableau-下载安装与密钥管理](/daishixiong/atoms/数据分析-tableau-下载安装与密钥管理.html)。
- 本动态图表教程全程用 Public 演示教学。
