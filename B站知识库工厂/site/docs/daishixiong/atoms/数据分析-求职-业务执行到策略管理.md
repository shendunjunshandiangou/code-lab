# 业务执行到策略管理

## 一句话定义

岗位晋升经典路线：先做业务执行（低门槛上手）→ 做出业务价值 → 转策略/管理岗拿高薪；分析师本质是策略岗的一种。

## 详细说明

**为什么先做执行**：
- 无门槛、先上手，做出业务价值比纯分析更重要
- 「怎么做」比「分析来分析去」更有价值
- 戴师兄本人从运营业务端转到数据分析

**经典路线**：

| 路线 | 路径 |
| --- | --- |
| 销售线 | 销售 → 销冠 → 销管 → 策略/产品/获客 → 总部中台 |
| 通用线 | 运营/产品 → 数分 → 中高管 |

- 分析师 = 基于数据做测算、实验，判断策略赚还是亏
- 纯数据分析/产品经理年薪 30～50 万区间提升痛苦；30 万是多数人软性天花板
- 到中高管时岗位名称消失，才是真正高薪
