# 流程分析侦探思维

## 一句话定义

分析基础是像侦探一样了解业务全流程——竞品为什么好、业务下滑要倒推每一步发生了什么。

## 详细说明

- 不了解业务信息就不可能有分析结论
- 类比恋爱：不知道对方喜好就无法吸引
- 双 11 下滑 → 倒推定目标、策划、准备、执行各步
- 先拆行为漏斗 → 再看每环节时序对比 → 60% 基本盘清楚
