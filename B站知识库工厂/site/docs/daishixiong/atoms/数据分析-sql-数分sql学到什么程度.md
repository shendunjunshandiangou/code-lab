# 数分 SQL 学到什么程度

## 一句话定义

数分岗位 SQL 分三档：有兴趣不设上限；求职需掌握语法+套路题+逻辑嵌套；工作中重点是业务需求翻译成 SQL。

## 详细说明

### 三档标准

| 档位 | 要求 |
| --- | --- |
| 兴趣驱动 | 不设上限，多学都会用上 |
| 求职 | 语法（WHERE/HAVING/聚合/窗口函数）+ **套路题**（连续登录、Top N）+ 逻辑嵌套 |
| 工作 | 把业务口语翻译成 SQL；很多需求因数据不支持而无法完成 |

### 笔试 vs 面试

- **笔试**：前几题考语法，后几题（4–5 题）趋近套路题
- **面试**：面试官随口出套路题，没时间提前准备

### 关键判断

做不出题不一定是 SQL 语法问题，可能是**数据处理思路**缺失（Excel 也不会做）。
