# 转行看相对提升原则

## 一句话定义

转行是否值得，看相对原岗位的提升而非绝对薪资；早上岸一天多享行业红利。

## 详细说明

- 例：传统行业应届生五六千 → 转行数分北上约 8000，相对原圈层是提升
- 大专学数分可能脱离大专就业圈层
- 数据型人才大势所趋，但坑位在减少；等所有人都会数分，它变成硬性技能（如字节各岗要求 AB 测试）
- 看自己的实际所得，不被画饼；有所得就该果断付出
