# 订单明细驱动分析思路

## 一句话定义

从成交额出发逆向梳理曝光→转化→成交流程，拿到订单明细+用户行为+用户画像三表做交叉分析，是零基础积累业务分析经验的核心方法。

## 详细说明

**三步串联**：
1. 拿到**订单明细** + **用户行为明细** + **用户画像维度**
2. 识别**核心成交用户**画像（性别/年龄等）
3. 盯核心用户在各环节比例 → 成交下降时诊断活跃度 → 产出精准策略

**行为分析**：页面停留、活动采样时间等不说谎，是策略有效度的直观呈现。

**逻辑链**：用户属性表 → 成交与活跃行为 → 业务质量贯通 → 定位哪部分用户策略没做好。
