# 三剂心态强心针

## 一句话定义

跳槽前心态崩了时，用「休息、为简历打工、抓大放小」三法调整——优于裸辞留硬伤。

## 详细说明

1. **赶快休息**：请假/病假、约饭、大餐、电影、游戏——无心理负担，精力恢复后心态平缓
2. **为简历打工**：工作是为塑造高薪履历，与恶心老板同事无关——降低被 PUA 概率
3. **抓大放小**：将军赶路不追小兔——有更重要目标时，路边干扰睁一只眼闭一只眼

**延伸（职业规划篇）**：工作外建立第二条投入曲线（游戏/博主/健身）；精神降维——「打工人都是蝼蚁，再厉害也改变不了太多」。
