# 五个职场跃迁觉醒点

## 一句话定义

戴师兄用五个觉醒节点串联从年薪 10 万到年创 100 万价值的职业路径：选对赛道 → 掌握全流程 → 升级战略视角 → 深耕头部行业 → 发展管理能力。

## 详细说明

| 觉醒 | 核心认知 | 对应薪资档 |
| --- | --- | --- |
| 第一次 | 选择赛道比努力更重要 | ~10 万 |
| 第二次 | 掌握分析全流程，从被动取数到主动创造价值 | 20~30 万 |
| 第三次 | 职级与权限是隐性增值；行业选择决定天花板 | ~30 万 |
| 第四次 | 深耕头部行业（电商）才有长期价值 | 40~50 万 |
| 第五次 | 管理能力才是突破 50 万瓶颈的关键 | 100 万+ |

- 数据分析师能很有确定性地完成 10 万到 50 万的过渡，但无法帮你达到职业终点
- 50 万之后每人方向不同，分析能力会泛化，不局限于数据、岗位、行业
