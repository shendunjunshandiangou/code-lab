# 笔试面试套路题方法论

## 一句话定义

SQL 笔试面试最常考「套路题」——连续登录、Top N、在线人数等固定模式；没见过就做不出，刷够题才能举一反三。

## 详细说明

### 为什么套路题卡住大多数人

- 不是别人更聪明，而是**你没见过这类题**
- 类比数学题：没刷过同类题，纯靠脑子想几乎不可能
- 学会套路后题变得简单

### 常见套路类型

- Top N（窗口函数排名）
- 连续登录（日期差/分组标记）
- 在线人数
- 留存/复购率口述

### 考察维度排序

1. **套路题**（最主要）
2. 语法（结合业务场景，不单考函数）
3. 逻辑嵌套（多层子查询）
