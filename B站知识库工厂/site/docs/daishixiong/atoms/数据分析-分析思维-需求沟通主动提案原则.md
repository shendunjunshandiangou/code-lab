# 需求沟通主动提案原则

## 一句话定义

数据分析需求沟通时，应主动拿更好的方案解决对方问题，而非被动执行——避免被牵着鼻子走和需求拉扯。

## 详细说明

**错误做法**：别人给什么需求就做什么，非常被动。

**正确做法**：自己对需求理解后提出更好方案与对方对比。

**类比**：
- 别人说要吃饱 → 给营养均衡的餐（既能吃饱又能吃好）
- 别人说要吃饱 → 给三个大馒头 → 吵架「不爱吃」→ 陷入需求拉扯

**分析师角色**：默认拿着更好的方案去解决对方问题，而不是将将好完成对方的东西。

**来源**：戴师兄入行时带教老师传授的经验。
