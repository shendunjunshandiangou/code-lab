# 杭州入门城市选择

## 一句话定义

数据分析入门首选杭州而非北京——电商小企业密集、用人标准相对低、研究生有人才落户补贴，上海北京更适合跳槽涨薪而非入门。

## 详细说明

**杭州优势**：
- 一大堆小企业搞电商、主播工作室等 → 数据分析需求多
- 用人标准比北京低
- 研究生人才落户可领补贴、申请廉租房，城市欢迎度高

**城市定位分工**：
- **杭州**：入门首选
- **上海/北京**：适合高薪分析师跳槽涨薪
- **河北等第二产业为主地区**：数据分析需求少，只能往北京投但竞争更激烈

**最快上岸路径（三件套）**：
1. 城市选杭州
2. 做出自学课项目（外卖 Excel 报表 + 金融 Tableau 看板）
3. 直接在杭州找数据运营/数据专员/数据分析，先别管公司大小
