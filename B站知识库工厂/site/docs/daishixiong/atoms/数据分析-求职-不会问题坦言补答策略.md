# 不会问题坦言补答策略

## 一句话定义

面试遇到不会的问题，坦言不了解并承诺回去研究补答（可发分析报告），不要瞎扯——瞎扯反而暴露分析能力问题。

## 详细说明

**推荐话术框架**：
- 坦言对该业务/问题了解不够深入
- 承诺今晚/明天搜索资料并做分析
- 通过微信/招聘软件发送具体分析报告

**适用场景**：
- 工具挑刺问到不会的（K-means 原理等）——正常，目的是探底
- 对公司业务场景不了解

**禁忌**：没有任何储备还硬说，会被认为分析能力有问题。
