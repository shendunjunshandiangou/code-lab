# 辩证看待劝退信息

## 一句话定义

劝退帖多因无追责风险；岗位无绝对好坏只有适不适合——把言论转成客观特点，自行验证后判断是否接受。

## 详细说明

1. 花 2–3 天收集劝进/劝退信息
2. 记「特点」而非优缺点
3. 客观验证：上限查职级、薪资查招聘网、替代性看是否工具人
4. 结合个人偏好判断能否接受
