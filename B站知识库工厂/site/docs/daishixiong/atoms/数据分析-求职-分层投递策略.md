# 分层投递策略

## 一句话定义

求职分三阶段：非意向岗位练手 → 非意向城市/行业海投 → 意向城市/行业定投，避免大厂冷冻期和不必要面评污点。

## 详细说明

| 阶段 | 做法 | 目的 |
| --- | --- | --- |
| 1. 乱投练手 | 投非意向岗位（想投数分先投产品/运营） | 验证简历硬实力，1% 响应即合格 |
| 2. 海投 | 投意向岗位，但避开意向城市/行业 | 测响应率，保护意向岗位不被投完 |
| 3. 定投 | 针对性改简历和打招呼话术，冲刺意向城市/大厂 | 提高成功率 |

- 想投上海电商：先投广州外/content/游戏/本地生活中小厂
- 非意向城市有响应 → 意向城市也会有响应
- 每日记录投递表，分析哪些岗位响应度高
- 复盘：看哪里投递有响应、哪里没有
- **公司体量顺序**：也可按小厂 → 中厂 → 大厂分批投递，先练手再冲刺（2024 就业局势视频）
- **基准数据（系统求职课）**：平均投递响应率约 1%、面试邀约率约千分之一，高于此即简历/面试较好；乱投 2～3 天没反馈就该改简历而非继续投；定投阶段打招呼语要按目标公司背景定制；机会有限（大厂被拒有半年冷冻期），本质是训练集/测试集思维——用表格量化管理，「像分析师迭代策略一样迭代简历和投递」
