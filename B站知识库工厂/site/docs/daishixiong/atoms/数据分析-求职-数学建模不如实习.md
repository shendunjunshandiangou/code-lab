# 数学建模不如实习

## 一句话定义

在校生求职时，数学建模比赛远不如实习有用——企业不搞数学建模，校内活动不能替代真实工作场景。

## 详细说明

**核心逻辑**：
- 企业不需要数学建模，随便问老板大多说不需要
- 指望建模比赛给简历加分，属于「没接触过外面世界」的稚嫩想法

**打猎类比**：
- 学打猎应该直接学打猎，不是在校把体测跑步练到满分
- 真正厉害的猎人用陷阱和技巧，不是跟猎物赛跑

**更有用的替代**：
- **实习**——直接对应企业需求
- **和导师做项目**——相对有用，练服从性、有人指导
- 积累任何东西**立即写进简历**，变成硬实例再投更好的岗

**学习需要衡量标准**：
- 没有指导就没有标准——就像学英语不跟外教对话，永远不知道够不够用
- 只有实战（实习/项目指导）才知道学得够不够
