# 笔试真题不会做四步应对

## 一句话定义

题库不可能穷尽，遇到不会的笔试/业务真题按四步应对：提升实力、总结思路模板、面试前情报准备、事后穷尽答案入题库；境界上走「刷题 → 临场发挥 → 经验硬钢」三级。

## 详细说明

**四步方法**：

1. **提升实力**：真题本质是企业真实面临的问题；工作中不回避问题，解决得越多下次面试能答的越多。
2. **总结思路模板**：数据诊断题（先看环节 → 下钻定位异常指标 → 结合业务沟通找原因）；业务瓶颈题（区分策略问题/产品质量问题/执行问题）；AB 测试题（记三类指标：核心/北极星指标、辅助指标、护栏指标（存疑，原音「互联指标」））。
3. **面试前情报准备**：搜目标公司的商业模式、行业主要玩家（盯头部企业动态即可抓住行业脉络）、用户画像（知乎/脉脉现搜即得）、基本策略（电商可注册商家后台走一遍流程，用淘宝教育、巨量学、多多大学等平台方教程）。
4. **穷尽答案**：不会的题事后一定找到答案、整理进个人题库文档；把补充解答发给面试官还能逆转印象。

**答题技巧**：先与面试官确认口径（如转化率是到加购还是到成交），再用「第一/第二/第三 + 其中第 X 最重要」的拆解排序结构争取思考时间、留容错。

**三境界**：刷题（补差距）→ 临场发挥（积累够了能自由组织）→ 经验硬钢（行业专家）。思路保 60 分，积累保 80 分，实战出色才是 100 分。

**典型送分题**：每科最高分学号（牛客套路题）、「部分上升总体是否一定上升」（辛普森悖论）、广告定价（不是定出来是测出来的，广告费一般不超过给商家带来收益的 50%）。
