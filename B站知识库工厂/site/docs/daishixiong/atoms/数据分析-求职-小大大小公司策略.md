# 小大大小公司策略

## 一句话定义

公司切换遵循「小大小」：先去小公司（难度低、无空窗）→ 大公司积累背书 → 再回小公司变现/养老。

## 详细说明

| 阶段 | 逻辑 |
| --- | --- |
| 先小 | 难度低、无空窗；如哈啰/青桔 → 美团 |
| 再大 | 积累背书和经验；校招进大厂很少，多数社招积累后进 |
| 再小 | 变现/养老；大厂跳小厂一般不降薪 |

**选公司条件**（大厂够不上时）：
- 成立 2 年以上、约 50 人规模
- 有持续业务和现金流
- 有代教或需在事中获得成长

**电商特殊路径**（见 [数据分析-求职-电商职业路径三部曲](/daishixiong/atoms/数据分析-求职-电商职业路径三部曲.html)）：
- 服务商入门 → **拼多多第一站**（其他平台不要现成电商人）→ 京东补全 → 品牌方变现
- 拼多多有竞业协议；加班换 30%～50% 涨幅后跳其他平台
