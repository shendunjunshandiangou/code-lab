# 通投与定制两版简历法

## 一句话定义

求职先用泛化「通投版」进面试池练手、看响应方向，再针对高响应赛道（如电商）改「定制版」——不要一上来只写单一行业。

## 详细说明

- **通投版**：技能/项目面面俱到，无亮点也可，结构清晰有数字有作品集
- **定制版**：响应好、薪资好、赛道好时再改，项目全换该行业
- 错误：只写「电商分析」→ 只有电商要你，不知真实竞争力
- 校招可写本地生活/互金/电商等多行业项目泛化练手
