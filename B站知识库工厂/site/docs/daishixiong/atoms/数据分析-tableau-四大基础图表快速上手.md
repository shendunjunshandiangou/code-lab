# Tableau 四大基础图表快速上手

## 一句话定义

Tableau 通过拖拽维度与度量到行列、颜色、大小等「架子」即可出图；可视化领域最常用的是柱状图、条形图、饼图、折线图、散点图。

## 详细说明

**柱状图**：维度 → 列，度量 → 行；排序；视图 → 适合宽度。

**条形图**：维度 → 行，度量 → 列（与柱状图行列互换）；「转置」按钮可互换二者。

**饼图**：度量 → 大小，维度 → 颜色；标记卡选「饼图」；视图 → 整个视图放大；Ctrl+拖拽可复制字段到标签。

**折线图**：日期字段先改为日期类型 → 列（选精确日期/天/月等粒度）→ 度量 → 行。

**散点图**：一度量 → 列，另一度量 → 行；维度 → 详细信息与标签。

**课后思考**（原视频布置）：
- 饼图能否用累计确诊替代新增确诊？（与数据结构有关）
- 折线图如何改周/月/季度/年粒度？
- 散点图如何给各省加颜色区分？

拖拽出图原理与图表应用场景见后续 Tableau 课程。
