# 大厂分工与小公司补位

## 一句话定义

大厂高度分工（每人做很少但专精，面试造火箭工作打螺丝）；小公司缺位严重需分析师多职补位——被安排救火说明你是高质量人才，每份工作都为下一份做准备。

## 详细说明

| 类型 | 分析师体验 |
| --- | --- |
| **大厂** | 分工细，策划/算账/定目标/执行/监控/复盘/迭代各有专人 |
| **小公司** | 报表+数据建设+会议纪要+投放+客服文案等多职；「四处救火」 |

**分工原理**：专人切菜更快——分析师专门训练取数、洗数、看板、专项分析，效率高于业务团队人人学数据。

**心态**：小公司补位 = 充分感受业务场景，为跳槽积累素材。
