# 劳动力供需与就业压力

## 一句话定义

就业压力源于劳动力需求不足——企业能不能赚钱决定招不招人，GDP 增速放缓和内需疲软是核心因素。

## 详细说明

- 劳动力需求取决于企业盈利能力和信心
- GDP 从 10% 降至 5% → 就业压力增加
- 内需疲软 → 企业家谨慎扩张
- 增长本质是信心
- 20 年前需求>供给；现在存量竞争

**普通人应对**：选能帮企业创造可量化价值的技能方向，立即行动而非等待。
