# 深度广度四问吸收法

## 一句话定义

有了深度知识框架后，用四问吸收广度信息：看到什么 → 想到什么 → 能否用自己的话讲出/重构 → 用在何处；关注改变量，并模型化世界。

## 详细说明

四问：
1. 我看到了什么？（说不清说明难点或无用）
2. 看到后想到知识体系哪一块？要不要结合？
3. 结合后如何重构进自己的体系？能否讲出来？
4. 能用在哪个场景？

配套：作品思维、正反馈、降低坚持成本（避免牺牲感）；阅读要能融合用自己的话输出。
