# 多见识跟对人六字诀

## 一句话定义

提能力的速记口诀：多见识（攒信息差与眼界）+ 跟对人（跟对老师/路径，拿到正确方法）。

## 详细说明

**多见识**：
- 区分「可复用方法论」与「行业信息差」；老员工强常因信息差而非天赋碾压。
- 深耕行业 → 攒业务信息差；偏创业 → 攒 0→1 方法论。
- 多交朋友聊天，很多信息差搜不到。

**跟对人**：
- 学习方法浓缩为三字「跟对人」；老师不匹配则成绩/进度容易崩。
- 与百关键词入行、项目作品路径配套使用。
