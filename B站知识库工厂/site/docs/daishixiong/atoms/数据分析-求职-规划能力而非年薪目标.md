# 规划能力而非年薪目标

## 一句话定义

长远职业规划应聚焦「具体能力如何成长」而非「毕业几年年薪 XX 万」——虚目标在当前环境无容错空间，只会徒增内耗。

## 详细说明

**误区**：定 P 级、年薪百万等与你现在无关的大目标。

**正确做法**：
- 基于当前职业经历把手牌打好，选收益最大化路径
- 研究年薪百万者具备什么经验，拆到每天学多久
- 找自己最舒适、可长期持续的提升方式——慢但坚持会快
- 不跟别人比节奏；不压迫自己才是可持续规划
