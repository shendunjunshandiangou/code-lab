# 海外先小后大投递法

## 一句话定义

海外求职可泛化国内「先小后大」策略——如先投邻国再投本国，成功案例证明逻辑可迁移；留外发展 vs 回国可随时根据个体画像选择。

## 详细说明

- 德国留子案例：先投邻国→再投本国→成功
- 留外若生活独立开心可继续发展；回国做跨境电商欧洲市场有优势
- 戴师兄无大量海外辅导案例，不做保票承诺
