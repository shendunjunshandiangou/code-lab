# 转行低起点薪资策略

## 一句话定义

转行入门时对薪资不要有要求——数分最低五六千，干满一年很快涨到 8000–1 万，仍比传统行业安全员 4000 好太多；光脚的不怕穿鞋的。

## 详细说明

**案例对比**：
- 安全员 EHS 国企：4000 元/月、离家 600km
- 数分入门：最低五六千，干满一年 8000–1 万
- 辅导案例：小白自学 3 个月涨薪 3000+

**策略**：
- 先入行再涨薪，不要因起点低而放弃
- 硕士有优势，但必须先有数据相关实习/作品集
- 没有大学学历的人把技能+项目学到位也能上岸

**心态**：「光脚的不怕穿鞋的」——已经是最低起点时，反而没有包袱。
