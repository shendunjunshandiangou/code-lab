# AI替代判断标准

## 一句话定义

核心价值是「单一确定性映射」（A→B→C 重复标准化）会被替代；做「不确定性决策、承担责任」不会被替代。

## 详细说明

**会被替代**：
- 取数、做表、简单 SQL 等重复标准化工作
- 有大量数据可参考的定性映射

**不会被替代**：
- 信息不足时的复杂决策（黄仁勋：未来最稀缺人才）
- 深度人际协作、建立信任、交付限价服务

**企业现实**：大厂不授权第三方 Agent；token 成本高于人力；企业不愿把数据给外部 AI。
