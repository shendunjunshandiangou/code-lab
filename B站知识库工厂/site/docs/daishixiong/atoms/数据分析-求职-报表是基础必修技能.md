# 报表是基础必修技能

## 一句话定义

报表是将业务行为量化呈现、对外说服的主要形式——如同步枪，不管什么兵种，几百米内都必须会。

## 详细说明

- 报表不是中间阶段职业，而是**基础专项能力**——像厨师必须会炒菜
- 高级厨师做佛跳墙，但基本功必须全会
- 1～3 年岗位需求多报表/可视化/取数——跳槽时也最契合
- 高级形态：树状跳转报表（指标低于辅助线 → 下钻 → 定位业务对象），注释写清阈值与策略含义
