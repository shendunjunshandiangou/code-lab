# 求职转型重塑身份

## 一句话定义

求职/转行是重塑职业身份——看待问题、表达与解决思路都要转变，不是单纯学技能、过笔试面试。

## 详细说明

类比：学物理知识≠能当物理学家；学编程≠自动成为程序员视角。

转型需系统性改变：业务问题视角、面试阐述方式、解决思路表达。因此需要系统性求职课程而非碎片化技巧。
