# 考研短暂逃避决策

## 一句话定义

很多人考研并非为学历质变，而是毕业圈压力下「拖一手」的短暂逃避——高情商说是重新选择机会，低情商说是中国宝宝自己的 Gap。

## 详细说明

- 父母/学校/同学压力 → 「考研了父母就不烦」
- 并非都觉得能考上，仍随大流考
- 若考前后不做正确决策（实习/考公/读博方向），考研价值无法放大
- 考研是人生长期规划一环，非必达目标——不想考就安心找工作

**与量变考研**：普本升一本等量变，同样时间私企工作 1～2 年更有含金量。
