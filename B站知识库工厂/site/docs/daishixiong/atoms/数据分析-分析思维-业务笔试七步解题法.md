# 业务笔试七步解题法

## 一句话定义

开放型业务笔试题按「提取信息→结构化→列指标→数学关系→找影响因素→总结规律→分类作答」七步作答；答案不重要，思路才重要。

## 详细说明

| 步 | 动作 |
| --- | --- |
| 1 | 提取关键信息，理清业务机制/对象/目标 |
| 2 | 结构化，划分问题范围 |
| 3 | 列关键指标：量化>抽象，长期>短期（短期过得去时） |
| 4 | 指标比较转化为数学公式 |
| 5 | 找出影响比较指标的因素 |
| 6 | 总结规律/定律（如长尾效应） |
| 7 | 结合实践（A/B 测试）分类讨论，结论须量化 |

**原则**：笔试不重要，分析思维才重要；经验靠积累，思路今天就能用。
