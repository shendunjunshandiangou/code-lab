# Boss 直聘第一渠道

## 一句话定义

当前招聘 Boss 直聘为第一渠道；脉脉等薪资 1.5 万+ 再考虑；期望薪资区间往高写才能被推高质量岗位。

## 详细说明

- 好互联网岗位集中在 Boss；响应度高
- 期望薪资过低 → 推荐池岗位质量差、响应少
- 简历含大厂/电商等关键词；响应低可换新号（注意历史版本风险）
- 双号同投一家公司可摸需求、第二轮更稳

**渠道全景（系统求职课）**：
- 社招：脉脉是 HR/高管最高频的优质岗位平台（吃瓜顺便招聘，主页帖子还能塑造职业形象）；Boss 直聘中小企业最多、体量第一；拉勾（已被 51job 收购）、智联（国企多）适合海投试手；国聘多为二手中转
- 校招：官网 + 内推为主渠道，辅以校招社群、公众号汇总（警惕付费求职服务割韭菜）
- 实习：内推最稳（实习生带实习生）；实习僧/应届生求职网质量低但适合零经验破冰
