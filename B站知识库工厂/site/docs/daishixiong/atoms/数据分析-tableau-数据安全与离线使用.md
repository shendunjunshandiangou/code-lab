# Tableau 数据安全与离线使用

## 一句话定义

Tableau 分析数据只在本地/企业环境，不上传官方；上传的仅为注册与许可信息；可完全离线使用。

## 详细说明

**两类数据**：
1. **被分析的业务数据**：仅在自己环境，**不会**传输到 Tableau 官方
2. **注册/许可信息**：单位、使用模块等管理信息（正常需要）

**云端托管**（Tableau 托管 Server）：云服务商安全通常高于个人电脑；风险更多来自 **不规范使用 Server**（如数据提取保存在错误环境），而非云本身泄露。
