# Tableau 工作簿格式与工作区类型

## 一句话定义

Tableau 工作簿分 `.twb`（无数据）与 `.twbx`（含数据）；工作区分为工作表（单图）、仪表盘（多图组合）、故事（交互汇报）。

## 详细说明

**工作簿格式**：

| 格式 | 特点 |
| --- | --- |
| `.twb` | 仅保存图表与连接配置，不含数据；文件小、便于保密 |
| `.twbx` | 数据打包进工作簿；便于练习、导出、离线操作 |

**三种工作区**（底部图标）：

1. **工作表**：制作单一图表
2. **仪表盘**：放置多个工作表，不能在仪表盘内新建图表
3. **故事**：类似 PPT，组合工作表与仪表盘，支持交互展示与项目汇报
