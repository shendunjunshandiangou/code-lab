# 转行不可能三角

## 一句话定义

换工作时行业、岗位、公司规模三者最多同时满足两个；想一步到大厂数分违背 HR 录用逻辑。

## 详细说明

示例：

- 小游戏公司数分 → 大公司数分：行业+岗位，可行
- 游戏运营 → 电商大厂数分：仅行业沾边，极难

**建议优先**：行业 + 岗位——先在互联网找能写 SQL 的数据岗；退一步先做能接触数据的运营/业务，1～2 年后再转数分。

培训机构「一步成功」常伴随伪造简历，背调风险极高。
