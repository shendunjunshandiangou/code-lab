# 年底找工作！小心！

<BiliPlayer :bvid="'BV1yFU9YpErv'" />


**摘要**：年底求职避坑只需问两个问题——「这个工作为什么会招我？」「我的前任为什么会离职？」观察面试官反应即可判断岗位风险；追问时加上定心丸话术，还能增加谈薪筹码。

## 一、两个必问问题

1. **这个工作为什么会招我？**
2. **我的前任为什么会离职？**

观察面试官的**回答内容和神情变化**，基本能判断是否有坑。这类问题很少被候选人问到，多数面试官会比较慌张——**越慌，问题可能越大**。

## 二、如何解读回答

| 面试官说法 | 可能含义 | 建议 |
| --- | --- | --- |
| 前任因个人发展离开 | 公司薪资/晋升通道可能不行 | 继续追问晋升机制 |
| 前任被裁员 | 岗位需求不明确或要求过高 | 追问岗位 KPI 和稳定性 |
| 不方便沟通 / 支支吾吾 | 有所隐瞒 | 视为潜在风险信号 |
| 希望新人带来新增量 | 前任可能没做出成绩，岗位有风险 | 追问前任具体产出 |

若回答**不直接**，都说明岗位存在潜在风险，需继续追问。

## 三、追问话术：给面试官吃定心丸

面试到最后看的是硬实力，**不要怕追问**。在问题末尾加上：

> 基于以上信息，我会更好地准备岗位，让领导和团队省力。

效果：

- 掌握主动权，面试官不会觉得你「不服管」
- 表明提问是为了入职后更好表现
- 若对方回答不好或回避，**还能增加谈薪筹码**——定薪时可以说「有些风险你没告诉我，会增加职业选择的隐患」，对方要么更直接回答，要么没有底气压薪

## 四、总结

- 多问问题 = 收集更多信息，选 offer 更全面客观
- 挑到面试官的「刺儿」，不管怎样都有助于决策
