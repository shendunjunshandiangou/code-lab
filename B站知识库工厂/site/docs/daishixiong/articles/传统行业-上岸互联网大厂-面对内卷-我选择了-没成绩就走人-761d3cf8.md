# 传统行业上岸大厂：残酷真相与选择

## 一、嘉宾路径

- 经济学本科 → **管培生曲线救国**（轮岗进数据分析，约 1/4 管培生走数据方向）
- 转行大厂**商业分析师** → 现离职攻读港理工商业分析

## 二、传统行业 vs 大厂

| | 传统行业 | 互联网大厂 |
| --- | --- | --- |
| 一句话 | 一潭死水 | 当牛做马 |
| 薪资 | 低 | 跳槽涨幅 **40% 以上** |

## 三、管培生避坑

- 市面管培评价参差；需看清轮岗是否**核心岗** vs 一线杀鱼
- 乐言项目轮岗均为核心岗，比例走数据方向高于预期

## 四、为何离开大厂读书

- 「不想再当牛做马，想活得像个人」
- 大厂绩效文化：短期无成绩 → 快速淘汰

## 五、对转行者的启示

- 管培/非对口岗可作为**曲线救国**进数据方向
- 大厂适合攒经验与薪资 jump，但需接受绩效压力
- 长期规划可包含「上岸→积累→再深造/转型」

## 文章正文

戴师兄访谈「带你成长」系列首位嘉宾乐言：湖大经济学本科 → 传统行业管培 → 大厂商业分析师 → 即将赴港理工读商业分析，27 岁，工作约 4 年（城乡规划要读 5 年所以少一年）。

**一句话对比**：传统行业「一潭死水」；互联网大厂「当牛做马」。跳槽大厂薪资涨幅**40% 以上**。

**为何第一份工作选管培？** 一直想做数据，但没相关实习，直接面数据分析难——**管培曲线救国**，轮岗机制轮到数据分析。市面管培评价参差（有人轮岗去盒马杀三个月鱼），乐言项目轮岗都是**核心岗位**，约 **1/4 管培生**走数据方向——超出预期；因公司在建数据中台，战略规划需要大量数据人才，管培生反馈也好，两方面相辅相成。

**大厂残酷真相**：做不出成绩会**迅速被干掉**，人人焦虑。乐言选择继续读书而非留大厂——「不想再当牛做马，想活得像个人」；出去读段时间，起码做一段时间的人。

**对转行者启示**：管培/非对口岗可作曲线救国进数据方向；大厂适合攒经验与薪资 jump，但需接受绩效压力；长期规划可包含「上岸→积累→再深造/转型」——乐言并非否定大厂经历，简历背书和业务经验已到手。
