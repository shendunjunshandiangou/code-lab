# 数据透视表：Excel 最被低估的功能

**摘要**：半面影直播课分两部分——Power Query 合并表课后作业（4–6 月销售汇总），以及数据透视表从创建到 GETPIVOTDATA 的完整讲解，含切片器、计算字段/项、值显示方式（占比与环比）、透视图与动态排名图案例。

## 一、课后作业：Power Query 合并 4–6 月销售数据

### 数据结构

- 4–5 月历史表：含区域、省份、小组列。
- 6 月分华东/华南/华西北三张表，缺区域/省份/小组，列顺序华西北与另两张略有不同（列名相同即可合并）。
- 人员表：销售工号 → 区域/省份/小组。

### 操作流程

1. 新建空表「销售汇总数据」→ 从文件夹导入三区域 6 月表 → 过滤 `~` 缓存文件、排除人员表和汇总表 → 合并文件。
2. 导入人员表 → **合并查询**（左外部，连接键=销售工号）→ 展开区域/省份/小组。
3. 导入 4–5 月数据 → **追加查询**到当前表。
4. 关闭并上载至 → **仅创建连接** → 右键加载最终表到工作表。

> 大数据量场景推荐：新建空表通过 PQ 加载，避免直接打开百万行 Excel 卡顿。

## 二、快速创建数据透视表

### 自定义快速访问工具栏

插入 → 数据透视表 → 添加到快速访问工具栏 → `Alt+数字` 一键插入（如第 4 个按钮即 `Alt+4`）。

### 数据源三种方式

| 方式 | 特点 |
| --- | --- |
| 选中连续区域 | 默认框选；新增行需手动更改数据源 |
| 按列整列选中 | 新增行刷新自动纳入 |
| 转为智能表（Ctrl+T） | 容器自动扩展，新增行带角标提示 |

## 三、行列值字段

- **行/列**：放离散维度（区域、省份、期数等）。
- **值**：放可聚合的度量（成交金额等）；离散字段放入值区默认「计数」。
- 销售工号被识别为数值时会显示科学计数法，需先转**文本**再计数。
- **双击单元格**：查看该汇总值背后的明细行。
- **复制透视表**：必须复制完整透视表（含所有组成部分），粘贴后才有「数据透视表工具」选项卡；部分复制只得到静态数据。

## 四、筛选与排序

### 字段筛选 vs 值筛选

- 在**维度字段**上做值筛选：按该维度汇总值筛选（如在「区域」筛选 >1000 万，筛的是各区域汇总，不是省份行）。
- 要在看到的行上筛数值：点击透视表**空白处**再添加筛选器（否则筛选项为灰色不可用）。

### 按度量排序

维度字段默认按文本/拼音排序。要按成交金额排序：右键维度 → 排序 → 其他排序选项 → 选度量字段降序。内外层维度需分别设置。

### 值字段设置 vs 开始栏数字格式

- **值字段设置**：对整个值字段统一应用格式。
- **开始栏格式**：只对选中的单元格生效。

### 日期字段特殊性

日期放入行区会自动组合为年/月/日。仅日期组合字段可在值字段设置中改数字格式；文本型日期无法组合也无法设格式——需先把原列转为日期类型。

## 五、布局与选项

- 报表布局：压缩形式 / 大纲形式 / 表格形式；可重复所有项目标签。
- 分类汇总：每组底部/顶部显示小计。
- **固定默认布局**：文件 → 选项 → 数据 → 编辑默认布局 → 导入当前透视表设置。
- 常用选项：空单元格显示为 `-`；更新时自动调整列宽；启用显示明细数据（双击看明细的前提）。

## 六、切片器与日程表

- **切片器**：可视化筛选器，可连接多张透视表（报表连接）。
- **日程表**：专用于日期字段的切片器，适合连续月份选择；断选月份（如只看 4 月和 6 月）不符合日期分析逻辑。
- 透视表过多时务必**命名**，否则报表连接无法辨认。

## 七、组合、计算字段与计算项

### 自定义组合

选中多个维度项 → 右键「组合」→ 生成新分组字段（如省份分组）。

### 日期组合

日期字段可组合为年/季度/月/周/日；按周组合需设置「起始于」对齐周一–周日。

### 计算字段 vs 计算项

| 类型 | 含义 | 示例 |
| --- | --- | --- |
| 计算字段 | **不同列**之间运算 | 客单价 = 成交金额 ÷ 成交客户数 |
| 计算项 | **同一字段内不同数据项**之间运算 | 18 期 − 12 期 |

计算项需点击维度内的**数据项**（如具体期数）才能激活，点击数值区则灰色不可用。

### 值汇总方式

求和、计数、平均值、最大值、最小值、乘积等。不重复计数在普通透视表中有限制，需 Power Pivot + DAX。

## 八、值显示方式（重点）

在「值汇总依据=求和」基础上设置：

| 显示方式 | 含义 |
| --- | --- |
| 总计的百分比 | 当前值 ÷ 全表总和 |
| 列汇总的百分比 | 当前值 ÷ 所在列总和 |
| 行汇总的百分比 | 当前值 ÷ 所在行总和 |
| 父行/父列汇总的百分比 | 当前值 ÷ 上一级维度汇总 |
| **差异** | 当前值 − 基准值 |
| **差异百分比** | (当前值 − 基准值) ÷ 基准值 |

### 月环比快捷算法

- 值显示方式选「**差异百分比**」。
- 基本字段选「月份」，基本项选「上一个」。
- 公式等价于：`(本月 − 上月) / 上月`。
- **前提**：透视表中必须包含上一个月的数据；筛掉前月则无法计算当月环比。

日环比选成交日期 + 上一个；年环比选成交年份 + 上一个。

## 九、GETPIVOTDATA 函数

从透视表提取数据的专用函数：

```
=GETPIVOTDATA("成交金额", 透视表左上角单元格, "区域", "华西北", "业务组", "北京三组", "期数", "6-7")
```

- 在透视表选项中勾选「生成 GETPIVOTDATA」后，点击透视表单元格自动生成公式，再改参数。
- **两个硬性要求**：
  1. **可见**：透视表中必须算出该值且未被折叠隐藏。
  2. **唯一**：查询条件能定位到唯一单元格；期数放在行内且无列汇总时，无法直接取期数总计。

## 十、数据透视图与动态案例

- 选中透视表 → 插入柱状图/折线图/饼图/组合图。
- 组合图：大数值放主轴，环比（百分比）放**次坐标轴**。
- 透视图与透视表行列配置强联动，切换行列后图表结构随之变化。
- 切片器可控制透视图实现动态交互。

### 单月 Top3 + 月环比排名图

难点：月环比需要多个月同时在透视表中，但展示只要单个月。

解法：透视表保留全部月份计算环比 → 透视图/辅助区域通过 GETPIVOTDATA 或筛选器取单月 Top3 展示。

## 十一、已知限制

- 普通透视表无法对维度做「不重复计数」（如某区域不重复销售工号数），需 Power Pivot + DAX `DISTINCTCOUNT`。
- 无法直接求中位数，可用 MEDIAN 函数或 Power Pivot。

### 课后作业：PQ 合并 4～6 月销售数据

**数据结构**：4～5 月历史表含区域/省份/小组；6 月分华东/华南/华西北三张表缺三列，华西北列顺序与前两张不同但列名相同可合并；人员表用销售工号匹配区域信息。

**流程**：新建空表「销售汇总数据」→ 从文件夹导入三区域 6 月表 → 过滤 `~` 打开缓存、排除人员表与汇总表 → 合并文件 → 导入人员表 **左外部合并**（连接键=销售工号）展开区域/省份/小组 → 导入 4～5 月 **追加查询** → 关闭并上载「仅创建连接」→ 加载最终表到工作表。

大数据量推荐：新建空表通过 PQ 加载流程，避免直接打开百万行 Excel 卡顿；PQ 处理比在工作簿里写公式快。Win+方向键或拖窗口可分屏操作。

### 快速创建透视表

- 插入 → 数据透视表 → 添加到**快速访问工具栏** → `Alt+数字`（如第 4 个按钮即 `Alt+4`）一键插入。
- **数据源**：① 选中连续区域（新增行需改数据源）；② 按列整列选中（新增行刷新纳入）；③ **Ctrl+T 智能表**（自动扩展，推荐）。

### 行列值字段

- **行/列**放维度；**值**放可聚合度量（离散字段默认计数）。
- 销售工号被识别为数值会科学计数法 → 先转**文本**再计数。
- **双击单元格**看明细；**复制透视表须复制完整**（含所有组成部分），否则只剩静态数据。

### 筛选与排序

- 在维度字段上做「值筛选」筛的是该维度汇总（如在「区域」筛 >1000 万），不是明细行；要在看到的行上筛数值 → 点透视表**空白处**再加筛选器。
- 按度量排序：右键维度 → 排序 → 其他排序选项 → 选度量降序；内外层维度分别设置。
- **值字段设置** vs **开始栏格式**：前者对整个值字段；后者只对选中单元格。
- **日期**入行区自动组合年/月/日；仅日期组合字段可在值字段设置改格式；文本型日期需先转日期类型。

### 布局与选项

压缩/大纲/表格形式；重复所有项目标签；分类汇总位置；文件 → 选项 → 数据 → **编辑默认布局**可导入当前透视表设置。常用：空单元格显示 `-`；更新时自动调整列宽；启用显示明细数据。

### 切片器与日程表

**切片器**可视化筛选，可**报表连接**多张透视表；透视表多时必须**命名**。 **日程表**专用于日期连续选择；断选非连续月份不符合日期分析逻辑。

### 组合、计算字段与计算项

- **自定义组合**：选中多个维度项 → 右键「组合」。
- **日期组合**为年/季/月/周/日；按周组合需设「起始于」对齐周一。
- **计算字段** = 不同列运算（客单价 = 成交金额 ÷ 成交客户数）。
- **计算项** = 同一字段内数据项运算（18 期 − 12 期）；须点击维度内**数据项**才可用，点数值区则灰。
- 值汇总：求和、计数、平均等；**不重复计数**普通透视表有限制，需 Power Pivot + DAX。

### 值显示方式（重点）

在「值汇总依据=求和」下：**总计的百分比**、**列/行汇总的百分比**、**父行/父列汇总的百分比**、**差异**、**差异百分比**。

**月环比**：值显示方式选「差异百分比」，基本字段选「月份」，基本项选「上一个」→ 等价于 `(本月−上月)/上月`；**须含上一个月数据**，筛掉前月则无法算当月环比。日环比用成交日期+「上一个」；年环比用成交年份+「上一个」。

### GETPIVOTDATA

```
=GETPIVOTDATA("成交金额", 透视表左上角, "区域", "华西北", "业务组", "北京三组", "期数", "6-7")
```

勾选「生成 GETPIVOTDATA」后点击单元格自动生成再改参数。**要求**：① 值在透视表中可见且未折叠；② 条件能定位**唯一**单元格（期数放行内且无列汇总时难取期数总计）。

### 透视图与 Top3 排名案例

选中透视表插入柱/折线/饼/组合图；大数值主轴、环比百分比**次坐标轴**。透视图与透视表行列强联动。

**单月 Top3 + 月环比排名图**：环比需多个月同时在透视表 → 展示层用 GETPIVOTDATA 或筛选取单月 Top3。

### 限制

普通透视表无法对维度做不重复计数（如区域不重复销售工号数）→ Power Pivot + `DISTINCTCOUNT`；无法直接求中位数 → MEDIAN 或 Power Pivot。

戴师兄强调：数据透视表是分析师高频操作，熟练 `Alt+4` 与值显示方式（尤其环比）可显著提速——「学会数据透视表，Excel 速度直接翻 3 倍」。
