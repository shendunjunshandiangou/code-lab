# 外卖大战下半场：谁送得快，谁活下来

<BiliPlayer :bvid="'BV1Ase4zDECT'" />


**摘要**：京东入局外卖、美团加码小象超市、阿里升级闪购、抖音布局本地生活——巨头争夺的是**线下最后一公里交付能力**。互联网本质是比特帮原子省时间，但商品交付必须靠原子完成。

## 一、互联网巨头的底层逻辑

- 过去「人找饭」→ 现在「饭找人」：线上信息（比特）降低传递成本
- 电商、闪购、线上购票、软件打车——都是**帮用户省时间**发家
- 但**商品交付**比特搞不定，必须靠线下原子跑完最后一公里

## 二、当前拼什么

**拼十面（线下）的交付能力**——外卖、生鲜、数码每条赛道都在卷。

### 以外卖为例

- 护城河有限：骑手管理、算法调度、商务拓展均可复制
- 外卖 = **闪电版电商**；叮咚做外卖迁移成本≈0
- 京东属**自卫反击**：18 年美团闪购偷袭 30 分钟万物到家；3 年前再补刀

## 三、为何经济下行仍跑马圈地

- 本地生活/即时零售是**高频刚需 + 存量博弈**
- 谁掌握交付网络，谁锁定用户时间与消费入口
- 对分析师：关注**履约时效、骑手效率、单均成本、跨品类协同**等指标

## 四、分析框架启示

- 看大厂动作不能只看 C 端补贴，要看**线下基建与供应链复用**
- 「送得快」= 调度算法 + 骑手密度 + 仓配网络 + 商户覆盖的综合能力

## 文章正文

京东高调入局外卖，同时在生鲜、家电等领域多渠道发力；美团加码小象超市，阿里升级淘宝闪购，抖音大力布局本地生活。全球经济下行，国内大厂为何仍在疯狂跑马圈地？戴师兄带大家回到起点，盘一盘互联网大厂到底在争什么。

过去没有互联网，懒得开火出门觅食，开门、下楼、点菜、等待、回家，一来一回至少半小时。现在动动手指就能下单，安心等美食上门——核心变化是**从「人找饭」变成「饭找人」**：线上能看到商品信息，省掉了跑腿成本。现实世界的你由原子构成，线上信息由比特构成；比特可以主动上门，原子 largely 可以躺平，入户时间大幅缩短。各大互联网巨头，本质上就是靠**帮大家省时间**发的家：电商闪购终结排队，线上购票让窗口消失，软件打车告别苦等。

但互联网不是万能的。有个环节比特死活搞不定——**商品本身的交付**。纯靠比特吃不上饭，原子也不能完全躺平；比特跑得再快，终究无法穿透现实，最终还得靠线下的原子跑完**最后一公里**。

所以巨头打架拼的是什么？拼的是**十面（线下）的交付能力**。外卖、生鲜、数码，每条赛道都卷出火星。以外卖为例，护城河其实很有限：骑手管理、算法调度、商务拓展，京东都能搞；模式上外卖说白了就是**闪电版的电商**，叮咚做外卖几乎是现有能力复制粘贴，迁移成本约等于零。京东这次更像**自卫反击**——吃瓜群众可能不知道，2018 年美团就用闪购偷袭「30 分钟万物到家」，直插京东腹地；3 年前再补刀联合苏宁抢 3C。再不打，家都被拆完了。除了狠狠反攻外卖，京东到家、七鲜超市、京东家电也在全面开花。外卖既不是单纯的绝地反击，也不是单纯的规模扩张，而是**十面战场的全面布局**。

战鼓敲响，胜负未分。美团以百万骑手筑起高墙，京东正用多年淬炼的线下能力撬动城门。这场战争揭示的残酷真相是：互联网的终局，不再取决于谁更擅长操作比特，而是谁能在那些比特无法替代的原子缝隙中，榨取出最后一分效率——**谁送得快，谁活下来**。
