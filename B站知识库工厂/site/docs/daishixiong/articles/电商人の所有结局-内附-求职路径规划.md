# 电商人の所有结局与求职路径

<BiliPlayer :bvid="'BV1Le411k7d5'" />


**摘要**：做电商必遇四类公司——**平台、品牌方、供应商（含工厂物流）、服务商**。互联网大钱在电商；进平台起薪高；品牌/供应链/服务商各有路径；分析师在电商赛道性价比最高（参见已有电商原子）。

## 一、四类公司

### 1. 电商平台（人上人）

- 淘天、京东、拼多多、抖音电商、小红书、得物、B 站等
- 中国 top 互联网公司 2/3 与电商相关
- 起薪不低，哪怕销售/店小二也能在大城市稳定生活（忙累）

### 2. 品牌方

- 联合利华、宝洁、可口可乐等——货架多数快消来自少数巨头
- 发展前景好，分析师可跟品类/渠道数据

### 3. 供应商（生产+仓储+物流一体）

- 现代电商产研运一体化；工厂不考虑供应链/物流会烂地里
- 数字化管理（如快反）是趋势

### 4. 服务商（最易忽略）

- 平台/品牌外包生态庞大：代运营、MCN、数据服务、ERP 等
- 入门门槛可能更低，接触多品牌场景

## 二、求职路径规划

- **目标平台/品牌** → 先服务商/供应商积累行业 know-how → 再跳头部
- 数分在电商：订单/GMV/转化/供应链指标丰富，**最优赛道之一**
- 与 [数据分析-职业认知-电商是分析师最优赛道](/daishixiong/atoms/数据分析-职业认知-电商是分析师最优赛道.html) 互补：本视频补全四类公司地图

## 三、结局可能性（框架）

- 视频按职业阶段展开电商人多种结局（晋升、转行、创业、自媒体等）；核心是先选对**公司类型**再谈路径

## 文章正文

戴师兄在甲流康复后随手讲解：做电商必遇四类公司——**平台、品牌方、供应商（含工厂物流）、服务商**——并给出从起点到「最终归宿」的求职路径。

**四类公司。** 电商平台（淘天、京东、拼多多、抖音电商、小红书、得物、B 站等）：中国 top 互联网公司约 2/3 与电商相关；进平台起薪不低，哪怕做销售/店小二也能在大城市稳定生活（忙累）。品牌方（联合利华、宝洁、可口可乐等）：超市货柜五种香皂可能三种联合利华、两种宝洁；发展前景好。供应商：生产+仓储+物流一体，现代电商产研运一体化，工厂不考虑供应链会「烂地里」。服务商最易忽略：代运营、MCN、数据服务、ERP 等外包生态庞大。

**第一份实习/工作选哪类？** 平台：一毕业进大厂很难落地，容易成「空中飞人」——知道平台策略但不知店铺如何从 0 到 1 起量，更像螺丝钉；但大厂背书吃香。品牌方：值得学但快消校招 985 起跳，进去后多组织乙方服务商，节奏极快，比平台更落地。供应链：**职业生涯中都不要想自己去**——靠人情世故推动，需食品工业等专业技术，苦哈哈干几年才摸清流程，不适合普通人。服务商：**最适合大多数人起点**——门槛低（杭州上海五六千起），能经历店铺从 0 到 1、投放起量等具体环节；服务多家企业可快速知道行业整体玩法；缺点是视野窄，只知执行不知为何做策略。

**第二份去哪？** 排除供应链。服务商之后应**先去平台**（不是直接去品牌方）——平台员工工资不如品牌方，晋升岗位也少；品牌方是**最后去的**，要带着平台成型经验去变现。路径：服务商→拼多多（有且只有一个首选）→其他平台→本地生活→品牌方。拼多多因商业间谍问题不要其他平台现成电商人；996 但脚踏实地；先卷出「优质牛马」认证再跳其他平台非常好跳；即将成为中国电商市值最高公司之一。建议再去京东（高客单价）抓全貌，有空去字节了解兴趣电商。大厂来回跳，分析师或运营年薪四五十万基本可保证。若不想做电商跳本地生活，会重新一轮成长，且本地生活连锁（酒店餐饮）不如快消/美妆赚钱快。

**最终归宿。** 品牌方是大多数电商人最终归宿——专员/市场总监，多品牌轮岗，年薪四五十万有保障；接手新品牌或传统子品牌（如中华牙膏）可卷到大总监，年薪百万起步。或自己创业做品牌/开店。其他结局：大厂当螺丝钉到 35 岁被裁；或成为供应链大佬（最不敢想但最有期望的一集）。从服务商干起→平台→品牌方，「起于市井，发于卒伍」，到小公司指导必能出成绩——入门门槛还非常低。
