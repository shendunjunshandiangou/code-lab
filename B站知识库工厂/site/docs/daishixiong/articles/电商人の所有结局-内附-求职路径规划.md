# 电商人の所有结局与求职路径

<BiliPlayer :bvid="'BV1Le411k7d5'" />


**摘要**：做电商必遇四类公司——**平台、品牌方、供应商（含工厂物流）、服务商**。互联网大钱在电商；进平台起薪高；品牌/供应链/服务商各有路径；分析师在电商赛道性价比最高（参见已有电商原子）。

## 一、四类公司

### 1. 电商平台（人上人）

- 淘天、京东、拼多多、抖音电商、小红书、得物、B 站等
- 中国 top 互联网公司 2/3 与电商相关
- 起薪不低，哪怕销售/店小二也能在大城市稳定生活（忙累）

### 2. 品牌方

- 联合利华、宝洁、可口可乐等——货架多数快消来自少数巨头
- 发展前景好，分析师可跟品类/渠道数据

### 3. 供应商（生产+仓储+物流一体）

- 现代电商产研运一体化；工厂不考虑供应链/物流会烂地里
- 数字化管理（如快反）是趋势

### 4. 服务商（最易忽略）

- 平台/品牌外包生态庞大：代运营、MCN、数据服务、ERP 等
- 入门门槛可能更低，接触多品牌场景

## 二、求职路径规划

- **目标平台/品牌** → 先服务商/供应商积累行业 know-how → 再跳头部
- 数分在电商：订单/GMV/转化/供应链指标丰富，**最优赛道之一**
- 与 [数据分析-职业认知-电商是分析师最优赛道](/daishixiong/atoms/数据分析-职业认知-电商是分析师最优赛道.html) 互补：本视频补全四类公司地图

## 三、结局可能性（框架）

- 视频按职业阶段展开电商人多种结局（晋升、转行、创业、自媒体等）；核心是先选对**公司类型**再谈路径
