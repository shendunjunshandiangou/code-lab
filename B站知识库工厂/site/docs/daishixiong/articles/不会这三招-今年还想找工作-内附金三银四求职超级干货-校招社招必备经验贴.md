# 不会这三招，今年还想找工作？

<BiliPlayer :bvid="'BV1JN41197va'" />


**摘要**：金三银四求职三招——爬虫/手动调研确定意向岗位、按岗位迭代简历（五条原则）、多平台海投（脉脉响应度最高）。辅以面试与谈薪要点。

## 第一招：确定意向岗位

- 用爬虫工具或**手动多页浏览**招聘网站，收集最新岗位信息
- 关注**最低工资平均数**（最高薪往往是吸引套路，谈下来只有底薪或中间数）
- 多看 JD——很多公司 JD 是百度乱写的，只有多看才能提炼真实技能要求
- 理性分析后选定明确求职方向

## 第二招：按岗位迭代简历

社招原理：照着岗位信息写，用人单位觉得「就是我要的人」。

五条注意：

1. **岗位和在职时间绝不造假**——背调查出来丢工作丢信誉
2. **方向明确**——投什么岗位只保留相关经历和技能，万能简历不存在
3. **可适当改写工作过程**——岗位时间不能动，但实现过程可调整，把 JD 技能融入描述（**写了就要能答**，面试一问就露馅）
4. **多写多迭代**——没响应 = 简历质量不过关；好简历至少改数十版，边投边学边改
5. 定期根据最新岗位信息更新简历

## 第三招：投递策略

响应程度（大致）：**脉脉 > Boss > 拉勾 > 猎聘 > 51 > 智联**

- 每天刷、至少投 7～10 个，看到新岗就投，一周内有反馈
- **大公司每年仅 1～2 次投递机会**，用完会被自动拒绝，谨慎投
