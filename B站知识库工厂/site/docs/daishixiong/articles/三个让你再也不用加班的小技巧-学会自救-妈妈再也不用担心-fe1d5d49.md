# 打工人 996 自救三技巧

<BiliPlayer :bvid="'BV1Z5411J7VR'" />


**摘要**：应对无效内卷加班的三法——周中批量完成、周末分批交付；按自己节奏安排工作（如固定健身时段）；提升工具效率（Excel/SQL/Python 等）。若老板坐背后无限派活，应优先考虑换环境。

## 技巧一：一次干完、多次交付

- **周中提前完成整周工作**，周末分批交付，营造「工作饱和、周末也在忙」的错觉
- 避免领导主动加活
- 副作用：治拖延、提效率、预留学习时间、应对突发

## 技巧二：用自己的节奏干饭

- 设定固定个人节奏（如每天固定健身），灵活调整工作时间
- 不必等同事都走才下班
- 身材与时间管理双管齐下

## 技巧三：提升生产力

- 年轻最宝贵的是时间；拖延会压缩试错成长空间
- 多学提升生产力的工具技能（Excel/SQL/Python 等）
- 休息好、能力强 → 工作自然高效

## 底线

若老板天天坐背后无限派加班 → **赶紧跑，命比工作重要**。
