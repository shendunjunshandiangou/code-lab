> **摘要**：梳理 21 款数据分析工具优先级。**BI 板块核心变化：Power BI 升为首选，Tableau 仍值得学但非首选；FineBI 等国内 BI 有公司再用时再学。**

## BI 工具（可视化/看板）

### Power BI — 2023 首选

- 微软 Excel 生态演进的 BI：数据处理、探索、可视化、看板、在线监控一体。
- **几百元** 即可本地制作 + 共享发布，云服务免维护；个人/小团队 **性价比最高**。
- 与 ChatGPT 同属微软生态；**Fabric** 将统一大数据/数仓/分析，Power BI 为分析界面——跟进 Power BI 可享 AIGC 技术红利。

### Tableau — 仍值得学，但非首选

- 最适合新手入门的可视化工具之一，无设计基础也能做出美观图表。
- 中国市场政策导致购买/服务体验被 Power BI 等拉开；国内使用 Tableau 的公司可能减少。
- **可视化能力与操作逻辑仍值得专业分析师学习**；学会后迁移其他 BI 很简单。

### 国内 BI（FineBI / Quick BI / 永洪等）

- 多为 Power BI + Tableau 的「缝合版」，**不推荐专门花时间学**。
- **公司用了再研究**；有 Power BI/Tableau 基础，几小时可上手。

### Tableau Prep Builder — 必学自动化

- 图形化流程化数据处理；本地/云端数据整合、上传数据库与 BI。
- 拖拽步骤即可；可定时自动跑，上班前挂任务——**效率飞升工具，必学**。

## Python 可视化相关

- 分析师 Python 必学 **Pandas** + 成型项目；**PyGWalker** 必掌握——Python 内 Tableau 式拖拽，出图效率高。

## 其他板块（框架）

| 领域 | 推荐 |
| --- | --- |
| 自动化 | 影刀 RPA |
| 办公 | Office 365（Excel/PQ/Pivot）、飞书云文档 |
| 数据库 | 本地 MySQL + DataGrip；SQL 刷题 |
| Python | Jupyter Notebook；Pandas 必学；爬虫不推荐深钻，用后羿/影刀 |
| 算法 | ChatGPT 应用；校招可学 sklearn/xgboost |
