# 5年大厂分析师感悟

**摘要**：22 年一线大厂月薪约 3 万、14-18 薪、总包税前 45 万。大厂一天开会到下午，晚上加班做数据建设和专项分析；小厂随时被抓开会、修报表。分析师越老越吃香，30 岁不会被必然优化。

## 一、薪资参考

- 22 年某一线宇宙大厂：月薪 3 万左右（浮动 ±5000）
- 14~18 薪，年底一般 3~4 薪
- 年薪总包税前约 45 万，税后 35~36 万

## 二、大厂一天

| 时段 | 工作 |
| --- | --- |
| 早餐时 | 刷手机确认报表/看板/邮件有无报错 |
| 10:00~16:00 | 各种日会、周会、专项会议 |
| 午休 | 可能跟业务同事吃午饭收集信息 |
| 17:00 后 | 理需求，准备晚上工作 |
| 19:00~22:30 | 加班：数据建设/SQL、专项分析、业务推进跟踪 |

**晚上三件事**：
1. 梳理数据建设（能自己做的 SQL/标签自己写，不行找数仓套近乎）
2. 专项分析（业务达成率、下滑原因、策略效果 → 云文档报告）
3. 业务推进（跟踪策略落地，跟业务拉扯目标）

> 该碰业务还是要碰——未来职业发展差距就在业务理解上。

## 三、小厂一天

- 会无明确时间规划，随时被抓开会
- 报表挂了修不好（上游数仓问题）
- 天天洗数据、做自动化建设
- 领导想看的分析数据往往没有，得跟开发撕逼
- 能力够强、数据建设搞定后越来越好

## 三、三个高频问题

### 1. 加不加班？

跟职业无必然关系，看公司文化和数据建设完善度。数据自动化程度高、老板友善 → 加班少。前期技能不熟练肯定加班。

### 2. 工作重复吗？

- 数据处理：每次应是全新思考；一直重复 = 技术能力有问题，没做自动化
- 分析报告：日报周报模板化是幸福（简单）；异常排查才棘手
- 越重复难度越低，越个性化难度越高

### 3. 30 岁会被优化吗？

- 经验老辣、一直出结果 → 不会被优化
- 以学习和成长潜力下滑为界，非年龄
- 技术能力强能搞定数据建设 → 也有一席之地
- 分析师离老板更近，往往不是第一批被优化的

## 四、转型路径

先做半年运营 → 工作中用 Excel/BI 做数据汇报 → 拉开差距 → 转行数据分析。
