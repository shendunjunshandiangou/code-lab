# 3 套面试话术：轻松应对离职原因与裁员

<BiliPlayer :bvid="'BV1Bm4y1c7ny'" />


**摘要**：被裁后面试被问原因，HR 提问方式往往不专业。两类敏感原因绝不能透露（能力短板、无法改变的环境因素）；只能围绕整体裁员、业务变动、技能错配三种情况，用「原因 + 思考 + 规划」框架正面回应并展现成长空间。

## 一、为什么不能如实回答

HR 问「为什么裁你不裁别人」非常不专业——很多裁员是高层决定裁整个业务线，并非个人能力问题。

确定候选人后首先要排除用人风险，因此有两类**绝对不能轻易透露**的原因：

1. **自身能力不足**（压力太大、绩效差、沟通问题）——暴露短板极大降低成功率
2. **无法改变的环境因素**（领导太坏、同事太菜、内斗、天天加班）——HR 会认为适应能力差

完美工作环境不存在。雷点过多应在面试中收集信息，**双向选择时果断开溜**。

## 二、能说的三个原因 + 统一框架

只能围绕以下三种原因，统一用 **原因 → 思考 → 规划** 框架回答：

### 2.1 整体裁员

- **原因**：外部政策和经济环境导致公司不行，与个人无关；涉及所有部门和试用期员工
- **思考**：之前对行业过于乐观，危机意识欠缺（强调是判断问题，非工作能力）
- **规划**：仔细研究了贵司所在行业，看到挑战，危机意识已加强；愿意相信行业前景

### 2.2 业务变动

- **原因**：外部原因导致整条业务线收缩，只保留收尾编制——公司砍掉的是**坑位**，不是「我这个人」
- **思考**：深究业务线收缩根源——业务只有赚钱才能创造价值
- **规划**：将更多精力投入业务营收，为公司赚钱争取更大发展空间；主动接受裁员，营造「跳槽」而非「被踢」的观感

### 2.3 技能错配

- **原因**：长期在某一领域积累，但公司优化编制后需承担 2～3 个岗位复合职能，无法深钻原领域，主动寻找新舞台
- **思考**：加深了对各类岗位职能的理解，认识到职能复合化是趋势
- **规划**：保持专业追求的同时拓宽能力边界，接受一定限度职能复合

## 三、应对 HR 刁难

- HR 可以带目的性提问，候选人也可以带目的性回答
- 逻辑清晰的回答给 HR 吃定心丸，加快招聘进度
- 遇到人身攻击式提问：用准备好的话术应对，**不要因对方言语否定自己**，当作免费模拟面试练经验后离开
- 有没有一种可能：离裁员不是你不行，而是**公司不行**

## 四、话术调整

以上模板需结合**真实经历**调整，不要照搬。
