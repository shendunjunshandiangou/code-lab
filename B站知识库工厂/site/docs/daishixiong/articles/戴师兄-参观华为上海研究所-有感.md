# 参观华为上海研究所有感

<BiliPlayer :bvid="'BV1b94y1q7Wx'" />


**摘要**：戴师兄参加华为上研 UP 主交流，参观 C 端裸眼 3D/XR 与 B 端 5G 基建方案（智慧医疗、矿井、远程手术等），感受华为 5G 设备小型化、集成化与工程落地能力，以及对科技展会的启发价值。

## 一、C 端体验

- **裸眼 3D 屏幕**：追踪眼球，物体立体呈现（类似苹果 XR 方向）
- 5G 使虚拟与现实交互；分辨率尚低但可想象黑料（暗场）后真实感

## 二、B 端与基建

- 智慧医疗、矿井、远程手术等：**5G 低时延、不丢包**使远程实时操控成为可能（4G 易模糊/丢包不敢做）
- 地下采矿仪表盘联网、无人化解决方案

## 三、5G 设备工程优势

| 特点 | 价值 |
| --- | --- |
| 航天级塑料外壳 | 抗腐蚀 |
| 发射器小型集成（~30kg 以下） | 欧洲部署单人可背，万站省人力 |
| 3G/4G/5G 集成 | 替换原 4G 杆即可覆盖 345G |

## 四、研发布局

- 昆仑玻璃、巴黎设计中心、俄罗斯算法中心等全球研发布局
- 技术储备强，招待规格在字节/阿里/网易/百度中属最高档

## 五、对从业者的启发

- 所见即所得 vs 听介绍：震撼完全不同
- 建议多逛**技术类/工业类展会**，所获启发可反哺业务与职业认知
- 5G 价值在**具体场景落地**（医疗、工业、XR），非概念本身
