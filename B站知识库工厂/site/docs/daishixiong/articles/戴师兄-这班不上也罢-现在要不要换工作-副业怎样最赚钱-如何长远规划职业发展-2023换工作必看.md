# 要不要换工作？副业与职业规划

<BiliPlayer :bvid="'BV1Wv4y1r7Mh'" />


**摘要**：满一年且有好下家可跳；未满一年优先苟住（稳定性=能力）。裸辞留硬伤，用三剂心态法调节。职业规划应规划**具体能力成长**而非虚渺年薪目标。副业：优先主业深耕，警惕割韭菜副业课。

## 一、要不要换工作

| 情况 | 建议 |
| --- | --- |
| 待满 1 年 + 更高薪下家 + 简历不花 | **果断跳**，骑驴找马，拿到 offer 再离职 |
| 未满 1 年 | 先苟；稳定性与能力同等重要 |
| 工作恶心 | 多数问题换公司仍存在；靠**能力提升**解决（不会游泳换池无用） |
| 心态崩了想裸辞 | **别裸辞**——履历硬伤、HR 稳定性质问、3～5 年后难找优质工作 |

### 三剂心态强心针

1. **赶快休息**：请假/病假、约饭、大餐、电影、游戏，恢复精力
2. **为简历打工**：工作是为塑造高薪履历，与恶心老板无关
3. **抓大放小**：将军赶路不追小兔，为目标忍小干扰

## 二、副业怎样最赚钱

- 主业未稳时，副业分散精力，**优先把主业做深**
- 警惕「副业月入过万」类割韭菜
- 真正副业往往来自主业能力外溢（咨询、培训、自媒体等），需长期积累

## 三、长远职业规划

### 误区

- 定「毕业几年年薪 X 万 / P 级」——无用，市场无容错随便弃牌重打

### 正确做法

- 基于**当前经历**，把手上牌打好
- 规划**具体能力如何成长**：研究年薪百万者具备什么经验能力 → 在当前业务场景如何积累 → 细分到每天学习时长
- 做不到细致执行 = 画饼 + 精神内耗

## 四、核心原则

- 外部环境不可控，**心态可调**
- 排解负面情绪优于裸辞留硬伤
