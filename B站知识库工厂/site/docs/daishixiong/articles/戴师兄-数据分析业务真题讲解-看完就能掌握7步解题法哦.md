# 网易数据分析笔试真题与七步解题法

**摘要**：以网易云音乐付费会员活动题（买一送一 vs 五折、联合会员成本收益评估）为例，讲解开放型业务笔试的**七步解题法**。核心原则：答案不重要，解题思路才重要；笔试不重要，分析思维才重要。

## 一、三种错误反应（必凉）

1. 一脸懵逼，不知从何答起
2. 知道套路但缺乏业务认知，信息不全瞎写
3. 想到什么写什么，不系统

## 二、七步解题法（七步成诗）

| 步骤 | 动作 |
| --- | --- |
| **1 提取关键信息** | 圈出题干重点，整理成结构化信息；理清业务机制、业务对象、分析目标 |
| **2 信息结构化** | 划分问题范围，有的放矢 |
| **3 列出关键指标** | 量化 > 抽象；长期 > 短期（短期过得去时优先长期）；短期长期同时考虑 |
| **4 量化数学关系** | 把指标比较转化为公式（> &lt; =）；「只有数学关系才能更好地判断好坏」 |
| **5 找影响因素** | 谁在影响比较指标？便于控制变量、分类讨论 |
| **6 总结规律** | 把指标、影响关系归纳为几条规律/定律（如长尾效应、新用户与折扣关系） |
| **7 结合实践分类作答** | 考虑 A/B 测试等实验手段；按影响因素分类讨论；**结论必须有量化指标** |

## 三、案例要点（会员活动题）

### 业务机制

通过丰富权益吸引付费会员 → 提升粘性与忠诚度（长期目标）；开卡活动是短期手段。

### 指标选择原则

- 用**活跃/互动/付费**等量化指标近似「忠诚度/粘性」
- 短期看开卡人数、收入；长期看核心指标增长率
- 长期观测至少 **1～3 个月**，年卡活动 ideally 等到**活动彻底结束**再复盘

### 数学关系示例

- 两方案 ROI 相同（买一年送一年 ≈ 半折买半年送半年）
- 比较开卡收入本质可能在比开卡人数（单价有倍率关系）

### 实践考量

- 活动前能否做 **A/B 测试** 或产品测试
- 用抽样数据 + 推断统计代表整体

## 四、分析师立场

- 业务只要开卡量 → 你会被牵着走；展现**长短期兼顾**思路才是好分析师
- 经验靠时间积累，**思路今天学完就能用**
- 面试/笔试失败没关系，能力有增长即可

## 五、课后练习

第二题（联合会员成本收益划分）与加问留作自练，思路与第一题类似但更复杂。
