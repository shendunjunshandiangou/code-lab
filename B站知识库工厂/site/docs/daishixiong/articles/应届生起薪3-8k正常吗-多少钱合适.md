# 应届生起薪 3-8K 正常吗？多少钱合适？

<BiliPlayer :bvid="'BV1cZopYQEB4'" />


**摘要**：成都小厂给应届生开 3～8K 是常见写法，3K 往往是实习期工资；数据分析应届生起薪大致 6～12K，一线大厂数分 offer 可达 1.8～2 万，关键看实习经历和校园背景。

## 一、3～8K 的 JD 怎么理解

以成都某小厂「3～8K」为例：

- **6～8K** 对成都应届生来说已经算不错
- **3K** 通常是拿毕业证之前的**实习工资**——哪怕在大厂，毕业前也会以实习生身份压薪资，毕业后才签正式劳动合同给正式薪资

## 二、各城市/岗位起薪参考

| 场景 | 起薪范围 |
| --- | --- |
| 一线大城市（北上广深）运营岗 | 普遍 6～8K，很难低于 5K |
| 数据分析应届（一般） | 6～12K，看实习经历和校园背景 |
| 美团、字节等大厂校招数分 offer | 约 1.8 万，甚至 2 万左右 |

起薪跟**实习分数、有没有大厂实习**强挂钩。

## 三、建议

1. **在校期间多积累实习经验**——这是拉开起薪差距的关键
2. **没有实习经历前，不要对岗位挑三拣四**
3. 成都有 6～8K 的游戏行业起点，已经是不错的选择
4. JD 写 3K 也是一种「负责任的态度」——提前劝退觉得薪资低的人

## 文章正文

群里有人问：成都某小厂给应届生开 3～8K 是认真的吗？感觉以前小厂职能都能开到 7～11K。戴师兄以成都为例做了解读。

**6～8K 对成都应届生已经算不错。** 区间里的 **3K 其实是实习工资**——拿毕业证之前，哪怕在大厂，往往也是以实习生身份压薪资、「白嫖」一部分劳动力；只有毕业后才签正式劳动合同、给正式薪资。所以看到「3～8K」的 JD，不要把 3K 当成毕业后的正式起薪。

**各城市、岗位参考：** 北上广深海等一线大城市，运营岗普遍起薪 **6～8K**，很难低于 5K。数据分析应届生起薪大致在 **6～12K**，具体看**实习经历和校园背景**——跟实习分数、有没有大厂实习强挂钩。如果能拿到美团、字节等大厂校招数分 offer，可能到 **1.8 万甚至 2 万左右**。

他的建议是：**在校期间多积累实习经验**；在没有实习经历之前，**不要对岗位挑三拣四**。如果在成都能有 6～8K 的游戏行业起点，已经是比较不错的选择。JD 写出 3K 下限，也是一种负责任——让觉得薪资低的小伙伴提前劝退，避免入职后心理落差。
