# 直播电商数据分析启蒙课：毕业答辩框架

**摘要**：浙江农林相关学分课学员围绕美妆个护等直播间做探索性分析答辩。共性方法是：用**单位时间人均 GMV**衡量主播/行为/商品的变现力，用**平均停留时长/在线人数**衡量留存与聚人气，再按直播阶段搭配主播、行为与商品顺序。

## 一、课设在练什么

不是背 BI 操作，而是把直播间当成业务对象做诊断：

1. **谁更能卖货**：比较主播的单位时间人均 GMV。
2. **谁更能留人**：比较平均停留时长、在线人数趋势。
3. **什么行为抬销量**：把主播行为（介绍商品、大屏展示、亲自试用表达感受、展示购买者评论等）与人均 GMV / 停留 / 在线人数交叉。
4. **卖什么、什么顺序**：找出销量王/销售王/潜力品，按「聚人气 → 留存 → 变现」排品，冷门品穿插。

## 二、反复出现的指标逻辑

| 指标 | 典型用途 |
| --- | --- |
| 单位时间人均 GMV | 变现能力：选「该上场卖货」的主播/行为/爆品 |
| 平均停留时长 | 留存能力：选「该暖场留人」的主播/行为 |
| 在线人数及趋势 | 聚人气：中期抬场、为后期变现铺垫 |

多组答辩结论形态相似（品牌案例如菲利普等）：

- 变现最强主播在**成交段**出场；留存最长主播在**蓄水段**出场。
- 感性营销、讲解售后、展示购买评论、亲自试用等，常与更高变现相关。
- 前期多用介绍商品 + 大屏展示拉停留；中期加大屏抬在线；后期用展示购买者评论促成。
- 人气不足时先推高吸引力/高销量的引流品，再推留存品，最后上高变现品；冷门品穿插带货。

（具体主播名、SKU、数值以各组 PPT 为准，字幕识别存在大量专名误差，此处不逐条当作精确事实。）

## 三、可迁移的分析框架

1. **先定义阶段目标**：引流 / 留存 / 变现，不要全程同一套动作。
2. **人、货、场行为分开度量**：主播、商品、话术行为各自用适合的指标，再组合排期。
3. **用表格交叉验证**：行为 × 人均 GMV、行为 × 停留、商品 × 在线人数趋势。
4. **输出可执行建议**：谁上场、何时上场、卖什么、排什么顺序、加哪些库存/时长——而不是只描述「数据很好看」。

本场为多组学生答辩 + 教师点评的长录屏，细节数字不作知识库权威数据源；沉淀的是上述直播电商分析套路。
