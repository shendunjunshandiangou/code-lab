# 数据分析职业规划万字总结（框架）

**摘要**：直播/Q&A 合集，覆盖数分职业路径、技术/业务/算法分工、跳槽时机（金三银四/金九银十）、校招社招差异、职场潜规则与高时薪逻辑。核心：待满 1～2 年再跳、校招难时年转社招基础岗、理解 HR 岗位规划逻辑。

## 一、职业路径三分法

| 方向 | 特点 |
| --- | --- |
| **技术向** | 取数、数仓、工具、看板 |
| **业务向** | 贴近业务决策、算账、策略 |
| **算法向** | 模型、预测、需数学/工程基础 |

- 多数分析师走**业务向**性价比最高
- 高时薪靠参与决策层算账，非仅工具操作

## 二、跳槽时机

- **原则**：待满 **1～2 年**再跳更利于简历与议价
- **金三银四**：年底跳槽者 spring 流动 + 校招岗位梳理，社招岗位一并明确
- **金九银十**：校招季带动岗位释放；但跳槽人员相对少，成功率需具体分析
- 有更好下家且满一年 → 可跳；裸辞/频繁短任 → 伤稳定性评价

## 三、校招 vs 社招（近年趋势）

- **近年校招难度极高**——戴师兄称见过最难一年
- 建议：校招通道难时，**同步投社招基础岗**（刷题面试与投递并行）
- HR 校招时会做全盘岗位梳理 → 弄清哪些校招、哪些社招
- 校招每年淘汰很多人 → 不必单吊校招渠道

## 四、职场潜规则

- 理解公司推荐/晋升/绩效的**隐性规则**
- 业务、HR、leader 各方利益诉求不同，分析师需在博弈中呈现数据
- 跳槽前了解目标团队数据成熟度与分析师权限

## 五、避坑要点

- 勿只做工具人大盘看板
- 勿被「开卡量/短期 KPI」牵着走（参见七步解题法长期思维）
- 技术业务算法选一路深耕，频繁横跳难积累
