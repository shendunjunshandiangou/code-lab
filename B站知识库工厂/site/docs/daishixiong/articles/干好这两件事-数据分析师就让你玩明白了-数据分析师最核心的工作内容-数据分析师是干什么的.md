# 干好这两件事，数据分析师就让你玩明白了

<BiliPlayer :bvid="'BV1DL411H7vZ'" />


**摘要**：成熟分析师的核心就两件事——①提供数据支持，让业务看到想看的数据；②基于分析为业务出谋划策。SQL/Tableau 解决取数呈现，会议沟通补齐数据发现不了的问题。

## 一、第一件事：数据支持

业务常看的报表类型有限，用 **SQL + Tableau** 即可覆盖：

| 工具 | 作用 |
| --- | --- |
| SQL | 与数据库对话取数，核心语句约七个，可快速入门 |
| Tableau | BI 可视化，比 Excel 更美观、数据量更大、支持网页协作 |

## 二、第二件事：数据分析与出谋划策

- 业务面对海量数据无力深入，但不研究又不行
- 分析师通过业务公式计算，挖掘数据背后信息（如速度 = 路程 / 时间）
- **并非所有问题都能靠数据发现**，需结合业务场景
- 需积极参会收集信息、验证假设、辅助决策（大厂分析师一天两三场会很常见）

## 三、成熟标准

做到以上两点，即具备成熟数据分析师能力：既能稳定供给数据，又能参与业务决策。
