# HR是如何骗你入职的！

<BiliPlayer :bvid="'BV19b421b76s'" />


**摘要**：戴师兄用「鉴定网络热门招聘陷阱」的方式，拆解 JD 里常见的文字游戏——表面光鲜的福利话术，背后往往藏着加班、低薪、偏远办公等真实处境。

## 常见招聘话术 vs 真实含义

| JD 写法 | 真实含义 |
| --- | --- |
| 工资面议 | 低到不好意思写，只能先把人骗过来再说 |
| 期权激励 | 用期权拴住员工好多年，未来不一定值钱 |
| 公司提供免费宵夜 | 廉价宵夜喂到饱，不到凌晨不准走 |
| 班车接送 | 公司偏到山里，不安排车的话通勤时间比上班还长 |
| 提供宿舍 | 公司位置偏远，宿舍是配套手段 |
| 团队年轻有活力 | 老员工全都跑路了，在职的都是新骗来的 |
| 可适当加班者优先 | 适当加七八个小时就好，不用太多 |
| 还有各种隐形福利 | 隐形到连 HR 都不知道是什么福利 |
| 回去等消息吧 | 再看看有没有更便宜的，没有再回来坑你 |
| 算上绩效工资，范围 4000 到 1 万 2000 | 保养得好，预计能活 80～500 岁 |

## 怎么用

看 JD 时不只看表面关键词，要把「面议」「期权」「免费宵夜」「年轻团队」这类话术翻译成真实工作条件，再决定值不值得投。
