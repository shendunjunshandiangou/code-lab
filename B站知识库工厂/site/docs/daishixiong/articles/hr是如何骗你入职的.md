# HR是如何骗你入职的！

## 常见招聘话术 vs 真实含义

| JD 写法 | 真实含义 |
| --- | --- |
| 工资面议 | 低到不好意思写，只能先把人骗过来再说 |
| 期权激励 | 用期权拴住员工好多年，未来不一定值钱 |
| 公司提供免费宵夜 | 廉价宵夜喂到饱，不到凌晨不准走 |
| 班车接送 | 公司偏到山里，不安排车的话通勤时间比上班还长 |
| 提供宿舍 | 公司位置偏远，宿舍是配套手段 |
| 团队年轻有活力 | 老员工全都跑路了，在职的都是新骗来的 |
| 可适当加班者优先 | 适当加七八个小时就好，不用太多 |
| 还有各种隐形福利 | 隐形到连 HR 都不知道是什么福利 |
| 回去等消息吧 | 再看看有没有更便宜的，没有再回来坑你 |
| 算上绩效工资，范围 4000 到 1 万 2000 | 保养得好，预计能活 80～500 岁 |

## 怎么用

看 JD 时不只看表面关键词，要把「面议」「期权」「免费宵夜」「年轻团队」这类话术翻译成真实工作条件，再决定值不值得投。

## 文章正文

戴师兄用「鉴定一下网络热门招聘陷阱」的口吻，把 HR 文案里常见的文字游戏逐条拆穿。视频很短，但每条对照都直指真实处境——表面越「人性化」，背后往往越需要警惕。

「工资面议」通常意味着低到不好意思写在招聘平台上，只能先把人骗过来面谈。「期权激励」是用未来不确定的收益拴住你很多年，期权本身未必值钱。「公司提供免费宵夜」配合的是廉价宵夜喂到饱、不到凌晨不准走的节奏。「班车接送」「提供宿舍」往往是因为公司偏到山里，不安排车的话通勤时间比上班还长。

「团队年轻有活力」的真实含义常常是：老员工都跑光了，在职的都是新骗来的。「可适当加班者优先」里的「适当」，往往指每天再加七八个小时。「还有各种隐形福利」——隐形到连 HR 自己都说不清是什么。「回去等消息吧」可能是再看看有没有更便宜的候选人，没有再回来坑你。

最夸张的一条是：「算上绩效工资，范围 4000 到 1 万 2000」——戴师兄接了一句：「保养得好，预计能活 80～500 岁。」

这类内容不是让你拒掉所有带「面议」「期权」的 JD，而是养成翻译习惯：看到话术先问「实际工作条件是什么」，再决定值不值得投。求职阶段，能读懂 JD 背后的真实含义，和能写 SQL 一样，都是避坑的基本功。
