# 与其纠结大环境，不如往这三个方向猛攻

<BiliPlayer :bvid="'BV116rrBhEaN'" />


**摘要**：框架级回答「明年好不好找工作」——从供需双方构建分析框架：岗位由企业提供（大环境），个人优势靠能力。结论指向三个猛攻方向（数字化/跨境/AI 应用等，详见视频分章节），勿只听培训机构贩卖焦虑或机会。

## 一、问题重构

- 勿问培训机构「好不好找」——必答有机会或贩卖焦虑
- 正确问法：**哪些行业/情况好找，哪些不好找**
- 分析师方法：从白纸构建框架 + 结合自身情况

## 二、找工作双向因素

| 侧 | 因素 |
| --- | --- |
| **个人** | 能力、简历、面试表现 |
| **环境** | 岗位供给（非个人能左右） |

- 用户常问的是大环境：能力差点能否靠局势上岸

## 三、分析框架（概要）

1. **岗位谁提供** → 看行业景气、政策、融资、裁员/扩招
2. **哪些方向岗位增** → 数字化、跨境、AI、本地生活等（视频分章展开）
3. **自身牌怎么打** → 专业、经历、城市、技能组合

## 四、三个猛攻方向（视频核心）

> 视频从宏观到微观论证三个值得投入的方向；框架级重写保留结论逻辑，具体行业数据见原视频分 P。

1. **数字化/数据相关岗位**——企业数据化持续，数分/运营/产品仍有需求
2. **跨境与出海**——国内供应链 + 海外需求差（电商、物流、市场）
3. **AI 提效能力**——会用 AI 做分析/办公者竞争力上升（非纯算法岗）

## 五、行动建议

- 停止空泛纠结，**选定向猛攻 + 边做边验证**
- 用招聘网站验证方向：搜 JD 数量、薪资、技能要求
