## 摘要

学数分应**以终为始**：先框架后细节、用算法数据选老师与课程、按在看人数判断投入性价比，并尽早投简历拿反馈迭代——AI 时代人类只需会「说清楚怎么做」和「做出结果」。

## 高效学习步骤

1. **选对老师**：B 站搜索数据分析，选播放量高 + 有大厂背书、拿过结果的博主；相信推荐算法与实时「在看人数」
2. **先学框架**：先看博主多篇「自学攻略」总结，对比迭代版本搭建知识地图，再跟具体课程
3. **按数据选课**：同一 UP 主下，**在看人数多**的短课优先于枯燥长课（如 Tableau 2.0 比 1.0 更易跟）
4. **推荐顺序**：Excel → SQL 入门 → Tableau 可视化（可跳过过干版本改跟 2.0）
5. **以终为始**：改好简历模板 → Boss 直聘直接投 → 看响应与面试反馈再决定深度学；不要纯应试式囤课
6. **AI 时代简化**：简历/JD 匹配可交 AI；人类核心学两样——面试能说清**怎么做**，以及**实际做出分析报告**（有图、有业务、逻辑详实）

## 作品门槛

可以没有报表、没有 BI 看板，但**必须有一份像样的分析报告**——从根本上解决面试作品焦虑。
