# 要不要换工作？副业与职业规划

**摘要**：满一年且有好下家可跳；未满一年优先苟住（稳定性=能力）。裸辞留硬伤，用三剂心态法调节。职业规划应规划**具体能力成长**而非虚渺年薪目标。副业：优先主业深耕，警惕割韭菜副业课。

## 一、要不要换工作

| 情况 | 建议 |
| --- | --- |
| 待满 1 年 + 更高薪下家 + 简历不花 | **果断跳**，骑驴找马，拿到 offer 再离职 |
| 未满 1 年 | 先苟；稳定性与能力同等重要 |
| 工作恶心 | 多数问题换公司仍存在；靠**能力提升**解决（不会游泳换池无用） |
| 心态崩了想裸辞 | **别裸辞**——履历硬伤、HR 稳定性质问、3～5 年后难找优质工作 |

### 三剂心态强心针

1. **赶快休息**：请假/病假、约饭、大餐、电影、游戏，恢复精力
2. **为简历打工**：工作是为塑造高薪履历，与恶心老板无关
3. **抓大放小**：将军赶路不追小兔，为目标忍小干扰

## 二、副业怎样最赚钱

- 主业未稳时，副业分散精力，**优先把主业做深**
- 警惕「副业月入过万」类割韭菜
- 真正副业往往来自主业能力外溢（咨询、培训、自媒体等），需长期积累

## 三、长远职业规划

### 误区

- 定「毕业几年年薪 X 万 / P 级」——无用，市场无容错随便弃牌重打

### 正确做法

- 基于**当前经历**，把手上牌打好
- 规划**具体能力如何成长**：研究年薪百万者具备什么经验能力 → 在当前业务场景如何积累 → 细分到每天学习时长
- 做不到细致执行 = 画饼 + 精神内耗

## 四、核心原则

- 外部环境不可控，**心态可调**
- 排解负面情绪优于裸辞留硬伤

## 文章正文

戴师兄总结打工人最关心的三个问题：要不要换工作、要不要搞副业、如何长远规划。

**要不要换工作？** 原则上待满一年且能找到更高薪下家、简历不花，**果断跳**——更高薪往往意味着更看重你的能力。但要**骑驴找马**，拿到 offer 再提离职，避免裸辞尴尬。未满一年想换：稳定性与能力同等重要——HR 看到半年就跳会犯嘀咕，至少一年才有磨合调整空间。工作太恶心？多数问题换公司仍存在——「不会游泳换再多泳池也没用」，靠**个人能力提升**解决当前问题才是正路。

心态崩了想裸辞？**稳住**。裸辞一时爽，往后每次找工作 HR 都会质问稳定性；裸辞多了简历花，3～5 年后甚至找不到优质工作。外部环境不可控，**心态可调**——排解负面情绪比裸辞留硬伤更优。

**三剂心态强心针**（亲自实践）：一，**赶快休息**——请假/病假、约饭、大餐、电影、游戏，精力恢复后心态自然平缓；二，**为简历打工**——工作是为塑造高薪履历，跟恶心老板同事无关，想清楚被 PUA 概率大降；三，**抓大放小**——将军赶路不追小兔，为目标忍小干扰，等满一年大换特换。

**长远职业规划**：走出误区——不要定「毕业几年年薪 X 万/P 级」，市场无容错随便弃牌重打。基于当前经历把手上牌打好，规划**具体能力如何成长**：研究年薪百万者具备什么经验能力→在当前业务场景如何积累→细分到每天学习时长。做不到细致执行=画饼+精神内耗。别盲目比较——找自己最舒适、可长期持续的提升方式，循序渐进。

**副业**：一个人不可能同时干好两件事。正确态度是用空闲时间探索**下一个增长曲线**（下一份更高收入的主业）。副业顺利则逐渐切换重心直到值得百分百投入；多数情况是副业占用精力却不足以媲美主业——**收回来认真干主业**收获更大。为兴趣搞副业则不要有太多收入预期，佛系探索。
