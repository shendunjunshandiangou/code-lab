# GMV 下滑 + DAU 暴跌：用业务公式拆解大厂数据分析题

**摘要**：大厂笔试经典题——GMV 下滑但 DAU 下滑更厉害。核心方法：先校验数据准确性，量化「更厉害」（绝对值+比例），用业务公式 GMV=DAU×成交率×客单价 拆解，判断后置指标挽单效应。

## 一、指标科普

- **GMV**：标价成交总额，滞后指标，反映卖得怎么样
- **DAU**：日活用户，前置指标，对营销/外部环境更敏感，波动性更大

## 二、分析流程

### 1. 数据准确性校验

新数据先验证准确性，常年正确的数据才直接分析。

### 2. 量化「更厉害」

对比 = 绝对值 + 比例：
- 绝对值：GMV 下滑 10%，DAU 下滑 20% → DAU 多滑 10 个百分点
- 比例：GMV 下滑 20%，DAU 下滑 60% → 预期 40% 实际 60%，比例从 1:2 变成 1:3

### 3. 业务公式拆解

```
GMV = DAU × 成交率 × 客单价
```

DAU 下滑更厉害但 GMV 没那么惨 → **成交率×客单价必然回升**（挽单效应）。

四种情况穷举，只有成交率上升、客单价上升、两者都上升三种成立。

### 4. 成交率上升场景（挽单）

**理发店施工例子**：外面施工无新客 → DAU 锐减 → 收回发传单人力 → 1V1 服务老客办会员卡 → 成交率上升。

**大厂补贴战例子**：京东/美团补贴后打不赢 → 减少获客投入 → 资源转向老客复购维护 → DAU 降但毛利率/ROI 升。

### 5. 客单价上升场景

- 直接涨价（硬涨/只推高价产品）
- 间接涨价（减少低价 SKU、捆绑销售等）

涨价天然导致用户流失，与 DAU 下滑形成因果。

## 文章正文

戴师兄拆解某电商大厂笔试真题：**GMV 下滑，且 DAU 下滑得更厉害，如何分析？** 大厂分析题达标思路只能及格，真正得分靠背后的业务经验与可执行判断。答题前须先做**数据准确性校验**——新数据先验证，常年正确的数据才直接分析。

### 指标与「更厉害」的量化

**GMV** 是标价成交总额，滞后指标，反映「卖得怎么样」。**DAU** 是日活，前置指标，对营销、外部环境更敏感，波动性更大。题目可翻译为：最近卖得不好，但活人用户丢得更凶。

「更厉害」须结构化：**对比绝对值 + 对比比例**。例如 GMV 下滑 10%、DAU 下滑 20%，绝对值上 DAU 多滑 10 个百分点；若 GMV 滑 20%、DAU 滑 60%，预期比例 1:2 变成 1:3，比例下滑更厉害。

### 业务公式：GMV = DAU × 成交率 × 客单价

DAU 跌得更狠而 GMV 没那么惨 → **成交率 × 客单价必然回升**（挽单效应）。穷举四种情况，只有成交率升、客单价升、两者都升三种成立。

**成交率上升（挽单）**：理发店外施工无新客 → DAU 锐减 → 收回发单人力、1V1 服务老客办会员卡 → 成交率升。大厂补贴战打不赢后减少获客、资源转向老客复购维护 → DAU 降但毛利率/ROI 升。

**客单价上升**：直接涨价（香奈儿式硬涨导致低端客户流失）、只推高价 SKU（手游金字塔只推千元档）、间接涨价（取消补贴/免费体验）。涨价天然导致用户流失，与 DAU 下滑形成因果，但可部分托住 GMV。

分析顺序：**先绝对值（前置指标天生更敏感）→ 再比例（套公式看成交率/客单价谁托底）→ 再穷举业务场景**。

### 得分标准与加分项

能说出「客单价和成交率必有上升」约 **60 分**（有逻辑缺案例）；穷举挽单与涨价场景约 **90 分**；结合时事约 **100 分**——如 UZI 名人堂皮肤定价 3286 元 vs 普通皮肤 20～200 元，有效拉高 GMV 但可能因广告推送导致 DAU 流失；可建议通行证先给粉丝免费皮肤再推高价。
