# 能进大厂的都是什么人？我能进吗？

<BiliPlayer :bvid="'BV17U9jYXEtx'" />


**摘要**：大厂助教直播对谈——校招进不了大厂可接受低起点再积累；筛选看学历+两段相关实习+硬技能+实习深度反思；**没实习比没学历更致命**；进大厂靠信息/能力积累与把实习挖深；中小厂可通过业务+中台双线争取认可；职场成长靠超越老板视角与真诚沟通。

## 一、校招进不了大厂怎么办？

- 低起点也有机会：第一份工作中小厂/传统企业，积累后仍可回大厂（如 4 年积累后进字节）
- 接受起点，规划职业生涯；大厂不是唯一解，情绪管理与抗压需自评
- 案例：双非+三份大厂实习可校招进大厂；学历好但零实习反而难进

## 二、大厂筛选标准（校招）

| 要素 | 说明 |
| --- | --- |
| 学历 | 漂亮学历是强背书，但非唯一 |
| 实习 | 至少两段与投递岗位相关的实体实习，最好一段大厂级 |
| 硬技能 | 工具会用，简历无明显瑕疵 |
| 实习深度 | 不只写执行内容，要反思串联、体现对业务的推动价值 |
| 信息积累 | 与大厂人交流，用「他们的话」沟通（指标、拆问题方式） |
| 运气 | 被哪个部门捞、面试官、发挥 |

**优先级**：没实习比没学历更致命。

## 三、如何做高薪大厂人 / 快速成长

### 3.1 中小厂双线争取认可（如玉助教）

1. 搞清业务领导、中台领导对数据的认知与 KPI
2. 与业务领导 N-1 建关系 → 拿真实业务逻辑 → 做出别人做不出的成果
3. 选中台领导 OPR 最重指标，帮推项目/写材料
4. 业务+中台同时「吹枕边风」→ 重要场合（季度会等）给曝光机会

### 3.2 超越老板视角（小鱼助教）

- 建立老板视角，甚至 +1、+2 视角；言行站在老板位置思考
- 领导软弱/退缩时顶上去；专业能力是关键时刻的「开门武器」，需软实力铺垫

### 3.3 职场融入

- 非职场：提升人际包容度，尊重不同生活方式
- 大厂：融入节奏与文化；与同事能否成为朋友影响团队生存

## 四、其他要点

- 脉脉等平台薪资帖勿全信（非实名、IT/算法居多）
- 跳槽涨幅看上一份薪资；内部晋升涨幅通常小于跳槽
- 向上管理：先完成领导布置的任务，再谈改善；让领导有掌控感
- 面试对不同面试官拟合风格：和气场合者深聊，对冷漠者 123 点规范化输出；对大佬放低姿态、真诚分享

## 五、快问快答摘录

- 外包大厂：没更好选择可去，简历响应度高于不知名公司
- 数分入行感：数据真正赋能业务策略时；汇报附图表获认可
- 总部数分支持地区：常用 SQL 查询库、优化慢查询、真诚回复消息
