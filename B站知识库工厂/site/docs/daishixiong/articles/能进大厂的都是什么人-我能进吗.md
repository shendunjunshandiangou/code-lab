# 能进大厂的都是什么人？我能进吗？

<BiliPlayer :bvid="'BV17U9jYXEtx'" />


**摘要**：大厂助教直播对谈——校招进不了大厂可接受低起点再积累；筛选看学历+两段相关实习+硬技能+实习深度反思；**没实习比没学历更致命**；进大厂靠信息/能力积累与把实习挖深；中小厂可通过业务+中台双线争取认可；职场成长靠超越老板视角与真诚沟通。

## 一、校招进不了大厂怎么办？

- 低起点也有机会：第一份工作中小厂/传统企业，积累后仍可回大厂（如 4 年积累后进字节）
- 接受起点，规划职业生涯；大厂不是唯一解，情绪管理与抗压需自评
- 案例：双非+三份大厂实习可校招进大厂；学历好但零实习反而难进

## 二、大厂筛选标准（校招）

| 要素 | 说明 |
| --- | --- |
| 学历 | 漂亮学历是强背书，但非唯一 |
| 实习 | 至少两段与投递岗位相关的实体实习，最好一段大厂级 |
| 硬技能 | 工具会用，简历无明显瑕疵 |
| 实习深度 | 不只写执行内容，要反思串联、体现对业务的推动价值 |
| 信息积累 | 与大厂人交流，用「他们的话」沟通（指标、拆问题方式） |
| 运气 | 被哪个部门捞、面试官、发挥 |

**优先级**：没实习比没学历更致命。

## 三、如何做高薪大厂人 / 快速成长

### 3.1 中小厂双线争取认可（如玉助教）

1. 搞清业务领导、中台领导对数据的认知与 KPI
2. 与业务领导 N-1 建关系 → 拿真实业务逻辑 → 做出别人做不出的成果
3. 选中台领导 OPR 最重指标，帮推项目/写材料
4. 业务+中台同时「吹枕边风」→ 重要场合（季度会等）给曝光机会

### 3.2 超越老板视角（小鱼助教）

- 建立老板视角，甚至 +1、+2 视角；言行站在老板位置思考
- 领导软弱/退缩时顶上去；专业能力是关键时刻的「开门武器」，需软实力铺垫

### 3.3 职场融入

- 非职场：提升人际包容度，尊重不同生活方式
- 大厂：融入节奏与文化；与同事能否成为朋友影响团队生存

## 四、其他要点

- 脉脉等平台薪资帖勿全信（非实名、IT/算法居多）
- 跳槽涨幅看上一份薪资；内部晋升涨幅通常小于跳槽
- 向上管理：先完成领导布置的任务，再谈改善；让领导有掌控感
- 面试对不同面试官拟合风格：和气场合者深聊，对冷漠者 123 点规范化输出；对大佬放低姿态、真诚分享

## 五、快问快答摘录

- 外包大厂：没更好选择可去，简历响应度高于不知名公司
- 数分入行感：数据真正赋能业务策略时；汇报附图表获认可
- 总部数分支持地区：常用 SQL 查询库、优化慢查询、真诚回复消息

## 文章正文

戴师兄邀请两位大厂助教（如玉、小鱼）直播对谈：校招进不了大厂怎么办、大厂筛选标准、如何做高薪大厂人。

**校招进不了大厂怎么办。** 如玉：低起点也有机会——第一份工作中小厂或传统企业，积累后仍可回大厂；人生是体验，没有「必须做到什么」的节点。小鱼：任何结果都要接受；中场或其他公司都是下一份工作的跳板。戴师兄：自己校招没进大厂，4 年积累后进字节；「接受起点、规划职业生涯，进大厂不是非常难的事」。选大厂须想清楚最吸引的点（如收入）与最大缺点（牺牲时间、数分多留一二线城市）能否接受。

**大厂校招筛选标准。** 如玉：①学历是强背书；②至少两段与投递岗位相关的实体实习，最好一段大厂级；③实习深入反思——不只写自己做的事，还要串联同事/加一的工作，完善简历；④硬技能（工具会用）、简历无明显瑕疵；⑤运气（被哪个部门捞、面试官、发挥）。小鱼补充：⑥**信息/人脉积累**——进大厂前是否与大厂人交流，能否用「他们的话」沟通（拆问题、看哪些指标）；⑦**能力积累**——对实习的深入思考，体现对业务的推动价值，「做再小的事都要有价值」。戴师兄：**没实习比没学历更致命**——见过双非+三份大厂实习校招进大厂，反过来学历好零实习反而难进。

**如何做高薪大厂人。** 如玉观察二三线城市：走专业线机会小，想高薪得带人（团队可能与数据分析有偏差），有管理经验后可回一线或留当地。小鱼：高薪要么早期进公司做到管理者，要么创业——「真正能拿钱的肯定不是基层员工」。脉脉薪资帖勿全信（非实名、IT/算法居多）；跳槽涨幅看上一份薪资，**跳槽涨得多于内部晋升**（跳槽是重新估值）。戴师兄分享涨薪路径：提前转正拿加薪→金九银十入职→过年前年终奖→年后 5～6 月数值晋升→再跳槽，12 个月内可能三次涨幅。

**如玉在中小厂争取认可的双线策略。** ①搞清业务领导、中台领导对数据的认知与 KPI；②与业务领导 N-1 建关系，拿真实业务逻辑，做出别人做不出的成果；③选中台领导 OPR 最重指标，帮推项目/写材料；④业务+中台同时「吹枕边风」，争取季度会等曝光机会。

**小鱼：超越老板视角。** 建立老板甚至 +1、+2 视角；领导软弱时顶上去；专业能力是关键时刻的「开门武器」，需软实力铺垫。职场融入：提升人际包容度；大厂要融入节奏与文化。

**其他。** 外包大厂：没更好选择可去，简历响应度高于不知名公司。数分入行感：数据真正赋能业务策略时；汇报附图表获认可。向上管理：先完成领导任务再谈改善，让领导有掌控感。面试拟合风格：对和气者深聊，对冷漠者 123 点规范化输出；对大佬放低姿态、真诚分享。
