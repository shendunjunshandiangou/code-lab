# 教大家一个在职场非常实用的爬虫小技巧，人人都能学会~

> 摘要：一条 2 分半的短教程，用一款无代码采集软件（口播未点名，下载链接在视频简介；"智能采集/深入采集"功能与后羿采集器一致）演示零代码爬取 B 站排行榜数据的完整流程：智能识别字段 → 深入采集详情页 → 启动采集 → 导出数据。

## 操作流程（以爬 B 站排行榜为例）

1. **下载安装**：到官网下载软件（链接在视频简介），安装并打开。
2. **智能采集**：复制目标页面网址，粘贴到软件的网址框，点击"智能采集"——它会自动识别网页上所有可爬取的字段，稍等识别完成即可。
3. **修改字段名**：有些字段自动命名不准，右键"修改字段名称"（例如把某字段改名为"播放量"），点击确定。
4. **深入采集**（可选）：点击"深入采集"可以爬取网页点击后的详情页信息。要新增字段时点"添加字段"，会出现一个笔状指针，直接指到想爬的数据上点击，识别后修改字段名（例如"关注数"），确定。
5. **启动采集**：点击保存 → 开始采集 → 启动，软件自动开始爬数据。
6. **导出数据**：爬完后点停止（或等软件自动提示爬完），点击"导出数据"，在预览处设置导出文件夹（记住存放位置），点导出，再点"查看导出的文件"，双击即可查看数据。

## 注意事项

- 爬取时长随数据量级不同而不同；免费版速度一般。
- 需要"点击进入下一层"的深入采集会比较慢；只爬列表页数据则非常快

## 文章正文

戴师兄在这条约 2 分半的短教程里演示：用一款**无代码采集软件**（下载链接在视频简介；功能含「智能采集」「深入采集」，与后羿采集器同类）**零代码爬取 B 站排行榜**——「一分钟教大家学会数据爬虫」。

**第一步，下载安装。** 到官网下载软件，安装并打开。

**第二步，智能采集。** 复制 B 站排行榜网址，粘贴到软件的网址框，点击「智能采集」。软件会自动识别网页上所有可爬取的字段，稍等识别完成。有些字段自动命名不准，可以右键「修改字段名称」——例如把某字段改成「播放量」，点确定。

**第三步，深入采集（可选）。** 点击「深入采集」可以爬取点击后的详情页信息。要新增字段时，点「添加字段」，会出现笔状指针，指到想爬的数据上点击，识别后修改字段名（例如「关注数」），确定。

**第四步，启动采集。** 保存 → 开始采集 → 启动，软件自动爬数据。爬取时长随数据量级不同；免费版速度一般。**需要点击进入下一层**的深入采集会慢；只爬列表页则非常快。

**第五步，导出。** 爬完后点停止（或等软件提示爬完）→「导出数据」→ 在预览处设置导出文件夹（记住位置）→ 导出 →「查看导出的文件」，双击即可查看。
