# 一周的实习经历，能写在简历上吗？怎么写？

<BiliPlayer :bvid="'BV1a9dfYTEXD'" />


**摘要**：实习不合适就尽早跑（第 1～3 天），否则老老实实做 1～3 个月；短期实习一般无背调，简历时长可灵活处理。关键不在纸面经历，而在面试能否讲出自动化报表、看板和分析报告等真实产出。

## 一、一周就跑：怎么处理

- 简历上会体现为很短的经历
- 若还没上手具体工作，可用「不太适应」或「学校有事」等理由离开
- 实习一般**不做背调**，简历写一周还是一个月看你自己
- **不要**明确承诺实习 3～6 个月却一周就跑——会让对方预期落差大

## 二、核心原则：要么快跑，要么做满

| 选择 | 做法 |
| --- | --- |
| 觉得不对 | 第 1～3 天就跑路 |
| 留下来 | 老老实实实习 1～3 个月 |

## 三、短期实习也要榨干价值

不管多基础，都要：

1. 尽可能把**数据处理过程自动化**
2. 参与会议，积累行业经验
3. 整理出**自动化报表、看板、分析报告**——这是面试谈资

**仅写在纸面上的实习经历没用**，面试讲出学到的东西才能展现硬实力。

## 四、第一份实习：不要挑

- 零实习找实习最难，有了一份之后找后面的会简单很多
- 进入前尽可能问清实习内容，或让领导展示后再入职
- 第一份实习能去就去，不要在内容上过度挑剔

## 文章正文

有小伙伴在群里问：实习只干了一周就扛不住了，想跑，会不会成为简历上的「污点」？

戴师兄先说结论：**实习不合适，就尽早跑；否则就老老实实做满 1～3 个月，中间态最尴尬。**

如果你一周就想走，简历上肯定会体现为很短的经历。但实习一般**不会做背调**，简历上写一周还是写一个月，看你自己怎么把握。需要注意的是：如果你明确承诺要实习三个月或六个月，却一周就跑，会让对方心里产生预期落差。若还没让你上手具体工作，用「不太适应」或「学校有事」等理由离开，通常也说得过去。

回到实习本身，能有一份实习还是要好好珍惜。不管工作多基础，都要尽可能把**数据处理过程自动化**，多参与会议、积累行业经验。一两周或一个月的实习，价值在于两件事：一是让实习经历「坐实」；二是整理出**自动化报表、看板、分析报告**——这些才是面试时的谈资。仅仅写在纸面上的实习经历没什么用，只有你在面试里把学到的东西讲出来，才能展现硬实力。

进入实习前，尽可能问清实习内容，或让领导展示后再入职。但如果是**第一份实习**，他建议不要挑——零实习找实习最难，有了一份之后，简历和面试才有发挥空间，后面会简单很多。
