# 被AI替代，还有的救！

<BiliPlayer :bvid="'BV1w6Q2BPEhQ'" />


**摘要**：讨论「会不会被替代」已无意义——关键是怎样会被替代、怎样不会、已被替代怎么办。从业年限不等于淘汰概率，做重复车轱辘工作的人资历越深反而越危险。

## 一、选项陷阱

按从业年限（0-3 年/3-5 年/5-10 年/10-20 年）判断谁更容易被淘汰是**选项陷阱**——就像问「男人还是女人更富有」，分类维度错了。

**真实规律**：不管什么年限，只要在做**重复工作**，资历越深越难调头，AI 来了瞬间被替代。

## 二、AI 强弱与人才稀缺性

| AI 强 | AI 弱 |
| --- | --- |
| 单一确定性映射（A→B→C） | 不确定性下决策 |
| 有大量数据可参考的标准化任务 | 信息不足时的假设与洞见 |
| 取数、做表、简单统计 | 人际协作、信任交付 |

**两种未来稀缺人才**：
1. 创新或信息不足场景下对目标负责的分析者
2. 提供人类专属情感陪伴的高级服务者

即：**高级脑力劳动**或**高级体力劳动**，重复性工作风险极高。

## 三、不被替代的能力

1. **有限信息下做决策**——连麦三五句话就能判断问题（分析师核心能力）
2. **深度人际协作**——建立信任、交付限价服务
3. **会做假设**——不是盖棺定论，而是提假设再用数据验证
