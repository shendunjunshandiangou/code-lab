# PLC 可编程逻辑控制器

## 一句话定义

PLC（可编程逻辑控制器，1970 年代出现）是工厂的"大脑/CPU"：可提前编程让产线具备"if…then…"的纠错与控制能力（识别空瓶再灌装、控制灌装量与速度），使流水线实现真正的全自动化，被视为继福特流水线之后对供应链的又一次重塑。

## 详细说明

- 解决的痛点：早期自动化没有纠错机制（瓶子没到位喷头照喷），厂商只能雇人盯着拉闸；流水线的错误千奇百怪、无法穷举。
- 1980 年代起 IBM、沃尔玛、丰田、通用、宝洁等流水线公司因 PLC 普及效率大增；如今罐装饮料产线每秒 10-20 瓶，电梯、写字楼、游乐场也遍布 PLC。
- 格局：西门子 Simatic 是 PLC 龙头，全球约 1/3 工厂用西门子控制器。
- 自动化的当代形态：机械臂、AGV 小车、无人仓库（自动堆垛机、厘米级入库）；西门子成都数字化工厂（全球首批灯塔工厂，主产 PLC、"自己生产自己"，组装传输用磁悬浮）；小米、格力的"黑灯工厂"。
