# Nikola One 发布会模型车骗局

## 一句话定义

2016 年 11 月 Nikola 发布会上展示的「能跑 1000 英里的世界最强新能源卡车」，实际是无氢能源技术的巨大模型车——外壳发布会前一周从欧洲运来，靠台下电线偷偷供电，Milton 反复强调 "fully functioning truck" 掩盖心虚。

## 详细说明

### 发布会表象

- 2016 年 11 月大型新闻发布会
- 宣称：**第一次能跑 1000 英里的世界最强新能源卡车**
- Slogan：*"Nikola One — for those who wish to leave the planet better than it was when we found it."*
- 媒体聚焦酷炫车头，对比特斯拉「弱爆了」

### 实际情况

| 事实 | 细节 |
| --- | --- |
| 技术 | Milton **没有任何氢能源技术** |
| 时间线 | 四个月前的「几个月后展示」只是夸下的海口；六个月前仅是个**骨架** |
| 外壳 | 发布会前**一周从欧洲加班运来** |
| 本质 | **巨大卡车模型，根本不能用** |
| 供电 | 台下有**一根线连到卡车上偷偷供电** |
| 可用部分 | 车头内触摸屏、座椅等；车头旁放**氢能源大 logo** 做视觉 |

### 心理操纵

- Milton 从未当着这么多人面骗，心虚，反复强调「这不是假车，能开、能开、能开」
- 公开场合的大胆谎言反而利用**人性弱点**——「当着这么多人面，谁好意思这么吹牛撒谎？」
