# 降息周期资产传导理论

## 一句话定义

理论上降息（其他变量不变）：利好美股/美债/黄金/海外股市/美国房价，利空美元；通胀传导滞后。

## 详细说明

- 美股：融资成本↓、折现率↓、营收刺激。
- 美元↓→黄金/大宗商品（美元计价）↑；海外资产相对吸引力↑。
- Fed 还考量股市、大选、美元走势；强调 data dependent 与预期管理。
