# A股上市三阶段与中介

## 一句话定义

A 股上市经历申报（招股说明书）→ 交易所+证监会审核问询 → 发行路演定价；投行/律所/会所组成中介三军。

## 详细说明

- 招股书：论证适合 A 股、国家战略、盈利、风险可控（「10 万字议论文」）
- 投行佣金约 5–7%，律所会所约 1–3%
- 2019 起注册制更透明，实质审核仍严；新国九条提高质量、强化退市
