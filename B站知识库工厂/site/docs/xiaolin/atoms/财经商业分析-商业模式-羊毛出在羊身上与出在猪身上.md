# 羊毛出在羊身上与出在猪身上

## 一句话定义

两种平台商业模式的根本分野：互联网模式先用免费内容/工具/补贴攒流量再靠广告变现（**羊毛出在猪身上**，如爱优腾）；Netflix 从租碟起家第一天就向用户直接收费、再把钱砸回内容（**羊毛出在羊身上**）——它的核心是娱乐不是互联网，互联网/算法/数据只是提高效率的工具。

## 详细说明

- 判别三问：是互联网模式还是娱乐（产品）模式？从广告收钱还是从用户收钱？羊毛出在猪身上还是羊身上？
- Netflix 的一致性：2002 年上市第一份财报写下的核心目标「为用户提供定制化、多样化的影视体验」20 年从未改变；60%–70% 支出始终花在内容上。
- 对照组：爱优腾以互联网思路起家（免费攒流量 + 广告），近年叠加付费会员仍被吐槽（会员还看广告、抢先点播加钱、多端重复收费）且连年亏损。
- 讲者升华：互联网是工具、算法是方法，能提效和加速扩张但**无法替代产品本身**——就像金融资本自己不生钱、只是提高资金运营效率；工具越给力的时代，越需要优质的产品。
