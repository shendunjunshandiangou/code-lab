# 金本位的顺周期缺陷与 1934 黄金收权

## 一句话定义
金本位（每发 1 美元须有 40% 黄金支撑）逼得美联储在越萧条时越紧缩、放大经济震荡；1934 年罗斯福借《黄金储备法》废除金本位并把黄金收归**财政部**（而非美联储），连同贴现率的实际控制权一起，把美联储架空成财政部的附属机构。

## 详细说明
- **顺周期机制**：早期美联储像"终极黄金商人"——收多少黄金印多少钞票。1929 年大萧条爆发后民众挤兑黄金，美联储黄金储备从 45 亿美元骤降至 32 亿美元，发钞能力被锁死：在最需要刺激的危机高潮反而被迫紧缩。
- **1934《黄金储备法》**：放弃金本位（美元不再需要黄金背书）+ 黄金国有化，民众上缴对象是财政部；还要求美联储把绝大多数黄金归财政部所有——一招收回美联储的黄金职能。
- **货币政策被接管**：贴现率基本上财政部说多少是多少，优先满足国债发行需求。时任美联储主席自嘲：名义上是央行主席，实际是"给财政部打下手做行政工作的人"。
- **二战低谷**：财政部摁着美联储签协议，要求以 2.5% 以下低利率购买国债帮政府融资——连表面独立性都没了（UP 主独立性评分 2 分）。
