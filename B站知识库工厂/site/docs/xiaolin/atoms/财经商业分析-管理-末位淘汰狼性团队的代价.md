# 末位淘汰狼性团队的代价

## 一句话定义
斯基林在安然推行极端狼性管理：年度互评+末位 10%–15% 直接开除，配合双倍于同行的薪酬和巨额奖金（25 岁新人年终奖 500 万美元）——短期造就了「每天工作 18 小时、不惜一切代价赚钱」的战斗力，长期则筛选并奖励了「为利益不择手段」的人格，成为系统性造假的组织土壤。

## 详细说明
- 机制设计：每年业绩评选+同事互相打分，尾部 10%–15% 直接淘汰；工资高出同行一倍；用极限运动激发「原始野性」。斯基林的用人标准原话：「不拘一格、不受约束、不按常规出牌、但能赚钱的人」。
- 短期效果：做市业务年增速 40%–50%，从天然气扩展到电力、塑料、钢铁乃至天气衍生品，「所向披靡」；2000 年做市业务占安然营收 95%。
- 长期代价：
  - **道德筛选逆向化**：淘汰机制惩罚守规矩者、奖励越界者——交易员能一边操纵加州电价致民众断电、一边为奖金欢呼；
  - **无人吹哨**：巨大骗局的每个环节都有知情者，但高奖金+高淘汰让所有人选择沉默；
  - **文化即风险**：肯老早年默许违规交易员（87 年两天亏 9000 万差点破产）已埋下「赚钱高于合规」的基因。
- 管理启示：激励强度越大越需要合规对冲；当考核只认结果不问手段时，组织会精确地长成激励的形状。
