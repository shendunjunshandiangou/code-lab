# 莫拉维克悖论

## 一句话定义

对人类本能简单的走/抓/平衡，机器人需极复杂计算与协调；对人类复杂的逻辑推理反而更易——机器人训练的核心悖论。

## 详细说明

- 婴儿避火 ~0.001 焦耳；机器人需百万级图像+触觉数据训练，单次推理耗能约为婴儿 180 万倍。
- 波士顿动力强在运动性能；大模型改善视觉/听觉，**触觉**仍是瓶颈。
