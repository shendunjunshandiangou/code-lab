# 住房开工与供给弹性

## 一句话定义

房屋供给周期长、弹性低，在人口密集发达区影响力通常不如需求；地广人稀处供给可能成为决定房价的首要因素。

## 详细说明

- 供给 = 市场净增出售房屋
- 常用指标：**Housing Starts**（新房开工）
- 土地有限的发达区：供给波动小，与房价有时呈正相关（非因果否定供给，而是周期长、弹性不足）
- 芬兰政府报告：分析美国经验后认为本国房价最主要因素是房屋供给
