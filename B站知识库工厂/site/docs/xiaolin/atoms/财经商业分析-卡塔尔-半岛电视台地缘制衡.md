# 半岛电视台与卡塔尔地缘制衡

## 一句话定义

卡塔尔创办半岛电视台（Al Jazeera）输出全球影响力，配合美军基地驻军，在中东沙特—伊朗夹缝中制衡周边大国。

## 详细说明

卡塔尔地处波斯湾「小豆」，财富伴动荡。中东分沙特阵营与伊朗阵营，卡塔尔虽为阿拉伯国家却不愿站边沙特，与伊朗较近，常遭沙特找麻烦。

哈马德制衡策略：
1. **美军驻军**——美国在中东最大军事基地之一；
2. **半岛电视台**——阿拉伯国家中最具影响力的全球性媒体，时常炮轰沙特、阿联酋；
3. 能源筹码与灵活外交。

美国撑腰 + 媒体影响力，使沙特虽看不顺眼却不敢轻举妄动。
