# 内幕消息陷阱与独立思考

## 一句话定义
利弗莫尔第二次破产的教训：「棉花大王」朋友给他内幕消息让他做多棉花，他半信半疑仍加杠杆入场，结果对方联合供应商清仓砸盘、棉价暴跌——由此他把「相信自己：别人的话可以听，但不能全信」刻进交易原则，即独立思考。

## 详细说明
- 案例结构：消息提供者本身是利益相关方（棉花大王手握巨量库存），「内幕消息」实为诱多出货的一环——信息不对称下，白给的消息往往是猎人的诱饵。
- 心理弱点：刚在 1907 年大胜后的膨胀期，自信外溢为轻信；「半信半疑却重仓」是最危险的状态——仓位与信念不匹配。
- 原则内涵：独立思考不是拒绝信息，而是把任何外部信息（包括大佬观点、内幕消息）都当作待验证的输入，用自己的分析框架消化后再决定仓位。
- 代价与修复：破产后他花整整六个星期不做一笔交易、只观察研究，等到一战钢铁股机会（98 买入、hold 至 145）翻本——用纪律修复了轻信的代价。
