# TCO 总拥有成本与数据中心采购

## 一句话定义

企业客户（云服务商、数据中心）买服务器 CPU 讲究 **TCO（Total Cost of Ownership，总拥有成本）**：购买价只是冰山一角，芯片 7×24 不间断工作的耗电、维护、散热等持续成本才是水下大头——AMD 在官网做了赤裸裸的对比工具（任选一款 EPYC 对一款英特尔至强，比价格/速度/用电/TCO），同级芯片便宜约 20%–50%、维护成本低 10%–30%，一年买上万台服务器的客户能省几个亿。

## 详细说明

- 与消费市场的区别：消费品买完就完了；服务器是持续经营性资产，采购决策高度理性、量化。
- 拉踩的底气：公开对标若造假英特尔「不得告死它」——敢比说明数据站得住；国内有广告法限制，国外可以直接这么比。
- 企业客户粘性的另一面：不像消费者说换就换，所以 EPYC 份额没有跃迁式飙升，但 AWS、Azure、谷歌云、IBM 云都已部分采用，国内互联网企业采购比例据称也不低。
- 背景：数据中心 CPU 曾完全是英特尔的天下、是其命根子——科技公司可能一半预算花在服务器上；抢这块业务的先决条件是能造出最顶尖的芯片（对应苏姿丰的高性能计算战略）。
