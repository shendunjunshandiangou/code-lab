# 独占策略与使命召唤筹码

## 一句话定义

独占（exclusive）是主机平台纵向整合的极致玩法——自家游戏只能在自家主机上玩，任天堂玩得最溜（Switch 81% 营收来自自家游戏）；《使命召唤》则是独占博弈的最大筹码：游戏史上经济回报最成功的 IP 之一，2021 年仅这一款就给索尼带来 15 亿美元收入，索尼还签了「2023 年底前新作只上 PS」的独占协议——微软收购动视暴雪直接威胁到这棵摇钱树，才引爆整场收购战。

## 详细说明

- 独占的竞争逻辑：会员制（游戏圈 Netflix）时代，平台上游戏尤其爆款的数量决定订阅吸引力；独占既拉新又锁客——若微软独占使命召唤，PS 用户可能大量流失、索尼老大地位动摇。
- 三巨头格局背景：PS 约占主机市场一半、Xbox 和 Switch 各约四分之一；索尼与微软用户和游戏重合度极高、硬刚十几年，任天堂靠封闭生态独立。
- 讽刺的攻防转换：法庭上微软摆数据——Xbox 独占游戏 56 个 vs PS 296 个，「独占这事就是行业一哥索尼搞出来的」——独占既是索尼的护城河，也成了它指控别人时的软肋。
- 索尼买不起的现实：论行业影响力索尼更胜一筹，论财力完全不是对手——全球拿得出 687 亿现金的科技公司只有微软、谷歌等寥寥几家，连苹果、腾讯手里都没那么多现金，索尼只能靠监管使绊。
