# 美国银行体系：证券化与资本全球渗透

## 一句话定义

宽松监管下美国发展出高度证券化的融资体系（发债、股票、衍生品替代银行贷款），资本能迅速渗透到全球回报率最高的角落——这种灵活的资本调配能力是靠银行简单放贷的国家做不到的。

## 详细说明

**成熟证券市场**：华尔街 1792 年就开始交易；借钱不一定找银行，可以去市场发债、发股票。

**衍生品市场**：如房利美、房地美把房贷打包成债券（MBS）在二级市场发行。

**双刃剑**：市场化为 2008 年次贷危机埋下隐患（过度放贷、过度杠杆、过低风控）；但好处是钱能迅速流向美国乃至全球高回报领域——"哪有潜在高回报，华尔街资本啪一下就过去了"，市场随之快速发展。
