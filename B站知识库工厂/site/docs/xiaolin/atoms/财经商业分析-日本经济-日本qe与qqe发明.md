# 日本 QE 与 QQE 发明

## 一句话定义

2001 年日本在全球首创 QE；2016 负利率 + QQE 目标压低 10 年期国债至 0.25% 以下，央行甚至成为股市最大单一股东之一。

## 详细说明

- 1999 零利率；2001 QE 四年资产负债表扩 2.5 倍
- 2006–07 停 QE 后 2008/2011 再受重创
- 安倍经济学第一支箭：黑田东彦负利率（商业银行部分 -0.1%）+ QQE
- 效果：资产价格狂欢，GDP/通胀改善有限
