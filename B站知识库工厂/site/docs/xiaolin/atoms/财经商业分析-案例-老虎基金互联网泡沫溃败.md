# 老虎基金互联网泡沫溃败

## 一句话定义
老虎基金 1990 年代末做空被高估的互联网股+做多蓝筹对冲，却在泡沫疯涨与投资人撤资夹击下从 210 亿美元缩至 7 亿，2000 年初关闭——半年后纳斯达克一年跌 75%，方向对了、时机没熬到。

## 详细说明
- 老罗（Julian Robertson）重基本面，认为 1995 年后大量不盈利即上市的互联网公司被严重高估，大举做空并买蓝筹对冲（多空策略）。
- 市场却发明新估值模型（客户价值×增长指数）为互联网公司定价，产业急速膨胀、股价持续疯涨。
- 双重挤压：股价上涨→持续补保证金；投资人质疑「互联网是未来怎么能做空」→疯狂撤资。管理规模 210 亿→7 亿美元，2000 年初关闭基金「这市场我看不懂了」。
- 关闭后半年泡沫破裂：纳斯达克一年跌 75%，互联网企业大量破产（.com bubble）。有人怀疑泡沫破裂前的最后疯涨正是老罗等大空头被轧空所致。
- 教训：判断对方向还不够，必须抓对时机——后者比登天还难。老罗在巨亏时果断金盆洗手，反而保住江湖地位。
