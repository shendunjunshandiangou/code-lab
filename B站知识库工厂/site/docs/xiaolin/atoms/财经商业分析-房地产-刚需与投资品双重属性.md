# 房地产刚需与投资品双重属性

## 一句话定义

住房既是人人需要的刚需，又是价值取决于未来市场预期价的投资品，导致「越涨越买」并占据家庭资产极高比重。

## 详细说明

- 刚需：不像股票债券可不买
- 投资品：价值不取决于实际效用，而取决于预期市场价
- 中国约 70% 家庭资产为房产；美国约 28.9%
- 越不富裕家庭，房产占比越高 → 连接经济体中几乎每个人
