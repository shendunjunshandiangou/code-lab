# 三代互联网与数据所有权

## 一句话定义
Web1.0（1991 起）只读——用户只能看信息（新浪、搜狐）；Web2.0（2004 Facebook 起）可读可写——用户创造内容、互动（微博、B站、抖音），并孕育出"免费产品吸引用户 → 收集数据 → 广告变现"的薅羊毛商业模式；Web3.0（以太坊创始合伙人 2014 提出、2021 走红）主张基于区块链让用户**拥有**自己的内容和数据——"拥有自己的羊毛"。

## 详细说明
- **羊毛经济学**：Web2 公司种草（免费产品）引羊（用户）、薅羊毛（数据→广告）。早期好产品稀缺、用户不在意数据，双方相安无事；互联网成熟后产品过剩，稀缺的变成了用户数据——数据所有权矛盾催生 Web3 叙事。
- **Web3 的真正野心**：不是单点技术创新，而是以区块链为底层、与现实隔离的一整套经济/生态系统——无中央管理者，靠激励机制自主运行、自我升级，有自己的货币、组织形式、服务模式。理解入口：把"比特币对支付系统的挑战"扩大到整个网络。
- **概念与现实的落差**：理想听着像科幻片，但商业版图现状"除了交易就是游戏"（交易所、DeFi、NFT、GameFi）——与元宇宙同病：远大叙事 vs 就这。
- 与元宇宙的关系：元宇宙是上层的沉浸式数字世界，Web3 是更底层的基础架构，元宇宙大概率构建在 Web3 之上。
