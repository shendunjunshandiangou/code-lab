# 求职规划三要素

## 一句话定义

找工作中最关键的一步是求职规划，分为明确方向、评估准备工作、安排时间线三块，为后续简历面试与实习打基础。

## 详细说明

1. **方向**：想进什么行业/岗位；结合专业、兴趣、性格与软硬技能要求
2. **准备**：硬技能、软技能、实习社团项目、简历面试技巧
3. **时间线**：各公司招聘节奏不同（如投行大一大二即开始探索项目），宜尽早排期

方向错误会导致「东一榔头西一棒」；专注一条线的竞争者深度不可比。
