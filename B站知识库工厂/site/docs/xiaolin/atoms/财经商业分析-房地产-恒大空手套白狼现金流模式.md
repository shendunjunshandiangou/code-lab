# 恒大空手套白狼现金流模式

## 一句话定义

1997 年一穷二白起家的许家印，靠极致现金流管理撬动最大杠杆：拿地款找政府谈分期 → 借政府背书向小银行借钱付首付 → 供应商先付一半、开盘后付尾款 → 房子盖一层就卖期房拿客户的钱——政府、银行、供应商、客户借了个遍，第一个楼盘回笼 8000 多万，是拿地款的 16 倍。

## 详细说明

- 房地产是重资本行业，拿地、盖楼、销售每个环节都要钱，这套模式让资金周转起来且自己大赚。
- 杠杆越大风险越大，但 2000 年代初地产黄金期"谁杠杆开得最大、玩得最野，谁挣得最多"——恒大高歌猛进扩张全国，2009 年港股上市，许家印以 400 多亿身家成中国大陆首富。
- 命门：2016 年负债已超 1 万亿，现金短债比一直小于 1（兜里的钱不够还当年的债），全靠房价上涨预期下借新还旧腾挪，一旦融资断流即资金链断裂。
- 该模式（期房预售 + 高杠杆周转）曾是全行业常态，也是后来三条红线针对的对象。
