# 加息的国内传导：僵尸企业与金融业

## 一句话定义
日本加息对国内的传导分层：政府借债成本是渐进上升（只影响新发国债）；50 多万家靠零息贷款存活的僵尸企业可能迎来倒闭潮；银行与金融系统则因利差空间恢复而受益；股市影响多空叠加、方向未知。

## 详细说明
- 政府端·渐进非突变：已发债务利率已锁定，只有新发国债利率变高，影响随新债占比上升慢慢体现——即使利率骤调 20%，政府也不会第二天破产。央行同时承诺维持国债购买量（"印钞机还开着，别恐慌别卖国债"），短期影响非常有限；但长期高债务下借债成本上升，轻则限制财政刺激力度、重则动摇政府信用。
- 企业端·僵尸企业：日本超过 50 万家僵尸企业长期靠近零利率贷款续命；央行只保国债、企业端撒手不管，很可能出现一小波倒闭潮。加息最底层、最广泛的作用就是抑制经济活力，严重时可引发信用危机。
- 金融业·利好：银行厌恶低利率——负利率年代利润空间被挤压极小；加息后银行利差恢复，整个金融系统被激活。
- 股市·未知：利好（资本回流、通胀正常化）与利空（抑制经济、压估值、日元走强伤出口）叠加，说不准。
