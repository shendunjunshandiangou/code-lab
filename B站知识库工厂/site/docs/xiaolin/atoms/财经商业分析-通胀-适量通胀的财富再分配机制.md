# 适量通胀的财富再分配机制

## 一句话定义
通胀是无时无刻不在进行的财富再分配：藏在床底的钱购买力持续流向新创造的钱，逼着人们去赚钱、生产、及时消费，从而抬升总需求与 GDP——这就是主流央行把 2%–3% 通胀设为目标而非零通胀的原因。

## 详细说明
- 直觉上货币越稳定越好，但完全稳定的货币让持币者太安逸；温和贬值制造「今年不花、明年变少」的紧迫感，激发消费与生产——「一潭死水，搅和搅和就激发出更大活力」。
- 对经济体而言钱只是媒介，奶茶 3 块还是 300 块不重要，重要的是更高的产值：温和通胀服务于 GDP 增长。
- 度的把握：美、欧、英、日、加、澳央行通胀目标约 2%–3%；多数其他经济体即使不公布明确目标，也希望控制在 2%–5%。把通胀维持在合理区间是央行头号目标，哪怕代价是牺牲短期增长甚至短期衰退——保住通胀是发达经济体的底线。
