# OpenAI 非营利控股结构与权力集中

## 一句话定义

OpenAI 的架构：非营利组织 OpenAI Inc（**没有股东、没有股东大会，董事会即最高权力且法理上无人能撼动**）掌控一个持股平台，持股平台持有营利实体 OpenAI Global 51% 的股份——因此微软 130 亿美元换 49% 股份却没有一个董事席位，哪怕奥特曼、布罗克曼和微软联手，从控制权角度也斗不过六人董事会；这个「半路出家转营利」的蹩脚结构正是 2023 年宫斗的根因。

## 详细说明

- 演变：2015 年成立时是纯非营利（大佬合投几亿美元，为 AI 理想）；因训练烧钱 + 需要股权激励员工，2019 年转型营利，投资人收益封顶 100 倍、超额回投公司。
- 权力链条：OpenAI Global（营利、员工为其工作、微软持股 49%）← 控股平台 51%（非营利组织 + 员工 + 投资人共同持股）← OpenAI Inc（非营利）← 六人董事会。**问题不在利益分配，在权力分配**。
- 类比：与三星财阀用极少股份层层控制整个财团异曲同工。
- 马斯克的教训：作为关键发起人捐了约 5000 万美元，但非营利组织没有股东概念，分道扬镳后彻底被抛弃、毫无追索。
- 创始人的情怀面：转型公告强调董事会多数成员不拿经济利益，奥特曼一点股份没拿（国会听证「我赚了很多钱，但没赚在 OpenAI」）；一般非营利组织无利益纠葛没问题，问题出在「没想赚钱→潜力太大→半路转营利」造成的权力过度集中。
