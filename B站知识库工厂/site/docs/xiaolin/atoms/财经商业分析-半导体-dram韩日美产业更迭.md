# DRAM 韩日美产业更迭

## 一句话定义

1970 英特尔 → 1980 日本五杰 80% → 1983 三星东京宣言 → 韩企逆周期投资主导；财团+政府补贴模式与日类似。

## 补充（三星王朝篇）

三星入局起点更早：1974 年李健熙在全员（含其父李秉喆）不看好的情况下，自掏 4 亿韩元收购濒危的"韩国半导体"公司改名三星半导体；约五年后李秉喆亲自将半导体定为集团核心业务并建成世界级工厂，恰好赶上 PC 普及带来的指数级需求——这笔"备胎"的个人押注成为三星电子日后统治 DRAM 的源头。
