# 劣质老板与离职决策

## 一句话定义

向上管理有边界：若老板业务能力差、态度横、不讲道理还爱发脾气，再好的沟通技巧也救不了——该找下家就找下家。

## 详细说明

小Lin说在讲完高级拍马屁和向上管理全套方法后，明确给出例外情形：

**建议离职的老板画像：**

- 业务能力也不怎么地
- 整天特横
- 不讲道理
- 整天发脾气

**逻辑：**

向上管理、站队思维、结果导向沟通，前提是老板也是理性组织角色、愿意共同把事办好。若老板本身不合格且难以改变，再高超的「拍马屁」也只是消耗自己。

**建议：赶紧找下家。**

这与「站在老板立场」「成为靠谱下属」形成对照——技巧用在值得合作的老板身上才有意义。
