# 多策略分散与杠杆容量

## 一句话定义

把资金均摊到多个互不相关的策略可大幅降低组合波动率（9 个不相关策略把 10% 的日标准差降到 3.33%，100 个降到 1%）；组合越分散，能承受的杠杆越高，预期收益越高。

## 详细说明

**数学逻辑**：单一策略日标准差 10% 意味着每天一个涨跌停级别的波动；均摊到 N 个互不相关的同风险策略，组合标准差按 1/√N 下降。

**LTCM 应用**：50 倍杠杆下资产波动 2% 就归零，短期风险靠分散化解——其每日收益标准差约 4500 万美元，看似巨大，但不足资产组合的 1%，且是在三五十倍杠杆之上。

**致命前提**：策略之间必须真正不相关。LTCM 用过去 5 年数据回测确认了独立性，但样本里没有黑天鹅——恐慌时刻所有资产相关性趋近于 1，分散瞬间失效。
