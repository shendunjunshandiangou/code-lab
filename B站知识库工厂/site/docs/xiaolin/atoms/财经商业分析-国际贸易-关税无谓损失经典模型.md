# 关税无谓损失经典模型

## 一句话定义

经典贸易模型：加关税使消费者剩余减少、生产者与政府剩余增加，但社会总剩余减少两个三角形区域——无谓损失，整体效率下降。

## 详细说明

- 无关税时社会总剩余为一定面积
- 加关税后总剩余缩小（两个小三角 = 无谓损失）
- 结论：消费者变差，政府与生产者变好，**全国整体变差**
- 经济学课本（一年级水平）主张贸易自由化效率最大，关税/补贴/限制降低社会总福利

**局限**：模型未考虑幼稚产业保护、谈判筹码、民意等现实因素——业界共识是无差别大规模关税不利，但针对性保护在特定时机可能长期有利。
