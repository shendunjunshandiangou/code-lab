# 跨国监管套利与反洗钱博弈

## 一句话定义

洗钱者利用各国监管规则差异跨国叠层设障，与监管者、国家间及银行内部形成持续博弈；加强一国审查会推动资本流向监管更「通畅」的枢纽。

## 详细说明

- **猫鼠游戏**：链条越长潜在破绽越多，但跨国追查人力与时间成本巨大。
- **避税天堂角色**：打着保护隐私吸引资金，反洗钱推进「推一下挪半步」——部分不配合全球反洗钱 partly 因配合会流失业务。
- **新加坡案后续**：加强审查后，部分富豪嫌麻烦转向香港设家族办公室——并非意味转香港的钱是脏钱，更多是嫌合规流程变繁琐。
- **气球隐喻**：金融发达地区反洗钱系统看起来密不透风，实际一捅就破。
- **博弈层次**：洗钱者 vs 监管者；国家 vs 国家；国内监管 vs 银行一线。
