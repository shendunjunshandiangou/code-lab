# 投行上下级制衡与内部跳槽

## 一句话定义
投行上下级之间是一种微妙的相互制衡：老板要在「淘汰不行的、留住优秀的、防范太强的」之间走钢丝，下属则在「保业绩」与「随时评估外部机会」之间摇摆；表面和气，任何一方势力变化都会打破平衡——摩根大通以鼓励内部跳槽的制度设计来疏导这种张力。

## 详细说明
- 老板的三层心思：下属不行→想办法弄走；下属优秀→防跳槽、要留人；下属太强→防止其势力盖过自己（真实案例：下属跳去买方，两年后跳回来成了老板的老板）。投行没有论资排辈，精英之间上级地位并不永久。
- 下属的两层心思：业绩不好→饭碗不保（开人是常态）；业绩好→怀疑自己被低估，考虑跳槽。
- 平衡的脆弱性：华尔街人人强势，任何一方筹码变化即触发流动——这解释了行业超高的人员流动率。
- JP 的制度化解法：建内部招聘系统、鼓励内部跳槽——内部招人成本低，又避免人才流向竞争对手；员工上班时间刷内部职位是公开的秘密（但要防老板看见，同事互相「放风」）。
- 与老板关系的现实建议：奖金基本由直属老板决定，跟老板关系不好时，靠「政治斗争」扳倒老板的概率极小，理性选择是找下家。
