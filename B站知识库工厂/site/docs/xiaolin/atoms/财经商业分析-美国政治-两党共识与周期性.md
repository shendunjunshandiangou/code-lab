# 两党共识与周期性

## 一句话定义

无论谁当选，美国都将继续增发国债、贸易保护主义（尤其对华）及口头降通胀；共和党上台常伴市场偏热周期；小费免税等选举噱头两党趋同。

## 详细说明

**共同政策**：
- 增发国债刺激经济（特朗普可能更多）
- 贸易保护主义，尤其针对中国（特朗普更激进）
- 口头降通胀，实际政策多反向，通胀或仍靠美联储加息

**选举噱头共识**：**小费不征税**——特朗普为争取摇摆州服务业人群先提，哈里斯跟进；无经济学依据，纯选举政治。

**市场周期性**：共和党减税 + 放松监管 → 市场偏热甚至过热；经济与市场需要时两党轮换。历史上两党交替与经济表现存在较明显周期关联。

**政府支出分野**（虽都花钱）：特朗普重国防；哈里斯重社会福利——量相近、方向不同。

**移民**：拜登任内 2023 年超 200 万非法移民涌入后收紧；特朗普主张激烈驱逐（1500 万–2000 万，数字不切实际）。
