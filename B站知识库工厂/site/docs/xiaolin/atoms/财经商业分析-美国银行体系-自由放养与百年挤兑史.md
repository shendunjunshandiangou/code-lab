# 美国银行体系：自由放养与百年挤兑史

## 一句话定义

美国银行体系"由松收紧"：从各州银行随意发钞的极度自由出发，历经 1866/1893/1907 三次大挤兑催生美联储（1913），大萧条 9000 家银行倒闭催生 FDIC 存款保险——崩一次收紧一点。

## 详细说明

**极度自由期**：独立战争后 13 个州"搭伙过日子"，无中央银行，谁都能发行银行票据；20 世纪初美国有 2 万家银行（现在约 5000 家，当时人口仅为现在的 1/4）。美国人反感"骑在头顶上的大公司"，两次尝试的中央银行（美国第一银行、第二银行）都在成立 20 年后被各州声讨解散；南北战争后才真正统一货币。

**挤兑与制度回应**：

| 危机 | 制度回应 |
| --- | --- |
| 1866/1893/1907 三次大挤兑（1907 靠政府求 JP Morgan 出面联合银行业救市） | 1913 年成立美联储 |
| 1929 大萧条，约 9000 家银行倒闭，罗斯福给银行"放假"阻断挤兑 | 设立 FDIC，为 25 万美元以内存款提供保险 |
| 2008 次贷危机 | 多德-弗兰克法案 + 巴塞尔协议 III |
