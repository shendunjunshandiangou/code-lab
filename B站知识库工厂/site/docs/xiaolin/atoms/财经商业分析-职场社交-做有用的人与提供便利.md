# 做有用的人与提供便利

## 一句话定义
职场小白提升社交价值的路径是让自己「有用」（当志愿者、搭把手、汇报进步、送祝福），并在求助时把事情做成对方的举手之劳（如替对方写好转发文案）。

## 详细说明
**让自己变得有用**（学生/小白初期提供不了资源时的替代方案，按力度递减）：
- 对方组织活动 → 去当志愿者；
- 对方查资料做研究 → 搭把手；
- 实在帮不上忙 → 汇报自己的进步，让对方觉得付出的时间有价值；
- 最起码逢年过节送个祝福。

**给对方提供便利**：求助的事对对方是举手之劳，获助概率大很多；很难搞则要掂量交情。
- 例子：想让 A 介绍认识 B，与其只说「帮我介绍一下」，不如把 A 要转发给 B 的介绍文案替 A 写好（你是谁、什么背景、为什么找 B），A 直接转发即可，免去编辑和「介绍不清楚」的顾虑。
