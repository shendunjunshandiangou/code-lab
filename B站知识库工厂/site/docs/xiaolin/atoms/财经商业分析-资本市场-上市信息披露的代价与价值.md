# 上市信息披露的代价与价值

## 一句话定义
上市公司须完全公开财务信息（季报、年报、未来展望与策略），代价是暴露商业机密（华为不上市的原因之一），但也可以是主动资产——京东上市就是为了用公开报表粉碎「资金链断裂」谣言。

## 详细说明
- 披露义务的逻辑：保障大众投资人的投资安全 → 财务信息完全公开，每季季报、每年年报，含未来展望与发展策略。
- 两难：不想全盘托出家底（发展规划、商业策略），但不写市场就认为你没规划、股价下跌。华为不上市的原因之一是不愿透露商业机密。
- 反例（披露的价值）：刘强东称京东上市「就是为了披露信息」——当时谣言频传京东资金链要断裂，供货商恐慌；辟谣会被讥「此地无银三百两」，不辟谣供货商寝食难安。上市公开报表后谣言不攻自破，供货商睡得安稳。
