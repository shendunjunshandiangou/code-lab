# 十五五高质量发展三大重点

## 一句话定义

「十五五」建议以「高质量」为高频主线（全文 25 次），经济发展三大重点方向为内需、产业、科技；另有一条并列主线为安全。

## 详细说明

| 重点 | 核心要义 |
| --- | --- |
| **内需** | 居民消费率明显提高；统一大市场破除地方保护 |
| **产业** | 新旧动能转换；智能化、绿色化、融合化；传统产业转型 + 新兴产业培育 |
| **科技** | 原始创新、攻克核心技术；超常规措施、新型举国体制 |
| **安全**（并列主线） | 民生兜底、产业链/网络/数据/AI 等安全、化解房地产/地方债/中小金融机构风险 |

产业并非取代经济：制造业 PMI 今年多在 50 附近徘徊，传统链条偏弱，产业被放在极关键位置——「老登不退场，小登上桌帮老登」。
