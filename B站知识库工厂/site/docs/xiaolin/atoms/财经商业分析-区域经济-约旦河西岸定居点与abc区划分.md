# 约旦河西岸定居点与 ABC 区划分

## 一句话定义

以色列在西岸建 250+ 犹太人定居点（逾 70 万人），按奥斯陆协定将西岸划为 A/B/C 三区——巴勒斯坦真正自治的 A 区仅 18% 且被切成 165 块孤岛。

## 详细说明

**定居点**：1967 年后以色列在无人区建现代化社区（小洋房、超市、大学、16 个工业区），房价低于以色列本土，吸引约 70 万犹太人；国际社会认定非法。

**ABC 区**：
- A 区 18%：巴勒斯坦完全自治
- B 区 22%：巴政府管理，以军联合管理
- C 区 60%：以色列完全控制，含大部分肥沃土地与约旦河谷

**双轨生活**：黄牌路畅通 vs 白牌路检查站；巴勒斯坦人常凌晨排队数小时过关；80% 人口仅获 17% 淡水资源。

**许可证制度**：建房、种树、进口书、出生登记等皆需以色列许可，严重阻碍经济交换与产业发展。
