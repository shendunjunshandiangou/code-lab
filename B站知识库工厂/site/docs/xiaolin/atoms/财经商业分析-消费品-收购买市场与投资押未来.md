# 收购买市场与投资押未来

## 一句话定义

成熟集团资本配置的两条腿：**收购**——产品已跑通、买的是市场潜力，金额大风险小、盈利导向、赚当下的钱（欧莱雅 174 亿人民币收购 Aesop，当时其在中国仅两家门店，典型"产品成熟、市场未开发"）；**投资早期初创**——可能连产品都没有、买的是技术与科研，风险大、研发导向、押宝未来。

## 详细说明

- 平衡的本质：布局当下 vs 布局未来——收购为集团提供盈利与增长保障，初创投资提供押注未来的期权。
- Aesop 是欧莱雅史上最大收购（2023 年，澳洲高端护肤品牌）。
- 科研端延伸：欧莱雅基金会与联合国教科文组织合办杰出女科学家奖（颜宁曾获奖）——早期布局延伸到基础科学。
- 该框架可推广：当行业增长放缓、技术范式可能突变时，现金牛业务并购守当下 + 风投组合押未来是大集团的标准应对。
