# 卡塔尔外籍劳工人口结构

## 一句话定义

卡塔尔近 300 万人口中公民仅约 31 万（11%），男性占 75%，印度籍最多，低失业率背后是严苛的务工准入与恶劣劳工条件。

## 详细说明

| 指标 | 数据 |
| --- | --- |
| 公民占比 | 约 11%（31 万 / 300 万） |
| 男性占比 | 约 75% |
| 最大外籍来源 | 印度（约 70 万） |
| 失业率 | 约 0.1%–0.3% |

基建体力活多由男性外来务工承担，无工作则难以入境，故失业率极低。但大量劳工低薪、保障不足，早期签证近乎半绑架；50°C 工地作业缺乏保护；2010 年后大规模工程致约 6500 外籍劳工死亡（估算）——人道主义问题严重。
