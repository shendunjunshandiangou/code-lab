# 关税作为谈判筹码与胆小鬼博弈

## 一句话定义

在米兰框架里，关税的主要目的不是收钱而是谈判：先加很高的关税，再用"降关税 + 军事保护伞"作筹码，要求各国配合对中国加税、投资美国或直接让利——这是一场看谁先扛不住的胆小鬼博弈（chicken game）。

## 详细说明

- 米兰建议的谈判话术："我可以降关税、提供军事保护，条件是你配合我一起对中国加关税，或投资美国，或直接给我写支票；不妥协就是高关税 + 没有保护伞。"这就是"重组全球贸易体系"的重要部分；贝森特采访也透露过要求贸易伙伴配合"攻击北京"。
- 胆小鬼博弈的成立前提：美国经济相对坚挺 + 安全防御伞筹码 → 米兰判断他国大概率先妥协。他没料到的是特朗普**同时跟所有国家**玩这个博弈。
- 解放日的 Excel 表（纳瓦罗主导，发布会前三小时才定、公式简单粗暴、连企鹅岛都加税）说明团队不在意具体数字——那只是谈判起点，像小商品市场"成本 50 喊价 1000"；但国与国之间没有这种砍价默契，直接引发全球金融市场恐慌。米兰、贝森特事后都撇清"数字不是我定的"。
- Bill Ackman 讽刺："有时候最好的谈判策略就是让对方觉得你是个疯子。"
- 后果：市场先教育了特朗普（债市暴跌→暂停 90 天），各国看出软肋——不用反制，市场就先治你，威胁分量大减。
