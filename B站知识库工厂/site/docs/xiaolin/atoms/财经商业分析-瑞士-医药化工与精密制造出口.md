# 瑞士医药化工与精密制造出口

## 一句话定义

保密银行业衰退后，医药化工（占出口半壁江山）与精密制造/手表（Swiss Made）成为瑞士人均 GDP 长期领先的核心产业引擎。

## 详细说明

- **医药**：罗氏、诺华等；医药行业平均月薪约 1.15 万美元；高人才聚集效应。
- **制造**：迅达、ABB、利博海尔；手表产量全球 &lt;2% 但利润 >50%。
- **Swatch 策略**：1983 年低成本塑料表扛住日本石英表冲击，后整合欧米茄、浪琴等。
- **出口结构**：医药化工占出口约一半；瑞士不愿加入欧盟 partly 因贸易顺差与产业结构差异。
- **经常账户**：长期巨额顺差支撑瑞郎需求。
