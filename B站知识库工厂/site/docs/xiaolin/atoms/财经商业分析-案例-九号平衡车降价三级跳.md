# 九号平衡车降价三级跳

## 一句话定义

九号公司把平衡车从 Segway 的 8 万元一路降到 1999 元的三级跳：牺牲国内用不上的性能 + 供应链人工成本优势做出 3 万元"风行者"（2B 景区租赁，2012 年营收 5000 万）→ 性能优化 + 规模效应推出 1 万元 Ninebot E（2C，2013 年营收两三亿、毛利 40%）→ 两年研发把电动车轮毂电机移植到平衡车、加入小米生态链，2015 年发布 1999 元九号 mini，年销从几万台跃至超百万台。

## 详细说明

- 创业起点：北航机器人背景的高禄峰 2009 年发现 8 万元的平衡车"有市场、有技术门槛、可代步"，与师弟王野 2012 年创立公司，创始团队 9 人故名 Ninebot。
- 定价洞察：每一级价格对应一个市场层（景区 2B → 有钱人玩具 → 大众消费品）；1 万元时全球平衡车市场容量仅几亿，天花板极低，降到两三千才能打开大众市场。
- 降价的底层逻辑不是"便宜多卖"，而是解锁产品的自传播属性（见相关原子）；技术路径是核心零件突破（轮毂电机）带动整车简化与成本骤降。
- 冷启动助攻：天使投资人胡海泉骑车出街、送明星朋友，免费完成早期传播。
