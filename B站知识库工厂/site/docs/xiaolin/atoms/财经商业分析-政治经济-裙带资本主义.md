# 裙带资本主义

## 一句话定义

普京整顿不听话的旧寡头后，树立听话的新寡头并国有化部分企业，超一半经济由寡头控制且与政府利益绑定——形成 crony capitalism（裙带资本主义），巩固寡头格局、阻碍创新。

## 详细说明

- 旧寡头：流放、坐牢、自杀
- 新寡头：如切尔西老板阿布在普京帮助下崛起
- 至今超一半经济通过听话寡头控制在政府手里
- 与日韩财阀类似，政府与寡头利益绑定越来越紧
- 导致腐败严重、创新受阻；2014 与 2022 西方制裁对经济影响巨大
