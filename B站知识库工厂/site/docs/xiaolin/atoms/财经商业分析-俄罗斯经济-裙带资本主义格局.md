# 裙带资本主义格局

## 一句话定义

普京整顿不听话的旧寡头、树立听话的新寡头管理半国有企业，形成政府与寡头深度绑定的裙带资本主义（Crony capitalism），超一半经济由此控制。

## 详细说明

**普京对旧寡头的整顿**：
- 七大寡头以为普京是「毛头小子」、傀儡
- 实际上大刀阔斧处理不听话者：流放、坐牢、自杀
- 收紧叶利钦时期经济政策，越来越多企业重新国有化

**新寡头体系**：
- 树立听话的新寡头管理半国有企业
- 案例：阿布在普京帮助下成为新寡头，才得以收购切尔西
- 目前超一半经济通过听话寡头控制在政府手中
- 政府与寡头利益绑定越来越紧，进一步巩固寡头局面

**与日本财阀、韩国财阀的类比**：都多多少少带有裙带资本主义特征。

**后续冲击**：
- 2008 年金融危机 + 油价下跌
- 2014 年油价暴跌 + 克里米亚冲突后西方制裁
- 2020 年疫情
- 2022 年俄乌冲突后更严重的新一轮制裁
