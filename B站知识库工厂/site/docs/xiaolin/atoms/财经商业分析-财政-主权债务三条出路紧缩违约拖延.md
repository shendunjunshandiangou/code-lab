# 主权债务三条出路：紧缩、违约、拖延

## 一句话定义
政府债务走投无路时只有三条路：紧缩（立竿见影伤经济、政治阻力极大）、违约/重组（信用崩塌十年难恢复）、拖着（借新还旧+印钱）——选举体制下民众只接受短期不痛苦的扩张，故「拖着」是大多数国家的共同选择。

## 详细说明
- **紧缩**：少花少借。小打小闹无用（美国政府因预算僵局多次关门，2019 年史上最长 35 天、80 多万公务员一个月没工资）；大规模紧缩对经济打击立竿见影，需要极大魄力。真正严苛紧缩的国家（欧债危机的意希西、97 危机的韩国）都不是自觉，而是借了外债、被 IMF 等债主逼着勒紧裤腰带。
- **违约/债务重组**：政府违约不像公司那样变卖清算——1902 年委内瑞拉破产时德英意真的出动军队炮轰港口讨债，如今不会武力讨债；最大后果是**信用崩塌**：融资成本飙升、货币被抛售、汇率暴跌，恢复至少十年八年。越发达的经济体越依赖信用，违约代价越大。
- **拖着**：借新债还旧债、或直接印钱花、或双管齐下。赌的是刺激砸到点上引发生产力跃升、用增长消化通胀与债务——概率极低，大概率持续恶化。明知不理性却是最普遍选择，根源在政治体制：一届届选举，民众厌恶紧缩、更受不了违约，三条路里唯有拖着是扩张性、短期最不痛苦的（参见希腊公投的教训——民意投票有多不靠谱）。
