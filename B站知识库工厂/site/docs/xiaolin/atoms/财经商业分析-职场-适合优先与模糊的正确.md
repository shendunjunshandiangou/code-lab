# 适合优先与"模糊的正确"

## 一句话定义

职业与专业选择的核心问题是"我到底适合干什么"（性格、习惯、爱好都算在内）：只有思维方式一致、真正适合的工作才能让人发光；"模糊的正确好于精确的错误"。

## 详细说明

**常见误区**：选专业只看高薪和前景，觉得"怎么吃苦都行"——工作两年后就会发现，不适合的岗位前两年还能装一装，往后自己痛苦、老板痛苦、公司也痛苦。

**决策方法**：不要纠结不出来就"走一步看一步"；至少先了解清楚有哪些选择、都是什么、怎么准备，再决定转行、努力还是躺平——想清楚再做的决定不后悔。

找到适合自己的位置会瞬间豁然开朗、直击核心。
