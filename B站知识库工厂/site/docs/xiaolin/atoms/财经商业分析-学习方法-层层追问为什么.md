# 层层追问为什么

## 一句话定义

通过不断问「为什么」带目的向下挖掘，排除无关信息，快速建立因果链式知识框架。

## 详细说明

- 例：硅谷银行 — 为何倒？→ 卖股融资 → 为何长期债风险大？→ MBS+加息+未对冲 → 为何不管理风险？→ 无 CRO
- 同样适用于商业问题（如奶茶店如何增利：收入−成本→提价/增量/拉新/复购）
- 主动搜索问题，排除被动信息流
