# 智能手机果链与国链演化

## 一句话定义

中国智能手机供应链早期随苹果（「果链」）壮大；折叠屏时代则由中国手机厂商主导铺路，供应链能力反向服务苹果，形成「国链」进化。

## 详细说明

**果链阶段**：业界常说中国智能手机供应链有苹果很大功劳，配套企业随 iPhone 等产品线成长。

**国链阶段**：折叠屏供应链发展主要靠中国手机厂商（如 OPPO 与飞凤联合研发）铺路。中国厂商先迭代技术、验证量产，供应链成熟后苹果等巨头再跟进采购——角色从「跟随果链」转向「国链定义标准、果链后续接入」。

该演化说明：高端消费电子的创新节奏与供应链主导权，正部分向中国品牌与本土精密制造企业转移。
