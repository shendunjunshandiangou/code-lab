# 庞氏骗局定义与运作机制

## 一句话定义

庞氏骗局（Ponzi scheme）用新投资人的钱偿还老投资人的本息，而非真实投资盈利；只要新资金流入大于流出，骗局就能循环维持。

## 详细说明

### 核心机制

- 承诺固定或稳定高收益（麦道夫案例：年化 10%–15%）
- **不拿资金做真实投资**，而是拆东墙补西墙
- 借新还旧：新投资人资金 → 支付老投资人收益 + 部分存入操作者账户
- 循环条件：**新人流入 > 旧人流出**

### 麦道夫案例中的规模

| 指标 | 数据 |
| --- | --- |
| 持续时间 | 约 20 年（70 年代起暗中运营） |
| 峰值规模 | 650 亿美元 |
| 知情人数 | 两只手数得过来（不到十人） |

### 与合法基金的区别

合法基金用募集资金投入真实资产，收益来自市场回报；庞氏骗局收益完全来自后续投资者本金。
