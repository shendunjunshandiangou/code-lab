# 总统敌视加息与选举周期施压

## 一句话定义
美联储与白宫的百年分歧几乎都是同一件事：美联储要加息紧缩时政府不让——因为加息抑制经济（总统政绩的关键指标）且抬高政府借债成本（战争与赤字融资的命门）；施压最猛的时段是选举年前一年、尤其连任选举年前一年。

## 详细说明
- **两大底层利益**：① 经济表现：选民用执政期经济好坏评价总统，加息 = 给自己的成绩单泼冷水；② 债务成本：利率越高政府借债越贵，所以每逢战争（二战、朝鲜战争、越战）财政部对美联储的压力格外大。
- **条件反射**：美联储一加息总统就跳脚，一降息就 happy。典型如特朗普第一任期：2018 年鲍威尔连续四次加息被骂到狗血淋头；2020 年疫情中鲍威尔紧急降至零利率 + QE，特朗普罕见夸赞"老鲍干得漂亮"。
- **反面教材·伯恩斯**：尼克松无法在法理上开除马丁，就提前宣布继任者伯恩斯"架空"马丁。伯恩斯上台后言听计从、主动降息助尼克松连任——水门事件录音曝光他致电尼克松邀功："我已把贴现率降到 4.5%""我已尽我所能帮您成为总统"。结果 1970 年代通胀失控，美联储独立性拱手让出。
- **制度反弹**：正因伯恩斯干得太烂，国会立法明确美联储**物价与就业双重目标**，要求主席每半年提交货币政策报告并出席两院听证会——制度层面加强审查。
- **中间人本质**：美联储主席夹在市场与政府之间——完全站市场会被白宫治，倒向政府则市场反噬美元。马丁（原则强硬间歇认怂）与格林斯潘（"和稀泥"、模棱两可的语言艺术、四朝元老）能久任，靠的正是这种微妙平衡。
