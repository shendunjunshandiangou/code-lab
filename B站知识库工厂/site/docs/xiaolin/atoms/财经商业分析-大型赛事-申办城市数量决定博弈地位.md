# 申办城市数量决定博弈地位

## 一句话定义

想办奥运的城市越多，IOC 越占主导（规模要求更高、分成更多）；申办者稀少时 IOC 须让步（降要求、多分利益），二者处于数十年动态博弈平衡中。

## 详细说明

**申办者多 → IOC 主导：**
- 可要求更大规模赛事
- 转播、赞助等利益分配中拿更多

**申办者少 → IOC 让步：**
- 降低场馆与规模要求
- 多分利益以挽回申办热情

**历史缩影：**
- 1970 年代惨淡后，IOC 给洛杉矶、1988 汉城、1992 巴塞罗那较优 terms；洛杉矶成功后赞助上升，88、92 较成功。
- 1990 年代后金砖等发展中国家经济走强，申办城市变多 → IOC 再度占主导，主办国成本上升。
- 近年申奥退潮（见 [财经商业分析-大型赛事-2024申办危机与奥委会让步](/xiaolin/atoms/财经商业分析-大型赛事-2024申办危机与奥委会让步.html)），IOC 转处相对劣势。
