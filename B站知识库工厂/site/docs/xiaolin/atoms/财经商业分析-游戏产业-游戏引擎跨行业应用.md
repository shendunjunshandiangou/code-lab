# 游戏引擎跨行业应用

## 一句话定义

游戏引擎虽因游戏诞生，但实时渲染能力已外溢至影视、航天、航空训练、自动驾驶等领域，随虚实融合或成多行业基础设施。

## 详细说明

| 领域 | 应用案例 |
| --- | --- |
| 影视/电视 | LED 虚拟制片：演员四周 LED 屏根据摄像头移动实时渲染，只渲染镜头可见区域；《曼达洛人》《西部世界》大量采用虚幻引擎 |
| 航天 | NASA 与 Epic 宣布用虚幻 5 创建火星资源与体验 |
| 航空训练 | 南航用腾讯游戏引擎模拟机场跑道、飞行中天气变化 |
| 自动驾驶 | 特斯拉测试自动驾驶/辅助驾驶时用游戏引擎模拟三番路况 |

**潜力判断**：
- 目前应用多在细分领域，仍被叫做「游戏引擎」
- 随虚拟世界与现实融合越来越紧密，很多领域都需要「这把铲子」
- 这也是腾讯投资 Epic、市场高估值 Unity/Epic 的重要原因
