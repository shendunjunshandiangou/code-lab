# 保险系高端养老社区模式

## 一句话定义
以泰康燕园为代表的保险公司养老社区采用「月费 + 保险/押金门槛」模式：月费 1.2 万–3 万，入住需购买约 200 多万的年金保险或缴大额押金，目标客群是来「享受」的活力老人。

## 详细说明
- **规模**：燕园（北京昌平）可容纳四五千人、已建三期，属五星级养老机构（北京约十几家）。
- **收费结构**：
  | 项目 | 标准 |
  | --- | --- |
  | 月费（活力老人，60–180 平房型） | 1.2 万–3 万/月 |
  | 月费（护理型，房间较小） | 2–3 万/月 |
  | 入住门槛 | 购买年金保险（北京约 200 多万）或大额押金 |
- **产品逻辑**：保险公司借养老社区绑定长期保单，客户用保险锁定未来入住资格——「月费+保费/押金」是保险系机构的通用模式。
- **体验设计**：双高度猫眼（轮椅友好）、随处紧急呼叫（一按保安/护士/管家三人到）、健身房器械旁设休息椅；健身房、泳池、书画、棋牌、电影院、大讲堂一应俱全。
- **客群差异**：驿站与普通养老院多为失能失智「刚需」；高端社区老人多可自理，结伴入住，共同理由是**丰富的文化娱乐生活**——「老人教老人」（95 岁教龄 4 年）如大学社团。
