# 种姓制度的经济影响

## 一句话定义
上千年的种姓制度把人按出身锁定职业（婆罗门祭司、刹帝利战士统治者……贱民只能做清洁疏浚且不得受教育），虽已法律废除但根深蒂固——至今仅 5% 婚姻跨种姓，高收入岗位存在隐形筛选，从经济学看是对劳动力配置效率的长期损耗。

## 详细说明
- 结构：四大种姓+种姓之外的「贱民」；各种姓对应固定职业、理论上不通婚。英国殖民时期甚至将其写入法律——分好级只需管住最上层，便于统治。
- 现状：独立后废除，但文化惯性极强——跨种姓交友都难，婚姻仅 5% 跨种姓；教育水平与贫富差距在种姓间依然显著。
- 政府矫正：学校招生、企业雇佣设低种姓比例指标（配额制）。
- 经济学视角：把劳动力按出身切成四份管理，天然妨碍人尽其才与流动性，是就业与效率问题的深层文化因素之一。（注：UP主声明此为个人观点，也有研究认为种姓制度影响不大。）
