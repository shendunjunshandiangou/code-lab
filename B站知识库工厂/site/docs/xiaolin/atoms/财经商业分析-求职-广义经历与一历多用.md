# 广义经历与「一历多用」

## 一句话定义
简历中的「经历」是广义的（课程项目、社团、课余相关活动都算），且同一段经历可按不同岗位命题取材、多种用途复用。

## 详细说明
- **广义经历**：不只实习/工作，还包括课程项目、课外项目、社团活动、课余自己做的相关事情。入选标准：自己真做过、面试时能对答如流、能体现真实兴趣——满足就大大方方写。
- **筛选方法**：把过往所有经历写进一个 Word 文档，再按目标岗位的关键词、关键技能逐条评估，挑最相关的进简历。
- **一历多用**：一段创业经历里可能同时包含市场调研、营销、战略分析——申商业分析岗突出调研与分析，申市场营销岗突出营销部分。经历之间可以相互转化。
- **借助外力**：不确定哪段经历相关，找做类似工作的师兄师姐看一眼——外行冥思苦想几天的问题，内行弹指一挥就能选出。
- 没有经历的根本解法仍是增加经历。
