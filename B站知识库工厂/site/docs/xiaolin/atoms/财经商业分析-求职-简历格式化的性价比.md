# 简历格式化的性价比

## 一句话定义
格式调整（formatting）是简历中性价比最高的一步：花不到 20% 的精力换取超过 80% 的第一印象效果。

## 详细说明
- 反面教材：格式一片狼藉的简历让筛选者根本不想读——「这都做不好还找什么工作」。格式如同面试前的梳洗打扮，是第一印象。
- 要点清单：
  - 该对齐的对齐、该加粗的加粗（忌通篇加粗）；
  - 字体种类不要多，**选常用字体**（导出 PDF 后对方缺字体会被替换成难看的系统字体）；
  - 字号 10–12 号；
  - 注意断句与留白，整体别太挤也别太空。
