# Dyson 轻奢定价自传播

## 一句话定义

Dyson 电吹风证明自传播不一定靠低价：主力用户女性更看重感性价值（买得爽、用得爽、能晒朋友圈更爽）而非性价比，小几千的轻奢定价反而制造了晒单动机——50 块的电吹风没人发朋友圈，几千块的收礼、晒小红书都有面子——同时维持了极高利润。

## 详细说明

- 与九号平衡车的对照：九号靠降价解锁转化率，Dyson 靠提价制造分享情绪——自传播路线取决于产品属性与目标人群的价值取向。
- 机制：轻奢价格本身成为传播素材（礼物、格调、身份感），实现"定价即营销"。
- 视频注明：不确定 Dyson 最初是否有意为之，但效果成立，已被当作自传播成功案例。
