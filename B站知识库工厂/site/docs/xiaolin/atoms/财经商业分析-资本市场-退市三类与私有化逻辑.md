# 退市三类与私有化逻辑

## 一句话定义

退市（de-listing）指股票不再能在公开市场买卖，分三类：①自愿退市（私有化）——公司为摆脱短期利益绑架、或资本 buyout 买断、或股价低于资产价值时低价回收（罕见）；②非自愿/强制退市——经营不达标、财务造假、法律要求；③"强制自愿退市"（有强制因素的自愿退市，如滴滴）。

## 详细说明

- 私有化动机：上市的隐性成本是被季报和舆论绑架，想投高风险长周期项目就把流通股收回、变私有公司，决策不必向大众汇报。
- 自愿私有化必付 premium（溢价），否则没人卖——马斯克 2018 年拟以 420 美元私有化特斯拉（时价约 350），推文次日股价涨 6%+；对散户而言自愿私有化通常是好事。
- Buyout（资本买断退市）：资本看中低估公司，买断→倒饬（降本、全球化）→增值→再上市或转手退出，是自愿退市里更常见的形式，属一级市场资本游戏。
- 低估值私有化：公司资产（如早年购置的房产）价值高于股票市值时理论上可低价回购退市，现实极少见。
- 退市≠完蛋：很多公司退市就是为了整装再上市；对一级市场资方而言，退市是武器库中的重要工具。
