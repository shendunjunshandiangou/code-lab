# 一切指标皆可期货

## 一句话定义

期货可追踪任何可量化指标——股指期货到期现金结算，无需实物交割；理论上播放量、利率、指数皆可期货化。

## 详细说明

- 标普 500 期货有 250 倍乘数，到期按指数点位清算多空盈亏
- 永续期货（加密货币）无到期，纯价格对赌
- 期货把指标金融化、可交易化，故为「金融产品之王」
