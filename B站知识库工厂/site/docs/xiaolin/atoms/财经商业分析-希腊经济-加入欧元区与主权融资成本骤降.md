# 加入欧元区与主权融资成本骤降

## 一句话定义

希腊 2001 年入区后，主权借债利率从约 18% 骤降至约 3%（与德法相当），因市场认为欧元区纪律与德法信用背书消除了违约风险。

## 详细说明

**入区动机**：
- 货币代表经济体信用；希腊信用恶化、德拉克马贬值/通胀压力大
- 换用健康欧元区的欧元可缓解通胀
- 享受自由贸易等一体化红利

**利率变化**：
- 入区前：约 **18%**
- 入区后：约 **3%**，与德国、法国相当

**市场逻辑**：
- 背后有德国、法国等大佬信用背书
- 欧元区对成员国财政有严格要求，市场认为「妥妥没风险」
- 借来的债仍须希腊自己还，但融资成本骤降

**后续行为**：希腊将债务/GDP 拉到 110%，与意大利长期飘在 100% 以上，辩称「再不发展经济就完蛋」。
