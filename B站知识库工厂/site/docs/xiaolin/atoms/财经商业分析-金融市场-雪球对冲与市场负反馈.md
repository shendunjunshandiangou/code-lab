# 雪球对冲与市场负反馈

## 一句话定义

券商发行雪球后须在期货市场对冲；未敲入时低买高卖减小波动，跌破敲入价后须大量卖出股指期货，与投资人亏损形成负反馈，加剧中证 500/1000 等指数下跌。

## 详细说明

投资人与券商收益相反；券商不对赌，靠佣金赚钱，须对冲大部分风险。

**未敲入**：股指跌 → 券商买期货；涨 → 卖。区间内低买高卖，减小波动。

**跌破敲入**：券商 Delta 急升 → **大量卖出股指期货** → 加剧下跌。

**2024 年初 A 股案例**：
- 2023 年初大涨，大量雪球在约 6500 点敲出
- 投资人于 6000–6500 点再买，敲入价对应约 4500–4875
- 今年初跌至该区间 → 大量券商同时抛售中证 500/1000 期货
- 两指数跌幅超沪深 300/上证，期货贴水扩大

雪球非大跌唯一原因，但推波助澜。历史：2015 港股雪球卖韩国、2020 疫情全球暴跌均有类似传导；投行仍持续发行因利润率高（如法外贸银行年赚上亿欧）且相当于发债融资。
