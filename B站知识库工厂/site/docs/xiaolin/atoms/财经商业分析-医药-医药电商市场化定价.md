# 医药电商市场化定价

## 一句话定义

2015 年起医药电商管制逐步放开；无门店降成本、覆盖面大带来规模效应与议价权，主攻家庭常备药与慢病药，一线城市可半小时达。

## 详细说明

- 京东大药房等相当于网上药房：无货架限制、品种多、竞争充分。
- 覆盖常备药、慢病药等非紧急需求；打通检—诊—药流程。
- 部分核心城市已支持医保支付；处方外流随互联网覆盖面扩大而增加。
