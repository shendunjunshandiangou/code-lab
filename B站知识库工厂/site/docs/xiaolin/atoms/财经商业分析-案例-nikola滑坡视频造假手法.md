# Nikola 滑坡视频造假手法

## 一句话定义

面对投资人质疑 Nikola One 迟迟不量产，Milton 2017 年前后发布「演示视频」：真车无法启动，被拖车拉到犹他州 3 度、3 公里微坡顶，靠重力滑下并做镜头处理，伪装成自主行驶——2016 年该车已在内部被放弃。

## 详细说明

### 背景

- 2017 年前后，距发布会已一年多，Nikola One **毫无进展**
- 外界质疑，Milton 需稳住投资人 → 放出「霸气」演示视频

### 造假步骤

1. 车**确为真车**（非特效），但**根本启动不了**
2. 怕车门掉下来，从里面用**胶条封住**车门
3. 拉到犹他州**鸟不拉屎的地方**
4. 利用当地**约 3 度微小斜坡**，长达 **3 公里**
5. 找更大**拖车**把 Nikola One 拖到坡顶
6. 让车**靠重力慢慢滑下**；镜头处理隐藏斜坡

### 后果

- 投资人以为车做出来了，**2018 年 C 轮继续投资**
- 视频里风光无限的 Nikola One，**2016 年内部已被放弃**
- Hindenburg 做空报告后来将此作为核心造假证据之一
