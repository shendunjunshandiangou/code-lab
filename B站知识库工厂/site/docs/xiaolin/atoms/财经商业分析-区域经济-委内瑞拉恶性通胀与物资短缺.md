# 委内瑞拉恶性通胀与物资短缺

## 一句话定义

马杜罗上台后油价下跌，进口 5 年降至 1/6；超市限号购物、武装安保、指纹验证，2017 年有人靠捡垃圾食品为生。

## 详细说明

- 查韦斯时代负实际利率（通胀 30–50% vs 贷款利率上限 29%）催生消费狂潮。
- 汽油持续补贴：1 美元加满油箱，全国排队数小时至数天；年走私损失估超 100 亿美元。
- 美国 2014 起扩大制裁，打击石油收入与融资。
