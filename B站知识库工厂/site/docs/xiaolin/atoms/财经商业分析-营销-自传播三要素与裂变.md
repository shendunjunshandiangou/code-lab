# 自传播三要素与裂变

## 一句话定义

自传播是不靠投放广告、让用户口口相传或自发传播的低成本获客方式（referral 邀请返利、拼团砍价、logo 购物袋、值得发朋友圈的产品皆是），做好要看三要素：①传播度（利益驱动已多被禁，情绪驱动——新奇/显酷/有格调——是主路径）；②正面传播比率（差评传播力可达好评的十倍甚至百倍，口碑必须极好）；③转化率（传播要落到真实购买才有收入和下一轮传播）。

## 详细说明

- 九号案例：平衡车"产品即广告"，上街就被围观问"三件套"（这是啥/在哪买/多少钱），天生传播度拉满；卡点在转化率——1 万元劝退路人，因此投研发降到 1999 元解锁裂变，营收 91 亿而营销费用仅 1.2 亿（1.3%）。
- 裂变的数学：一传二、二传三、十传百，效果爆炸式；用户先看到别人在用再种草，接受度远高于广告。
- 三要素相互制约：传播度高的产品差评也扩散得快；转化率与价格强相关，但并非所有产品都该走低价（见 Dyson 反例）。
