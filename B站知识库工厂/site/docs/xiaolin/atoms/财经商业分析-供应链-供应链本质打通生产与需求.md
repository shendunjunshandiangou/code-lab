# 供应链本质：打通生产与需求

## 一句话定义

供应链管理的本质是打通生产端与需求端之间的流转——技术和模式再复杂，解决的都是这个问题；经典教学案例是餐厅特价菜：推出"宫保鸡丁半价"等于主动把需求引导到特定菜品，从而可以精准备料、少备其他食材，用需求管理反向优化库存。

## 详细说明

- 餐厅从买菜、洗菜、切菜到端上桌就是一条微型供应链；菜单几十上百道菜意味着大量新鲜食材库存。
- 特价菜的三层作用：促销吸客（表层）→ 需求引导 → 库存管理与供应链效率（深层）。
- 衡量指标：库存周转（Inventory Turnover）——生产出的东西平均多久卖出去，本质是在考虑需求。
- 百年供应链进化的主线：流水线加速生产 → 考虑库存与需求 → 自动化/数字化/智能化，追求工人、原料厂房、信息反馈的极致效率，近年再叠加安全与可持续。
