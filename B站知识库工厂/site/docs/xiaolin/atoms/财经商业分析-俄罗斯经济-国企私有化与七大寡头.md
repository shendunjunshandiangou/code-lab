# 国企私有化与七大寡头

## 一句话定义

叶利钦休克疗法中的国企私有化，使前苏联大国企以极低价格落入极少数人手中；1996 年叶利钦与七位银行大佬秘密交易，造就掌控俄罗斯经济命脉的七大寡头。

## 详细说明

**私有化悖论**：理论上民众可入股大国企，实际上大国企以极低价格落入极少数人手中，财富迅速集中。

**1996 年秘密协定**：经济一塌糊涂、车臣战争失利、支持率跌至第四的叶利钦，秘密召集七位掌管银行界的大佬——大佬助其连任，叶利钦保护其荣华富贵。数月后叶利钦实现大逆转。

**七大寡头（Seven bankers）**：
- 俄语直译「七大银行家」
- 掌控银行、天然气、石油、媒体等经济命脉产业
- 叶利钦行事也需看其脸色
- 人选之后不断变化，但寡头操控经济、影响政治的格局长期延续
