# 多邻国 CEO 管理成长与靠谱团队

## 一句话定义
多邻国 CEO 路易斯每年接受董事会与下属共同撰写的绩效评估，其多年短板是回避冲突和想微管理；他总结的高管标准是"靠谱"——交代的事可以完全忘掉、一定会被解决，且随年龄增长他愈发看重契合度与忠诚而非纯能力。

## 详细说明
- CEO 也被考核：董事会 + 直接下属（高管团队）每年共同给他写 performance review。
- 短板一 · 回避冲突：对事直说不难（"这个项目不行"），对人极难（"你做得不行"），甚至有下属当场哭；与一个人共事约五年后才能自如直接反馈。
- 短板二 · 微管理倾向：自认"想当 micromanager"，但公司过百人后不可能；如今懂得高管在各自领域都比自己强（营销/产品/工程），充分授权效果更好——"虽然我心里还是想自己上"。
- 靠谱团队："不给我添麻烦"是最高评价——交代即忘、必被解决并回报；团队靠多年试错筛选（不合适时双方都知道，走人或辞退）。
- 用人观演进：能力是底线，年龄越大越看重 chemistry 与忠诚；了解一个人的成长经历（独生子？移民？虎爸虎妈？）比看指标更能理解他被什么激励（金钱/精进/使命）。
- 自评成功特质：极度投入（obsessed，做了十几年每天投入）+ 擅长把自己放进用户的鞋子里（每天学自己不擅长的科目：音乐、瑞典语——因为大多数用户同样不擅长）。
- 个人成长：幸福感来自解决问题，但学会了对亲人先倾听共情而非"解决问题"。
