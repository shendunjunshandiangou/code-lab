# 挪威 GPFG 透明被动投资

## 一句话定义

全球最大主权基金（~1.8 万亿美元）：72% 股票、禁止私募，官网公开全部持仓——像无法购买的全球 ETF，1998 至今年化约 6.3%。

## 详细说明

- 550 万人口，石油天然气积累（名「养老基金」实为能源基金）。
- 国会批准每年取约 3% 作财政（政府支出约 20%）。
- 个人长期合理预期收益约 4%–7%（扣管理费再打折）。
