# Alpha 零和与 Beta 时代红利

## 一句话定义
对冲基金靠策略赚 alpha——你赚就有人赔的零和博弈；价值投资把钱配置给能创造更大价值的公司，分享的是时代发展的红利（beta），是正和游戏。

## 详细说明
- **Alpha（对冲基金模式）**：靠找交易策略获取超额收益，本质是零和博弈——收益来自交易对手的亏损。
- **Beta（价值投资模式）**：把资金给到能创造更大价值的公司，让它们更好地「生钱」，投资者分享企业成长与时代红利。
- **三种财富创造方式的对照**（讲者借马斯克与巴菲特互怼引出）：
  1. 投机/策略交易获取高收益（对冲基金大佬）；
  2. 投资创造价值的公司（巴菲特：被马斯克讥为「只会 allocate capital」）；
  3. 亲自创造实业、带动技术发展（马斯克：被巴菲特讥为「炒炒货币」）。
- 讲者立场：三者没有对错，取决于个人对金钱和社会价值的认知。
- 对普通人的意义：拿不到 alpha 没关系，坚持长期参与就能拿到 beta——这正是「不懂就买大盘」的理论依据。
