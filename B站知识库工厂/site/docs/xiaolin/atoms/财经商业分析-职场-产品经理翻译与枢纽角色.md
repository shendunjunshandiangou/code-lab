# 产品经理：翻译与枢纽角色

## 一句话定义

产品经理是"翻译 + 枢纽（hub）"：把用户/运营需求翻译成程序员能听懂的话、把技术产出翻译成用户看得懂的东西；同时决策需求、连接开发与运营各端，对沟通与项目管理要求极高。

## 详细说明

**工作流**：思考交互设计（如 B 站一键三连、长按点赞三连）→ 画原型图流程图 → 找设计做 UI → 找程序员开发 → 催进度。

**适合的人**：讲者观察，做产品特别好的都是职场话术强、特别能张罗事、沟通能力极强的人——聚会上最张罗事的往往就是产品经理。

**背景要求**：多数产品岗对专业无明确要求，但必须熟悉公司业务、用户和产品，并多少懂点技术以便实现需求。应届进大厂约 15–18K，高于市场岗。
