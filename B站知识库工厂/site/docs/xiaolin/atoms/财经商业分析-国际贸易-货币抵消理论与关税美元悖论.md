# 货币抵消理论与关税美元悖论

## 一句话定义

米兰用"货币抵消理论"论证关税不引发美国通胀：他国央行让本币补偿性贬值以保出口，出口国承担大部分关税成本；但这暗含悖论——关税推升美元，而美元高估正是贸易逆差与制造业空心化的根源，与关税初衷自相矛盾。

## 详细说明

- 大规模关税两大副作用：①通胀——特朗普可不在乎，但美联储在乎：通胀→加息→政府债务成本上升→触碰特朗普核心利益（财政）；②对手国以牙还牙反制，美国出口制造业反受打击。
- 米兰花七八页论证货币抵消：美国加税→他国货币贬值→美元升值抵消涨价→无通胀。
- 悖论：美元升值 = 空心化根源。视频评价：米兰谈特里芬困境时客观理性，谈关税影响时"避重就轻、先有答案再硬凹论证"。
- 案例印证：2018-2019 关税战期间人民币等货币确有贬值对冲，但 2025 年全面关税下美元反而走弱，理论并不稳固。
