# 财阀与财团演变

## 一句话定义

明治四大财阀（三井/三菱/住友/安田）二战后被美国解散控股层，1950s 起以总裁会+相互持股重组为**财团**星系网络，Top3 为三菱/三井/住友。

## 详细说明

- **财阀**：家族控股伞状，控约 40% 国民财富；二战后伞尖撤销、资产强制抛售。
- **财团**：金曜会（三菱）、木曜会（三井）等定期开会；相互持股、互为高管。
- Top3 占国内约 60% 销售额；4–6 位伊藤忠、丸红、住友（三合）。
