# 黄氏定律与芯片持续奔跑护城河

## 一句话定义

GPU 性能约每 2 年翻 3 倍（黄氏定律），芯片业迭代极快导致产品难形成超 2 年护城河——持续研发投入与组织奔跑能力才是护城河。

## 详细说明

- **摩尔定律**：晶体管约 18 个月翻倍；CPU 已难跟上
- **黄氏定律**：GPU 约每 2 年性能翻 **3 倍+**（讲者强调是英伟达自我卷出来的结果）
- **商业含义**：今天最强芯片 2 年后必落后；3dfx 等「5 年落寞」案例常见
- **英伟达**：研发投入约营收 **25%**（互联网巨头约 10–15%）；约每 **6 个月**一代芯片；RTX 光线追踪等多次颠覆自身技术路线
- **行业结局**：小鱼吃虾米、大鱼吃小鱼，常以收购收场；互联网巨头不敢轻易入局
