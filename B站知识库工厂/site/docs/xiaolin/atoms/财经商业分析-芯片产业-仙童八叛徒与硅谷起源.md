# 仙童八叛徒与硅谷起源

## 一句话定义

1947 年贝尔实验室造出第一个晶体管（获诺奖），团队中的威廉·肖克利科研宗师但管理灾难（固执、种族歧视、多疑）；1957 年他手下八位顶尖科学家（「八叛徒」）集体出走成立**仙童半导体**——半导体人才的摇篮、硅谷的起源：十年间八位创始人纷纷再出走，1968 年诺伊斯和摩尔创立英特尔，一年后销售主管桑德斯创立 AMD——英特尔与 AMD 同根同源、缠斗至今。

## 详细说明

- 出身差距：英特尔含着金汤匙（人才资金俱全，轻松融 250 万美元、两年 IPO）；AMD 生于 70 年代半导体创业混战期，小鱼小虾多如牛毛没人投——最早投桑德斯的个人投资人竟是英特尔老大诺伊斯。
- 仙童的结局：高管与科学家的持续流失基本宣告其出局，但它孵化的公司定义了硅谷。
- 桑德斯也不弱：AMD 成立三年后（1972 年）即上市。
