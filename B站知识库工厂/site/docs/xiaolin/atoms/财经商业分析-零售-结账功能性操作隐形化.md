# 结账等功能性操作的隐形化

## 一句话定义
零售技术演进的一个趋势：扫码、结账等功能性操作的存在感会越来越弱，直至对消费者「隐形」。

## 详细说明
- 物联网时代的代表是 **Amazon Go**：拿了就走，无需收银。
- 现在 Zara、优衣库门店结账时，把衣服往筐里一放即可完成识别结算（RFID），连扫码都不用。
- 趋势判断：扫码这类功能性操作在未来零售中的存在感会持续减弱，把消费者的注意力释放给「体验」本身。
