# 医院医保限价与以药养医

## 一句话定义

医院处方药主战场与医保高度重合；医保局集采压价 + 省级平台公开议价且不得加价，使医院渠道价相对透明——与全球医药分离模式形成对比。

## 详细说明

- 医保局集合全国医院需求与药企谈判，规模效应压价。
- 医院自采须在省级采购平台公开议价，**进价即售价，不得加价**。
- 全球多数国家医药分离以避免过度开药，但通常更贵且需完善商业保险；中国正部分过渡，「以药养医」属更大体系话题。
