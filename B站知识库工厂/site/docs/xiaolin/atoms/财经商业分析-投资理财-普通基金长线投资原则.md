# 普通基金长线投资原则

## 一句话定义

普通基金因费率高昂、专为专业管理付费，**应用于长线投资而非短期炒作**——不必天天盯盘、频繁换手。

## 详细说明

- 基金设计鼓励长期持有：资金在池子里越久，平台越受益；短期卖出有惩罚性费率
- 7 天内卖出可收 **1.5%** 交易费；高频买卖等于给基金平台「捐款」
- 已交高昂管理费、托管费请专业人士管理，投资者不必自己瞎操心
- **本期最重要结论之一**：普通基金是用来做长线投资的，不是用来短期炒的
