# Insta360 影石创业史

## 一句话定义
Insta360（影石）由 91 年生的南京大学软件学院毕业生刘靖康创立：从大学连环创业到 2014 年立项便携全景相机，历经 Nano（2016）、One（2017）等产品迭代，2021 年 9 月通过上交所科创板上市委审议，创始人成为 A 股首位 90 后创始人。

## 详细说明
- **创始人背景**：刘靖康本科期间是典型技术黑客创业者——大二做随机视频交友网站（一周注册破万，覆盖近半数在校男生）、破解宿舍/情侣关系、用电话按键音解析大佬电话号码；大三大四做校园手机直播 App「V 直播」。
- **起点**：2014 年被俄罗斯 AirPano 的空中全景视频震撼，认定全景是场景分享的终极方案；当时拍全景需多台相机拼圈+电脑长时间处理，门槛极高，遂立项便携全景相机。大四未毕业即敲定 IDG 六十多万美元天使轮。
- **至暗时刻**：软件团队做硬件，demo 搞了大半年且质量差（出门带螺丝刀/电烙铁/热熔胶「维修三件套」，曾在过夜高铁上通宵抢修到凌晨四点）；工厂嫌订单小、团队嫩不接单，全员上产线拧螺丝，UI 设计师因此辞职。
- **关键节点**：2015 年底想出「插在手机上的全景相机」（边拍边传边剪）→ Nano；2016-01 CES 参展获经销商关注+新融资，团队破百人；2016-07 Nano 上线首月收入超 1000 万人民币，随后因市场教育不足业绩下滑；靠创意内容+KOL 营销打开海外，2017 年海外营收破亿；2017-08 切入运动相机赛道推出 One 一炮走红。
- **产品线**：Nano（2016）→ One（2017）→ One X（2018）→ Go 拇指相机（2019）→ One R 模块化相机（2020，与徕卡联合设计）。营收从 2017 年 1 亿多增至 2021 年预估十几亿人民币（GoPro 同期年销售额六七十亿人民币仍居第一）。
- **方法论总结**：产品上在成熟市场找突破口、不硬刚搞差异化（Think Bold）；营销上不打低价促销，用创意影像制造震撼体验。
