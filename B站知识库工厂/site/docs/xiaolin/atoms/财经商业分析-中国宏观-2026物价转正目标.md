# 2026 物价转正目标

## 一句话定义

2025 年 CPI 目标 2% 未达标（接近零、PPI -2.6%）；2026 年明确**「推动价格总水平由负转正」**，物价成为与政府高度重视的新重点。

## 详细说明

| 指标 | 衡量对象 |
| --- | --- |
| CPI | 居民消费端（衣食住行） |
| PPI | 工业品出厂/生产端 |
| GDP 平减指数 | 最终商品与服务整体 |

「价格总水平」市场普遍认为更可能指 GDP 平减指数，报告未明确。

调控路径：需求侧促销费促投资 + 供给侧反内卷深入整治。
