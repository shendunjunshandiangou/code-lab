# AI捆绑投资与系统性风险

## 一句话定义

AI 巨头的股权投资本质是「钱 + 算力订单」捆绑：大模型公司融到巨额资金的同时，承诺出去万亿美元级未来订单，降低单家风险却放大全行业系统性风险。

## 详细说明

**与普通投资的区别**：
- 英伟达、亚马逊、谷歌的投资多是「承诺最多 X 亿美元」
- 条件：未来须用投资方算力、达到指标才继续投钱
- OpenAI 已粗略承诺出去超过 **万亿美元**订单
- 不能只看「融资 1100 亿」——融了钱又立刻欠出去万亿美元级订单，把自己与别人死死绑住

**风险结构**：
- **单家公司**：股份合作捆绑降低自身风险（多艘船绑一起抗风浪）
- **整个行业**：系统性风险增大——「火烧赤壁」
  - 算力需求不达预期
  - 电力跟不上
  - 某家公司爆雷
- **乐观情形**：若需求持续爆发、算力旺盛，则绑定各方皆大欢喜

**OpenAI、Anthropic 的处境**：表面众星捧月、融资光鲜，实则融资同时让渡大量未来算力与合作承诺，并不「舒服」。
