# 普通基金费率结构

## 一句话定义

普通基金（共同基金/LOF）费用名目多、合计高：管理费 + 托管费约 **2%/年**，另有申购费、赎回费及短期惩罚性交易费，无论经理表现好坏都要交。

## 详细说明

| 费用 | 说明 |
| --- | --- |
| 申购费 | 买入时缴纳 |
| 管理费 | 基金经理管钱，年化约 **0.25%–2%**，常见约 **1.5%** |
| 托管费 | 资金存托管银行（如招商银行），约 **0.25%/年**，基金公司代收转交 |
| 赎回费 | 卖出时；鼓励长期持有 |

- 管理费 + 托管费按年收取，合计约 **2%/年**
- 美股语境：**expense ratio** 含 management fee 及 12b-1 等所有给基金的费用
- 短期高频操作有惩罚：如 7 天内卖出收 **1.5%** 交易费
- 投资者须掂量：经理是否值这 2% 年费？
