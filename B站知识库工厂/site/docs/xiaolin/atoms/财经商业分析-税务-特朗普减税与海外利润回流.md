# 特朗普减税与海外利润回流

## 一句话定义

特朗普 2017 年减税法案把企业所得税从 35% 降到 21%，海外存量利润按 8%–15.5% 的优惠税率汇回，同时规定此后海外利润**不论是否汇回都要交税**——一句话「给你减税，但钱必须拿回来」；苹果随即把 2500 多亿美元现金转回美国、一口气缴税 380 亿美元，全美超万亿美元现金回流、带来超千亿美元税收。

## 详细说明

- 该法案直接封死了「海外利润不汇回就不缴税」的递延口子，是美国对跨国避税架构的收网。
- 回流的钱去向：苹果近十年回购自家股票约 **6000 亿美元**——足够买下中国任何一家上市公司，这些回购金额单独算一家公司就是全球第十大上市公司；按一年回购近 1000 亿、市值 3 万亿计，相当于每年 3% 以上直接回报给投资人，高于很多银行存款利率。
- 与巴菲特重仓苹果的关联：巴菲特的价值投资需要「持续挤奶的奶牛公司」（可口可乐、美国银行式持续回报），苹果开启大规模回购后开始「出奶」，伯克希尔三千多亿盘子近一半重仓苹果一只股票。
- 讲者的长线观察：苹果看似躲美国税收，但这些钱大部分最终还是回到美国政府兜里——90 年代美国政府内部资料就显示监管层料定避税天堂的钱早晚回流。
