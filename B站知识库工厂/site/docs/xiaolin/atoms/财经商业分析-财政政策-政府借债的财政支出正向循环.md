# 政府借债的财政支出正向循环

## 一句话定义

政府借债增支可启动双重良性循环：财政支出刺激就业与税收，经济好转又降低借债成本、扩大支出空间——这是理解希腊宁可打破赤字红线也要大量借债的核心逻辑。

## 详细说明

政府借债主要为了花钱（财政支出），而非仅为贪腐（希腊当时确有贪腐，但借债逻辑更深）：

**循环一：财政支出 → 税收**
```text
财政支出↑ → 刺激经济、增加就业 → 收入↑ → 纳税↑ → 政府更有钱 → 继续增支
```

**循环二：信心 → 融资成本**
```text
经济更好 → 市场信心↑ → 借债利率↓ → 借更多钱 → 财政支出再↑
```

两个循环叠加，解释了希腊为何打破债务/GDP 与赤字红线仍要借债——希望进入「良性螺旋」。

**前提与风险**：需市场持续信任、经济向好、能「借新还旧」。2008 年信用收紧后该逻辑逆转为恶性循环——也是希腊危机的理论支点。
