# 科创板国家战略定位

## 一句话定义

科创板要求企业阐述符合国家战略（如十四五数字化、VR/AR）；硬科技须证明专利落实到产品，符合者审批可极快（如中芯国际约 1 个月）。

## 详细说明

- 上交所规定「面向国家重大需求」；不能仅靠买专利包装
- 影石例：逐项列技术来源、应用产品、研发负责人
- 2019–今申请 1158 家，成功约 647；制造/软件信息占超 2/3
