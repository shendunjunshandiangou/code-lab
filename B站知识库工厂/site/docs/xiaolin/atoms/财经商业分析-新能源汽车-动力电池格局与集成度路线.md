# 动力电池格局与集成度路线

## 一句话定义

动力电池市场规模 2014 年约 85 亿美元→2023 年四五百亿美元，格局已成型（宁德时代约 40% + 比亚迪 + LG 能源合计约 70%）；后来者提升电池性能有两条思路——电芯材料（能量密度）与提高集成度（CTC/CTB），新玩家更可行的是后者。

## 详细说明

- 高集中度 = 高进入门槛。行业成型后，续航（200 公里→1000 公里）与 800 伏快充等体验仍在快速迭代。
- 思路一：电芯材料提能量密度——特斯拉大圆柱电池、宁德时代/比亚迪固态电池路线；需要深厚材料研发功底。
- 思路二：提高体积效率/集成度——电池是电芯打包成的，从买电池包进化到买电芯自己加工（打包成土字形/T 字形/凹字形），或 CTC（Cell to Chassis 电芯直连底盘）、CTB（Cell to Body 电芯直连车身）。
- 小米案例：与宁德时代合作研发 CTB 一体化 800 伏电池包，集成率 77.8%，电池更薄让座舱更大；电芯与泄压阀倒置，极端情况能量向下释放提高安全性——合作研发 + 差异化创新，避开自己不擅长的材料战场。
