# 名义 GDP 与实际 GDP 的汇率陷阱

## 一句话定义

经济增长率用本国货币经通胀调整计算；GDP 国际排名用名义 GDP（当年汇率折美元）——汇率变动可导致「德国超越日本成第三大经济体」类误导性头条。

## 详细说明

**案例（2023–2024）**：
- 新闻：德国抢走日本世界第三大经济体称号
- 德国 2023 增速倒数第二（仅次于沙特），日本真实 GDP（日元计）其实在涨
- 关键：日元大幅贬值 → 名义 GDP 换美元后暴跌 → 被德国反超
- 结论：二者谁都难长期稳坐第三；头条「德国很强、日本很弱」有迷惑性

**讽刺性建议**：日本动用外汇储备买日元保面子——说明汇率对名义排名影响巨大。
