# 大红筹架构

## 一句话定义

境内企业绕道境外上市的标准组合拳：在开曼注册上市主体 → 控股一个香港公司 → 香港公司控股境内经营主体——开曼壳在纳斯达克等境外市场挂牌，变相实现境内公司上市融资；香港层的作用是把大陆对外资分红的 10% 税率降到 5%。

## 详细说明

- 动因：A 股上市有利润门槛，烧钱期的互联网/科技公司只能去境外融资；而大陆企业因监管限制很难直接在纳斯达克挂牌，需要海外壳做上市主体。
- 注册地选开曼：零税率避税天堂，京东、阿里、B 站的上市主体都是开曼公司。
- 底层逻辑：无论股权结构图多复杂，本质都是"境内经营主体通过各种方式串联到境外主体、由境外主体上市"。
- 与小红筹的区别：大红筹用直接控股连接境内外；受外资准入限制的行业用协议控制（VIE）。
