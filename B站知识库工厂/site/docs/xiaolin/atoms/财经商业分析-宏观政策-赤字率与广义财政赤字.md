# 赤字率与广义财政赤字

## 一句话定义

名义赤字率只计一般债；超长期特别国债、专项债等专款专用不计入，中国此类债务较多，**广义赤字可超 8%**，远高于名义 4%。

## 详细说明

| 类型 | 是否计入赤字率 | 2026 规模（报告） |
| --- | --- | --- |
| 一般债（赤字） | 是 | 赤字率 4%，5.89 万亿 |
| 超长期特别国债 | 否 | 1.3 万亿 |
| 特别国债 | 否 | 3000 亿 |
| 地方专项债 | 否 | 4.4 万亿 |

- 2025 赤字率从 3% 提至 4%（史上最高），2026 延续 4%——措辞「继续实施更加积极」重点在「继续」。
- 横向对比：美国名义赤字约 6%、英法超 4%；中国名义 4% 但广义超 8%。
- 专款专用债偿还不依赖一般预算税收收入。
