# 失去货币主权的希腊困境

## 一句话定义

加入欧元区后希腊丧失独立印钞权，信用危机时无法像拥有本币的国家那样通过增发货币应急，只能眼睁睁看着融资断裂与经济下滑。

## 详细说明

信用萎靡时，拥有货币主权的国家通常可增发本币刺激经济（有通胀风险，但能解燃眉之急）。

希腊境况完全不同：
- 此前执意加入欧元区，占尽低息融资等好处
- **印钞权归欧元区**，希腊说了不算
- 危机中：钱印不了、债借不着，只能干着急

这是欧元区「一国货币政策、多国财政」体制的典型代价——危机国失去汇率与货币两大调节工具。
