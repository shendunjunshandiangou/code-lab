# 人脉背后的信息与认知收益

## 一句话定义
人脉带来 offer 的表面原因是「认识人」，隐性的真正收益是社交过程中获取的行业信息和认知提升——它们才是让你准备得更好的敲门砖。

## 详细说明
- 案例：一位在纽约上学的学生想进投行，通过校友圈和领英把某大投行一个组的十来个人全部约见聊了一遍，面试时相当于把熟人见了一遍，自然拿到 offer。
- 表面看是因为「熟」，但他整个招聘季喝了不下一两百次咖啡，每次对话获得的行业信息和职场理解，才是真正帮他拿到 offer 的关键——没有人会因为一次咖啡就录用你，录用是因为你准备得更好。
- 类比：大学新生通过学长学姐了解选专业、备考等信息，如同拿到「通关秘籍」；职场信息更不透明、潜规则更多，找资深者点拨的价值更大。
- 这类信息过来人大多愿意分享：一顿饭很难立刻变成利益关系，但信息拿到了就是自己的，在社交早期效果立竿见影。
