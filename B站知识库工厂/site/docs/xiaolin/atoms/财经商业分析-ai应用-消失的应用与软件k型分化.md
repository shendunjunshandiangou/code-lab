# 消失的应用与软件K型分化

## 一句话定义

Coding 被 AI 代替后，软件壁垒从开发能力转向「解决什么问题、积累什么数据」；生态或 K 型分化——大厂做大众应用，长尾极度个性化且更长。

## 详细说明

**「消失的软件」（Disposable Software）**：
- 用户可随时手搓应用、用完即弃，迭代频率远高于传统 App
- 开发能力不再是核心壁垒
- 关键是解决了什么问题、积累了什么数据

**K 型分化预测**（马竞）：
- 几个大厂做大家都用的应用
- 其余极度个性化
- **长尾会更长**：以前长尾应用难被搜到、难变现；在灵光圈等平台，任何想法都可低成本变成应用与内容

**时间判断**：不会太久——基模迭代快、中国移动端使用深度与时间优势。

**最大卡点**：让所有 C 端用户体验 Web Coding / Wish Coding 的魅力。
