# 路口经济与贸易总量 337%

## 一句话定义
衡量贸易依赖度要看贸易总量（出口+进口）/GDP 而非净出口——新加坡该比例高达 337%（1960 年以来从未低于 200%），像一个繁忙的交通路口：对外贸易才是头号经济重镇，内需反而是次要指标。

## 详细说明
- 指标辨析：A 国进出口各 1 万亿、B 国零贸易，两者净出口都是零，但贸易依赖度天差地别——应看贸易总量/GDP。
- 对比：美国 27%，中国 38%（已算很高），新加坡 337%——贸易总量是 GDP 的三倍多。
- 路口比喻：各国像带围栏的小区（美中是大小区），新加坡像交通要塞路口——地理位置重要、路宽、"过路费"低、有交警维持通畅；从政策到银行对外企完全敞开，与多数国家保护本土产业的做法相反。
- 引申：正因经济以跨境流动为主，新加坡的货币政策以汇率而非利率为锚。
