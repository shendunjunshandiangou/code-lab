# 上市光环效应与再融资成本

## 一句话定义
上市的另外两大好处：名声（「上市公司」对 2C 消费者和商业伙伴的光环与信誉背书）和降低再融资成本（成熟度与信息披露背书使后续贷款、发股发债更便宜）。

## 详细说明
- **光环效应**：对面向消费者的 2C 公司价值巨大——消费者一看「上市公司」就觉得是有信誉的大公司；对个人头衔亦然（「上市公司 CEO」）。
- **降低再融资成本**：公司发展需要持续融资（银行贷款、二级市场再发股发债），上市公司的成熟度、信誉度、信息披露构成背书，借债成本大大降低。极端例子：JPMorgan、沃尔玛等高评级公司借债利率与美国国债相当，十年期不到 2%。
