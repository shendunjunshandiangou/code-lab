# GENIUS 法案与稳定币监管框架

## 一句话定义

美国 GENIUS 法案（参议院已通过）及各国类似立法要求：发行牌照、100% 高质量储备、破产优先偿付、反洗钱与信息披露——合规成币圈信用关键。

## 详细说明

**GENIUS 核心**：发行需牌照；储备为短期国债等且不可再抵押；风险管控与信息披露。

**合规意义**：监管模糊时主流不敢涉足；立法明确红线是利好。Circle、Coinbase 合规抢占美国市场；OKX 交罚款、关链上钱包换牌照；美股上市=合规证书（币圈缺信用不缺钱）。

**风险案例**：USDT 信息披露不足、注册在维京群岛，可能因不合规被赶出美国市场。
