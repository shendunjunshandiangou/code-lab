# 中国双积分政策

## 一句话定义

2018 年起实施的汽车行业「双积分」制度，对新能源与燃油车分别计分，覆盖面比美国单一碳积分更广，允许燃油车通过降油耗获得积分。

## 详细说明

- **新能源积分**：生产/销售新能源车获得
- **油耗积分**：燃油车提高技术、降低油耗也可得分
- **案例**：理想汽车 2021 年靠卖分收入两亿多元
- **性质**：过渡性供给端政策，随电动化普及市场将逐步萎缩
