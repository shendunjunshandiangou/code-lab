# 微软 Bing 整合与谷歌 Code Red

## 一句话定义

ChatGPT 威胁谷歌 93% 搜索垄断与 60% 广告收入；微软借 Bing+ChatGPT 先发，谷歌仓促 Bard 发布会失误致市值蒸发约 1000 亿美元。

## 详细说明

| 公司 | 动机 | 关键动作 |
| --- | --- | --- |
| 微软 | Bing 仅 ~3% 份额 | 2.07 整合 ChatGPT（Copilot for Web），须 Edge；充分准备的一小时发布会 |
| 谷歌 | 搜索 ~93%，广告 ~60% 收入 | 内测 LaMDA/Bard 强但不愿自毁摇钱树；2.08 仓促发布会：40 分钟前段跑题、忘带手机、Bard 广告事实错误 |

- **成本**：ChatGPT 日运行约 **100 万**美元；单次查询能耗约为传统搜索 **10–100 倍**
- **谷歌 Code Red**：集中全公司力量应对 AI 赛道
- **讲者评价**：AI 战争第一仗，谷歌惨败但仍是强者，后续待定
