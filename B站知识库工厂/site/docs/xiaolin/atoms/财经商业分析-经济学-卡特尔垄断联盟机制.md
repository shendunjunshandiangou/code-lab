# 卡特尔垄断联盟机制

## 一句话定义
卡特尔（Cartel）是为避免过度竞争损害整体利益而结成的垄断联盟；石油史上从七姐妹到 OPEC 再到 OPEC+，随着市场越开放、成员约束力越弱，垄断力一次次衰减。

## 详细说明
- 定义：联手控制一种资源（产量/价格/地盘）的组织。对经济资源配置不利，但对成员利益极大——监管者和消费者想方设法阻止，商家想方设法组建。
- 刚需稀缺资源（如石油）形成卡特尔的利益极高，故石油市场百年间多次形成不同卡特尔。
- **约束力递减规律**：七姐妹是公司联盟，可用事先商定的股份约束成员的责任与利益；OPEC/OPEC+ 是主权国家联盟，成员在生产、经营、财务上完全独立，无法互相监管、无法律约束，一国偷偷增产别国无可奈何。
- 结果：OPEC+ 对油价已很难产生实质性影响，沙特只能用「玩命增产、杀敌一千自损八百」惩罚不守约者；很多经济学家认为 OPEC 不能算（有效的）卡特尔。
