# 心理战 PsyOps 与选举咨询

## 一句话定义
PsyOps（Psychological Operations，心理战）是通过心理学手段影响目标人群行为的军事技术；由于「改变心理→改变行为」的逻辑同样适用于选民，这套技术被商业公司移植到选举咨询，成为政治营销的灰色武器。

## 详细说明
- 军事起源：SCL 公司成立于 1990 年，做战争心理咨询，曾帮美英军方参与阿富汗、伊拉克战争——例如劝服穆斯林青少年不加入基地组织、劝村庄投降。
- 商业迁移逻辑：选举与战场同构——都是「影响大量人群的决策」；且各国选举资金规模大，是心理战技术最大的民用变现场景。
- 实践范围：SCL 以政治咨询身份介入阿根廷、特立尼达和多巴哥、泰国、印度、马来西亚、意大利、肯尼亚等多国大选。早期（无大数据时代）靠传统心理策略；2013 年后与 Facebook 数据结合，成立子公司 Cambridge Analytica，升级为「数据驱动的心理战」。
- 关键升级：大数据解决了心理战的瞄准问题——过去只能对整个人群设计单一策略，有了个体画像后可对每类人定制信息（见 micro-targeting）。
