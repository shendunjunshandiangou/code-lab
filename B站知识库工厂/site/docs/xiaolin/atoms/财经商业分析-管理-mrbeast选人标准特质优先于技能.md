# MrBeast 选人标准：特质优先于技能

## 一句话定义

MrBeast 管理 400 名员工的选人哲学：**宁要一个具备正确特质、愿意干十年的人来培养，也不要一个进门就很懂技术但特质不对的人**——三年后前者会远远强过后者；他点名的特质：①**可教练性**（工作难度一直上升，不学习技能线迟早被难度线甩开）；②**极度痴迷**（不给钱也会在业余时间干这个、研究新相机——因为热爱而非想当好员工）；③**长期承诺**（最糟的是投入多年员工却辞职）；④**高自主性**（「我讨厌需要告诉别人去做事」）；⑤**极端责任感**（分包出去也要每天跟进承包商进度，而非一周后才过问）。

## 详细说明

- 背景：没有导师、没有有经验的父亲，从十几岁到二十出头「搞砸了无数次」才想清楚要找什么人。
- 关键教训：早期招了很多传统媒体、好莱坞背景的人，他们想把一切系统化、把好莱坞搬到 YouTube——行不通；**做没人做过的事（单视频 2 亿播放）需要非常规的人**，而非成熟行业的流程专家。
- 早年入职的隐含门槛：把整个未来赌在一个 YouTube 频道上很吓人，来的人必须真信这件事——反向筛选出高信念密度团队。
