# 中投与汇金体系

## 一句话定义

中国主权基金两大派系：外管局系（华安等四朵金花，~1.09 万亿，极神秘）与中投 CIC（~1.33 万亿，中央汇金控股国有金融命脉，2025 年证金+AMC 无偿划转扩容）。

## 详细说明

- 外储约 3.2 万亿美元；美债从 1.2 万亿降至约 7500 亿。
- 汇金持工行/建行/中行等，资产约 7.76 万亿人民币。
- 中投海外 2014–2023 十年收益率 6.57%；48% 另类资产（含一带一路等）。
