# 铂力特钛合金3D打印消费级突破

## 一句话定义

铂力特与 OPPO 联合将航天级 3D 金属打印下放到折叠屏铰链，实现 0.15mm 厚、0.02mm 精度的钛合金量产件，是增材制造首次规模化进入 3C 核心结构件。

## 详细说明

**铂力特背景**：西安企业，源自西工大研究团队，为长征火箭、嫦娥六号、空客飞机等供应零部件，十余年积累使其位居全球 3D 金属打印头部。

**攻关过程**：

- OPPO 目标厚度 0.15mm，铂力特原量产最薄约 0.3mm，目标激进；双方决定联合攻关。
- 钛合金强度高、重量轻，但熔点高、注射成型平整度差，3D 打印极难。
- 需从零深度配合：粉末粒度、温度、风场流动等逐项仿真调整；铂力特在 3C 项目首次实验室预演，动用国家级项目技术设备。
- 最终精度 0.02mm，成品 0.15mm，结构复杂件薄如纸。

**产业意义**：经验与标准可支撑后续更天马行空的产品；证明高精尖增材技术可在成本可控前提下找到消费级场景（折叠屏铰链），为缓解手机行业材料同质化价格战提供工艺多元化思路。
