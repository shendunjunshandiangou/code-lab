# GDP支出法公式CIGNX

## 一句话定义

支出法 GDP = 消费(C) + 投资(I) + 政府购买(G) + 净出口(NX)，是各国最常用核算方式；其中「投资」指企业生产经营投入，不含买股票买基金。

## 详细说明

**公式**：
```text
GDP = C + I + G + NX
```

**关键辨析**：公式中的 **I（投资）** ≠ 日常金融投资
- ❌ 买股票、黄金、基金、房产投机
- ✅ 企业买厂房、设备等生产性投入

「股市大量资金涌入导致 GDP 上涨」——不成立。股票是股权转让，与生产无关；券商手续费、投行上市服务费才计入。

宏观分析常用此公式拆解：消费、投资、政府、进出口哪块拉动或拖累增长。
