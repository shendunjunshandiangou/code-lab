# 金融三角色与基金嵌套

## 一句话定义
再庞大的金融系统追本溯源只有三类角色——融资者（需要钱做业务的）、中间商（牵线/服务的）、投资人（出钱的，分债权与股权）；基金是替不专业的投资人"汇集资金、按特定需求投向特定产品"的角色，基金间层层嵌套（A 投 B、B 投 C）是金融显得复杂的重要原因，每层按各自承担的风险分走相应收益（"扒皮"），本身也是资源配置的合理环节。

## 详细说明
- **三角色**：融资者（小林式的企业）、中间商（银行、投行、交易所……）、投资人（存款人=债权投资人、股民=股权投资人）。现实中主体既可以是个人也可以是机构，且一家公司可同时扮演多角色。
- **基金**：很笼统的概念——私募基金、风投基金、养老基金、对冲基金皆是。价值在于专业化代客配置：普通人不知道该买哪家公司，交给基金。
- **嵌套与"扒皮"**：你投 A 基金 → A 投 B 基金 → B 投 C 基金 → C 投项目；项目赚的钱被逐层分成。这不是贬义——每个环节按承担的风险获得相应收益，是风险与收益逐层匹配的机制。
- **认知要点**：金融看似复杂庞大，是因为无数个角色互相交织 + 产品层层包装；拆到底层永远是"钱从有闲钱的人流向能创造价值的项目"。
