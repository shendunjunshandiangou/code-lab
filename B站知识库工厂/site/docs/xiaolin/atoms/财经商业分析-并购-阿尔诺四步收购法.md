# 阿尔诺四步收购法

## 一句话定义

阿尔诺数十年收购 70 余个奢侈品品牌，手法高度重复：备资盯目标、危机时出手、瓦解股东内讧、收购后清洗管理层并重定位包装——「三年一小战、十年一大战」。

## 详细说明

### 四步框架

1. **找准目标，备好资金**
2. **伺机而动**——对方危机、内部矛盾或股价暴跌时大打出手
3. **瓦解核心股东，内讧渔翁得利**
4. **收购成功后清洗管理层，重新定位包装，拉回股价给投资者信心**

### 节奏

- 玩法已是司马昭之心，行业普遍提防
- 典型节奏：1989 LVMH、1999 Gucci、2009 Hermès、2019 Tiffany——约十年一大战

### 局限

- 并非常胜：Gucci 毒丸、Hermès 家族铁壁均告失败
- 有钱的企业很多，四步法是必要条件而非充分条件
