# ImageNet AlexNet 与 GPU 算力转折

## 一句话定义

2012 年 AlexNet 用神经网络 + 英伟达 GPU 在 ImageNet 竞赛错误率 16.4%，领先第二名约 10 个百分点，证明 GPU 算力使神经网络从理论走向实用。

## 详细说明

- **瓶颈**：神经网络理论早有，长期缺算力无法落地
- **2012 ImageNet**：第二名至第四名错误率约 26–29%；AlexNet **16.4%**，碾压优势
- **意义**：学界震动；英伟达 all in AI 显卡优化与软件栈
- **后续**：ChatGPT 训练用 1 万+ 张 A100；A100/H100 成算力衡量标准
