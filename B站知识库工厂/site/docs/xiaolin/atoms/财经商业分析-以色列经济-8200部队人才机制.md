# 8200 部队人才机制

## 一句话定义

以色列军方情报分支 Unit 8200 以 18–21 岁顶尖年轻人为主力、服役数年即退役的"流水席"制度，配套全国高中生测试筛选（甚至提前特训 14–15 岁天才），实质成为国家级人才筛选与培训机器，是高科技创业的一大支柱。

## 详细说明

**机制**：几千人规模；在全国高中生中通过年度/半年度测试选出最顶尖者（须为犹太人）；对 14–15 岁有潜力者提前进行三年编程/黑客基础特训。工作类似"黑客"——在全球互联网搜集情报，服役期间身份极度保密。

**外溢效应**：退役校友上万人——智商顶尖、受过军事化管理培训、群体凝聚力强，20 出头退伍即成为行业精英，大批投身创业。知名成果：被谷歌 13 亿美元收购的 Waze、纳斯达克上市建站公司 Wix 的创始人都出自 8200。

**定位**：8200 不只是情报部门，而是"军队即孵化器"的人才机制——军事投入反哺高科技产业。
