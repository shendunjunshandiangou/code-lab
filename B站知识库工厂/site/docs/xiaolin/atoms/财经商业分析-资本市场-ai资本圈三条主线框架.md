# AI资本圈三条主线框架

## 一句话定义

美国 AI 资本关系可归纳为三条主线：英伟达投资全行业、所有人投资 OA 两家换订单、捆绑式合作形成「一条船上的人」——同级竞争、不同级捆绑。

## 详细说明

**牌桌角色**（小 Lin 主观比喻）：
| 角色 | 特征 |
| --- | --- |
| OpenAI、Anthropic | 天才年轻创业者，势头猛、风险大 |
| 谷歌、微软、亚马逊、Meta | 老钱巨头，培养+投资拉拢年轻人 |
| 英伟达 | 鼎盛中年大佬，统筹整桌、把蛋糕做大 |
| 博通 | 靠谱供应商新贵 |
| 甲骨文、AMD、英特尔 | 费劲上桌，争取更大蛋糕 |
| SpaceX | 另一套路数，未来可能强力竞争 |

**三条主线**：

1. **英伟达投资所有人**：扶持行业 + 控股，锁定 GPU 订单
2. **所有人投资 OpenAI 与 Anthropic**：换取算力订单；2025 年两者计划上市，融资更活跃，大厂脚踩多船，一捆绑成同船人
3. **捆绑本质**：股权投资附带算力承诺，形成交叉持股与订单网络

**简单规律**：同级之间竞争，不同级之间捆绑。半年前资本版图还与今天不同，许多箭头是近两个月才加上。
