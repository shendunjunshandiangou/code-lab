# SpaceX史上最大IPO与市销率估值

## 一句话定义

SpaceX 完成人类史上最大 IPO，以接近 100 倍市销率空降美股，融 750 亿美元仅稀释 4.2% 股份，估值高度依赖「马斯克溢价」而非当前利润。

## 详细说明

**IPO 规模**：
- 人类史上最大 IPO；此前美股纪录为 12 年前阿里巴巴
- 融资 **750 亿美元**，稀释仅 **4.2%**
- 对比：阿里巴巴 IPO 融 250 亿、稀释约 15%

**估值争议**：
- 上市即全美市值第七，超过特斯拉
- 估值约 **1 万多亿**；沃尔玛约 7000 亿营收，Meta/博通/特斯拉等营收 600 亿+，SpaceX 营收不到 **200 亿**
- SpaceX 亏损，无法算市盈率（PE）

**市销率（PS = 市值 ÷ 销售额）**：

| 公司 | 市销率（约） |
| --- | --- |
| 苹果 | 10 倍 |
| 特斯拉 | 15 倍 |
| 英伟达 | 25 倍 |
| SpaceX | **接近 100 倍**（讲者称 IPO 空降为美股最高档） |

**高估值的好处**：
- 以极低成本融资
- 成为选择此时上市的重要原因之一

**马斯克溢价**：
- 估值夸张很大程度来自马斯克多年商业履历带来的信任
- 同样业务两家公司，CEO 是马斯克 vs 普通人，市场定价不同
- 买 SpaceX 的人买的是对马斯克这个人的信任
