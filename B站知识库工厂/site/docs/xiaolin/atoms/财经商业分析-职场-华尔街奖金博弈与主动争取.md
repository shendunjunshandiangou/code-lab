# 华尔街奖金博弈与主动争取

## 一句话定义
华尔街奖金由直属老板主导决定，发放当天（communication day）是一场双向心理博弈：员工要管理表情、传递「it could be better」的信号，因为「会哭的孩子有奶吃」——同等优秀时，老板会让不争不抢的人吃亏，把资源倾斜给不满足就可能离职的人。

## 详细说明
- 决定机制：老板组集体讨论，但直属老板起决定作用——这是「必须与老板搞好关系」的制度根源。
- communication day 的博弈细节：老板逐个约谈告知奖金与来年工资；员工若不满要当场表达并问改进方向；若满意则要做好**表情管理**——不能让老板看出高兴（否则他觉得给多了），而老板也深知这一点（「太后眼里明镜似的」）；回座位时全组都在观察你的表情猜结果。
- 会哭的孩子有奶吃：老板视角下，两个同等优秀的下属，一个给多少都 OK、一个天天追问目标与回报、不给够就走——理性选择是让省心的那个吃点亏。因此在实打实的奖金面前应主动争取、该哭哭该闹闹。
- 边界条件：老板不傻，日常共事谁干得好一清二楚——**活干得不好，再哭再闹都没用**；争取的前提是业绩过硬。
