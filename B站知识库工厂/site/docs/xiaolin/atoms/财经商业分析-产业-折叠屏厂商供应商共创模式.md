# 折叠屏厂商供应商共创模式

## 一句话定义

折叠屏时代厂商从组装转向工程师/产品经理角色，牵头共创；供应商（如三星显示）选客户看技术牵引与方向，非仅订单金额——「乙方中的甲方」。

## 详细说明

- 三星显示 ≠ 三星手机；可供给 OPPO/华为/摩托罗拉。
- 中国产业集群 + 工程师红利支撑精密共创；库克称中国制造优势在集群与工程师规模。
