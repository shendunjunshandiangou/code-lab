# 日内瓦自由港与艺术品避税

## 一句话定义

日内瓦自由港是免税 + 严格保密的「隐形艺术博物馆」，约 120 万件藏品、估值超 1000 亿美元，成为艺术品投资、避税与洗钱通道。

## 详细说明

- **机制**：过境商品免关税，只收储藏费；空壳公司匿名存放，实物可不移动却多次转手。
- **费用**：普通尺寸画作约 1000 美元/月储藏费；小储藏间约 1.2 万美元/月。
- **2005 年后**：新进口文物须声明来源；2009 年前存量仍大量匿名。
- **与保密战略关系**：20 世纪保密吸引全球财富的核心环节之一；21 世纪随国际压力收紧。
