# 中国制造升级四阶段

## 一句话定义

以空调业为缩影的中国制造升级路径：①被看中市场、被投资（外资合资进入）→ ②承担生产重任的制造工厂（代工与产能扩张）→ ③供应链技术完善、能主动思考甚至制定行业标准的研发基地 → ④积极对外投资、具备全球知名度与行业整合能力。

## 详细说明

- 每个阶段的关键跃迁条件：①→② 低成本劳动力 + 产业链外溢；②→③ 利润再投入研发 + 质量认证接轨（3C）；③→④ 资本积累 + 危机窗口的海外并购。
- 空调业的落点：年产 2 亿多台、占全球 80%，从生产制造大国到研发大国、品牌大国。
- 该框架可平移分析其他行业（手机、汽车、光伏等）所处阶段与下一步瓶颈。
