# 国际收支与双顺差升值

## 一句话定义
国际收支（BOP）衡量一国资金总体是净流入还是净流出（贸易、投资、借贷、赠予全算）；顺差 = 外汇市场对本币需求大于供给 = 升值压力——所谓货币"被高估/低估"通常就指国际收支余额指向的压力方向；中国入世后长期"双顺差"是最典型的非均衡案例。

## 详细说明
- **三个概念的包含关系**：国际收支 = 经常账户 + 资本账户；经常账户包含贸易平衡。多数国家（美国除外）左右资金流动的主要是贸易，所以三个词常被混用。
- **均衡与非均衡**：多数国家经常账户与资本账户方向相反、可互相抵消（如美国：经常账户逆差 + 资本账户顺差，对汇率影响不确定）；**中国 2001 入世后长期双顺差**——贸易赚钱 + 外资涌入同时发生，结果是 2014 年前人民币持续单边升值、央行外汇储备飙升超 3 万亿美元。双顺差、升值、外储激增三者相互关联而非巧合。
- 反向案例：澳大利亚长期经常账户逆差，与其国债六成以上由外国套息资本持有、需持续对外付息有关。
