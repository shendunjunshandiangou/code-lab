# 珠宝与投资需求的跷跷板平衡

## 一句话定义
黄金的珠宝需求与投资需求高度负相关：经济好时买首饰、投资需求弱；萧条时不买首饰但避险投资需求上升——理论上互相抵消稳定金价，实际金价仍剧烈波动，推手是投资需求。

## 详细说明
- **机制**：经济好 → 有钱有信心 → 买首饰金表消费，钱同时流向股市/房产/创业，黄金投资需求弱；经济差 → 首饰消费缩水，但避险保值需求上来。
- **实证**：2020 年疫情初期，珠宝需求骤降、投资需求猛涨（主要增长点是黄金 ETF）。
- **意义**：这对冲关系解释了为什么实体黄金总交易量波动不大；但金价的剧烈波动来自投资需求内部——尤其「爱」与「投机」两个隐藏需求的激发（见相关原子）。
