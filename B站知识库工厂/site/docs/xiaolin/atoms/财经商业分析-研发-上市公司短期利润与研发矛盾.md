# 上市公司短期利润与研发矛盾

## 一句话定义

美股上市公司每分钱需追求股价与资本效率最大化，与需要长期投入的硬科技研发存在天然张力。

## 详细说明

- 类比超市门口摆高毛利小件最大化空间效率
- 研发到产品链路漫长；波士顿动力为典型
- 工业界反馈链可更短（华为案例：2 年商业化 vs 高校约 10 年）
