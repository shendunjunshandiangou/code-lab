# SPE 空壳实体转移负债

## 一句话定义
安然 CFO 法斯托设立大量 SPE（Special Purpose Entity，特殊目的实体）空壳公司：SPE 以安然股票为抵押借债融资，再用借来的钱购买安然资产，通过记账手段把安然的负债和亏损转出表外——账面负债降低、所有者权益和利润被抬高；由于抵押品是安然股票，这套玩法**只在股价上涨时成立**。

## 详细说明
- 操作链条：设 SPE → SPE 抵押安然股票融资 → 用融资买安然资产/承接安然亏损 → 安然报表上负债变股权、亏损消失。
- 庞氏特征：股价涨→SPE 抵押品充足→可继续转移亏损→财报好看→股价再涨（左脚踩右脚）；股价一旦下跌，抵押品缩水，整个结构瞬间失效——2001 年股价半年跌 40% 后玩法终结，直接引爆崩盘。
- CFO 角色异化：正常 CFO 做预算、出报表；法斯托是「在线救火员」——哪笔投资亏了四千万，就被叫去「捣饬捣饬弄没了」。
- 教训：表外实体+自家股票抵押 = 把公司偿付能力与股价绑定的死亡结构；分析财报时需警惕大量关联 SPE 与表外安排。
