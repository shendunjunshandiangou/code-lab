# Auto-GPT 自主 Agent 雏形

## 一句话定义

Auto-GPT 开源项目接入 GPT-4，结合编程、搜索、长期记忆，可自设目标、自提问自迭代执行任务——Agent 方向早期探索。

## 详细说明

- **与 ChatGPT 区别**：不只给答案，而是接工具链自主运行
- **流程**：用户给目标 → 模型自问「该怎么做」→ 自行搜索/执行/优化
- **讲者定位**：半玩票性质，但展示 Agent 想象力
- **趋势**：从聊天/生图走向**应用层渗透**（搜索、办公、金融、教育、电商、编程 Copilot 等）
