# 关税博弈囚徒困境与以牙还牙

## 一句话定义

贸易博弈中两国若单次决策，各自最优均为贸易保护，但整体最差；重复博弈下「以牙还牙」与 WTO 等利益共同体可化解关税困境，推动长期自由贸易。

## 详细说明

**囚徒困境模型**：A/B 国各可选开放或保护；单方保护总更优 → 均衡为都保护，但整体收益低于都开放。

**化解一：以牙还牙（Tit for Tat）**
- 先开放，警告对方若保护则立刻反击
- 英国 18–19 世纪单边降关税，美德继续保护并反超——英国未还击是没落重要因素

**化解二：利益共同体**
- WTO（166 国、98%+ 贸易）、欧盟、USMCA
- WTO 仲裁 + 报复配额：美欧飞机补贴各获 75 亿/40 亿美元报复额度；美加木材争端自 1982 年

**谈判筹码**：关税「利己不一定，损人一定」——特朗普常用威胁（退出 WHO、退出 NAFTA、加征关税）迫使重谈（USMCA）。
