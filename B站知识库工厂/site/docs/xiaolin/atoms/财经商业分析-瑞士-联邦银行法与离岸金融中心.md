# 瑞士银行保密法与离岸金融中心

## 一句话定义

1934 年瑞士《联邦银行法》将银行保密写入联邦法，吸引全球高保密需求客户，使瑞士成为跨境私人财富离岸中心。

## 详细说明

法案核心：银行不得向任何人（含美国政府、联合国、瑞士政府）透露客户信息，除非客户同意或刑事申诉。

效果：二战后瑞士银行业一飞冲天；2000 年初全球约 **1/4** 跨境私人财富集中在瑞士。客户中不乏逃税、洗钱等灰色资金（二战时纳粹、希特勒本人亦在瑞银存金）。

2014 年瑞士签署美国 **FATCA**，须向美国提供美国纳税人账户信息，打破传统保密机制。
