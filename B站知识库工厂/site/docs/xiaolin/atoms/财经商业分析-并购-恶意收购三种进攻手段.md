# 恶意收购的三种进攻手段

## 一句话定义
董事会不同意时，收购方强制拿下上市公司的三条路径：偷偷买入（creeping tender offer）、代理权斗争（proxy fight）、公开邀约收购（tender offer），可排列组合使用。

## 详细说明
| 手段 | 逻辑 | 难点 |
| --- | --- | --- |
| 偷偷买入 creeping tender offer | 绕过董事会，在公开市场/向大股东悄悄买到 51% 投票权，然后改组董事会 | 很多人不愿卖；一旦暴露会遭防守反击 |
| 代理权斗争 proxy fight（智取董事会） | 不必买到 51%，把半数以上董事会席位变成自己人即可换血掌权；多一个席位就多一分话语权 | 极度考验政治手腕（Carl Icahn 最擅长） |
| 邀约收购 tender offer | 跳过董事会向全体股东公开溢价收购 | 表面公开公正，实际靠「后端压榨」形成胁迫（见相关原子） |

- 前提：恶意收购只对股份制/上市公司可行——若目标被某人 100% 控股且不卖，出多少钱都没辙。
- 恶意收购（hostile takeover）翻译成「敌意收购」更确切：本质就是双方没谈妥。
- 友好收购（双方谈妥成交）才是大多数情况。
