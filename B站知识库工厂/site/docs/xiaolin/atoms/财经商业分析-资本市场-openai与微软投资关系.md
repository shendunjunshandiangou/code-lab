# OpenAI与微软投资关系

## 一句话定义

微软是 OpenAI 最早、最成功的战略投资人之一，以 130 亿美元获约 27% 股权，投资回报约 17 倍，奠定了 AI 资本圈「大厂押注独立大模型」的范式。

## 详细说明

**微软投资**：
- 前前后后共投 **130 亿美元**
- 占股约 **27%**
- OpenAI 估值约 8500 亿美元时，微软股权估值约 **2290 亿美元**
- 投资回报约 **17 倍**

**对比软银**：孙正义投 OpenAI 超 **600 亿美元**（约为微软四五倍），只得约 **13%** 股权。为凑这笔钱，清仓手上 **58 亿美元英伟达股票**；孙正义称是「哭着把英伟达卖掉的」。

**格局含义**：
- 早期 OpenAI 独一档，微软押宝极成功
- 亚马逊等后来者觉得再投 OpenAI 没意思（微软股份太高），转扶持 Anthropic
- 微软没有非常出圈的自研大模型，通过入股 OpenAI 留在中心牌桌，但也逐渐被边缘化，并与 Anthropic 保持暧昧

**英伟达对 OpenAI**：签 300 亿美元战略合作意向书（本可最高 1000 亿，因 OpenAI 估值太高而缩水），黄仁勋自认「上车晚了」。
