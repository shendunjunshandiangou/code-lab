# 操纵比赛 Match Fixing

## 一句话定义

博彩利润极高激励操纵赛果；贿赂门将和裁判最有效（成功率约 70%–80%）。

## 详细说明

- Match fixing = 赛前「固定」结果
- 案例：新加坡 Perumal 操纵 80–100 场；2025 年土耳其 371/571 名裁判涉嫌；板球教练 Woolmer 疑被灭口（传闻）
- 虽违法，回报率陡峭仍有人铤而走险
