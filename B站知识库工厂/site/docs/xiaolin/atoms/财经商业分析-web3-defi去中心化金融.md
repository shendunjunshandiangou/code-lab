# DeFi 去中心化金融

## 一句话定义
DeFi（Decentralized Finance）是币圈交易规模发展到一定程度后的制度创新：不要银行、券商、清算所等中介，把现有金融体系的业务（借贷、保险、清算、衍生品、杠杆交易）搬到区块链上，用智能合约自动执行，为币圈提供货币流动性。

## 详细说明
- **为何火爆**：围绕 Web3 做全新应用一炮打响难度大，而利润点集中在币圈；币圈商业格局又已基本成形、自己造币难有颠覆——做"币圈的金融衍生层"、用现成金融玩法挖掘存量潜力，成为资本热衷的方向。
- **关键基础设施**：稳定币——解决加密货币波动过大（比特币年化波动 80-90%）的问题，让 Web3 世界可以"持有美元"。
- 本质是金融功能的复刻：金融的三角色（融资者/中间商/投资人）在链上重演，只是中介被代码替代。
