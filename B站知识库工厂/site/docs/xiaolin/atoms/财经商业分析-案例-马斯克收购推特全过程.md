# 马斯克收购推特全过程

## 一句话定义

马斯克 2022 年以 440 亿美元收购推特历经毒丸计划、毁约诉讼、突然履约、大规模裁员与 CEO 公投辞职闹剧，被小Lin评为年度最一波三折并购案。

## 详细说明

### 收购时间线

| 时间 | 事件 |
| --- | --- |
| 2022-04-04 | 披露已持股 9.2%（29 亿美元）成第一大股东，推特股价单日涨 27% |
| 4 月上旬 | 推特邀其入董事会（条件：持股≤14.9%），马斯克拒绝 |
| 2022-04-15 | 宣布以每股 54.2 美元（较 4/1 收盘溢价 38%）「一口价」全盘私有化，总价 440 亿美元 |
| 4 月中旬 | 推特启动一年期毒丸计划（15% 触发线）争取调查时间；一周后董事会确认资金真实、溢价足够，全票接受收购——恶意收购转友好收购 |
| 7 月 | 马斯克起诉推特，指未按约定处理机器人账号 |
| 10-17 | 宣布不告了、继续收购 |
| 10 月下旬 | 正式入驻，成为拥有者与 CEO |

### 入主后动荡

- 推订阅会员制、为缩减成本裁员超 **3/4**
- 发现裁多了又往回捞人
- **12 月 19 日**：发起投票「是否应卸任推特 CEO」，1750 万人参与，**57.5%** 选 Yes
- 两天后称若找到「愿意当 CEO 的蠢蛋」就辞职
- 马斯克形容推特是「引擎着火、即将坠落的飞机」

### 评选理由

同期还有微软收购动视暴雪（FTC 反垄断诉讼）、博通收购 VMware（欧盟反垄断），但马斯克收购推特因反复变卦、440 亿美元如「闹着玩」被评为最离奇曲折。
