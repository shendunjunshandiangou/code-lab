# 属地属人原则与双重征税

## 一句话定义

跨国收入面临两套征税逻辑：**属地原则**（在我地盘产生的收入都要交税，几乎所有国家采用）与**属人原则**（我的居民在哪赚的钱都要交税，大部分大国采用）——两头都收会造成双重征税抑制跨境投资，因此各国普遍签双边税收协定互相让渡。

## 详细说明

- 示例：A 国居民老王去 B 国赚 1 万，两国各收 20%，双重征税后到手仅 6400——没人愿意跨境投资。
- 双边协定的常见安排：两边各少收一点、或只按属地征税等（实际税法远比简化版复杂）。
- 这套框架是理解一切避税操作的起点：避税 = 压低所在地税率（迁册/移居）或压低赚钱地税率（转移利润归属地，尤其是难以界定产生地的无形资产收入）。
