# 碳积分与存款准备金率类比

## 一句话定义

碳积分在车企间的买卖机制，与商业银行间通过隔夜拆借调节准备金余缺、提高资金使用效率，在逻辑上异曲同工。

## 详细说明

- 央行要求商业银行保留一定**存款准备金率**（中国约不到 10%）
- 缺资金的银行与有闲置资金的银行通过**隔夜拆借市场**互借，利率调节流向
- 碳积分 likewise：有的车企积分富余、有的不达标，通过**积分买卖**让资源流向最需要的企业
- 二者都是行政配额 + 市场调剂的组合
