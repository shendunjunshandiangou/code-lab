# 新加坡三只主权基金分工

## 一句话定义

新加坡金管局（后卫稳汇率）、GIC（中场分散 1/3×3）、淡马锡（前锋全股权私募）——收益一半交政府（占政府收入约 1/5），一半滚雪球（NIR）。

## 详细说明

- 规模约 6200 亿 / 8010 亿 / 2880 亿美元。
- 经常账户顺差 GDP 15%+，双顺差积累外储后分拆管理。
- 《新加坡为什么这么有钱》补充：GIC + 淡马锡合计约 1.2 万亿美元（人均可分 22 万美元）；与中东、挪威"卖石油"的主权基金不同，这些钱是靠贸易与投资自己赚的。2000 年前主要扶持本土经济，2002 年起投向全球，现约 70% 投资在海外；均为政府 100% 控股但按私营企业方式运营。
- 风格对比：**淡马锡**激进，约 4000 多亿美元基本全是股权投资（持股腾讯、阿里、美团、Visa、BlackRock 等，管理规模比巴菲特还大）；**GIC** 稳健，教科书式分散配置（各类股票、债券、抗通胀债、房地产、私募）。
- 反哺财政：每年投资回报一半资助政府（新加坡财政收入最大来源、约占 1/5），一半滚回组合；央行与政府完全融合、毫不避讳，靠政府清廉与执行力保证机制不被滥用。
