# 云游戏市场格局

## 一句话定义

云游戏尚不成熟（2022 年约 24 亿美元、占产业 1% 几），表面王者是英伟达 GeForce Now，但基础设施多为 AWS/Azure，GPU 多为英伟达——多层「背后赢家」结构。

## 详细说明

**参与者**：
| 角色 | 业务/地位 |
| --- | --- |
| 微软 | Xbox Game Pass Ultimate、Azure 为索尼 PS5 云游戏提供商 |
| 亚马逊 | Luna；AWS 为云业务王者 |
| 索尼 | PS5 云游戏，背后用 Azure |
| 英伟达 | GeForce Now，占云游戏收益超一半，订阅用户超千万，领先微软等 |

**产业链嵌套**：
- 英伟达 GeForce Now 领先，但背后云用 **AWS**
- 云游戏服务商不是 AWS 就是 **Azure**
- 两家云的 GPU 又多为 **英伟达**
- 英伟达是「背后的背后的大赢家」

**英伟达路径**：1993 年起专注显卡，显卡收入至今占 40%+；推动算力优化成为 AI 巨头，市值超万亿。
