# 游戏机三巨头与纵向整合

## 一句话定义

索尼、任天堂、微软构成传统游戏机三巨头；行业从「开发—发行—平台」分立，演变为中下游平台凭借资本优势吞并上游开发商的纵向整合格局。

## 详细说明

**早期结构（二三十年前）**：
- 开发商开发游戏
- 发行商发行
- 平台分发
- 三块相对独立，井水不犯河水

**整合动因**：
- 上游开发成本越来越高、周期变长，难以独立生存
- 中下游发行商与平台靠近用户，有资本优势
- 逐步吞并上游开发商 → 纵向整合

**三巨头**：索尼、任天堂、微软。全球历史最畅销游戏机前十名基本被索尼与任天堂包揽（如 Game Boy、PS2）。

**索尼生态**：围绕 PS 构建强生态圈，以各类独家内容巩固老大地位；硬件卖得好的同时，收入大头来自生态（卖游戏与会员），游戏机硬件仅占很小一部分。
