# 银行的系统性风险与强周期

## 一句话定义

银行既是资源配置中介又决定货币量，是经济体的"红绿灯"式枢纽；叠加强周期性（经济好时放贷赚钱、差时坏账激增），一旦出事连锁反应极强，因此是最需要政府监管和帮助的行业。

## 详细说明

**枢纽比喻**：一个行业连接的玩家越广越深，出问题的连锁反应越强——银行坏了不是"一盏路灯坏了"，而是"路口红绿灯坏了"，交通彻底混乱。

**强周期**：看美国历年银行倒闭数量，基本能对上历次大衰退。

**监管如修树**：怎样最大程度发挥银行作用、又最好地控制风险是技术活；各国"修剪风格"不同，长出形态各异的银行体系，银行模式又反过来影响国家经济模式——最典型的两棵树是中国（由紧放开、政策主导）和美国（由松收紧、市场导向）；日本则围绕财团、每个财团中心有一家大银行输血。
