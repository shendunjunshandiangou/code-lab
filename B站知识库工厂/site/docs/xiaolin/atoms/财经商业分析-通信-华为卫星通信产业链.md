# 华为卫星通信产业链

## 一句话定义

5G 受限后华为布局 6G 时代卫星通信；Mate 50/60 卫星短信/电话带动芯片订单从几万→几百万台，降本并带动全行业跟进。

## 详细说明

- 6G 重点或从提速转向覆盖率改善；竞品含摩托罗拉、SpaceX、苹果 Globalstar。
- Mate 60 Pro 可在无基站区域（雪山/沙漠）通话；规模效应摊薄卫星通话费用。
