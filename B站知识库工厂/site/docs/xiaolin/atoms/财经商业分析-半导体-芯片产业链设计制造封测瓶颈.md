# 芯片产业链设计制造封测瓶颈

## 一句话定义

芯片含设计、制造、封测三大环节数千道工序；国内设计与制造不成熟，EDA/IP 依赖美国，制裁相当于「学霸没笔」。

## 详细说明

- **设计**：EDA 工具必须，三大主流厂商来自美国；IP 如 ARM 授权制裁后无法更新。
- **制造**：2020 第三轮制裁后台积电等无法代工华为芯片。
- **封测**：相对国内条件较好。
