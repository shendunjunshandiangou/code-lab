# 罗斯福沙特石油换安全协定

## 一句话定义
1945 年罗斯福亲赴沙特达成口头协议：沙特给美国商人石油优先开采权，美国提供武器、军事保护和黄金——奠定美沙同盟，沙特至今仍是全球最大军火制造国美国的头号客户。

## 详细说明
- 背景：1938 年美国商人在沙特发现当时已知最大石油区，美国主张「中东归英国，沙特是我发现的归我」。
- 协定内容：优先开采权 ↔ 武器 + 军事保护 + 大把黄金。此后美国在沙特近乎建立「国中国」，自采自卖，按月支付武器与钱。
- 1953 年伊朗政变后，美国控制住中东两大石油国（沙特、伊朗），以其为中东支柱：提供武器控制局势、制衡苏联，并确保油价稳定供给。
- 美国后来还转与沙特、科威特等订立五五分成协定——比英国压榨伊朗的模式温和，产油国坐享一半分成故较听话。
- 长尾影响：美国在各时期以各种理由向各国卖武器（先伊朗、后伊拉克），中东遍地武器成为后续战乱的伏笔。
