# 中投与中央汇金体系

## 一句话定义

中投 CIC 约 1.33 万亿美元（不计入外汇储备）；中央汇金持股国有金融命脉（工行建行等），2025 年证金与 AMC 无偿划转股份至汇金。

## 详细说明

- 中投海外+国际：约 1/3 股票、16% 固收、48% 另类；2014–2023 海外收益率 6.57%
- 中央汇金总资产约 7.76 万亿人民币，是中投体量主体
- 07 年成立初期投黑石、摩根士丹利遇金融危机非议
