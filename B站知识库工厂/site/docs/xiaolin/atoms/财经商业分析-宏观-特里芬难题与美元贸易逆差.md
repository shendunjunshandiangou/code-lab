# 特里芬难题与美元贸易逆差

## 一句话定义

美元作世界储备货币 → 全球资本经金融账户流入美国 → 美元升值 → 半世纪持续贸易逆差；整体 BOP 近平衡（经常账户逆差+金融账户顺差）。

## 详细说明

- 贸易差额=储蓄-投资 为恒等式，非因果；广场协议 1985 曾短期缩小逆差。
- 仅关税难根本缓解，若金融账户持续净流入。
