# 瑞郎避险货币与负利率

## 一句话定义

瑞郎是真·避险货币（非日元式套息交易驱动）；动荡时资本涌入导致升值打击出口，瑞士央行曾设欧元下限、零利率、负利率 -0.75%，2015 年突然取消下限致瑞郎暴涨 15%。

## 详细说明

**机制**：经济坚挺+政治稳定+中立 → 全球资本涌入；经常账户顺差约万亿美元（全球第三）。

**政策震荡**：
- 2011 年：欧元兑瑞郎下限 1.2 + 利率降至零
- 2014 年：负利率 -0.75%
- 2015 年 1 月：取消下限，瑞郎暴涨 15%，多家基金爆仓

**与日元对比**：日元避险多因套息交易平仓与资本回流；瑞郎负利率下做空风险大，套息交易少。

**央行上市**：可买股票但无投票权，市值约 3.5 亿瑞郎，更像永续债券；约 1/4 外汇储备买美股。
