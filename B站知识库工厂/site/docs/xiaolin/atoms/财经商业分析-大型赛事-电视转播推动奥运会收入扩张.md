# 电视转播推动奥运会收入扩张

## 一句话定义

电视转播技术从局部到全球覆盖，使奥运观众规模与转播费从百万美元级暴涨至数亿美元，成为奥运商业化的关键转折。

## 详细说明

| 届次 | 转播里程碑 |
| --- | --- |
| 1936 柏林 | 首次电视转播，约 5 万观众（柏林周边） |
| 1948 伦敦 | 约 50 万观众 |
| 1960 罗马 | 首次全球电视转播 |

- 1960 年起夏季奥运转播费从约 120 万美元涨至 1984 年数亿美元量级。
- 技术进步使「盘子」越来越大，主办国与 IOC 开始争夺转播收益分配。
