# 新加坡双顺差与外汇储备积累

## 一句话定义
新加坡央行的巨额外汇储备来自"贸易顺差 + 经常账户顺差"的双重积累：企业出口赚来的美元到央行换成新币，加上持续大于流出的外资流入——钱不是印的，是实打实赚来的。

## 详细说明
- 贸易顺差：出口持续大于进口；企业把赚来的美元换成新币，美元沉淀在央行。
- 经常账户顺差（"钱的顺差"）：资本流入持续大于流出，除贸易顺差外，很大一块是外部投资流入。理论上即使贸易收支平衡，只要外资持续流入国家也会变有钱——代价是必须持续保持吸引力（1997 亚洲金融危机正是资本突然撤离导致的）。
- 吸引力清单：低税率、汇率稳定、友好营商环境、英语、高教育水平与顶尖人才、政治稳定——跨国公司在亚洲设点很难绕开新加坡（香港条件类似，容易经商指数全球第三紧随其后；国际投行的亚洲总部除日本外基本在港新二选一或都要）。
- FDI 数据（2022）：1408 亿美元，人口不到 600 万的国家排全球第三（仅次美中）；结构上金融占超一半、工业已不多；来源以美国、日本、英国为主，对外投资最大对象是中国大陆，与荷兰互投关系密切。
- 用途：外储既是汇率调控的弹药，多余部分交给 GIC、淡马锡投资增值。
