# 美联储慢加快降节奏

## 一句话定义

美联储降息风格「慢加快降」：加息慢让市场消化；降息快避免「等更低利率」推迟消费；首步幅度传递路径预期但不可一次过猛。

## 详细说明

- 2024.9 降息 50bp 开启周期；期货预期 1 年内 5.25%→~2.75%。
- 不能一次砍 200bp——避免「Fed 急了」引发恐慌。
- 就业数据下修（80 万）+ 通胀下行 → Jackson Hole 明确转向。
