# 结构化思维与知识树

## 一句话定义

学习新领域的核心是先在脑中建立知识框架（「树」），再填充细节；有框架则理解、表达、增量信息都更高效。

## 详细说明

- 繁杂信息需分类与关联，而非逐片记忆
- 例：宏观经济 = GDP/CPI/就业 → 政策 → 金融市场
- 先框架后细节，比抠单点提升更大
- 框架可简单、线状，对自己有用即可
