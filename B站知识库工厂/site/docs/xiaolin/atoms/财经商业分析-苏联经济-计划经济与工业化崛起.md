# 苏联计划经济与工业化崛起

## 一句话定义

斯大林时期苏联实行中央计划经济，在方向明确时调动效率极高，三个五年计划使苏联从落后农业国迅速变为工业大国。

## 详细说明

**计划经济机制**：经济资源从分配、生产到部分消费均按计划执行。物资紧缺时中央一纸命令即可调拨；发展重工业时一纸命令即可动员全国建厂，无需等待市场自我调节。

**适用场景**：方向非常明确或需要大规模整体改革时，计划经济极为高效。

**成果**：
- 1928 年后斯大林前三个五年计划效果显著
- 1920–1940 年苏联人均 GDP 翻了三倍多
- 同期 1929 年美国正经历大萧条，西方普遍遭殃
- 二战后国际地位进一步提升，又经历一段高速增长
