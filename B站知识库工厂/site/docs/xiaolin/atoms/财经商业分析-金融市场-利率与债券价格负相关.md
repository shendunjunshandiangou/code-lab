# 利率与债券价格负相关

## 一句话定义

相同期限的债券，利率与价格铁打负相关——利率跌则债券价格涨，利率涨则债券价格跌。

## 详细说明

长期利率由债券价格反推，关系来自计算恒等式。

**理解方式**：利率涨了，政府得以更高利息发新债 → 原来低息旧债吸引力下降 → 需求减少 → 价格下降。

- **相同期限**：负相关，铁打不动
- **不同期限**：关系没那么直接；长期利率可看成预期未来若干短期利率的组合

美国中期利率跟着涨，一方面因美联储加隔夜利率，另一方面因市场预期其将坚定加息抗通胀。
