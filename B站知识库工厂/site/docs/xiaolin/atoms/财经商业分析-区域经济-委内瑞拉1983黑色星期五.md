# 委内瑞拉 1983 黑色星期五

## 一句话定义

1983.2.18 玻利瓦尔从 1:4.3 贬值至 1:7.5 并实施外汇管制；进口依赖致物价近乎翻倍，拉开长期危机序幕。

## 详细说明

- 背景：1980 年代油价暴跌 + 美国加息 + 1982 墨西哥债务危机。
- 1989 佩雷斯激进自由化（大调整）因缺乏法制环境失败；1994 银行业危机，半数银行倒闭。
