# 德国救希腊两难与救人即救己

## 一句话定义
危机中最有底气的德国面对希腊只有死局三选一：不救则希腊违约——留在区内摧毁市场对欧元区的信任，踢出去则开创先例、多米诺骨牌推倒整个欧元区；何况希腊的债主主要是德法机构——救希腊就是救自己。

## 详细说明
- 德国的地位：欧元区主心骨；信用收紧时资金流向安全地，德国资金反而更充足。
- 三难推演：①不救→希腊必违约；②违约后留在欧元区→「违约还能留着」重创欧元区信用；③踢出欧元区→今天踢希腊、明天踢意西葡，全踢完欧元区荡然无存。结论：只能救。
- 救人即救己：希腊的债权人很大程度上是德法的银行与机构，希腊违约德法直接受损。
- 救助的憋屈与强硬：欧元区不能像美国那样印钱刺激（不能让全区为希腊陪绑），德国只能自掏腰包凑钱，同时强硬要求希腊减支紧缩、「按德国人的风格来」——尽管希腊此前把财务标准当耳旁风、还靠高盛造假混入欧元区。
- 一般化启示：货币联盟中「大哥」的隐性担保在危机时刻会变成显性账单；被市场认定的兜底预期没有退出选项。
