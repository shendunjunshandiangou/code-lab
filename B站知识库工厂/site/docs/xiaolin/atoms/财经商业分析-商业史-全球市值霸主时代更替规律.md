# 全球市值霸主时代更替规律

## 一句话定义
过去 55 年全球市值最高公司基本每 5 到 10 年更替一次，每个霸主都是对应时代经济科技浪潮的缩影：登顶有偶然性，但时代背景下必然会有这样一家公司冒出来。

## 详细说明

| 时期 | 霸主 | 背后的时代浪潮 |
| --- | --- | --- |
| 约1967–1986 | IBM | 大型计算机崛起 |
| 1987–1991 | NTT（日本电信电话） | 日本股市泡沫 |
| 约1993–1998 | 通用电气 GE | 多元化管理（无明确时代浪潮） |
| 1998–2003 | 微软（与GE轮替） | 个人电脑与软件 |
| 约2005–2011 | 埃克森美孚 | 石油与油价长牛 |
| 约2011至今 | 苹果 | 智能手机 |
| 2019/2022 | 沙特阿美（间歇登顶） | 能源价格与美联储货币政策博弈 |

解读这一榜单的两个注意点：

1. **市值最高 ≠ 最牛 ≠ 最赚钱**，只代表二级市场最受追捧；股价受该国股票估值环境影响很大，股市泡沫会让市值虚高（典型如 1990 年前后全球市值前十有七家日本公司）。
2. 很多大公司（尤其各国大型国企）未上市，不在榜单内。
