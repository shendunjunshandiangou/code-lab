# 灌溉系统比喻与资金流向股市

## 一句话定义
央行像灌溉系统的水阀：只能决定放多少水，管不了水往哪流——宽松资金大部分流进股市、房地产等「蓄水湖」而非实体经济「稻田」；这是「Don't fight the Fed」的底层逻辑。

## 详细说明
- **机制**：经济不景气时，金融机构拿到便宜钱不愿放贷、更愿投资（美国主要是买股票）；上市公司低成本融资后也倾向回购自家股票——无论降息还是 QE，大部分资本最终流向股市。
- **比喻**：央行想浇稻田（实体经济），但只能开上游水阀；一小部分水到稻田，大部分流进蓄水湖（股市、房地产、海外资产）。这是全球央行的共同困境：能决定印多少钱，不能决定钱花在哪。
- **推论**：美联储目标是物价与就业，从没说要让股市涨，但结果是一刺激股市就狂涨——「炒股最重要的是看美联储，不是看财报」；华尔街名言 **Don't fight the Fed**。
- 华尔街反向施压：「QE 不能停，这药不能停」。
