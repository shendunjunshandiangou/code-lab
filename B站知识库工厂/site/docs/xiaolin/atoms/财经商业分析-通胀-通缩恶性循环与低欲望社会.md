# 通缩恶性循环与低欲望社会

## 一句话定义
通缩下持币即赚钱，个体最优策略是「非必要不花钱」+躺平不奋斗，导致需求与生产双萎缩的恶性循环；价格便宜不是问题，全社会不工作、不生产、不消费的低欲望状态才是深层问题。

## 详细说明
- 机制：同样 100 块明年买得更多→藏钱床底财富自动增值→不努力赚钱、尽量不花钱→总需求萎缩→企业减产降价→进一步强化通缩预期。
- 典型案例：日本 90 年代泡沫破裂后陷入通缩与信用萎靡，30 年间央行怎么刺激、政府怎么借债，需求就是起不来（叠加文化与老龄化因素）。
- 预期的粘性：一旦人们形成通缩预期，通缩会越陷越深（与通胀预期自我实现同理）。
