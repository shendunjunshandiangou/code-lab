# 产业政策组合拳：硅基仿生案例

## 一句话定义

国家通过**顶层设计 + 科研基金 + 税收优惠 + 审批绿色通道 + 医保集采**等多维政策辐射，集中力量扶持重点产业——硅基仿生动态血糖仪打破进口垄断为典型。

## 详细说明

十四五将高端医疗器械列为重点，深圳硅基仿生获：

- 科技部重点研发项目资金（三项技术）
- 所得税 25% → 15%
- 创新绿色通道：审批 2–3 年 → 1 年拿医疗器械认证
- 纳入医保集采，提升国产份额

成果：自研传感器、医疗芯片、AI 算法；2021 年第一代获批，现第三代并出口 100+ 国家/地区。
