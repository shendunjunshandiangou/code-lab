# 简历命题作文思维

## 一句话定义
简历不是罗列光辉经历的个人传记，而是针对特定岗位的命题作文——目标是论证「我就是具备你想要的经历、知识、技能的候选人」；因此找不同工作需要不同版本的简历。

## 详细说明
- **误区**：把所有觉得光彩的经历（竞赛奖、长跑第一、好人好事）全写上。HR 只会觉得「全面发展真棒」，但与岗位无关就直接错失机会。
- **原理**：不同工作需要的技能不同（金融→建模分析、程序员→coding、数据岗→数据分析），HR 挑简历只找相关信息。
- **拿到命题的方法**：
  1. 仔细读目标岗位的职位描述（JD）与所需知识技能，对照自身经历把自己描述成「那个人」；
  2. 询问从事目标工作的师兄师姐「这个岗位需要什么样的人」。
- 这是简历四步法（明确目标→筛选经历→包装经历→格式化）的第一步，决定后续所有取舍。
