# 瑞士银行业保密制度兴衰

## 一句话定义

1934 年银行法第 47 条奠定全球保密标杆，吸引 1/4 跨境资产；2009 年后向美欧交客户信息，保密名存实亡，金融业占 GDP 从 11.2% 降至 8.9%，2023 年瑞银收购遭挤兑的瑞信。

## 详细说明

**巅峰**：全球约 1/4 跨境资产在瑞士；46% 全球亿万富翁为 UBS 客户；低税+保密+影视文化（007、华尔街之狼等）。

**转折**：1997 梅利揭露瑞银销毁大屠杀资产文件 → 1998 年和解 12.5 亿美元；2009 年瑞银向美国罚 7.8 亿并提供 4000+ 账户；与美欧签信息交换协议。

**现状**：瑞银 2023 年收购瑞信后管理资产 5.8 万亿美元（超瑞士 GDP 七倍）；银行业对 GDP 贡献不到 10% 但名片效应巨大。
