# 领英找人聚焦相关性

## 一句话定义
在领英等平台找社交对象时，不要广撒网，而要按「相关性」聚焦：看经历找有关联的人、按目标方向筛选、优先校友，相关度越高回复概率越大。

## 详细说明
- 平台：领英及其求职招聘平台「领英职场」。
- 找同行业的人：对某公司（如字节跳动）感兴趣时，与其直接搜公司名一顿乱加，不如点开对方经历，找到与你有关联的再加。
- 先调研方向再筛：如想做数据分析，先搜「数据分析」，再筛选目标公司的人；请求越聚焦、相关度越高，对方答应概率越大。
- 校友是高度相关的人群：不管平级还是向上社交都能瞬间拉近距离，搜人后按学校筛选即可；用领英找内推时相关性更加重要。
