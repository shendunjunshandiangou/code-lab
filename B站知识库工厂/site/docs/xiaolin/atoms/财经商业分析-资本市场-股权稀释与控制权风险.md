# 股权稀释与控制权风险

## 一句话定义
上市的首要成本是股权稀释：创始人从 100% 控股变为要看股东脸色，丧失完全自主权，甚至可能像 Lululemon 创始人那样被资本运作投票踢出公司——这是 Dyson 等公司拒绝上市的核心原因。

## 详细说明
- 上市前创始人可完全自主决策；上市后公众股东进入，发展策略需顾及股东意见。
- 极端情形：Lululemon 创始人（「薯片哥」）在公司上市后，公开市场资本来回运作使其发言权渐失，最终被投票出局。
- Dyson 创始人不上市的最主要原因：不想稀释股权、不想失去对公司的控制权。华为不上市也有股权结构不符合上市条件等因素。
- 对策参考：科技公司常用双重股权（AB 股）等结构保控制权（见相关原子）。
