# 内容再投资飞轮与拍摄自由

## 一句话定义

MrBeast 的核心打法是**把每段视频赚的钱几乎全部投进下一段视频**：频道本身是亏钱的（平均每段视频制作 250 万美元、算上员工成本 450–500 万，每月两段，「去年亏了一大笔」），逻辑是「**专注做人们喜爱的视频而不是想着赚钱，人们能感受到热情、会不断回来**」——流量和 IP 做到极致后，再靠 Feastables 等外部业务变现；烧钱的内容必须有其他业务收入支撑，才能实现「拍摄自由」。

## 详细说明

- 与传统内容生意的区别：不评估单视频 ROI，视频是获客和品牌资产而非利润中心——未来赚钱靠的「显然是 Feastables 和其他业务，不是视频的广告收入」。
- 超高成本制作是竞争壁垒的一部分：2 亿播放来自「做从没人做过的事」，而没人做过通常因为太难、太贵——烧钱本身筛掉了模仿者。
