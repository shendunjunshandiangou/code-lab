# 俄罗斯休克疗法

## 一句话定义

叶利钦时代突然放开价格管制、进出口、利率并将国企私有化，让市场主导——1992 年通胀 2500%、失业率升至 14%、GDP 约腰斩，国企低价落入寡头之手。

## 详细说明

**休克疗法**（Shock Therapy）：从计划经济硬切至自由市场。玻利维亚（1985）、波兰（1989）有成功案例，但对俄罗斯过于激进。

| 后果 | 数据 |
| --- | --- |
| 通胀 | 1992 年 **2500%** |
| 失业率 | 从 &lt;5% 升至 **14%** |
| GDP | 约**腰斩** |

四大措施：放开价格管制、放开进出口、利率自由化、国企私有化——第四条导致国企以极低价格落入极少数人手中。
