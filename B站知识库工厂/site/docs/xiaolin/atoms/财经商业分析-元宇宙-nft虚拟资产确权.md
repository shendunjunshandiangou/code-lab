# NFT 虚拟资产确权

## 一句话定义

NFT（Non-Fungible Token）用区块链使虚拟物品不可替代、去中心化确权——复制粘贴无效，解决虚拟世界资产归属混乱问题。

## 详细说明

- **问题**：虚拟世界 Ctrl+C/V 即可复制「房子/画作」，权属不清
- **NFT 作用**：全球可验证某资产归某地址；无需中心化权威机构发证
- **案例**：Beeple 画作 **6900 万**美元；Decentraland 林俊杰买地 **12.3 万**美元；平台 2021 下半年成交额约 **400 亿**美元（以太坊计价）
- **买家心理**：赌未来融入元宇宙的早期虚拟资产（类比早期比特币）
- **讲者判断**：前景**不太明朗**；与元宇宙终极形态仍众说纷纭

## 补充（Web3 篇，2022 视角）

- **交易风口**：NFT 最大交易平台 OpenSea 融资 3 亿美元、估值 133 亿美元、月活约 150 万；但炒作情绪衰减——Twitter 第一条推文从拍卖价 290 万美元跌到几千美元。
- **品牌营销用法**：想"浅浅涉足" Web3 的企业多从 NFT 下手——明星争买无聊猿、Adidas 购入无聊猿并穿上自家服装、LV 推出 Louis: The Game、Nike 在 Roblox 建 Nikeland；只要 NFT 有热度流量，即可用于拉新与用户互动。
- **投机品画像**：讲得出精彩故事 + 稀缺 = 理想炒作标的，看不清价值时先进场的多是炒家。
