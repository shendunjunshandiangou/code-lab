# 七姐妹卡特尔与 As-is 协定

## 一句话定义
1928 年三大石油巨头在苏格兰秘密达成 As-is 口头协议（划分地盘、锁死价格、避免竞争），后扩至七家形成「七姐妹」，攥住中东几乎所有开采权和全球 85% 石油储备，并用「转移开采地」威胁让产油国打税费价格战。

## 详细说明
- 背景：标准石油拆分后市场大乱战，1920 年代价格战使油价长期低位、无人赚钱；巨头意识到不如联合控价。
- As-is Agreement（1928，苏格兰秘密会面，口头协议）：避免竞争、分割市场地盘、价格定死。后又有四家加入，七家公司组成秘密联盟「七姐妹」（外号出自被拒入伙的意大利国有石油公司董事长）。该协定连各国政府都不知情，20 多年后才泄露。
- 控制力：伊朗、伊拉克、沙特及海湾国家几乎全部开采权 + 全球 85% 石油储备。
- 压榨手法：税费越少付越好；哪国要求提高税费就威胁「去隔壁采」，作为唯一客户迫使各国政府在税费上互相竞底。
- 产油国无力反抗的四个原因：①部分国家尚未独立、政权被欧美控制；②国有化会招致欧美制裁、出口受限；③没钱没技术开采；④七姐妹控制的欧美市场是几乎唯一的销售渠道。背后还有西方政府撑腰（BP 背后是英国政府）。
