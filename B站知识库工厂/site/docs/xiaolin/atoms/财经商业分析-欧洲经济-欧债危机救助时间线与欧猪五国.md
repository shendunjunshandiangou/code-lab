# 欧债危机救助时间线与欧猪五国

## 一句话定义
2009 希腊曝出造假起，救助像按葫芦起瓢：希腊、爱尔兰、葡萄牙、西班牙、塞浦路斯相继伸手，累计 5440 亿欧元（近半给希腊），2018 年希腊还清最后一笔才告终；重灾五国葡意爱希西被戏称 PIIGS「欧猪五国」。

## 详细说明
- 时间线：
  | 时间 | 事件 |
  | --- | --- |
  | 2009 | 希腊新政府自曝债务超 GDP 110%（上限 60%） |
  | 2010-01 | 曝出造假：09 年赤字实为 15.4% 而非 3.7%；国债利率飙至 10%，欧元下跌 |
  | 2010-05 | 欧盟+IMF 给希腊 1100 亿欧元，条件紧缩 |
  | 2010 末 | 爱尔兰为救银行债务 30%→近 120%，获 850 亿欧元 |
  | 2011-05 | 葡萄牙辟谣后半年认怂，获 780 亿欧元 |
  | 2011-07 | 希腊不够花，拟再给 1000 亿欧元；西意国债暴跌、VIX 蹿升、标普将意大利 A+ 降至 A；欧洲央行印钱买西意国债；英国自保先 QE 750 亿英镑 |
  | 2011 末 | G20/欧盟令希腊私人债权人承担 50% 损失——「大臭棋」：市场看穿欧盟不团结，全区债券集体下跌；新刺激因英国、匈牙利反对流产 |
  | 2012 | 标普一口气下调法国等 9 国评级；3 月再给希腊 1300 亿；西班牙（失业率 20%）获 1000 亿；塞浦路斯获 100 亿 |
  | 其后 | 欧盟设立 ESM（欧洲稳定机制）：让市场出钱替代成员国掏腰包 |
  | 2018 | 希腊还清最后一笔救助贷款，紧急救助全部结束 |
- 买单人：问题基本靠德国和 IMF 掏钱解决；欧洲央行 QE 累计印钱上万亿欧元才止住大出血。
- 后遗症：欧元区 GDP 至今未回到危机前十多年的水平；西希失业率长期 15% 左右（德法约 5%）；有经济学家估计希腊 2060 年才能完全恢复；意法德近半民意支持退出，英国干脆脱欧。
- 决策为何慢：20 多个国家达成一致本来就难——货币联盟的集体行动困境。
