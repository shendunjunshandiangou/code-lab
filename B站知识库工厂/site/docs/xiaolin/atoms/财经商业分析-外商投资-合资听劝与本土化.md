# 合资听劝与本土化

## 一句话定义

2011 后中国从「世界工厂」转向「巨大市场」，合资外企需本土化适配——上汽大众途观加长、与大疆/讯飞/腾讯合作智驾与方言助手为典型。

## 详细说明

- 2011 第三产业 FDI 超制造业；税收优惠收缩。
- 途观 L Pro：L2+ 智驾识别中国特殊路况（绿化带、三轮车、施工队）；副驾娱乐屏、方言语音。
- 与本土生态合作（微信上车、华为 HiCar）是「听劝」的一部分。
