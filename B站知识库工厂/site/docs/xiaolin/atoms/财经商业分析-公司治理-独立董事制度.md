# 独立董事制度

## 一句话定义

独立董事指不在公司任职、与公司无重要业务联系、能公正独立判断的董事；大部分国家的司法体系要求上市公司配置一定比例独董，以防「董事会全是自家高管、聚一起给自己涨薪放 300 天年假」——纽交所和纳斯达克要求至少一半独董，很多美国公司超四分之三，麦当劳曾一度 100% 全是独董。

## 详细说明

- 人选逻辑：越大的公司越需要德高望重又有管理经验的人——不能全找老教授，所以美国大公司独董多为其他行业/公司的现任或前任高管，由此产生「高管在彼此董事会互相乱窜」的现象：苹果董事会里有波音、强生、BlackRock 的人；波音董事会里又是 KPMG、GE、Aetna 的人。
- 中美落差：大陆公司的独董多少有「花瓶凑数」意味（董事长安排人），近年监管在整治；美国独董是董事会主流。
- OpenAI 的三名外部董事即独董——不是特殊制度也不是傀儡，属行业标准；员工逼宫信里也要求「再找两个新的靠谱独董进来」。
- 独董比例高不等于外人掌控公司——因为美国董事会大部分时候是「摆设」，详见 [财经商业分析-公司治理-不可替代性即隐性股权](/xiaolin/atoms/财经商业分析-公司治理-不可替代性即隐性股权.html)。
