# 脱欧谈判拉锯与协议内容

## 一句话定义
2017–2020 年脱欧谈判历经梅姨协议三次被国会否决、两度延期、约翰逊重新大选拿下 365 席才通过方案；最终协议保留零关税贸易、改行打分制移民、把北爱海关线画在爱尔兰海上。

## 详细说明
**拉锯时间线**：
- 2017-03-29 英国正式提出脱欧申请，限期两年谈判，谈不成即 no deal；
- 梅姨困境：先与欧盟谈成协议、再让国会通过——对欧强硬谈不成，太软国会不通过（名言「Brexit means Brexit」）；2017-04 提前大选反而从 330 席掉到 317 席，失去绝对控制；
- 2018-11 与欧盟谈成协议 → 2019-01 国会否决；2019-03 再提交再否；延期至 4-12 又否；欧盟延期至 10-31；2019-06 梅姨辞职；
- 2019-07 约翰逊接任，10-17 达成新协议、两天后又被国会驳回；欧盟延期至次年 1-31；约翰逊解散国会重新大选，2019-12-12 保守党赢 365 席，一周后方案通过；
- 2020-01-31 脱欧正式启动，2020-12-31 正式完成。

**协议要点**：
1. 贸易：继续自由贸易、零关税；
2. 移民：人员不再自由流动，英国实行打分制移民系统（人才可进），英国人也不能自由进出欧盟；
3. 北爱边境：不建墙，北爱与欧盟自由贸易，北爱货物进入不列颠主岛需审查（反之亦然）。

**代价**：谈判期悬而不决的不确定性是巨大成本——英欧跨国公司 40% 总部设在伦敦，搬与不搬悬置、计划搁置，严重拖累英国生产力；苏格兰（留欧派）提出脱英公投。
