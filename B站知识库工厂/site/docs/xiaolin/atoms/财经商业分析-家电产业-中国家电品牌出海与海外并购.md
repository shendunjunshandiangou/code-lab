# 中国家电品牌出海与海外并购

## 一句话定义

中国家电"产品出海易、品牌出海难"（海外消费者一听是中国货就"再考虑考虑"），破局三板斧：质量对标国际（3C 认证，海尔拿 001 号证书）、大事件营销提知名度（2008 北京奥运合作伙伴）、以及借 2008 金融危机低价收购外国品牌（三洋白电、斐雪派克、GE 家电）直接买下其海外市场。

## 详细说明

- 品牌困境：张瑞敏 1998 年海外调研——不提品牌对方说产品挺好，一说是中国的就犹豫；老牌（大金、东芝等）积累了几十年信任，消费者没理由冒险换新品牌。
- 3C 认证：2001 年底中国推出国家强制性产品认证与国际接轨，第一张证书发给空调行业。
- 并购的妙处：直接获得目标品牌的海外市场 + 技术协同 + 渠道整合，比正面硬刚成熟品牌快得多；前提是"当了多年工厂积累下的资本"+ 危机窗口期对手虚弱。
- 通用模式：制造代工积累资本 → 危机期逆势收购品牌与渠道 → 完成从 OEM 到全球品牌的跳跃。
