# 游戏引擎「卖铲子」逻辑

## 一句话定义

Unreal 与 Unity 撑起了全球大部分游戏的开发引擎；年收入仅数亿美元，但相当于游戏领域卖铲子，估值增长极快。

## 详细说明

**什么是游戏引擎**：
- 开发游戏用的工具，含预编组件（树、草、材质等）
- 开发者调用即可搭建世界，无需逐像素从零开始
- 相当于「构建好基本世界设定，开发者在上面写剧本」

**两大引擎**：
- **Unreal（虚幻）**：Epic 旗下，虚幻 5 演示画面震撼
- **Unity**：市占率略高

**Epic 案例**：
- 《堡垒之夜》年收入超 50 亿美元，贡献 Epic 95%+ 收入
- 但 Epic 更值得讨论的是引擎，而非单款游戏
- 2012 年腾讯 3.3 亿美元买 40% 股权；2020 年索尼 2.5 亿美元仅买 1.4% → Epic 估值 8 年翻 20 多倍
- 元宇宙热潮时 Unity 市值一度超 500 亿美元（年收入约 11 亿、持续亏损）

**核心能力**：**实时渲染**——电影特效可能数小时渲染一帧，游戏引擎可实时渲染，打开跨行业可能性。
