# 股价隐含收购成功概率

## 一句话定义

收购宣布后，目标公司股价在「原价」与「收购价」之间的位置粗略反映市场对收购成功概率的定价：动视暴雪原股价约 65 美元、微软收购价 95 美元，消息宣布后股价跳到 81 并稳定在 80 左右——粗略反推市场认为成功概率约五五开；此后整场反垄断战中，股价越接近 95 代表市场越看好，成了战况的实时计分板。

## 详细说明

- 逻辑：若 100% 能成，股价应立刻跳到收购价；若必败则趴在原地——中间位置即概率加权（忽略基本波动的粗略估算）。
- 实时投票器的表现：欧盟发难当天股价下跌；CMA 3 月改口当天涨 5%；CMA 4 月正式否决当天暴跌 12%；FTC 败诉后从 82.7 跳到 91；CMA 表态撤销禁令后飙到 94——每个监管节点都被市场即时定价。
- 这也是并购套利（merger arbitrage）策略的基础：赌收购成败赚取价差。
