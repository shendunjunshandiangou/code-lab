# 婴儿潮老龄化需求拐点

## 一句话定义
1963–1973 年是建国以来新生儿最多的十年（每年超 2500 万），这批人自 2023 年起步入 60 岁，叠加其子女多为独生子女的赡养结构，中国养老需求正进入快速释放期。

## 详细说明
- **人口结构**：1963–1973 年婴儿潮每年新生儿超 2500 万；从 2023 年（视频发布年）开始陆续跨过 60 岁大关。
- **赡养压力**：婴儿潮一代的子女多为独生子女，「一对夫妻养四个老人」的结构使家庭养老难以为继，推动社会化养老需求。
- **市场逻辑**：需求 → 商业激励 → 机构投资与供给增加 → 消费者获得更好的养老选择。当前多数机构难以盈利，经济利益能否闭环是行业发展的关键。
- **态度层面**：不应把老人抽象成弱势群体符号——他们展示的是回忆与荣誉而非财富；「现在的他们，正是未来的我们」。
