# 华为三轮制裁与麒麟绝版

## 一句话定义

2019–2020 年三轮 BIS 制裁切断 EDA/ARM 与台积电代工；2020.10 麒麟 9000 成绝版，2022 库存耗尽后只能推 4G 手机。

## 详细说明

- 2019.5–8：先进芯片/技术需 BIS 许可；EDA 三大美商占 70%+。
- 2020.5：凡用美国专利/软件/技术的外国公司亦需许可，台积电/三星/中芯无法代工。
- 麒麟 9000：5nm、153 亿晶体管，2020.10 发布；Mate 60 麒麟 9000S 2023 年突现。
