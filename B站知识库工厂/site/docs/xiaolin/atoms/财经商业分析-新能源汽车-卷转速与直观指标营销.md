# 卷转速与直观指标营销

## 一句话定义

电机转速是极其直观的可比指标（不像审美、舒适度那样主观），"我一万六你两万，那你就是比我厉害"——新玩家卷转速容易快速做出"第一、唯一"的营销人设。

## 详细说明

- 小米对标保时捷做高性能轿跑，立项超级电机，技术路线"装配一代、量产一代、预研一代"：
  - 装配：与联电、汇川两家龙头供应商合作 V6/V6s，21000 转；
  - 量产：自研 V8s，27200 转（当时全球第一）；
  - 预研：实验室 35000 转。
- 一般规律：当行业主体验（续航、快充）趋同后，竞争会转向可量化、可传播的单点指标；直观指标既是技术竞争也是传播策略。
