# 华盛顿共识与新自由主义

## 一句话定义

1989 年 IMF、世界银行与美国政府为指导拉美改革推出华盛顿共识：减少政府支出、国企、监管与干预，利率货币贸易等均由市场主导——叶利钦将其照搬至俄罗斯，与苏联计划经济形成极端对立。

## 详细说明

**新自由主义**（Neoliberalism）：政府极少干预，极度自由市场。

**华盛顿共识**核心：政府「什么都别管」——减少支出、国企、监管、干预，全由市场做主。本为拉美设计，叶利钦认为美国靠此发展很好而引入俄罗斯。

问题：苏联近一个世纪计划经济，与自由市场是两个极端，直接照搬风险极大。
