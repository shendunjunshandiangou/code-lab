# MrBeast 收入结构去广告化

## 一句话定义

MrBeast 的收入大头早已不是平台广告：按他自己的粗算，巧克力品牌 **Feastables 约占 40%**、AdSense 广告分成约 10%、品牌合作约 15%，其余来自玩具、ViewStats、Lunchly 等业务——「今年巧克力的收入会超过整个 YouTube 频道（AdSense + 品牌合作全加起来）」；光靠主账号收入「养不活自己」，自有品牌才是烧钱视频帝国的经济支柱。

## 详细说明

- 时间线：约四年前（视频越做越大、越亏越多时）开始做视频之外的生意——「我需要从别的地方赚钱」。
- Feastables 的差异化叙事：**道德采购可可**，证明巧克力不必靠不道德的原料来源——给消费品牌加上价值观标签。
- 流量→渠道的转化：亚马逊 1 亿美元真人秀合作、沃尔玛铺货（Jimmy 带博主逛沃尔玛看自家巧克力）——顶级 IP 自带议价权和线下渠道敲门砖。
- 这是「羊毛出在猪身上」的创作者经济版本：内容（羊）免费甚至亏钱，从消费品（猪）身上把钱赚回来；与帕梅拉「流量→自有品牌」的商业漏斗同构。
