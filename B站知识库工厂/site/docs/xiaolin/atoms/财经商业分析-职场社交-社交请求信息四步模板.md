# 社交请求信息四步模板

## 一句话定义
给陌生人发社交请求的模板：自我介绍 → 说明联系 → 说明来意 → 提出清晰的小 ask；联系越紧密、问题越相关、ask 越小越清晰、附带潜在价值，回复概率越高。

## 详细说明
四步结构与提高回复率的原则：
1. **自我介绍**；
2. **有联系说明联系**——联系越紧密回复概率越高（如共同认识某人、校友）；
3. **说明来意**——展示自己已做的研究，表明是认真探讨、不是广撒网；
4. **清晰的小 ask**——问题与对方越相关、回复成本越低越好；若能附带潜在价值（如未来合作可能）概率更高。

示例（想开奶茶店、找同行师姐请教）：
> 师姐您好，我是小林，XX 大学食品饮品研究学院 XX 级毕业生。我从 XXX 那里得知师姐深耕奶茶行业。我目前想开一家奶茶店，正在做饮品市场研究，发现年轻消费者对椰奶、燕麦奶等植物奶偏好越来越高。能否在您方便时花 15–20 分钟通个电话，了解行业饮品偏好趋势？也想了解您公司情况，看未来供应链方面有没有合作可能。

效果：ask 清楚且小、对方回答难度低、提供了潜在价值，同时显得真诚、有功课。
