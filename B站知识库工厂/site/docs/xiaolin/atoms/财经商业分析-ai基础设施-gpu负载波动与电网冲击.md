# GPU负载波动与电网冲击

## 一句话定义

AI 训练时 GPU 负载「暴脾气」式猛冲猛落，吉瓦级数据中心对电网形成巨大冲击，是审批监管排斥数据中心落地的重要原因之一。

## 详细说明

**电网平衡原则**：输出 = 负载输入；负载波动，输出须跟着动，否则能量反灌。

**传统调节**：火电可多烧煤，反应快。

**新能源问题**：风光不可预测，接入越多电网稳定性挑战越大；传统 UPS 到一定程度也难处理。

**GPU「暴脾气」**：
- 训练：若干张卡一起干 → 负载突然猛冲
- 保存/停下 → 负载突然猛落
- 电网视角：付费负载一会很大、一会没了

**冲击后果**：
- 吉瓦级（如 1.5 吉瓦）负载突然消失
- 局部短路切走 → 发电机可能停机 → 大面积停电
- 能理解为何居民与监管不愿数据中心建在附近

这与绿电不稳定叠加，使数据中心不仅是用电大户，还须具备**构网**能力支撑电网。
