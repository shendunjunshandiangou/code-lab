# 银发经济规模与需求升级

## 一句话定义
银发经济指满足老年人需求的产品与服务市场；中国规模约 5.4 万亿元、占全球约 5.56%，但中国老年人口占全球 20%，市场空间巨大，且正沿马斯洛需求层次从防护型产品向精神需求升级。

## 详细说明
- 物质/安全层：能爬楼梯的轮椅、防摔安全气囊（老人摔倒时如汽车气囊弹出保护）、关节发热理疗仪等防护产品已五花八门。
- 规模错配：中国银发经济约 5.4 万亿元，仅占全球约 5.56%，而中国老年人口占全球 20%——供给远未匹配人口占比，增长空间大。
- 精神需求层（可预见的增量）：老年大学（教拍照、微信、拍视频甚至当 UP 主）、老年社交（不止广场舞和打牌）、老年婚恋平台、老年恋综等。
