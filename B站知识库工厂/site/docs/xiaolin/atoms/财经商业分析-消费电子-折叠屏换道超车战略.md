# 折叠屏换道超车战略

## 一句话定义

在苹果占据全球智能手机约 85% 利润、高端约 75% 份额的背景下，荣米 VOV 借折叠屏颠覆直板形态、冲击高端，因苹果尚未占领该品类。

## 详细说明

- 整体市场缩水，仅高端增长
- 折叠屏：视觉冲击 + 大屏效率（分屏办公等）
- 2019 华为曾以 17.6% 份额超苹果；荣耀等以「挑战者」姿态迭代第二代、第三代产品
