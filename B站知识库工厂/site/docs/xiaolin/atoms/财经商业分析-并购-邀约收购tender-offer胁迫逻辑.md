# 邀约收购（Tender Offer）的胁迫逻辑

## 一句话定义
邀约收购表面是「市价 100 我出 150」的公开善意报价，实际靠「拿到 51% 后压榨剩余股东」的预期制造恐慌，迫使股东争先接受——因此本质上是一种强制收购手段。

## 详细说明
博弈推演（以市价 100、邀约价 150 为例）：

1. 收购方公开向全体股东溢价邀约；
2. 若凑到 51% 投票权 → 改组董事会 → 对剩余 49% 股东强制推行「以 80 元把你们的股票卖给我」的方案，票数碾压、不服也得卖；
3. 回到邀约时点：每个股东都会想——就算我不想卖，只要别人凑够 51%，我就沦为被 80 元强买的「可怜的 49%」；
4. 于是恐慌性接受 150 的邀约，收购方顺利凑够股份。

- 这个「前端溢价、后端压榨」的阴招在 1970–80 年代美国非常盛行；现实操作与监管远比概念复杂，但逻辑内核如此。
- 它也是毒丸计划诞生的直接背景：董事会需要工具为股东争取时间，避免被突袭式邀约稀里糊涂吞掉。
