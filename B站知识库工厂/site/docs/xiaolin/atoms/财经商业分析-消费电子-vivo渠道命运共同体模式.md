# VIVO 渠道命运共同体模式

## 一句话定义

VIVO「总部+代理」模式通过长期供货与总业绩分红绑定渠道，优先让零售商与省代赚钱，支撑 20 万+ 门店并化解传统经销不信任。

## 详细说明

- 打破厂商与代理多次博弈、串货问题
- 2011–2012 功能机危机：认订单、承担成本、补偿代理降价 → 两月清库存
- 本质：**风险共担**换长期共赢
