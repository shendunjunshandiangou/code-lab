# 中国税收结构以企业端为主

## 一句话定义

中国政府税收第一大项是**国内增值税**（占政府总收入约三分之一），加上企业所得税、消费税——大头全部直接向**企业端**征收；与之形成鲜明对比，**个人所得税只占政府总收入不到 7%**（全国个人交的所得税约等于中国烟草一家公司交的税），而美国个税是政府收入第一大来源、占约一半——这与中国过去偏向生产型社会的发展路径分不开。

## 详细说明

- 央地分成结构（2023 年，中央一般公共预算收入 10.0 万亿、地方 11.7 万亿）：全归中央——进出口相关税费、车辆购置税、消费税；全归地方——房产税、土地使用税、土地增值税、契税等；共享大头——增值税五五分成、企业所得税中央六地方四。
- 增值税机制：商品流转中每个环节只对**增值部分**征税（进货 300、出厂 500，对 200 征税）。
- 中国消费税 ≠ 国际消费税：国际上一般对消费者普征（价外加收）；中国只针对少数商品——影响环境的（汽车、成品油、鞭炮）、不利健康的（烟、酒）、奢侈品，是带惩罚性的税收，1.6 万亿全归中央。
- 烟草是隐形巨头：1.6 万亿消费税里半壁江山来自烟草；2023 年烟草行业工商税总额约 1.2 万亿——日均交税 30 多亿，其中约三分之二是消费税；对比贵州茅台年缴税 200 多亿。
- 讲者提醒：传统经济学模型说「税跟谁收都一样、最终共同承担」，但实际效果非常不一样。
