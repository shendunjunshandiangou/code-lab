# 研发商业化三阶段

## 一句话定义

技术商业化一般经历基础研究（扩知识）、应用研究（解决问题可专利）、开发（可卖产品）三阶段，越往后离商业越近。

## 详细说明

- 基础研究：企业很少资助（如基础数学）
- 应用研究：如网络优化模型
- 开发：直接产出产品
- 硅谷早期估值：无产品时可按 PHD 人头估
