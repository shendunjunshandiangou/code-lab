# CUDA 与 GPGPU 生态护城河

## 一句话定义

英伟达 2006 年推出 CUDA，让 GPU 可用 C 语言等做通用并行计算；硬件垄断 + 软硬件深度协同形成 AI 时代核心护城河。

## 详细说明

- **背景**：GPU 本为 3D 图形并行渲染设计；GPGPU 需突破编程门槛
- **CUDA**：2006 年正式推出，让普通开发者/学生通过 library 即可做加速计算
- **投入**：长期高成本、短期「不合理」，华尔街曾不满（应用场景少）
- **扩展**：航天、生物制药、天气预报、能源勘探等大量采用
- **护城河逻辑**：硬件垄断前提下，显卡与 CUDA 融合越默契，竞争对手越难替代；类比苹果软硬件生态
