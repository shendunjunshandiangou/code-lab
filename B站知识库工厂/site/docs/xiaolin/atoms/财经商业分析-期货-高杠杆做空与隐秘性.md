# 期货高杠杆做空与隐秘性

## 一句话定义

期货支持约 20 倍杠杆、做空无借券限制、交易主体完全隐藏——2024 年 SOFR 单笔 1180 亿美元无人知是谁。

## 详细说明

- 类别：利率 SOFR（量最大）、股指、大宗商品、VIX 等。
- A 股量化「中性/增强」常多空股指期货对冲 Beta。
- 隐秘性：Nick Leeson 搞垮巴林银行时连本行都不知头寸。
