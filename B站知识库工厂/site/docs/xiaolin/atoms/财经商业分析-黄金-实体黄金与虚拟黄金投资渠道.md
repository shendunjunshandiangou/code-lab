# 实体黄金与虚拟黄金投资渠道

## 一句话定义
私人投资黄金分实体（金条金币，踏实但交易成本 3%–20%）与虚拟（OTC/期货/ETF，黄金躺金库、线上交易所有权）；全球三大交易中心为伦敦（LBMA 现货批发）、纽约（COMEX 期货）、上海（上期所）。

## 详细说明
### 实体黄金
- 形式：金条、金币；央行标准金砖 400 盎司（12.4 公斤，按现价约 100 万美元）。
- 优点：价值密度高、握在手里踏实（1.5 亿美元身价可装进一个行李箱——但重两吨拖不动）。
- 缺点：交易成本高（买卖价差约 3%，商场渠道易超 10%，综合 5%–20%）；储存安保成本高（怕偷抢）。
- 反直觉数据：实体黄金仍占每年黄金净投资的 2/3 以上。

### 虚拟黄金
- 交易量巨大（甚至超过美国国债），三种形式：
  | 形式 | 特点 |
  | --- | --- |
  | OTC 柜台 | 交易量约一半，「批发市场」，集中于伦敦 LBMA（认证金库存金超 1 万吨） |
  | 期货 | 成本低、专业性高；COMEX+上期所占全球 97% |
  | ETF | 省心如股票、成本稍高，规模仍是期货零头 |
- 黄金涉及实物运输储存，天然形成集中市场——与股票多交易所并存不同。
