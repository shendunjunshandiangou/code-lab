# 能源转型与石油需求峰值

## 一句话定义
全球变暖共识（巴黎协定控温 1.5℃）加速清洁能源转型，石油使用量一半在汽车，故各国力推电动车；机构预测石油消耗量 2035–2040 年达峰——「石油时代的结束不会是因为石油用光了」。

## 详细说明
- 气候背景：石油燃烧产生二氧化碳，过去 200 年全球升温 1℃，2030 年可能达 1.5℃；2015-12 超过 150 个国家和地区签订巴黎协定，努力把升温控制在 1.5℃ 以内。
- 结构：全球石油使用量约一半用于汽车——这是各国大力推进电动汽车转型的重要原因。
- 前沙特石油部长名言：「石器时代的结束不是因为没有石头了；石油时代的结束也不会是因为石油被用光了。」
- 现实节奏：能源转型虽激进，但当代经济对石油依赖仍很高，多家机构预测石油消耗量 2035–2040 年才达峰后缓慢下降——可预见的长时间内石油仍是世界焦点、中东仍举足轻重。
