# 瑞幸造假退市与翻身路径

## 一句话定义

瑞幸咖啡 2020 年财务造假曝光（先否认、调查后纸包不住火），5 月底被纳斯达克勒令退市，股价从鼎盛的 50 多美元跌到不足 2 美元；但退市后一年多在粉单市场涨超 500%——靠继续开店、领导班子大换血、重新提交 2019 年财报、与 SEC 和解（约 12 亿人民币罚款）、2021 年申请破产重组与债权人和解，一步步铺路重回主板。

## 详细说明

- 财务造假是强制退市里"自己作死"的一类：严重违法，监管不允许继续公开交易。
- 翻身的关键动作：①业务未死——退市不影响持续经营，业绩反弹；②合规修复——补交真实财报、按时披露；③了结监管责任——与 SEC 和解交罚款；④了结债权责任——破产重组不是"不行了"，而是与债权人达成和解的工具。
- 投资视角：强制退市股票转粉单后流动性差（有的每周仅限特定日交易），但若公司基本面能起死回生，低位持有存在高赔率机会。
