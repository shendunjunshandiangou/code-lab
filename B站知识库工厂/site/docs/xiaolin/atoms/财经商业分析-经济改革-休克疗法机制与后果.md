# 休克疗法机制与后果

## 一句话定义

休克疗法是直接硬推市场经济的激进改革：取消计划、放开管制，让市场「无形的大手」自行调节——对俄罗斯造成恶性通胀、GDP 腰斩和长期寡头后果。

## 详细说明

**机制**：不走循序渐进的「软着陆」，而是「一锤子」直接切换到自由市场。类比武侠小说中身受剧毒后先把筋骨全打断再让其重新长好。

**叶利钦四大措施**：
1. 放开价格管制
2. 放开进出口
3. 利率自由化
4. 国企私有化

**历史先例**：
- 1985 年玻利维亚：成功抑制高速通胀
- 1989 年波兰：短暂衰退后走上正轨

**俄罗斯短期后果**（经济学家可预见）：
- 价格突然放开 + 借新债还苏联旧债 → 1992 年通胀 **2500%**（年初奶茶 10 卢布，年末 250 卢布）
- 7 年内失业率从 &lt;5% 升至 14%
- GDP 约腰斩，同期美中 GDP 持续上涨

**最严重后果**：私有化造就寡头垄断，影响延续至今（见相关原子）。
