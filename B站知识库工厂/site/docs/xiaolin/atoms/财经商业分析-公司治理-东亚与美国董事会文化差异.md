# 东亚与美国董事会文化差异

## 一句话定义

东亚公司（大陆、日韩）的董事长是绝对老大：通常也是最大股东，董事会过半是他的人、CEO 由他安排或亲自兼任，格局一家独大；美国公司的董事长只是**维持秩序的角色**（像英国议会的议长，坐中间但不是他说了算）——苹果董事长亚瑟·莱文森、特斯拉董事长罗宾·德诺姆几乎无人知晓，很多公司干脆不设董事长。

## 详细说明

- 东亚的同盟工具：国内一度流行**一致行动人协议**——签约双方所有决策共进退，绑定扩大股权影响力。
- 东亚的冲突模式：一旦董事长控制不了多数席位、反对派出现，就上演「万宝之争」式厮杀，直到争出新老大、恢复一人独大。
- 美国模式的佐证：OpenAI 董事长布罗克曼能在毫不知情的情况下被踢出董事会——因为他的权力并不比其他董事大多少。
- 人数常识修正：「董事会必须奇数好投票」的课本理论早过时，特斯拉、阿里、亚马逊都是偶数；一般 5–15 人（太多难组织、太少易掐架）。
