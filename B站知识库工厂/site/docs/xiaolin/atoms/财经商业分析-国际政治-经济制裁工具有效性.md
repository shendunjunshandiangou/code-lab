# 经济制裁工具有效性

## 一句话定义

制裁工具：贸易禁运、资产冻结、金融/SWIFT、技术封锁；效果取决于替代靠山（如古巴曾靠苏联）与目标经济韧性。

## 详细说明

- 古巴：1963 起最长制裁之一；苏联解体后 Special Period GDP -1/3。
- 制裁 rarely 单独致命，需与国内治理/结构叠加。
