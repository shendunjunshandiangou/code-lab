# 日本金融市场化与信用过度扩张

## 一句话定义
日本泡沫真正的引擎不是低利率本身，而是一条信用过度扩张的逻辑链：金融市场化（美国施压 + 自然演进）→ 大公司改到债券市场低成本融资、抛弃银行贷款 → 银行贷款贷不出去，却仍背着央行"窗口指导"的放贷指标 → 只能下沉给信用差的公司和个人放抵押贷款（房价在涨、有抵押就贷）→ 没有偿还能力的人拿着低息贷款涌入股市楼市——"都借钱疯狂炒股"比"疯狂炒股"危险得多。

## 详细说明
- **传统机制**：国民保守、储蓄进银行，银行贷给企业；政府除利率外靠**窗口指导（Window Guidance）**指挥银行的放贷节奏与方向，长期运行稳定。
- **市场化的冲击**：70 年代起日本被美国施压全面开放金融市场与外汇交易，债券市场随之放开；低利率环境下大型上市公司发现发债比银行贷款更便宜省事，1985-90 年企业大举借贷且主要靠发债。
- **银行的困境与堕落**：利差薄 + 客户流失，但窗口指导的 KPI（"这个月几百亿必须贷出去"）还在，于是转向信用差的企业乃至个人——只要有房产抵押就放低息贷款，赌的是房价上涨兜底。
- **要害**：链条里最严重的问题不是全民炒作，而是**信用的过度扩张**——泡沫由借来的钱吹起，破裂时通过坏账直接击穿银行体系（对照：不加杠杆的泡沫破裂只是财富蒸发）。
- 与欧元区 2000 年代"核心-外围资本流动 + 信用过度扩张"同一性质：廉价信贷错配到无偿付能力的主体。
