# 公募基金与私募基金分类

## 一句话定义

基金按募集方式分为**私募基金**（私下募集、高门槛、高风险高灵活）和**公募基金**（公开募集、大众可买、监管严格），普通投资者能买的主要是公募基金。

## 详细说明

### 私募基金

- 私下募集，普通大众**买不了**
- 面向有钱且风险承受能力强的投资人（对冲基金、PE/VC 等）
- 监管相对宽松，可使用期权、期货、衍生品、杠杆、做空、恶意收购等
- 回报波动极大：年化可达一两百%，也可能亏光
- 基金经理分成可观；小团队管理数十亿至上百亿资产的情况常见
- 公众常有反感（如 2021 年初散户集体买 GME/AMC 打击对冲基金空头）

### 公募基金

- **公开募集**，任何人可买
- 很多人投入毕生储蓄、养老钱 → **监管更严**
- 禁止杠杆、做空等高风险操作，风险相对可控
- 体量：2020 年美联储报告，美股市值约 **1/3** 由公募基金持有；中国约 **8%** 且持续增长
