# 智能合约与 DApp

## 一句话定义
早期区块链（比特币）只是记录交易的公开账本；2015 年以太坊引入**智能合约**——链上可运行"If this then that"式程序，由此诞生完全去中心化、自动运行的应用 **DApp（Decentralized Apps）**，理论上一切互联网应用都能搬上区块链并把数据归属权还给用户（如上传到链上的视频无人能下架），这是 Web3 应用层的技术底座。

## 详细说明
- **能力跃迁**：账本（只能记录）→ 计算平台（可执行逻辑），DeFi、NFT、GameFi 都构建其上。
- **理论与现实**：理论上可重构一切应用，实际操作难度极大；当前主流公链存在交易速度慢、能源浪费、效率低的技术瓶颈。
- **不可篡改的双刃剑**：钱转错要不回、密码丢失无人可救；每年平均超 20 亿美元加密货币被黑客盗走且无法追回——人性化容错的缺失是普及的重大障碍。
- 底层应用延伸：区块链物联网、链上链下数据连接（预言机）、去中心化存储等方向也在发展。
