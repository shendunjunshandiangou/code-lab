# 中小银行客户集中双刃剑与羊群挤兑

## 一句话定义

中小银行天然要聚焦一小拨客户（一个地区或一个领域）——正是集中服务产生比较优势才能发展起来（硅谷银行深耕创业圈、Silvergate 深耕加密圈），但客户同质化意味着**存款同涨同跌、恐慌同频共振**：硅谷创业者抱团、消息灵通，彼得·蒂尔等大佬一号召，一天 420 亿美元（四分之一存款）挤兑——号召的那一刻已宣判死刑，就算银行健健康康也撑不住。

## 详细说明

- 讲者称之为「半个小问题」：没法指望中小银行把客户摊开，集中是它们存在的理由；但需认识到这种模式的内生脆弱性。
- 羊群效应的放大链：客户圈子重叠 → 信息瞬时传遍 → 集体行动 → 挤兑自我实现。健康与否已不重要，恐慌本身就是杀手。
- 银行系统通病：因为恐慌可怕，银行想尽办法掩盖问题；越掩盖市场越怕、越草木皆兵——导致银行**轻易不出事、一出事就是大事**的两极分化。
- 恐慌可跨机构传染：SVB 倒闭后银行股集体下挫、20 多家停牌、USDC 一度脱钩——个例演变为整个中小银行的信用危机。
