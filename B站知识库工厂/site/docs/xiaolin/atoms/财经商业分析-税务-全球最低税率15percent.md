# 全球最低税率 15%（Global Minimum Tax）

## 一句话定义

各国政府斗不过跨国巨头的税务规划师，最终想出一劳永逸的终极补丁——**全球最低税率 15%**：不管避税架构多花哨、税名义上在哪国交，只要企业实际税率不到 15%，就得在总部所在国补齐差额；视频发布时尚未完全落地，预计 2024 年上线。

## 详细说明

- 设计思路：不再逐个封堵漏洞（Double Irish 堵了有 Single Malt、Green Jersey），而是直接给实际税负设下限，让「把利润挪到低税地」失去意义——「哪怕去火星整个三明治」也没用。
- 背景：苹果 2017 年前海外平均税率约 3%–4%，欧盟 130 亿罚单在法律上抓手勉强，说明现有各国税法框架难以约束跨国架构，需要多边协调的制度性方案。
- 讲者的收束视角：表面是苹果避税，实际是各国政府间的较量——市场永远处在政策制定者、监管者与参与者的动态博弈中。
