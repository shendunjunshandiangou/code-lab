# 欧元区入会标准

## 一句话定义

加入欧元区须满足两大财政门槛：政府债务/GDP 低于 60%、年度政府赤字/GDP 低于 3%——旨在防止信用差、花钱大手大脚的国家拖累整个货币区稳定。

## 详细说明

欧元区不是谁想加就能加，有严格准入要求：

| 指标 | 门槛 | 逻辑 |
| --- | --- | --- |
| 政府债务/GDP | &lt; 60% | 政府不能欠太多债 |
| 政府赤字/GDP | &lt; 3% | 政府一年净支出不能超 GDP 的 3% |

设计意图：欧元区须保证整体稳定；若拉入信用不佳、财政失控的成员，可能波及整个欧元区的安全与稳定。

希腊案例：1990 年代末债务/GDP 已达 100%，远超 60%；正常改革路径漫长，希腊等不起，转而寻求「藏债」造假入区。
