# 财团星系结构与主银行商社

## 一句话定义

财团内分金融、综合商社、实业三大类；**银行**（血管/父亲）提供资金，**商社**（母亲/最强大脑）管贸易情报投资，构成星系中枢。

## 详细说明

- 商社：进口资源、出口制成品；情报分析+战略投资，兼**影子银行**。
- 主银行：危机时作白衣骑士（三井银行救丰田案例）。
- 小公司需「认爸爸」选派系才能长大；有潜力企业易被财团收入星系。
