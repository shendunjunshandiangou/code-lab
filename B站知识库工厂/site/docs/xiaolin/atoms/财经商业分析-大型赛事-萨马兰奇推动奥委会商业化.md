# 萨马兰奇推动奥委会商业化

## 一句话定义

1980 年起任 IOC 主席的萨马兰奇认为 IOC 须商业化、自给自足才能办更大奥运；1972 年 IOC 资产约 200 万美元，2013–2016 周期收入约 50 亿美元。

## 详细说明

- 1980 年代前奥运商业化很弱，过度商业化与奥林匹克无私精神相悖。
- 1972 年 IOC 账上资产约 200 万美元。
- 萨马兰奇推动 IOC 走上商业化道路，2013–2016 周期收入约 50 亿美元，数量级差异巨大。
- 其任内促成 TOP 全球赞助计划等机制；视频称其「不但是中国人的朋友，还是钱的朋友」。
