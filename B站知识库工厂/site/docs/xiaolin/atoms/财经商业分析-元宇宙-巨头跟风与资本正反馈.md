# 元宇宙巨头跟风与资本正反馈

## 一句话定义

Meta 宣布 all in 后，微软（Hololens/Minecraft/Mesh）、苹果、字节、腾讯等被迫加大投入，形成巨头→资本→人才→估值→更多资本的风口正反馈链。

## 详细说明

- **微软**：Meta 宣布 **4 天后**称也有元宇宙（Hololens MR、Minecraft、Mesh 协作）
- **其他**：苹果加大 VR 研发；字节、腾讯并购/表态
- **正反馈**：巨头涌入 → 资本涌入 → 资源整合 → 行业估值升 → 投机者涌入 → 产业加速
- **讲者**：行业被商业运作「绑架」往前跑；5–10 年能否兑现价值或打水漂**说不准**
- **Epic**：Fortnite + Unreal 引擎；虚拟演唱会百万观众——游戏线目前能摸到的最远
