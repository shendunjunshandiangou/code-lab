# VIVO 影像与屏下指纹差异化

## 一句话定义

VIVO 以 HiFi 切入、2016 起以自拍场景重做影像（蔡司+自研 V1 芯片），2018 与汇顶共研全球首款屏下指纹，实现份额跃升。

## 详细说明

- Xshot 光学参数超前 → X7 前置柔光抓自拍需求，份额 8.2%→14.8%
- 2021 份额 21.5% 中国第一；2018 屏下指纹失败损失与汇顶五五分
- 2014 起接触蔡司，2020 正式合作
