# 投行经历的职业价值与出路

## 一句话定义
投行出来的人出路多样——跳其他投行、去买方、创业、周游世界——但「从没听过任何人说那几年是浪费时间」：高压环境换来的是专业语言、资源积累和被验证过的意志力，类似高考式训练的长期职业资产。

## 详细说明
- 常见出路：其他投行（平台间跳槽）、买方（对冲基金/资管等）、自主创业、间隔性生活（周游世界）。
- 隐性收获：
  - **专业语言内化**：入行时研报满眼缩写「两眼一抹黑」，看习惯后变成自己的语言；
  - **平台资源**：全行研究报告随便看、CFA 等考试公司报销——利用程度全看个人；
  - **意志力凭证**：「姐连这都经历过来了」——高压经历回头看是值得感激的成长期，与回忆高中军训同构。
- 华尔街中国人越来越多的一个解释：经历过高考「军训式生活」的人自带这种意志力，在高压筛选环境中有比较优势。
- 心态提示：压力大到想爆炸是常态，但多数人离开后回望都认为值得——评估一段高强度职业经历，应看它留下的能力与记忆资产，而非当下的舒适度。
