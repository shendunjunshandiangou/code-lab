# 股票回购信心信号

## 一句话定义

股票回购（share buyback）是公司用手头现金从市场买回部分自家股票，是给市场和投资人注入信心的强心剂：一说明现金流充足，二暗示"这个价格买回来值"；苹果十年回购 4000 多亿美元、居美股之首，是其市值一度达 3 万亿美元的重要推手之一。

## 详细说明

- 与私有化的关系：私有化是把流通股全部收回，回购是部分收回——股价偏低时更常见的操作。
- 有意思的实证：经济学家统计显示公司其实也判断不准自家股价被高估还是低估，但回购行为本身展现的决断与信心确实能提振市场。
- 与中国的工具对照：中国央行 2024 年的"回购增持再贷款"就是为上市公司回购提供专项资金支持。
