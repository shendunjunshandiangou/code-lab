# 次贷危机触发希腊债务危机

## 一句话定义

2008 年次贷危机通过信用收紧、旅游/海运收入崩塌、次级债抵押品暴雷三条路径同时打击希腊，使其无法继续「借新还旧」，引爆主权债务危机。

## 详细说明

希腊经济高度依赖海运与旅游，财政长期靠低息借债滚动。次贷危机带来三重打击：

| 渠道 | 机制 |
| --- | --- |
| 融资断裂 | 全球避险、信用收紧，投资人不愿买希腊国债 |
| 实体经济 | 危机中旅游与海运收入锐减 |
| 抵押品暴雷 | 此前向德法借债时抵押美国次级债券；危机中债券暴跌，希腊国库被掏空 |

巴菲特名言应验：**潮水退去，才知道谁在裸泳**。

此前欧洲经济向好时，大家对希腊高债务睁一只眼闭一只眼——2004 年雅典奥运会照办；只要信心强，债务/GDP 远超 60% 也能靠信用滚债。**金融玩到底就是信用**。
