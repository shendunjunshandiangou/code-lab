# 撒切尔主义供给侧改革

## 一句话定义
1979 年撒切尔夫人对英国进行供给侧自由化改革——金融去监管、减税、打击工会、控通胀、国企私有化——奠定伦敦全球金融中心地位，但代价是失业率飙至近百年最高的 12%。

## 详细说明
- 供给侧改革：不刺激消费，从产业端入手——金融市场与银行业去监管、自由化，减税，打击工会，控制通胀，并将战后大量国有化的产业私有化，用市场经济激活经济。这套理念被称为「撒切尔主义」。
- 时代背景：1980 年代全球主题就是市场自由化（与苏联戈尔巴乔夫改革方向类似，但英国的执行效果好得多）；英国经济自此逐渐追赶德法。
- 金融业崛起为支柱产业，伦敦成为全球第二大金融中心。佐证：美元最重要的利率 Libor（London Interbank Offered Rate）就是伦敦银行间拆借利率；全球金融中心格局为欧洲伦敦、北美纽约、亚太香港/新加坡。脱欧后不少公司开始迁出伦敦。
- 代价与争议：失业率飙升至 12%（近百年最高），社会动荡；私有化动了很多人的蛋糕，撒切尔因强势而极具争议——但也正因强势，英国才被快速推上自由市场经济道路。
