# 十五五经济与安全双主线

## 一句话定义

「十五五」两大关键议题是经济与安全：经济侧抓内需/产业/科技，安全侧兜住民生与系统性风险，并适应外部环境由「机遇期」到「机遇与风险并存」的判断转变。

## 详细说明

**安全主线升温**：近年文件「安全」频率明显升高——无安全则增长站不住。

**外部环境措辞演变**：
| 时期 | 判断 |
| --- | --- |
| 十六大起 | 重要战略机遇期 |
| 二十大起 | 战略机遇和风险挑战并存，不确定难预料因素增多 |

**对外**：积极友好、开创共赢；继续**一带一路**、推进**人民币国际化**。

**对内安全兜底**：
- **民生**：收入、就业、教育、养老、医疗
- **国家安全体系**：粮食、能源、产业链 + 网络、数据、人工智能、生态等
- **风险化解**：房地产、地方政府债务、中小金融机构——有序化解，严防系统性风险

经济与安全并行，构成未来 5 年政策总框架。
