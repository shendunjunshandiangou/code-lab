# 多邻国绿猫头鹰 IP 网络共创

## 一句话定义
多邻国吉祥物"多儿"是"一半刻意设计、一半网络共创"的超级 IP：官方设计了绿猫头鹰形象，而其"阴阳怪气"的人格由网友对推送通知的解读发明，团队再顺着网络梗迭代放大。

## 详细说明
- 刻意部分：第一版 App 就加入吉祥物，因为创始人相信吉祥物能促使用户回来学习。
- 共创部分：App 图标是猫头鹰的脸，系统通知旁总显示图标，用户误以为通知是"猫头鹰发的"，把通知语气人格化到猫头鹰身上；团队顺势写出有个性的通知（如"这些提醒好像没用，我先不发了"），网友称其 passive aggressive，并做出"猫头鹰绑架你全家逼你上课"的梗图。
- 迭代机制：网上出现受欢迎的梗 → 团队"多做一点那样的"→ 人格与网友共同进化。
- 好角色公式：角色必须**极度想要一个东西且存在障碍**——多儿想要的是"你去上课"，障碍是"你太懒"。
- 边界管理：内部原则叫 wholesome unhinged（健康的疯感）——疯但安全、不过度性化，保住家庭品牌属性；给猫头鹰开 OnlyFans 的提案被 CEO 直接否决，男友抱枕则通过。
- 商业效果：全球用户大量二创、官方下场自黑互动，衍生 Lily、Dolly、Eddie（存疑）等 IP；在中国与美团、瑞幸、QQ 音乐等联名（含"多儿去世"营销事件），品牌被视为"很酷"。
