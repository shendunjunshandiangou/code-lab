# BVI VISTA 信托与控制权保留

## 一句话定义
一般信托的受托人受「谨慎投资原则」约束、不得把管理权还给委托人；英属维尔京群岛（BVI）的 VISTA 信托废除了这一约束，让委托人既控制资产又实现法律隔离，因此深受中国互联网大佬欢迎。

## 详细说明
- **一般规则**：受托人有谨慎投资原则，要对信托财产负责，不能把管钱的权利转让回委托人。若委托人同时管钱又是受益人、资产还不在名下，属于**伪信托**，隔离效力可能不被承认。
- **BVI 特殊性（VISTA 信托）**：允许委托人决定谁来管钱——包括「委托自己管」。结果是：委托人 = 管理人 = 受益人，但资产不在自己名下，离婚、追债都动不了。
- **使用者**：刘强东（京东股份）、马云（阿里）、黄峥（拼多多）、王兴（美团）等都在维尔京群岛设有此类信托。
- 这解释了为什么中概股创始人的持股结构里频繁出现 BVI 实体。
