# 政府发债两条红线：利率与通胀

## 一句话定义
真正限制政府发债的不是债务上限而是两条红线：①利率——国债发多推升无风险利率、抑制经济且付息压垮财政（可靠央行 QE 配合绕过）；②通胀——政府永远绕不过去的硬约束，唯一解法是化疗式紧缩。

## 详细说明
- **红线一：利率**。国债供给增加→价格下降→无风险利率上升→全经济利率跟涨（效果等同加息，抑制一切）；政府自身发债成本也上升，若税收连利息都不够付，就沦为「给全球债主提供固定收益的基金」、陷入债务漩涡。日本若平均利率达 5%，全部财政收入不够付息——但实际仅 11% 财政收入用于付息且逐年下降，因为央行印钱买国债压低长期利率（2013 安倍经济学配套 QQE；后来明牌 YCC 把十年期利率压在 0.25% 以下、2022 年上调至 0.5%）。故利率红线可由政府+央行配合绕过。
- **红线二：通胀**。需求不足、就业不足都能花钱解决，通胀一来所有政策小聪明全部失效，历史上唯一解法是紧缩——像化疗，好坏细胞一起杀、靠抑制整体经济给物价降温；狠不下心继续刺激就滑向恶性通胀（委内瑞拉）。美国 2021 年若预见大刺激引发高通胀+暴力加息，绝不敢花那么多钱。
- **债务红线不是绝对数字**：只要不触两条红线，发多少债是增长与稳定的权衡——丹麦/瑞典/荷兰/瑞士稳健（债务比 GDP&lt;50%），美日追求高增长，中国中央稳、地方债城投债问题多。低利率低通胀+政府信用好时，债务数字再吓人经济也没事（疫情前美欧日）；环境一变红线就移动——美国十年期利率从 0.6% 升至 3.7%、通胀从 0 到 4.9%，日本约 260% 的债务率两年前不算高、现在就有问题了。
