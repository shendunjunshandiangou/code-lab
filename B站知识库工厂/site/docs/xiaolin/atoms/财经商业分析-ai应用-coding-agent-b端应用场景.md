# Coding Agent B端应用场景

## 一句话定义

Coding Agent 当前核心收入与渗透在 B 端：程序员提效、金融分析师做财报分析、医疗企业等——SpaceX 招股书称 AI 市场潜在价值 90%+ 在企业端。

## 详细说明

**核心场景**：
1. **程序员**：Vibe Coding，让机器写代码而非人写
2. **金融分析师**：财报分析（投行等场景）
3. **医疗**：Claude Code 正在进军医疗企业

**代表产品**：Anthropic Claude Code（目前不在中国售卖）

**与 Chat 的本质区别**：Coding Agent 可理解成「看过世界上非常多代码的超级程序员」——给自然语言要求后拆解为写代码、自检错误、交付应用；大模型对话则是「学过很多文字」。

**OpenAI 动向**：组织架构调整，增加帮助 B 端服务落地，不再纯研究大模型。
