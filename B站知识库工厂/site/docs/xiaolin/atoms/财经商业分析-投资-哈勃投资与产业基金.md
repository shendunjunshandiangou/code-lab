# 哈勃投资与产业基金

## 一句话定义

2019 华为成立哈勃投资战略投资部门，近年发展产业基金形式吸引社会资本，整合专业性与资金推动国产替代。

## 详细说明

- 国产替代需上下游深度磨合研发，非「图纸一扔代工」。
- 哈勃投资支持半导体、材料等萌芽期产业；与昆仑玻璃、芯片等案例联动。
