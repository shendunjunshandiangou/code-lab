# 新加坡港口 + 下游加工一条龙

## 一句话定义
当人工成本上升、制造业难以继续扩张时，新加坡把高附加值产业的下游环节与港口中转结合：原油过境顺便精炼、芯片过境顺便封装测试，用"一条龙服务"保住了制造业地位。

## 详细说明
- 背景：经济起飞后人工成本上涨，低附加值制造难以为继；对策一是向更高技术领域升级，二是发挥全球贸易大港优势。
- 模式：挑选高附加值产品做下游加工——中东原油运来精炼加工后直接运往目的地；芯片上游设计制造做不了，就做下游组装、封装、测试，"连运输带封测，你睡觉的功夫全搞好"。
- 本质：产业下游业务 × 港口中转 = 物流与加工的一体化服务，让制造业至今仍是新加坡经济的重要一环。
