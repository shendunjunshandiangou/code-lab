# 商业金融入门书单

## 一句话定义

小Lin亲测有效的精简书单：《When Genius Failed（拯救华尔街）》——LTCM 故事书，看华尔街真实的杠杆、风险与环环相扣；《巴菲特致股东的信》——50 多年智慧的价值投资框架，说人话、挖本质；《魔鬼经济学》——经济学思维启蒙，结论反直觉、教你不人云亦云；《原则》——达利欧的自我成长与决策原则。

## 详细说明

- 选书标准：不搞几十本收藏式书单，只推自己读完真觉得有帮助的，并说明适合谁读。
- 延伸书目：《Dynamic Hedging》（塔勒布，衍生品交易，特别数理）、《Too Big To Fail》（纽约时报记者视角的 08 危机）；《牛奶可乐经济学》《薛兆丰经济学讲义》未读过不评价。
- 行为金融学方向：任选一本即可，内容大同小异（心理偏差如何影响投资与商业行为）——先知非理性，才能理性决策。
- 宏观经济：没有一本能让人"读完入门"的神书——利率、汇率、货币、期货、地缘政治交织太复杂，靠持续积累 sense。
- 巴菲特论做空的示范：做空量小、有轧空（short squeeze）风险、借贷成本高——几个词精辟总结巨量资料，体现"挖掘底层逻辑"的功力。
