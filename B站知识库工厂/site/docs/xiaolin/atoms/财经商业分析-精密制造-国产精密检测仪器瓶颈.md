# 国产精密检测仪器瓶颈

## 一句话定义

精密剪裁制造依赖可探测金属内部精度的高精度检测仪器，该设备被美国、德、日、瑞等国垄断，高价进口制约国产精密制造产业链在剪裁路线上的竞争力。

## 详细说明

对极精密零部件，肉眼无法判别质量，须「后验」式高精度探测。此类仪器基本被美国、德国、日本、瑞士包圆，单台数百万至上千万元。

**成本传导案例**：

- **医疗**：心脏介入手术需血管造影机实时观测血管；德国进口大型造影机单台成本数千万，推高手术费。
- **航天**：发动机叶片等零部件检测，单次成本可达数十万甚至更高。

因此，依赖剪裁制造 + 进口检测仪器的路线，构成国产精密制造产业链的卡点之一。增材制造等替代工艺的意义，部分在于减少对这条「剪裁—后验检测」路径的依赖。
