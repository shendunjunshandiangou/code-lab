# 大小政府理念与两党经济逻辑

## 一句话定义

民主党倾向大政府：加税（富人）、政府救济、全民医保、刺激就业；共和党倾向小政府：减税、放松监管、减少开支，理念接近里根经济学与自由市场。

## 详细说明

**民主党（哈里斯）**：政府先收钱再大包大揽社会福利——育儿补贴、social security、教育、医疗；多招公务员；加强金融监管（Dodd-Frank）、反垄断。

**共和党（特朗普）**：减税 + 放松金融监管与传统能源监管 + 减少政府开支 + 控制货币供给；推动解雇联邦工作人员；削弱 Dodd-Frank 与反垄断。

**里根经济学**（80 年代高通胀后）：减税、放宽监管、减政府开支、控货币——政府靠边，带动全球金融自由化浪潮。

**类比**：民主党像严管班主任；共和党像撒手掌柜——市场自由竞争，过热时再干预。

与经济学流派略似对应：凯恩斯主义（政府刺激）≈ 民主党；货币主义（自由竞争、美联储控利率）≈ 共和党，但不绝对。
