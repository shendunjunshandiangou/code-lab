# 房价需求四因素框架

## 一句话定义

小Lin说总结的房价需求分析框架：人口、经济（含外部投资）、利率、政策；前两者为长期市场因素，后两者为短期行政因素。

## 详细说明

| 因素 | 要点 |
| --- | --- |
| 人口 | 10 年尺度首要；结构 + 流动（底特律、加澳移民） |
| 经济 | 人均收入、就业；须理解背后产业逻辑 |
| 外部投资 | 开放城市国际游资（葡、希度假别墅） |
| 利率 | 降 1% 粗算约等于 100 万 25 年贷款「便宜 10 万」 |
| 政策 | 限购、税收；短期立竿见影 |

四条件齐 → 易涨；但低利率下四条件也易培育**信用泡沫**，勿盲从。

**案例**：摩纳哥无所得税 + 区位 → 全球最高均价之一（约 5 万美元/平米）。
