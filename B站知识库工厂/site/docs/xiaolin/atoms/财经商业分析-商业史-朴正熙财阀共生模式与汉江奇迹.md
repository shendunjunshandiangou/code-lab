# 朴正熙财阀共生模式与汉江奇迹

## 一句话定义
1961 年朴正熙军政府先抓财阀立威，再与之达成交易——"听我的、银行收归政府（钱我说了算），政府给政策与优惠贷款让你们飞黄腾达"——用听话能干的财阀充当经济崛起的引擎，开启了财阀时代与汉江奇迹；1987 年民主化后权力关系反转：总统五年一换而财阀长存，财阀反向渗透政府，形成"青瓦台魔咒"与"三五定律"。

## 详细说明
- **军政府时期（1961-1987）**：财阀再大也得听朴正熙、全斗焕的。政府通过优惠政策与贷款扶持财阀完成五年计划，财阀借势壮大——国家与财阀深度绑定。
- **民主化后的反转**：总统短任期 vs 财阀永续，操作空间归了财阀。李健熙威逼利诱渗透政府，可影响大选、政策与经济方向；汉城申奥这类国家大事也要靠财阀出面搞定国际奥委会。
- **青瓦台魔咒**：民选总统几乎人人卷入贪腐——本质是财阀政治的必然。
- **三五定律**：财阀犯罪一审最高判五、二审改判三、缓刑五年，再提前释放或特赦——李健熙（1996、2008 两次获罪均被特赦）、李在镕、乐天辛格浩/辛东彬、现代郑梦九皆例。韩国司法"高调抓国家级人物 + 挠痒痒式放人"并存："法治在韩国——有，但不多。"
- **量级**：三星集团收入 ≈ 韩国 GDP 的 22%（近十年常年约 1/5），"韩国人一生逃不过税收、死亡和三星"。
- 对照：日本财阀战后被拆解为松散财团（见相关原子），韩国财阀则保留家族集中控制——两条路径的分野。
