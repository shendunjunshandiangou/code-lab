# 大选政策对美债与资本市场影响

## 一句话定义

特朗普减税预计推升联邦债务/GDP 至 2028 年约 116%，哈里斯约 109%；特朗普政策短期刺激经济、吸引外资回流，偏利好股市。

## 详细说明

长期美元利率看**国债**：供给增加 → 价格下跌 → 利率上涨。

彭博估算：
- 特朗普上台：2028 年债务/GDP **116%**
- 哈里斯上台：约 **109%**
- 两党都会扩大债务，特朗普可能更多

**债务口径**：名义债务超 35 万亿美元，含政府内部持有（社保、医保、公务员基金）；联邦政府整体对外净债务约 **GDP 99%**。

特朗普：企业减税 + 富人减税 + 投资减税 → 外资回流（上届已有）+ 短期经济刺激 → **资本市场/股市偏利好**。
