# 脱欧不确定性成本

## 一句话定义
无论脱欧本身好坏，多年谈判悬而不决产生的「不确定性」就是巨大的经济成本：企业无法决策、计划搁置，直接拖累一国生产力。

## 详细说明
- 欧盟与英国的跨国公司中约 40% 总部设在伦敦；2016 年公投后脱欧方式长期悬而未决，企业不知道总部搬还是不搬，凡事要做几手准备，发展计划搁置等待谈判结果。
- 「你说要杀要剐给个准信，我好摆个姿势等着」——不确定性让企业既不能按留欧安排也不能按脱欧安排，极大影响了英国的生产力。
- 一般化结论：政策的不确定性本身独立于政策内容构成成本，重大制度变更拖得越久、代价越大。
- 附带的政治后遗症：支持留欧的苏格兰提出脱英公投，国内进一步割裂。
