# 印度独立后计划经济与许可证制度

## 一句话定义
被「英国式自由贸易」压榨两百年后，独立的印度走向另一个极端：工业银行业全面政府管控、照搬苏联计划经济、拒绝外资，并独创许可证制度（License Raj）——办企业须打通七八十个部门审批，拿到证即进保险箱。

## 详细说明
- 心理动因：领教了殖民者的「自由贸易」，独立后选择严格自我保护与封闭。
- License Raj 机制：几乎所有企业无论做什么都要申请许可证；审批链条长达七八十个部门，需要大量关系运作；获证者获得类垄断地位、失去竞争压力。
- 双重恶果：①核心产业政府垄断+无市场竞争→工业整体拉垮（结构性跳步的根源）；②半个世纪的审批寻租把腐败嵌入整个经济运行——上学、看病、考驾照（60% 的人不考试就拿证）都要打点，透明国际 2020：39% 与政府打交道的印度人行过贿，亚洲最高。
- 结果：独立后经济发展缓慢，直到 1991 年危机倒逼改革。
