# 巴勒斯坦 WBG 地位与西岸加沙分裂

## 一句话定义

巴勒斯坦为联合国非会员观察国，世行称 WBG（西岸+加沙）；2007 年内战后分裂为法塔赫管西岸、哈马斯管加沙，经济形态截然不同。

## 详细说明

- 2012 年升格非会员观察国；138/193 国承认，美法德日韩等未承认
- 世界银行、IMF 称 WBG 而非直接称巴勒斯坦
- 2022 年人均 GDP：西岸约 4458 美元，加沙约 1257 美元；以色列约 5.466 万美元

**分裂**：2006 哈马斯赢得选举，2007 年内战后加沙由哈马斯控制，西岸由巴解/法塔赫主导政府控制，格局延续至今。
