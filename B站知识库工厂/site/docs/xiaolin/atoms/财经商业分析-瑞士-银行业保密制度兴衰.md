# 瑞士银行业保密制度兴衰

## 一句话定义

1934 年银行法第 47 条确立的绝对保密曾是吸引全球 1/4 跨境资产的杀手锏，2009 年后在美欧压力下崩溃，标志瑞士「保密时代」结束。

## 详细说明

- **巅峰**：全球约 46% 亿万富翁为 UBS 客户；UBS 管理资产约 5.8 万亿美元。
- **用途**：逃税（开曼空壳 + 瑞士账户）、政客贪污（马科斯案等）、毒枭洗钱。
- **转折**：1997–1998 瑞银大屠杀资产文件案；2009 年瑞银罚 7.8 亿并向美国交 4000+ 账户；与美欧签 CRS 信息交换。
- **后果**：2008 年后金融业竞争力下降；2023 年瑞银收购瑞信。
- **GDP 贡献**：银行业直接贡献不到 10%，但名片效应巨大。
