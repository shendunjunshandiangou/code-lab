# 人均收入与人均GDP区别

## 一句话定义

人均 GDP 是生产总值÷人口；人均收入仅含个人平均收入，一般低于人均 GDP，两者不可混用。

## 详细说明

「GDP = 所有人收入总和」不准确。理论上**全要素收入 = 生产总值**，但收入还包括企业利润等，不单是工资。

- **人均 GDP**：整个国家生产价值的人均
- **人均收入**：仅个人平均收入

**2024 年美国**：
- 人均 GDP：**85,800** 美元
- 人均收入：**73,200** 美元

一国所有生产价值、所有收入、所有开支，理论上应相等——即 GDP 的三种算法本质一致。
