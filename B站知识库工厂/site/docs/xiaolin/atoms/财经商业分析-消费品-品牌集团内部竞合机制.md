# 品牌集团内部竞合机制

## 一句话定义

欧莱雅集团化经营"更绝的一点"在品牌营销：子品牌共享供应链与研发，但销售营销完全独立、互相竞争（同集团也要争预算）；集团再在可接受限度内强制经验共享（兰蔻的好招数推给赫莲娜、修丽可）——竞争 + 共享的机制持续逼出新创意。

## 详细说明

- 营销对消费品的分量：全球前几大广告主中欧莱雅第四、宝洁第一、联合利华第三，全是消费品公司；营销比拼的核心是创意。
- 基金公司类比：各基金经理本各自为战，老板要求把赚钱策略分享给同事——原本能躺赚的人被迫继续创新；共享不是全盘托出，而是限度内的强制交流。
- 机制启示：大集团保持创新活力的关键不是资源本身，而是"内部赛马 + 强制知识流动"的组织设计。
- 创意花絮：兰蔻买下南法玫瑰庄园让品牌故事与链条具象化；巴黎奥运把赛场放到名胜古迹旁、喊"百年奥运重回法国"——浪漫仪式感全有还省钱。
