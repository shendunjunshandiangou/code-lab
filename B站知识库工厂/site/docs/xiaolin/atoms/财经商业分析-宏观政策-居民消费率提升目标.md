# 居民消费率提升目标

## 一句话定义

「十五五」明确要提高居民消费率（居民消费/GDP），中国目前不足 40%、远低于全球平均 56% 与美国近 70%；政策发力重点可能在消费意愿，并绕不开促就业、增收入、稳预期。

## 详细说明

**指标定义**：居民消费率 = 居民消费占 GDP 比重。

**横向对比**：
- 中国：&lt; 40%
- 全球平均：~56%
- 美国：~70%

**措辞力度**：纲领文件少见定量指标；本次由十二五的「上升」改为「**明显提高**」，可参考十二五 5 年提高约 3 个百分点的历史效果，期待更高力度。

**两大决定因素**：
1. **消费能力**：中国居民收入占 GDP 比例并不低
2. **消费意愿**：远低于发达国家 → **未来政策很可能重点刺激意愿**

可能手段：消费券、带薪假期、新消费场景等。

**底层三角**：促就业、增收入、稳预期（工作稳定性、育儿、养老）——「我不消费是因为不喜欢吗」的回应。
