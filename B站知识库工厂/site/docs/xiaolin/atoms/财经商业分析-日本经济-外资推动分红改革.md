# 外资推动分红改革

## 一句话定义

外资持股上升后要求分红回报；2015 起监管推动增分红，2020 巴菲特访日劝商社分红，东证规定回购/分红——近年日股上涨的重要推手。

## 详细说明

- 原财团内部利益优先，盈利留企业不分红，「日本挣钱日本花」。
- 外资：「我又不是你亲爸爸，不分红干嘛投你？」
- 与交叉持股解体并行，资本市场更面向外部股东。
