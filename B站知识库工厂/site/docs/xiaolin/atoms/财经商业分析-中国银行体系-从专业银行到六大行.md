# 中国银行体系：从专业银行到六大行

## 一句话定义

中国银行体系几十年浓缩了银行进化史：计划经济时期两家银行只做执行 → 改革开放后四大专业银行分地盘 → 1998 年坏账后引入股份制银行竞争、四大行转为国有商业银行并扩为六大行、政策性业务剥离给三家政策性银行。

## 详细说明

**计划经济时期（1949 后）**：不需要中介调配，只有中国人民银行（印钱 + 理论上的存贷款 + 财政部下的拨款，什么都干）和中国银行（只管外汇外贸）。

**改革开放（1978 后）**：央行从财政部独立、专管货币政策；借贷业务归四大专业银行——工行管商业、建行管基建、中行管外汇、农行管农业，"行如其名"、井水不犯河水。问题：毫无竞争，谁都不缺存款、贷款方是绝对弱势；效率不足与过度放贷间接导致 1998 年大量银行坏账。

**引入竞争**：

- 成立 12 家股份制银行（民生、光大、招商、广发等）；
- 城乡信用社合并升级为城商行、农商行；
- 四大行取消业务地盘限制，转为国有商业银行，后扩为六大行；
- 修铁路、基建等政策性任务剥离，成立国家开发银行、农业发展银行、进出口银行。

**现状**：中国大行资产规模居全球前四；银行资产占中国金融体系资产总量 85% 以上，占据主体性、主导性地位。
