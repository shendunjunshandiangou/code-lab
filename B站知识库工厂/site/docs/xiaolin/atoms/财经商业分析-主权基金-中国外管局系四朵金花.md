# 中国外管局系四朵金花

## 一句话定义

外管局约 3.2 万亿美元外汇储备由华安、华美、华欧、华新等驻外机构分投全球；华安投资约 1.09 万亿美元，为全球第三大主权基金。

## 详细说明

- 华安（全球）、华美（美）、华欧（欧）、华新（亚）
- 约 55% 美元资产，有意减持美债（峰值 1.2 万亿→约 7500 亿）
- 梧桐树、银杏树、曼陀罗等统称外管局系，信息极神秘
