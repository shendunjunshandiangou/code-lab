# 特朗普关税战2.0与解放日关税

## 一句话定义

特朗普第二任期关税战 2.0 从无差别针对中国扩展为普适关税 + 行业保护 + 解放日对等关税，核心是用关税政治讨价还价，真正约束来自资本市场（美债恐慌后暂停 90 天）。

## 详细说明

**理由有效性**：
- 有限：逆转逆差（美元/金融账户问题）、增政府收入（预计约 3000 亿/年，占收入 6%）
- 有效：制造业回流、国家安全、**讨价还价**

**三条战线**：
1. 北美（加墨威胁，现 80%+ 仍美加墨协定，均关税 &lt;5%）
2. 钢铁铝 50%、汽车 25%
3. **解放日**（4/2 行政令）：基准 10% + 对等关税（中 34%、欧 20%、日 24% 等）；市场暴跌后暂停 90 天

美国均进口关税从 2%–3% 升至估 **17%**。

**成果**：约 3000 亿/年关税收入 + 1.5 万亿美元投资承诺；**代价**：信任消耗、美元下跌、各国或建新联盟。WTO 上诉机构因美国拒任命法官而瘫痪（脑死亡）。
