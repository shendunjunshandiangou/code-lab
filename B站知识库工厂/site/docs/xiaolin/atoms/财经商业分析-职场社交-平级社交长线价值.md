# 平级社交长线价值

## 一句话定义
与资历相当的人（平级）社交回复率高，且随着彼此一起晋升，未来会形成分布在各机构、各团队的广泛资源网络，是被低估的长线投资。

## 详细说明
- 向上社交有难度：发很多邀请往往只有几个回复；平级社交（跟资历差不多的人）回复率高得多。
- 小Lin 在投行工作时老板的建议：平时多跟别的组的 Analyst、Associate（同级别的人）走动。好处一是工作交流更顺畅，二是你们会一起往上升，未来他们会成为各大投行、各个组的骨干——前提是你自己也要往上升——届时资源就非常广。
