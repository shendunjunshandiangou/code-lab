# 可口可乐二战军供爱国营销

## 一句话定义
二战限糖令重创糖水厂商之际，可口可乐董事长罗伯特·伍德拉夫借垄断地位与国际产业链和美军达成战略合作——"不管军队在世界哪个角落、成本多高，保证美国大兵 5 分钱买到一瓶可口可乐"——二战期间共售出 100 亿瓶（一半供军队），把品牌与爱国、传统、家庭死死绑定，培养出一代终身死忠粉，为此后几十年的品牌营销打下地基。

## 详细说明
- **危转机**：1941 年美国参战后糖价飞涨、民用糖受限，卖糖水的本该最惨；伍德拉夫在艾森豪威尔的要求与支持下四处建厂，每天上千万瓶送往世界各地美军。
- **心智效应**：战场上的士兵喝到"小红瓶"就有回家的感觉——情感锚点比销量更值钱；战后这批人成为终身用户。
- **战果**：1948 年可口可乐总利润 12.6 亿美元，受限糖令拖累的百事仅 2600 万美元、不及零头。
- **方法论**：借国家叙事做品牌（爱国营销）+ 用渠道跟着军队全球建厂（顺带完成国际化布点）——一次合作同时完成心智占领与产能出海。
