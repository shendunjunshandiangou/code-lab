# Shein 小单快反模式

## 一句话定义

Shein 把供应链反馈速度做到极致的"小单快反"：社交媒体内容判断流行趋势 → 设计巨量新款 → 每款先只做一两百件（小单）→ 全部上架电商平台测试真实用户反馈 → 受欢迎的立刻量产（快反），一天可上千个新款，快速席卷全球。

## 详细说明

- 快时尚是把供应链反馈发挥到极致的行业：苹果一年一更新，Zara、H&M 靠持续飞速上新崛起，恨不得几天上百个新品——核心是最快知道哪些品受欢迎并量产跟上。
- 本质是用真实销售数据替代需求预测，规避了耐克式 ERP 预测失灵的风险。
- 代价：极致的快带来环保、质量把控等一系列问题，Shein 为上市正在设法解决。
