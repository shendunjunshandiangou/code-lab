# 人均GDP六种口径与调整

## 一句话定义

同一国人均 GDP 可因是否调整通胀、是否换美元、是否用购买力平价而产生截然不同的走势图——伊朗等国六张图即典型。

## 详细说明

每年 GDP 要跨国、跨年对比，须经不同调整 → 不同调整 = 不同走势（发展中国家尤甚）。

**三个调整维度**：统一汇率？统一购买力？统一通胀？

**伊朗六指标示例**：

| 口径 | 调整 |
| --- | --- |
| 现价本币 | 无 |
| 不变价本币 | GDP 平减指数剔本国通胀 |
| 现价美元 | 当年汇率换美元 |
| 不变价美元 | 再剔美元通胀 |
| PPP 人均 | 购买力平价 |
| PPP 不变价 | PPP + 剔通胀 |

**GDP 平减指数**：衡量整个经济活动平均物价变化（不同于消费者一揽子 CPI）。

**统计误差悖论**：换算步骤越多、指标越「完美」，人工估算越多，**误差可能越大**——尤其伊朗等数据不足、官方汇率与黑市汇率差别大的国家。

**日常折中**：多用**当年本币 GDP 按当年汇率换美元**——调整少、可横向对比。
