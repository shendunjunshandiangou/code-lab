# 麦道夫造假应付监管

## 一句话定义

麦道夫为应付 SEC 突袭检查，开发虚假交易软件、雇人假交易、伪造报表，甚至用门口冰柜给打印报表降温——好莱坞级别的造假手段配合监管走过场，使泡沫持续酝酿。

## 详细说明

### 造假手段清单

| 手段 | 细节 |
| --- | --- |
| 虚假交易软件 | 开发一套假系统，雇人在门口假模假式交易 |
| Bloomberg 伪装 | 检查时打开 Bloomberg 窗口，假装下单成交 |
| 虚假报表系统 | 专门开发假报表生成系统 |
| 冰柜降温 | 门口放冰柜；突袭检查时把打印报表放进去降温，不易被发现 |

### 配合因素

- SEC 检查多为**突袭检查**——麦道夫针对性设计应对流程
- 官方**睁一只眼闭一只眼** + 走过场式走访
- 麦道夫本人心理素质极好，检查时自己也吓出一身冷汗，但表面镇定

### 说辞掩护

面对质疑「如何做到 15% 稳赚不赔」，麦道夫称利用**期权定价偏差**进行**无风险套利**——衍生品定价误差中攫取利益。类似策略有量化对冲基金在研究，但无人能达到麦道夫收益水平。
