# 重复博弈变一次博弈极端化

## 一句话定义

长期做生意是**重复博弈**——不能做得太过、要考虑对方感受、要看未来；而一旦一方挑明关系有明确终点（欧盟宣布 2027 年后彻底不需要俄气），博弈就退化为**一次博弈**：反正之后就结束了，此前能多抠一个子儿都是赚、不必再顾忌面子和未来——于是双方政策同时极端化：俄罗斯找理由限制供给（涡轮坏了）、欧盟想方设法压价（限价令），双方基本无视之前签好的合同。

## 详细说明

- 博弈论经典结论的现实版：重复博弈中合作可以自发维持（背叛会被未来惩罚），一次博弈中背叛是占优策略——给合作关系设定「死期」等于亲手拆掉合作的激励基础。
- 应用启示：分析商业或国际关系冲突时，先判断双方处在重复博弈还是一次博弈——关系预期的改变往往比单次利益冲突更能解释行为突变。
- 本案的第三方搅局：OPEC+ 在双方对峙时减产 200 万桶/天火上浇油——一次博弈的混乱局面对旁观者可能是「千载难逢的赚钱时机」。
