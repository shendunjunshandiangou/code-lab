# 普京时代经济复苏

## 一句话定义

1999 年后近 10 年俄罗斯 GDP 增速超 5%、人均 GDP 从不足 2000 美元涨至 2008 年 1 万美元——除降税减监管与卢布贬值外，小 Lin 认为最主要原因是赶上油价上涨的黄金时期。

## 详细说明

| 指标 | 变化 |
| --- | --- |
| 人均 GDP | &lt;2000 美元 → 2008 年 1 万美元 |
| 失业率 | 13% → 6% |
| 工业生产 | 增 75% |
| 平均工资 | 翻八倍 |
| 贫困人口 | 30% → 14% |

**原因**：初期市场化政策（降税、减监管）+ 1998 危机后卢布贬值提升出口竞争力 + **油价从 1998 年 &lt;15 美元涨至 2008 年 >100 美元**（核心）。

但未能利用高油价期优化结构，反而进一步巩固能源依赖与寡头格局。
