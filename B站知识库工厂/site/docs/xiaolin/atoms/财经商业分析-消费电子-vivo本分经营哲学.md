# VIVO 本分经营哲学

## 一句话定义

VIVO 源自步步高段永平「本分」思想——平常心、找对的事并坚持做对——演化为长期埋头种因、合作共赢的经营哲学。

## 详细说明

- 渠道：先零售商、再省代、最后总公司赚钱
- 危机：功能机积压仍认订单、补偿代理
- 产品：不盲目跟风互联网模式，HiFi→影像→屏下指纹
- 四个 Happy：用户、员工、伙伴、股东
