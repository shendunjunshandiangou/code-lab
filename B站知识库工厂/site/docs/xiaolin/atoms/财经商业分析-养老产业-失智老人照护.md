# 失智老人照护

## 一句话定义
中型养老院中阿尔茨海默症（失智）老人占比可超过一半，远高于大众想象；照护要点是顺应情绪、帮助家属接受病情、防走失。

## 详细说明
- **比例**：顺义某民办养老院院长称，入住老人超过一半有不同程度阿尔茨海默症。
- **行为特征**：情绪不稳定（突然高兴/生气）、表达不清、常见被害妄想（怀疑别人偷东西）。
- **照护方法**：
  1. **顺着说**：对被害妄想回应「行，我晚上就报警」，老人便把你当同盟；对抗只会激化。
  2. **家属沟通**：家属常拒绝承认老人失智（把反常当「变了性格」），需要日常持续沟通，家属自行查证后接受——一旦认可，后续照护配合度大增。
  3. **防走失**：失智老人爱走、爱跟人走且外表看不出患病；在老人背后贴联系电话标牌，靠这个低成本手段真正找回过多名走失老人。
- 专业启示：需要有经验的专业人员，「跟照顾小孩一样」需要耐心与情感投入。
