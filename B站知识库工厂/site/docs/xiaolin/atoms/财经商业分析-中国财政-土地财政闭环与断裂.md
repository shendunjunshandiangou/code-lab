# 土地财政闭环与断裂

## 一句话定义

土地财政的闭环：地方政府 KPI 是 GDP 但收入不够用 → 2014 年新预算法前不许自己借债，遂把土地相关权益抵押给城投公司融资 → 借钱修路搞基建（想致富先修路）→ 人和企业进来、经济活力起来、GDP 和人口双升 → 税收变高 + **地价上涨带动土地出让金水涨船高** → 拨款给城投还债 → 开启新一轮融资投资；正常运转时非常漂亮合理（房地产因此被称为经济发动机），但也让地方积累了大量表外隐性债务，而当土地出让金从 2021 年 8.7 万亿跌到 2024 年估算 4.5 万亿（近乎腰斩），闭环断裂、地方陷入债务泥潭，才有了大规模化债。

## 详细说明

- 评判负债健康的标准不是多少，而是**负债对应的投资能否产生超过利息的回报**——2000 年代初中国飞速发展期搞基建「怎么投怎么有」，显然还得起；近年疫情冲击、经济震荡、房价下跌让项目投资回报率出问题，逻辑才逆转。很多发达健康经济体负债率都很高，欠债少 ≠ 健康。
- 政企合作模式五花八门（BT、BOT、BOO、TOT、PPP 等），核心都是政府希望**用尽量少的债务撬动尽量大的项目**。
- 隐性债务始于 90 年代税改，急速膨胀于 2008 年 4 万亿刺激之后。
- 根本出路（讲者结语）：土地财政减弱背景下，能否推进城投平台公司改革、找到可持续发展模式，把更多资源实打实投入民生。
