# Tiffany 疫情压价收购案

## 一句话定义

2019 年阿尔诺与 Tiffany 敲定收购后，借 2020 年疫情股价大跌与美国对法品牌加征关税为由压价毁约打官司，最终降价完成收购——展现其六亲不认的谈判风格。

## 详细说明

### 背景

- 2019 年阿尔诺对 Tiffany 埋伏已久，谈判敲定收购意向

### 压价博弈

- 2020 年初疫情爆发，奢侈品市场受挫，Tiffany 股价大跌
- 2020 年 7 月美国宣布对法国品牌加收关税，钻石价值预期下降
- 阿尔诺要求按新价格买，否则不买了
- 双方打官司，两败俱伤

### 结局

- Tiffany 让步，阿尔诺压低价格完成收购
- 再次扩大 LVMH 奢侈品版图
