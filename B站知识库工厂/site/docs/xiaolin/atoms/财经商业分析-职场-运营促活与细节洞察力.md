# 运营：促活与细节洞察力

## 一句话定义

运营的本质是"运营和维护人（用户）"，通过内容与活动不断促活、刺激变现；最适合的人除了熟悉业务、沟通强，还要极其细心——干活不出错，且能从小细节发现苗头。

## 详细说明

**工作内容**：活动策划、内容运营、用户管理都算运营——读书 APP 搞活动刺激买会员、外卖 APP 推当季新品刺激消费、B 站毕业歌会留住用户。

**人的画像**：

- 熟悉业务、熟悉产品；
- 沟通能力强（常与用户打交道）；
- 非常细心：第一干活不能错；第二能从细节发现苗头——例如从朋友圈谁给谁点赞看出关系微妙，就是好的洞察力；
- 性格天马行空但常犯错的人不适合。

**行情**：大厂方方面面都要运营、需求量大；薪资约 10–15K，略低于市场和产品；专业不限。
