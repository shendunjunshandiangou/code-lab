# 居家、社区、机构养老格局

## 一句话定义
中国目前 90% 以上老人居家养老（靠子女）、约 7% 依托社区、仅 3% 机构养老，且机构养老偏高端化，普惠化仍有很长的路；趋势是走向商业化、社会化。

## 详细说明
- 现有机构养老门槛高：保险公司、地产公司建的养老院动辄需缴近百万保费才有入住资格，相当于二三线城市一套房。
- 日本经验（老龄化最深的国家）：公私立并存的完整体系——轻度照护（做饭、集体娱乐）便宜且覆盖率高；重度护理需排队申请、公立养老院月费也合人民币六七千。即便如此，日本最普遍的仍是居家养老，甚至有老人去泰国、马来西亚等低成本国家养老。
- 国内试点：北京西城区由社区中心专门服务，养老院配云系统（老人身体信息、健康状况、看病记录）与辅助监测（摔倒、走失即刻通知家人）。
- 行业最贵的不是设备而是**人**：护理人员在包括中国在内的大部分国家严重供不应求，急需专业培训与监管；日本至今短缺，甚至出现身体较好的老人照顾另一位老人的情况。
