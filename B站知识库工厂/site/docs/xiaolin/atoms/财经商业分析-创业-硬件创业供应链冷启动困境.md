# 硬件创业供应链冷启动困境

## 一句话定义
硬件初创公司找代工的死结：工厂按订单量接活（开口就问「要几千台」），而新品类小团队既给不出量、又没有过往产品背书，被视为不靠谱——只能自建产线硬扛。

## 详细说明
- 与软件创业不同，硬件必须过「样机→量产」两道坎：样机可以靠团队死磕（Insta360 早期 demo 经常坏，出门带螺丝刀/电烙铁/热熔胶随时维修），量产却必须依赖供应商。
- 供应商筛选客户的两个标准恰好卡死新创团队：①订单规模（新品类没有销量基础）；②信誉背书（一帮 90 后、零产品历史）。
- Insta360 的无奈解法：把公司各岗位员工（包括 UI 设计师）都拉上生产线拧螺丝；设计师因此认定公司不靠谱而辞职。
- 破局点在于融资与订单起量后供应链态度反转：CES 参展获经销商关注+新一轮融资后团队破百人、专业化分工，「小作坊时代」才结束。
