# 关键点位心理效应

## 一句话定义

技术分析"看似有效"很大程度源于市场心理的自我实现：股价多次冲不过某关键点位后，大家形成"过不去"的共识并集体卖出，使其真的过不去；一旦突破，共识反转引发追涨。

## 详细说明

例子：股价多次冲击 3000 点回落后，市场形成"3000 是坎"的预期——每次接近 3000 大家都卖，于是真的过不去；可一旦突破 3000，原来认为过不去的人反手买入，价格急速上涨。若此时你按旧规律卖出，会亏得很惨。

**含义**：在关键点位附近，价格/收益率分布会偏离正常形态，出现异常；过度依赖历史规律要承担更大风险。价格最终仍归结于供需与价值。
