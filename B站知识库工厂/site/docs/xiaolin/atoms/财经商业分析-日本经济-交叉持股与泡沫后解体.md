# 交叉持股与泡沫后解体

## 一句话定义

泡沫破裂后银行坏账+合并、2001 持股限制法+外资施压，交叉持股从 1990 年代 30%+ 降至 2017 年 10% 以下，财团资本绑定减弱。

## 详细说明

- 1997 保险/证券破产；三菱/三井/住友系银行合并重组。
- 监管要求资本充足率 → 银行变卖持股；外资股东看回报、施压减交叉持股。
- 长期：抑制创新（人才流向大集团），但给软银/优衣库等非财团留出空间。
