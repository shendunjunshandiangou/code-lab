# 任务确认与 On the Same Page

## 一句话定义

老板布置任务时，别只说「好的老板」——用具体问题对齐需求、维度和时间节点，达成 on the same page（同一页上、达成共识），避免拍在马腿上白干。

## 详细说明

**反面案例（拍在马腿上）：**

老板让做「奶茶产品分析」，员工不敢问细节，两天后交出化学成分分析，老板其实要的是竞品分析——相当于老板饿了半天等你上菜，上来的是表演节目。耽误进度还得回头重干。

**老板视角：**

- 大多数老板很忙，没时间把所有细节一个一个讲明白
- 布置任务时，**多数是在等你去问问题**
- 有些老板会主动问「你有没有问题」，但更多老板心想：你说没问题，我就等着看了

**正确策略：**

当老板给你布置任务，有问题千万别只说「好的老板」，要用问题体现求知进取和对项目的重视。

**示范对话要点：**

1. 确认对象：「您是要分析最近推出的新款珍珠奶茶吗？」
2. 确认维度：「从受众口味、价格、珍珠弹性等维度，和 A/B/C/D/E 竞品对比，您觉得可以吗？」
3. 确认时间：「周末之前给您，您看可以吗？」

这就是 **on the same page**——跟老板在同一页上、对需求达成共识。
