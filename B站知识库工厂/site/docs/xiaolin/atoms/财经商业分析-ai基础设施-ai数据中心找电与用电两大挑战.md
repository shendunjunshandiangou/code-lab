# AI数据中心找电与用电两大挑战

## 一句话定义

AI 数据中心在电力上须同时解决「找电」（拿到足够稳定电源）和「用电」（高效稳定送达 GPU），中美电网条件差异导致美国大厂不得不「自己发电、上天入地」。

## 详细说明

**两大挑战**：
1. **找电**：能否拿到足够电量
2. **用电**：如何高效稳定送给服务器

**中国**：电网相对稳定；「东数西算」顶层设计，八个节点骨干网，总量角度相对够用。

**美国**：
- 电网老旧，按州管理
- 几吉瓦数据中心=中小城市，居民反对、并网审批排队数年
- 大厂不得不自建电源

**美国找电主要路线**：

| 路线 | 特点 |
| --- | --- |
| 核电 | 微软重启三里岛（20 年购电协议、2028 点火）；稳定零碳，但新建约 5～10 年 |
| 天然气 | 马斯克田纳西燃气轮机；快（数月可用），但有排放 |
| 绿电+储能 | 谷歌、亚马逊大规模买；不稳定 |
| 出海 | 阿联酋、西班牙、泰国等监管较松地区 |
| 太空 | 马斯克、谷歌设想中 |

**用电侧**：供电（进线→UPS→馈线）+ 散热（液冷/TMU），须保证毫秒级不断电。
