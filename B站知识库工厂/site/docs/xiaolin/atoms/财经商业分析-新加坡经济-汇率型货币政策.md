# 新加坡汇率型货币政策

## 一句话定义
新加坡金管局（MAS）不调控利率而直接管理汇率：新币贬得太厉害就动用外汇储备买入、涨得太厉害就卖出——因为其货币主要跨境流动，控制汇率比控制利率更能直接对付输入型通胀，但前提是央行必须极其有钱。

## 详细说明
- 逻辑：央行首要目标是控物价。多数央行用利率，因为利率控制的是资金在国内的流通速度（水龙头大小）；而新加坡的货币主要在本国与海外之间流动——海外物价上涨会直接传导进来，最直接的对策是**让新币升值**，进口就没那么贵了。利率与货币供应量（M1、M3）则交给市场自由浮动。
- 与港币的区别：不与美元硬绑定，新币兑美元近年在 0.69–0.76 区间浮动——既稳定又保留调控空间。
- 稳定的价值：贸易与投资伙伴最怕的不是升值或贬值，而是剧烈波动——汇率稳定才敢把钱和供应链放进来（对照日本央行在日元暴跌时被迫干预）。
- 苛刻门槛：让本币贬值可以印钱去卖；让本币升值必须动用美元欧元等外汇储备买本币——外币印不出来。需要巨额外储，这一条排除了 99% 的国家。
