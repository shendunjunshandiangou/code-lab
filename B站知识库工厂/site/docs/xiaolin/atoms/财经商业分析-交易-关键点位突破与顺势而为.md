# 关键点位突破与顺势而为

## 一句话定义
利弗莫尔的核心策略升级：股价不是完全随机游走，存在关键点位（现代所称压力位/支撑位）——短期在区域内震荡，一旦突破往往沿突破方向持续运动；与其在区间里赚薄利，不如等待突破后顺势重仓（如跌破 80 至 78 时大量做空），放长线钓大鱼。

## 详细说明
- 策略演变背景：他早年的区间低买高卖在赌场有效、在真实股市失灵，原因有二——①真实市场价格变化快，报价与成交之间存在滞后；②实际交易成本更高、流动性时好时坏。结论：短线高频在（当时的）真实股市行不通。
- 突破策略要点：区间内波动油水有限且噪音大；突破关键点位是趋势确认信号，此时入场时效性要求低、盈亏比高。通俗说法接近「追涨杀跌」，但完整体系远比口号复杂。
- 顺势而为的极致运用：1907 年挤兑危机中顺势做空（「你快不行了我再推你一把」）大赚；JP Morgan 劝停后又顺势反手做多，一天赚 100 万美元。
- 时代局限提醒：这类规律在早期不成熟市场才有肥厚利润；现代市场机构/央行/投行/保险/量化/高频/散户混杂，机会稍纵即逝。另见关键点位的心理学解释——技术位「有效」很大程度是共识的自我实现。
