# 利率对房价黄金汇率影响

## 一句话定义

利率升高理论上利空房价与黄金（借贷/机会成本上升），但在通胀环境下资金可能流向房地产与黄金抗通胀；对汇率则主要看相对利差——加息方货币有升值压力。

## 详细说明

### 房价

- **利空**：利率升高 → 借贷成本升高
- **利好**：高利率通常在通胀期；若通胀持续、股市跌，资金可能流向房地产
- 影响**不确定**，与股市类似存在矛盾方向

### 黄金

- **利空**：高利率 → 债券收益率增加 → 资金可能卖黄金买债券；过去几十年黄金与国债有明显负相关
- **利好**：美元通胀环境下，大家买黄金抗通胀
- 影响同样**不确定**

### 汇率

- 利率 = 投资该国货币的**回报率**
- 美联储加息 → 美元回报率变高 → 美元有升值压力
- 关键看**相对利率（利差）**：美元加息 + 人民币降息 → 利差拉大 → 美元相对人民币有升值压力
- 利率是不小因素，但通胀、贸易顺差、市场信心、政府干预等影响因素非常多
