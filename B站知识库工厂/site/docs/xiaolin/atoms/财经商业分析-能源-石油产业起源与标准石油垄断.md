# 石油产业起源与标准石油垄断

## 一句话定义
1859 年德雷克在宾夕法尼亚钻出第一口油井（威士忌桶成为计量单位「桶」，约 159 升），1900 年前美国占全球产量六七成、产业链被洛克菲勒标准石油垄断，1911 年被最高法院拆成 34 家公司。

## 详细说明
- 1859-08-27 美国商人埃德温·德雷克（Edwin Drake）在宾夕法尼亚州首次成功钻出石油，用威士忌桶盛装——Barrel 从此成为石油最重要的计量单位（1 桶≈159 升）。
- 石油燃烧效率远高于煤、管道运输方便，产量从 1860 年 50 万桶 → 1870 年 2000 万桶 → 1920 年 4.5 亿桶/年。
- 洛克菲勒的标准石油不惜重金收购上下游、垄断全产业链以控制油价，造就当代最富有的人；1911 年美国最高法院裁定拆分为 34 家公司。
- 按当时美国开采量估算，美国石油十年将采光，引发全球找油潮：墨西哥、伊朗、委内瑞拉、伊拉克、科威特相继发现大油田；1938 年美国商人在沙特发现当时已知最大石油区。
- 术语：地下挖出的是原油（Crude Oil），加工后才是石油。
