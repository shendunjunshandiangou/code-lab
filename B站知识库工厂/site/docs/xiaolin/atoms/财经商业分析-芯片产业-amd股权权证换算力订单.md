# AMD股权权证换算力订单

## 一句话定义

AMD 与 OpenAI、Meta 签订「算力换股权」协议：提供数吉瓦算力，同时发行低价 warrant 让对方可购入约 10% 股份——上大模型中心牌桌不得不「出血」的典型资本玩法。

## 详细说明

**OpenAI × AMD**（2024 年底）：
- AMD 未来几年提供总计**六几吉瓦**算力
- 须专门为 OpenAI 发行最多 **1.6 亿股 warrant 权证**，行权价 **0.01 美元/股**
- 相当于 OpenAI 可一分钱一股购买 AMD 股票——条款复杂，非白送
- 可理解成 AMD 牺牲约 **10% 股权**换 OpenAI 大订单，成本很高

**Meta × AMD**（2025 年 2 月）：
- 几乎复刻 OpenAI 协议：六几吉瓦算力换 **10% 股权**

**背景**：要上 AI 资本圈中心牌桌，芯片厂商不得不让利；OpenAI 同时与英伟达战略合作、跟博通自研芯片，不想吊死一棵树上。
