# 麦道夫障眼法与信用背书

## 一句话定义

麦道夫庞氏骗局能维持 20 年的关键「障眼法」：用合法做市商威望、纳斯达克地位、慈善形象和正儿八经证券公司背书转移注意力，使所有人默认「麦道夫不可能骗人」。

## 详细说明

讲者将庞氏骗局比作魔术——原理简单，关键在于**障眼法**（转移注意力）。

### 障眼法构成

| 元素 | 作用 |
| --- | --- |
| 合法证券公司 | 做市商业务真实运营 30 年 |
| 纳斯达克先驱地位 | 行业技术领袖、最大做市商之一 |
| 政府与行业头衔 | 半数以上来自官方机构 |
| 慈善活动 | 伟光正正面形象 |
| 社交崇拜 | 亲友「像被施了魔法」般崇拜 |

### 与骗局的关系

视频前半段大量讲述麦道夫正面履历——**与庞氏骗局「半毛钱关系都没有」**，但这恰恰是骗局规模能做到 650 亿美元的重要原因。所有人都会觉得：麦道夫先生怎么会骗人？根本不会有任何怀疑。

### 资产管理业务的隐蔽性

70 年代起，在正常做市商业务之外偷偷开展资产管理（类基金业务），与公开合法业务形成「一明一暗」结构。
