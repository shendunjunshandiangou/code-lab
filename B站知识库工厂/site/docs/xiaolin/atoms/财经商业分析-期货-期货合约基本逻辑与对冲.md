# 期货合约基本逻辑与对冲

## 一句话定义

期货约定未来以约定价格交易商品；生产商可做空、采购商可做多对冲价格风险；绝大多数交易者到期前平仓，实物交割极少。

## 详细说明

- 可紧密追踪商品价格而无需持有实物（如黄金期货）
- 原油期货市场实物交割不到 1%
- 08 年后大宗商品投机沉寂，2024 年可可豆期货因供给冲击暴涨 200%
