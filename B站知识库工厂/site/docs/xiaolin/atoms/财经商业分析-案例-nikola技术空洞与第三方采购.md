# Nikola 技术空洞与第三方采购

## 一句话定义

Nikola 自称前沿高科技企业，实际几乎无自研——核心技术从第三方购买，与氢气相关专利仅两个；Milton 同时虚构太阳能板、天然气压缩机、交换机技术等资产，靠酷炫产品线（皮卡、水上摩托、军用越野车等）吸引投资人。

## 详细说明

### 技术谎言清单

| 声称 | 实际 |
| --- | --- |
| 自有太阳能板 | 没有 |
| 自有天然气压缩机 | 没有 |
| 自有交换机技术 | 是别家的 |

### 研发与专利

- 大多数技术从**第三方购买**
- 查 Nikola 专利：与氢气相关的仅**两个氢能源电池充电专利**
- 核心技术「要啥没啥」

### 营销产品线

Milton 极善市场营销，推出一系列酷炫产品：

- 内裤、皮卡、水上摩托、军用越野车等
- 「特斯拉可没有水上摩托」——投资人特别吃这套
