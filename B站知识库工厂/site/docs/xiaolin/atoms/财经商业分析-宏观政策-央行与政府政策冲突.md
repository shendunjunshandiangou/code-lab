# 央行与政府政策冲突

## 一句话定义
央行首要目标是物价稳定、政府天然倾向刺激经济，在经济萎靡叠加高通胀时两者政策容易「打架」，其根源是选举制度下政府的短期激励。

## 详细说明
- 本质是**经济增长与物价稳定之间的抉择**：央行的首要目的永远是控制通胀，倾向牺牲经济增长换物价稳定（当前共识：长期看物价稳定更重要）。
- 政府总是更希望刺激经济，甚至宁愿冒短期通胀恶化的风险——因为大部分国家的政治体制尤其是选举制度，使政府选择大众短期内更希望看到的刺激政策。
- 案例：2022 年英国央行连续 7 次加息抗通胀，特拉斯政府却推出 1000 多亿英镑的财政刺激（迷你预算案），与央行拧着干，最终引发股债汇三杀。
- 欧元区更典型：欧洲央行政策各国政府无法干涉，只能各自财政刺激；美国政府与美联储也未必一条心。
