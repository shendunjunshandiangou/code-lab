# 瑞信逃税定罪与 2014 转折

## 一句话定义

2014 年瑞信因帮美国公民逃税被定罪并罚 26 亿美元（同业最重），同年瑞士签 FATCA，标志瑞信「融化」开始。

## 详细说明

2000 年初国际组织对瑞信逃税等业务展开调查。2014 年 5 月定罪成立，罚 **26 亿美元**（瑞银、法兴等亦被罚但未定罪、力度较轻）。

讽刺的是：股价罚后未跌反微涨——市场早知逃税存在，认为「总算罚完、可接受」。同年瑞士签 **FATCA**，打破联邦银行法保密传统。
