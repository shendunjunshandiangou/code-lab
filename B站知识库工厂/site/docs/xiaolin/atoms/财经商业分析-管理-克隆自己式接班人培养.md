# 克隆自己式接班人培养

## 一句话定义

创始人对核心能力（如 MrBeast 的内容判断）不放心外包时，MrBeast 的解法是「**克隆自己**」：找到认识的最有干劲的人（Tyler），让他**搬来同住两年、7×24 跟随**、旁听每一次谈话，再逐步移交责任——并肩工作四年后，同一个提案问 Tyler 和问 Jimmy，99% 的情况下答案一模一样；「你花大量到荒谬的时间和一个人一起做你想让他做的工作，直到他真的像你一样思考」。

## 详细说明

- 适用场景：创始人即产品灵魂（「这是我的孩子，我为内容而活」）、核心判断难以文档化时，用超高强度的贴身传承代替流程化交接。
- 本质是把隐性知识（品味、直觉、权衡）通过长期共同决策转移，而非培训技能；目标明确——「让他最终在写视频这件事上取代我」，且成功了（Tyler 已在公司近七年）。
- 代价与前提：在单个人身上投入巨大，依赖对方的长期承诺——与其选人标准（要干十年的人）互为因果。
