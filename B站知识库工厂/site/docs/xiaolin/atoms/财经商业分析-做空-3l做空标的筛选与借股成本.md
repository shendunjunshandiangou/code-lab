# 3L 做空标的筛选与借股成本

## 一句话定义
浑水创始人的选靶标准 3L——Large（够大才赚得到钱）、Liquid（流动性好才借得起股票）、Lying（存在欺诈）——三者互相冲突：越大越流动的公司越不容易造假，故调查式做空的标的极其有限。

## 详细说明
- **Large**：体量够大，做空才值回研究成本（否则忙几个月赚三五万）。
- **Liquid**：流动性决定借股成本（由借股供需决定）：
  | 股票 | 借股年化利率 |
  | --- | --- |
  | 苹果（长期有人持有） | ≈0.25% |
  | 特斯拉（做空需求高） | ≈2.5% |
  | GME（2021 年初逼空期） | ≈80%，偶尔超 100%（跌到零也不赚钱） |
- **Lying**：要做全世界第一个发现问题的人，且随做空/调查机构增多难度递增。
- 三者冲突 → 调查式做空机构体量小（浑水 2.6 亿美元/8 人）、案例有限；美股仅 3%–5% 股票被做空且多为多空策略。
