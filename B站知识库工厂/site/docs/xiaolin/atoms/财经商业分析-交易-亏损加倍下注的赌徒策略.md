# 亏损加倍下注的赌徒策略

## 一句话定义
「亏了就加大赌注赌回来，总有一天能赌赢」的策略（double down / 类马丁格尔逻辑）在有无限资金时数学上似乎成立，但现实中资金有限，一次极端行情就会导致满盘皆输。

## 详细说明
里森版本的策略循环：

1. 交易出现亏损 → 藏进幽灵账户；
2. 冲进市场加大头寸豪赌，赌赢则亏损填平；
3. 赌输则向总部要更多保证金，加更大杠杆再赌；
4. 「总有一天能赌赢，最后总能把钱赚回来。」

为什么衍生品市场特别适合这种「赌法」：期货/期权杠杆可以做得很大、灵活性高、交易不对外公开，只要有保证金就能不断加注。

心理机制（讲者归纳为「人性」）：
- **知道背后有保险柜**（能藏亏损），决策就大手大脚；
- **赌输了**想翻本；**赌赢了**更难收手——里森 9 个月惊险填平 400 万英镑后当晚发誓收手，一周后就因「我赚回 400 万为什么没人看到」的不甘心重新下场。

结果：2 万英镑 → 120 万 → 400 万 → 2300 万 → 1.6 亿 → 4 亿英镑，每一步看似「利益最大化」，实则一步错步步错。
