# 美元 50 年走势复盘

## 一句话定义
1971 年脱离布雷顿森林体系后，美元半个世纪的涨跌可以用四大因素（通胀、利率、国际收支、风险/避险）串起来解读：1985 年"高利率+通胀回落+拉美危机避险"三重叠加冲上历史高点，广场协议掉头向下；此后每轮大波动几乎都是这几股力量的主次更替。

## 详细说明
- **1971-1979 跌**：脱钩布雷顿森林体系（美元还是不是世界货币？——风险）+ 石油危机滞胀，一路下行。
- **1980-1985 涨至历史顶**：沃尔克把利率拉到 20% 压通胀 + 拉美债务危机避险资金涌入。
- **1985-1990s 初跌**：美元过强伤贸易（对日逆差恶化）→ 广场协议五国联手抛美元；随后降息 + 1987 股灾。
- **1994-95 背离案例**：墨西哥债务危机全球避险 + 美国在加息，美元按理该涨却跌——因为危机正是美元资本乱搞造成、美国指标又差，**这轮避险避的就是美元**，资金转向日元与德国马克。
- **1995-2002 八年上涨**：互联网投资机会虹吸资本 + 亚洲/俄罗斯金融危机避险，双轮驱动。
- **2000s 阴跌**：互联网泡沫破裂、中国入世崛起、欧元登场，美国经常账户严重逆差——持续加息也压不住；2007 年降息 + 信用风险，美元至历史低位。
- **2008 反直觉上涨**：次贷危机源自美国、美联储大降息，但全球恐慌避险仍买美元。之后低利率 + QE，低位震荡。
- **2015 涨**：缩表退出 QE、开启加息预期。
- **2022 快涨**：全球通胀、美联储加息超 5% + 逆全球化地缘风险避险。
- **无法解释的部分**：2014 年底美元经历半个世纪最快大涨，当时未加息、利率在跌——硬解释（基本面强、日欧陷通缩债务泥潭、加息预期）说得通，但"为何突然猛涨"没人真知道。汇率分析的本质是判断多因素中谁主谁次，专业分析师也只能猜个大概。
