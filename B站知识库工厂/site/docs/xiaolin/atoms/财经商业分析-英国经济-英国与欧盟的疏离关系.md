# 英国与欧盟的疏离关系

## 一句话定义
英国入欧曲折（拒绝首邀、被戴高乐两次否决、1973 年才加入），入欧后又处处搞特殊（1975 公投闹退、拒用欧元、不满捕鱼配额与会费），始终是欧盟的边缘游离成员。

## 详细说明
- 欧共体初建时曾邀请英国，英国因产业国有化的保守路线和亲美立场拒绝；后悔再申请时，法国总统戴高乐嫌其亲美，1963、1967 两次否决，连谈判都拒绝；1969 年戴高乐下台后英国才于 1973 年加入。
- 入欧后的疏离表现：地理上隔英吉利海峡；经济纠纷不断（英吉利海峡的鱼被限制只能捕 1/5、还要交会费）；1975 年刚加入就公投闹退出；德法推欧元时英国拒绝加入、坚持用英镑。
- 英国人的傲气加上欧盟规则约束，使双方都没把英国当核心成员看待；经济好时相安无事，2010 年后欧债危机让不满情绪上头。
- 直接导火索：2015 年叙利亚难民潮。默克尔牵头按人口比例接纳上百万难民，大量难民涌入英法德，影响英国社会安定与就业，法国又发生恐袭，英国民众对欧盟产生强烈抵触。背景数据：1998 年起英国每年净流入几十万人，人口密度 277 人/平方公里（美国仅 34），移民问题长期是英国抵触欧洲的痛点。
