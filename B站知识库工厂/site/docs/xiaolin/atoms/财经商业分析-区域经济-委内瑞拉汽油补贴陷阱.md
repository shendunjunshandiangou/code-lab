# 委内瑞拉汽油补贴陷阱

## 一句话定义

政府持续砸钱维持近乎免费汽油，全国排队加油、年走私超 100 亿美元；马杜罗不敢取消——涨价或引发政权危机。

## 详细说明

- 设备老化、精炼能力弱，汽油仍靠进口；补贴将成本从金钱转化为排队时间。
- 市场化按国际油价收费的加油站存在但不排队，多数民众仍选免费站。
- 「天下没有免费午餐」——补贴占财政巨大份额，经济崩溃后仍硬着头皮维持。
