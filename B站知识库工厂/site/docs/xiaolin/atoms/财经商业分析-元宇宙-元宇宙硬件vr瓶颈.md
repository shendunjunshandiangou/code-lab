# 元宇宙硬件 VR 瓶颈

## 一句话定义

消费级 VR 眼镜在舒适性、便携性、刷新率、分辨率上仍不足，佩戴易晕、眼酸，难长时间使用——是元宇宙愿景与现实之间的核心障碍。

## 详细说明

- **现状**：比早年改进，但与宣传视频体验差距大
- **症状**：晕眩、眼酸 → 无法长时间佩戴 → 难以支撑「改变生活」的沉浸交互
- **讲者判断**：若设备体验接近宣传，元宇宙才真正临近
- **对比**：Roblox 等多数用户仍对屏幕，非 VR 沉浸
