# 专项再贷款与政策调控

## 一句话定义

专项再贷款是中国央行精准调控的主要工具：央行向大行提供资金但限定投向（"钱可以找我要，但必须贷到这个行业去"），在市场竞争与政策把控之间取得平衡。

## 详细说明

**背景**：中国引入银行业市场竞争的同时，保留政府与央行的精准调控能力——银行体系未完全成熟时一是更稳、二是更好把控方向。

**应用实例**：保交楼、碳减排支持工具 2000 亿、科技创新、普惠养老，以及供给侧结构性改革配套资金。

**体系含义**：这是中国"由紧放开、政策主导调控贷款"模式的核心机制，也是银行在中国金融体系中占主导地位（资产占比 85%+）的制度基础之一。
