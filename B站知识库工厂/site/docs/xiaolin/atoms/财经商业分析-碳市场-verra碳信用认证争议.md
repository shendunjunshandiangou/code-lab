# Verra 碳信用认证争议

## 一句话定义

全球最大碳信用认证机构 Verra 被调查指 90% 以上认证存疑，暴露碳抵消项目「全凭一张嘴」核算的系统性风险。

## 详细说明

- Verra 牵头制定碳汇算体系，为 GUCCI、壳牌、欧莱雅、麦肯锡等认证减排量
- 企业用认证抵消自身排放，宣传「零碳产品」
- 2023 年初媒体与科学家调查：大量项目如「圈定本已存在的森林称阻止砍树」、种植吸收量夸大
- 批评：美国碳交易市场猫腻的负面影响或已大于正面效果
- 讲者：中国对减排申报与碳交易持谨慎态度有其道理
