# 直接上市与借壳上市

## 一句话定义
IPO 之外的两条上市捷径：Direct Listing 不发新股不融资、不经投行直接把现有股票放上公开市场（适合 Spotify、Coinbase 等自带知名度的公司）；借壳上市（反向收购）则由私有公司收购上市公司、把业务装进去变相上市。

## 详细说明
- **Direct Listing（直接上市）**：不需要投行运作和寻找投资人，直接将现有股票放到公开市场交易；不发新股、不融资。前提是公司在大众中有影响力——「一发行大家就知道是啥」。案例：Spotify、Slack、Coinbase。
- **借壳上市（反向收购）**：私有公司收购一家（通常经营不善的）上市公司，把自己的业务装进去，变相完成上市——与常规「大上市公司收购小私有公司」方向相反。
- **与 SPAC 的区别**：SPAC 的「壳」不是真公司而是专为合并而设的空壳（见相关原子）。
