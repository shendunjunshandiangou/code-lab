# 固定赔率与庄家抽水

## 一句话定义

博彩公司在均衡赔率（庄家零利润）基础上双向下调，使隐含概率之和超过 100%，从每笔投注抽取 3%–30% 不等「水钱」。

## 详细说明

- **均衡**：100 押 A + 200 押 B，总池 300 → A 赔率 3.0、B 赔率 1.5。
- **抽水**：调至 2.9 / 1.4 等，庄家稳赚。
- **比例**：胜平负 3%–5%；实时盘/串关 25%–30%。
- **偏好**：通常冷门爆冷对庄家更有利——热门球队非理性下注多。
