# 高级拍马屁与向上管理思维

## 一句话定义

「高级拍马屁」不是低级奉承，而是一种向上管理思维：话说到老板心里、把事办到、解决老板难处，让老板知道你跟他同一战线解决问题。

## 详细说明

职场里跟老板关系处不好，常见原因不是不会「拍马屁」，而是没掌握高级拍马屁背后的思维方式。

**低级 vs 高级：**

| 维度 | 低级拍马屁 | 高级拍马屁 |
| --- | --- | --- |
| 目标 | 让老板一时爽 | 把事办到、解决老板难处 |
| 表现 | 空洞奉承 | 话说到老板心里 |
| 本质 | 话术技巧 | 思维方式 |

**向上管理核心：**

- 不是言听计从，而是**共同发展、一致对外**
- 让老板知道你跟他**同一战线解决问题**
- 老板是上级，下属辅助他做事；大家都在给组织办事
- 问题解决好老板脸上有光，解决不好最后背锅的还是老板

跟老板关系处理好不一定立马飞黄腾达，但至少每天心情愉快；关系不好则每天上班跟上刑。
