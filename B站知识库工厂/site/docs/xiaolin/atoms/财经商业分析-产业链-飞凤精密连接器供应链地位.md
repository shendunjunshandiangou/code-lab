# 飞凤精密连接器供应链地位

## 一句话定义

飞凤精密以连接器（connector）制造见长，长期为诺基亚翻盖机、MacBook 转轴及主流折叠屏铰链供货，是消费电子精密结构件供应链中的关键隐形冠军。

## 详细说明

- **历史客户**：90 年代诺基亚翻盖机转轴、苹果 MacBook 中间开合转轴等。
- **折叠屏合作**：OPPO 约四五年前入局折叠屏即与飞凤联合研发铰链，四年多迭代六代；新款折叠屏铰链仍由双方联合生产。
- **苹果动向**：消息称苹果或于 2026 秋推出折叠屏手机、2027 年推出折叠屏平板与电脑，铰链拟找飞凤合作——苹果折叠屏迟迟未大规模入局，铰链技术成熟度可能是因素之一。

飞凤在供应链中地位高，但公众知名度相对较低，属于典型「链上关键、屏外低调」的精密制造企业。
