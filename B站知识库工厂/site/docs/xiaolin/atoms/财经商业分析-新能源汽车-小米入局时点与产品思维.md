# 小米入局时点与产品思维

## 一句话定义

小米入局电动车的时点"不但不晚还挺聪明"：电动化已基本成型（可合作+差异化），智能化才刚开始且恰是小米的优势主战场；配合把产品思维用到极致——技术发布会实为收集用户反馈、真听劝、价格吊足胃口。

## 详细说明

- 判断入局时机的框架：看产业各板块所处阶段——成型板块（电池）借力龙头做差异化，早期板块（智能化）全力自研抢位。
- 雷军与顺为资本早已布局新能源/AI 赛道，投资标的不下 100 个（含蔚来、小鹏）。
- 制造端：有资金实力就跳过代工阶段直接自建厂；采用一体化大压铸（简化流程+减重），用 AI 从 1000 多万种金属组合研发"泰坦合金"，成为国内唯一自研大压铸设备集群和压铸材料的车厂（9100 吨集群）。
- 产品思维案例：技术发布会开完，夸的骂的都是用户反馈；用户嫌尾翼拼音太大，几天后就调小；价格一直不公布持续听声音（视频停在 SU7 定价悬念）。
