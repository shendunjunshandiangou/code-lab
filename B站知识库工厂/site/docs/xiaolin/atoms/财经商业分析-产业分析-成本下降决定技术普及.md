# 成本下降决定技术普及

## 一句话定义

一个技术能否商业化普及，关键不在于"造出来"，而在于成本降到大众可承受：电动车 1834 年就被发明（比燃油车还早），却直到近年成本下降才普及；AI 爆火的重要原因也是算力成本大幅下降。

## 详细说明

- 特斯拉 2006 年商业计划书四句话即是成本路线图：先造跑车（Roadster，名人客户）→ 用利润造买得起的车 → 再造更大众的车 → 提供零排放能源选择——一步一步把成本打下来。
- 冰箱、洗衣机在中国普及同理：供应链规模化带来成本下降。
- 反例的价值：洗碗机成本已降（欧美普及率 75%+、中国年产 800 多万台且技术成熟），在中国普及率却不到 2%——说明成本只是必要条件，消费认知与使用场景适配（"两怕一没有"）同样能构成阻力。
