# 奥运会三大收入结构与分配

## 一句话定义

2013–2016 周期 IOC 收入约转播 70%、赞助 20%、门票与周边 10%；转播与 TOP 赞助由 IOC 主导后拨给主办国，小项广告等归主办国。

## 详细说明

| 收入类型 | 占比（约） | 归属 |
| --- | --- | --- |
| 转播权 | ~70% | IOC 主导，分一部分给主办国 |
| 全球赞助（TOP 等） | ~20% | IOC 主导，分一部分给主办国 |
| 门票、周边 | ~10% | 奥运村内小牌、电视插播等小广告归主办国 |

IOC 策略：占大头的转播与全球赞助牢牢握住，主办国承担成本却只能从剩余与小项获益。
