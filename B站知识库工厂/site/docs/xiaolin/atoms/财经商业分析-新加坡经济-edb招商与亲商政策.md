# 新加坡 EDB 招商与亲商政策

## 一句话定义
李光耀 1961 年成立经济发展局（EDB），以"降税 + 简化流程 + 削弱工会"三板斧主动向世界 500 强招商，把新加坡打造成外资建厂的首选地，奠定制造业起飞的基础。

## 详细说明
- 背景判断：新加坡太小、资源一般，快速致富必须靠工业和外资；配套建工业园区，并成立专事工业融资的新加坡发展银行。
- EDB 打法：给 500 强写信、打电话、亲自拜访——"便宜、大港口、政府全力配合"。
- 三大诚意：
  1. **税收**：先给建厂减免，后来关税降到零；企业所得税从超 40% 一路降到 17%（可能低于外企本国税率），个税也有优惠——由此成为著名避税天堂；
  2. **流程**：推行"单一窗口"，没有反垄断、审批牌照、外汇管制——世界银行"容易经商"排名全球第二（仅次于新西兰，香港第三）；
  3. **劳工**：60 年代工会强、常罢工，出台政策削弱工会，为外企"把路铺平"。
- 效果与代价：60 年代末起跨国公司蜂拥建厂——先是服装玩具等劳动密集型，1968 年德州仪器、1970 年惠普入驻激活电子工业；制造业占 GDP 从 10%（1960）升到 25%（1980），GDP 增速超 10%、失业率几乎为零。代价是外企遍地、一定程度上削弱了本土企业。
