# Trevor Milton 与 Nikola 创始背景

## 一句话定义

Nikola 创始人 Trevor Milton（1982 年生）市场营销专业出身，创办 Nikola 前已有十几年创业经历；其创业逻辑是「只要敢忽悠，拿钱不用愁」，以 sales and marketing 而非技术驱动氢能源卡车叙事。

## 详细说明

| 要素 | 内容 |
| --- | --- |
| 人物 | Trevor Milton（特雷弗·米尔顿），1982 年生，创办 Nikola 时不到 40 岁 |
| 教育 | 犹他州州立大学，专业 Sales and Marketing（市场营销） |
| 创业特点 | 与电池/技术/续航导向的电动汽车创业者不同，主打市场营销 |
| 前史 | 创办 Nikola 前已有十几年创业经历，讲者称其为「惯骗子」（具体细节略） |
| 核心领悟 | 「只要敢忽悠，拿钱不用愁」 |
| 创立时间 | 2015 年创立 Nikola，最初想做烧天然气的卡车 |
