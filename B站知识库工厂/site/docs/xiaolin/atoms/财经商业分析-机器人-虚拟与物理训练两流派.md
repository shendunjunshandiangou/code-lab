# 虚拟与物理训练两流派

## 一句话定义

机器人泛化需海量物理数据；**虚拟仿真**（英伟达 Isaac）快但难复刻真实物理，**物理实景**（特斯拉工厂）真实但成本高——可并行推进。

## 详细说明

- LLM 有百亿级语料；机器人不可能摔 1000 亿次鸡蛋。
- 泛化能力距通用 AI 仍远：新环境无预训练即适应是最大挑战之一。
