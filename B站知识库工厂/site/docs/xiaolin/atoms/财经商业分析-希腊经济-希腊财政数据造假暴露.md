# 希腊财政数据造假暴露

## 一句话定义

2009 年起希腊政府赤字率经层层审计从自称的 6.7% 暴增至 15.4%，叠加高盛藏债丑闻，债券从 A 级（约 3%）降为垃圾债，市场信用崩塌。

## 详细说明

欧元区要求年度赤字率 &lt; GDP 的 3%。希腊 2009 年数据连环翻车：

| 阶段 | 公布的赤字率 |
| --- | --- |
| 希腊自称 | 6.7% |
| 首次调查 | 12.7% |
| 审计复查 | **15.4%** |

同时，与高盛合谋藏债等见不得人的操作接连曝光。

**市场反应**：
- 经济发展不好还「骗人」——信用受惩罚
- 主权债券从与德法同级的 A 级（约 3% 利率）直接降为**垃圾债**

类比：学生成绩不好尚可理解，人品不行则市场严惩。
