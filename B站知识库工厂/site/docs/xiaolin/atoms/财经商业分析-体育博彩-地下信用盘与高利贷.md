# 地下信用盘与高利贷

## 一句话定义

地下赌球主流为「信用盘」——下注时不预交钱，输了再结算，降低门槛并利用赌徒高估运气；庄家同时开展高利贷，信用盘比收保证金更利于拉客与放贷。

## 详细说明

- **金字塔代理**：116 案总投注 4840 亿，层层抽水 1%–15%。
- **Owen Hansen 案**：2015 年 FBI 端掉，从 N  FL 名人赌球拓展至毒枭洗钱，判 21 年。
- **合法对比**：英美澳等有牌照 KYC；中国仅体彩（返奖率 65%–75% vs 欧美约 90%）。
