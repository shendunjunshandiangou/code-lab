# 集中重仓与分散持股的大盘相关性

## 一句话定义
当持股数量达到 20–30 只时，组合表现与大盘的相关性可超过 90%，等于变相买指数；想跑赢大盘就必须找到极好的股票并集中重仓。

## 详细说明
- 精挑细选二三十只股票的「王大爷式组合」，跑一年下来和大盘差不多——因为分散到这个数量后，组合与大盘相关性已达 90% 以上，跟买基金没有太大差别。
- 巴菲特的做法：过去几十年组合中约 **50% 的资产押在 3–5 只股票**上；某年年底组合的 47% 押在苹果一家公司（盘子规模约 5000 亿美元）。
- 这与「分散风险」的常识相反，前提是对公司有极深的研究和极坚定的信心——集中是超额收益的来源，也是能力圈的试金石。
- 反面推论（讲者版）：普通人若没有这种研究能力，与其伪装选股，不如直接买大盘。
