# 信托设立成本与适用门槛

## 一句话定义
信托设立本身不贵（美国律师费约 1000–3000 美元），贵在长期管理费；一般流动资产达 100 万美元（约千万人民币）以上且有财产保护/传承需求时才值得考虑。

## 详细说明
- **设立成本**：律师起草法律文件，美国约 1000–3000 美元。
- **管理成本**：需专业信托公司作为受托人，按委托人意愿定制方案并长期执行，管理费持续发生——这是信托「贵」的主要部分。
- **适用门槛**：流动资产（含房产）100 万美元 / 千万人民币以上，且有遗产管理、婚姻隔离、债务隔离、避税、隐私保护等需求。
- 定位：信托是一种金融创新，普通人现阶段未必用得上，但了解其功能边界有助于识别相关新闻与营销话术（「不管什么钱放进信托就安全」是营销号误导）。
