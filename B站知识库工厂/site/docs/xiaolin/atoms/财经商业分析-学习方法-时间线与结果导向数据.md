# 时间线与结果导向数据

## 一句话定义

沿时间线从简单起点梳理复杂体系，并用 GDP、股价等结果导向数据定位关键波动节点再深挖原因。

## 详细说明

- **时间线**：经济/文化类常用；ChatGPT、国别经济均可从早期讲起
- **结果导向**：GDP 突变→挖油价（俄罗斯）；通胀突变→挖改革（土耳其）；公司看**股价**过滤谣言与花边
- 配合**一手信息**：国别基础宏观数据、公司年报、创始人访谈
