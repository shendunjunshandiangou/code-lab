# 能力圈：不懂不买

## 一句话定义
巴菲特在纺织厂惨败后确立的铁律：只投自己真正理解的公司；不懂的行业宁可错过（互联网泡沫），懂了再重注（2016 年买苹果）；普通人不懂就买大盘。

## 详细说明
- **原则来源**：伯克希尔纺织厂的大亏让巴菲特确立「但凡我买、我投，就一定要懂」。
- **正面验证一（不为所动）**：互联网泡沫（tech bubble）时期人人疯抢科技股、嘲笑巴菲特落伍；泡沫破裂，市场再次验证了他的克制。
- **正面验证二（懂了重注）**：2016 年买苹果并不违背「不懂科技股」——他理解了苹果的消费品/品牌属性后大规模、持续买入，最终占组合约一半，2020 年后收益验证判断。当时市场都在争论苹果是否高点，如同后来争论特斯拉。
- **规模的烦恼**：盘子太大后，全世界能投的股票没几只；巴菲特自己说「如果资金规模小一点，我肯定比现在赚得多得多」——价值投资收益率随规模递减。
- **普通人版本**：一定要买你懂的公司，不懂就去买大盘；人云亦云，就摆脱不了韭菜的宿命。
