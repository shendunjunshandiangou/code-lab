# 美日国债利差驱动日元汇率

## 一句话定义
国债收益率可近似看作投资该国货币的收益率，因此两国国债利差与汇率高度相关——2022 年日元从 115 暴跌至 140 多，走势与美日十年期国债利差几乎完全吻合。

## 详细说明
- 机制：日本国债收益率相对美国越高 → 持有日元回报越高 → 资金流入、日元升值；反之贬值。这也是美元加息时许多国家被迫跟随加息的原因——不跟就面临贬值。
- 2022 年链条：俄乌冲突推高油价 + 美国前期刺激 → 美国通胀 → 美联储加息 → 美债利率上行 → 美日利差扩大（美英利率超 3%，日本被 YCC 压在 0.25%）→ 日元暴跌；做空日元成为全球对冲基金的 Trade of the Year（日元体量大、流动性好）。
- 注意：两条曲线如此吻合有巧合成分，拉长时间线差异仍大——影响汇率的因素很多，不能只用利率解释。
- 贬值的两面：正常贬值利好出口（日本历史上长期"求贬值"）；但暴跌带来输入型通胀、总需求下降，政府要的是可控缓慢的贬值而非过山车。
