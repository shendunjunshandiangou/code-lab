# 信用杠杆与股价依赖循环

## 一句话定义
上市巨头的生存本质是「用未来盈利能力和信用借钱」：借新还旧+增发股票的持续融资全靠信用支撑，因此维持市场信心（股价、评级）可以比真实现金流更「重要」——这解释了大公司财务造假的动机结构，也解释了为什么信心崩塌时企业会在几个月内从明星跌到破产。

## 详细说明
- 信用杠杆逻辑：小公司赚多少是多少，造假只是骗自己；上市巨头靠信用背书快速扩张——债到期借新债、随时增发股票，融资成本和可得性都由信用决定。评级机构由此获得对实体公司的「生杀权」：评级下调→股价暴跌→借不到钱→资金链断裂，业务再好照样完蛋。
- 安然的自我强化环：财报好看→市场有信心→股价上涨→SPE 玩法可持续→资金流入→继续美化财报→股价再涨……环能滚多久，骗局就能撑多久；「上季度到底赚了多少现金」反而不重要。
- 配套的信心工程：政界游说+媒体形象（Lay）、威逼利诱分析师给买入评级（Fastow）、审计安达信睁一只眼闭一只眼；2000 年虚增 1.1 亿美元流媒体营收让股价两天涨 37%。
- 反身性崩塌：环的每一节都依赖股价——2001 年股价半年跌 40%，SPE 失效→融资中断→资金链断裂→市值 600 亿的全美第七大公司三个月内破产。
- 分析启示：判断高杠杆扩张型公司，核心不是利润表，而是「信心中断时它还能活多久」——现金流、债务期限结构与信用依赖度。
