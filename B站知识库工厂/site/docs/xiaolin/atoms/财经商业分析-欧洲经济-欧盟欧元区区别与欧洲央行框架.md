# 欧盟欧元区区别与欧洲央行框架

## 一句话定义
欧盟是政治经济联盟（27 国，基本签协议即可入），欧元区是以欧元为法定货币的国家集合（19 国，须满足通胀与财政的严格门槛）；欧洲的权力框架是：各国政府管财政、欧盟委员会商讨经济政策、欧洲央行管货币。

## 详细说明
- 欧元与「美元化」的本质区别：使用美元的国家对美元发行和利率毫无话语权（美联储说什么是什么）；欧元是成员国**联合发行**，组团建立欧洲央行共同决定货币量与利率——代价是无休止的成员国博弈（如行长人选之争：总部已在法兰克福，法国就争行长位）。
- 三层框架：国家政府（财政政策+国家政策）/ 欧盟委员会（集体经济政策）/ 欧洲央行（货币政策）——货币统一而财政分散，是欧元区结构性矛盾的根源。
- 入区门槛针对「物价稳定」而非经济规模：对通胀率、政府债务和支出有严格要求（不许大手大脚借钱花钱）。1999 年首批仅 11 国；希腊条件不够、靠高盛造假 2001 年蒙混入区。
- 入区好处：①效率（更大市场，商品资本服务快速流通）；②稳定（卢森堡、马耳他等几十万人口小国自有货币易波动）；③隐性的「莫须有信心」（见利率趋同原子）。
