# 基金定义与运作机制

## 一句话定义

基金是把众多投资者的资金汇集起来，交由基金经理（portfolio manager，PM）统一投资管理的金融产品，核心理念是术业有专攻。

## 详细说明

- **运作方式**：散户/个人把钱凑在一起 → 交给专业基金经理管理
- **角色**：基金经理（portfolio manager，简称 PM）负责投资决策
- **逻辑**：让专业的人管钱，普通投资者无需亲自选股择时
- 建立基金思维框架的目的：了解分类、区别、投向、收益分配及经理是否真比散户更会管钱，避免人云亦云被忽悠
