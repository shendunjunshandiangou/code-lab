# 以色列建国背景与第一次中东战争

## 一句话定义

一战后英国接管巴勒斯坦地区（阿拉伯人占 3/4、犹太人 13%），复国运动与大屠杀推动犹太人 20 年翻 8 倍涌入；1947 年联合国 181 号决议分治，1948 年以色列独立、次日爆发第一次中东战争，以色列获胜并控制了比决议更多的土地。

## 详细说明

**移民潮**：复国运动 + 欧洲犹太人大屠杀等因素，二战结束时当地犹太人口达 60 万、占 1/3；与原住阿拉伯人的暴力冲突不断升级。

**181 号决议**：二战后英国把烂摊子甩给联合国；分割方案"跟拼图似的"。犹太人欣然接受（国际承认建国 + 大片土地），1948 年宣布独立；周边阿拉伯国家（埃及、伊拉克、约旦、叙利亚、黎巴嫩等）坚决反对，次日进攻。

**结果**：以色列能打 + 国际犹太人和美国暗中支持，1949 年停火时以色列控制的土地反而多于决议方案；约 70 万阿拉伯人流亡（阿方称"大灾难"）。这场战争开启了延续至今的阿以纷争。

**经济反差**：如今以色列人均 GDP 约 5.5 万美元，碾压土耳其、伊拉克、约旦乃至沙特，超过英德日、直逼美国。
