# 迪拜外来劳工依附经济

## 一句话定义

88% 人口为外来打工者；护照常被收走、50°C 下超长工时；本地人福利优厚，蓝领工资约为本地人 1/10。

## 详细说明

- 年上千劳工「自然死亡」；印度使馆估年约 100 名印度劳工自杀
- 2017 年立法要求每周一天休息但未完全贯彻
- 廉价南亚劳动力是迪拜奇迹的「黑暗支柱」
