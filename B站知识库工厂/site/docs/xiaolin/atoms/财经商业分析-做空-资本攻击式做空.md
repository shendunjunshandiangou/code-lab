# 资本攻击式做空

## 一句话定义
资本攻击是大佬专属的做空：用海量资本给高估资产一个向下的原始推动力、带动市场跟卖，理论上是价格回调机制，实际常引发恐慌踩踏——故金融危机期间监管通常禁止做空。

## 详细说明
- 机制：大佬判断资产高估且众人有同感但价格仍居高位 → 海量资本带头砸盘 → 市场反思跟卖 → 强反馈把价格拉回「合理区间」。
- 现实缺陷：人是非理性的，回调不会恰好停在合理区间，恐慌可致踩踏、暴跌甚至金融危机——空头挨骂的重要原因。
- 信心逻辑：金融本质是信用；「市场觉得你行你就行，不行也行」。资本空头进攻等于当众宣判你不行，对市场信心是毁灭性打击，走得不稳的公司被踹一脚可能一蹶不振。因此危机期间监管禁止做空。
- 代表：利弗莫尔 1929 年经一百多个经纪人偷偷大举做空美股，间接触发大崩盘与大萧条，获利上亿美元、被斥发国难财；索罗斯 1992 做空英镑——但做空外汇是双向对称的（空一币=多另一币），与做空股票不同，更接近纯粹的资本攻击。
