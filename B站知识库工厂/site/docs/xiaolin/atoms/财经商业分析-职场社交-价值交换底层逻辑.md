# 价值交换底层逻辑

## 一句话定义
职场社交最底层的逻辑是价值交换：对方帮你的三类动机（招人、圈子美名、纯做好事）价值递减，你能提供的价值越高，社交成功率越高。

## 详细说明
对方非亲非故为什么要花时间帮你？三种情况：

| 情况 | 对方的动机 | 被回复难度 |
| --- | --- | --- |
| 你很优秀，对方正好在招人 | 招至麾下当得力助手（职场上招得力助手不容易） | 最低 |
| 有共同圈子或联系（如师兄师姐） | 帮你是好事，还能在圈子里留下「乐于助人」美名 | 中等 |
| 毫无关系 | 纯粹做好事，图自我满足感 | 最高 |

- 越往下，你对对方的价值越低，得到回复越困难。
- 因此社交除了认识人、拿信息、要内推，真正该问自己的是：我的价值在哪里？我能给对方提供什么？
- 强调价值交换不是让人功利：切忌一上来就索取（如「你们公司有机会能不能推荐我」）；核心是让自己变得更有价值。
