# 期货合约与未来价格对赌

## 一句话定义

期货是约定未来某时以约定价格交易的标准化合约；进化至可追踪任意可量化指标（股指、利率、播放量），到期多现金结算——本质是对未来某时点价格的对赌。

## 详细说明

- 对冲：预期卖则做空锁价，预期买则做多。
- 绝大多数持仓到期前平仓；原油实物交割 &lt;1%。
- 「一切指标皆可期货」——成交面值超外汇，为全球最大交易市场。
