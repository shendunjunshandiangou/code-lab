# 希腊紧缩政策经济后果

## 一句话定义

三马主导的紧缩政策打破希腊财政支出正向循环，引发反向恶性循环：数年 GDP 累计萎缩约 25%、2013 年失业率约 25%（35 岁以下约 50%），高质量人才外流德法。

## 详细说明

希腊接受救助后开始节衣缩食：增税、减支、还债。

2010 年赤字从 15% 多降至 11% 多，但财政支出减少触发**反向恶性循环**（与「政府借债正向循环」相反）：

| 指标 | 数据 |
| --- | --- |
| 2010 年 GDP | 下降 10% |
| 2010–2013 | 持续危机 |
| 累计 GDP 萎缩 | 约 **25%** |
| 2013 年失业率 | 约 **25%** |
| 35 岁以下失业率 | 约 **50%**（「毕业即失业」） |

**社会冲击**：
- 除刚需产业外近乎民不聊生
- 欧盟人员自由流动，高质量人才流向德法
- 小 Lin 希腊同事：四兄妹中两人在国外工作养全家

数字下降不仅是经济指标，而是每个希腊人切身感受的巨大冲击。
