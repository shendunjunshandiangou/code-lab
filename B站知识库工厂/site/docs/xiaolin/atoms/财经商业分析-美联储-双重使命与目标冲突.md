# 美联储双重使命与目标冲突

## 一句话定义
美联储的两大法定目标——稳定物价（stable prices）与最大就业（maximum employment）——经常互相冲突：刺激就业需要宽松、宽松推高通胀；如何拿捏是货币政策的核心难题，主流经济学认为物价稳定更重要。

## 详细说明
- **冲突机制**：宽松货币 → 刺激就业 → 通胀上行；紧缩压通胀 → 抑制经济活力 → 失业上升（沃尔克 20% 利率使失业率超 10%）。
- **武器的化繁为简**：美联储早期工具繁多，后来主要只控制**联邦基金利率**（银行间隔夜拆借利率）——平时说的加息降息即指它。短期利率决定钱的流通速度，实际控制货币量；存款、房贷、国债等其他利率由市场供需决定，美联储无法直接控制但相关性很高。
- 该框架适用于各国央行：目标多元且冲突，工具集中于短期利率。
