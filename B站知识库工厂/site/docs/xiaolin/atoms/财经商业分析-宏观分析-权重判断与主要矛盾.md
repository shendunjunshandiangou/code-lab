# 权重判断与主要矛盾

## 一句话定义

分析经济这类复杂问题时，**逻辑链条成立、故事好听 ≠ 是主要因素**——关键在各因素的权重：「印度股市大涨带动经济腾飞」逻辑没毛病，但印度股民占比仅 5%–10%，权重太小不是主因（换成股民过半、以金融市场为融资中心的美国就可能成立）；「阿斯麦营收大涨推动荷兰经济」听着有理，但 2023 年阿斯麦股价亮眼、万亿美元 GDP 的荷兰却排增长倒数第三——地产萎靡消费疲软的权重远大于一家明星公司，就像说「茅台业绩好推动中国经济」一样扯。

## 详细说明

- 考试归因类比：「不及格因为线性代数没学好/做题太慢」是合理找主要矛盾；「因为第三题做错了」——剩下二十七道错题怎么解释？听着弱智，但在复杂系统里权重误判天天发生。
- 同一逻辑在不同对象上权重不同：股市财富效应在美国是主因候选、在印度只是小噪音——分析必须带着国别结构（融资体系、参与率）判断。
- 内容创作观：讲者刻意不把研报里二三十个影响因素全部罗列（那样反而省事），而是筛掉不重要的细节、理清主干——「整理树干」是最难的环节；传达的是底层逻辑而非细节堆砌。
