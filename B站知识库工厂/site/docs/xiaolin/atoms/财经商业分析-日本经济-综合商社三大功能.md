# 综合商社三大功能

## 一句话定义

日本综合商社除贸易外，还有**情报分析**与**产业投资**两大暗线功能，巴菲特增持五大商社即看重其资源与布局能力。

## 详细说明

- **贸易**：铁矿石/原油进口，汽车家电出口；进日本绕不开商社。
- **情报**：长期跨国经营，信息能力极强（王进喜照片推测油田为都市传说，不保真）。
- **投资**：巴西铁矿、东非天然气、东芝早期入股；高风险业务银行不贷时商社可给。
