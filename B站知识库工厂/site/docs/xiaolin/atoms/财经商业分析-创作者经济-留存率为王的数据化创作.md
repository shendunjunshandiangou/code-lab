# 留存率为王的数据化创作

## 一句话定义

MrBeast 认为视频最重要的量化指标不是播放量、不是点击率，而是**留存率**：用 YouTube 后台的留存曲线看观众每一秒的流失，把每段视频切成小段落逐段复盘「大家在哪不感兴趣了、为什么」，由此打磨出一套成熟模式——快节奏剪辑、鲜艳画面、紧凑音效、不断设置新悬念点，节奏远快于一般综艺，一个点接一个点，让人**生理性地、不自觉地**看下去。

## 详细说明

- 播放量/点击率衡量的是「多少人来」，留存衡量的是「内容本身留不留得住人」——后者才是内容质量的直接反馈，也决定平台算法的长期推荐。
- 对争议的回应方式也是数据化的：面对「撒钱作秀」「挑战危险」「扭曲价值观」等质疑，他主要看数据看留存——大多数人喜欢，就优化、就做。
- 撒钱只是表象，底层是极强的专业性与对观众的理解；与 Netflix 用行为数据取代打分数据的思路同源（[财经商业分析-流媒体-推荐算法从打分到行为数据](/xiaolin/atoms/财经商业分析-流媒体-推荐算法从打分到行为数据.html)）。
