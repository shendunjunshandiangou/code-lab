# 投行求职长周期路径

## 一句话定义

进入高盛等投行不是临毕业投简历即可，常见路径从大一大二探索项目起，经社团技能积累、大三暑期实习至 Return Offer，周期长达数年。

## 详细说明

- 大一大二：Early Insights、Spring Week、Mentorship 等探索性项目
- 大三：社团经历、技能、简历；暑假 **Summer Analyst** 实习
- 实习表现好 → **Return Offer**，上岸后不必再找工作
- 与 Google、麦肯锡等路径及技能要求均不同 → 须先明确方向再专项准备
