# Nikola 融资估值与套现时间线

## 一句话定义

Nikola 从 2016 年 A 轮 1600 万美元（估值 3 亿）滚雪球至 SPAC 上市估值 33 亿、股价一周翻五倍；Milton 在 2019 年融资后悄悄套现 7900 万美元并购奢华庄园，全程一辆车未量产。

## 详细说明

| 时间 | 事件 | 金额/估值 |
| --- | --- | --- |
| 2016 | A 轮融资 | 1600 万美元，估值 **3 亿美元** |
| 2016 后 | 签战略合作（含博世等） | — |
| 2017 | B 轮融资 | 4400 万美元，估值 **9 亿美元**（一年翻三倍） |
| 2018 | C 轮融资（滑坡视频后） | 2140 万美元，估值 **11 亿美元** |
| 2019 | 新一轮融资 | **2.5 亿美元**，估值 **30 亿** |
| 2019 后 | Milton **悄悄套现** | **7900 万美元** |
| 2019 后 | 购犹他州奢华建筑 | **3200 多万**（类似皇家园林） |
| 2020 年 6 月 | 与 VectoIQ SPAC 合并上市 | 融资近 **8 亿美元**，估值 **33 亿** |
| 上市后 | 股价 | **一周内翻五倍** |
| 市值对比 | 一辆车未造 | 市值**超过福特** |
| Milton 个人 | — | 跻身 **10 亿美元俱乐部** |

### 关键事实

- 2016 年 Nikola One 已在**内部被放弃**，靠视频和忽悠继续融资
- 创始人套现本应引起警觉，投资人视而不见
- 2020 年 6 月上市后，贾跃亭法拉第未来 2021 年也 SPAC 上市
