# AMD 收购 ATI 与商誉减记

## 一句话定义

2006 年势头正盛的 AMD 想横跨 CPU/GPU 获得协同效应，在英伟达（市值 75 亿、比自己 86 亿还大，怕反被吞）和 ATI（约 50 亿）之间选了后者：掏空 18 亿现金 + 借 25 亿外债 + 11 亿股票豁命拿下——随后两年**商誉减记 26 亿美元**（50 多亿买的东西发现只值 20 多亿），战略正确但买太贵，现金流崩紧，开启最黑暗的十年。

## 详细说明

- 黑暗十年的多线溃败:英特尔在收购宣布后第三天发布酷睿 2、祭出 Tick-Tock 战术（一年更制程一年更架构、自己卷自己）重占市场；AMD 既跟英特尔抢 CPU、又跟英伟达抢 GPU、还要自己设计+生产，债务累至 50 亿美元。
- 断臂求生：2008 年拆分卖掉晶圆厂（即 GlobalFoundries）；CPU 份额从近 50% 跌到 20%，服务器 CPU 从 30% 跌到 1%，市值跌破 20 亿美元，卖总部大楼换 1.6 亿美元，华尔街认为 AMD 将退出历史舞台。
- 历史的辩证：正是买了 ATI，AMD 才成为唯一横跨 CPU/GPU 两大市场的巨头，2011 年的 APU（单芯片融合 CPU+GPU）和后来垄断游戏主机市场（PS5/Xbox/Steam Deck）、乃至 2023 年 AI 显卡的二级助推都源于此——**战略对、价格错**的经典案例。
