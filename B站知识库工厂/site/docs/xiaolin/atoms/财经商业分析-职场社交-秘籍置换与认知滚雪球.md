# 秘籍置换与认知滚雪球

## 一句话定义
把从 A 处学到的信息消化后，在与 B 的社交中展示，赢得 B 的深度分享，如此「用 A 的秘籍换 B 的秘籍、再换 C 的秘籍」，认知在一次次社交中滚雪球式提升。

## 详细说明
- 与 A 社交拿到信息，消化成自己的东西；下次与 B 聊天时不经意间展示，B 会觉得你有想法、有见解，留下好印象。
- 反向视角：职场老手面对一问三不知的小白，只会讲讲皮毛；面对「有点东西」的小白，可能聊嗨了把行业本质倾囊相授。
- 因此社交是正循环：每次对话既是获取信息，也是为下一次更深入的对话积累筹码；聊一圈下来，认知显著提升，还会显得「什么都知道、谁都认识」。
