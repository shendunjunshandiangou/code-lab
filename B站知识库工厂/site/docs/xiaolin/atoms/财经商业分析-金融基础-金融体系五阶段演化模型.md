# 金融体系五阶段演化模型

## 一句话定义
金融的本质是"金=钱、融=融汇流动"——让钱流到能产生更大价值的地方，从而提高社会整体福利与资源配置效率；整个现代金融体系可以用五个演化阶段还原：1.0 个人借贷 → 2.0 银行（存贷中介）→ 3.0 股权投资（入伙）→ 4.0 投行/券商/FA（撮合服务）→ 5.0 公开市场（IPO 面向大众发行）。

## 详细说明
以"小林开奶茶店"为线索：
- **1.0 个人借贷**：小林向老王借 100 块、还本付息 110——一笔交易让出借人得利息、创业者得资金、消费者得产品，已概括金融全部本质。
- **2.0 银行**：老王们的钱有限、彼此又不认识，银行汇集存款、放贷给缺钱的人，居中赚利差——让钱系统性流动。
- **3.0 股权投资**：银行只做固定利息的低风险贷款，高风险项目（如"量子瘦身奶茶"）借不到钱；愿担高风险、求高回报的金主以"入伙分成"方式出资——股权投资。典范：Peter Thiel 2004 年 50 万美元买 Facebook 10.2% 股份，2012 年套现 10 亿美元，8 年 2000 倍。
- **4.0 投行/券商/FA**：创业者找不到足够多的投资人，投行以圈内资源 + 项目包装能力撮合双方，收服务费。
- **5.0 公开市场**：企业股份公开发行给大众（IPO/敲钟），散户也能成为股东；交易所（纽交所/港交所/上交所）承载，投行继续提供上市服务。
- 到 5.0，全社会的闲钱可按风险偏好流向不同渠道，不同风险的项目也各有渠道拿到钱——现实中无数融资者、投资人、中间商交织，系统看似庞大，本质仍是这五层。
