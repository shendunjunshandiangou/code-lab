# 三来一补与制造业 FDI

## 一句话定义

改革开放早期外商投资以三来一补（来料加工/装配/来样 + 补偿贸易）为主；1992 年 FDI 取代借款，制造业占外资六七成，1992–2010 出口翻 26 倍。

## 详细说明

- 出卖劳动力为主；技术较强国家对外投资常见补偿贸易。
- 1991 外汇 200 亿美元+；2011 突破 3 万亿。
- 92 年开放扩大至长江沿岸、边境、内陆省会。
