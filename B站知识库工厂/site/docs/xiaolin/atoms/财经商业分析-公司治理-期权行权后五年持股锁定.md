# 期权行权后五年持股锁定

## 一句话定义

特斯拉 2018 年马斯克激励计划规定：期权行权换成股票后，须额外持有五年，防止 CEO 解锁后立即变现离场，给投资者更长久的利益绑定保障。

## 详细说明

2018 年计划比 2012 年更苛刻的一点：董事会担心马斯克拿到期权后**立刻换成股票并卖出变现**。

**规则**：每完成一档目标获得的期权，行权换成股票后，须**额外持有 5 年**才能自由处置（讲者表述）。

**设计意图**（讲者分析）：
- 把马斯克长期绑在特斯拉
- 对投资者而言，CEO 大量持股是信心信号
- 与 [财经商业分析-公司治理-激励计划市值持续六个月达标](/xiaolin/atoms/财经商业分析-公司治理-激励计划市值持续六个月达标.html) 共同构成「长期价值」导向，而非短期套利

2012 年计划无此五年锁定要求；2018 年加码体现了董事会在第一轮计划被快速完成后对「留人」的更强约束。
