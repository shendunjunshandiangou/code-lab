# 别人恐惧我贪婪：逆向抄底

## 一句话定义
「别人贪婪的时候我恐惧，别人恐惧的时候我贪婪」——当好公司因市场恐慌（而非基本面恶化）被错杀时逆向买入；前提是先验证基本面没坏。

## 详细说明
经典案例：

| 时点 | 事件 | 操作与结果 |
| --- | --- | --- |
| 1960 年代 | 美国运通抵押物丑闻，华尔街恐慌「运通快不行了」 | 巴菲特到消费者中调研，发现大家照常使用——判断只是华尔街新闻，抄底后两三年翻 2–3 倍 |
| 1973 经济危机 | 市场普跌 | 抄底华盛顿邮报，后翻 40 倍 |
| 黑色星期一之后（字幕原文 88 年，存疑） | 股灾 | 抄底大量买入可口可乐 |
| 2008 金融危机 | 银行业恐慌 | 抄底大量银行股、金融股 |

关键限定：**不是一跌就买**。有的下跌是基本面真坏了（业务不行），有的只是市场噪音、甚至竞争对手的黑报告——所以逆向抄底必须以「先找到好公司、看准基本面」为前提。运通案例中的「群众调研」（后被称为 scuttlebutt 式调研）就是区分两者的手段。

「危机」二字：别人看到危，价值投资者找机。
