# 民办为主与公办福利院高门槛

## 一句话定义
公办福利院数量极少、入住条件苛刻（市级以上劳模 80+、失独老人 70+、因公/见义勇为致残等，符合还要排队），因此中国社会化养老的现实格局是：民办为主、政策指导、辅以补贴。

## 详细说明
- 北京市第一福利院与第五福利院已合并；入住条件包括：市级以上劳模且 80 周岁以上、失独老人 70 周岁以上、因公致残或见义勇为致残（字幕不清晰处按语境修正）——绝大多数普通老人无法进入。
- 民办机构覆盖三个档次（北京实地样本）：
  | 形态 | 规模 | 月费 |
  | --- | --- | --- |
  | 社区养老驿站 | 二三十人 | 几百–9000 元 |
  | 中型养老院 | 百余人 | 5000–1 万多 |
  | 高端养老社区（保险系） | 数千人 | 1.2 万–3 万 + 约 200 万保险/押金门槛 |
- 北京共 450 多家养老机构（容纳 100 人以上），五星级约十几家。
