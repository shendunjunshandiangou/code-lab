# 价值投资三要素

## 一句话定义
巴菲特价值投资的精髓可归纳为三点：找到好公司（并重仓）、低价买入（别人恐惧我贪婪）、长期持有（吃复利与时代红利）；根基是「价格在长期是价值的体现」。

## 详细说明
1. **找到好公司，重仓**：与分散投资相反，只有集中重仓才能跑赢大盘（见集中持股相关原子）。选股偏好商业模式简单、现金流稳定的行业（保险、消费品）。
2. **低价买入**：相信基本面、无视噪音，在市场恐慌时逆向抄底（运通、华盛顿邮报、可口可乐、2008 银行股）。
3. **长期持有**：经济小周期约 5–8 年、大周期几十年，公司实现价值需要时间；美股 60–70 年代几乎没涨，80–90 年代翻了好几倍。

底层信念：即使股价短期暴涨暴跌看似与价值无关，**价格在长期是价值的体现**，这一根基不因技术和商业模式变化而动摇。

对普通人的推论：不了解行业和公司，就相信时代红利，长期持续参与大盘。
