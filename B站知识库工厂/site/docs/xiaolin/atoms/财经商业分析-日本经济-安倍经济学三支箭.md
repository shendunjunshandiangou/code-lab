# 安倍经济学三支箭

## 一句话定义

2012 起安倍晋三打包三支箭：激进货币宽松、财政刺激、结构改革；最有效为第一支箭（黑田），第二三支效果有限甚至消费税加税压制需求。

## 详细说明

- 第三箭：增女性就业、放松监管等，多名存实亡
- 第二箭：减税+增支，但消费税 5%→8%→10% 每次加税消费大跳水
- 第一箭：负利率 + QQE，推升股债
