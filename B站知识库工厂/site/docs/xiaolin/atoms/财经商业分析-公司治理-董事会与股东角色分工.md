# 董事会与股东的角色分工

## 一句话定义
董事会是全体股东选出的一小撮代表，负责任免管理层、决定并购等重大事项；董事会成员未必是大股东——推特整个董事会持股合计不到 3%。

## 详细说明
- **普通股东**：公司的所有者，靠投票权表达意志，人数众多、利益分散。
- **董事会**：股东选出的代表机构，替全体股东做重大决策（任命/监督管理层、批准兼并收购等）。
- **持股与权力分离**：董事会成员不一定是大股东。推特董事会中持股最多的是创始人（2.4%），其余成员合计不到 0.3%，整个董事会持股不到 3%——却握有是否接受收购的决定权。
- 这种分离正是恶意收购攻防的舞台：收购方可以绕过董事会直接争取股东（邀约收购），也可以夺取董事会席位（代理权斗争）；董事会则可用毒丸等工具为股东利益（或自己的位子）而战。
- 部分公司有特殊条款（如重大事项需董事会 70% 以上同意），会改变博弈结构。
- **三层一环套一环**（OpenAI 宫斗篇补充）：管理层（CEO 领衔）负责日常运营决策；董事会任免管理层、几个月开一次会；更大事务上升到股东大会投票，其中最关键的是任命新董事。董事会不爽管理层可开掉 CEO；大股东不爽董事会可召开股东大会重组董事会——说到底谁股份大谁说了算。而 OpenAI 这类非营利组织没有股东，链条在董事会处断裂，权力天花板即董事会。
