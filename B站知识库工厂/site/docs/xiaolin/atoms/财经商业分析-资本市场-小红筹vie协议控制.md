# 小红筹 VIE 协议控制

## 一句话定义

针对外资持股受限行业（如网络传播类，外商股东可能导致许可证办不下来）的红筹变体：香港公司在境内先设一个外商独资企业（WFOE），WFOE 与境内运营公司签一堆 VIE 协议——取得其收益和控制权——用"协议控制"替代"股权控股"连接境外上市主体与境内业务。

## 详细说明

- 结构图特征：B 站等公司股权结构图里的虚线部分就是协议控制。
- 与大红筹的唯一区别：中间那层从直接控股换成协议控制，其余（开曼上市主体 + 香港层 + 境内主体）完全一致。
- 这就是常被"不明觉厉"的 VIE 架构：因签的那堆协议叫 VIE 协议（Variable Interest Entity，可变利益实体）。
- 风险本质：境外股东对境内业务的权利建立在合同而非产权上——依赖协议的可执行性。
