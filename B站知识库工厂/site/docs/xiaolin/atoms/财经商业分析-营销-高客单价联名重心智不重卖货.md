# 高客单价品牌联名：重心智不重卖货

## 一句话定义
对奢牌护肤品、电动车这类高客单价产品，联名的目的不是当场卖货，而是通过资源整合置换塑造品牌形象、提高粉丝身份认同、在潜在客户群里**占领心智**——等用户真正产生需求时第一个想到你；执行上还常引入第三方场景扩大"朋友圈"并分层触达。

## 详细说明
以 2023 年特斯拉 x 赫莲娜联名冠名 EDC 电音节（7-10 月，北上广深、苏州等 7 城巡回，主题"能量之旅"）为例：

- **为什么拉上电音节**：两家客单价都高，只互享粉丝则朋友圈受限；EDC 同样是"能量"调性、受众是年轻群体，一加一加一，扩大触及面并强化三者的共同印象。
- **触达与留资**：电音节联名周边、体验区、拍照打卡，快闪活动可体验绿宝瓶产品——以更低成本让更多用户先用上产品；用户体验注册时留下联系方式，相当于把潜在用户拉进自己的私域朋友圈。
- **分层对症下药**：面向电音节/快闪的广泛人群送周边、小样（低成本）；面向特斯拉车主这类消费力强的潜客，送进阶护理体验、推进阶产品。
- 联名形式不限于共同出产品（LV x Supreme、优衣库 x KAWS、Adidas x 椰子鞋），也可以是共同办活动（campaign）。
