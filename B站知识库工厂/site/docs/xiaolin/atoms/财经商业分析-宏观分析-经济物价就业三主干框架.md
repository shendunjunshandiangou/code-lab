# 经济物价就业三主干框架

## 一句话定义

分析任何国家经济的知识树主干只有三样：**经济（GDP）、物价（CPI）、就业（失业率）**——央行和政府的目标说白了就是让这三个数字好看；往下是二级数据归因（PMI、核心通胀、工资增长、零售、非农、CPI 拆分到租房/石油/食品），再看财政政策、货币政策与主要资产价格（股票、国债、房地产、外汇）——一套自上而下、由主到次的分析框架。

## 详细说明

- 考试类比：三主干像总分，知道不及格后翻卷子找错题就是查二级数据；「怎么改」对应政策工具。
- 侧重随时期切换：美国前些年重就业（看非农），近两年焦点转向通胀（CPI 干扰多，须往下拆分项）。
- 框架的用法：细节复杂到蒙圈时**跳到上一层**，不被细节牵绊、找到主要矛盾；没有框架看新闻是「信息输入很多、脑袋一片空白」，有了框架每次输入都在完善这棵树。
- 讲者补充：还有一个对民生很重要但经济学家不太关心的指标——**贫富差距**。
- 复杂性所在：难的不是单个指标，而是这些要素之间的联系；越专业的人看得越细，但底层看得很简单。
