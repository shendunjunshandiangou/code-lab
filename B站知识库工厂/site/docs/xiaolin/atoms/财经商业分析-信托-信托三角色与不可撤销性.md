# 信托三角色与不可撤销性

## 一句话定义
信托由委托人（交出财产）、受托人（保管打理）、受益人（分享收益）三方构成契约；财产入托后不属于任何一方，信托本身是独立个体，且一般不可撤销。

## 详细说明
- **起源**：十字军东征时士兵把妻儿家产托付给留守兄弟，归来发现被霸占，此后立法保护这种契约关系——「言而有信，受人所托」。
- **三角色**（以士兵为例）：委托人=士兵，受托人=帮忙照看的「老王」，受益人=妻儿老小。
- **财产独立性**：钱放进信托后，不属于委托人、不属于受托人、也不属于受益人——这是信托能实现婚姻隔离、债务隔离的法律基础。
- **不可撤销性**：定了就定了，委托人自己也撤不回来（梅艳芳母亲毕生打官司也没能把钱取出）。
- 本质：一种金融创新，实现其他金融产品达不到的功能——财产保护与财富传承。
