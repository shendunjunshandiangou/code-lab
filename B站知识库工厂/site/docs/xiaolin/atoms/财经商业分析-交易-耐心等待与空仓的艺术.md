# 耐心等待与空仓的艺术

## 一句话定义
利弗莫尔认为交易中最难的不是分析和策略，而是**坐着不动**：「当机会还不够明显的时候，不要着急入场」——没有好想法时的轻举妄动，是绝大多数亏损的来源；耐心持币等待大机会，才配得上重仓出击。

## 详细说明
- 原则表述：机会不明显时不入场；耐心是交易过程中最难的部分。看似朴实，实为其全部原则中最精彩的一条。
- 行为学解读：交易者的本能是「必须做点什么」——盯盘产生行动冲动，空仓带来错过焦虑（FOMO）；而好交易极其稀少，频繁出手必然稀释质量。
- 实践样本：二次破产后六个星期零交易、纯观察，等来一战钢铁股大机会；1929 年前长期蛰伏「伺机而动」，直到龙头股集体疲软的危机信号出现才隐蔽建立世纪空仓。
- 与其他原则的关系：耐心解决「何时入场」，让利润奔跑/止损解决「入场后怎么办」，独立思考解决「信号从哪来」——四者构成完整闭环。
