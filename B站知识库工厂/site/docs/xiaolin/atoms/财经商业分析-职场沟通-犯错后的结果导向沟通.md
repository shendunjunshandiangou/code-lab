# 犯错后的结果导向沟通

## 一句话定义

搞砸任务或犯错时，老板要的是事实、解决方案和结果——先认错补做把事办好，报告交上去后再解释原因，别一上来编理由。

## 详细说明

**老板此时最关心什么：**

老板是**结果导向型**的，出现错误、纰漏或影响结果的事情时，他最关心：

1. 事实是什么
2. 第一时间想到解决方案
3. 把事先给解决了

**反面做法：**

员工开始编理由：忙、陪客户、家人有事……老板又不傻，什么急事、谁在搪塞心里明镜似的。更关键的是：**老板根本不 care 你那堆破事儿**，他急的是报告什么时候能出来。

**买饭类比：**

让老王买饭，两小时没回来——你想的是「我的饭呢」，不是「老王两小时奇遇记」。老板更是如此。

**正确做法：**

1. **先认错 + 给出补救方案**：「不好意思是我耽搁了，实在抱歉。您看我今天补上，您看行吗？」
2. **立刻去办**：老板说行就去干
3. **事后再解释**：报告递上去后，若真有急事原因，再心平气和说明——既体现担当，又把问题解决

一般人犯错第一反应是解释「我没错」，但老板也在气头上听不进去。应先安抚「受惊的老马」，再谈原因。
