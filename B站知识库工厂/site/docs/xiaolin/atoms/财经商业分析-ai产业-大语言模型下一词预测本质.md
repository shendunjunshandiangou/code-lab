# 大语言模型下一词预测本质

## 一句话定义

ChatGPT 等大型语言模型本质是据上下文计算「下一个词/句」出现概率，是语言统计模型而非真正理解——像极聪明的小孩模仿大人说话。

## 详细说明

- **机制**：前文 → 从海量参数中算下一词概率（如「我很…」→ 开心/健康/着急）
- **能力来源**：上千亿参数 + 海量文本找规律，无需显式教「什么是编程/脚本」
- **局限**：逻辑错误、编造答案（hallucination）、道德离谱输出
- **图灵余论**：当 99.9% 答对，「真理解 vs 模仿」意义可能不大
- **突破**：极大提升人↔机文字沟通效率，人可直接说人话
