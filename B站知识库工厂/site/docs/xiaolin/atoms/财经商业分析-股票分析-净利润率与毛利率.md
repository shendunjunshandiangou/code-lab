# 净利润率与毛利率

## 一句话定义

净利润率 = 净利润/总收入，衡量盈利能力；高利润率公司更能抵御成本上涨；高速扩张亏损公司应看毛利率。

## 详细说明

**行业基准**：A 股/美股整体约 5–10%；金融、芯片、软件、烟草、医药可达 10–20%+。须与行业平均比较。

**案例**：茅台 2021 中报净利率约 50%；苹果约 25%。

**亏损高增长公司**：补贴/研发导致净利润为负时，看扣直接成本后的毛利率。
