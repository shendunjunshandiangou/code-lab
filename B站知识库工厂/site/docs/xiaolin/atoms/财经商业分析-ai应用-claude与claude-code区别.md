# Claude与Claude Code区别

## 一句话定义

Anthropic 的 Claude 是聊天框问答产品；Claude Code 是像真实程序员一样写代码、debug、部署并直接交付可用成果的 Coding Agent。

## 详细说明

**公司定位差异**：
- OpenAI：Chat 模式让普通人免费使用，C 端认知度高
- Anthropic：早期核心 bet 在 **coding（代码生成）**解决问题

**产品对比**：

| | Claude | Claude Code |
| --- | --- | --- |
| 形态 | 聊天框 | 超级程序员助理 |
| 能力 | 相对简单问题；复杂问题多给代码片段，用户自己上线部署 | 写代码、debug、上线部署，**直接交付可用成果** |
| 用户 | 广泛 C 端 | 程序员、金融分析师、医疗企业等 B 端为主 |

**为何火爆**：非常明确提高程序员生产力（Vibe Coding）；金融分析师用于财报分析；「动嘴就行」。

小 Lin 案例：有人用简单几句话让 Claude Code 自己上网搜集信息编出完整网页。
