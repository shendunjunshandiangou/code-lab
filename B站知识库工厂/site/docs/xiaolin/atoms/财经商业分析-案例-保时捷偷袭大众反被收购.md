# 保时捷偷袭大众反被收购

## 一句话定义
2008 年保时捷试图通过偷偷买入（creeping tender offer）蛇吞象式收购大众，眼看持股要过 50%，却因金融危机爆发资金链断裂，反被大众收购——偷偷买入策略最著名的翻车案例。

## 详细说明
- 策略：绕过董事会，在公开市场悄悄积累大众股票，目标超过 50% 投票权后掌控这家比自己体量更大的公司。
- 转折：2008 年金融危机爆发，保时捷自身资金链先断裂。
- 结局：猎手变猎物，保时捷反而被大众收购。
- 教训：偷偷买入型恶意收购高度依赖持续融资能力，宏观流动性风险可以瞬间逆转攻守。
