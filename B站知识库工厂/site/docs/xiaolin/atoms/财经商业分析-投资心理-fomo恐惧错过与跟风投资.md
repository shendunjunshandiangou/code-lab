# FOMO 恐惧错过与跟风投资

## 一句话定义

FOMO（Fear of Missing Out，害怕错过）指投资人在新风口涌入时因怕错失机会而头脑发热跟风投入——Nikola 案例中，部分投资人不看报表产品，凭直觉「哪热往哪钻」，与 2018 年后特斯拉暴涨引发的「追竞争对手」心态叠加。

## 详细说明

### 两类投资人

- **一类**：看市场、报表、产品
- **另一类**：凭直觉，哪热往哪钻

### FOMO 机制

- 当很多人涌入新机会和风口时，投资人头脑发热
- 「我就怕错过」→ 赶紧投
- 2018 年后特斯拉股价飞涨，没赶上的投资人「肠子悔青」
- 有人追特斯拉，有人捧「特斯拉竞争对手」

### Nikola 案例中的表现

- 电动汽车门槛极高，但投资人管不了那么多
- **能不能量产、电池技术先靠边站**，目标非常明确：**上市**
- 创始人套现 7900 万美元，投资人**视而不见**
- 讲者结论：若不做细致背景调查就强制塞钱，没有 Milton 也会有其他人出来
