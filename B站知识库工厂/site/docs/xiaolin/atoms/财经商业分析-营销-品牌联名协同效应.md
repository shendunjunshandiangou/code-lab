# 品牌联名的协同效应

## 一句话定义
好的品牌联名不是把两个 logo 放一块的"一加一约等于二"，而是要有品牌协同效应（synergy）——两个品牌**气质相符**、相互加持、共同强化调性；气质不符的联名轻则尴尬、重则给品牌抹黑。

## 详细说明
- 底层逻辑是"近朱者赤、近墨者黑"：你平时跟谁在一起，会影响别人对你的印象判断（整天晒和大佬聚餐的照片，别人就觉得你是大佬）。
- 成功案例：
  - **红牛 x GoPro**：表面八竿子打不着，但品牌形象都是动感、极限，常年联合赞助赛事；太空定点跳伞被 GoPro 拍下，把两者形象展现得淋漓尽致。
  - **Apple Watch x 爱马仕**：爱马仕给苹果带来极致工艺，苹果给爱马仕带来科技创新，相得益彰。
  - **LV x Supreme**：LV 抬高 Supreme 的定位，Supreme 让 LV 更潮、拓展设计风格。
  - **特斯拉 x 赫莲娜**：乍看不搭，细品共享三重气质——科技先锋（电车先锋 / 世界第一瓶面霜的研发先锋）、能量（新能源 / 绿宝瓶"能量瓶"）、绿色（绿色能源 / 绿宝瓶），还产生"精华里的特斯拉"这种高效的心智类比。
- 失败案例：2019 年喜茶 x 杜蕾斯，尺度把握不好引发观众反感，还遭到有关部门批评。
