# 特多「Do So」运动：心理战操纵选举

## 一句话定义
2009-2010 年特立尼达和多巴哥大选中，SCL 为印度裔政党策划了表面「反政治」的「Do So（不要投票）」青年运动，利用两族群家庭管教差异实现不对称弃票，最终帮助雇主翻盘获胜——是「不直接拉票、通过改变特定人群行为赢选举」的经典心理战案例。

## 详细说明
- 背景：特多政坛分两大阵营——印度裔派与黑人派；SCL 受雇于印度裔派做选举咨询。
- 操作：以年轻人为突破口发起「Do So」运动（配标志性手势），号召年轻人不要投票、与「肮脏的政治」隔绝。活动不针对任何党派，纯粹迎合年轻人对政治的叛逆心理，因此在社交媒体上病毒式传播。
- 不对称机关：两派**成年人**投票行为不受社交媒体运动影响；**年轻人**易被带动。而黑人家庭对孩子投票放任自流，印度裔家庭家教森严——孩子再叛逆也会被家长押去投票。结果黑人青年大面积弃票、印度裔青年照常投票。
- 结果：18–35 岁人群投票结果向印度裔派扭转 40%，整体扭转 6%，恰好让印度裔派赢得大选。
- 方法论意义：说服一个人「改投」很难，让特定人群「不投」容易得多；心理战的杠杆点是找到目标行为在不同群体间的**执行差**（这里是家庭管教强度），设计一个表面中立的触发器。
