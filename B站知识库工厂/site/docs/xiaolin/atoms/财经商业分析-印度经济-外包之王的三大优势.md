# 印度外包之王的三大优势

## 一句话定义
印度占全球一半以上外包业务，靠三大优势：人力成本极低（资深工程师年薪一万多美元）、庞大的理工科人才库（硅谷 1/4 科创公司由印度人创办）、近 3 亿人说英语（殖民遗产）——2019 年外包收入约 700–1000 亿美元。

## 详细说明
- **便宜**：管理十几人的资深工程师年薪一万多美元（纽约最低工资年化 3 万美元）；OpenAI 训练 ChatGPT 的人工反馈外包到印度，时薪不到 2 美元。
- **理工人才**：印度家庭极度重视理工科/编程教育；谷歌、微软、Adobe、IBM 等顶尖公司 CEO 均为印度本土读完大学后赴美的印度人；硅谷科创公司约 1/4 由印度人创办。
- **英语**：仅 10%–20% 受过良好教育者说英语，但绝对数近 3 亿（单拎出来居全球人口第四）；印度因此成为全球呼叫中心与技术支持中心。
- 客户面：谷歌、亚马逊、Adobe、英特尔、苹果、IBM 及几乎所有华尔街投行都把部分 IT/客服业务外包印度。
- 定位：外包是 1991 改革后印度崛起的支柱产业，也塑造了「跳过工业直奔服务业」的结构。
