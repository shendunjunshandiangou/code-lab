# 低借贷成本与杠杆放大

## 一句话定义

收敛交易的价差极小（十几个基点），必须靠杠杆放大收益；杠杆策略能否成立取决于借贷成本——LTCM 凭资本与声誉拿到与大投行相当的融资条件，杠杆放大 30–50 倍，把 1% 的价差收益变成 30%–50%。

## 详细说明

**为什么 LTCM 可以、个人不可以**：普通人乃至高净值客户借钱要付利率（如 3%）加抵押，成本高于价差，找到套利机会也无法盈利。LTCM 有几十亿资本、明星团队背书，投行们认定它低风险、抢着提供最优惠的借款与抵押条件，借贷成本低到与大投行相当。

**逻辑链**：模型找到微小价格不合理 → 超低成本借债加杠杆 → 收益被放大到可观水平。这也是当今大型量化基金的通用逻辑之一。
