# 普惠金融与民营数字银行

## 一句话定义

大银行做小微贷款不经济（固定成本高、单笔利润几百块还担风险），2013 年普惠金融上升为国家战略后，2014–15 年试点民营数字银行（微众、网商等），以极低运营成本填补小微金融空白。

## 详细说明

**小微金融的经济学困境**：银行竞争集中在大客户（大国企项目几十亿上百亿，赚 1% 即千万上亿）；小微企业"两万块过桥贷、下月就还"的需求，银行忙一通只赚几百块还要承担违约风险——不是看不上，是固定成本决定大范围做小微不经济。但中国有上千万小微企业被这类资金卡住。

**政策与试点**：2013 年十八届三中全会把发展普惠金融提到国家战略层面；2014–15 年首批试点五家民营银行，含 2014 年腾讯为第一大股东的微众银行（中国第一家数字银行）、2015 年背靠阿里的网商银行。

**数字银行成本优势**：纯线上、无网点——国内银行一个账户年运营成本约 20–100 元，微众约 2 元；以低 IT 成本作为传统银行的补充，起到资源配置"塞缝"作用。

**体系定位**：形成以国有大行为核心、股份制银行与民营银行多元覆盖的体系；"需要大钱大规模找政策性银行/六大行，需要渗透靠互联网"。
