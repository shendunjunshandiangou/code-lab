# 中日美游戏资本三路径

## 一句话定义

**欧美**：硬件生态+量大管饱 3A+并购填 Game Pass；**日本**：主机起家+小而美 IP；**中国**：盗版环境→征途 F2P→手游原始积累→全球投资。

## 详细说明

- TGA 穿透：日系 36%、中国 25%、美系 20%。
- 王者荣耀 2023 流水 391 亿 vs 黑神话全周期乐观 ~60 亿。
- 成熟市场头部 25 款占 73% 时长；巨头靠持续 IP 降风险非单款爆款。
