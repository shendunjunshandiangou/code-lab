# 洗碗机中餐场景本土化改造

## 一句话定义

欧美洗碗机为西餐浅盘设计，中餐的挑战是**深碗、重油**，厂商需围绕中餐场景做本土化改造，并逐一拆解"两怕一没有"：强水压冷水冲净重油（演示 10 秒洗净猪大油/蚝油/老抽）、抽屉式简化操作、水流悬浮喷淋臂自清洁、重型滑轨按 20 年寿命设计、免费上门设计解决安装改造。

## 详细说明

- 洗得干净且省水：家用机耗时长是因为清洗后还有漂洗、加热消毒杀菌、烘干流程；用水靠循环反复利用，比手洗省水。
- 抽屉式 vs 拉门式：拉门式需开门-拉碗篮-放碗-推回-关门，抽屉式只需拉开-放碗-关上；喷淋臂取消固定轴、靠水流悬浮，每次洗碗自行冲净。
- 耐用性对抗"坏了难修"：嵌入式安装维修需整机拆出，故用更贵的重型滑轨并隐藏在高压水冲不到处。
- 安装服务补"没地方装"：免费上门设计，锯窄柜门、垫高机身、现场切割封边，两趟上门即完成橱柜改造。
- 成本账（2024 年初）：抽屉式约 1.4 万 + 改造 600-800 元；主流拉门式线下均价六七千、线上四五千；每次洗涤耗材水电约一两元。
