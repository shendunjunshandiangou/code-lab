# 社区养老驿站与「一碗汤的距离」

## 一句话定义
养老驿站是嵌入社区的小型养老形态（住二三十人），提供助餐、上门助老和小规模入住照料；「一碗汤的距离」形容其最大优势——家人送来一碗汤还是热的。

## 详细说明
- **规模与分布**：北京官方名单共 814 家养老驿站，按能力分一型到三型（三型仅 13 家），服务一个街道、几个小区的老人。
- **服务分层与费用（北京，2023 前后）**：
  | 类型 | 月费 |
  | --- | --- |
  | 可自理/简单辅助（助餐、上门服务） | 几百到上千元 |
  | 居家住家保姆 | 5000–7000 元 |
  | 入住驿站照料 | 7000–9000 元 |
- **服务内容**：每周菜谱自选的助餐（自取或送上门）、修脚/理发/保洁、换尿管等医疗辅助。
- **运营要点**（向荣驿站案例）：技能好培训，难的是像照顾小孩一样用心——听老人反复倾诉、中立调解老人间摩擦；监控画面接入民政部门。向荣已实现盈亏平衡，而多数养老机构难以盈利——经济利益是行业可持续发展的底层推动力。讲者设想优质驿站未来可能像学区房一样带动周边房价。
