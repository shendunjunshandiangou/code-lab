# 新品类市场教育难题

## 一句话定义
全新品类产品最大的销售障碍不是渠道而是认知：消费者不知道它是干嘛的、更不会主动搜索，经销商和电商平台只能承接存量需求、无法替你教育市场。

## 详细说明
- Insta360 Nano 上线首月收入超 1000 万，两三个月后业绩下滑：经销商囤了货，但查设备激活量发现终端根本卖不动。
- 原因诊断：全景相机概念太新（2016 年），商场里消费者得听人解释半分钟功能+半分钟使用场景才明白，教育成本极高。
- 渠道的局限：经销商和亚马逊等电商平台适合成熟产品——用户有需求会搜索，价格/产品有优势就能成交；新品类**根本没人搜**，渠道再多也无用。
- 结论：新品类的市场教育只能品牌方自己做（Insta360 的解法见创意内容营销原子）。
- 侧写：如今全景相机应用已广（冬奥会/Super Bowl 转播、VR 看房、VR 游戏），说明教育成功后品类天花板会被打开。
