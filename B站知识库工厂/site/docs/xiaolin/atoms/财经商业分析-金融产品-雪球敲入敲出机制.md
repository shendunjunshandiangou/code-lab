# 雪球敲入敲出机制

## 一句话定义

雪球是挂钩股指的结构化产品：每月观察敲出价（如 103%）与敲入价（如 75%），未大跌则获高票息，跌破敲入且未回升则按指数跌幅亏本金，本质接近卖出看跌期权。

## 详细说明

典型结构：15% 年化票息、2 年、挂钩中证 500。

| 情形 | 结果 |
| --- | --- |
| 月收盘 ≥ 敲出价 | 敲出（自动赎回）：本金 + 已计票息 |
| 全程在两线间 | 到期：本金 + 全部票息 |
| 曾跌破敲入且未涨回敲出价 | 相当于持有指数，按跌幅亏本金 |
| 跌破敲入后涨回敲出价 | 敲出，仍获票息 + 本金 |

仅第三种严重亏损；其余获约 15% 年化——**大概率赚小钱、小概率亏大钱**。投资人常加 3–4 倍杠杆，跌破敲入触发追保，补不起则爆仓。
