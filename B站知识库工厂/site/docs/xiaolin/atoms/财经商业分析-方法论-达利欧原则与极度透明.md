# 达利欧原则与极度透明

## 一句话定义

桥水基金创始人瑞·达利欧在《原则》中的核心方法：自我成长五步循环（设定目标→发现问题→分析问题→得到方案→执行）；open-minded 开放式思维与对自己的极度坦诚（不给错误找任何借口、直面解决方案）；公司层面的 radical transparency 极度公开透明（所有话摆在明面上说 + 同事互评打分体系）。

## 详细说明

- 原则存在的理由：人的心理和潜意识充满非理性，事先定好的原则让人在生活、工作、投资中客观评价、理性决策。
- 桥水文化的筛选效应：很多人觉得互评+透明的文化残酷待不下去，但自信、渴望自我成长的人很适应；达利欧用这套原则把桥水做成全球最大对冲基金。
- 与巴菲特书信的互补：巴菲特给的是投资框架与商业价值观（坚持长期价值、在商业中做好人），达利欧给的是自我管理与组织管理的操作系统。
