# 老板视角与站队思维

## 一句话定义

站在老板立场思考：他要的是把事办好、把问题解决；下属不停解释让他窝火，一心帮着解决则让他暖心——向上管理就是传递「我跟你同一战线」的信号。

## 详细说明

两个典型场景（任务没说清、犯错延期）从老板角度出发点都一样：**怎么样我能把这个事给办好，把问题解决了**。

**老板的心理账本：**

- 问题解决好 → 老板脸上有光
- 问题解决不了 → 最后背锅的还是老板
- 下属不停解释 → 老板心里窝火
- 下属一心帮着解决问题 → 老板心里暖心

**站队思维 vs 言听计从：**

跟老板沟通最主要的不是让老板一时爽，而是让他知道你跟他**同一战线解决问题**——一致对外，不是说啥听啥，而是**共同发展**。老板是上级，下属辅助他；大家都在给组织办事。

**实践原则：**

只要开始站在老板立场思考问题，沟通会丝般顺滑。高级拍马屁的本质，就是传递「我在帮你把事办好」这一信号。
