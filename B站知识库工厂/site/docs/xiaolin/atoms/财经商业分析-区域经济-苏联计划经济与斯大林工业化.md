# 苏联计划经济与斯大林工业化

## 一句话定义

斯大林时代苏联实行中央计划经济，在方向明确时效率极高，1928 年后三个五年计划使苏联从农业国变为工业大国，但缺乏市场调节也为后期停滞埋下隐患。

## 详细说明

- **模式**：资源分配、生产乃至部分消费均按中央计划执行，无需等待市场自我调节
- **成效**：1920–1940 年人均 GDP 翻了三倍多（同期西方经历大萧条）
- **适用条件**：方向非常明确或需大规模整体改革时效率极高
- **后期问题**（1960 年代起）：经济结构复杂化后计划难以为继；领导「指哪打哪」导致贪腐、创新受损，1964–1985 年进入停滞时期
