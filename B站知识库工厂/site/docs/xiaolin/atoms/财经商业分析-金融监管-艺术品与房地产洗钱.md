# 艺术品与房地产洗钱

## 一句话定义

通过艺术品估值弹性与房产消费/证明功能，把脏钱转换为看似合法的高价资产或「卖房所得」等收入来源。

## 详细说明

### 艺术品

- 拍卖会隐私保护好、可收现金；估值波动大（1 万到 1 亿）。
- 常见套路：低价买入 → 「专家」高估 → 高价卖给自己的空壳公司；也可用于行贿（巴西较常见）。
- 2012 年墨西哥限制艺术品现金交易后，市场腰斩——合理推测原渠道含相当比例洗钱（并非所有交易都是洗钱）。

### 房地产

- **消费现金**：部分国家（如迪拜）可现金买房且不查收入。
- **财富证明**：空壳公司赃款买房再卖，卖房所得变合法收入，银行通常不再追问当初买房钱来源。
- **推高房价**：洗钱犯不太在意绝对房价、愿意加价，是推动迪拜、新加坡、香港、伦敦、纽约房价的因素之一。
- 新加坡案涉及 207 处房产，两人在迪拜有超 3100 万美元房产。

### 黄金（同类资产转换）

摩洛哥团伙：英国现金 → 巴黎 → 比利时换黄金（附伪造收据）→ 迪拜卖金 → 空壳账户；数年间 2.5 亿美元，2017 年因审计 Dubai 珠宝商异常曝光。
