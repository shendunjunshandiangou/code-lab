# 欧洲三驾马车与希腊救助

## 一句话定义

欧洲三驾马车（European Troika，欧洲央行 + 欧洲委员会 + IMF）为防止希腊破产引发欧元区多米诺效应，三轮向希腊提供约 2890 亿欧元救助，条件为财政紧缩（增税、减支）。

## 详细说明

**为何必须救**：
- 希腊破产会触发投资人抛售西班牙、意大利、葡萄牙国债
- 可能产生多米诺骨牌，引发欧元区整体信用危机甚至欧盟解体
- 德国不愿看到欧元区成立不到 10 年就解体

**三驾马车组成**：欧洲委员会、欧洲央行、IMF（简称「三马」）

**救助规模与轮次**：
| 轮次 | 规模（约） | 备注 |
| --- | --- | --- |
| 第一轮 | 1100 亿欧元 | 要求紧缩 |
| 第二轮 | 1300 亿欧元 | 2012 年，债务已达 170% |
| 第三轮 | 600 多亿欧元（3 年计划） | 2015 年希腊服软后，条件更苛刻 |
| **合计** | **约 2890 亿欧元** | 2018 年还清 |

**救助条件**：提高税收、减少支出、裁员降薪——以紧缩换贷款。

**立场逻辑**：三马背后是德法等成员国，非慈善机构；自身经济也难，借钱给希腊须保证能还、稳住市场情绪。
