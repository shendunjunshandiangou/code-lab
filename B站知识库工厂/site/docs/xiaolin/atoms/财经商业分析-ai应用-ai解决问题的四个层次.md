# AI解决问题的四个层次

## 一句话定义

AI 解决问题可分四层：简单问答、SOP 复杂问题、需自主推理的长程问题、虚拟与现实结合的开放问题——资本估值取决于 AI 能爬到哪一层。

## 详细说明

| 层次 | 类型 | 机制 | 代表形态 |
| --- | --- | --- | --- |
| 一 | 简单问题 | 问答、对话 | Chat 聊天框 |
| 二 | 复杂问题 | 有 SOP，按规则逐步执行，可控 | Coding Agent |
| 三 | 长程问题 | AI 自主推理如何一步步解决 | 大模型 + Coding Agent |
| 四 | 开放问题 | 虚拟思考落地到物理世界 | 世界模型、机器人 |

**资本逻辑**（蚂蚁灵光产品总监马竞）：Anthropic 估值反超 OpenAI，因资本看到 AI 能解决更复杂、更长程的真实问题，解放生产力并创造 GDP 一部分。
