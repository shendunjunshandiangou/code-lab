# 避税天堂零税率与所在地转移

## 一句话定义

香港、新加坡、摩纳哥及加勒比海岛的资本利得税为零，BVI、巴哈马连公司经营利润也免税；公司通过迁注册地、个人通过买房移居（巴哈马买房即成居民）把"所在地"挪到零税率地区，无形资产收入（书、歌、专利）则因产生地模糊可被转移到税收洼地。

## 详细说明

- 规模：全球跨境交易约一半经过避税天堂；BVI 三万多人口、总资产 1.4 万亿美元（人均 4000 多万美元）；全球人均 GDP 前几名（摩纳哥、卢森堡、百慕大、开曼）正因富豪扎堆。
- 赚钱地转移：实打实的经济活动改不了赚钱地，但知识产权收入难界定产生地——苹果、微软、辉瑞把海外研发基地放爱尔兰即此逻辑。
- Shakira 案例（连环空壳）：巴哈马买房拿税收居民身份（0%）→ 与卢森堡政府谈判把税率降到 2% → 架构为卢森堡公司 ← BVI ← 开曼 ← 巴哈马公司 ← 本人，中间层层空壳税率全零，最终只交 2%；多层壳是为隐藏名人身份。
