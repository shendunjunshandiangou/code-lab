# 以色列 2B 科技与被巨头收购模式

## 一句话定义

以色列科技公司多为 2B 模式：几十人团队研发尖端技术，然后以数十亿美元卖给靠近市场的美国巨头——这是"很少听说以色列科技大公司"的原因。

## 详细说明

**典型案例**：

- 芯片公司 Mellanox：2020 年被英伟达以 69 亿美元收购；
- 自动驾驶公司 Mobileye：被英特尔以 153 亿美元收购；
- 2C 导航应用 Waze（类似谷歌/高德地图）：被谷歌以 13 亿美元收购。

**模式成因**：

1. 以色列离消费市场远，自己发展 C 端成本高；
2. 投资者都是专业投资人，卖给大公司比独立发展更划算；
3. 以色列与美国关系紧密，美国巨头喜欢收购以色列科技公司——有潜力成为巨头的公司很快被买走。
