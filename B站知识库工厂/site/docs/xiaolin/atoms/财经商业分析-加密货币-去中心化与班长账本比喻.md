# 去中心化与班长账本比喻

## 一句话定义

加密货币的核心思想是去中心化：传统金融像班集体选班长老王记账（政府/央行/银行是中心化角色，财产安全依赖"老王人品"），2008 年金融危机让市场对央行信任骤降，中本聪提出让每个人手里都有一本独立账本、交易向全网广播记录的新体系——比特币；支撑它的两大技术是区块链与非对称加密。

## 详细说明

- 去中心化要解决的问题：没有班长时，如何保证所有人的"吆喝"都被听见、按规则记录、不可篡改、账户信息不泄密——区块链 + 非对称加密就是中本聪的答案。
- 体系、原理、代码全公开，谁都能发币：2021 年已有 4000 多种加密货币，比特币占 56%、以太坊 12%，本质大同小异。
- 判断新事物生命力的框架：不光看技术好坏，看它带来的收益是否显著大于被动蛋糕者的损失、应用范围是否够广——是则具备长期颠覆性。
