# 银行与投行角色区别

## 一句话定义
银行是"用自己的资产负债表做生意"：吸储放贷、赚利差、自己承担信用风险；投行（高盛、大摩、小摩、中金、中信等）是"卖服务的中介"：帮客户发股、发债、上市、交易衍生品，赚服务费/中介费，撮合之后的盈亏归交易双方——这也是投行被称为"卖方"（sell side，卖的是服务）的原因。

## 详细说明
- **银行（2.0 角色）**：钱是自己（储户）的，贷出去收不回来风险自担，所以偏好低风险固定收益的放贷。
- **投行（4.0 角色）**：不出本金，靠圈内投资人资源和项目包装能力牵线搭桥（"码农不会画 PPT，投行说你给钱我全搞定"）；只要客户有资本市场需求（发股/发债/IPO）就提供服务收费。
- **二级市场服务**：证券发行后的流通市场里，投行继续做交易服务——卖衍生品等特殊产品、提供市场流动性。
- **边界模糊**：现实中大公司业务全包——阿里、字节、腾讯本是典型融资方，做大后设投资部充当投资人，蚂蚁金服这类又充当中间商，"公司大了什么业务都做"。
