# 药店便利溢价逻辑

## 一句话定义

药店是 OTC 主战场，针对紧急小病的「现在就要」需求，地租 + 持证药剂师成本决定一定溢价；医保定点与外卖比价约束漫天定价。

## 详细说明

- 类比线下便利店：着凉了买感冒灵、吃坏肚子买止泻药等紧急场景。
- 开在居民区/商业区，需配备药剂师，成本高于普通零售。
- 大型连锁多为医保定点，药价上报受监管。
- 外卖（京东秒送、美团闪购）延伸覆盖 3–5 km；各省药店比价小程序加速竞争。
