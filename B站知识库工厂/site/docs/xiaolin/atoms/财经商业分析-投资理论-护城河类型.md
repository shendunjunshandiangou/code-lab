# 护城河类型

## 一句话定义

护城河（门槛/核心竞争力）决定商业模式被验证后竞争对手能否轻易复制；常见类型包括品牌、网络效应、用户粘性/转换成本、专利/牌照/高初期投入。

## 详细说明

| 类型 | 强度 | 例子 |
| --- | --- | --- |
| 品牌 | 强 | 苹果、茅台——溢价仍买、送人认 |
| 网络效应 | 强 | 微信、视频平台——赢者通吃 60–70%+ |
| 用户粘性/转换成本 | 较弱 | 电话卡、WiFi、健身卡首年优惠 |
| 专利/牌照/高投入 | 视情况 | 专利产品、限量牌照 |

**反例**：辣椒珍珠奶茶无专利，竞品跟进后价格战——产品好但不是护城河。

需选熟悉行业，能看懂公司如何运营赚钱。
