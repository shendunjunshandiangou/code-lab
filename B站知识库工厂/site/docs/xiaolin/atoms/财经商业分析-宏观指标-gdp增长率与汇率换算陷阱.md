# GDP增长率与汇率换算陷阱

## 一句话定义

官方公布的 GDP 增长率是本币剔除通胀的真实增速；用两年美元 GDP 相除算出的增速含汇率波动，短期评判常不合理，各国数字会对不上。

## 详细说明

除美国外，各国「公布增长率」与「美元 GDP 两年相除增速」常不一致——都没算错。

**中国 2024**：
- 美元 GDP：18.74 万亿 vs 18.27 万亿 → 相除仅 **2.6%**
- 官方公布：**5.0%**（本币不变价）

**中国 2021**：
- 官方 **8.6%** vs 美元数相除 **21.3%**

**原因**：
- 官方增速 = 本币真实生产增长（剔通胀）
- 美元相除 = 含汇率波动，短期不合理
- 长期有人仍爱看美元口径，见仁见智

读经济数据时务必分清：说的是哪个口径的增长率。
