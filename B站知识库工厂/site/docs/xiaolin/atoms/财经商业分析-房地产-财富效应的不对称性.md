# 房地产财富效应的不对称性

## 一句话定义

房价上涨时多数家庭不卖套现、未买者反而攒钱，消费拉动不明显；房价下跌时负财富效应极强，对总需求打击远大于股市波动。

## 详细说明

- 股票涨跌通过财富效应影响消费，但对经济体总量是换手
- **上涨**：一套房家庭暗爽但不卖；准买家攒钱 → 效应抵消，甚至消费下降
- **下跌**：资产缩水 + 高杠杆 → 消费骤降
- 2000 互联网泡沫蒸发 6.2 万亿美元，美国零售仍增长；2008 房市蒸发 3 万亿，零售连续 10%+ 负增长
