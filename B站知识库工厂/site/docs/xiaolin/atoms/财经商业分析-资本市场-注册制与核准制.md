# 注册制与核准制

## 一句话定义
注册制（美国）下证监会只审材料、材料合格即可注册上市；核准制（中国传统）下证监会还要审查公司背景、决定你能不能上，周期更长——这是互联网巨头多赴美/港上市的重要原因之一。

## 详细说明
- **注册制**：公司备好材料交证监会，材料审核通过即可注册上市。
- **核准制**：先审材料，再查公司背景「看你到底行不行」，能否上市由证监会决定，公司只能等。中国过去为核准制，现正逐板块向注册制转化。
- 互联网公司赴美/港上市的原因组合：①核准制排队时间长（机构投资者急于退出，不愿等三四年）；②A 股挂牌对盈利有要求，而互联网公司上市时几乎都不盈利（美国的互联网公司也一样）；③引入了海外投资人/美元基金，股东成分不符合 A 股要求。
