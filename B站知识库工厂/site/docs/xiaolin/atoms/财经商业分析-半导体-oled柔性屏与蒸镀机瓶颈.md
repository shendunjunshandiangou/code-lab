# OLED 柔性屏与蒸镀机瓶颈

## 一句话定义

柔性屏依赖 OLED（可弯）而非 LCD（硬）；三星曾垄断 OLED 及蒸镀机供应，国内京东方 2017 年才排到设备，2019 前后第六代产线量产支撑国产折叠屏。

## 详细说明

- 2013 三星 CES 展示柔性 OLED；2018 柔宇 FlexPai；2019 三星 Fold / 华为 Mate X
- 蒸镀机当时几乎仅日本一家能产，多被三星买走
- 三种能量产形态：横折内折、横折外折、竖折
