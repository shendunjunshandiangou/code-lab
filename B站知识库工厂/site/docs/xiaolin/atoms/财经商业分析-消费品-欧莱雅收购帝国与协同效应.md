# 欧莱雅收购帝国与协同效应

## 一句话定义

全球最大美妆集团欧莱雅（营收占全球美妆七分之一、比第二名高 60%）旗下近 40 个品牌只有 3 个是自己的，其余全靠收购；集团化的核心价值是协同效应——研发、供应链、渠道全品牌共用一套，典型如 1996 年收购美宝莲买的就是其铺好的中国线下渠道。

## 详细说明

- 收购史：1909 年染发剂起家；1957 年资本高手达勒任 CEO 后，1963 上市、1964 收兰蔻（高端化）、1971 收碧欧泉（进军男士）、1985 收阿玛尼香水线，形成集团化格局。
- 美宝莲案例：收购看中的不是品牌而是渠道；2003 年再收国内生产线——此后 YSL、薇姿、理肤泉进中国"粮草和路都铺好了"。家族企业比喻：爸爸先打江山疏通关系，孩子们进来跟叔叔阿姨喝喝酒就行。
- 完整闭环：收购积累体系（研发/供应链/渠道/营销）→ 再收购有潜力的小品牌 → 体系助推小品牌全球化盈利；外因是全球化几十年总有高增长经济体（近十几年是中国，现为其全球第二大市场）承接扩张。
