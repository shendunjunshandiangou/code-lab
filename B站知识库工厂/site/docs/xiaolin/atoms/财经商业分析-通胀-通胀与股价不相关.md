# 通胀与股价不相关

## 一句话定义
IMF 对 71 个经济体的回归显示：无论发达还是发展中国家，股价表现与通胀不相关，且高通胀期股票一般下跌——中介变量是政策：高通胀触发财政货币双紧缩，紧缩才是压垮股价的原因。

## 详细说明
- 直觉误区：以为通胀→物价全涨→资金涌入股市房产抗通胀→正相关；数据不支持。
- 政策中介机制：高通胀→政府与央行大概率紧缩→股价下跌。
- 两个例外印证机制：①土耳其高通胀期未采取紧缩，股票一年翻三倍——没有紧缩就没有下跌；②能源公司股票在高通胀期历史表现优异（2022 年埃克森美孚成全球表现最好公司之一），因为它们是成本推动型通胀的受益方。
