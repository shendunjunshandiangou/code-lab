# 瑞士永久中立与刺猬战略

## 一句话定义

瑞士 1815 年确立永久中立后未参与外国战争；二战靠「国家堡垒计划」刺猬战略（自毁桥梁隧道、上万掩体）与金融圆滑（无差别接纳各方资金）躲过战火。

## 详细说明

**地理**：阿尔卑斯山+侏罗山+莱茵河/博登湖环绕，连接外界仅靠少量隧道桥梁，军事上易守难攻。

**刺猬战略**：反坦克炮藏山体/伪装谷仓；关键通道埋炸药随时自毁（部分至 2015 年才清理）；上万掩体。希特勒曾制定攻瑞计划。

**圆滑**：二战期间瑞郎为全球唯一可自由流通兑换货币；1998 年承认从纳粹获得当时价值 4.4 亿美元黄金。

**历史**：13 世纪起雇佣兵闻名（梵蒂冈瑞士卫队至今）；1815 年维也纳会议确立中立，吸引 50+ 国际组织总部。
