# 各国关税谈判分层格局

## 一句话定义

2025 年美对各国关税大致分层：铁哥们 10%、谈判让步国 15%–20%、强硬/逆差国 25% 起叠加惩罚；特朗普以投资、政治、市场开放灵活要价。

## 详细说明

| 层级 | 国家 | 结果要点 |
| --- | --- | --- |
| 10% | 英、澳 | 英最快；10 万辆汽车配额 10% |
| 15%–20% | 欧盟、日韩 | 欧：15% + 7500 亿能源 + 6000 亿投资；日韩：巨额对美投资换 15%（含汽车） |
| 25%+ | 中国、印度 | 中：日内瓦战术休战，有效关税估 ~32%；印：26%→25% 后再因俄能源购 +25% 至 ~50% |

**欧盟内部分歧**：德国护汽车、法国护农产品、东欧重安全。

**谈判逻辑**：有钱（欧日韩瑞）→ 要投资；有政治需求（巴加）→ 谈政治；依赖度低（印）→ 腰板更直。

除中印外，多数国家口头不满、私下实质性让步。
