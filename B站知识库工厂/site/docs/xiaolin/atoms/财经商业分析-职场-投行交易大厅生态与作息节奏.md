# 投行交易大厅生态与作息节奏

## 一句话定义
投行二级市场业务集中在 trading floor（交易大厅）：大平层、两排对坐、人均八块屏幕；作息由市场决定——债券岗比股票岗早、MD 比员工早（四五点起步），刚开市与闭市前是全天最忙的两个时段。

## 详细说明
- 空间布局：大平层一排排对坐（如《华尔街之狼》），屏幕多到看不见对面的人；外圈办公室属于大 MD，但他们平时也坐在大家中间。辨认老板两招：屏幕少（不用盯盘、主要打电话）+ 坐一排最外侧（方便沟通）。
- 作息梯度：普通员工约 8 点到岗（股票晚、债券早）；大 MD 六点、六点半到（要跟伦敦汇报），住城外的四五点即开始工作；有 MD 凌晨三点多起床先跑五公里。
- 交易日节奏：9 点半开市钟一敲，大厅如菜市场——打电话、隔排对喊、柱子三面挂电视；trader 需同时盯订单、聊天窗口、Bloomberg、新闻和自己 book 的仓位与风险敞口。**开市（订单涌入）与闭市前（整理对冲）最忙**——「下午三点半找股票 trader 聊天 is not a good time」。
- 情绪常态：砸键盘、扔鼠标、骂人是压力发泄的常见形式；做成大单则全组欢呼。根源是资金量巨大——行话「百万叫 buck、十亿叫 yard」，「five bucks」= 500 万美元；旁边斯文的小哥可能一天赚 5000 万美金。
- 午餐文化：桌上吃、无午休；新人负责给全组买饭；很多人午餐只吃沙拉——少吃防下午犯困，健康形象是给外人看的（晚上回家狂吃）。
