# GDP定义与核算边界

## 一句话定义

GDP（国内生产总值）是特定时期内一国区域内生产的所有最终产品的市场价值，但只计有偿交易、只计最终产品、按生产地域而非所有者国籍统计。

## 详细说明

**三个易忽略边界**：

1. **地域原则**：中国人在越南全资建厂 → 算越南 GDP；算国民属性用 GNP
2. **最终产品**：中间品（如卖给餐馆的鸡蛋）不算，避免重复；终端产品（西红柿炒鸡蛋）算
3. **有偿交易**：无偿劳动（父亲给儿子做玩具）不算——无市场价、难记录

**常见计入/不计入**：
- ✅ 新商品购买、改装增值、新房、终端餐饮、库存产出、中介服务费
- ❌ 二手交易、股票买卖及涨跌、金融投资、二手房（翻新增值部分除外）
- 黑市/非正规：理论上应算，实际难统计（中低收入国约 1/3、发达国约 15%）

股票涨跌不直接进 GDP，但可能通过财富效应间接带动生产与消费。
