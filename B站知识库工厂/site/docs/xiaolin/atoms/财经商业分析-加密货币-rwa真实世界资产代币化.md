# RWA 真实世界资产代币化

## 一句话定义

RWA 将现实资产（房产、债券、基金等）映射上链；稳定币是 RWA 一种；贝莱德链上货币基金 BUIDL 规模约 29 亿美元，蚂蚁参与香港 Ensemble 沙盒做充电桩/光伏 RWA。

## 详细说明

- 链上资产不可篡改、交易透明，可挂智能合约
- 贝莱德 2025 年 3 月与 Securitize 发布链上美元货币基金 BUIDL
- 蚂蚁研究充电桩、光伏、换电站 RWA，参与香港 Ensemble 沙盒
- 香港、新加坡视 RWA 为加密世界潜力领域，亦是稳定币主战场
