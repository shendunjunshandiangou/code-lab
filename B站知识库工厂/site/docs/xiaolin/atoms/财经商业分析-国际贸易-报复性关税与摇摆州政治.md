# 报复性关税与摇摆州政治

## 一句话定义

美国关税政策受摇摆州关键产业工会影响；对手国报复性关税常精准瞄准这些州（如加拿大 2018 年清单），形成贸易谈判中的「痛点外交」。

## 详细说明

**美国侧**：钢铁关税惠及宾夕法尼亚（匹兹堡钢铁之城）；钢铁工会超 120 万人、总部在滨州——关键摇摆州。特朗普、拜登、小布什、卡特、里根均实施钢铁保护。

**加拿大 2018 报复**：名义对等 125 亿美元，清单瞄准亨氏番茄酱（滨州）、橙汁（佛州）、卫生纸/蛋黄酱（威斯康星）——专打摇摆州。

**结果**：约一年后美加撤销钢铁关税，达成 USMCA。
