# 上市直接与维护成本

## 一句话定义
上市要花一年到数年时间和数千万级费用（投行抽成几个点+律师会计师），且有达不到挂牌盈利/资产门槛而失败的风险；上市后还有审计、发布会、IR 部门等持续维护成本。

## 详细说明
- **时间**：美国上市约 1–1.5 年；国内 1–2 年，长则 3–4 年，是全公司级的大工程。
- **金钱**：投行先收几个点（融 100 亿拿走一两亿），另需律师、会计师，数百万美元级；账上没几千万办不了。
- **失败风险**：交易所对盈利业绩有要求（如纳斯达克对现金流/资产/收入的细致门槛），花了钱也可能上不去。互联网公司早期烧钱不盈利，美国互联网公司上市时几乎没有一家盈利——这也是它们难上 A 股的原因之一。
- **维护成本**：季报年报编制、审计公司审核费用、财报发布会、回应媒体与华尔街分析师提问——需要专门的 IR（Investor Relations，投资者关系）部门。
