# 现金密集型产业洗钱

## 一句话定义

最经典的洗钱入口：利用餐馆、洗衣店、赌场、金店等日常有大量合法现金进出的产业，把脏钱混入报表，同时解决消除痕迹与合理化收入。

## 详细说明

- **操作示例**：洗衣店日赚 1 万，混入 5000 脏钱报 1.5 万也合理。
- **历史**：1920 年代美国黑帮开洗衣店，「Money Laundering」词源。
- **局限**：依赖实体业务量，效率难上去；毒枭不会苦心经营洗衣店；现多作链条入口，通过代持或空壳公司隔离风险。
- **赌场洗钱（变种）**：去赌场用现金换筹码，象征性玩几把再换回钱/支票，称「赌博赢来的」——近二三十年大型赌场监管趋严，多用于小额。
