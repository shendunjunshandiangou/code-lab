# 奥委会与主办国转播费分成博弈

## 一句话定义

奥运转播费因品牌 IP 而非主办地产生，IOC 从早期 1%–4% 逐步提升至 2000 年前后约 50%、2016 里约近 70%，与主办国形成长期分成博弈。

## 详细说明

**早期 IOC 较理想主义：**
- 1948–1968：IOC 从转播费拿约 1%–4%
- 至 1980 年约 10%

**变化动因：**
- 转播费规模暴涨；IOC 自身商业化；观众看的是「奥运会」品牌，与举办地无关。

**分成演变：**
- IOC 要求比例升至三四十%
- 2000 年前后约 50%（字幕「5%」疑 ASR 误识）
- 2016 里约时 IOC 分成接近 70%

**对比：** 夏季奥运转播费约 25–30 亿美元；世界杯约 40 亿美元。无论主办国盈亏，IOC、FIFA 等 IP 方往往获利丰厚。
