# 股权转债权受偿优先级套利

## 一句话定义

公司濒危时大股东用分红现金回购自家高息债券，实质是**把股权转换成债权**：一旦破产清算，股票基本一分不值，债权却排在受偿顺位前列、可通过变卖资产获得偿还——同时还能吃到信用恶化带来的超高票息（许家印两笔恒大债票息 13% 和 13.75%，为 2018 年新加坡全市场发行中最高），一鱼两吃。

## 详细说明

- 形象比喻：公司像一艘正在下沉的巨轮，大股东一边变卖船上值钱的玩意儿，一边赶紧往船顶跑（船沉时最后被淹的位置）。
- 高票息的来源：正因公司信用出问题才发得出 13%+ 的债——大股东比市场更清楚公司底细，却以「借钱给自己公司」的姿态获得高收益。
- 与「信心信号」的反差：小额认购自家债是对公司有信心的表现；大规模包圆式认购则更可能是清偿顺位安排。
- 制度背景：整套操作利用不同国家/地区规则差异（BVI 实体、境外分红、新加坡发债），站在每个地区都合规——国际资本游戏的猫鼠博弈本质。
