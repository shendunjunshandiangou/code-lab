# 流动比率财务健康

## 一句话定义

流动比率 = 流动资产/流动负债；>1 必须，建议 >1.5；&lt;1 可能一年内资不抵债。

## 详细说明

类比丈母娘看资产负债：高不一定优秀，低则非常危险。与盈利能力配合使用。
