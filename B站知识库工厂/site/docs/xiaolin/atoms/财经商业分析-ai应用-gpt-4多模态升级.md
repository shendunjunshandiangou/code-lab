# GPT-4 多模态升级

## 一句话定义

GPT-4 相对 GPT-3.5 显著升级图像理解（常识+幽默）、回答准确性及各类考试表现——美国律师考试从最差 10% 跃至前 10%。

## 详细说明

- **多模态**（API 图片输入）：识别 VGA 插手机、出租车后熨衣等非常态场景；手绘草图 → 网页代码
- **准确性**：相对 3.5 各类考试分数普遍提升
- **玩法**：Swift 动画、贪吃蛇、律师函、智能合约等——**不到半年**能力跃迁（讲者）
- **商业**：OpenAI 微软投资后不再公开参数量；Plus 订阅可用 GPT-4，部分功能（图片输入）初期仅 API 开放
