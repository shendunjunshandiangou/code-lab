# 技术分析局限与量化碾压

## 一句话定义

技术分析是"看图说话找规律"（分析历史价格、成交量、波动率预测未来），但个人投资者不宜过分依赖：量化机构在算法与速度上形成碾压，且量价规律只对应短期供需，不适合中长期投资。

## 详细说明

**技术分析是什么**：分析股票过去的走势（价格、成交量、波动率等），预测未来价格变化——股评人指着 K 线讲"双肩""放量""量涨价涨建议买入"即属此类。

**不建议个人投资者依赖的两个原因**：

1. **量化机构碾压**：量化对冲基金、交易公司雇佣数学/物理 PhD 建模找规律（iceberg 冰山模型、ninja 忍者模型等），市场供需一出现微小偏差，算法资金自动进场；还会对新闻做 Text Mining（文本挖掘），把文字转成交易信号。个人投资者没有算法、机器、服务器，无法竞争。
2. **周期错配**：价格短期由供需决定、长期由价值决定；量价分析适合短线，而个人投资者持仓通常为几个月到几年的中长期，用技术分析风险大。
