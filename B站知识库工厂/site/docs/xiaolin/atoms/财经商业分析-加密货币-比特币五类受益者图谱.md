# 比特币五类受益者图谱

## 一句话定义

分析比特币生态"谁在吃蛋糕"的五类受益者：①投机者（波动率高+流动性好+概念酷必然招投机，反过来又推高波动）；②机构投资者和公司（组合分散风险）；③蹭流量者（媒体与马斯克式舆论玩家）；④产业链中间商（挖矿产业不大，真正赚钱的是收手续费的交易所）；⑤黑市交易者（当时每秒约 7 笔的性能限制下，实际支付应用多集中于黑市）。

## 详细说明

- 分析方法论：看一个新事物的生命力，先画受益者图谱——谁受益、谁被动蛋糕、受益是否大于损失。
- 投机≠骗局：大多数金融产品都有投机属性，比特币只是发挥到极致；年化波动率动辄 100%+，是美股五倍。
- 蹭流量的间接助推：媒体不买币但追逐流量的报道推波助澜币价；马斯克"Bitcoin to the moon""狗狗币上月球"即典型。
- 主流化渗透：Mastercard、PayPal 开始接受比特币；Morgan Stanley 向高净值客户开放交易、高盛开发业务、多家公司排队申请加密货币 ETF。
