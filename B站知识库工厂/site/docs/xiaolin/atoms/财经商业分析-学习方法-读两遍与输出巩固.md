# 读两遍与输出巩固

## 一句话定义

读长文/论文先忽略细节抓框架（摘要、资产负债表、根本原因），第二遍按需补细节；通过书写脉络或 presentation 输出以巩固记忆。

## 详细说明

- 第一遍：摘框架性信息（如「一文读懂」类结构）
- 第二遍：针对具体问题补充细节
- 输出：合上书写脉络、做汇报；无人逼时也自建输出环节
