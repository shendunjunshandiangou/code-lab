# 霍华德·休斯的跨界征服式创业

## 一句话定义
霍华德·休斯（1905-1976）以"做到顶尖就厌倦、换个领域再征服"的模式横跨石油、电影、飞机制造、民航、拉斯维加斯地产赌场多个行业——19 岁执掌家族石油公司，24 岁拍出影史最贵电影，30 岁造出世界最快飞机，38 岁前创下 91 小时环球飞行纪录并获国会奖章，39 年收购环球航空（TWA）扭亏为盈独霸民航十年；斯坦·李承认钢铁侠的灵感原型正是他。

## 详细说明
- **起点**：德州石油大亨之子，16、18 岁时父母先后病逝，经与亲戚诉讼后 19 岁成为石油公司董事长兼唯一主人；随即从莱斯大学辍学、雇好管理层撒手不管——"有钱、有时间、有才华、没人管"。
- **驱动力是征服欲而非财富**：第一部制片电影被伯父骂"破烂"、无人关注这个纨绔子弟，埋下"我不服"的种子；此后每个领域的进入几乎都始于受挫或被轻视（飞行比赛倒数→自建飞机公司造出 H-1，时速 566 km/h 破世界纪录；被赌场老板驱赶→掷资 1300 万买下酒店）。
- **飞行带动产业**：91 小时环球飞行（1938，纽约往返，2.5 万人接机）让公众从害怕坐飞机转为向往，随后顺势收购濒临破产的环球航空，提供跨大西洋民航航线，连续十年称霸全球民航——个人 IP 宣传直接转化为产业机会。
- **结局的另一面**：严重强迫症晚年恶化（怪癖缠身、与世隔绝），无子女，助手趁机稀释其商业帝国；一生收购的酒店、赌场、餐厅、航空公司、电台最后无一盈利，1976 年去世后 400 多人争夺遗产。
- **UP 主点评**：天才与疯子的界线本就模糊，人类的进步常由有才华的"疯子"推动；all in 是很多大佬的共性——想要人上人的收益，就得承担倾家荡产甚至丧命的风险。
