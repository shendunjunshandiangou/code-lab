# 信用深度绑定与 credit crunch 链条

## 一句话定义

房产贷款以房子为抵押且银行承担跌价风险，使房地产与信用体系深度绑定；房价暴跌可触发比股市泡沫更可怕的信用危机（Credit Crunch）。

## 详细说明

**信用扩张**：17 个发达国家贷款/GDP 从 1950 年约 15% 升至近 70%，是过去 50 年发达城市房价涨幅超工资的核心推动力。

**与股票抵押对比**：股票杠杆跌价可清仓，券商风险低；房产跌价时银行收房卖出仍可能亏，且交易成本高。

**链条**：房地产崩 → 信用崩 → 银行崩 → 经济崩（流动性抽干、企业破产、总需求下降）。

- IMF：2008 年 23 个双泡沫国家中 21 个严重衰退
- 日本 90 年代、欧债、美国次贷等重大危机多与房地产相关
- 修复银行体系极难；欧元区国家不能自行印钱救银行
