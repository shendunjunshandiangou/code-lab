# 空壳公司洗钱网络

## 一句话定义

当代洗钱核心手段：在避税天堂匿名注册大量无实际业务的空壳公司并开户，作为资产网络基本盘，利用跨国监管差异层层转移赃款。

## 详细说明

- **注册便利**：英属维尔京群岛、百慕大、开曼、巴拿马等——网上几分钟可注册，无需个人信息，雇当地人当负责人。
- **开户**：可通过代理人或信托公司受托开户。
- **墨西哥黑帮经典套路（1990 年代）**：卢森堡空壳公司 A 存大额定期存单 → 直布罗陀公司 B 在巴黎以存单担保贷款 → 贷款进入巴拿马空壳公司 C——跨三国银行保密法设障。
- **合法用途**：空壳公司在合伙控股、税务规划中广泛使用，问题出在部分避税天堂监管不到位。
- **2017 年后**：迫于美欧反洗钱压力有所加强，但推进「推一下挪半步」。
- **核心逻辑**：利用国家间监管规则不同；若全在一国政府易够着，跨国则人力与时间成本巨大。
