# 美国推动稳定币三大目的

## 一句话定义

美国加急推进稳定币立法主要为：维持美元主导地位（99% 挂钩美元）、增加短期国债需求（次要）、争夺加密世界话语权——非主要为 3 年后 1 万亿短债需求。

## 详细说明

1. **美元化**：非美元国家广泛使用 ≈ 变相放弃本币货币政策。
2. **美债**：泰达 2024 年净买入美债 331 亿（全球第七）；但多为 3 月内超短债，美联储利率影响更大——小Lin 认为非主因。
3. **话语权**：压不住就拥抱；欧洲、新加坡、香港已先行立法。
4. **特朗普**：支持加密、禁止数字美元、家族发币等。
