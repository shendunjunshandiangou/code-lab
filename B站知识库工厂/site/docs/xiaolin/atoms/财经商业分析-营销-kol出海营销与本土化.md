# KOL 出海营销与本土化

## 一句话定义
小众垂直品类难以直接触达目标用户，但领域 KOL（海外叫 Influencer）自带精准粉丝群，且天然完成本土化——用本土语言、文化、场景展示产品，远胜官方宣传片的机械翻译。

## 详细说明
- Insta360 借鉴了当时国内新兴电子产品出海的两个标杆：一加手机与大疆 DJI 的 KOL 打法。
- 匹配逻辑：全景相机太小众，广撒网找不到目标用户；找到攀岩/冲浪/滑雪等领域 KOL，让他给粉丝一讲，精准用户全知道了。
- 成功要素：①产品新颖好玩，KOL 自愿尝试；②拍出的影像本身精彩，社交媒体传播度高（One 曾冲上日本推特热搜前十）；③KOL「超额完成」本土化——东西最终卖给当地人，KOL 用本土语言/文化/场景展示，比官方宣传片翻译成各国语言有效得多。
- 适用条件：产品具备内容生产属性（拍出来的东西本身就是广告素材），营销与产品形成飞轮。
