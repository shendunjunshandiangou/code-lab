# 美国全球征税与海外囤积现金

## 一句话定义

美国自 1920 年起对本国公司实行**全球征税**（注册在美国，全球利润都要向美国交税），但留了一个口子：**海外利润只要不汇回美国就先不用交**——于是美国跨国公司把利润长期囤在海外，苹果 2017 年海外现金达 2500 多亿美元，比当时整个英国的外汇储备还多。

## 详细说明

- 逻辑类似美国公民/绿卡持有者的全球收入都要向 IRS 报税，但公司版本存在「递延」口子。
- 这个口子是整套海外避税架构的第一层：利润先用转移定价集中到低税率国家，再无限期滞留海外，美国 35% 的企业所得税（2017 年前）便一直悬而不缴。
- 规模：2018 年美国跨国公司仅在爱尔兰囤积的现金就超过 1 万亿美元。
- 上世纪 90 年代美国政府内部资料显示，监管层清楚跨国公司在避税天堂存钱却睁一只眼闭一只眼——因为料定这些钱早晚会回流美国；2017 年减税法案（汇回优惠税率 + 此后不论汇回与否都征税）正是收网动作。
