# 风险调整后收益评价

## 一句话定义

评价投资回报不能只看收益率，还必须衡量风险；麦道夫基金 15% 年化且零波动的「神迹」曲线，在风险调整后反而极度可疑。

## 详细说明

### 核心原则

- 光看收益不够，必须同时看**风险**
- 银行存款 3% 你也愿意存，因为几乎**零风险**
- 高收益 + 低/零风险 = 极度反常，应高度警惕

### 麦道夫案例

| 指标 | 标普 500 | 麦道夫基金 |
| --- | --- | --- |
| 走势 | 波动明显 | 一直上涨、无波动 |
| 年化收益 | 与市场相关 | 约 15% |
| 风险 | 市场风险 | 看似为零 |

讲者评价：麦道夫回报曲线「跟存银行差不多」，但收益率是 15%——如果真有这么个基金，简直就是神一般的存在。

### Markopolos 的量化发现

Harry Markopolos 用公开数据模拟发现：

- 麦道夫收益率与美股收益率**相关系数不为零**（应有关联）
- **波动率几乎为零**——不管市场暴涨暴跌，永远最赚钱

若真用期权套利对冲，「把全世界期权买过来都对冲不了这个风险」——数据上不可能实现。

### 巴林银行里森案例（补充角度）

1994 年里森靠幽灵账户对倒做出 2800 万英镑虚假利润时，**其他银行的资深交易员**已经起疑：在基本对冲掉风险的情况下不可能赚这么多，收益风险比（Sharpe ratio，夏普比率）高得离谱。但巴林管理层因不理解衍生品业务，把质疑斥为「嫉妒」——同行的风险调整视角是有效的欺诈预警，管理层的失明才让骗局延续。

### 投资启示

完美直线收益曲线是识别欺诈的重要信号，专业机构和散户都应从风险调整角度审视「稳赚不赔」承诺。
