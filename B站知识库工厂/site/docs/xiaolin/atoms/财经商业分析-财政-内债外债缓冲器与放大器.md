# 内债外债：缓冲器与放大器

## 一句话定义
内债（本币国债）可靠央行印钱变相兜底、是经济的缓冲器；外债（通常美元计价）印不出来、资本来去放大周期、是经济的放大器——历史上的国家级债务崩溃几乎都源于外债违约。

## 详细说明
- **内债的变通**：完善体制下政府与央行分开、不能直接印钱给政府，但央行可以去市场上买国债（即量化宽松），中间多倒一手，本质是政府管央行借钱。金本位时代尚有约束，法币时代政府较容易靠印钱规避内债违约——「内债不是债」说法虽极端（内债也是债），但可控范围大得多。
- **外债的脆弱**：以美元计价、印不了，现金流出问题只能求人，极易违约。债务即杠杆：形势好时资本涌入放大优势，出问题时资本齐撤加剧崩盘。
- 案例：①七八十年代拉美——外资涌入后石油危机+美元加息，阿根廷/巴西/墨西哥集体违约；②欧债危机——欧元区把内债变成了外债（成员国无法印欧元），希腊/意大利/西班牙濒临违约只能受欧盟与 IMF 紧缩条件救助，财政政策从调节器变成波动放大器；③97 亚洲金融危机、98 俄罗斯、01 阿根廷、2022 斯里兰卡，直接原因都是外债还不上。
- 内债违约统计（1980–2022）：全球共 84 次；大国内债违约基本与外债一起发生（98 俄罗斯、01 阿根廷、2012 希腊）——外债反正要违约、信用板上钉钉受冲击，干脆内债一块赖掉。阿根廷 20 年破产三回。
- 美国的特殊性：全世界的美元债对它都是「内债」。
