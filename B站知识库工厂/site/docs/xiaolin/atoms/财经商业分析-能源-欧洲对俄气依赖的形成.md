# 欧洲对俄气依赖的形成

## 一句话定义

欧洲对俄罗斯天然气依赖度从 2010 年的 26% 升至 2019 年的超 40%（德国超 50%），且 2018 年还在推北溪二号，其成因是「居安不思危」的三重叠加：①主动收缩本土供给——英国 2000 年代停掉大部分海上油气井、多国关闭储气设施（维护贵，反正俄气便宜）、德国产量躺平；②福岛核事故后弃核——法国全停检修（至今一半未复）、德国决定关停全部核电站；③三个心理支撑——便宜方便、自认甲方（俄 45% 财政收入靠油气、天然气出口 80% 输欧，「它更需要我」）、绿色转型过渡心态（再过二三十年就不需要化石能源了，依赖只是暂时的）。

## 详细说明

- 讽刺的注脚：特朗普 2018 年在联合国警告德国过度依赖俄气，在场德国官员不屑而笑——正因为这话出自特朗普之口。
- 结构性矛盾：欧洲一边大幅降低本土化石能源开采、投入清洁能源研发，一边仍高度依赖俄气——青黄不接的转型窗口期为半个世纪最严重的能源危机铺好了路。
- 「甲方幻觉」的破产：2022 年欧盟以为自己是主导方而放话制裁，结果俄石油转卖中印、能源收入反增 38%，欧洲反被断气拿捏（详见 [财经商业分析-案例-北溪断气与欧洲能源反噬](/xiaolin/atoms/财经商业分析-案例-北溪断气与欧洲能源反噬.html)）。
