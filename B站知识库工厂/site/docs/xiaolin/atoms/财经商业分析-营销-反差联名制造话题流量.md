# 反差联名制造话题流量

## 一句话定义
两个品牌的反差感越大，联名越自带出圈话题；话题带来传播与曝光，相当于抓住免费流量，大幅降低营销获客成本——这是"奢牌 x 大众品牌"这类大尺度跨界联名的核心收益。

## 详细说明
- **Fendi x 喜茶**：表面看奢侈品跟大众茶饮联名会"拉低身价"，实则各取所需——Fendi 近年缺爆品出圈，需要触达一二线精英之外的潜在客户；喜茶则借 Fendi 拉高品牌。巨大反差本身成了出圈话题：不少人"人生第一次全款入手 Fendi"，连雷军都发微博晒联名款顺便给小米 13 打广告。
- **特斯拉 x 赫莲娜**：大尺度跨界本身自带话题，马斯克和特斯拉都是话题之王；有话题就有传播和曝光，品牌塑造与潜客触达的效果才能最大化。UP 主判断这次是赫莲娜主动找的特斯拉——借话题之王出圈。
- 前提仍是气质协同：反差要建立在可自洽的共同调性上（见协同效应原子），否则会翻车。
