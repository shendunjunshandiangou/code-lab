# 双层爱尔兰与荷兰三明治

## 一句话定义

2014 年前爱尔兰税法不按经营地征税，而**按公司由谁管理和控制征税**——跨国公司让爱尔兰子公司被百慕大等地的另一家公司控制，就连爱尔兰 12.5% 的低税率也不用交；配合在百慕大、荷兰、爱尔兰设多层分公司倒转利润的完整架构，得名 **Double Irish（双层爱尔兰）** 与 **Double Irish with Dutch Sandwich（双层爱尔兰加荷兰三明治）**。

## 详细说明

- 完整链条：成本分摊协议让爱尔兰公司享有知识产权 → royalty 把各国利润集中到爱尔兰（各国无利润）→ 钱不汇回美国（美国税递延）→ 爱尔兰公司实际控制人在百慕大（爱尔兰税也不交）。效果不是一分不交，而是降一个数量级——苹果那些年海外税率不到 4%，2011 年苹果销售国际实际税率万分之五、2014 年低至万分之 0.5。
- 选爱尔兰而非维京/开曼的原因：12.5% 已够低（2018 年前美 35%、中 25%），且设在发达国家**体面**——库克 2013 年才能在美国听证会上说「我们不会把钱挪到加勒比小岛避税」；更关键的是上述「管理控制地」漏洞。
- 漏洞关闭与新变种：迫于美国施压，爱尔兰 2015 年堵洞但留了 5 年过渡期；税法律师随即造出 **Single Malt**（中间绕道马耳他，微软、Meta 采用）和苹果的 **Green Jersey**（在泽西岛设空壳多倒一手）。
- 采用者是**大部分**美国跨国公司：谷歌、Meta、苹果、亚马逊无一例外；医药巨头同样扎堆爱尔兰。
