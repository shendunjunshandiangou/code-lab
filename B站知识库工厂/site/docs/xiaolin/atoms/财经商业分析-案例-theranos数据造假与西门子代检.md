# Theranos 数据造假与西门子代检

## 一句话定义

Theranos 明知爱迪生机器无法工作，参观时用西门子验血机在后场代检、报告只写少数准确数据；上线后对样本稀释（兑水）违反检测规定，导致大规模误诊与漏诊。

## 详细说明

**参观套路**：抽一滴血 → 带客人参观餐厅办公室 → 研发团队用西门子机器检测 → 1 小时后出示结果。拜登参访时亦如此。

**上线后**：用西门子测约一半指标；样本不够则稀释。健康人被误诊糖尿病、高血压、肺结核；梅毒患者超 30% 测不出。

**Holmes 公开否认**：崩盘后在 CNBC 坚称「从未用过第三方检测机器、测试都是最高质量」，但硬事实与 17 万+ 用户数据已无法掩盖。
