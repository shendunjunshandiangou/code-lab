# 美妆科技模糊目标与开放式创新

## 一句话定义

美妆的科技创新目标天然模糊——"更美"是主观的（不像通信业有速度/延迟等明确指标指导研发），目标不确定就适合**广撒网投初创**分散风险而非重金买成熟公司；欧莱雅称之为开放式创新（BOLD 创投基金、Big Bang 美妆科技创造营），选初创最看中的是**专利**。

## 详细说明

- 背景：行业老大最怕动荡期——增长放缓 + AI 爆发，"别哪天美妆界搞出个 OpenAI 抄了老巢"；既得利益者最慌。
- 两个创新方向：生物科学（找 PhD）与科技。美妆科技的特殊性：欧莱雅自称"create beauty"企业，连包装实验室都在研究什么材质颜色让人觉得更美——目标多元复杂。
- 开放式创新案例：红外吹风机（BOLD 基金投的深圳科创公司合作项目）、纳米晶片导入仪（2020 年中国 Big Bang 创造营孵化）。
- 选项目逻辑（开放式创新负责人访谈）：没有自上而下的赛道配置计划，最看中初创是否有独特的、与众不同的技术（专利）。
