# 休斯商业化改造拉斯维加斯

## 一句话定义
1960 年代的拉斯维加斯还是黑帮、毒品横行的"堕落之地"；休斯因被赌场酒店驱赶而掷资 1300 万美元买下该酒店，随后两年连续买下大量赌场酒店，对黑帮势力进行商业化改造（黑帮成员变成他的生意伙伴），为拉斯维加斯日后的正规化、商业化、企业化奠定了基础。

## 详细说明
- **起因**：休斯在拉斯维加斯一家酒店包下整层长住数年但从不赌博，老板嫌他不产生赌收要赶人；休斯一气之下直接买下酒店。
- **当时的维加斯**：远非今日度假圣地——本地人天黑不敢出门，外地客抱着"赌上性命过把瘾"的心态而来。
- **改造逻辑**：用企业化经营替代黑帮式经营。休斯多年开公司积累的商业能力 + 雄厚资本，让原有黑帮势力被收编进正规生意，"带着他们一起赚钱"。
- **启示**：资本进入灰色地带的正规化改造，可以重塑一座城市的产业底色；这也是休斯"黑道白道都能搞"的注脚。
