# 政府工作报告措辞解读法

## 一句话定义

政府工作报告每年框架相近，**措辞的微小变化**（如「稳健→适度宽松」「积极→更加积极」）往往暗示政策走向与力度，需结合上下文具体分析。

## 详细说明

- 报告结构：回顾 → 总体目标与政策取向 → 重点任务 → 其他部署。
- 十五五开局之年还强调「统筹发展和安全」。
- 不能机械套用措辞变化，需对照历年表述与具体量化指标（GDP 目标、赤字率、专项债规模等）。
