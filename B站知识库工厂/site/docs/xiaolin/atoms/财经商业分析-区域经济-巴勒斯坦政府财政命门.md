# 巴勒斯坦政府财政命门

## 一句话定义

巴勒斯坦政府不能印币（用以色列新谢克尔），2/3 以上收入依赖以色列代收税款且常被拖欠，加上巨额安保开支与援助下降，陷入持续赤字。

## 详细说明

- 与以色列相关税收（关税、增值税等）由以政府代收，月扣 3% 后转交——占巴政府收入 2/3 以上
- 代收金额与转交时间由以色列掌控，常找理由拖欠
- 政府需大量安保人员（超 5 万）镇压内部反抗并配合以色列，开支沉重
- 国际援助年约 10–30 亿美元，因腐败严重近年下降
- 年赤字十几亿美元，常发不出公务员工资
