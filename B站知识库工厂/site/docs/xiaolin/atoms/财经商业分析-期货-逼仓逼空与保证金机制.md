# 逼仓逼空与保证金机制

## 一句话定义

高杠杆+保证金：价格小幅变动即可使对手保证金不足被强平；多头可逼空头（现货不足）、空头可逼多头（如负油价），形成死亡螺旋。

## 详细说明

- 327：万国先被逼仓，后反向 8 分钟逼多；双逼仓典型案例。
- 期货合约由多空配对动态产生，数量可远超可交割现货。
- 闭市前几分钟定价博弈：双方砸资本争对自己有利的结算价。
