# 1MDB一马基金设立与骗局结构

## 一句话定义

2009 年纳吉政府设立 1MDB 国家发展基金，刘特佐说服其以发展项目名义发债，实际成为三人（刘/纳吉/高盛）合谋从公众借债不还的提款机。

## 详细说明

1MDB 名义投资商业区、发电站等，可光明正大发债。三大角色：策划者刘特佐、掩护者纳吉、执行者高盛。
