# 委内瑞拉石油诅咒与荷兰病

## 一句话定义

石油出口换大量美元致本币升值，制造业农业退化；95%+ 出口为石油、60%+ 食品进口——经济丧失免疫力，油价一跌即崩。

## 详细说明

- **荷兰病**：资源出口 → 本币升值 → 本国产品对外变贵、进口变便宜 → 非石油部门萎缩。
- 委内瑞拉形成「玩命出口石油、进口一切」模式；1980 年代油价跌时打击巨大，因非石油部门已完全退化。
- 查韦斯时代进一步以价格管制、官方汇率扭曲市场，加剧结构失衡。
