# 马斯克被踢出 PayPal

## 一句话定义

2000 年马斯克的 X.com 与另一公司合并成后来的 PayPal、马斯克任 CEO，董事会看他不爽，彼得·蒂尔趁马斯克在澳大利亚度假时集体把他开除并踢出董事会——「董事会开掉明星 CEO」的经典先例；但马斯克手握 PayPal 股票，几年后套现数亿美元成立 SpaceX、加入特斯拉，因此有人调侃「人类的科技进步要感谢彼得·蒂尔当初把这个猖狂小子踢出 PayPal」。

## 详细说明

- 与 OpenAI 宫斗的对照：形式相似（董事会突袭开除 CEO），但合理性不同——PayPal 当时刚合并、军心不稳，马斯克并非不可替代；奥特曼则是公司绝对的灵魂人物，开除引发全员逼宫。
- 与股权的关系：马斯克虽失去控制权但保有股票，经济利益成了他后续创业的弹药——对比他在 OpenAI 捐 5000 万美元却因非营利无股东概念而被彻底抛弃。
- 佐证「董事会何时有牙齿」：管理层地位不稳、公司动荡时，董事会的任免权才真正落地。
