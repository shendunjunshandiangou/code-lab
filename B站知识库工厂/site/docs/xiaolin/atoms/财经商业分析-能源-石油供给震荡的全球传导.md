# 石油供给震荡的全球传导

## 一句话定义

石油高度市场化，霍尔木兹海峡 2000 万桶/日堵滞会瞬间推高全球油价，与出口目的地无关；天然气区域性更强。

## 详细说明

- 供给受限 → 全球所有人感受油价上涨之痛
- 天然气：欧洲、亚洲涨 50%+，美国本轮几乎未受影响（与俄乌战争时欧洲天然气暴涨类似）
- 各国战略储备一般可扛 3–6 个月
