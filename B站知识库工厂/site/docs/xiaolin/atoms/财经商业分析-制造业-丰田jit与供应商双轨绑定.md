# 丰田 JIT 与供应商双轨绑定

## 一句话定义

资源匮乏的日本视库存为暴殄天物，丰田把福特模式改良成 JIT（Just-in-time 及时生产）追求零库存"用时再产"；为让供应商接"又小又急"的单，把供应商分两轨——核心零部件供应商入股绑定、专建产线优先供货，非核心供应商"海王战术"多找备胎并相互竞价。

## 详细说明

- 起点：1929 年丰田喜一郎参观福特底特律工厂，"流水线可以学，垂直整合不适合我"。
- 思想进化：福特只求生产快，丰田把需求端纳入供应链管理——不能光低头造，还得抬头看市场要什么；库存周转（Inventory Turnover）指标即源于此。
- 这套供应商双轨打法沿用至今，苹果即采用同款战术（库克落地 JIT、砍 SKU、轻资产外包）。
- JIT 不一定是最优解，但在供应链管理思想上是确定的进化。
