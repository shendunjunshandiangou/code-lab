# 天然气管道经济学与长期合同

## 一句话定义

天然气是气态、难运输（车运效率低易泄漏，早年液化技术不成熟），最经济的方式是**管道**——初期成本极高、工期两三年到六七年，但建成后**边际成本极低**（只需维护查漏）；这种「高固定成本 + 低边际成本」的基建型经济特征，决定了天然气交易天然倾向国家间、地区间**十年起步的超长期大单**，而非「来两口气尝尝」的现货小交易，也把买卖双方深度绑定。

## 详细说明

- 历史起点：1970 年西德与苏联签长期天然气协议（苏联供气、西德提供建管道的钢铁），1973 年管道建成——天然气最大出口国与欧洲最大经济体就此绑定；此后 20 年即使冷战、即使里根劝阻，欧洲越买越多，1989 年德国约三分之一天然气来自苏联。
- 管道的软肋是**途经国风险**：苏联解体后通德管道多经乌克兰，俄罗斯为分散地缘风险修建绕行管道，承载量最大的**北溪一号**经波罗的海直达德国、不经任何第三国（无过路费、无途经国风险）——这也使它成为俄欧博弈中最关键的阀门。
- 对比：石油可轻松海运转卖给任何买家，天然气管道锁定了买卖对象——2022 年制裁下俄罗斯石油可转卖中印，天然气则难以转手（另一条出路是 LNG，见 [财经商业分析-卡塔尔-北方气田与LNG出口](/xiaolin/atoms/财经商业分析-卡塔尔-北方气田与lng出口.html)）。
