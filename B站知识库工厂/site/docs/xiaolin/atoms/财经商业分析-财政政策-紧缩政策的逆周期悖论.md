# 紧缩政策的逆周期悖论

## 一句话定义

经济管理的基本逆周期逻辑是经济差时宽松刺激、经济好时收紧防过热；三马在希腊衰退期强制紧缩，违背该逻辑，事后广受诟病。

## 详细说明

国家经济管理最简单的底层逻辑：

| 经济状态 | 应有政策取向 |
| --- | --- |
| 经济好 | 收紧，防止过热，积累积蓄 |
| 经济差 | 宽松，刺激需求，尽快回正轨 |

不论财政政策、货币政策还是其他工具，基本逻辑应如此。

**希腊矛盾**：
- 2010 年前后希腊明显处于经济不好阶段
- 三马救助条件却是紧缩（增税、减支）
- 显然违背逆周期原则

**三马为何仍这么做**：
- 三马当然懂逆周期逻辑
- 但立场是德法等债权国：自身难保，借钱给希腊必须保证能还上
- 稳定市场情绪优先于希腊短期增长

该悖论是理解希腊「魔鬼十年」政策争议的关键。

**英国 2010 紧缩（同类案例）**：
- 卡梅伦政府以 08 金融危机为鉴，在全球危机最严重时推行削减开支（目标 -40%）——美国大放水、中国四万亿刺激，英国（及整个欧洲）却在紧缩。
- 无奈之处同样是欧债危机：欧盟贸易/货币捆绑造成成员国信用捆绑，政府若扛着信用风险继续刺激很可能崩盘，只能靠欧央行降息印钱。
- 长期代价：紧缩延续到疫情前，被视为此后英国经济萎靡、脱欧、2022 债务危机与地方政府排队破产的深层根源之一——政府本应是经济的逆周期缓冲器，紧缩把自己架了起来。
