# 中国需求推动 2000 年代油价长牛

## 一句话定义
2000–2008 年油价一路飙升至 147 美元/桶的历史最高，短期因素有伊拉克战争、OPEC 减产，但十年持续上涨的根本原因是中国 2001 年入世后 GDP 与石油需求量的爆发式增长。

## 详细说明
- 2001-12 中国加入 WTO，此后 GDP 一路飙升，石油需求量随之暴涨，成为拉动全球油价的最大增量。
- 2008 年油价达 147 美元/桶历史峰值；同年次贷危机爆发，因全球衰退风险油价暴跌到 32 美元；OPEC 减产+市场确认危机非长期性后，油价快速反弹至 80 美元左右。
- 启示：分析大宗商品长周期，需求侧的结构性力量（新兴大国工业化）往往比供给侧的短期事件更根本。
