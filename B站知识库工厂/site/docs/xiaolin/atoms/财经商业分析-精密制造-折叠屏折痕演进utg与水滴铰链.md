# 折叠屏折痕演进 UTG 与水滴铰链

## 一句话定义

折痕来自屏幕+铰链双核心；路径 CPI 塑料 → 2020 三星 UTG 超薄玻璃 → 2021 华为/OPPO 水滴铰链 → 2023 UTG+水滴成业界收敛。

## 详细说明

- UTG：玻璃薄至可弯折；三星与肖特 3 年独家，后康宁第二供应商。
- 水滴铰链：U 型 180° 折 → 水滴形分散压力，折痕变宽变浅。
- 二手市场折痕深浅价差可达一倍。
