# 地下赌球与洗钱关联

## 一句话定义

博彩大量现金往来适合洗钱；庄家帮犯罪分子运输/分层赃款利润可超博彩本身（Hansen 送 100 万现金提成 10%）。

## 详细说明

- 地下盘服务器常放越南/柬埔寨；加密货币 increasingly 用于结算。
- 赌球易延伸至洗钱、黑帮——运动与赌博/犯罪 historically 高相关（非必然）。
