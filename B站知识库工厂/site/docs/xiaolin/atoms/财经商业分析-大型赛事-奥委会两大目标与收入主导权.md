# 奥委会两大目标与收入主导权

## 一句话定义

IOC 两大目标：做大奥运 IP（赛事规模、声势、场馆）；从转播与赞助等大头获取经济利益并主导分配——主办国承担成本，IOC 握住收入主动权。

## 详细说明

**目标一：IP 做大做强**
- 选有野心、愿把赛事办大的主办国合作。
- 对主办国而言，声势浩大 = 赤裸裸的成本（场馆、基建）。

**目标二：经济利益**
- 转播、赞助等大头由 IOC 主导，再分一部分给主办国。
- IOC 想留大头，主办国自然分得少。

**两层博弈：**
1. 赛事举办规模要求
2. 收入分配比例

**关键变量：** 申办城市/国家数量（见 [财经商业分析-大型赛事-申办城市数量决定博弈地位](/xiaolin/atoms/财经商业分析-大型赛事-申办城市数量决定博弈地位.html)）。
