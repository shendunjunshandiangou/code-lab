# Spoofing 诱骗交易

## 一句话定义
Spoofing（诱骗交易）指挂出大量不打算成交的订单制造虚假供需信号、把价格推向自己想要的方向后撤单获利——是明确违法的市场操纵行为，华尔街交易员入职第一课就会被警告。

## 详细说明
- 操作流程（以做多为例）：①先在 101 真实买入；②挂大量虚假买单，市场误以为买盘汹涌纷纷跟买、卖单撤走；③价格被推到 110–111 时卖出真实持仓获利；④虚假订单从头到尾不成交、随时撤销。
- 有效前提：资金/挂单量够大，足以显著扭曲盘口供需信号。
- 合规红线：属于「首当其冲」的违法行为。真实案例：交易员误把买单下成卖单、撤回重下，第二天就被合规部门（compliance)约谈怀疑 spoofing——监控之严可见一斑。
- 与高频交易的军备竞赛：2006 年后机器高频交易基金涌入压缩了手动 spoofing 的利润，Sarao 转向算法自动化，虚单量指数级增长，一度占标普 E-mini 期货全市场日订单量的 7%，闪崩当天占三分之一。
- 监管后续：Sarao 配合调查后，监管机构数年内查处了德意志银行、瑞银、汇丰等多家大行的市场操纵行为，罚款数以亿计。
