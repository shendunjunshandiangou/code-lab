# 俄罗斯七大寡头

## 一句话定义

1996 年叶利钦与七位银行界大佬秘密协定——助其连任则保其荣华富贵，形成掌控银行、天然气、石油、媒体等命脉的 Semibankirschina（七大银行家），寡头操控经济影响政治至今。

## 详细说明

**三大后果**：

1. **遏制竞争与创新**：目标变为排除竞争者、守地盘，而非提供更好的产品
2. **腐败与黑帮**：Krysha（「房顶」，即保护费）盛行；阿布与别列佐夫斯基曾公开涉及 20 亿美元保护费官司
3. **财富集中与资产外流**：最富 98 人持 4210 亿美元；最富 10% 占 89% 财富；海外资产估算相当于 GDP 23%，未计入官方统计
