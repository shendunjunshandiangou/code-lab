# 造车三大块价值量拆分

## 一句话定义

站在造车角度，电动汽车可拆成三大块：电动化（电池电机电控"三电"，约占整车价值量 50%）、智能化（智驾 + 座舱，约 20%）、造车相关（制造/供应链/内饰等，约 30%）——这是穿透研报几十个模块和发布会营销名词的基础框架。

## 详细说明

- 电动化是电车的基础，其中电池占总成本约 40%，是最核心最贵的零部件。
- 智能化占比因车型差异很大，20% 只是量级概念；电动化与智能化是业界公认两大块。
- 分析新玩家（如小米）入局策略时，逐块看格局：电动化已成型→卷差异化；智能化早期混战→全力自研；制造→视资金实力决定代工还是自建。
