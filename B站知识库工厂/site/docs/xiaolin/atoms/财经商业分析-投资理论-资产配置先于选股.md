# 资产配置先于选股

## 一句话定义

专业投资组合构建的第一步是 asset allocation（资产配置：投哪个国家、发达/发展中市场、股还是债），之后才是 security selection（证券选择）；宏观决策优先于个股决策。

## 详细说明

金融专业课中投资组合的标准顺序：先做资产配置，再做证券选择。

宏观层面的可选项并不少：除了 A 股/港股/美股/创业板，还可以买覆盖发展中国家或特定板块的基金/ETF，一次买一篮子，分散风险。

配套的宏观观察指标：GDP、CPI、non-farm payroll（非农就业）等；另类指标如格林斯潘的"男性内裤指数"——内裤是必需品、销量稳定，衰退时人们省着买，销量下滑；衰退期内裤销售额回升被视为经济好转的先行指标。

讲者推荐了根据 Ray Dalio 著作制作的宏观经济原理科普视频（约三四十分钟）作为入门。
