# 奥运会早期门票信托模式

## 一句话定义

1896–1930 年代现代奥运以主办国组织、IOC 宣传、观众购票为主，门票收入入信托供下届支出，非盈利导向的极简模式。

## 详细说明

- IOC 1894 年由顾拜旦创立；1896 年雅典首届：14 国、241 运动员、42 项赛事。
- 主办国负责场地组织，IOC 负责宣传，观众买票提供收入。
- 门票收入放入信托，下次办奥运从中扣款，金额不大，更多是精神支持。
- 该模式持续 30 余年，直至电视转播出现（1936 柏林首次电视转播）。
