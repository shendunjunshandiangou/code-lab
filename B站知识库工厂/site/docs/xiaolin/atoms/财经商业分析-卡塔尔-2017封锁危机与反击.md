# 2017 卡塔尔封锁危机与反击

## 一句话定义

2017 年沙特联盟对卡塔尔全面封锁，卡塔尔靠天然气筹码、伊朗领空与土耳其粮食合作化解危机，2021 年封锁解除且 13 条要求均未兑现。

## 详细说明

**背景**：2010 年阿拉伯之春后沙特不安；卡塔尔与伊朗走近，2017 年沙特联合阿联酋、巴林、埃及实施 Blockade——非普通制裁，飞机禁飞越沙特领空，美方人员撤离，连骆驼都需运出。

**13 条要求**（核心）：减少与伊朗关系；停止与土耳其军事合作等。

**冲击**：90% 粮食靠进口，曾面临粮食短缺；IMF 2017 年下调经济前景。

**反击**：
- 与土耳其签大量粮食订单，卡塔尔成土耳其第三大投资人；
- 年付伊朗约 **1 亿美元** 换取领空飞越权；
- 制裁后短暂衰退即恢复增长，降低对沙特、阿联酋依赖。

2021 年沙特取消封锁；2022 俄乌战争推高欧洲 LNG 需求，进一步利好卡塔尔。
