# 主动型基金与被动型基金

## 一句话定义

**被动型基金**（如多数 ETF）按指数配置，经理任务是减小追踪误差；**主动型基金**（普通基金）由经理在划定范围内选股，决策直接决定收益。

## 详细说明

### 被动型（ETF 为主）

- 基金经理按指数被动配置，指数涨跌与经理个人无关
- 核心任务：成本可控下**尽量减小追踪误差、降低交易成本**
- 国内称「交易型开放式指数基金」，名字里常带「指数基金」

### 主动型（普通基金）

- 基金经理在基金名称划定的范围内主动选股
- 范围示例：沪深 300、小盘股、军工股、消费股等——名字里写清楚
- 买什么、买多少直接决定收益
- 投资人有行业偏好可选对应主题基金，由经理在范围内再精选

### 普通投资者两大比较维度

1. **费率**：普通基金贵，ETF 便宜
2. **谁来管钱**：主动选股 vs 被动追踪
