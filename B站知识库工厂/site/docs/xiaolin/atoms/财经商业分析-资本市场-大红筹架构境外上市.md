# 大红筹架构境外上市

## 一句话定义

境内企业境外上市的标准组合拳：在开曼注册上市主体（京东、阿里、B 站皆是）→ 中间加一层香港公司（香港持有大陆公司股份的分红税率 5%，比外资直接持股的 10% 省一半）→ 香港公司控股境内经营主体——由开曼壳在纳斯达克等境外市场挂牌，变相实现境内公司上市融资。

## 详细说明

- 动因：A 股上市有利润门槛，烧钱的互联网/科技公司前期不盈利只能去境外；而大陆企业因监管限制难以直接在纳斯达克挂牌，需要海外壳做上市主体。
- 开曼作为上市主体注册地的优势：零税率 + 成熟的离岸公司法制 + 境外交易所认可。
- 底层逻辑一句话：境内经营主体通过各种方式串联到境外主体、由境外主体上市——结构图再复杂，本质如此。
