# 复利的长期威力与年化收益对比

## 一句话定义
年化收益率的差距在长期会被复利指数级放大：同样 1000 元投 50 年，3% 国债变 5000 元，7–8% 股市变 4 万元，巴菲特的约 21% 年化则变成 4000 万元。

## 详细说明
| 投资方式 | 年化收益率 | 1000 元投 50 年后 |
| --- | --- | --- |
| 国债 | 约 3% | 约 5000 元（5 倍） |
| 股市大盘 | 约 7–8% | 约 4 万元（40 倍） |
| 巴菲特 | 约 21–22% | 约 4000 万元（4 万倍） |

- 巴菲特 50 年平均年化收益率约 21%，单看数字不惊人，放在 50 年时间线里就是复利的冲击。
- 讲者引申：一年赚 100%、200% 不算什么；能长期 50 年、哪怕每年只赚 20%，就是 4 万倍收益——这是「长期持有」的数学基础。
