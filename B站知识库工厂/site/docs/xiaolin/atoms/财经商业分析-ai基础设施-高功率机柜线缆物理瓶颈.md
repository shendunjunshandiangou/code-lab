# 高功率机柜线缆物理瓶颈

## 一句话定义

服务器功率提升受物理限制：功率=电压×电流，电流越大线缆越粗，机柜空间有限导致「线太粗进不去」，约 300 千瓦/机位时需供电、液冷、工程系统级重构。

## 详细说明

**物理公式**：功率 = 电压 × 电流
- 功率↑ → 要么电压↑，要么电流↑（更多「水」过管道）
- 电流大 → 线缆必须更粗 → 机柜背后空间有限，**线太粗真的放不下**
- 电流大、线缆粗 → 发热高、损耗大

**临界点**（华为何波）：
- 功率提升在未来是**颠覆性**的，现在缓慢提升
- 到约 **300 千瓦/机位**时，所有东西需**系统级重构**
- 例：一根电缆可能接近奶粉罐那么粗，若干根怎么「进得去」

**行业方向**：
- 黄仁勋：未来方案或采用**高压直流/800 伏**，让更多电进得去
- 华为认可方向，但走多元组合而非单一路线
- 强电改标准牵涉全国电器生态（如电网 50Hz），非纯技术问题；开断关等零配件须全面适配

下一数量级：不光电压电流，供电、液冷、运营工程都需重构。
