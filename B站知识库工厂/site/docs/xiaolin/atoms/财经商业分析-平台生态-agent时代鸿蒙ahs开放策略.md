# Agent时代鸿蒙AHS开放策略

## 一句话定义

面对 Agent 时代，鸿蒙向合作伙伴开放 AHS 对接能力，使 App 能力 Agent 化并可经小艺等系统入口被一句话调用，降低消费者使用深度。

## 详细说明

余承东阐述鸿蒙应对 Agent 时代的调整：

- 与合作伙伴开放 **AHS 对接**能力
- 合作伙伴将自身能力 **Agent 化**，与系统 AI 对接
- 消费者可通过**小艺**一句话调用他人 Agent 完成功能——使用入口更浅

战略含义：操作系统竞争从「App 图标入口」转向「意图 + Agent 编排」；谁先完成能力开放与调用标准，谁更易绑定开发者与用户。鸿蒙将 Agent 对接作为生态下一阶段的基础设施能力。
