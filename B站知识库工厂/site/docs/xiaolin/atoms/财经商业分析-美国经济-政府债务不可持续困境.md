# 政府债务不可持续困境

## 一句话定义

政府债务是美国乃至全球政府长期最大的潜在风险：利率上升不断推高负债成本，短期问题不大（美日政府付得起利息），但**不可持续**——不能经济一出问题就靠政府借钱买单；而削减开支对政府「更多是嘴上说说」，短期内永远有更紧急的事需要刺激花钱，两党博弈、政府关门等限制手段效果也不佳。

## 详细说明

- 结构性拖延症：问题重要但不紧急，政治上的行动激励永远输给眼前的刺激需求——这是全球政府债务问题的共同病灶。
- 美国的表现：债务刹不住车地不停突破上限；政府关门大战成为常态化戏码。
- 与利率的联动：高利率时代延长会持续放大存量债务的利息负担，形成「利息吃掉财政空间→更依赖借新还旧」的循环。
