# 北方气田与卡塔尔 LNG 出口

## 一句话定义

卡塔尔北方气田占全球天然气储量约 15%，哈马德押注液化天然气（LNG）技术后成为全球第二大 LNG 出口国，开采成本极低。

## 详细说明

1971 年壳牌在波斯湾发现北方气田（The North Field），与伊朗共享但以卡塔尔侧为主。早期管道运输不划算，气田长期难以变现。

1995 年哈马德上台后倾国力投资 LNG 全产业链：零下 160°C 液化使体积缩至 **1/600**，便于海运。卡塔尔现为全球第三大天然气出口国、第二大 LNG 出口国；气田储量大且「一碰就出气」，开采成本低廉。
