# 中美市场主动基金超额收益差异

## 一句话定义

主动型基金跑赢大盘的比例在 A 股远高于美股：国内 10 年期约 **70%** 跑赢中证全指/800，美欧不到 **20%**——差异源于市场有效程度与散户结构。

## 详细说明

### 数据对比

| 市场 | 主动基金跑赢大盘比例（10 年期） |
| --- | --- |
| 中国（中证全指/800） | 约 **70%** |
| 美国、欧洲 | 不到 **20%** |

### 原因分析（有效市场视角）

**美股等成熟市场**

- 信息发达，量化交易挖掘充分
- 市场更接近有效市场
- 主动选股套利空间小

**大 A 股**

- 散户多，多数人不看年报、听风追涨杀跌
- 价格存在大量不合理性
- 机构有**信息差**和**专业性**，可在不那么有效的市场中选股跑赢
- 这也是 A 股投资者常感「被割韭菜」的原因之一

### 讲者笼统结论

| 市场 | 建议 |
| --- | --- |
| 不那么成熟（大 A） | 主动型（普通基金）平均表现优于 ETF，但须**挑对基金经理** |
| 较成熟（美股） | 主动型空间小；不懂挑就买 ETF，费率更低 |
| 例外 | 巴菲特、文艺复兴等数十年超额收益的「bug 级」人物始终存在 |
