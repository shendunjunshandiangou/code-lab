# 多邻国五大运营原则

## 一句话定义
多邻国员工手册的核心是五条"不是口号、而是真实做法"的运营原则：着眼长远、做得有趣、先做出来、展示而非讲述、不断提高标准。

## 详细说明
1. **Take the long view（着眼长远）**：把公司当作要存在一百年来经营，而非只看明天——最重要的一条。
2. **Make it fun（做得有趣）**：产品要有趣，在公司工作本身也要有趣。
3. **Ship it（先做出来）**：不无休止争论想法好坏，做出来再验证。
4. **Show don't tell（展示而非讲述）**：汇报时别讲"我熬了多少夜、多辛苦"，直接展示成果。
5. **Raise the bar（不断提高标准）**：永远比之前做得更好。

CEO 认为这五条不是 aspirational（愿景式）的空话（如"做到最好"），而是公司实际的行为方式；公司规模变大后这五条文化保持得依然不错。
