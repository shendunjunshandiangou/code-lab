# 马斯克SpaceX控制权结构

## 一句话定义

SpaceX 通过 A/B/C 三类股、B 类十票加权、B 类股东选举过半董事等设计，使马斯克在上市后仍保持约 84% 投票权，控制权「可以上市但不能 public」。

## 详细说明

**马斯克过往控制权教训**：
- PayPal 蜜月飞机上被董事会罢免
- OpenAI 狼狈出局
- 特斯拉投票权仅 **20% 出头**，常需以辞职要挟董事会

**三类普通股**：

| 类别 | 投票权 |
| --- | --- |
| A 类 | 1 股 1 票 |
| B 类 | 1 股 10 票 |
| C 类 | 无投票权 |

**上市前后持股**：
- 上市前：12.3% A 类 + 93.6% B 类 → 投票权 **85.1%**
- 本次对外只融 **A 类**；B 类仍由马斯克控制
- 上市后投票权约 **84%**

**董事会控制**：
- S-1 规定 B 类股东有权选举**董事会过半**董事
- 马斯克占 B 类 **93.6%** → 无论多少轮 A 类融资，始终控制董事会
- 美国三大公共养老金联名警告结构「新奇又极端、闻所未闻」

**资本市场观感**：
- 买 SpaceX 的人信任马斯克本人
- 部分人认为独揽控制权可减少资本裹挟，属仁者见仁
