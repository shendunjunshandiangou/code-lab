# 个人贷款利率三部分构成

## 一句话定义

银行给个人的贷款利率 = 无风险利率 + 借款人风险溢价 + 银行利差，其中无风险利率是整个利率体系的根基。

## 详细说明

以 10 年期房贷、银行报价 **5%** 为例：

| 组成部分 | 示例 | 说明 |
| --- | --- | --- |
| 无风险利率 | 3% | 见 [财经商业分析-货币政策-无风险利率与国债基准](/xiaolin/atoms/财经商业分析-货币政策-无风险利率与国债基准.html) |
| 风险溢价 | 1.8% | 根据财务状况、收入、抵押等，模型算出违约概率；风险越大溢价越高 |
| 银行利差 | 0.2% | 市场竞争与银行自身情况 |

- 贷给苹果或世界银行 → 风险溢价极低
- 新闻里「美联储加息 50 基点」「央妈降息」等，**不是**银行挂牌的一年期/五年期/十年期存款利率
