# 互换便利与回购增持再贷款

## 一句话定义

2024 年央行为股市创设的两个工具：**证券基金保险公司互换便利（SFISF，5000 亿额度）**——金融机构把股票质押给央行换成流动性更好的国债，卖国债回笼的钱只能买股票；**股票回购增持专项再贷款**——上市公司直接找央行借钱回购股票。本质都是把钱从央行辗转导向股市。

## 详细说明

- SFISF 的作用：帮国家队资本拿到更多资金支撑股市——用央行信用把"不好变现的股票"临时换成"好变现的国债"。
- 这类工具属于**刺激性政策**（区别于化债类稳定性政策），是央行直接为资产价格托底的创新尝试。
- 认知技巧：政策名字再绕（"证券基金保险公司互换便利"），抓住"再贷款/互换 + 央行"关键词即可判断资金链路。
