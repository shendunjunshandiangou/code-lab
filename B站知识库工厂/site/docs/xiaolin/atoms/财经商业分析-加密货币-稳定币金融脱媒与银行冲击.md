# 稳定币金融脱媒与银行冲击

## 一句话定义

稳定币跳过银行与支付中介（金融脱媒），各国禁止稳定币付息以限制其成存款工具；银行担忧丧失放贷能力与经济体连接，但仍纷纷发行自有稳定币「打不过就加入」。

## 详细说明

- 储备 100% 理论上部分回流银行，但银行丧失灵活资金与放贷能力
- 央行对加密世界货币政策调控、流通速度、信贷规模尚无成熟方案
- 美国定义「支付稳定币」——主要支付，少存钱投融资
- 摩根大通、PayPal、Stripe、OKX 等已纳入稳定币；Jamie Dimon 曾称比特币骗局现推摩根币
