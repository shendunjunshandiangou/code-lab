# 赛后场馆资产再利用困境

## 一句话定义

IOC 建议赛后场馆办赛事/演唱会、奥运村改售商品房，但里约、东京商品房未卖出，雅典、里约场馆大量闲置，维护仍须每年数百万美元。

## 详细说明

**理想路径（IOC 建议）：**
- 场馆：办演唱会、体育赛事
- 奥运村：改商品房出售

**成功案例：**
- 2008 北京：奥运村改商品房出售，后续房价飞涨；鸟巢仍可办体育赛事、演唱会。

**失败案例：**
- 里约、东京：本想改售商品房，未卖出
- 雅典、里约：赛后场馆闲置废弃
- 闲置非零成本：维护每年仍须数百万美元

**盈亏对比（视频举例）：**
- 较成功：1984 洛杉矶、2008 北京
- 较惨：2004 雅典、2016 里约、2020 东京（2021 举办）
