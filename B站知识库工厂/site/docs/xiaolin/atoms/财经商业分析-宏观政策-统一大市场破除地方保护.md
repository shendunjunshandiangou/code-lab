# 统一大市场破除地方保护

## 一句话定义

「十五五」内需第二方向是做强国内大循环、破除地方保护，通过统一基础制度让要素自由流通，释放 14 亿人口超大市场潜力。

## 详细说明

**问题**：国内市场存在地方保护——本地采购优先、税收优惠、财政补贴争抢新兴产业等。地方向上负责 GDP，招商关系上升通道，竞争激烈；过度本地偏向形成保护主义，不利于公平竞争与全国需求整合。

**目标**：制定统一基础制度与规则，促进商品、资金、人才、技术更自由通畅流通 → 企业竞争更公平，激发全国需求潜力。

**经济学逻辑**：地方政府竞争本身可以是好事，但若演变为系统性本地保护，则从全国视角降低效率、割裂大市场。
