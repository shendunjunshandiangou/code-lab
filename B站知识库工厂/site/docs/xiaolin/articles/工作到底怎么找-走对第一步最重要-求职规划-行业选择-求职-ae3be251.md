> **摘要**：Offer帮创始人 Lindsay 强调求职规划是找工作中最关键的一步。规划分三块：明确方向（行业/岗位与自身匹配）、评估准备工作（硬技能/软技能/经历/简历面试技巧）、按时间线排期（不同公司路径差异大，宜尽早开始）。

## 为什么规划最重要

规划为后续所有准备打基础。方向对了，后面通畅；方向错了，在错误路上越走越远。

**反例**：想进高盛不是快毕业投简历、一面拿 offer 那么简单。常见路径：大一大二参加 Early Insights、Spring Week、Mentorship 等探索项目 → 大三积累社团与技能 → 大三暑假 Summer Analyst 实习 → Return Offer 上岸。Google、麦肯锡等公司与岗位对技能、软实力要求各不相同。

若没想清楚方向，「东一榔头西一棒」使力分散；而专注高盛投行的同学「毕生绝学」钻研一条线，深度不可比。**前期规划决定竞争起点。**

## 规划第一块：明确求职方向

先搞清楚「到底想找什么」——相当于方向确认，才知道做什么准备、如何安排各公司招聘时间线。

常见方向需结合：
- 专业知识、兴趣爱好、性格
- 各方向常见专业背景
- **硬技能**与**软技能**要求

做法：了解各方向做什么、是否感兴趣；评估技能要求是否愿意学、能否达到。

## 规划第二块：准备工作评估

针对选定方向，评估需完成的准备：

| 类别 | 内容 |
| --- | --- |
| 硬技能 | 岗位所需专业知识与工具 |
| 软技能 | 交流沟通、人脉等 |
| 相关经历 | 实习、社团、项目 |
| 求职技巧 | 简历、面试；把经历写得「来了就能干活」 |

## 规划第三块：时间线

类似开学 Syllabus：知道何时期中、期末，才能塞入准备、不错过 deadline。

- **大一开始**：时间充裕，可探索多方向、多实习、多备技能
- **临毕业才找**：时间紧张，须把「无限工作量」塞进有限时间

找工作准备工作多，**必须尽早规划**。规划可随实际情况调整，非一成不变。
