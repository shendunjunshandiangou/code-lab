## 摘要

美国半世纪贸易逆差「冠绝全球」——表面因进口>出口，公式 **贸易差额=储蓄-投资** 只是恒等式。关键：**美元作为世界储备货币**，全球资本经金融账户流入美国→美元升值→贸易逆差持续；整体国际收支近乎平衡（经常账户逆差+金融账户顺差）。

## 常见解释及局限

- 美国人爱消费 / 政府赤字 → 短期可能，但**汇率应自动调节**（逆差→本币贬值→纠正）；美国却逆差近 50 年 → 调节机制「失灵」？

## 汇率自动调节 vs 广场协议

- 理论：持续逆差→卖本币买美元→本币贬值→出口增进口减。
- **广场协议**（1985）：美日德等升汇、美元贬 → 美国逆差短期收缩；但 1995 后逆差变本加厉。
- 美国很少直接干预外汇；95 年后逆差扩大非因「人为压本币」。

## 全貌：三大账户

| 账户 | 美国特征 |
| --- | --- |
| 经常账户 | 长期逆差（主要是贸易） |
| 金融/资本账户 | 长期顺差（买美债/美股/FDI 流入） |

- 全球资金涌向美国资产 → **美元升值** → 进口更便宜、出口更难 → **贸易逆差**。
- 逆差≠「钱被赚走」——是账户结构；华尔街、科技公司、政府受益于资本流入。

## 美元特权的循环（特里芬难题）

- 出口国赚美元 → 央行收储汇 → 部分回流买美债/美股 → 支撑美元与逆差。
- 仅对个别国家加关税→进口结构重构，**金融账户持续流入则逆差难根本缓解**。

## 特朗普政策含义

- 关税短期或改结构；长期若资本仍涌入美国，逆差压力仍在。
