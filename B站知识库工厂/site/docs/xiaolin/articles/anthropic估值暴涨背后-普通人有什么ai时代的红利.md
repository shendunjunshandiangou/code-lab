> **摘要**：OpenAI 与 Anthropic 年内上市、估值逼近万亿，2025 年被称作 Agent 落地元年。蚂蚁灵光产品总监马竞解析：资本看好 AI 解决更复杂真实问题；中美 Agent 路线分化（美国 B 端、中国 C 端移动端）；普通人红利在「许愿编程」与 AI 原住民创作生态。

## 背景：资本热与 Agent 元年

OpenAI、Anthropic 都在卯劲上市，市值直逼万亿美元。OpenAI 2025 年 3 月融资估值约 **8500 多亿**美元；Anthropic 已递交招股书，估值约 **9600 多亿**美元，首次反超 OpenAI。

2025 年也被称作 **Agent 真正落地元年**——从「你问我答」到能写代码、做应用。Agent 对整个商业社会意味着什么？大厂有什么新玩法？普通人能抓什么机会？

本期为小 Lin 主持的「灵光对话」播客，嘉宾为蚂蚁集团灵光 App 产品总监**马竞**。

## Anthropic 估值为何反超

马竞认为，核心还是资本看到了 **AI 到底能解决什么真实问题**。

### AI 解决问题的四个层次

| 层次 | 类型 | 特征 | 代表 |
| --- | --- | --- | --- |
| 第一 | 简单问题 | 问答、对话 | ChatGPT 聊天框 |
| 第二 | 复杂问题 | 有 SOP，按规则逐步执行，更可控 | **Coding Agent**（如 Claude Code） |
| 第三 | 长程问题 | 除 SOP 外需 AI 自己推理、思考如何一步步解决 | 大模型 + Coding Agent 结合 |
| 第四 | 开放问题 | 虚拟世界与现实世界结合 | 世界模型、机器人 |

资本看到 AI 已能解决更复杂、更长程的问题，真正解放生产力，并创造 GDP 的一部分——这是 Anthropic 估值反超的逻辑。

## Claude 与 Claude Code：聊天框 vs 超级程序员

**Anthropic** 在中国认知度一度不如 OpenAI——后者用 Chat 模式让普通人免费使用 AI。

Anthropic 早期核心方向是 **基于 coding（代码生成）解决问题**：
- **Claude**：聊天框形式，解决相对简单问题；复杂问题上难像 Claude Code 那样完全交付
- **Claude Code**：像真实程序员和助理——你说要求，它写代码、debug、上线部署，**直接交付可用成果**

Claude Code 在全球受很多企业追捧：明确提高程序员生产力，也在金融、医疗等 B 端提供效能——「动嘴就行」。

**应用场景**（B 端为主）：
- 程序员：Vibe Coding，让机器写代码
- 金融分析师：财报分析（小 Lin 投行背景也相关）
- 医疗企业：正在进军

小 Lin 举例：有人用简单几句话让 Claude Code 自己上网搜集信息编出完整网页，那一刻很震撼。

## 中美 Agent 路线差异

| 维度 | 美国 | 中国（灵光路径） |
| --- | --- | --- |
| 主战场 | B 端企业应用（SpaceX 招股书称 AI 市场 90%+ 在企业端） | C 端移动端 |
| 基础设施 | PC、SaaS 中小企业支付意愿高 | 手机普及率远高于 PC，很多行业不需 PC |
| 类比 | 信用卡阶段 | 跳过信用卡直达移动支付；跳过 PC Coding 直达移动端 **Wish Coding（许愿编程）** |

**为何灵光选 C 端而非 B 端**：
- 蚂蚁有普惠基因（移动支付）
- 中国 SaaS/企业软件生态与美国本质不同：美国中小企业中腰部占比高、支付意愿高；中国 B 端生产力可能通过其他方式解决
- 中国大量小微企业可通过移动端 Web Coding 低成本解决问题
- 真正用过 Claude Code 的中国用户极少——生产力平权是目标

Claude Code 等产品目前不在中国售卖；各路径殊途同归，探索如何解决复杂、长程、开放问题。

## AI 产业链：钱落在哪一层

马竞观点：基础模型是**人才密集 + 资本密集**，头部几家大厂研发；**工具层和应用层百花齐放**。

灵光定位：**应用层、面向 C 端**——先帮用户获取信息、解决问题、探索世界，在中国环境下让更多普惠用户用上最前沿 AI。

## 灵光与「闪应用」：C 端 Wish Coding

灵光去年底上线，被记住的点是**「一句话生成闪应用」**，**30 秒内**生成、上线、部署——全球能快速做到的公司相对较少。

**数据**：已沉淀 **3000 多万**闪应用。

**主要类别**：
1. **日常工具**：情绪管理、记录与解读情绪、可视化拟人化（类似 Inside Out）
2. **文字游戏/互动内容**：小说转可交互悬疑游戏，可加图片、音乐
3. **测试类**：性格测试、MBTI 等（曾与 MBTI 创始人做沙龙）
4. **烟火气实用**：「如何给女朋友拍照」——拍场景后给摆姿指导
5. **财务类**：家庭资产负债表（流动/固定资产、消费贷/房贷等；目前多手动输入，后续 API 授权打通）

**差异化**：闪应用内容形式**可改、可交互、创作成本极低**——自然语言生成后可随时调整角色、背景等，比传统文字/图片/视频更个性化。小 Lin 自己搓了「电子衣柜」。

**社区**：2025 年 4 月上线**灵光圈**——浏览、使用、修改他人应用，类似开源社区迭代。

## AI 原住民与创作生态

灵光不把用户叫「App 开发者」（太重），而称 **AI 原住民**——核心是有想法，不需要懂代码。深圳黑客松参与者从十几岁到 60 岁，非常广泛、时尚。

形态类似自媒体演进：门槛降低 → 平台涌现 → 从内容创作者到**应用创作者**。灵光圈想做这件事：
- 传统内容创作者转来
- 完全 AI 原生、每天指挥 Agent 完成生活任务的年轻创作者

**好创作者特质**：真正有想法（Coding Agent 后想法可由「超级程序员」实现）；适应灵光生态、跟随平台鼓励方向。

**平台鼓励方向**：
1. **知识科普**（如用动物视角看世界给小孩讲大自然）——AI 时代人应有更丰富精神世界、知道自己要什么
2. **情绪/人文关怀**
3. **可交互游戏内容**

## 商业化与壁垒

- 灵光上线半年，**免费**，优先用户价值
- 有用户自己做应用去社交媒体卖——顺应推出灵光圈社区
- 未来：**双面市场**（供给 + 内容消费），设计可持续机制
- **核心壁垒**：优质内容资产沉淀 + **自然语言生成闪应用**的基础设施全球领先（基模、后训练、产品优化完整链路）；移动端创建 App 体验优化较好

**版权**：后续在版权与知识产权保护上做更多工作。

## 「消失的应用」与生态影响

大家讨论**消失的软件**（Disposable Software）：
- Coding 已被 AI 代替，软件壁垒不再是开发能力，而是**解决了什么问题、积累了什么数据**
- 生态可能 **K 型分化**：几个大厂做大众应用，其余极度个性化
- **长尾会更长**：以前长尾应用难被搜到、难变现；灵光圈上任何想法都可低成本变成应用与内容
- 时间不会太久——基模迭代快、中国移动端使用深度与时间优势

**最大卡点**：让所有 C 端用户体验 Web Coding/Wish Coding 的魅力——下载灵光 App，输入简单指令（如「帮我创建一个拍照姿势指导的闪应用」），生成、调整、收藏或发布到灵光圈。

## 普通人能抓什么红利

1. **不必是程序员**：有想法 + Agent/Coding 工具 = 可交付应用
2. **C 端移动端红利**：中国基础设施适合跳过 PC 直达手机端创作
3. **AI 原住民身份**：从消费者同时成为创作者，在灵光圈等平台沉淀内容资产
4. **长尾个性化需求**：记账、情绪、小游戏、知识科普等「次抛」应用成本极低
5. **B 端视角**：金融分析师、程序员等已用 Claude Code 提效——职业场景亦在变革
