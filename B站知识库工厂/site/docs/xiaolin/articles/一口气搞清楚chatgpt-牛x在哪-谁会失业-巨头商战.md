> **摘要**：从图灵测试、模式匹配聊天机器人到 Transformer 与 GPT，讲者串起 ChatGPT 技术演进、OpenAI 从非营利到 capped-profit 的融资转折、RLHF 训练逻辑，以及微软 Bing 整合 vs 谷歌 Code Red 的搜索商战；并讨论大语言模型「下一词预测」本质与套路性工作失业风险。

## 聊天机器人演进：从 ELIZA 到 Transformer

- **1950 图灵测试**：文字对话中难辨人/机 → 可认为机器智能
- **1966 ELIZA**：心理治疗师设定 + 关键词 if-then（约 200 个），非真智能
- **1995 ALICE**：模式匹配（pattern matching）更强，仍预设回答
- **2001 SmarterChild**：机器学习 + IM 平台，3000 万用户、日收 10 亿条消息；**2007 被微软收购**
- **2010 年代神经网络**：需大量数据 + 算力；RNN 处理文字顺序慢、长句易忘
- **2017 Transformer**（谷歌论文）：并行学习大量文字，BERT/GPT 中的 T 均指此架构

## OpenAI 与 GPT 路线

- **2015**：马斯克、彼得·蒂尔等注资 **10 亿**成立 **OpenAI**（非营利，成果公开）
- **2018**：马斯克为避免与特斯拉 AI 利益冲突退出董事会
- **2019**：资金压力 → 转为 **capped-profit**（投资回报上限 **100 倍**，超额归 OpenAI）；**微软注资 10 亿** + 建全球第五超级计算机 → 成果不再完全公开
- **GPT 演进**：GPT-1（1.2 亿参数）→ GPT-2（15 亿）→ GPT-3（**1750 亿**，2020）
- **问题**：纯自监督训练，加大参数量改善有限——缺「好坏」反馈
- **RLHF**（人工反馈强化学习）：人类标注对话质量 → 2022.03 GPT-3.5 → 2022.11 **ChatGPT** 对话优化版

## ChatGPT 为何震撼

- 两个月月活破 **1 亿**（扩张速度史上最快）
- **本质**：大型语言模型计算「下一个词/句」的概率，非真正理解；像记忆力极好、不懂含义的小孩学大人说话
- **局限**：逻辑错误、编造答案（hallucination）、道德伦理离谱输出
- **突破意义**：极大提升人↔机文字沟通效率——人可直接说人话，模型自行「翻译」为任务执行

## 微软 vs 谷歌：搜索商战

| 维度 | 微软 | 谷歌 |
| --- | --- | --- |
| 立场 | 搜索老二（Bing ~3%），有动力 all in | 搜索垄断（~93%），广告收入 ~60% 来自搜索 |
| 动作 | 2023.01 再投 OpenAI **100 亿**；2.07 发布会整合 Bing + ChatGPT（Copilot for Web）；须用 Edge | 内测 Bard/LaMDA 更强但不愿自毁摇钱树；2.08 仓促发布会 Bard 事实错误 + 忘带手机 → 市值蒸发约 **1000 亿** |
| 成本 | ChatGPT 日运行约 **100 万美元**；单次查询能耗约为传统搜索 **10–100 倍** | 曾发 **Code Red**，集中全公司力量 |

**微软投资结构（讲者）**：OpenAI 利润 **75%** 先分给微软直至 **100 亿**回本；微软持股约 **49%**，仍有 capped 回报上限。

## 谁会失业：套路性工作

- 技术革新双刃剑：长期 GDP 可能上升，短期部分人失业
- **套路性工作**易被替代：有固定模板的故事、熟练闭着眼写的代码、批量网文、基础财务报告/设计/法律建议等
- **不是整职业消失**，而是工作中套路部分被 AI 接管
- **教育冲击**：美国 18+ 学生约 **90%** 用过 ChatGPT 写作业；体系未准备好 → 学校/公司一度一刀切禁止
- **版权、秩序**：AI 生成内容归属、交通系统类比——「车都会飞了但规则还没建好」

## 图灵论文余论

与其问「机器能否像人一样思考」，不如问「机器能否做人类能做的事」——当 99.9% 情况答对，模仿与理解之辩意义可能不大。
