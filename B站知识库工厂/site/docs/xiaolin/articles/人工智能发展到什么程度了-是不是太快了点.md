> **摘要**：ChatGPT 发布不到两月，GPT-4、微软 Copilot、文心一言、Midjourney V5 等密集落地。讲者区分分析型 AI 与**生成式 AI**，盘点 GPT-4 多模态升级、Alpaca 低成本微调、办公/搜索/金融/Auto-GPT 等应用，并讨论 AGI 争议与「取代你的是使用 AI 的人」。

## 生成式 AI vs 分析型 AI

- **分析型**（早已普及）：大数据、AlphaGo、人脸识别等——分析已有数据
- **生成式 AI**（Generative AI）：生成全新文字、图片、代码、音视频——AI 具一定「创造力」，向通用人工智能迈进一大步

## 2023 年 3 月密集事件（讲者时间线）

- 3.14 谷歌 PaLM API → 3.15 OpenAI **GPT-4** → 微软 Office 全家桶整合 → 3.16 百度**文心一言** + Midjourney V5 → 华为/阿里/360/商汤等跟进
- 趋势：不只发布底层大模型，更快速**嵌入应用**

## GPT-4 主要升级

- **多模态理解**（API 图片输入）：识别图像 + 常识与幽默（VGA 插手机、出租车后熨衣、剪线气球飞走等）；可根据手绘草图生成网页代码
- **准确性提升**：相对 GPT-3.5 各类人类考试分数普遍提高；美国律师考试从最差 **10%** 跃至前 **10%**
- **应用玩法**：Swift 代码、动画、贪吃蛇、律师函、以太坊智能合约等——**不到半年**从「小学生」到「大学生」水平（讲者判断）
- **商业机密**：微软投资后 OpenAI 不再公开参数量与细节

## 大模型生态与低成本微调

- 市场已有十数个通用大模型（GPT-4、PaLM、文心、通义千问、Llama、盘古、Claude 等）
- **Alpaca**（斯坦福）：基于 Meta 模型 **600 美元**微调，达 GPT-3.5 类似水平且**开源**——未来或人人可训小模型
- **评测困境**：难像手机参数对比；参数量不可尽信，需实测；或出现专门 AI 评级机构

## 应用层：从 API 到行业渗透

| 方向 | 案例 |
| --- | --- |
| 搜索 | New Bing、Google Bard、Perplexity |
| 办公 | Notion AI、**Microsoft 365 Copilot**（Word/PPT/Excel/Outlook 嵌 GPT-4） |
| 图片/视频/音乐 | Midjourney、DALL·E、Stable Diffusion、Adobe Firefly、Runway |
| 金融 | BloombergGPT（500 亿参数 + 数十年数据）；同花顺等接入，股价翻倍（讲者观察） |
| 自主 Agent | **Auto-GPT**（接 GPT-4 + 编程/搜索/记忆，自设目标自迭代） |
| 其他 | Duolingo Max、Shopify 商品描述、GitHub Copilot |

**二次训练**：巨头自训（如 Bloomberg）；中小公司基于开源/API 做领域微调——「通识教育后再上专业课」。

## 元宇宙 vs 生成式 AI 的商业对比

- 元宇宙/VR：愿景大但**当下商业价值低**，雷声大雨点小
- 生成式 AI：**当下**即有巨大商业利益 + 未来无限可能 → 微软催 OpenAI 未完全调试即推 GPT-4；各厂被迫宣布布局

## 风险与 AGI 争议

- **AGI**（通用人工智能）：全面超越人类、可能有自主意识
- 2023.03.22 微软 **155 页**论文称 GPT-4 现 AGI 早期征兆（专家反驳）；同日马斯克等 **2.4 万+** 签名公开信呼吁暂停训练强于 GPT-4 的模型至少 6 个月——竞赛不会因此停止
- **当下 ChatGPT**：仍是「词语接龙」概率游戏，不知自己在说什么；但进化速度极快
- **就业**：取代你的不是 AI，而是**使用 AI 的人**——多了解、多试用
