小Lin 在华为 HDC 开发者大会现场，从「钱花在哪、怎么激励开发者」的视角审视鸿蒙生态——不只看发布会数字，而看生态是否进入自循环。

## 鸿蒙现状：数字与临界点

最新数据：终端设备超 **6600 万**（今年目标 1 亿），开发者超 **1100 万**。

PC、移动生态本质是开发者与用户「滚雪球」，增长非线性，存在关键临界点——业界称**生态裂谷**：

| 市场占有率 | 生态状态 |
| --- | --- |
| &lt; 19% | 平台须砸钱补贴拉开发者 |
| ≥ 19% | 进入自循环：不开发鸿蒙会流失真实用户，开发者主动适配 |
| ≥ 35% | 自加速：更多开发者、更多用户正向飞轮 |

鸿蒙已过第一个临界点（「活下来了」），正冲向第二个。今年 HDC 规模约为去年两倍，年轻开发者明显增多；**中小开发者**被推到更核心位置——鸿蒙星光大道整条街均由中小开发者展示。

## 中小开发者现场样本

### 灵动小组件
桌面自定义卡片，可链接 App 或纯装饰。团队组合跨界（医生、动画师、农学背景）。开发约两年半，用户量约 **126 万**，流水约 **100 万**。

### 文档解析 App
将图片中文字、图表、公式提取并复制到 Word（保留公式格式与表格）。与鸿蒙合作将模型部署到手机，可实时扫描并调用底层能力。选择鸿蒙原因：软硬一体合作，亦有投资层面因素；To C 完全免费（在鸿蒙下载链接免费）。

### 悟空图像
对标 Photoshop 的专业图像工具，叠加 AI 自然语言交互（如「画成猪」即变形）。深耕专业图像赛道，国内对标 PS 仅其一家。鸿蒙开发难度更大（原生新系统、语言与适配需重做），但参加「鸿蒙有礼」活动后用户量从「小几百万」增至 **1300 万**——单场活动带来明显商业化。

### 在校生与夫妻档
- 大三学生：私有云音乐播放器，光影随歌词点亮；已独立开发十余 App，辅以 AI 助手与女友协作。
- 夫妻档：上传图片生成痘痘贴图纸，多用于爱豆应援。

## 余承东访谈要点

**中小开发者的战略意义**：早期鸿蒙需要底座能力与头部应用（树干）；树干长成后，枝繁叶茂靠中小开发者。举例：MUCAI 一句话生成歌曲，满足普通人作词谱曲梦想。

**有机会的赛道**（相对其他操作系统）：
1. 鸿蒙仍处起步阶段，各赛道空间大；
2. 创新体验：空间感、3D 主题/小组件、3D 空间影音、照片生成 3D 模型并可打印；
3. Agent 时代：已向合作伙伴开放 AHS 对接，App 能力可 Agent 化，经小艺一句话调用他人 Agent，降低使用入口深度。

**年轻开发者**：技术进步下，年轻人对时代与生活理解更深，能做出贴合年轻消费者的产品，形成「年轻人做、年轻人爱」闭环。

## 衡量指标的变化

鸿蒙已过「活下去」阶段，评估生态的指标应从「有没有某某 App、用户数、开发者数」等纯数字，转向：**生态里是否出现有意思、新的、只属于鸿蒙的东西**。中小开发者与年轻开发者，是看见生态「烟火气」的关键。

## 小结

看鸿蒙不能只听发布会：钱花在激励开发者、会场给中小开发者核心展位、头部应用补齐后转向「树干 + 枝叶」生态结构，才是实诚进展。19% 临界点已过，商业化案例（悟空图像、灵动小组件等）表明补贴与活动可转化为用户与流水；下一阶段胜负手在差异化体验与 Agent 时代的系统能力开放。
