# 天坑岗位避雷清单

## 一句话定义
数据方向求职四类雷区：付费内推是诈骗、专职数据标注零积累、数据录入校对不要主做、专职爬虫涉逆向易违法；数据专员/数据运营则可以去。

## 详细说明

| 类型 | 判断 | 理由 |
| --- | --- | --- |
| 付费内推/上缴工资 | 诈骗，坚决不碰 | 实测有中介称多家大厂"包进、一周走完流程"，代价上缴工资 60%~80%；某大厂 OD 裁员裁的就是这批人。"保证入职"合同无法律效力。凡让你掏钱找工作的都是骗焦虑 |
| 专职数据标注 | 坚决不去 | 每天点几十万张图，一年下来毫无积累；大厂标注全是外包。判断标准：看岗位考核指标是标注量还是背 GMV/KPI。工作中零星打标逃不掉，但不能是主业 |
| 数据录入/校对/文员 | 不要主做 | 录手写表、逐表核对，积累不到东西 |
| 专职爬虫 | 不要去 | 专职岗一定涉及逆向破解，容易坐牢，见 [[数据分析-工具-爬虫三不原则合法边界]] |
| 数据专员/数据运营 | **可以去** | 只要能接触到数据就有操作空间；转行拆两步更丝滑（先数分再换行业，或先入行业做运营再转数分） |

另：软考等证书对数据分析求职没用——团队里没有任何人有证。
