# 简历呈现优先级顺序

## 一句话定义

在职者简历按「分析经验 > 工具技能 > 项目经验 > 个人作品」排序；学生靠作品+分析思维体现；四项皆无则先学习再改简历。

## 详细说明

**在职者优先级**：
1. **分析经验**（最重要）：写个人简介首句——指标体系搭建、商业智能监控、专项分析等
2. **工具技能**：Excel/SQL/Tableau/Python 各到什么程度
3. **项目经验**：实习/工作中具体事项
4. **个人作品**：链接放最后

**学生无业务场景**：可写职业规划、热点行业见解、竞品分析等体现分析思维。

**分析经验表达模板**：公司现状 → 收集信息 → 判断方向 → 优化动作 → 遇到问题 → 如何解决 → 结果。

四项都不具备时：先按漏斗模型积累实力，不要从零硬改简历。
