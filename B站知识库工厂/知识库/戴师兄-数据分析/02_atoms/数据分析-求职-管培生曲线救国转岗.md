# 管培生曲线救国转岗

## 一句话定义

无数据分析实习时可选管培生轮岗机制「曲线救国」——在核心轮岗中自学 SQL+Tableau，用成果面试转岗数据分析。

## 详细说明

**操作要点**：
- 选管培前看清轮岗岗位是否核心，避免纯一线坑（如「杀三个月鱼」）
- 非数据岗也可考 SQL 权限、自主做看板/分析项目
- 公司建数据中台时，管培生转数据比例可能较高（案例约 1/4）

**嘉宾乐言案例**：
- 湖大经济学，无实习 → 管培 → 轮岗数据中台
- 运营岗考 SQL 权限 + Tableau 看板 → 转岗面试顺利，奖金更高
