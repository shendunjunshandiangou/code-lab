# 分析流程全流程覆盖进阶

## 一句话定义

分析师职业成长本质是分析流程覆盖范围的逐级扩展：从单一取数 → 看板报表 → 融入管理流程 → 跨团队行业专家。

## 详细说明

**覆盖进阶路径**：

1. **数据提取**（入门）：别人要什么数就给什么数
2. **看板 + 报表**（进阶）：从项目确立介入，搭建监控看板
3. **融入管理流程**（战略视角）：收益核算、部门贡献量化、新业务赋能、公司级数据建设
4. **跨团队行业专家**（头部）：在一线行业争抢一线资源，成为分析专家

**厨师类比**：能不能从买菜、切菜到炒菜全流程完成并获得食客好评，还是只会炒菜或只会切菜。

**关键跃迁动作**（20~30 万档）：
- 需求沟通时主动提案更好方案
- 分析报告业务视角先行——先说策略效果，再展开成因
- 参与 A/B 测试等业务决策讨论
