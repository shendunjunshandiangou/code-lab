# 大厂分析师一天工作流

## 一句话定义

大厂分析师白天开会、晚上加班做数据建设/专项分析/业务推进——该碰业务还是要碰，差距在业务理解。

## 详细说明

| 时段 | 内容 |
| --- | --- |
| 早餐 | 确认报表/看板有无报错 |
| 白天 | 各种会议到下午 4-5 点 |
| 晚上 | 数据建设 SQL、专项分析报告、业务推进跟踪 |

- 报表报错要最快速度修好，否则被投诉/降绩效
- 数仓关系要维护好（奶茶套近乎）
- 业务推进花时间多但未来职业发展关键
