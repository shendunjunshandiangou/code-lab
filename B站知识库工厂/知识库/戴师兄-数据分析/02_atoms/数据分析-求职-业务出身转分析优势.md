# 业务出身转分析优势

## 一句话定义

先走运营/管培等业务一线再转数据分析，如同特种兵回去做参谋——同样都是参谋，地位完全不同。

## 详细说明

**戴师兄路径**：被迫先选运营 → 反思后认为正确；有业务出身的人在团队里地位不同。

**嘉宾乐言路径**：
- 无实习 → 管培生曲线救国 → 轮岗进数据中台
- 运营岗考 SQL 权限、自主做 Tableau 看板 → 转岗面试顺利

**核心**：懂业务不是「懂业务的数据分析」，而是「懂数据分析的业务」——先做完业务再回中台，否则难以服众。
