# 年底求职两问避坑法

## 一句话定义
年底求职向面试官问「为什么招我」和「前任为什么离职」，观察回答与神情即可判断岗位潜在风险。

## 详细说明

### 两个问题

1. 这个工作为什么会招我？
2. 我的前任为什么会离职？

### 解读信号

| 回答 | 可能风险 |
| --- | --- |
| 前任因个人发展离开 | 薪资/晋升通道可能不行 |
| 前任被裁员 | 岗位需求不明确或要求过高 |
| 不方便沟通、支支吾吾 | 有所隐瞒 |
| 希望新人带来新增量 | 前任可能没做出成绩，岗位有风险 |

面试官越慌张，问题可能越大。回答不直接即视为潜在风险，需继续追问。
