# 必学四工具与掌握程度

## 一句话定义

经典路径 **Excel → SQL → Tableau → Python** 四件套可覆盖分析师日常；各工具学到「能求职」的最低线有明确标准。

## 详细说明

| 工具 | 顺序 | 学到什么程度 | 推荐资源 |
| --- | --- | --- | --- |
| Excel | 1 | 基础函数、作图、透视表 | Office 自带教程；泽秀老师课程 |
| SQL | 2 | 七语句 + 基础函数；增删改了解 | 《SQL 必知必会》+ SQLZOO + 牛客 20 题 |
| Tableau | 3 | 半天入门；能搭**仪表盘** | 官方社区；《商业仪表盘可视化解决方案》 |
| Python | 4 | 控制流+函数；NumPy/Pandas/Matplotlib/Sklearn；**完整建模项目** | 松鼠书+LeetCode；科赛项目 |

**Python 用途限定**：大规模处理 + 高精度建模；**不做**日常可视化（交给 BI）。

**Python vs R**：选 **Python**（深度学习生态更全）。
