# 三级求职阶段模型

## 一句话定义
求职从 Level1 证明能干活 → Level2 证明是优质求职者 → Level3 证明是「你家的」优质牛马（JD 对口、非你不可）。

## 详细说明

| 阶段 | 要求 |
| --- | --- |
| Level 1 | 证明你是牛马（买方市场，降预期） |
| Level 2 | 证明你是**优质**求职者（有竞争对比） |
| Level 3 | 证明你是**对方家的**优质牛马——工作内容与 JD 对口，体现首选 |

提高响应率的核心：从 Level 2 迈向 Level 3 的 JD 对口。
