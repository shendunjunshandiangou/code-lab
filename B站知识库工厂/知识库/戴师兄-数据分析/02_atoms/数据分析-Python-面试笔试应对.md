# 面试笔试应对

## 一句话定义

Python 面试：**直接说会**并追问业务需求；简历问法结合效率/业务；陌生场景承认未用过该包但学得快；笔试离线可借助 AI，在线需适度积累但不必海量刷题。

## 详细说明

**面试三场景**：

1. **会不会**：说会 → 反问「具体什么需求、用哪个包」（60 万包不可能全会）
2. **简历经历**：如实答用法，强调**效率提升**（增长可选）
3. **陌生场景**：未用过该包，但会 Python、新包上手快 → 反问团队用何包/是否接手代码

**笔试**：
- 邮件离线题：可用 AI 辅助
- 在线/线下：需基础积累，但**不必大量刷题**——岗位与业务经验更重要

**底层逻辑**：企业雇人看业务效率与增长；工具服务于业务，非为刷题。
