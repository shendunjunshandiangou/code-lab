# 数据透视表创建与数据源

## 一句话定义

数据透视表通过拖拽字段快速做多维汇总；创建效率可通过快速访问工具栏 + Alt 快捷键提升，数据源推荐智能表以自动扩展。

## 详细说明

### 快速创建

1. 插入 → 数据透视表 → 添加到快速访问工具栏。
2. 按 `Alt+数字`（如 Alt+4）一键插入。

### 数据源三种方式

| 方式 | 新增行处理 |
| --- | --- |
| 框选连续区域 | 需手动「更改数据源」 |
| 整列选中 | 刷新自动纳入新行 |
| 智能表（Ctrl+T） | 自动检测新行并扩展 |

### 复制透视表

必须复制完整透视表（含所有组成部分），粘贴后才有「数据透视表工具」；部分复制只得静态数据。
