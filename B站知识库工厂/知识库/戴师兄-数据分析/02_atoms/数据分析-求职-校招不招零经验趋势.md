# 校招不招零经验趋势

## 一句话定义
大厂校招本质不招零经验应届生，倾向实习直接转正；社招持续招高质量复合型人才，纯报表执行岗已不再是分析师。

## 详细说明

- **社招**：一直在招，要求很高，招高质量人才，不再补执行层
- **校招**：不招零经验，没有多余岗位花一年培养，倾向实习转正
- 校招不是为应届生设计，是为**有能力的人**设计
- 纯做报表 = 数据专员（初级单一技能），分析师需从前负责到后（指标体系 + 数据建设 + 分析 + 出谋划策）
- 两大趋势：**职能高度复合** + **自负盈亏**（背营收指标）

**补充（转行必看 / 没数分经历）**：
- 校招难度可比拟考 985 甚至更难（需学历 + 实习 + 实战能力）
- 感觉不如同龄人优秀 → 校招可直接放弃，全力社招「经验不限」岗位
- AI 时代：企业不培养应届生，有 1 年经验者 + AI 提效 10%～20% 即可满足岗位
- 应届生 7 月毕业与社招入职无本质区别，可提前找社招实习错位竞争
- 校招黄金上岸点：毕业后**第二个金九银十**（毕业后半年最难伪造简历）
