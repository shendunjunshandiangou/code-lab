# 敏捷 BI 与复杂报表

## 一句话定义

复杂报表是落后需求；企业应走向敏捷 BI 面向决策；分析师应理解报表背后真实需求，用交互可视化实现，而非被落后需求绑架。

## 详细说明

**复杂报表**：服务于传统企业在特定阶段的数据呈现习惯，对大部分公司 **不是方向**。

**敏捷 BI**：面向业务决策；个人分析师 **不应被复杂报表需求绑架**——强行用敏捷工具填复杂报表 bug 等于被绑架，应 **直接拒绝**。

**正确做法**：超越显而易见的需求 → 理解领导真正要什么 → 用 Tableau 等 **交互可视化** 实现。
