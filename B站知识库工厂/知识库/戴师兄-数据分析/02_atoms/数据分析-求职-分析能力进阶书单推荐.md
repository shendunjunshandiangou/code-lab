# 分析能力进阶书单推荐

## 一句话定义
靠书籍积累垂类分析能力可支撑至月薪 1.8～2 万，超过此水平才需系统学习数据分析思路。

## 详细说明

**五本分析思维经典**（戴师兄推荐）：

| 书名 | 核心价值 |
| --- | --- |
| 《金字塔原理》 | MECE 拆解、SCQ 表达 |
| 《世界观》 | 拥抱变化、体系自洽、实践检验真理 |
| 《模型思维》 | 24 种模型工具箱 |
| 《事实》 | 十种思维误区与数据说话 |
| 《精益数据分析》 | 30+ 商业案例 |

| 层次 | 推荐 |
| --- | --- |
| 零基础认知 | 《深入浅出数据分析》通读 + 《深入浅出统计学》看到不懂为止 |
| 入门 | 《精益数据分析》 |
| 产品/运营思维 | 梁宁《增长思维 30 讲》《产品 30 讲》、《真需求》 |
| 案例 | 《幕后产品》（网易云音乐 0→1） |
| 进阶 | 于君《产品方法论》 |

有这些积累，在某一垂类行业的分析能力基本够用，**直到月薪 1.8 万或 2 万以上**才需系统学习分析思路。
