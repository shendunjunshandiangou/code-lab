# 数分学习顺序误区

## 一句话定义

先学 Python 是常见误区；正确顺序为 Excel → BI → SQL → 最后 Python（或无代码 Prep Builder 替代）。

## 详细说明

**误区**：一上来学 Python——多数场景用 Excel 报表、大厂用 BI 看板；强依赖 Python 往往说明数据建设差。

**正确顺序**：

| 步骤 | 工具 | 作用 |
| --- | --- | --- |
| 1 | Excel | 理解数据处理在发生什么 |
| 2 | BI | 报表美化、动态图表、提效 |
| 3 | SQL | 与 BI 表连接、Excel VLOOKUP 逻辑相通 |
| 4 | Python | 前三者不够再用；或 Prep Builder 拖拽替代 |
