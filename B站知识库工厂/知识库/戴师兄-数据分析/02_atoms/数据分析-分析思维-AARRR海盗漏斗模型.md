# AARRR海盗漏斗模型

## 一句话定义

获取→激活→留存→变现→推荐五阶段流量漏斗，各阶段对应不同指标，与 OSM 连用可系统性提升成交额。

## 详细说明

- 名称来自英文谐音梗（海盗吆喝声）
- 本质是通用漏斗模型，各业务阶段顺序可能不同（如拼多多先推荐再留存）
- 与 OSM 连用：目标提升成交额，策略优化每个环节各 10%，结果可提升 20~30%
- 产品体验好 + 推荐 = 零成本获客
