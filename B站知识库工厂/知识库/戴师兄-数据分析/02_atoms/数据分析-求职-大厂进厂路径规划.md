# 大厂进厂路径规划

## 一句话定义

进大厂像跳棋——校招走「实习滚雪球」，社招走「先转型再积累再冲击」， rarely 一步到位。

## 详细说明

### 校招

| 背景 | 路径 |
| --- | --- |
| 名校 | 大厂实习 → 留任 → 带经历校招 |
| 普通院校 | 第一份实习随便找、迈出第一步 → 学硬技能+作品 → 第二份大厂实习 → 边实习边校招 |

### 社招（首份工作不在大厂）

1. 第一份工作干满 1 年，不裸辞
2. 自学完成岗位转型
3. 去规模不错公司积累 1 年，做与大厂相关业务的项目
4. 岗位+行业匹配后冲击头部大厂

代运营等上下游公司是常见跳板。
