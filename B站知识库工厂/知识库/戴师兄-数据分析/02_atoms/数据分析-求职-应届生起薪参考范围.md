# 应届生起薪参考范围

## 一句话定义
数据分析应届生起薪大致 6～12K（看实习和背景），一线大厂数分校招可达 1.8～2 万；JD 里的 3K 通常是实习期工资。

## 详细说明

| 场景 | 起薪 |
| --- | --- |
| 一线大城市运营岗 | 6～8K，很难低于 5K |
| 数据分析应届（一般） | 6～12K |
| 美团/字节等大厂校招数分 | 约 1.8 万，甚至 2 万 |
| 成都等地区 6～8K | 对应届生已算不错 |

JD 写「3～8K」时，3K 往往是**拿毕业证前的实习工资**，毕业后才给正式薪资签劳动合同。
