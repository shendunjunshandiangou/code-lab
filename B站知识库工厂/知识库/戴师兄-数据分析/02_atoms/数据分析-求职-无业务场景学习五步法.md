# 无业务场景学习五步法

## 一句话定义

没有工作数据和业务场景时，按「公开数据→Excel→SQL→Tableau→Python」五步学习，并用代码存档和作品集证明实力。

## 详细说明

| 步骤 | 学到什么程度 | 如何证明 |
| --- | --- | --- |
| 数据来源 | 和鲸/Kaggle 等公开数据集 | 选对时事/量级合适的数据集 |
| Excel | 报表制作 + 数据透视表/图，老板站背后也能改 | 实操熟练即可 |
| SQL | 核心查询语句 + 牛客刷题 | **每行代码存档** + 刷题记录 |
| Tableau | 简易联动仪表盘 | 作品截图/链接 |
| Python | pandas + sklearn；可视化优先 Tableau | 项目/建模作品 |

**SQL 特别说明**：编程空口无凭，面试可能现场手撕代码；存档最难代码 + 牛客刷题量最有说服力。
