# Pandas 分析师定位

## 一句话定义

分析师 Python **只需掌握 Pandas**（复杂/海量数据处理）+ **成型项目**证明能力；可视化用 PyGWalker 或 BI；爬虫用后羿，勿深钻。

## 详细说明

**环境**：Jupyter Notebook 打草稿；PyCharm 可选。

**必学**：
- 基础语法（控制流、函数）
- **Pandas** — 覆盖 90%+ 分析师 Python 处理需求
- **PyGWalker** — Python 内类 Tableau 拖拽（2023 版推荐）

**必做**：完整「获取→清洗→分析」**项目**（非小游戏/demo）。

**不做**：
- 日常可视化（BI 更高效）
- 爬虫攻防（requests/scrapy ROI 低、合规风险）
- 全包精通（NumPy/Sklearn 按 JD 补充）

**与 R**：新手选 Python。
