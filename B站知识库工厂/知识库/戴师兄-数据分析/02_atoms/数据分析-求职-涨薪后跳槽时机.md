# 涨薪后跳槽时机

## 一句话定义

涨薪后跳槽至少留 3 个月薪资流水——HR 只认流水，刚涨薪就跳会被压薪；稳定 3 个月后再跳，薪资博弈上无话可说。

## 详细说明

- HR 谈薪时只认**薪资流水**，不认口头承诺
- 刚涨薪就跳槽 → HR 可能趁机压薪
- **建议稳定 3 个月后再跳**（极限 1 个月）
- 3 个月后可以说「提升得差不多了」，HR 在薪资拉扯上没话讲

**反面案例**：
- 7 月才许诺涨薪 30%、想金九银十跳 → 还得再干到 8–9 月，又被拖小半年
- 不如早点跟老板谈涨薪，或明确做什么能加快进程
