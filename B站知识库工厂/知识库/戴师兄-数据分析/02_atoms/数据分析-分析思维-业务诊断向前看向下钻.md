# 业务诊断向前看向下钻

## 一句话定义
指标异常诊断的基本方法：向前看（业务流程往前倒查各环节是否同步异常）+ 向下钻（拆解到具体维度/人群找下滑来源）。

## 详细说明

典型面试题：「DAU 某天下降如何找原因？」

### 向前看

业务流程往前倒：
- 活跃下滑 → 注册环节是否已下滑？
- 中间转化（登录）是否出问题？
- 登录后是否因 bug 无法进入功能导致流失？

### 向下钻

DAU 下滑 → 是哪个维度/人群在下滑？

向前看 + 向下钻基本能判断大部分原因。做向前看/向下钻前还需做分析前置准备。业务诊断是分析师笔试面试必考基本功。
