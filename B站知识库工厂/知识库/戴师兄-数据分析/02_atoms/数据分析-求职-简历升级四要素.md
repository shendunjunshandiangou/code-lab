# 简历升级四要素

## 一句话定义

除 JD 对口外，用稀缺、权威、喜好、认同四条线强化简历——你会别人不会、有大厂背书、热爱行业、口碑好。

## 详细说明

| 要素 | 含义 |
| --- | --- |
| 稀缺 | 某些技能只有你会 |
| 权威 | 大厂/名企工作或重要项目 |
| 喜好 | 真心热爱目标行业与岗位 |
| 认同 | 前同事/领导评价好 |

与 [[数据分析-求职-转行简历模拟升级法]] 配合使用；仍无法写出满意经历时，需回到硬技能与作品集积累。
