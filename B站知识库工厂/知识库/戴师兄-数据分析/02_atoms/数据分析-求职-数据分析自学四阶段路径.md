# 数据分析自学四阶段路径

## 一句话定义

自学分入门（认知+基础工具）和求职（SQL 进阶+Python 项目+建模）两阶段共八个目标，完成后可达初级数据运营/分析师水平。

## 详细说明

### 入门阶段

| 目标 | 内容 |
| --- | --- |
| 1 认知 | 《深入浅出数据分析》通读 + 《深入浅出统计学》看到不懂为止 |
| 2 工具 | Excel → SQL（6～10 天通关查询）→ Tableau → Python 三件套 |
| 3～4 | 衔接求职阶段 |

### 求职阶段

| 目标 | 内容 |
| --- | --- |
| 1 输出 | 无业务场景时集中产出一份竞品/行业分析报告（~1 万字） |
| 2 SQL | 开窗函数 + 牛客刷完；Hive 如岗位需要 |
| 3 Python | pandas/numpy/matplotlib + **和鲸项目复现**（必须做项目） |
| 4 建模 | sklearn 入门 + 建模项目（简历需要时） |

**进阶三方向**（入行后再深究）：算法建模、数据工程/数仓、深度业务专精。
