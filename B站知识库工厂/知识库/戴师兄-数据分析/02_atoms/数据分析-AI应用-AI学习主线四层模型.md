# AI学习主线四层模型

## 一句话定义

学 AI 主线 = LLM 大模型 + Harness 生产力壳 + Skills 稳定 SOP + CLI 省上下文；新产品按此四层归类即可判断边界。

## 详细说明

| 层 | 作用 | 代表 |
| --- | --- | --- |
| LLM | 会说话，无手脚 | GPT、Claude、KIMI、豆包 |
| Harness | 赋予工具调用与长时工作 | Claude Code、Codex、Cursor、OpenClaw |
| Skills | Markdown SOP，触发式读取，防自由发挥 | 分析报告五步 SOP 等 |
| CLI | 替代 MCP 一次性喂接口，Agent 自查 help | 终端命令封装 |

- OpenClaw = Harness（远程通讯+记忆+cron），需挂载其他模型
- 个人 AI 效能取决于：模型强度 × Harness × Skills 是否调到工作场景
