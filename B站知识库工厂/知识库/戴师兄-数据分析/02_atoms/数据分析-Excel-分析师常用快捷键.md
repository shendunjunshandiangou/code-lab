# 分析师常用快捷键

## 一句话定义

分析师日常最高频的 Windows 与 Excel 快捷键组合，用于文件访问、窗口管理、单元格导航、筛选、填充和截图，减少鼠标操作。

## 详细说明

### Windows 通用

| 快捷键 | 功能 |
| --- | --- |
| `F2` | 重命名 |
| `Ctrl+Z` | 撤销 |
| `Win+E` | 打开此电脑 |
| `Win+D` | 显示/恢复桌面 |
| `Win+方向键` | 窗口分屏 |
| `Alt+F4` | 关闭窗口 |

### Excel 核心

| 快捷键 | 功能 |
| --- | --- |
| `Ctrl+Shift+L` | 开关筛选（拿到表格默认操作） |
| `Ctrl+方向键` | 跳转到数据区域末端 |
| `Ctrl+Shift+方向键` | 选中到数据区域末端 |
| `Alt+=` | 快速求和 |
| `Ctrl+;` / `Ctrl+Shift+;` | 当前日期/时间 |
| `Alt+Enter` | 单元格内换行 |

### 填充技巧

- 选两个数拖拽 → 等差数列；选一个数拖拽 → 复制。
- 按住 `Ctrl` 拖拽 → 强制复制。
- 按住 `Shift` 拖拽列 → 插入列而非覆盖。
