# 副业第二增长曲线

## 一句话定义

副业正确态度是用空闲时间探索「下一收入更高的主业」——不能长期主业副业两头干，交叉探索期结束后必须二选一。

## 详细说明

| 场景 | 做法 |
| --- | --- |
| 追收益 | 副业 = 第二增长曲线；收益超主业 → 逐步切换重心至 100% 投入 |
| 副业不及主业 | 收回精力认真干主业，收获更大 |
| 追兴趣 | 无收入预期，当爱好；有变现机会再加大投入 |

**核心**：一个人不可能同时干好两件事。副业顺利时感觉好，但交叉期很难持久。
