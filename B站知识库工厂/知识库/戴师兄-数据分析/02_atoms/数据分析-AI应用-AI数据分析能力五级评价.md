# AI数据分析能力五级评价

## 一句话定义

测评 17–20 款 AI 做数据分析，分人上人/顶级/夯/NPC/拉五级；底线是能上传数据且出图表，夯级（Claude/GPT/KIMI/MiniMax）可端到端辅助分析。

## 详细说明

| 等级 | 标准 |
| --- | --- |
| 拉 | 不支持上传或无输出 |
| NPC | 能上传但只给文字方案、不出图/文件 |
| 夯 | 能出图表报告，可辅助分析（收费） |
| 顶级/人上人 | 接近资深分析师产出 |

多数国产模型卡在 NPC：描述性统计行，但不出图。
