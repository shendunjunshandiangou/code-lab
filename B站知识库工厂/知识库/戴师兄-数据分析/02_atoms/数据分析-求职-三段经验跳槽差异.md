# 三段经验跳槽差异

## 一句话定义

HR 对「应届生 / 1～2 年 / 3～5 年」三类候选人的评价维度完全不同——工具要多会、中期要即插即用、资深要有行业玩法。

## 详细说明

| 阶段 | HR 视角 |
| --- | --- |
| **应届生** | 看潜力；无潜力则比性价比——工具多会，别押宝单点；买方市场 |
| **1～2 年** | 职场「不想要」档；需成型专项能力或即插即用，否则不如招校招自己培养 |
| **3～5 年** | 需明确行业规划与玩法经验；找执行层以上，非纯基层 |

**对应策略**：
- 应届：花枝招展展示综合实力
- 1～2 年：岗位需要什么赶紧补
- 3～5 年：行业经验 + 专业技能双齐全才能卖出高价
