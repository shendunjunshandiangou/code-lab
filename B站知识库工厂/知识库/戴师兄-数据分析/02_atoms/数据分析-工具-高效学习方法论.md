# 高效学习方法论

## 一句话定义

学数分应**以终为始**：算法+在看人数选课、先框架后细节、早投简历迭代；AI 时代人类专注「会做」和「说清怎么做」。

## 详细说明

1. **选对老师**：播放量 + 大厂背书 + 实时在看人数验证质量
2. **先框架后课程**：通读博主多篇自学攻略，对比迭代版本
3. **数据选课**：同 UP 下在看多的短课优先于过干长课
4. **以终为始**：简历模板 → Boss 直聘投递 → 看响应再深学（勿纯囤课）
5. **AI 分工**：简历/JD 匹配交 AI；人学**方法论+做出分析报告**
6. **作品底线**：可以没有 BI 看板，但需**有图有业务逻辑的分析报告**

可套用产品、运营等其他方向。
