# 先就业再择业最优路径

## 一句话定义

最优转行路径：先找有业务场景的工作，再把 Excel/SQL/BI/Python 应用到岗位，简历凸显数分能力后投递。

## 详细说明

**业务场景定义**：能跟业务/用户/老板沟通，做「接下来怎么做才能更好赚钱」的决策。

各工具在现岗作用：

| 工具 | 作用 |
| --- | --- |
| Excel | 日常高效输出、美观报表 |
| SQL | 扩大数据权限，处理百万级数据 |
| BI | 配合 SQL 提效，汇报美观（颜值即力量） |
| Python | 压舱石，不受工具框架限制 |

步骤：先把工具应用到工作岗位 → 简历提炼数分能力 → 开始投递。
