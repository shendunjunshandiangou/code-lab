# 先沾边再头部转行路径

## 一句话定义

转行行业切换遵循「先沾边再头部」：先去与目标行业相关的边缘/垂直公司，再跳一线平台。

## 详细说明

| 原行业 | 路径 |
| --- | --- |
| 房地产/土木 | 链家/贝壳 → 美团酒店/民宿（长租） → 本地生活 |
| 本地生活 | 餐饮/物流/客运/旅游 → 美团/滴滴/哈啰/高德/携程 |
| 金融/银行 | 网贷（数据建设完善） → 美团钱包/京东白条 → 互联网金融/蚂蚁金服 |
| 电商 | 代运营 → 小品牌方 → 淘宝/拼多多/京东 → 得物/小红书 |
| 土木天坑 | 美团/携程酒店事业部 → 本地生活 |

- 抓住行业窗口（如美团民宿开放长租房）跳转最丝滑
- 与 [[数据分析-求职-行业岗位只能换一个]] 配合：先进行业再换岗
