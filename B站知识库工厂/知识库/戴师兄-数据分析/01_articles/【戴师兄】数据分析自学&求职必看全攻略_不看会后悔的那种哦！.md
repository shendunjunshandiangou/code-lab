# 数据分析自学与求职全攻略（2021 版）

**摘要**：自学分入门与求职两阶段——先读《深入浅出数据分析/统计学》建立认知，再按 Excel→SQL→Tableau→Python 通关工具；求职阶段用竞品分析报告等集中输出证明能力；SQL 刷牛客、Python 做和鲸项目复现；入门即可达初级分析师水平。

## 一、入门阶段（四个目标）

### 目标一：建立分析认知

1. **《深入浅出数据分析》**：通读一遍，理解如何用数据解决简单业务问题
2. **《深入浅出统计学》**：看到理解不了的地方就停，不要打击积极性

### 目标二：掌握基础工具

| 工具 | 学习要点 | 时间参考 |
| --- | --- | --- |
| Excel | B 站免费课 + 实操 | — |
| SQL | 先看 SQL 认知视频再学；6～10 天可通关查询语句 | 几十小时 |
| Tableau | 能搭仪表盘 | — |
| Python | pandas + numpy + matplotlib 三件套 | — |

学 SQL 前必看认知视频（MySQL/PostgreSQL 区别、要不要装数据库、Hive 等），避免装数据库浪费数天。

### 目标三 & 四

- 完成基础工具后进入求职阶段资源

## 二、求职阶段（四个目标）

### 目标一：直面问题并输出

- 无业务场景时：**广泛涉猎 → 集中输出一份**（如竞品分析报告 1 万字以上）
- 可针对意向公司做竞品分析（如 B 站 vs 抖音/快手）
- 不要广泛涉猎又广泛输出，质量优于数量

### 目标二：SQL 进阶

- 掌握开窗函数、数据转置（转置更应在 Excel/Tableau 实现）
- 尚硅谷 SQL 学到 P101（查询）即可；刷完牛客 SQL 实战
- 大厂难题可能超牛客难度，当场不会可回来慢慢解
- Hive SQL 如岗位需要，100 条以内定制语句搞懂即可

### 目标三：Python 实战

- pandas ≈ Excel 透视表前置；numpy 科学计算；matplotlib 出图
- **必须做项目**：和鲸社区高赞项目复现，换数据集即变自己的项目
- 只会三件套没项目 = 白学

### 目标四：建模入门（简历需要时）

- 推荐 sklearn 教程 + 和鲸建模项目
- 大多数公司数据量不足以支撑有效建模（如 1000 条数据预测无意义），但 JD 常要求

## 三、入门 vs 进阶差距

入门+求职阶段学完后可达**初级数据运营/分析师**水平。以下三方面拉开差距，但入行后再深究：

1. **算法建模**（数学要求高）
2. **数据工程/数仓**
3. **深度业务专精**

## 四、求职配套

- 简历、投递、笔试、面试、选 offer、入职全流程覆盖
- 2021 年求职环境有变化，薪资与岗位发展需结合最新视频补充

## 文章正文

这是 2021 版最新最全的数据分析自学与求职全攻略直播。相比主页置顶的旧版，本次根据 3～4 月求职环境变化更新了自学路径和求职路径——不仅含简历，还覆盖投递、笔试、面试、选 Offer 乃至入职，全程可互动提问。

### 入门阶段四个目标

**目标一：建立分析认知。** 正式学工具前通读《深入浅出数据分析》——有趣、场景化，从 CEO 视角给任务、用信件/文件形式教提取信息、拆分问题、总结数据并布置作业；「不会数据分析的人买一本通读一遍，就能用数据分析解决简单业务问题」。之后读《深入浅出统计学》——很多高校数分设在统计学院下，图表制作需要统计基础；**看到完全理解不了、学不进去的地方就停**，不要打击积极性，正反馈为零为止即可，建模内容留求职阶段。

**目标二：掌握基础工具。** Excel 无折扣，工作无法开展。SQL 先**必看戴师兄 SQL 认知视频**——MySQL/PostgreSQL 区别、要不要装数据库、Hive 是否要学等，「学 SQL 前不看，90% 以上会踩坑，装数据库装一两天都有」。分析师查询语句 **6～10 天、几十小时可通关**；跟元旦三天高密度 SQL 入门视频刷题、记笔记，掌握核心语句运行原理。细节如 LIKE 模糊查询可查漏补缺未明学院教程（更全面但更慢）。SQLZOO 习题刷完「毫无难度」，入门阶段 SQL 不必再精进——目标是快速搭建框架、覆盖日常大多数数据需求。爬虫：「学得好牢里蹲」；大厂分析师主要取自己业务库数据，爬虫作核心竞争力已偏技能点；**学了 SQL 不能不学 Excel，Excel 必须掌握**。Excel 推荐泽秀老师基础教程（王佩丰老师也可），掌握常用函数和数据透视表。BI 工具（Tableau/Power BI/FineBI 等）在大规模数据、复杂探索性分析时必须学；推荐 Tableau，操作最符合可视化原理；未明学院 Tableau 课程基础图表都会做——「入门阶段完全不用花一分钱，按目标逐步解锁就能学会」。

### 求职阶段四个目标

**目标一：直面问题并输出。** 深入理解《精益数据分析》——专业术语多、默认有业务经验，没经验读也能获启发，强调指标和量化，「对分析师是较好知识框架」。广泛积累：36 氪、人人都是产品经理等，如 POP MART 护城河、B 站/虎牙/抖音/斗鱼直播赛道推演等；**广泛涉猎 + 集中输出一份**——不要广泛涉猎又广泛输出；产出深度产品/行业分析报告，**少说上万字**，不比毕业论文难。可针对意向公司做竞品分析（如去 B 站则分析 vs 抖音快手）；知乎搜框架、收集信息、横向比较、产出结论，更好还可推演未来。

**目标二：SQL 进阶。** 掌握开窗函数、数据转置——转置更应在 Excel/Tableau 实现。尚硅谷 SQL 学到 P101（查询）即可；牛客网 SQL 实战刷完；大厂难题可能超牛客，当场不会回来慢慢解。Hive SQL 岗位需要时，100 条以内定制语句搞懂即可。

**目标三：Python 实战。** pandas ≈ Excel 透视表前置；numpy 科学计算；matplotlib 出图。**必须做项目**——和鲸社区高赞项目复现，换数据集即变自己的项目；「只会三件套没项目 = 白学」。

**目标四：建模入门（简历需要时）。** 推荐 sklearn 教程 + 和鲸建模项目。大多数公司数据量不足以支撑有效建模（如 1000 条数据预测无意义），但 JD 常要求。

### 入门 vs 进阶

入门+求职阶段学完可达**初级数据运营/分析师**。拉开差距的三方面（入行后再深究）：算法建模（数学要求高）、数据工程/数仓、深度业务专精。

### 求职配套

简历、投递、笔试、面试、选 Offer、入职全流程覆盖；2021 年环境有变化，薪资与岗位发展需结合最新视频补充。
