> **摘要**：分两部分——Tableau 下载安装、注册试用、密钥管理与学生账户申请；以及用各省疫情数据快速上手连接数据并制作柱状图、条形图、饼图、折线图、散点图五大基础图表。

## 一、下载、安装与账户

### 1.1 下载

- 从视频下方 Tableau 下载网址进入版本列表，选择最新版本。
- 滚动到页面底部，按 Win / Mac 对应链接下载（大陆已不支持 Win32，需到台湾站点；老版本 Win32 可用独立链接）。
- 下载较慢时可复制链接到迅雷加速。

### 1.2 注册与安装

1. 下载页右上角「登录」→「创建账户」，填写姓名、邮箱、密码。
2. 邮箱收到激活邮件，点击链接完成激活。
3. 双击安装程序，**勾选默认两项，不要自定义安装**（自定义易出现路径问题；系统盘需预留 2–3 GB）。
4. 安装完成后 Tableau 通常自动打开；首次打开可「立即开始试用」，输入登录信息即可。

### 1.3 试用与密钥

- 每台电脑首次安装可 **14 天试用**；试用结束后需购买密钥或申请学生账户。
- **每个有效密钥最多激活两台电脑**，超出可能被追责；多机使用需先停用旧设备上的密钥。
- 管理路径：**帮助 → 管理产品密钥 → 激活 → 使用产品密钥激活**，粘贴密钥即可。

### 1.4 学生账户

- 在读学生可凭学生证等证明 **免费申请一年** Tableau 激活码，在读期间可续期。
- 填写信息须与后续提交的学历证明/学生卡 **完全一致**（姓名、学校、在读期限等）。
- 提交后等待 1–3 个工作日审核；成功则邮件发送密钥，失败可按要求重新上传证明。

## 二、连接数据

### 2.1 数据源与连接方式

- 练习数据来自 Tableau Public 社区（视频下方下载链接），示例为「全中国各省疫情数据」Excel。
- 打开 Tableau → **连接到文件**：
  - `.xlsx` / `.xls` → Microsoft Excel
  - `.csv` → 文本文件
- **连接到服务器**支持主流数据库（后续课程详讲）。

### 2.2 实时 vs 数据提取

| 模式 | 说明 |
| --- | --- |
| 实时 | 每次操作直接读源文件 |
| 数据提取 | Tableau 预加载为 `.hyper` 格式，**数据量大时速度更快**，一般推荐使用 |

提取后可通过「刷新」重新拉取源数据变动。

### 2.3 保存格式

| 格式 | 说明 |
| --- | --- |
| `.twbx` | 工作簿 **含数据**，便于练习、导出、后续操作 |
| `.twb` | **不含数据**，体积小、保密性好；接收方需自行加载数据源 |

学习练习建议用 `.twbx`。

### 2.4 界面三件套

底部三个图标：

- **工作表**：单一图表
- **仪表盘**：组合多个工作表（不能在仪表盘内新建图表）
- **故事**：类似 PPT，可组合工作表与仪表盘做交互汇报

## 三、五大基础图表

数据字段示例：省份（维度）、新增确诊/新增治愈/新增死亡/累计确诊（度量）、日期。

### 3.1 柱状图

1. 省份 → **列**，新增确诊 → **行**
2. 排序（降序）；湖北离群值高 → 右键省份 **排除**，或拖入筛选器取消勾选
3. 新增确诊 → **标签**（常显数值）；或 → **工具提示**（悬停显示多指标）
4. 右键标签 → **旋转标签**（横向更易读）
5. 视图 → **适合宽度**

### 3.2 条形图

- 清空工作表后：省份 → **行**，新增确诊 → **列**，排序
- 同样排除湖北；筛选器可 **应用到此数据源的所有项**，后续同数据源工作表自动排除

### 3.3 饼图

1. 新增确诊 → **大小**，省份 → **颜色**
2. 「标记」卡 → 显示形式选 **饼图**
3. 视图 → **整个视图** 放大
4. Ctrl+拖拽复制字段到标签；数值也可放标签
5. **课后思考**：能否用累计确诊替代新增确诊做占比？（与数据结构有关，需观察数据）

### 3.4 折线图

1. 日期字段需先改为 **日期** 类型
2. 日期 → **列** → 选 **精确日期（天）**；新增确诊 → **行**
3. **课后思考**：如何改为周/月/季度/年粒度？（类比 Excel 日期分组）

### 3.5 散点图

- 新增死亡 → **列**，新增治愈 → **行**
- 省份 → **详细信息**；Ctrl+复制到 **标签**
- **课后思考**：如何让各省份像饼图一样用颜色区分？

## 四、通用技巧

- 右侧默认显示 1000 行，可改大数字查看总行数
- 顶部 **T** 按钮：快速显示/隐藏标签
- 「转置」按钮可自行探索（条形图与柱状图互换）
- 本讲侧重 **快速上手**，拖拽出图原理与四大图表应用场景见后续课程

## 文章正文

课程分两块：P1 讲 Tableau 下载安装、注册试用、密钥与学生账户；P2 用各省疫情 Excel 快速上手连接数据并做**柱状图、条形图、饼图、折线图、散点图**五类基础图表（讲者称可视化「四大天王」加散点等，本讲快速过原理）。

**下载安装**：从视频下方 Tableau 下载链接进版本列表选最新版，滚到页底按 Win/Mac 下载。大陆已不支持 Win32，需台湾站点；老 Win32 可用独立链接。下载慢可复制链接到**迅雷**。安装前在下载页注册：登录→创建账户，填姓名邮箱密码，邮箱点激活链接。安装程序**勾选默认两项、不要自定义安装**（路径问题多；系统盘留 2–3GB）。装完首次打开「立即开始试用」，每台电脑首次可 **14 天试用**；到期需购密钥或学生账户。

**密钥管理**：**帮助→管理产品密钥→激活**；**每个密钥最多两台电脑**，超出可能被追责，多机需先停用旧设备。学生凭在读证明可申请**一年免费**激活码，在读可续；填写信息须与后续学生证/学历证明**完全一致**（姓名、学校、在读期限），等 1–3 工作日，成功邮件发密钥，失败可重新上传清晰证明。

**连接数据**：练习数据来自 Tableau Public 社区（视频链接），示例「全中国各省疫情」Excel。打开 Tableau→连接到文件：`.xlsx/.xls` 用 Microsoft Excel，`.csv` 用文本文件；连服务器支持主流数据库（后续课讲）。单表会自动加载；多表界面需把工作表拖到画布。连接方式：**实时**每次读源；**数据提取**预加载为 `.hyper`，数据量大时更快，一般推荐提取；源变动用「刷新」。保存：**.twbx 含数据**（练习推荐）；**.twb 不含数据**，体积小、保密好，对方需自行加载数据源。界面底部三图标：**工作表**（单图）、**仪表盘**（组合多表，不能在仪表盘里新建图）、**故事**（像 PPT 组合汇报）。右侧默认显示 1000 行，可改大数字看总行数。

**柱状图**：省份→列，新增确诊→行，降序排序。湖北离群值高→右键省份**排除**，或拖省份到筛选器取消勾选湖北。新增确诊→**标签**常显数值；或→**工具提示**悬停看治愈/死亡等多指标。右键标签**旋转标签**横向更易读；视图**适合宽度**。顶部 **T** 按钮快速显隐标签。

**条形图**：清空工作表，省份→**行**，新增确诊→**列**，排序，同样排除湖北。筛选器可设为**应用到此数据源的所有项**，后续同数据源工作表默认排除湖北。课后：点「转置」按钮思考会发生什么。

**饼图**：新增确诊→大小，省份→颜色；标记卡选**饼图**；视图→**整个视图**放大。Ctrl+拖拽可复制字段到标签。课后思考：能否用**累计确诊**替代新增确诊做占比？需观察数据结构。

**折线图**：日期字段改类型为**日期**；日期→列选**精确日期（天）**，新增确诊→行。课后：如何改周/月/季/年粒度？类比 Excel 日期分组自己摸索。

**散点图**：新增死亡→列，新增治愈→行；省份→详细信息，Ctrl+复制到标签。课后：能否像饼图一样用颜色区分各省？
