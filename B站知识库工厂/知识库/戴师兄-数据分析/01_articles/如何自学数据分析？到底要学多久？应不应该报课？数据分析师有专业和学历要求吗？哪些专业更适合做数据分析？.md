## 摘要

自学只需弄清两件事：**学什么**、**怎么学**。本文为框架级重写，工具路径部分：全能分析师「懂数据」含全链路技术认知；入门 3 个月可上岗；工具顺序 Excel → SQL → 可视化 → Python 延后。

## 学什么：五大能力之「懂数据」（工具相关）

1. **数据全链路**：采集 → 呈现 → 图表 → 自动化/算法部署；需有清晰认知（各厂实现不同）
2. **数据结构**：列式标准表；知分析主题 → 知要哪些维度 → 对应表头
3. **数据挖掘**：行业标准指**大数据+算法**场景；很多 JD 写「挖掘」实际做 Excel 分析
4. **数据库**：数据存哪、如何管理自己的代码与下载文件

> 另四能力（懂业务、懂分析、懂沟通、懂行业）见视频其他分P，本重写聚焦工具路径。

## 怎么学：时间与阶段

- **入门约 3 个月**可胜任实习生/专员/数据运营
- 阶段目标：用数据解决简单业务问题 → SQL 取数 → Excel 处理 → 图表/BI

## 文章正文

戴师兄在视频里把自学数据分析拆成两件事：**学什么**、**怎么学**。全能分析师的「懂数据」能力，需要建立数据全链路认知——采集、呈现、图表、自动化/算法部署，各厂实现不同但框架要清晰。还要理解数据结构：列式标准表；知道分析主题 → 知道要哪些维度 → 对应表头。行业里「数据挖掘」常指大数据+算法场景，但很多 JD 写「挖掘」实际做的是 Excel 分析。数据库则回答数据存哪、如何管理自己的代码与下载文件。另四能力（懂业务、懂分析、懂沟通、懂行业）见视频其他分 P。

**入门约 3 个月**可胜任实习生、专员或数据运营——阶段目标是用数据解决简单业务问题：SQL 取数 → Excel 处理 → 图表/BI 呈现。

### 工具学习路径

| 阶段 | 内容 | 资源要点 |
| --- | --- | --- |
| 分析方法 | 解决简单业务问题 | 《深入浅出数据分析》 |
| 取数 | SQL 七语句 + 连接/子查询 | 未明学院 SQL 教程 + SQLZOO |
| 补充取数 | **后羿采集器**（无需编程） | 戴师兄爬虫/采集视频 |
| 处理 | Excel 函数 + **透视表** | 泽秀老师课程 |
| 可视化 | Excel 图表 + 任一 BI 基础 | Tableau/Power BI/FineBI；**面试前突击公司所用工具** |
| Python | 工作一段时间后按需 | 百万级/特殊处理再学 |

顺序上：Excel → SQL → 可视化 → Python 延后。可视化工具不必全学——**面试前突击目标公司用的那一款**即可。Python 等工作一段时间、遇到百万级数据或特殊处理需求再学，入门阶段不必贪多。

视频还讨论了应不应该报课、数据分析师有没有专业和学历要求、统计/计算机/商科等哪些专业更适合——这些与工具路径无强绑定，需结合个人基础与目标岗位判断：有自律和免费资源检索能力可自学；需要结构化督促和项目反馈时再考虑报课。专业背景影响入门速度，但不决定上限——关键是按路径做出作品、用实习和面试反馈验证学习成效。
