# 【2024最新版】Excel 全面升级（上）：报表复现

**摘要**：本集通过复现「劳戴」制作的销售汇总报表，系统讲解 Excel 数据分析基础——认识原数据、窗口分屏、MAX/SUMIF/SUMIFS 条件汇总、环比计算、IFERROR 容错、引用方式（相对/绝对/混合引用）及条件格式美化。

## 一、认识原数据

示例数据共 15 列、1657 行（含 1 行表头），核心字段包括：

| 字段 | 说明 |
| --- | --- |
| 成交日期 | 2020 年 4–5 月数据 |
| 销售工号 | 销售唯一标识（真实场景还有姓名等敏感信息，课程已脱敏） |
| 产品类型 | 现金贷/消费贷等借贷产品 |
| 期数 | 还款期数，期数越长利息通常越高 |
| 成交客户数 / 成交金额 | 核心度量 |
| 区域 / 省份 / 小组 / 业务组 | 销售组织架构 |
| 客单价 | 成交金额 ÷ 成交客户数 |
| 成交年份 / 成交月份 | 用 YEAR、MONTH 从成交日期提取 |

**分析师默认操作**：选中任意列，`Ctrl+Shift+L` 开启筛选；状态栏可查看选中列的聚合统计（求和、计数等）。

## 二、报表复现准备工作

1. 新建工作表「报表复现」。
2. **视图 → 新建窗口**，拖拽实现分屏：左边对照原报表，右边动手复现。
3. 单元格用字母列 + 数字行定位，区域用 `A2:H3` 表示左上角到右下角。

## 三、标题与筛选器

- 合并 A1:B1，输入「最新数据时间」，右对齐。
- C1 用 `=MAX(A:A)` 返回成交日期最大值（报表处理的最后一天）。
- **日期本质**：Excel 中日期是特殊数字，43982 对应 2020-05-31；显示为日期需设置单元格格式为「短日期」。
- G4 输入「月份筛选」，H4 用**数据验证 → 序列**，来源填 `4,5`（英文逗号分隔）。
- 标题 A2:H3 合并，公式 `=H4&"月销售数据汇总"`，实现随筛选器联动的标题。

## 四、核心指标：SUMIF 条件汇总

月累计成交金额（B6）：

```
=SUMIF(成交月份列, H4, 成交金额列)
```

含义：找出成交月份等于筛选器月份的行，对这些行的成交金额求和。

成交客户数（D6）同理，只改求和列。复制公式时注意**引用偏移问题**——判断区域和条件单元格应加 `$` 绝对引用（F4 快捷键），求和列随公式位置变化。

客单价（F6）：`=B6/D6`

## 五、环比计算

**公式**：`(本期 - 上期) / 上期`，化简为 `本期/上期 - 1`

- 5 月环比的分母是 4 月，因此复制 SUMIF 公式后将月份条件 `-1` 得到上期值。
- **数据前提**：算 5 月环比时，原数据必须存在 4 月；只有 4–5 月数据时，4 月环比需要 3 月会报错。
- **IFERROR 容错**：`=IFERROR(环比公式, "-")` 报错时显示横杠。

同比只需把「上期」换成「去年同期」（如 20 年 5 月 ÷ 19 年 5 月 - 1）。

## 六、多维度明细：SUMIFS

省份成交情况需同时判断**月份 + 区域 + 省份**三个条件，单条件 SUMIF 不够，改用 SUMIFS：

```
=SUMIFS(求和列, 条件区域1, 条件1, 条件区域2, 条件2, 条件3区域, 条件3)
```

**引用方式设置**（向下/向右填充时）：

| 引用对象 | 引用方式 |
| --- | --- |
| 求和列、条件区域 | 绝对引用 `$A:$A` |
| 筛选器 H4 | 绝对引用 `$H$4` |
| 区域/省份判断条件（A11、B11） | 混合引用 `$A11`（锁列不锁行） |

## 七、产品成交与占比

- 期数维度用 SUMIFS，条件为期数 + 月份。
- 占比 = 每期成交额 ÷ 总成交额（分母绝对引用）。
- 总计行用 SUM 或 `Alt+=` 快速求和。

## 八、格式美化要点

- 数值：千分位、0 位小数；环比/占比：百分比格式。
- 条件格式：环比箭头（>0 绿色↑，<0 红色↓）；成交金额数据条；占比色阶。
- **自定义格式 trick**：用 `[Green]0.0%↑;[Red]0.0%↓` 在显示层加符号，不改变原值。
- 加人民币符号应通过单元格格式，而非在数值前输入 `¥`（会变成文本无法计算）。
- 视图取消网格线，加边框分区。

## 九、关键习惯

- 所有括号、逗号用**英文输入法**。
- 函数嵌套多层时，保持清晰的制作逻辑——复杂函数是一层层简单组合。
- 做环比前先确认分母月份数据是否存在。

## 文章正文

本集通过复现同事「劳戴」做的销售汇总报表，把 Excel 做业务报表的核心操作串起来——不是孤立记函数，而是**从认识原数据到交差一份可联动筛选的月报**。

**认识原数据**：示例 15 列、1657 行（减表头 1657 行数据）。字段包括成交日期（2020 年 4–5 月）、销售工号、产品类型（现金贷/消费贷及期数）、成交客户数、成交金额、区域/省份/小组/业务组、职务类别（管理层/执行）、客单价（金额÷客户数）、成交年/月（YEAR/MONTH 提取）。敏感信息已脱敏。分析师拿到表的默认动作：选中列看状态栏聚合，`Ctrl+Shift+L` 开筛选；可排序看成交极值（最低 15 元、最高 96000 元）。

**复现准备**：新建「报表复现」表；**视图 → 新建窗口**分屏——左看原报表，右动手做；区域用 `A2:H3` 这类左上角:右下角写法。

**标题与筛选器**：A1:B1 合并写「最新数据时间」，C1 用 `=MAX(A:A)` 取成交日期最大值——Excel 里日期本质是数字，43982 即 2020-05-31，设「短日期」格式才像日期。G4「月份筛选」，H4 **数据验证→序列**，来源 `4,5`（必须英文逗号）。标题 A2:H3 合并，公式 `=H4&"月销售数据汇总"`，随筛选器联动。

**核心指标 SUMIF**：B6 月累计成交金额 `=SUMIF(成交月份列, H4, 成交金额列)`——找出月份等于筛选器的行再求和。D6 成交客户数同理。从 B6 复制到 D6 时引用会偏移，因默认相对引用；判断区和 H4 应 **F4 锁绝对引用**（`$`），求和列随位置变。F6 客单价 `=B6/D6`。

**环比**：公式 `(本期-上期)/上期` 化简为 `本期/上期-1`。5 月环比分母是 4 月——复制 SUMIF 后把月份条件 **减 1** 得上期。同比把「上期」换「去年同期」即可。课堂练习：观众自己算成交客户数环比。客单价环比：4 月客单价 = 4 月金额÷4 月客户数（前面已算），再与 5 月客单价组合。筛到 4 月时报 `#DIV/0!`——**原数据只有 4–5 月，4 月环比需要 3 月**，没有分母就报错；外层套 `=IFERROR(...,"-")` 显示横杠。讲者强调：函数嵌套多层不可怕，**清晰制作逻辑 + 一层层组合**即可攻破。

**明细 SUMIFS**：省份成交需同时判月份+区域+省份，单条件 SUMIF 不够。`SUMIFS(求和列, 条件区1, 条件1, …)`——熟练工常用 SUMIFS 替代 SUMIF 方便加条件。填充时：求和列、条件区域、H4 **绝对引用**；A11/B11 区域省份用 **混合引用 `$A11`**（锁列不锁行）。向下向右填充后改环比（月份减 1）、改求和列算客户数；客单价注意分母零值也 IFERROR。总计行 SUM 或 `Alt+=`；总计环比复制核心指标思路。
