# 今年数分面试难度如何？面经详解

**摘要**：字节校招数分题对社招是「零分难度」、对校招约 50～60 分。大厂社招持续招高质量复合型人才，校招本质不招零经验——倾向实习转正。清北也会被挂，实习和专业远比学历重要。以下以字节校招面经逐题拆解准备要点。

## 一、当前招聘行情

### 1.1 大厂还招人吗？

- **社招**：一直在招，但要求很高，招的是高质量人才，不再补执行层
- **校招**：本质上**不招零经验应届生**，倾向**实习直接转正**，没有多余岗位花一年培养

### 1.2 两大趋势：职能复合 + 自负盈亏

- **职能高度复合**：分析师不止写报告，还要做数据建设、仪表盘、自动化报表、指标体系、帮业务出谋划策
- **自负盈亏**：运营/分析师需背营收指标，未达标可能被裁（PMI 机制）

纯做报表的岗位已是「数据专员」级别，不是分析师。分析师要从前负责到后，一开始就要把指标体系梳理清楚。

### 1.3 校招的本质

校招不是为应届生设计，是为**有能力的人**设计——从毕业生中提前摘出优质苗子。走到哪，校招还是社招，**核心都是能力**。

## 二、校招筛选标准

### 2.1 学历 vs 实习 vs 专业 vs 作品

| 因素 | 重要性 |
| --- | --- |
| 学历 | 达标即可，清北也会被挂；名校扩招后不再是核心竞争力 |
| 实习 | **远比学校重要**——两份大厂实习 vs 清北零实习，选后者 |
| 专业 | 理工科有优势，但**专业对口 ≠ 有能力** |
| 个人作品 | **最核心**——能独立建立指标体系、做出分析报告即证明能力 |

### 2.2 校招最低实习配置

| 目标 | 最低配置 |
| --- | --- |
| **大厂** | 至少 2 份数分实习 + 其中 1 份大厂任意岗位背书 |
| **中场** | 至少 1 份数分实习 + 证明工具技能和分析思路 |
| **小厂** | 1 份相关行业实习 + 成熟工具技能 |

**零实习找校招非常难。** 相关实习 = 行业或岗位至少有一个相关（游戏运营实习去面新能源数分不搭）。

大厂面试弱考察工具（默认笔试+实习已证明），重分析思维和软实力；中场可能继续追问工具。

## 三、字节校招面经要点

### 3.1 一面：简历深挖 + 基础题

**简历深挖**：
- 至少 2 份实习才能进校招面试
- 简历每句话都要能讲清背景、做法、产出
- **禁止吹牛逼**：实习生写「优化效能 50%」不成立；写项目整体提升 5%～10%、自己是其中一份子可以
- 简历写不出 = 经历太少或工具没学明白

**经典题型**：

| 题型 | 要点 |
| --- | --- |
| DAU 某天下降如何找原因 | 业务诊断基本功；**向前看、向下钻** |
| 关键指标选取 | 先映射业务流程量化指标，再按目标（增长/复活/召回）筛选 |
| 功能点优缺点 | MECE 拆解；从**价值**和**时间**两维度分析 |
| AB 测试流程 | 选版本 → 选评估指标 → 回收结果；流量够大时假设检验非必须 |
| 最不能接受的事 | **服从性测试**——只答「因自己疏忽没做好」；拿到 offer 前穿袜子后穿鞋 |
| 反问环节 | 前两面问「我还需准备什么/一面有什么需提升」；最后一面问行业 3～5 年挑战、如何快速成长 |

**向前看、向下钻**（业务诊断）：
- **向前看**：业务流程往前倒——注册、登录、转化各环节是否同步下滑
- **向下钻**：DAU 下滑是哪个维度/人群在下滑

### 3.2 二面：项目深挖 + 产品洞察

- 准备一个充分的项目/分析报告介绍——**作品到位则无需额外准备**
- **致命题**：平时用哪个产品？只刷单一 App 会被淘汰；分析师需体验知乎、微博、小红书、B 站、抖音等
- 可能被要求展示手机 App 列表——可专门建「互联网流量观察」文件夹
- 产品优化题需明确目标（用户量/商业化），回到**营收**一般不会错

### 3.3 三面：总监面商业洞察

- 谈抖音电商竞争力、与快手/淘宝直播区别——靠平时积累，现编很难
- 关键指标选取反复出现——方法论同上
- 优缺点问法：答「人没有优缺点只有特点，看公司需要什么，我的特点是 1/2/3/4/5」

## 四、通用建议

1. **求职是长期积累**——格局修养比表面工具技能更重要
2. 分析师要**客观理性、量化**，这是基本品格
3. 拿到 offer 前服从性测试要通过；拿到 offer 后才有谈判筹码
4. 多看面经、多积累产品观察和商业洞察

## 五、其他 Q&A（直播片段）

- 大健康行业：学会业务+数分技能，针对性投递，拿到 offer 再说
- 字节本地生活：卷，无相关经验去是「送」；建议先去美团学经验再跳字节变现
- 能学到东西的岗位往往薪资较低；高薪岗位是对技能变现，不是来学习的
- 永远不要主动提加薪——让领导主动给，或直接跳槽
