> **摘要**：梳理 21 款数据分析工具优先级。**BI 板块核心变化：Power BI 升为首选，Tableau 仍值得学但非首选；FineBI 等国内 BI 有公司再用时再学。**

## BI 工具（可视化/看板）

### Power BI — 2023 首选

- 微软 Excel 生态演进的 BI：数据处理、探索、可视化、看板、在线监控一体。
- **几百元** 即可本地制作 + 共享发布，云服务免维护；个人/小团队 **性价比最高**。
- 与 ChatGPT 同属微软生态；**Fabric** 将统一大数据/数仓/分析，Power BI 为分析界面——跟进 Power BI 可享 AIGC 技术红利。

### Tableau — 仍值得学，但非首选

- 最适合新手入门的可视化工具之一，无设计基础也能做出美观图表。
- 中国市场政策导致购买/服务体验被 Power BI 等拉开；国内使用 Tableau 的公司可能减少。
- **可视化能力与操作逻辑仍值得专业分析师学习**；学会后迁移其他 BI 很简单。

### 国内 BI（FineBI / Quick BI / 永洪等）

- 多为 Power BI + Tableau 的「缝合版」，**不推荐专门花时间学**。
- **公司用了再研究**；有 Power BI/Tableau 基础，几小时可上手。

### Tableau Prep Builder — 必学自动化

- 图形化流程化数据处理；本地/云端数据整合、上传数据库与 BI。
- 拖拽步骤即可；可定时自动跑，上班前挂任务——**效率飞升工具，必学**。

## Python 可视化相关

- 分析师 Python 必学 **Pandas** + 成型项目；**PyGWalker** 必掌握——Python 内 Tableau 式拖拽，出图效率高。

## 其他板块（框架）

| 领域 | 推荐 |
| --- | --- |
| 自动化 | 影刀 RPA |
| 办公 | Office 365（Excel/PQ/Pivot）、飞书云文档 |
| 数据库 | 本地 MySQL + DataGrip；SQL 刷题 |
| Python | Jupyter Notebook；Pandas 必学；爬虫不推荐深钻，用后羿/影刀 |
| 算法 | ChatGPT 应用；校招可学 sklearn/xgboost |

## 文章正文

距离上一次数据分析工具推荐已过去一年多，随着生产力提升和 AIGC 发展，2023 年分析师该学什么、按什么顺序学，戴师兄梳理了 21 款专业工具，按必学推荐和需要时再补充分了优先级，并给出下载地址和使用教程，覆盖从入门到商业实战的全阶段需求。

**BI 板块变化最大，Power BI 升为首选。** 它是微软生态下由 Excel 演变而来的商业智能工具，集成数据处理、探索、分析、可视化、看板搭建和在线监控——仅用这一款就能覆盖大部分分析和应用需求。价格美丽，几百元即可本地制作看板并共享发布，微软级云服务免维护，是个人和小团队性价比最高的 BI 平台。更关键的是，Power BI 与 ChatGPT 同属微软生态，正在享受 AIGC 技术迭代；最新框架 Fabric 在内置 AI 加持下，将统一大数据、数据库、数据科学、数仓、分析、监控等资源，Power BI 作为分析界面调用 Fabric 内数据——「认真学好 Power BI、跟进更新，未来就能享受微软基于 AIGC 的技术红利」。

**Tableau 仍值得学，但已非首选。** 它依旧是最适合新手入门的可视化工具之一，没有设计基础也能做出配色舒适、布局合理的图表。但中国市场政策导致购买、使用、服务体验持续被 Power BI 和国内厂商拉开差距，国内使用 Tableau 的公司可能减少，分析师不得不学新 BI 替代。Tableau 强大的可视化能力和操作逻辑仍值得专业分析师研究；学会后迁移其他 BI 很简单，但 Power BI 才是更面向未来的选择。FineBI、Quick BI、永洪等国内外 BI 多是 Power BI 和 Tableau 的「缝合版」，**不推荐专门花时间学**——公司用了再研究即可，有 Power BI/Tableau 基础，几小时内就能上手。

**Tableau Prep Builder 是必学自动化工具。** 业内公认、用过都说好的图形化流程化数据处理工具，可独立实现本地表格、混合云端数据、上传数据库和 BI 平台；操作只需选择步骤拖拽点击，学习成本极低，可一键运行、配置脚本后定时自动处理。很多小伙伴学会后上班前就让 Prep 自动执行，完成数据准备和上传，与各种「工具人操作」彻底说拜拜——「真正意义上的效率飞升工具，必学必学必学」。

**影刀 RPA 是另一款解放双手的工具。** 所有电脑上需要反复执行的操作都可以制成流程并定时运行：每天从后台下载表格、复制粘贴图片到群聊、新建投放计划……影刀都能自动完成。掌握它，非分析工作上的重复机械操作大幅节省，也是上班提效的必备工具；市面上已出现大量 RPA 工程师岗位，学影刀还能拓展职业路径。

**办公工具：Office 365 + 飞书云文档。** 分析师是企业里最需要对信息整合、输出的岗位，Office 365 正迭代为 Microsoft 365，Copilot 等 AIGC 应用将全面实装，基于云算力和在线搜索的办公软件才是主流。Excel 基础功能（函数、透视表、图表）仍是学习成本最低、覆盖最广的基础；Power Query 和 Power Pivot 能在 Excel 里实现高级 BI 功能，方便直接分享文件、降低协作成本——但对 Excel 数据类型和 SQL Server 以外数据库支持较差，复杂转换 Prep 更合适；在 ChatGPT 加持下，微软系工具「加满准没错」。飞书云文档支持封面、标题高亮、插入图片流程图、协作编辑和多维表格，美观实用，学习阶段熟悉它，工作后快速上手钉钉/企业微信类场景，性价比很高。钉钉等同类工具工作时公司有要求再迁移即可。

**数据库与 SQL：本地 MySQL + DataGrip。** 2023 年学习路径已非常明确：数据库无脑选本地 MySQL，语言只学 MySQL 查询语句并不断刷题；操作工具必用 DataGrip，界面专业，学生可免费试用。有时间可补充云端数据库搭建——云服务才是实战主场景，面试入职能加分；增删改语句虽分析师日常少写，但笔试常考，有备无患。Hive SQL、大数据框架、Navicat 等，入职后各公司自有工具，到时再补；想拓宽职业选择，可提前学数据工程知识，很多中小厂对数据建设和开发能力有要求。

**Python：Jupyter + Pandas + 成型项目 + PyGWalker。** 编程环境初学只需 Jupyter Notebook 打草稿、输出分析报告；若不从事 Python 开发，不必学 PyCharm。分析师日常只有复杂或海量数据才需 Python，**Pandas 是唯一直接必学的工具包**，覆盖 90% 以上处理需求；但更重要的是做出**成型的数据处理与分析项目**——改小游戏、做简单报告都是闹着玩，面试官看重的是处理复杂数据的系统能力。基础语法有时间要多练；**PyGWalker 必掌握**，在 Python 里实现 Tableau 式拖拽出图，效率很高。其余知识点按求职方向或工作需要再补。爬虫是独立专业领域，与数据分析关系不大；必学工具是后羿采集器（智能识别网页字段），影刀也能爬数据且能看到每步操作，适合理解流程。若钻研 Python 爬虫可学 requests、scrapy、selenium，但戴师兄劝退：「投入产出比远没有分析领域高，学得越好越在法律边缘试探——收手吧，阿伟。」

**算法建模：跟进应用，校招可学 sklearn/xgboost。** 当前技术开发红利窗口对普通人已不大——算法团队最低学历要求常在 985 本硕加海外名校，一线拿高薪的多是博士和教授。普通人性价比最高的方式是跟进 ChatGPT 等成型算法工具的应用，让商业场景分析始终领先。还没用上 ChatGPT 要好好反思——未知领域信息获取、分析拆解、学习答疑效率碾压其他渠道。校招或走数据科学方向，sklearn 和 xgboost 是机器学习领域使用频率最高的库，笔试面试准备基本足够；深度学习属算法工程方向，有觉悟卷学历再了解即可。

转眼已覆盖商业智能、自动化、办公、数据库、Python、爬虫、算法建模七个领域、20 余款工具——按优先级学，需要时再补充，祝学习顺利、早日找到满意工作。
