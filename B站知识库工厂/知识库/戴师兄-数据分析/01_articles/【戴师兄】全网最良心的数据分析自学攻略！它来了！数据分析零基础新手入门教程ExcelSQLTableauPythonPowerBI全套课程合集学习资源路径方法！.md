## 文章正文

戴师兄最新版数据分析自学攻略在一个网页上就能用最快的速度掌握全部内容——视频可直接观看，电子书可直接下载，全是 B 站优质资源，且已编排好最科学的学习顺序。这份反复修改 50 多次、帮助过不下百位小伙伴入职大厂的攻略，符合目前一线数据分析师的工作技能标准。师兄从零基础运营转数分，结合近几年大厂经历，从分析方法、Excel、BI、SQL、Python 五个部分全方位梳理自学路径，每个部分都有清晰的学习目标和成长路径。

**分析方法分四个阶段。** 第一阶段是**建立认知**：数据分析为什么重要、背后原理是什么、主要流程有哪些、如何在企业创造实际价值——搞清楚这些，初期就搭建完善知识框架，后续查漏补缺、逐个击破，效率会高很多。戴师兄制作了两期视频：第一讲讲数据如何从埋点获取到算法建模、一步步创造业务价值；第二讲举例解答数据分析方方面面问题，一直讲到职业发展，知识点来自从业踩坑的血泪总结。第二阶段是**用数据分析解决问题**，从简单的开始，推荐《深入浅出数据分析》——所有问题从实际业务场景引入，讲如何在沟通中提取信息、拆解问题，有真实数据的分析过程和结论参考，还有练习题；覆盖销售、用户绩效等真实场景，专业方法一个不差，「比付费课程还良心」，入门阶段融会贯通这一本完全够用。第三阶段是**广泛涉猎**：坚持在 36 氪、人人都是产品经理、简书阅读方法和案例，每天至少一篇，好的文章总结做笔记——一方面见世面、知道主流商业模式和分析角色，不跟互联网场景脱轨；另一方面广泛积累分析经验，很多面试题和业务问题早就有人给过解决方案；观点碰撞中自己的分析思路也会越来越清晰。戴师兄转行那大半年每天通勤都在看书和文章跟着思考，「能转行成功绝对离不开这一阶段的积累」。第四阶段是**产出报告**：推荐《精益数据分析》，30 多个真实数据应用案例，硅谷创业者人手一本的经典；配合知乎系统攻略，做出深度行业或产品分析报告，极大提升求职成功率。

**工具学习，第一个且只能是 Excel。** 它在数据处理上最灵活、功能最丰富、掌握人数最多，能轻松打基础，解决日常大多数低量级数据问题。学习时要快速掌握核心功能、产出个人作品——Excel 功能太碎，不以明确结果为目标会很没逻辑。戴师兄制作了 Excel 教程，学完就能做联动筛选、智能填充字段的仪表盘，几乎覆盖所有核心功能；实战中查漏补缺，不会的就搜，官网也有最专业解答，不断积累函数并定期总结，也能成为 Excel 大神。

**BI 工具：Tableau 和 Power BI 都要会。** Excel 超过 100 万行无法读取，10 万行以上都可能卡死；Tableau 和 Power BI 能轻松处理千万级甚至亿级数据。Excel 在不掌握 VBA 时操作不能反复执行、自动化程度低；BI 可反复执行，还能发布到服务器自动从数据库取数，「啥都不用干每天看最新看板」。其他 BI 多是换皮版，公司用哪个学哪个，结果两个都要会。零基础、Excel 也在初学阶段，推荐先学 **Tableau**——完全基于可视化原理设计，不仅掌握工具，更理解可视化底层逻辑，对以后学任何可视化工具都有帮助；学戴师兄自制 Tableau 教程即可，补充可看未明学院、优达、Tableau 官方社区大神作品。若已是 Excel 大神，推荐直接学 **Power BI**——同为微软工具，可视化和操作逻辑相通，上手更快；资源推荐孙兴华 ZZ 大神全套教程，学完全面掌握，不必再买付费课。

**SQL 是分析师必须掌握甚至精通的技能。** 所有分析岗位笔试面试都会考察，也是日常从数据库取数的主要工具。戴师兄 SQL 教程迭代五次，以笔试和实战程序设计深入讲解底层原理，知识点、习题、笔记一应俱全，「只学这一个视频就能入门 SQL」；之后到牛客网把 SQL 真题刷一遍，「基本上就出师了」。需系统掌握可补充菜鸟教程；偏数据工程可看黎曼猜想 MySQL 教程（近 500 万播放）；Hive SQL 是 SQL 变形，尚硅谷 Hive 教程补充即可，岗位真有要求再学。

**Python 一定是最后学的。** 一般业务规模下 Excel 和 BI 完全能处理，轮不到 Python；现在还在推荐「纯 Python 数据分析」的，「不是菜就是坏，纯属误人子弟」。Python 初期学习成本巨高，没明确目的大概率学不会还劝退。一线分析师 Python 场景主要是：公司用不了 BI 只能 Python 处理海量数据、个别 BI 无法实现的处理、建模分析——前两种在数据建设完善的公司很少遇到；Python 最大收获其实在简历上，「你会他不会你就有优势」，但是加分项不是必选项，很多小伙伴不会 Python 一样找到大厂工作。系统学的话：数据分析推荐孙兴华 Python 数据三部曲（基础语法、numpy、pandas、matplotlib）；机器学习推荐菜菜的 sklearn 教程；实战看上硅谷机器学习推荐系统项目；算法竞赛可去 Kaggle 和天池，名次能写简历；和鲸社区有优质项目可链接分享。R 语言、SPSS 等主要用于科研或传统分析，互联网公司基本不用，不必掌握。

**数学和统计学：校招建议系统学，社招不必专门花 time。** 校招推荐 3Blue1Brown 高数原理和宋浩概率论与数理统计；社招大多数数学统计实践已封装在工具软件里，不必手推，现学现卖、百度都能学会，「学习数据分析的目的是解决问题，不是证明自己掌握了某一个知识点——统计学不会成为绊脚石」。

攻略完成之际，戴师兄见证了很多小伙伴转行成功、薪资改善——数据分析是值得掌握的工具技能，欢迎专注提高工作效率、用数据力量在工作岗位上做出独特贡献。
