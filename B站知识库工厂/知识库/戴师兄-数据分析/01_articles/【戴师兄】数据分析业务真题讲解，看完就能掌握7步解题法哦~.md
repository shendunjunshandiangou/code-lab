# 网易数据分析笔试真题与七步解题法

**摘要**：以网易云音乐付费会员活动题（买一送一 vs 五折、联合会员成本收益评估）为例，讲解开放型业务笔试的**七步解题法**。核心原则：答案不重要，解题思路才重要；笔试不重要，分析思维才重要。

## 一、三种错误反应（必凉）

1. 一脸懵逼，不知从何答起
2. 知道套路但缺乏业务认知，信息不全瞎写
3. 想到什么写什么，不系统

## 二、七步解题法（七步成诗）

| 步骤 | 动作 |
| --- | --- |
| **1 提取关键信息** | 圈出题干重点，整理成结构化信息；理清业务机制、业务对象、分析目标 |
| **2 信息结构化** | 划分问题范围，有的放矢 |
| **3 列出关键指标** | 量化 > 抽象；长期 > 短期（短期过得去时优先长期）；短期长期同时考虑 |
| **4 量化数学关系** | 把指标比较转化为公式（> < =）；「只有数学关系才能更好地判断好坏」 |
| **5 找影响因素** | 谁在影响比较指标？便于控制变量、分类讨论 |
| **6 总结规律** | 把指标、影响关系归纳为几条规律/定律（如长尾效应、新用户与折扣关系） |
| **7 结合实践分类作答** | 考虑 A/B 测试等实验手段；按影响因素分类讨论；**结论必须有量化指标** |

## 三、案例要点（会员活动题）

### 业务机制

通过丰富权益吸引付费会员 → 提升粘性与忠诚度（长期目标）；开卡活动是短期手段。

### 指标选择原则

- 用**活跃/互动/付费**等量化指标近似「忠诚度/粘性」
- 短期看开卡人数、收入；长期看核心指标增长率
- 长期观测至少 **1～3 个月**，年卡活动 ideally 等到**活动彻底结束**再复盘

### 数学关系示例

- 两方案 ROI 相同（买一年送一年 ≈ 半折买半年送半年）
- 比较开卡收入本质可能在比开卡人数（单价有倍率关系）

### 实践考量

- 活动前能否做 **A/B 测试** 或产品测试
- 用抽样数据 + 推断统计代表整体

## 四、分析师立场

- 业务只要开卡量 → 你会被牵着走；展现**长短期兼顾**思路才是好分析师
- 经验靠时间积累，**思路今天学完就能用**
- 面试/笔试失败没关系，能力有增长即可

## 五、课后练习

第二题（联合会员成本收益划分）与加问留作自练，思路与第一题类似但更复杂。

## 文章正文

本次直播以网易数据分析笔试真题为例，讲解开放型业务题的**七步解题法**。戴师兄强调：**答案不重要，解题思路才重要；笔试不重要，分析思维才重要**——经验靠时间积累，思路今天学完就能用。

**题目背景：** 各大平台推出付费会员策略，网易云音乐通过丰富权益吸引用户成为付费会员，提升粘性和忠诚度。第一问：开卡活动选「买一送一」还是「五折」？第二问：网易云音乐与网易严选联合会员（买音乐会员送一年严选 Pro），分析师如何评估两个会员的成本收益，给出具体评估指标？

**三种必凉反应：** 读完一脸懵逼、不知从哪答；大概知道套路但生活经验不足（没用过严选、概念不清）只能瞎写；不管了想到什么写什么——这三种下做出的答案都不系统全面。

**七步解题法（七步成诗）：**

**第一步，提取关键信息。** 像高中应用题一样，先明白题目到底要干什么。第一题关键信息：通过丰富权益吸引付费会员；对象网易云音乐；要开卡；两种方式（买一送一 vs 五折）；哪个更好并给出理由。第二题：联合会员的成本收益如何划分、用什么指标评估。

**第二步，信息结构化。** 把圈出的信息整理成结构化框架，划分问题范围，有的放矢。

**第三步，列出关键指标。** 原则：**量化优于抽象，长期优于短期**（短期能过得去时优先长期；短期长期需同时考虑）。敲黑板必须记住：**「只有量化的指标才能更好地反映现实事实」**——看似分析事实，实际分析最能表现事实的数据（即指标）。用分析矩阵把指标分成量化/抽象 × 短期/长期四类。短期量化：开卡人数、开卡收入。短期抽象：用户好感度提升（难量化）。长期量化：日活月活、核心行为（听歌频率、评论点赞等）、付费指标（会员之外是否继续购买专辑等）。长期抽象：用户忠诚度/粘性——产品和运营常提，但音乐 App 版权竞争激烈，「真有宁愿不听这首歌只用一个工具的用户吗」——需用可量化指标近似。

**第四步，量化数学关系。** 把指标比较转化为公式（>、<、=）。例如两方案 ROI：买一送一 ≈ 直接五折买半年送半年，收入可能相同；比较开卡收入本质可能在比开卡人数（单价有倍率关系）。「只有数学关系才能更好地判断好坏。」

**第五步，找影响因素。** 谁在影响比较指标？便于控制变量、分类讨论。例如用户类型（新用户 vs 老用户）、活动周期、竞品动作等。

**第六步，总结规律。** 把指标和影响关系归纳为几条规律/定律，如长尾效应、新用户对折扣更敏感等。

**第七步，结合实践分类作答。** 考虑 A/B 测试或产品测试；按影响因素分类讨论；**结论必须有量化指标**。

**常见四大错误（戴师兄批卷经验）：** 只讨论不总结——分类讨论好坏却不给「当 A 情况则…当 B 情况则…」的明确结论；思考不全面——只列两个方案的长处、不比短处；结论不量化——全文没有一个数字；指标不定义——说「用户数」「核心指标」但没有口径（已是较高水平才会犯的「幸福烦恼」）。

**分析师立场：** 业务只要开卡量，你若被牵着走就输了；展现长短期兼顾的思路才是好分析师。第一题长期观测至少 **1～3 个月**，年卡活动 ideally **等活动彻底结束**再复盘——用活跃/互动/付费等量化指标近似「忠诚度/粘性」。活动前能否做 A/B 测试；用抽样数据 + 推断统计代表整体。面试/笔试失败没关系，能力有增长即可。第二题联合会员成本收益划分留作自练，思路与第一题类似但更复杂。
