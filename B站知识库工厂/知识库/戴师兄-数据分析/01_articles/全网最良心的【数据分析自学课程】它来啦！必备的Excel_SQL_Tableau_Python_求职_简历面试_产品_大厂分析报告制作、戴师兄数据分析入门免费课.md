这是戴师兄**免费入门课合集**的导航性长视频，数十小时、二十多个分 P，把认知入门、Excel、SQL、Tableau/BI、Python、大厂分析报告制作、求职简历面试等模块串成一条可跟练的路径。配合「全网最良心的数据分析自学攻略」收藏页按顺序学习效果最好。

**课程开篇：数据分析不是什么花哨东西。** 戴师兄用「观测·实验·应用」定义数据分析——客观认知并改造世界。街角早餐店老板凭经验判断「今天多备点油条」「雨天少进豆浆」，本质上已在用数据做决策，只是**隐性、非系统化**；当摊位变成连锁、SKU 上百、多城市开店，靠脑子记不住，才需要 Excel 记台账、SQL 取数、BI 看板、Python 处理海量日志。这个直觉很重要：**数据分析不是只看板/算法/炫技，而是把业务问题翻译成可观测、可实验、可应用的指标与结论。**

**合集模块结构（按工具路径）：**

**认知入门**分 P 建立「数据分析是什么、在企业如何创造价值」——与自学攻略里「建立认知」阶段呼应，建议先看完再进工具。

**Excel 模块**从函数、透视表、图表到 Power Query 等核心能力，跟练即可产出可展示的小作品（如销售报表、联动筛选小仪表盘）。Excel 是后续一切工具的「手感基础」——不懂表格行列逻辑，SQL 的 FROM/SELECT 也难建立直觉。

**SQL 模块**配合戴师兄 SQL 入门免费课（分 P 刷题），从 SELECT/FROM/WHERE 到 JOIN、GROUP BY、窗口函数，以笔试和实战取数为导向；学完需到牛客/SQLZOO 刷真题。

**Tableau / BI 模块**含光速入门、动态图表、动态仪表盘等长课；零基础优先 Tableau 理解可视化原理，Excel 大神可直接 Power BI。操作步骤、数据集、安装激活细节在各分 P 演示，本文字不穷举点击路径。

**Python 模块**定位在 Excel/SQL/BI 之后：Jupyter + Pandas 处理复杂或海量数据，PyGWalker 等拖拽出图；强调**成型项目**而非语法炫技。

**分析报告模块**讲大厂报告制作方法论——如何从业务问题到指标框架、图表叙事、结论建议，与求职阶段「竞品/行业深度报告」产出目标一致。

**求职模块**覆盖简历四步法、面试模拟、作品呈现，可与《系统求职课》《简历制作攻略》交叉使用。
