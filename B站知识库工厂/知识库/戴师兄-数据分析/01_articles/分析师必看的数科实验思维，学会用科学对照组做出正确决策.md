# 数科实验思维：对照组与正确决策

**摘要**：嘉宾瑞老师分享——中国「数据科学」岗位≈数据分析师，核心是**更科学地利用数据做决策**。以天降红包营销活动为例，说明无对照组则效果几乎全是假的；A/B/C 测试本质是为回答「若没做策略，平行世界会怎样」。

## 一、数据科学在中国

- 算法工作多交给算法专家；留给 DS 的工作≈**DA（数据分析师）**
- 定义：**利用数据更科学地做决策**，帮 stakeholder 做 solid 决策

## 二、核心问题：没有对照组，效果几乎都是假的

### 场景：天降红包营销活动

老板问：策略有没有效？下次要不要大力推广？

### 错误分析（小 A）

- 活动期 vs 基期环比 → 涨 10% → 结论「有效」
- **问题**：没回答「如果不发红包会怎样」——可能是季节/自然增长

### 正确思路：构建对照组

- A/B/C 测试**不是为了算显著性本身**，而是构建对照组
- 回答：**平行世界里没做策略会发生什么**

## 三、实验设计要点

- 实验组 vs 对照组（未触达策略的用户）
- 控制混淆因素（时间、渠道、用户结构）
- 用推断统计将抽样推广到整体

## 四、对分析师的启示

- 业务说「活动涨了」→ 追问**增量是否由策略导致**
- 异动分析、策略评估、投放复盘均须**对照思维**
- 与大纲中 [[数据分析-分析思维-AB测试实验基础]] 一脉相承，本视频从商业分析场景深化

## 五、延伸

- 商业分析若只做描述性对比（环比/同比），易误导决策
- 科学决策 = 描述 + **因果推断意识** + 实验或准实验设计

## 文章正文

戴师兄邀请瑞老师分享：**没有对照组，你看到的效果几乎都是假的。**

在中国，数据科学岗位本质上≈**数据分析师（DA）**——算法工作交给算法专家，留给 DS 的核心是**更科学地利用数据做决策**，帮 stakeholder 做 solid 决策。

**案例：天降红包营销活动。** 老板问：策略有没有效？下次要不要大力推广？

小 A 的做法：活动期 vs 基期环比，涨 10% → 结论「有效」；下钻到场域，80% 增量来自直播，直播场中老用户贡献 80% → 建议加大直播老用户投入。小 B 的做法：把用户分成参与/未参与活动两组，发现参与用户人均金额显著更高 → 建议整体加大投入。

两种分析看起来都「有效」，但**都没回答关键问题：如果不发红包会怎样？** 可能是季节因素或自然增长。A/B/C 测试的本质**不是为了算显著性**，而是**构建对照组**，回答：「平行世界里没做策略会发生什么？」

瑞老师强调：商业分析若只做描述性对比（环比/同比），易误导决策。正确思路需要实验组 vs 对照组（未触达策略的用户），控制混淆因素（时间、渠道、用户结构），用推断统计将抽样推广到整体。

对分析师的启示：业务说「活动涨了」→ 追问**增量是否由策略导致**；异动分析、策略评估、投放复盘均须**对照思维**。科学决策 = 描述 + 因果推断意识 + 实验或准实验设计。
