# 数据分析师各阶段薪资详解

**摘要**：数分薪资取决于工具掌握程度与业务分析经验；实习生日薪 150～300 元；应届生 8～16 万年薪；工作 1 年后分四阶段成长，1～3 年达 25 万较正常；P7 级可冲击百万；刚毕业重学习轻薪资。

## 一、薪资两大决定因素

1. **工具掌握程度**：满足数据需求的速度与质量
2. **行业/业务分析经验**：能否做出正确商业决策

## 二、实习生（未毕业）

| 技能 | 日薪 |
| --- | --- |
| Excel + SQL 取数 | ~150 元/天 |
| Tableau / Power BI | ~210 元/天 |
| Python/R + 基础建模 | 200～300 元/天 |

工作内容以满足数据需求为主。

## 三、应届生

- 工作构成：**80% 数据需求 + 20% 简单分析**
- 仅做 Excel 报表、无收入权限：**6000～9000** 月薪
- Python 熟练：**1 万起步**；大厂/名校 +2000～3000
- 综合年薪：**8～16 万**

## 四、工作 1 年以上：四阶段模型

### 第一阶段：普通工具人

- 会 SQL 和高级工具，编程弱或无，无业务分析经验
- 月薪 **1.2～1.8 万**，年薪 ~20 万
- 不学新东西、不在业务中思考会**常年停留**

### 第二阶段：高级工具人

- 工具全会，Python + 基础建模，有业务分析经验，做出过有效数据决策
- 月薪 **1.5 万起**，最高见过 3.5 万
- 阿里系 P5～P6，年薪 **20 万起，天花板 ~40 万**
- 戴师兄自评处于此阶段

### 第三阶段：真·数据分析师

- 全身心投入业务，承担「分析大脑」
- 阿里系 P7，5 年+ 经验，月薪 **3 万起**
- 含股票/业务分成可达**年薪百万**

### 第四阶段：数据负责人

- 独立负责一条或多条业务线，带数据团队
- 合伙人级别，待遇上不封顶

## 五、总结建议

- 刚毕业是搬砖工具人，**不要太看重薪资**，关键看能否快速成长
- 1～3 年达年薪 **25 万左右**较正常
- 数分技术+业务双路径清晰，**付出有回应、越老越吃香**
- 近年企业 P7 年薪很多来自数据方向

*注：一线城市平均数，可按城市/公司/个人实力增减。*

**总结。** 刚毕业是搬砖工具人，**不要太看重薪资**，关键看能否快速成长；1～3 年达年薪 25 万左右较正常。数分技术+业务双路径清晰，「付出有回应、越老越吃香」；近年企业 P7 年薪很多来自数据方向。若对数据敏感、喜欢动脑筋解决问题，建议深入了解或动手学习数据分析。
