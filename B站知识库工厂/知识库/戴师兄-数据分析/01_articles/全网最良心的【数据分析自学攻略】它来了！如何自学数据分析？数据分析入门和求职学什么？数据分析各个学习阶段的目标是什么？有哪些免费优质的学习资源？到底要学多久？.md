## 摘要

经典自学攻略：**约 3 个月入门** → 实习生/专员/数据运营。工具路径：先分析方法 → SQL → Excel → BI（面试前突击）→ Python 工作后再学。本文为框架级重写。

## 入门阶段目标

用数据分析解决**简单业务问题**（销售额下滑原因、活动 ROI 等）

- 方法书：《深入浅出数据分析》（业务场景 + 拆解 + 练习）

## 工具链（按视频顺序）

1. **SQL**：七核心语句 + 表连接/子查询 → 未明学院教程 + SQLZOO 规划刷题
2. **取数补充**：**后羿采集器**（一键采集，无需编程）
3. **Excel**：常用函数 + 数据透视表 → 泽秀老师课程
4. **可视化**：Excel 图表 + **任一 BI 基础**（Tableau/Power BI/FineBI）；**各公司 BI 不同，面试前再学目标公司工具**
5. **Python**：入门阶段**不优先**；一般 Excel 够用，百万级或特殊场景工作后再学

## 进阶阶段（简述）

- 进阶 BI、Python 数据处理、统计/建模、行业报告产出

## 文章正文

数据分析师一直是高薪岗位的代名词。早期分析师工作比较基础——处理业务数据需求、准备 Excel 报表，解决实际问题的机会不多，因为系统分析需要业务基础和熟悉数据的时间；一开始提供基础数据支持就能快速上手，为以后解决更复杂问题做准备。这些基础工作并不难，学习目标明确的话，**大多数小伙伴努努力，三个月内就能快速入门**，胜任实习生、专员、数据运营等基础数据型岗位。

**入门阶段第一个目标：用数据分析解决简单业务问题。** 比如今天销售额下滑了，用数据和图表说明原因；公司办促销活动，计算投入产出比。市面上免费付费课程大多变着法教工具，很少教「如何用数据分析解决实际问题」。戴师兄找了两周，选出《深入浅出数据分析》——用对话和大量图文还原业务场景，逐步拆解；有问题引入、沟通中提取信息、拆解问题，有分析过程和结论参考，还有练习题；覆盖销售、用户绩效等真实场景，「比付费课程还良心」，入门阶段融会贯通这一本完全足够。

**获取数据：掌握 SQL 七大核心语句及表连接、子查询。** 推荐未明学院 SQL 零基础教程，边看边操作；搞定后到 SQLZOO 刷题，按练习规划进行，刷完能应付大部分 SQL 取数。入门阶段还可补充**后羿采集器**——输入网址一键采集，完全不需要编程。取出数据后，**Excel 常用函数和数据透视表**就足够处理——一般业务场景 Excel 完全能处理，工作初期很少接触需 Python 处理的百万级数据；「入门讲究快速使用，一开始就学编程一般直接劝退」。Excel 教程推荐泽秀老师 B 站代表作。

**可视化：Excel 图表 + 任一 BI 基础。** 人是视觉动物，正确图表帮助分析，也是支撑结论的关键；做 PPT 都要附数据图表。掌握 Excel 和任意智能图表工具的基础即可——Tableau、Power BI、FineBI 都行，但**各公司 BI 不同，一般面试前突击学目标公司工具即可**，「提前学一大堆可能白学，出图原理大致相通，学会 Tableau 其他的八九不离十」。Tableau 推荐未明学院零基础教程。

**进阶：成为一线大厂分析师。** 很多小伙伴自学一段时间后陷入「无法提升分析思维、不知道工具用在哪」——根因是一直学工具、从没学「数据分析如何解决问题」，潜意识没有清晰目标就会迷失。大厂不要高薪数据工具人，「工具掌握再多，不能帮企业解决实际问题，迟早被更先进工具或学习能力更强的新人取代」。求职阶段第一目标是**直面问题并用数据分析能力解决它**——养成勤思考、多实践，遇到问题不说做不了、只说怎么做；业务问业绩提升方法，若觉得跟己无关就失去思考机会，「业务能解决还要咱们干什么」；认真思考就会发现必须结合数据分析，用上工具、做研究假设、补充行业新玩法，工具用上了、思维活跃了、还能学到新知识——「真正的分析思维和能力都是练会的，不是看会的」。没有业务场景也可以积累：《精益数据分析》30 多个真实案例跟着思考；36 氪、人人都是产品经理、简书每天一篇做笔记；最后产出深度行业或产品分析报告，知乎有系统攻略，最好研究目标公司产品和行业。

**高级 SQL：** 掌握开窗函数、数据转置（转置更应在 Excel/Tableau 实现）；腾讯网易滴滴等大厂笔试重难点。推荐黎曼猜想 MySQL 教程（近 200 万播放），学完多练习，牛客网历年大厂 SQL 真题全刷一遍就出师。Hive SQL 岗位有要求再看尚硅谷教程，不是重点。

**Python 登场时机：** Excel 预览限 100 行、灵活处理最好在 10 万行以下，业务规模提升后不得不上 Python。推荐尚硅谷 Python 技术 + 黑马 Python 数据分析，掌握函数控制流和 numpy、pandas、matplotlib；牛客网 Python 数据分析真题至少刷 100 题，每天三小时约一个月搞定。算法建模分析师只需掌握 sklearn 机器学习模型——自动推荐、人脸识别等高级算法是算法工程师的事；国内能做好的算法团队不多，很多公司对分析师算法能力是幻想。推荐莫凡 sklearn 教程，系统学看上硅谷机器学习和推荐系统项目；书籍《统计学习方法》《西瓜书》，深入可看花书；最后在和鲸社区做从分析到建模的完整项目，链接写进简历。

**常见问题：** 数学统计学**不需要专门花时间学**——实践已封装在工具里，现学现卖足够，「目的是解决问题不是证明掌握知识点」。学多久？入门 50～100 小时；找大厂工作至少**三个月到半年**持续积累，每天两小时以上——高薪职业有门槛很正常。就业前景：分析师本质是用数据帮企业量化结论甚至决策，是未来公司核心岗位；待遇高、空间大；对职业前景有疑虑，一是个别「工具人」以偏概全，二是教育机构过度吹嘘——跟一线大厂分析师聊聊，能真切感受价值和魅力。
