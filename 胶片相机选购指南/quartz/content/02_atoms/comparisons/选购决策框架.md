# 选购决策框架

## 一句话定义
在选具体型号之前，先回答三个问题来决定相机类型方向。

## 三个核心问题

### 1. 你只是想随手记录，还是想学摄影？
- **随便拍拍** → 傻瓜机（PS 机）或一次性相机
- **想深入学习** → 手动单反或旁轴，带测光

### 2. 你愿意每次按快门前花多少时间？
- **越快越好（2 秒内）** → 傻瓜机或全自动单反
- **可以接受 30 秒调整参数** → 半机械单反（有 A 档）
- **享受慢慢来的过程** → 全机械单反

### 3. 你愿意带多重的相机出门？
- **口袋级别（< 300g）** → 傻瓜机、半格机
- **可以接受挂脖子上（500-800g）** → 手动单反
- **不介意背包（> 1kg）** → 中画幅

## 决策路径

```
你想要的体验是？
├── 和数码一样方便 → 全自动单反 / 高端 PS 机
├── 和数码不同，有仪式感 → 机械手动单反
├── 最小最轻，随身带 → 傻瓜机 / 半格机
├── 画质至上，不嫌重 → 中画幅
└── 只想试试水 → 一次性相机 / 百元傻瓜机
```

## 为什么重要

多篇文章反复提到一个共识：最贵的相机不一定是"最好的"，"最好的"是那台你愿意每天带出门、愿意频繁使用的相机。这个决策框架帮助你从"我想买一台胶片相机"这个模糊需求，收敛到具体的相机类型。
