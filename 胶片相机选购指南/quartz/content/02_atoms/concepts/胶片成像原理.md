# 胶片成像原理

## 一句话定义
胶片成像是光化学过程——底片上的卤化银颗粒感光后在显影液中还原出银颗粒，形成物理影像。这和数码的光电转换（CMOS 传感器）有着本质区别。

## 成像流程

```
光线 → 镜头 → 快门打开 → 底片上的卤化银颗粒感光（形成潜影）
→ 显影液将感光的卤化银还原为银颗粒（曝光越强，银越多，密度越高）
→ 定影液去除未感光的卤化银
→ 底片上呈现明暗相反的负像
→ 扫描或放大 → 正像照片
```

## 彩色胶片的特殊性

彩色负片叠加了三层感光层——分别感光蓝、绿、红光，同时配有黄、品、青成色剂。经过显影后，每个感光层形成对应补色的彩色负像。这就是为什么彩色负片的底片呈棕色（橙色片基层 + 有色染料）。

## 与数码的本质区别

| 维度 | 胶片 | 数码 |
|------|------|------|
| 成像原理 | 光化学（卤化银） | 光电转换（CMOS） |
| "传感器" | 每卷胶卷都是一个独立的"传感器" | 固定于机身 |
| 风格切换 | 换胶卷 = 换传感器风格 | 换机身或后期处理 |
| 高光过渡 | 化学密度曲线渐进，高光柔和 | 电子截断，高光生硬 |
| 暗光表现 | 弱，暗部几乎无细节 | 强，ISO 可调且噪点可控 |
| 不可逆性 | 物理化学反应不可撤销 | 数字文件可无限复制/修改 |

## 关键含义

每换一卷胶卷，就等于换了一个新的"传感器"。同是 135 机身，装不同的卷——柯达金 200、Portra 400、富士 C200——出片风格完全不同。这种灵活性在数码世界需要换机身甚至换系统才能实现。
