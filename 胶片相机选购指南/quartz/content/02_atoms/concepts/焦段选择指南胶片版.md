# 焦段选择指南（胶片版）

## 一句话定义

焦段选择决定你的视角和拍摄风格，胶片入门从 50mm 标头开始是最稳妥的策略——先在这个焦段拍几卷，形成自己的偏好后再扩展。

## 详细说明

### 焦段速查表

| 焦距 | 视角类型 | 典型场景 | 注意事项 |
|------|---------|----------|---------|
| 28mm | 广角 | 街拍、建筑、需要冲击力的场景 | 离被摄者近，透视变形明显 |
| 35mm | 小广角 | 人文纪实、环境人像、日常扫街 | 比 50mm 容纳更多环境信息 |
| 50mm | 标准 | 人像、扫街、万能入门 | 视角接近人眼自然视野 |
| 85mm+ | 长焦 | 演唱会、远距离抓拍、压缩感 | 体积大、手持稳定性要求高 |

（表格内容基于老王的总结）

### 50mm 为什么是万能入门焦段

1. **视角自然** — 接近人眼单眼视野，所见即所得，构图直觉友好
2. **大光圈版本性价比极高** — f/1.7 或 f/1.9 的 50mm 镜头是各系统中最便宜的定焦之一，几百块就能买到成像优秀的大光圈标头
3. **题材覆盖广** — 人像能拍（大光圈虚化），扫街能拍（不打扰被摄者），日常记录也够了
4. **套机标配** — 大多数入门胶片单反套机标配 50mm f/1.7 或 f/1.8

### 安全快门法则（胶片版）

> 安全快门速度 ≥ 焦距的倒数

| 焦距 | 理论安全快门 | 老相机实际选择 |
|------|-------------|---------------|
| 28mm | 1/30s | 1/30s 或 1/60s |
| 35mm | 1/30s~1/60s | 1/60s |
| 50mm | 1/50s | 1/60s 或 1/125s |
| 85mm | 1/85s | 1/125s |
| 135mm | 1/135s | 1/250s |

老相机的快门档位不一定有精确对应的速度，原则是**选更快的档位**（宁快勿慢）。

### 选购策略

1. **第一支镜头** — 从 50mm 标头开始。便宜、好用、什么都能拍
2. **先拍再买** — 先用 50mm 拍 3-5 卷，感受自己在这个焦段下的构图习惯
3. **发现偏好再扩展** — 如果总想拍得更广 → 考虑 35mm 或 28mm；总想拍得更近/更远 → 考虑 85mm+
4. **关注卡口兼容性** — 不同品牌机身的镜头卡口不同，入手前确认镜头能装到你的机身上
