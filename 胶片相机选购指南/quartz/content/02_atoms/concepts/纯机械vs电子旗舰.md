# 纯机械 vs 电子快门旗舰

## 一句话定义
F2 和 F3 的选择本质是一个哲学问题——你要纯机械的永恒可靠性，还是要电子快门的精准便利。

## 详细说明

### 两代旗舰对比

| 维度 | 尼康 F2（1971） | 尼康 F3（1980） |
|------|----------------|----------------|
| 快门类型 | 全机械 | 电子（1/60s 机械备份） |
| 没电时 | 所有快门速度可用 | 只有 1/60s |
| A 档 | 无 | 有 |
| 重量 | 约 730g | 约 700g（F3HP） |
| 测光 | 需配测光顶 | 内置测光 |
| 可换取景器 | 是（DE-1/DW-1/DP 系列） | 是（含高眼点 HP 版本） |
| 设计哲学 | 永不依赖电池 | 电子精准 + 机械备份 |

### 这个选择不只是尼康的

类似的"纯机械旗舰 vs 电子旗舰"选择在所有大品牌中都存在：

| 品牌 | 纯机械旗舰 | 电子旗舰 |
|------|----------|---------|
| 尼康 | F2 | F3 |
| 佳能 | — | F-1 / New F-1 |
| 宾得 | MX（非旗舰但全机械） | LX |
| 美能达 | SRT 系列（非旗舰） | X-1（XK/XM） |

### 怎么选

- 如果你追求的胶片体验是"和数码完全不同、每一张都亲手控制" → 选纯机械
- 如果你想要数码时代的舒适感延续到胶片上 → 选电子快门 + A 档
- 没有对错——这是一个关于"你为什么拍胶片"的问题

### 一个实用的视角

F2 和 F3 的价格差距并不大。选 F2 的人通常不是在省钱，而是在为"全机械"的仪式感买单。选 F3 的人也不是在偷懒，而是在选择"把精力放在构图上，把曝光交给相机"。
