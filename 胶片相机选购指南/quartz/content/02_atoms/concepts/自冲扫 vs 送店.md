# 自冲扫 vs 送店

## 一句话定义
送店冲扫是新手的最佳路径——除非你每月拍几十卷，否则自冲扫省不了钱也省不了心。

## 送店冲扫

### 流程
```
拍完 → 回卷取出 → 包装好 → 寄给冲扫店 → 店家冲洗 → 扫描仪数字化 → 
百度网盘/邮箱发你 → （可选）底片回邮
```

### 优点
- 省时省力，寄出去等结果
- 专业设备（自动冲卷机 + 专业扫描仪）
- 质量稳定，尤其 C-41 彩色负片（标准化工艺）
- 新手只管拍，剩下的不用操心

### 缺点
- 有等待时间（通常 2-5 天，视店家排期）
- 每卷 15-50 元（SP3000 最便宜，哈苏 X5 最贵）
- 扫描风格被店家决定（除非加钱选扫描仪）

## 自冲扫

### 最低设备投入
| 设备 | 价格（约） |
|------|-----------|
| 显影罐 | 100-200 元 |
| 量杯、温度计 | 50-100 元 |
| 药水（C-41 套装） | 150-300 元/套 |
| 暗袋（装卷用） | 50-100 元 |
| **冲卷设备合计** | **约 350-700 元** |
| 扫描仪（入门级） | 1000-3000 元 |
| 翻拍设备（数码相机+微距镜头+翻拍台+光源） | 2000-5000+ 元 |
| **完整自冲扫** | **约 2500-6000 元起步** |

### 隐性成本
- **学习成本**：药液比例、温度（C-41 要求 38°C 精确控温）、搅拌方式、时间控制——每一个变量都影响出片。新手很难第一次就达到专业冲扫店的稳定水平。
- **时间成本**：冲一卷黑白约 30 分钟，彩色（含温控准备）约 1 小时
- **药水有效期**：开封后药水有使用期限，拍太少的话药水过期浪费可能比送店还贵

### 胶片猫的结论
"除非你每个月要拍几十卷，否则将胶卷寄给专业冲扫店更加省时、稳定，也更划算。"

## 什么情况下值得自冲扫

- **黑白胶片爱好者**：黑白冲洗流程简单（不需要严格温控），药水便宜，出片效果个性化空间大
- **大量拍摄者**：每月 10 卷以上，自冲扫开始有成本优势
- **享受过程的人**：把冲洗当作胶片体验的一部分——和做菜一样，自己动手是乐趣

## 起步建议

1. 新手：先送店，什么都不用操心。SP3000 扫描约 15-25 元/卷。
2. 拍了十几卷后：如果对扫描风格有自己的偏好，可以尝试哈苏 X5 扫描（约 30-50 元/卷）——色彩过渡更细腻。
3. 愿意投资时：先试自冲黑白（简单、容错高），再考虑彩色。
4. 翻拍（用数码相机拍底片）：门槛最高但画质上限最高——需要全画幅数码机 + 1:1 微距镜头 + 翻拍台。
