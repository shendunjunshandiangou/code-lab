# 冲扫店选择

## 一句话定义
不同冲扫店扫同一卷胶卷，出来的颜色和画质差异很大——选店不是"淘宝搜个销量高的就行"。

## 为什么不同店结果不同

许多人以为扫描是客观的"还原底片"，其实不是。冲扫店在扫描过程中做了大量主观判断：
- **去色罩算法**：底片是棕色反相的，转成正常颜色这一步依赖扫描仪的色彩引擎和店家的设定
- **曝光/对比度调整**：同一张底片，A 店设偏暗、B 店设偏亮——出来的反差不一样
- **色调偏移**：即使是同一台 SP3000，不同店的默认调色风格也不同——有的偏青、有的偏品、有的偏暖

> 这就是为什么"胶片的色彩"既是胶卷本身决定的，也在很大程度上是冲扫店决定的。

## 选店五维评价

| 维度 | 怎么看 | 好/坏信号 |
|------|--------|----------|
| **SP3000 调色风格** | 看店家样片 | 偏青偏品 vs 偏暖偏黄——选你喜欢的 |
| **底片回邮速度** | 商品描述/评价 | 快（3-5 天） vs 慢（2 周以上） |
| **划痕率/脏点率** | 差评区找 | 干净 = 重视除尘；脏 = 流水线粗放 |
| **客服响应/态度** | 售前问一句 | 积极回应 vs 只回自动话术 |
| **数字文件交付** | 网盘还是邮件 | 网盘链接长期有效 vs 限期下载 |

## 不同扫描仪的店铺定位

| 扫描仪 | 店铺常见定价 | 风格定位 | 适合 |
|--------|------------|----------|------|
| 富士 SP3000 | 15-25 元/卷 | 色彩浓郁、偏日系——"胶片味"的来源 | 日常出片、社交分享 |
| 诺日士（HS-1800） | 20-35 元/卷 | 色彩中规中矩、还原度高 | 想要更准确的色彩 |
| 哈苏 X5 / Flextight | 35-80+ 元/卷 | 细腻、动态范围大、色调中性 | 重要照片、需要大幅输出 |

## 第一次送冲扫的备注模板

下单时在备注栏写清楚这几条，避免 90% 的沟通问题：

```
1. 胶卷类型：彩色负片 C-41 / 黑白 / 反转片 E-6（选一个）
2. 扫描仪：SP3000 标准套餐
3. 底片回邮：需要（请用硬质底片袋寄回，避免折损）
4. 不剪断底片：是（整卷保留，方便以后翻拍）
5. 分辨率：标准（约 1800×1200，够社交用）
6. 划痕处理：请尽量除尘和修复
```

## 翻车怎么办

- 整卷偏色 → 换一家店重新扫（底片还在的话）。**底片是唯一凭据**，没回邮就无处申诉
- 几张过曝/欠曝 → 大概率是你拍摄时的问题，不是冲扫
- 全卷雾化/有光斑 → 可能是冲洗药水问题或漏光，找店家沟通
- 划痕多 → 这家店除尘不认真，下次换店
