# 从数码过渡到胶片的路径

## 一句话定义

已有数码摄影经验的人过渡到胶片，核心挑战不在于曝光三要素，而在于胶片特有的操作流程和思维方式——增量体验取决于你选择的相机类型是否能打开一扇新的门。

## 详细说明

### 路径一：利用现有 DSLR 练习（Jie's Lens 建议）

如果你已经有一台数码单反/微单：

1. 把它调到 M 档
2. ISO 固定 200 或 400
3. 练习手动调整快门和光圈
4. 数码的即时反馈让你不花胶卷钱就掌握曝光基础
5. 一旦基础扎实，转移到胶卷只是换一个记录载体

核心优势：零额外成本获得曝光直觉，不需要等冲洗、不需要在错误上花胶卷钱。

### 路径二：从机械相机切入（萌大统领的分析）

有数码使用经验的人，傻瓜机和机械相机之间的选择值得深思：

- 数码相机已经是一台高度自动化的机器——操作逻辑和傻瓜机高度重叠
- 再买一台操作逻辑类似的傻瓜机，"增量体验"很有限，只是换了个成像介质
- 机械相机提供完全不同的操作体验——纯手动过片、机械快门声、全手动控制——这是数码给不了的体验维度
- 对于追求"玩胶片"而非"用胶片"的人来说，机械相机带来的学习曲线本身就是价值

### 胶片猫的悖论

"胶片反而比数码更适合入门学摄影——因为每张都要钱，迫使你慢下来思考。"

但对于已有数码经验的人，这个限制不是教育性的，而是选择性的——你已经会思考了，现在更需要的是感受胶片特有的节奏。

### 过渡后需要新学的内容

| 技能 | 说明 |
|------|------|
| 手动上卷与回卷 | 数码完全没有这个概念 |
| 手机测光 | 老相机内测光不可靠时的替代方案 |
| ISO 不可中途更改 | 一卷胶卷固定一个感光度，不能每张换 |
| 拍完无法回看 | 需要培养对曝光的信心和对构图的预判 |
| 冲扫流程认知 | C-41 还是 E-6、选什么扫描仪、找哪家冲扫店 |

### 过渡后不需要重新学的内容

曝光三要素、安全快门法则、构图基础、对焦逻辑——这些在数码阶段已经积累，直接迁移即可。
