# 佳能 A-1

> [!image] 佳能 A-1 外观参考
> 点击查看 [Canon A-1 SLR camera](https://www.google.com/search?tbm=isch&q=Canon%20A-1%20SLR%20camera)

## 一句话定位
FD 系统功能之王，P/A/S/M 四种曝光模式全覆盖。

## 基本参数

| 参数 | 值 |
|------|-----|
| 发布时间 | 1978 |
| 快门类型 | 电子快门 |
| 曝光模式 | P + A + S + M |
| 卡口 | 佳能 FD 卡口 |
| 对焦方式 | 手动 |
| 电池 | 需要 |

## 优点
- **PASM 四档全覆盖**：手动单反中功能最完整的机型之一
- **FD 镜头群**：可用佳能手动时代所有经典镜头
- 保有量大，零件好找

## 缺点
- 电子快门依赖电池
- FD 卡口法兰距短，无法转接到现代佳能 EF 机身上使用

## 价格参考（2026）
- 二手市场：约 800-1,500 元

## 适合谁
- 想要一台功能完整、不挑场景的 FD 口单反的人

## 不适合谁
- 想要在现代数码机身上共用 FD 镜头的人（法兰距限制）
