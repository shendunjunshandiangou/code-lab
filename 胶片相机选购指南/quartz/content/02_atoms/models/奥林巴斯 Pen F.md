# 奥林巴斯 Pen F 系列

> [!image] 奥林巴斯 Pen F 外观参考
> 点击查看 [Olympus Pen F half-frame camera](https://www.google.com/search?tbm=isch&q=Olympus%20Pen%20F%20half-frame%20camera)

## 一句话定位
有史以来最小的可换镜头单反系统，半格 + 全机械。

## 基本参数

| 参数 | 值 |
|------|-----|
| 发布时间 | 1963 |
| 快门类型 | 全机械快门 |
| 曝光模式 | M（全手动） |
| 卡口 | Pen F 专用卡口 |
| 对焦方式 | 手动 |
| 电池 | 不需要 |
| 画幅 | 135 半格（一张 135 底片拍两张） |

## 优点
- **极小单反**：把一整套单反系统塞进了半格相机的机身里，有史以来最小的可换镜头单反系统之一
- **可换镜头**：有专属镜头群
- **半格经济**：一卷 36 张的胶卷可以拍 72 张——拍摄成本减半
- 全机械，无需电池
- 工业设计独特

## 缺点
- 半格画幅，画质不如全幅 135
- Pen F 镜头群小，不好找
- 半格竖构图和取景习惯需要适应

## 价格参考（2026）
- 二手市场：约 1,500-3,000 元

## 适合谁
- 追求极致小巧机械相机的人
- 想体验半格拍摄的人

## 不适合谁
- 追求画质极限的人
- 想要 A 档便利的人
