# 宾得 KM / KX

> [!image] 宾得 KM 外观参考
> 点击查看 [Pentax KM camera](https://www.google.com/search?tbm=isch&q=Pentax%20KM%20camera)

## 一句话定位
K1000 的姐妹型号，K 卡口经典入门机械单反。

## 基本参数

| 参数 | 值 |
|------|-----|
| 发布时间 | 1975 |
| 快门类型 | 全机械快门 |
| 曝光模式 | M（全手动） |
| 卡口 | 宾得 K 卡口 |
| 对焦方式 | 手动 |
| 电池 | 不需要（或仅供电测光） |

## 优点
- **K 卡口生态**：镜头选择多且便宜
- 全机械，简单耐用
- 经典的金属机身手感

## 缺点
- 功能基础，只有全手动
- 测光表老化
- 不如 K1000 知名

## 价格参考（2026）
- 二手市场：约 400-800 元

## 适合谁
- 想入门 K 卡口系统、预算有限的人

## 不适合谁
- 想要更知名型号的人（K1000 存量更大）
