# 徕卡 M6

> [!image] 徕卡 M6 外观参考
> 点击查看 [Leica M6 camera](https://www.google.com/search?tbm=isch&q=Leica%20M6%20camera)

## 一句话定位
带测光的经典 M——但张有伯觉得不值

## 基本参数
| 参数 | 值 |
|------|-----|
| 发布时间 | 1984 |
| 镜头 | Leica M 卡口 |
| 快门类型 | 机械快门（电子测光） |
| 曝光模式 | M（内置测光表辅助） |
| 对焦方式 | 手动（旁轴联动测距） |
| 电池 | 测光需要（快门不需要） |

## 优点 / 缺点

**优点**
- 内置测光表（M3/M2/M4 都没有）
- 张有伯认为比 M3 更优雅
- Leica M 卡口，传奇镜头生态
- 旁轴取景体验独特（框线预览、取景器外视野可见）
- 做工精湛，保值率极高

**缺点**
- 极贵，15000-25000 元
- **张有伯明确表示"这个价位不值得"**
- 旁轴对焦并非人人适应（黄斑对焦有一定门槛）
- Leica 税——镜头、配件、保养都贵得离谱
- 取景器可能出现眩光问题（早期批次）

## 价格参考（2026）
~15000-25000 元

## 适合谁 / 不适合谁

**适合**：预算充裕的旁轴爱好者、徕卡收藏者、追求极致机械质感的人

**不适合**：预算有限的人（张有伯不推荐）、不喜欢旁轴对焦方式的人、追求性价比的实用主义者
