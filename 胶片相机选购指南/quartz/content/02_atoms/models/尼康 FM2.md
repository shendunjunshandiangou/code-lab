# 尼康 FM2

> [!image] 尼康 FM2 外观参考
> 点击查看 [Nikon FM2 camera](https://www.google.com/search?tbm=isch&q=Nikon%20FM2%20camera)

## 一句话定位
豆瓣四大神机之一，全机械快门标杆，纯机械可靠性的巅峰。

## 基本参数

| 参数 | 值 |
|------|-----|
| 发布时间 | 1982 |
| 快门类型 | 全机械纵走焦平面快门 |
| 最高快门 | 1/4000s |
| 曝光模式 | M（全手动） |
| 卡口 | 尼康 F 卡口 |
| 对焦方式 | 手动（裂像+微棱镜） |
| 电池 | 不需要（或仅供电测光表） |
| 重量 | 约 540g |

## 优点
- 全机械——没电池也能拍所有快门速度
- 快门最高 1/4000s（同级别中非常出色）
- 做工扎实，可靠性在手动单反中数一数二
- 尼康 F 卡口，镜头群庞大且便宜

## 缺点
- 价格已被炒高（网红神机溢价）
- 只有 M 档，没有 A 档——每次按快门都需要手动设置光圈和快门

## 价格参考（2026）
- 二手市场：约 1,500-2,500 元

## 适合谁
- 追求纯机械可靠性、想深入学习手动曝光的人

## 不适合谁
- 想要 A 档便利的人（选 尼康 FE）
- 预算有限的人（网红溢价不值得硬追）

## 相关机型

- [[尼康 FG]] — 更小巧的入门选项
