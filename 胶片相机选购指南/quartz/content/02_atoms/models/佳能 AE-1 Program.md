# 佳能 AE-1 Program

> [!image] 佳能 AE-1 Program 外观参考
> 点击查看 [Canon AE-1 Program camera](https://www.google.com/search?tbm=isch&q=Canon%20AE-1%20Program%20camera)

## 一句话定位
AE-1 的升级版，多了 P 档更方便。

## 基本参数

| 参数 | 值 |
|------|-----|
| 发布时间 | 1981 |
| 快门类型 | 电子快门 |
| 曝光模式 | P + S + M |
| 卡口 | 佳能 FD 卡口 |
| 对焦方式 | 手动 |
| 电池 | 需要 |

## 优点
- **P 档（程序自动曝光）**：比 AE-1 多了一个全自动模式，更方便
- **保有量大**：和 AE-1 共享 FD 生态系统
- 操作简单，适合新手

## 缺点
- 电子快门依赖电池
- 同样有网红溢价

## 价格参考（2026）
- 二手市场：约 1,000-1,800 元

## 适合谁
- 想要 AE-1 的经典属性 + P 档省心操作的人

## 不适合谁
- 预算有限的人（有更便宜的 FD 口选择）
