# 奥林巴斯 Pen D3

> [!image] 奥林巴斯 Pen D3 外观参考
> 点击查看 [Olympus Pen D3 camera](https://www.google.com/search?tbm=isch&q=Olympus%20Pen%20D3%20camera)

## 一句话定位
操作极简的半格机，除了 ISO 什么都不用调。

## 基本参数

| 参数 | 值 |
|------|-----|
| 镜头 | 固定镜头 |
| 快门类型 | 电子（硒光测光 — 免电池） |
| 对焦方式 | 固定焦距（泛焦） |
| 曝光模式 | 全自动 |
| 唯一手动设置 | ISO（由胶卷决定） |
| 画幅 | 半格（135 全格的 1/2，72 张/卷） |

## 优点
- 操作极简——只需要把镜头上的 ISO 拧到所用胶卷的值
- 光线不足时取景框弹出一面小红旗阻止拍摄（防翻车设计）
- 硒光测光免电池
- 72 张/卷，实验成本低

## 缺点
- 半格画质打折扣
- 泛焦无法精确对焦
- 硒光测光可能随年龄老化失效

## 价格参考（2026）
- 二手市场：约 500-1000 元

## 适合谁
- 第一次玩胶片、想体验半格 72 张的快感
- 需要一台完全不用动脑的随身机
