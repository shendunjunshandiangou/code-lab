# 玛米亚 RB67

> [!image] 玛米亚 RB67 外观参考
> 点击查看 [Mamiya RB67 camera](https://www.google.com/search?tbm=isch&q=Mamiya%20RB67%20camera)

## 一句话定位
除了重以外没有任何缺点——67 画幅巨无霸。

## 基本参数

| 参数 | 值 |
|------|-----|
| 发布时间 | 1970 |
| 画幅 | 6×7（120 胶卷，10 张/卷） |
| 快门类型 | 镜间快门（部分版本机械） |
| 曝光模式 | M（全手动） |
| 卡口 | 玛米亚 RB 卡口（可换镜头） |
| 对焦方式 | 手动 |
| 电池 | 不需要（部分后期版本需要电池供测光） |
| 特点 | 模块化：可换后背、取景器、镜头；旋转后背（横竖切换）；皮腔对焦 |

## 优点
- **张有伯评价"除了重以外没有任何缺点"**：镜头锐利、做工扎实、功能完善
- **6×7 巨大底片**：底片面积是 135 的约 4.5 倍，放大能力和细节表现极强
- **旋转后背**：无需旋转机身即可在横拍和竖拍之间切换，天才设计
- **腰平取景视觉冲击力大**：6×7 画幅的腰平取景画面震撼
- **皮腔对焦**：近摄能力强，适合静物和产品拍摄
- **价格合理**：3,000-5,000 元

## 缺点
- **极其沉重**：携带出门等于健身——"只有体力好的人才能带出门"
- **操作缓慢**：装卷、对焦、过片流程繁琐，不适合快节奏拍摄
- **体积庞大**：比哈苏 500 还大一圈，非常不便携

## 价格参考（2026）
- 二手市场：约 3,000-5,000 元

## 适合谁
- 追求极致画质、最大底片面积的深度玩家
- 体力好、愿意扛重型器材出门的人
- 棚拍或固定场景拍摄为主的人

## 不适合谁
- 需要便携的人（选勃朗尼卡 ETR）
- 不喜欢重型器材的人
- 快节奏街拍/旅行拍摄的人
