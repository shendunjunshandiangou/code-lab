# 禄来 SL35 / SL35E

> [!image] 禄来 SL35 外观参考
> 点击查看 [Rollei SL35 camera](https://www.google.com/search?tbm=isch&q=Rollei%20SL35%20camera)

## 一句话定位
禄来的 135 单反尝试，SL35 系列是入门，SL2000F/3003 是模块化先驱。

## 基本参数

| 参数 | 值 |
|------|-----|
| 发布时间 | 1970（SL35） |
| 快门类型 | 全机械快门（SL35）/ 电子快门（SL35E） |
| 曝光模式 | M（SL35）/ A + M（SL35E） |
| 卡口 | QBM 卡口（禄来） |
| 对焦方式 | 手动 |
| 电池 | 需要（SL35E） |

## 优点
- **禄来品牌**：双反/中画幅传奇品牌的 135 单反
- SL35E 功能更完整
- 有蔡司为 QBM 卡口制作的镜头

## 缺点
- QBM 卡口生态小，镜头难找
- 不是禄来最擅长的领域（禄来的核心在中画幅/双反）
- 维修难找师傅

## 价格参考（2026）
- 二手市场：约 500-1,500 元

## 适合谁
- 禄来粉丝想体验品牌的 135 单反
- 小众收藏者

## 不适合谁
- 追求实用性和维修便利的普通用户
