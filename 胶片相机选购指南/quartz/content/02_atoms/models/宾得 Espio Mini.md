# 宾得 Espio Mini

> [!image] 宾得 Espio Mini 外观参考
> 点击查看 [Pentax Espio Mini camera](https://www.google.com/search?tbm=isch&q=Pentax%20Espio%20Mini%20camera)

## 一句话定位
还未被市场爆炒的高素质傻瓜机，32mm 定焦在 PS 机中很罕见，被多位 UP 主评价为"对焦速度天花板级"。

## 基本参数

| 参数 | 值 |
|------|-----|
| 发布时间 | 约 1995 |
| 镜头 | 定焦 32mm f/3.5（罕见焦段） |
| 对焦方式 | 自动对焦 |
| 曝光模式 | P 档全自动 |
| 闪光灯 | 内置 |
| 过片/回卷 | 全自动 |
| 机身 | 塑料 |
| 特殊功能 | 宽幅模式（模仿哈苏宽幅视觉效果） |

## 优点
- **32mm 焦段**：不大不小，介于广角和标准之间，随身带拍什么都顺手
- **对焦速度**：萌大统领评价"所有傻瓜机里排第一或第二，超级超级快"——适合抓拍
- **开机速度快、自动对焦果断、测光稳定、镜头解析力高**——德松的评价
- **未被爆炒**：普通黑色版千元左右，限量银色版贵一些
- **宽幅模式**：增加可玩性
- 集各种优点于一身，确实对得起旗舰定位

## 缺点
- 塑料机身，质感不如金属机
- 外观少了点"社交属性"——德松原话："不然谁还用 Contax T2/T3 呢？"
- 和所有 90 年代 PS 机一样有电子老化风险

## 价格参考（2026）
- 普通黑色版：约 1,000-1,500 元
- 限量银色版：更高

## 适合谁
- 想要高素质傻瓜机但不想花 Contax 级别钱的人
- 重视对焦速度和抓拍体验的人
- 喜欢实用＞品牌的理性玩家

## 不适合谁
- 追求金属质感和社交属性的人
- 一定要最顶级光学素质的人（不是不够好，但打不过 Contax T3/CM）
