# 富士 ST-801 / ST-901

> [!image] 富士 ST-801 外观参考
> 点击查看 [Fujica ST801 camera](https://www.google.com/search?tbm=isch&q=Fujica%20ST801%20camera)

## 一句话定位
富士早期 M42 螺口单反，可以玩转海量 M42 老镜头。

## 基本参数

| 参数 | 值 |
|------|-----|
| 发布时间 | 1970 年代 |
| 快门类型 | 全机械快门（ST-801）/ 电子快门（ST-901） |
| 曝光模式 | M（全手动） |
| 卡口 | M42 螺口 |
| 对焦方式 | 手动 |
| 电池 | 不需要（ST-801）/ 需要（ST-901） |

## 优点
- **M42 螺口**：可以搭配从东德蔡司到苏联 Jupiter 的海量 M42 镜头
- ST-801 有当时非常先进的 LED 测光显示
- 富士 EBC 镀膜在当时是一流水平

## 缺点
- M42 螺口换镜头慢
- 测光可能已失效
- 富士 M42 系列存量不多

## 价格参考（2026）
- 二手市场：约 400-800 元

## 适合谁
- 想玩 M42 镜头群、又喜欢富士品牌的人

## 不适合谁
- 追求快速换镜头体验的人
