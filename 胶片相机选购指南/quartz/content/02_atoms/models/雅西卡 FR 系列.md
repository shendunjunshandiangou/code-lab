# 雅西卡 FR 系列

> [!image] 雅西卡 FR 系列 外观参考
> 点击查看 [Yashica FR camera](https://www.google.com/search?tbm=isch&q=Yashica%20FR%20camera)

## 一句话定位
雅西卡 CY 卡口的电子进阶，比 FX-3 多了自动功能。

## 基本参数

| 参数 | 值 |
|------|-----|
| 发布时间 | 1977 |
| 快门类型 | 电子快门 |
| 曝光模式 | A + M |
| 卡口 | CY 卡口（Contax/Yashica） |
| 对焦方式 | 手动 |
| 电池 | 需要 |

## 优点
- **CY 卡口**：可兼容蔡司镜头
- **A 档自动曝光**：比纯手动的 FX-3 方便
- 价格相对合理

## 缺点
- CY 卡口镜头不如 K 卡口/F 卡口丰富
- 蔡司镜头不便宜——便宜机身的本质是让你把钱花在镜头上

## 价格参考（2026）
- 二手市场：约 500-1,000 元

## 适合谁
- 想在 CY 卡口里找一台自动曝光机身的人

## 不适合谁
- 追求纯机械的人（选 FX-3）
