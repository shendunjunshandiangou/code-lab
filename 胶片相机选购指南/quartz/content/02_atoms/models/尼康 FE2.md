# 尼康 FE2

> [!image] 尼康 FE2 外观参考
> 点击查看 [Nikon FE2 camera](https://www.google.com/search?tbm=isch&q=Nikon%20FE2%20camera)

## 一句话定位
FE 的升级版——1/4000s 高速电子快门

## 基本参数
| 参数 | 值 |
|------|-----|
| 发布时间 | 1983 |
| 镜头 | Nikon F 卡口 |
| 快门类型 | 电子（最高 1/4000s，机械备份 1/250s） |
| 曝光模式 | A / M |
| 对焦方式 | 手动（裂像 + 微棱对焦屏） |
| 电池 | 需要（LR44 × 2） |

## 优点 / 缺点

**优点**
- A 档光圈优先，日常拍摄轻松自如
- 1/4000s 高速快门，相比 FE（1/1000s）大幅提升
- TTL 闪光灯支持
- Jie's Lens 自用机型，实战验证
- 比 F3 更轻更便宜，性价比极高

**缺点**
- 电子快门依赖电池，没电只有 1/250s 机械备份
- 不如 FM2 全机械皮实（FM2 无电池也能全速工作）
- 电子元器件老化风险

## 价格参考（2026）
~800-1500 元

## 适合谁 / 不适合谁

**适合**：想要 A 档便利的入门/进阶用户、尼康 F 卡口镜头用户、追求轻便的日常拍摄者

**不适合**：必须全机械无需电池的用户（选 FM2）、需要顶级做工的人（选 F3）
