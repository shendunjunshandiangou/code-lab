# 禄来 3.5F

> [!image] 禄来 3.5F 外观参考
> 点击查看 [Rolleiflex 3.5F TLR camera](https://www.google.com/search?tbm=isch&q=Rolleiflex%203.5F%20TLR%20camera)

## 一句话定位
双反断层领先——轻便优雅的腰平取景体验

## 基本参数
| 参数 | 值 |
|------|-----|
| 发布时间 | 1958-1960 |
| 镜头 | 固定 75mm f/3.5 Planar / Xenotar |
| 快门类型 | 机械 |
| 曝光模式 | M |
| 对焦方式 | 手动（腰平取景器） |
| 电池 | 不需要 |

## 优点 / 缺点

**优点**
- 张有伯称之为"双反领域的断层领先"
- 轻便——在中画幅里算相当便携
- 优雅的腰平取景体验，与单反取景方式完全不同
- Planar 镜头画质优秀
- 设计经典，收藏价值高

**缺点**
- 昂贵（7000-10000+ 元）
- 固定镜头，无法更换
- 双反取景方式需要适应（左右颠倒）
- 维修师傅稀少，专业保养不易找
- 6×6 方形画幅，不是每个人都能驾驭

## 价格参考（2026）
~7000-12000 元

## 适合谁 / 不适合谁

**适合**：追求极致双反体验的玩家、腰平取景爱好者、收藏与实用兼顾的用户

**不适合**：预算有限的用户、需要换镜头的场景、不喜欢方形构图的人
