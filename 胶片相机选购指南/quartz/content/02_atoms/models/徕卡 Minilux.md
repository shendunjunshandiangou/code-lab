# 徕卡 Minilux

> [!image] 徕卡 Minilux 外观参考
> 点击查看 [Leica Minilux camera](https://www.google.com/search?tbm=isch&q=Leica%20Minilux%20camera)

## 一句话定位
钛合金颜值担当——萌大统领说"实际拍得少"。

## 基本参数

| 参数 | 值 |
|------|-----|
| 发布时间 | 1995 |
| 镜头 | 40mm f/2.4 Summarit 定焦 |
| 快门类型 | 电子 |
| 曝光模式 | 全自动 |
| 对焦方式 | 自动对焦 |
| 电池 | 需要 |
| 机身 | 钛合金 |

## 优点
- **钛合金机身**，设计出众——PS 机里的颜值天花板
- **Summarit 镜头**：徕卡血统，光学素质优秀
- 徕卡品牌光环——拿着有面子
- 小巧紧凑，口袋机的便携性

## 缺点
- 萌大统领亲述："钛合金、颜值高，但实际拍得少"——买回去吃灰的风险不小
- **臭名昭著的 E02 错误**：排线断裂是通病，维修费昂贵且零件难找
- 用 premium 的价格买了一台可能随时挂掉的 90 年代电子设备
- 张有伯也提醒：高端 PS 机（含 Minilux）断排线、液晶缺笔画是常见问题

## 价格参考（2026）
- 约 5,000-8,000 元

## 适合谁
- 徕卡收藏爱好者
- 愿意为颜值和品牌溢价买单的人
- 对 E02 风险有心理准备、能接受维修成本的人

## 不适合谁
- 把相机当工具、追求可靠性的人——选更皮实的机器
- 预算敏感的实用派——同价位有更可靠的选择
- 追求"高使用率"的人——萌大统领都说了"拍得少"
