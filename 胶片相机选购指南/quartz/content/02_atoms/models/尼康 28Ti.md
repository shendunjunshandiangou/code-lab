# 尼康 28Ti

> [!image] 尼康 28Ti 外观参考
> 点击查看 [Nikon 28Ti camera](https://www.google.com/search?tbm=isch&q=Nikon%2028Ti%20camera)

## 一句话定位
28mm 口袋机的画质最优解，但操作极其难受。

## 基本参数

| 参数 | 值 |
|------|-----|
| 发布时间 | 1994 |
| 镜头 | 28mm f/2.8，六片四组带非球面 |
| 快门类型 | 电子 |
| 对焦方式 | 自动 |
| 电池 | 需要 |
| 重量 | 约 310g |

## 优点
- 28mm 镜头变形和暗角控制优秀
- 28mm 口袋机中画质最强
- 精工设计的指针表盘——独特的视觉特征（但也容易出问题）

## 缺点
- **操作体验极差**：机身大、握持硌手、镜头偏右、开关旋钮太小需要用指甲抠
- 指针表盘"华而不实，还容易出问题"
- 郝习惯原话："操作非常难受，拍摄体验不好，但是 28mm 口袋机的画质最优解"

## 价格参考（2026）
- 二手市场：约 5,000-8,000 元

## 适合谁
- 坚持 28mm 焦段、追求在这个焦段下最好的 PS 机画质
- 能忍受不太友好的操作手感
