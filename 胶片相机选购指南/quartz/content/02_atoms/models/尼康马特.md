# 尼康马特

> [!image] 尼康马特 外观参考
> 点击查看 [Nikkormat camera](https://www.google.com/search?tbm=isch&q=Nikkormat%20camera)

## 一句话定位
几百块入尼康门——扎实可靠的入门机械单反

## 基本参数
| 参数 | 值 |
|------|-----|
| 发布时间 | 1965（FT，后续有 FTn、FT2、FT3） |
| 镜头 | Nikon F 卡口 |
| 快门类型 | 机械（最高 1/1000s） |
| 曝光模式 | M |
| 对焦方式 | 手动 |
| 电池 | 测光需要（机械快门不需要） |

## 优点 / 缺点

**优点**
- 极便宜，几百元就能入手
- Nikon F 卡口，和 F3/FM2/FE 共用镜头生态
- 机械扎实，用料远超现代入门机
- 张有伯推荐千元以下入手
- 全机械机身，无电池也能全速拍摄

**缺点**
- 设计老旧，操作速度慢
- 快门速度有限（最高 1/1000s）
- 比后续机型更重
- 测光表可能不准或已失效
- 过片手感不如高级机型

## 价格参考（2026）
~400-800 元

## 适合谁 / 不适合谁

**适合**：预算极低但想用尼康 F 卡口的用户、F 口镜头用户需要备用机身、学生党

**不适合**：追求轻便和操作效率的人、需要高速快门的场景、不喜欢老式操作逻辑的人
