# 康泰时 T2

> [!image] 康泰时 T2 外观参考
> 点击查看 [Contax T2 camera](https://www.google.com/search?tbm=isch&q=Contax%20T2%20camera)

## 一句话定位
钛金属蔡司 PS 机，T3 的前代，对焦逻辑拖后腿。

## 基本参数

| 参数 | 值 |
|------|-----|
| 发布时间 | 1990 |
| 镜头 | 38mm f/2.8 Sonnar + T* 镀膜 |
| 快门类型 | 电子 |
| 对焦方式 | 自动（半按锁定距离 → 按下后镜头再对焦 → 快门曝光） |
| 电池 | 需要 |
| 重量 | 约 300g |

## 优点
- 钛金属机身，蔡司镜头，做工一流
- 成像素质优秀，Sonnar 镜头 + T* 镀膜组合
- 在高端 PS 机中相对"没那么贵"（比 T3 便宜不少）

## 缺点
- **对焦流程极蠢**：半按快门锁定距离 → 按下快门后镜头再对焦 → 然后才打开快门曝光——整个过程增加了不可忽略的快门迟滞，且无法修改
- 除此之外没什么大坑
- 作为 30+ 年的电子设备，故障风险不可忽视

## 价格参考（2026）
- 二手市场：约 4,000-7,000 元

## 适合谁
- 想要蔡司镜头 + 钛机身的 PS 机、能接受快门迟滞的人
- T3 太重预算但不想放弃蔡司体验的人
