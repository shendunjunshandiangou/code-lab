# 雅西卡 35 GX

> [!image] 雅西卡 35GX 外观参考
> 点击查看 [Yashica Electro 35 GX camera](https://www.google.com/search?tbm=isch&q=Yashica%20Electro%2035%20GX%20camera)

## 一句话定位
千元内最夯——40mm f/1.7 大光圈旁轴。

## 基本参数

| 参数 | 值 |
|------|-----|
| 发布时间 | 1975 |
| 镜头 | 固定 40mm f/1.7 |
| 快门类型 | 电子快门 |
| 曝光模式 | 自动曝光 + 手动控制 |
| 对焦方式 | 旁轴黄斑手动对焦 |
| 电池 | 需要 |
| 外观 | 经典复古旁轴造型 |

## 优点
- **张有伯千元以下第一推荐**：自动曝光 + 大光圈 + 复古颜值，约 600 元
- **f/1.7 大光圈**：在旁轴固定镜头机型中少见的大光圈，暗光可用性高
- **40mm 焦段**：介于 35mm 和 50mm 之间，街拍和人像兼顾
- **颜值在线**：经典旁轴外观，颜值党友好

## 缺点
- **电子元件老化**：50 年历史的电子产品，测光和快门可能不准
- **黄斑可能变暗**：老旁轴的通病，对焦体验打折扣
- **固定镜头**：不能换镜头，焦段自由度有限

## 价格参考（2026）
- 二手市场：约 600 元

## 适合谁
- 千元以下预算、想体验旁轴的新手
- 喜欢固定镜头、简单操作的玩家
- 追求性价比的颜值党

## 不适合谁
- 担心老电子设备可靠性的人
- 想要可换镜头系统的人
