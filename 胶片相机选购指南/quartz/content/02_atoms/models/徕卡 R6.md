# 徕卡 R6 / R6.2

> [!image] 徕卡 R6 外观参考
> 点击查看 [Leica R6 camera](https://www.google.com/search?tbm=isch&q=Leica%20R6%20camera)

## 一句话定位
徕卡 R 系列中机械感最强的型号，和美能达合作开发的产物。

## 基本参数

| 参数 | 值 |
|------|-----|
| 发布时间 | 1988（R6）/ 1992（R6.2） |
| 快门类型 | 全机械快门 |
| 曝光模式 | M（全手动） |
| 卡口 | 徕卡 R 卡口 |
| 对焦方式 | 手动 |
| 电池 | 不需要（或仅供电测光） |

## 优点
- **机械感最强**：徕卡 R 系列中最手动、最机械的型号
- 纯机械快门，无需电池
- 徕卡 R 镜头素质顶级

## 缺点
- **R 卡口限制大**：法兰距是所有 135 单反中最长的，无法转接其他卡口的镜头
- 副厂镜头少，原厂 R 镜头贵
- 维修极度困难且昂贵
- 性价比低（不比其他日系机械单反更可靠）

## 价格参考（2026）
- 二手市场：约 5,000-10,000 元

## 适合谁
- 徕卡收藏者、已有 R 镜头的用户

## 不适合谁
- 追求实用性的普通摄影爱好者（日系机械单反性价比更高）
