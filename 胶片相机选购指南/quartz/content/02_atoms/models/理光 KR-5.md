# 理光 KR-5

> [!image] 理光 KR-5 外观参考
> 点击查看 [Ricoh KR-5 camera](https://www.google.com/search?tbm=isch&q=Ricoh%20KR-5%20camera)

## 一句话定位
便宜实用的宾得 K 卡口入门机，胜在配件通用性。

## 基本参数

| 参数 | 值 |
|------|-----|
| 发布时间 | 1970 年代 |
| 快门类型 | 全机械快门 |
| 曝光模式 | M（全手动） |
| 卡口 | 宾得 K 卡口 |
| 对焦方式 | 手动 |
| 电池 | 不需要（或仅供电测光） |

## 优点
- **K 卡口生态**：可以使用宾得 K 卡口的所有镜头，镜头选择极多
- **便宜实用**：性价比高
- 功能中规中矩，够用

## 缺点
- 做工一般，不如宾得自家机身
- 功能基础，没有亮点
- 二手存量不多

## 价格参考（2026）
- 二手市场：约 300-600 元

## 适合谁
- 预算有限、想用 K 卡口系统的人

## 不适合谁
- 追求品质做工的人（直接买宾得）
