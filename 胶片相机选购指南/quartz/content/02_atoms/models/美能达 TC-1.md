# 美能达 TC-1

> [!image] 美能达 TC-1 外观参考
> 点击查看 [Minolta TC-1 camera](https://www.google.com/search?tbm=isch&q=Minolta%20TC-1%20camera)

## 一句话定位
为极致小巧放弃太多——画质一般、噪音大、修不了。

## 基本参数

| 参数 | 值 |
|------|-----|
| 发布时间 | 1996 |
| 镜头 | 28mm f/3.5 G-Rokkor |
| 快门类型 | 电子 |
| 对焦方式 | 自动 |
| 电池 | 需要 |
| 重量 | 约 185g |

## 优点
- **极致小巧**：所有 PS 机里最小的之一
- Sport 按键：按一下锁定快门速度，实现对焦和曝光分开控制——唯一的亮点设计
- G-Rokkor 镜头名号响亮

## 缺点
- 全开光圈暗角已经非常严重
- 锐度和解析力不如 GR1s
- **电机噪音极大**：为了做到极致小巧，把对焦/过片/回卷全交给一个电机——马力大、声音大、特别吵
- **排线极易断裂**：过于紧凑的设计让镜头排线余量不足，断了之后很多修机师傅都不愿意修
- 郝习惯评价："可以说这个机器为了体型放弃太多东西了"

## 价格参考（2026）
- 二手市场：约 5,000-8,000 元（炒作严重）

## 适合谁
- 极致便携追求者，接受画质和可靠性的妥协
- 收藏用途 > 实用用途

## 不适合谁
- 追求实用性和画质的人——GR1s 或 Klasse W 是更好的选择
