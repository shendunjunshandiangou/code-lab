# 奥林巴斯 Pen EE-2

> [!image] 奥林巴斯 Pen EE-2 外观参考
> 点击查看 [Olympus Pen EE-2 camera](https://www.google.com/search?tbm=isch&q=Olympus%20Pen%20EE-2%20camera)

## 一句话定位
一卷拍 72 张的半格口袋机，每张成本减半。

## 基本参数

| 参数 | 值 |
|------|-----|
| 发布时间 | 1968 |
| 镜头 | 28mm f/3.5（半格等效约 40mm） |
| 快门类型 | 电子（硒光测光 — 免电池） |
| 对焦方式 | 固定焦距（泛焦） |
| 曝光模式 | 全自动 |
| 电池 | 不需要 |
| 画幅 | 半格（135 全格的 1/2） |

## 半格意味着什么
使用普通 135 胶卷，每张照片只消耗正常画面的一半底片面积。一卷 36 张的胶卷可以拍 72 张——每张胶卷成本从约 2 元降到约 1 元。代价是底片面积减半，放大后颗粒更粗、细节更少。

## 优点
- 72 张/卷——不需要省着按快门，适合旅行/大量记录
- 硒光测光免电池，全自动曝光
- 体积小巧，口袋友好

## 缺点
- 半格画质打折扣——不适合追求高画质的场景
- 泛焦镜头——无法精确对焦

## 价格参考（2026）
- 二手市场：约 500-800 元

## 适合谁
- 想大量体验胶片、不在意每张最高画质的人
- 旅行中想"随便拍不心疼"的人
