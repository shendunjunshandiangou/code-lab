# 美能达 α-7

> [!image] 美能达 α-7 外观参考

## 一句话定位
807Si 的下一代，蜂窝测光 + 点测直读，AF 时代的末代精品。

## 基本参数

| 参数 | 值 |
|------|-----|
| 发布时间 | 2000 |
| 快门类型 | 电子快门 |
| 曝光模式 | P + A + S + M |
| 卡口 | 美能达 A 卡口 |
| 对焦方式 | 自动对焦 |
| 电池 | 需要 |

## 优点
- **十四点蜂窝测光可视化**：菜单中可以看到每个蜂窝点的欠曝/过曝档数——测光系统的极致透明化
- AF 实用成熟
- 相比 807Si 更完善的旗舰体验

## 缺点
- 价格比 807Si 贵不少（约 900 元 vs 200 元）
- 电子化程度高，故障风险存在

## 价格参考（2026）
- 二手市场：约 900 元

## 适合谁
- 想要美能达 AF 系统完整体验的人

## 不适合谁
- 极致性价比追求者（807Si 更值）
