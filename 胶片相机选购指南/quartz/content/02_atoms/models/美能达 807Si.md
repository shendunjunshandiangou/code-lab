# 美能达 807Si

> [!image] 美能达 807Si 外观参考
> 点击查看 [Minolta 807Si camera](https://www.google.com/search?tbm=isch&q=Minolta%20807Si%20camera)

## 一句话定位
被遗忘的专业级自动单反，功能对标旗舰，价格低到不像话。

## 基本参数

| 参数 | 值 |
|------|-----|
| 发布时间 | 1990 年代 |
| 快门类型 | 电子 |
| 曝光模式 | P + A + S + M |
| 对焦方式 | 自动对焦 |
| 过片 | 自动（马达驱动） |
| 闪光灯 | 内置 |
| 卡口 | 美能达 A 卡口 |

## 优点
- 全自动操作逻辑和现代数码单反几乎一样——自动对焦、自动过片、全模式曝光
- P/A/S/M 四种曝光模式全部支持
- 因外观不够复古，在二手市场几乎无人问津 → 价格极低
- 美能达 A 卡口镜头群丰富且便宜

## 缺点
- 外观"现代"，没有传统胶片机的复古感——很多人买胶片机就是为了那个机械美学
- 电子化程度极高，任何一块电路板坏了就是报废
- 自动对焦速度不如现代相机

## 价格参考（2026）
- 二手市场：约 200-500 元

## 适合谁
- 数码用户想要一台"最接近现代单反使用体验"的胶片机
- 实用主义优先，不在意复古外观的人
