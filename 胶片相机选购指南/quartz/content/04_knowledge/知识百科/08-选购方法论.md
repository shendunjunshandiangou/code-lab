# 选购方法论

> 前面讲了相机类型、品牌、胶卷、操作。这篇把所有这些串成一个"怎么选"的通用框架。
> 如果你的情况比较典型，可以直接跳去选购决策篇看具体推荐。

## 五步选机法

```
① 确定场景 → ② 确定预算 → ③ 选择类型 → ④ 选择品牌/卡口 → ⑤ 选择具体机型
```

每一步都在缩小范围。

### 第一步：确定你的使用场景

问自己：**我主要在什么时候、什么地方拍照？**

| 场景 | 最合适的相机类型 | 原因 |
|------|----------------|------|
| 日常随手记录、出门揣口袋就拍 | PS 机（傻瓜机） | 最小、最方便、自动 |
| 旅行、边走边拍、不需要太快节奏 | 电子快门单反（有 A 档） | 可换镜头适应不同场景 |
| 街头人文、安静低调 | 旁轴或小型 PS 机 | 安静、不引人注意 |
| 真正想学摄影、理解曝光 | 全机械单反 | 每一步都自己控制 |
| 啥也不懂、就想试试看 | 一次性相机 | 最低门槛体验装 |

### 第二步：确定预算

预算不只是机身——**要算上胶卷和冲扫的持续花费**。

| 总预算（含前3个月） | 相机预算 | 能买什么类型 |
|---------------------|----------|-------------|
| ~800 元 | ~500 元 | 一次性相机 / 百元 PS 机 / 自动单反（F-601、807Si 等） |
| ~2500 元 | ~1500 元 | 好 PS 机 / 电子快门单反 + 标头 / 入门全机械单反 |
| ~6000 元 | ~4000 元 | 经典机械单反 + 多支镜头 / 高端 PS 机 / 旁轴入门 |

### 第三步：选择相机类型

科普入门和相机类型详解已经覆盖了每种类型的细节。这里给一个快速判断：

- 你希望相机帮你做大部分决定？ → PS 机 或 全自动单反
- 你希望自己控制曝光但保留一点自动？ → 电子快门单反（有 A 档）
- 你希望一切都自己掌控？ → 全机械单反
- 你对器材重量极其敏感？ → PS 机 或 半格机

### 第四步：选择品牌/卡口

品牌系统速查已详解了六个品牌。核心逻辑：

**卡口锁定你的镜头生态**。定品牌之前，先问问自己：
- 我以后想在这个系统里升级机身吗？
- 这个卡口的镜头多不多？贵不贵？

尼康 F 卡口的镜头最多最便宜，是最稳妥的选择。美能达 MC/MD 和宾得 K 的性价比也很高。奥林巴斯 OM 以精巧见长。

### 第五步：选择具体机型

到了这一步，范围已经很窄了。在同一品牌同一类型内选择，差异主要在"你用不用的到那些多出来的功能"：

- 需要更高的快门速度（拍运动/大光圈在阳光下）→ 选 1/2000s 以上的机型
- 需要 P 档全自动 → 选带 P 档的电子机型
- 需要纯机械可靠性 → 选全机械机型
- 需要轻便 → 选最小巧的那个型号

## 平替策略

热门机型 = 高溢价。理解平替逻辑可以帮你用一半的钱买到几乎一样的体验。

**平替的核心思路**：
1. **同品牌、同卡口、不同型号**：FM2 → FE（同为尼康 F 口，FE 有 A 档且便宜 2/3）
2. **同类型、不同品牌**：mju-2 → 宾得 Espio Mini（同为全自动 PS 机，对焦更快）
3. **同镜头素质、不同机身名气**：雅西卡 T3 Super（蔡司 T\* 镜头）vs 康泰时 T2（同样的蔡司 T\* 镜头，贵 4-5 倍）

## 一步到位 vs 先试水

这是选购中最常见的观点冲突。两种逻辑各有道理：

### 一步到位（支持方）
- 买个贵的反正能出二手，胶片相机不像数码那样贬值——等于不要钱
- 一步到位避免了"先买便宜的 → 不满意 → 再买 → 再升级 → 总花费反而更多"
- 好的机器手感、可靠性、体验确实更好

### 先试水（支持方）
- 在拍到 20-30 卷之前，你很难真正知道自己喜欢什么类型的相机
- 几百块的入门机 + 几十卷胶卷的拍摄经验 > 一万块的旗舰机 + 没拍几卷
- 第一台相机的作用不是"买到对的"，而是"通过使用来发现自己真正想要什么"

### 折中建议
- 如果预算有限（< 2000 元），先试水——买一台电子快门单反或平价 PS 机，拍到 20 卷再决定
- 如果预算宽裕（4000+），你可以一步到位——但要记住"最好的相机是愿意带出门的相机"这个原则
- **不要为第一台相机花掉你全部的胶片预算**——留下一半钱买胶卷。拍了 20 卷的 500 块相机比拍了 2 卷的 5000 块相机更能帮你进步
