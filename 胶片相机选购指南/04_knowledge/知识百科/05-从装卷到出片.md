# 从装卷到出片：胶片拍摄全流程

> 这篇讲实际怎么操作——从装卷到收到扫描文件的每一步。

## 一、装卷

装卷是新手最容易翻车的步骤。片头没钩牢、齿孔没咬合 → 拍了一大半才发现胶卷根本没走 → 一卷空白，浪费冲扫费 + 你的时间。

### 手动单反装卷步骤

1. 拉起回卷手柄，打开后盖
2. 胶卷放入左侧胶卷仓，按下回卷手柄固定
3. 拉出片头，将片头插入右侧卷片轴的槽中
4. 用拇指轻按胶卷使其平整贴合，慢慢转动过片杆确认齿孔已咬合（两边齿孔都套上了）
5. 合上后盖
6. **空拍 2 张**——前 2 张因装卷时已曝光，不能用。计数器从第 3 张开始
7. **观察回卷手柄**：过片时回卷手柄如果跟着转动 → ✅ 装卷成功。如果不转 → ❌ 片头脱钩，需要重新装

### 傻瓜机/自动相机装卷步骤

大部分自动相机会在合上后盖后自动上卷到第一张——你只需要把片头拉到指定标记处，合盖，相机会自动处理。但还是要观察液晶屏上的计数器是否在走动。

## 二、拍摄

### 测光

**内置测光表**：取景器里通常有指针或 LED 指示灯。指针居中或绿灯亮 = 曝光正确。

**内置测光可能不准**：30-40 年的老相机了。测光表老化、替代电池电压不同，都可能导致测光偏差。如果你的第一卷出现了整体过曝/欠曝——很可能是测光表不准。

**手机测光**：用免费的 Light Meter APP。操作很简单：ISO 设为胶卷感光度 → APP 给出光圈+快门组合 → 在相机上设置。更可靠，推荐新手用。

**晴天 16 法则**（无测光工具时的救命稻草）：
- 大晴天 → f/16，快门 1/ISO
- 白云朵朵 → f/11
- 阴天 → f/8
- 乌云压顶 → f/5.6

### 曝光铁律：宁过勿欠

拍负片时、不确定曝光时：**宁可过曝一点，不要欠曝。** 负片过曝的细节可以救，欠曝的暗部就是一片纯黑——没有信息就是没有信息。

实际操作：当你用手机 APP 测光得到 1/180s，相机上最接近的两个档位是 1/125s 和 1/250s → 选 1/125s（多给半档）。

### 曝光补偿：你天天在用的功能

用 A 档或 P 拍时，相机自动算快门。但相机会被"骗"——大片白雪、逆光人像这些场景，相机会把白色拍成灰色、把逆光的人拍成剪影。**曝光补偿**就是你手动微调相机的判断，和数码相机上调 ±EV 一模一样。

胶片机上三种操作方式：

| 相机类型 | 怎么补偿 |
|----------|----------|
| 有专用旋钮（如奥林巴斯 OM、佳能 AE-1P） | 拨 `+1 +2 0 -1 -2` 旋钮 |
| 只有 ASA 旋钮的老相机 | **调 ASA**：调低 = 过曝，调高 = 欠曝（400 的卷设 320 = 过曝半档） |
| 全机械 M 档 | 直接换快门速度 |

> 老玩家默认就把 ASA 调低半档（如 400 设 320），等于全程"宁过勿欠"——能获得更干净的高光和更饱满的暗部。详见 [[曝光补偿]]。

### 安全快门

手持拍摄时避免手抖糊片的底线：快门速度 ≥ 1/镜头焦距。50mm 镜头 → 不慢于 1/60s。28mm 镜头 → 不慢于 1/30s。

## 三、回卷

拍完一卷后，相机过片杆转到阻力变大（拍到底了）→ **不要硬转硬掰**——会撕裂齿孔！

1. 按下相机底部的回卷释放小按钮
2. 打开回卷手柄，顺时针旋转
3. 转着转着突然阻力消失（片头从卷片轴脱离）→ 再转几圈确保完全卷回暗盒
4. 打开后盖，取出胶卷

**绝不中途打开后盖**：拍完前开了后盖 → 已拍照片全白。如果后盖被不小心打开了，立刻合上——可能只毁了最近几张，前面的因为卷在收片轴上可能还保得住。

## 四、冲扫

把拍完的胶卷寄给淘宝冲扫店——但**不是随便搜个销量高的就行**。不同店扫同一卷胶卷，出来的颜色差异很大。

### 选店要素

| 维度 | 怎么看 |
|------|--------|
| SP3000 调色风格 | 看店家样片——偏青偏品 vs 偏暖偏黄，选你喜欢的 |
| 底片回邮速度 | 3-5 天 vs 2 周以上 |
| 划痕率/除尘 | 差评区找信号，干净 = 重视除尘 |
| 客服响应 | 售前问一句看态度 |

常见连锁店风格（仅供参考）：画英雄（SP3000 稳、偏日系）、菲林日记馆（偏暖复古）、DustyFilm（划痕少工艺细）。

### 扫描仪选择

| 扫描仪 | 价格 | 风格 | 适合 |
|--------|------|------|------|
| 富士 SP3000 | 15-25 元/卷 | 浓郁偏日系，"胶片味"来源 | 新手首选 |
| 诺日士 | 20-35 元/卷 | 中规中矩，还原度高 | 想要准确色彩 |
| 哈苏 X5 | 35-80+ 元/卷 | 细腻、动态范围大 | 重要照片、放大输出 |

### 第一次送冲扫的备注模板

```
1. 胶卷类型：彩色负片 C-41
2. 扫描仪：SP3000 标准套餐
3. 底片回邮：需要（用硬质底片袋）
4. 不剪断底片：是（整卷保留方便以后翻拍）
5. 分辨率：标准（约 1800×1200，够社交用）
6. 划痕处理：请尽量除尘和修复
```

详见 [[冲扫店选择]]。

## 五、检查底片

底片是唯一能验证冲扫质量的凭据：
- 看底片边上文字是否完整显影（如 "KODAK GOLD 200"）——检查冲洗质量
- 看片基颜色是否正常（应为棕黄色，不应发蓝或发黑）
- 检查有没有划痕、灰尘
- 好的冲扫 = 干净的底片 + 饱满的色彩
