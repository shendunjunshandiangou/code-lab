# 新手避雷手册

> 前人踩过的坑，你不需要再踩一遍。这篇是从几十个 UP 主的翻车经验中提炼出来的。

## PS 机炒作的黑名单

以下机型被严重炒作，**新手慎重购买**（不是说它们不好，而是它们的价格远高于它们的实际价值）：

| 型号 | 被炒到多少 | 实际合理价 | 炒高的原因 |
|------|-----------|-----------|-----------|
| 康泰时 T3 | 10000-15000+ | ~6000-8000 | 明星同款 + 钛金属 + 精美做工 |
| 康泰时 T2 | 4000-7000 | ~2000-3500 | T3 的小弟，跟着被炒 |
| 奥林巴斯 mju-2 | 2000-3000 | ~800-1200 | 滑盖设计好看、存世量大被堆高 |
| 美能达 TC-1 | 5000-8000 | ~2000-3000 | 被捧为"最神口袋机"，但画质不如 GR1s |
| 尼康 35Ti/28Ti | 5000-8000 | ~3000-5000 | 精工指针表盘好看，但操作体验极差 |

**替代思路**：别死盯着一台热门机型，学会"平替"逻辑——同焦段、同档光圈、不同品牌，成像差距远小于价格差距。

- T3 平替 → 雅西卡 T3 Super（蔡司 T\* 镜头，~1000 元）
- mju-2 平替 → 奥林巴斯 mju-1（成像差异不大，~500-800 元）
- 35Ti/28Ti 平替 → 富士 Klasse W（28mm 更均衡省心，~3000-5000 元）

## PS 机的电子故障

PS 机都是 20-30 年前的电子设备——电子组件的寿命是有限的。以下故障极其常见：

1. **排线断裂**（最常见）：镜头伸缩时排线反复弯折导致。几乎所有的 PS 机都会遇到。有些修得好，有些修不好
2. **液晶缺笔画/黑屏**：理光 GR1 系列最臭名昭著的问题，"常常修不好"
3. **自动对焦不准**：对焦马达老化——二手买的 PS 机最应该先测试的就是对焦
4. **闪光灯不工作**：电容老化，常见的电子问题之一

**怎么降低风险**：
- 买之前问问卖家排线是否换过、对焦是否准确
- 买有 7 天无理由的平台，拿到手第一卷测试所有功能
- 做好"这机器可能一年后就要进维修店"的心理准备
- 普通傻瓜机（几十几百块的）：坏了直接扔，修比买还贵

## 闲鱼/二手平台买相机的验机清单

收到相机后立刻检查以下内容：

- [ ] 快门各个档位是否正常工作（1s、1/60s、1/250s 各试一张——慢速明显拖长、快速清脆利落）
- [ ] 光圈环转动是否顺畅，叶片有无油污或卡涩
- [ ] 对焦环转动是否顺畅，有无卡阻
- [ ] 测光表是否工作（和手机 APP 同步测，看是否接近）
- [ ] 取景器内部是否干净（有无霉菌、灰尘、划痕）
- [ ] 电池仓是否干净（有没有电解液泄漏腐蚀的迹象——这是死刑）
- [ ] 快门帘幕是否完整（布帘有无破损、钢帘有无变形）
- [ ] 后盖海绵密封是否完好（漏光是老相机最常见的毛病）

## 过期卷的危害

不要看到"过期卷便宜"就心动。以下是你可能遇到的情况：

- 感光度从 ISO 400 降到实际只有 ISO 50-100——等于废了你一整个下午的拍摄
- 颜色偏得完全无法纠正（蓝到发紫、绿到发黄）
- 发霉受潮——霉菌孢子进入你的相机，以后的新卷也可能被交叉污染
- 花了 25 元冲扫费，等了 3 天，拿到 36 张无可挽救的废片

**替代方案**：新卷柯达金 200 才 40-50 元。多花 20 块买的不只是胶卷，是 36 张按下快门时的那份不白费。

## 最常见的翻车场景

| 翻车 | 结果 | 怎么避免 |
|------|------|----------|
| 装卷时片头没钩牢 | 拍了一卷全白 | 过片时观察回卷手柄是否跟着转 |
| 误开后盖 | 已拍照片全部或部分过曝 | 拍完前绝不碰后盖 |
| 拍到最后一格时用力过猛 | 齿孔撕裂，碎片留在机身内 | 感觉阻力变大就停止过片，开始回卷 |
| 老相机测光不准 | 整卷过曝或欠曝 | 用手机 APP 测光做比对；不确定时负片多给半档 |
| 胶卷屯了几年不拍 | 潜影退化、片基变形 | 拍完尽快送冲扫，不要囤"等以后一起洗" |
| 托运时胶卷被 X 光扫描 | 整卷雾化 | 胶卷随身携带、不要放托运行李 |

## 老相机电池问题

很多老相机用的电池型号已经停产（如 PX625 汞电池）。替代电池电压不同会导致测光不准——这是非常普遍的问题。

解决方案：
- 纯机械相机：不用关心，快门不靠电池
- 有测光表的相机：用锌空气电池（WeinCell）替代，但寿命短；或用 LR44 加转接环（电压偏高会导致测光偏暗）。最靠谱的方案是用手机 APP 测光

## 闪光同步速度——买相机时容易忽略的隐形约束

不同机身的"闪光同步速度"从 1/60s 到 1/250s 不等。超过这个速度用闪光灯，画面会出现"半幅黑片"——快门帘幕挡住了部分画面。

| 相机类型 | 典型同步速度 |
|----------|-------------|
| 老式全机械单反（如尼康 FM2） | 1/250s |
| 多数电子单反（如尼康 FE、佳能 AE-1） | 1/125s |
| 纯机械旁轴 | 1/60s 甚至更低 |
| 叶片快门（部分中画幅、禄来双反） | 全速同步 ✅ |

**选购提醒**：如果你打算白天户外用闪光灯补光（拍人像很常见），同步速度越高越好。纯机械旁轴在这块是个短板。详见 [[闪光灯基础]]。
