# 选购决策：帮你选相机

不管你是完全零基础还是已经看了知识百科，这个分步引导会帮你从模糊的"想买胶片相机"一步步收敛到具体机型。

## 你需要做的

四步走，每步缩小一圈范围：

1. **[Step 1 — 确定你的场景](Step1-确定你的场景.md)**：你主要在什么时候拍？
2. **[Step 2 — 确定你的预算](Step2-确定你的预算.md)**：机身 + 胶卷 + 冲扫一共准备花多少？
3. **[Step 3 — 推荐方案](Step3-推荐方案.md)**：基于前两步，给出 2-3 套具体推荐
4. **[Step 4 — 买了之后](Step4-买了之后.md)**：第一卷拍什么、去哪冲扫、怎么开始
