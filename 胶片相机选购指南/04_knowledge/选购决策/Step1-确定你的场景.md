# Step 1：确定你的使用场景

相机的类型取决于你想怎么用。先不要看具体型号，选你的场景：

## 场景 A：日常随手拍，记录生活

> *"我就是想有台相机，出门带着，看到什么拍什么，不折腾。"*

你的核心需求：**方便、便携、不操心**。

→ 推荐相机类型：**PS 机（傻瓜机）** 或 **半格机**

原因：零门槛操作，口袋尺寸。你不需要学曝光三要素也能拍出好看的胶片照片。

→ 如果还想更省心：**全自动单反**（如尼康 F-601、美能达 807Si）——操作和数码一样，自动对焦+自动曝光，但可以换镜头。

→ 去 [Step 2](Step2-确定你的预算.md)

---

## 场景 B：旅行/出门玩

> *"我经常旅行或周末出门，想带一台相机记录途中看到的。"*

核心需求：**便携、可应对多种光线和题材**。

→ 推荐相机类型：**PS 机（变焦款）** 或 **电子快门单反（有 A 档 + 可换镜头）**

原因：旅行场景下，变焦的灵活性比定焦有天然优势。或者带一台电子快门单反 + 一支标准变焦或两支定焦（如 35mm + 50mm），适应性强。

→ 去 [Step 2](Step2-确定你的预算.md)

---

## 场景 C：街头人文摄影

> *"我喜欢在城市里走走拍拍，捕捉人的状态，不是拍那种摆好姿势的。"*

核心需求：**小巧、安静、不引人注意**。

→ 推荐相机类型：**旁轴** 或 **小型 PS 机**

原因：旁轴是最经典的街拍工具——取景器和快门都安静、机身小、不引起路人注意。PS 机也是好选择——开机即拍，反应快。

→ 去 [Step 2](Step2-确定你的预算.md)

---

## 场景 D：想认真学摄影

> *"我不只是想拍胶片照片，我是想真正理解曝光、对焦、构图，提升我的摄影技术。"*

核心需求：**全部手动控制、可换镜头系统**。

→ 推荐相机类型：**电子快门单反（有 A 档）** 或 **全机械单反**

原因：手动操作 + 可换镜头 = 长期可玩性最高。从 A 档开始逐步过渡到全手动是最平滑的学习曲线。全机械单反是终极的学习工具——每一步都是你主动做的决定。

→ 去 [Step 2](Step2-确定你的预算.md)

---

## 场景总结

| 你属于  | 推荐类型          | 核心关键词        |
| ---- | ------------- | ------------ |
| 日常记录 | PS 机 / 半格机    | 方便、口袋、不操心    |
| 旅行   | PS 机变焦 / 电子单反 | 便携、灵活、可换镜头   |
| 街拍人文 | 旁轴 / 小型 PS 机  | 小、安静、快       |
| 想学摄影 | 电子单反 / 全机械单反  | 手动、可换镜头、长期可玩 |
