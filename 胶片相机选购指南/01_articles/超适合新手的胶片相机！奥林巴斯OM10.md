奥林巴斯 OM 系列单反以小巧精致著称，OM-1 是其中的经典标杆。但 OM-10 作为它的简化版，用不到一半的价格提供了接近的体验——300 元起步就能拿到一台有 A 档自动曝光的奥林巴斯单反。

OM-10 于 1978 年上市，比 OM-1 晚了几年。它保留了 OM 系列最核心的优点：机身小巧、做工精致、可以接入奥林巴斯优秀的 Zuiko 镜头群。和 OM-1 全机械不同，OM-10 默认是一台光圈优先自动相机——你设定光圈，它帮你决定快门速度。如果你想要全手动控制，需要额外购买一个 Manual Adapter 配件装在机身上。

对新手来说，A 档意味着更低的入门门槛。你不需要每次按快门都测光——设定好光圈，相机自动匹配快门，对焦，拍。配合奥林巴斯的 Zuiko 镜头——尤其是 50mm f/1.8 这颗廉价但素质极好的标头——花很少的钱就能感受到高质量胶片的成像。

不过要注意：OM-10 的价格正在涨。随着胶片入门热度上升，它在二手市场的价格已经从两三百慢慢往上走。Manual Adapter 也不像以前那么好找了。

## 核心知识点

| 维度 | 内容 |
|------|------|
| 年份 | 1978 年 |
| 快门 | 电子快门（需电池） |
| 曝光模式 | A 档（默认），加 Manual Adapter 可全手动 |
| 卡口 | OM 卡口，兼容 Zuiko 镜头群 |
| 价格 | 300-600 元（2024 年），在涨 |
| 适合 | 喜欢 OM 颜值、想要 A 档便利的新手 |

## Claude 的补充分析

OM-10 在你考虑奥林巴斯系统时是一个值得看的替代选项。OM-1 作为豆瓣四大神机之一，价格已经被推高不少。OM-10 的核心体验——OM 卡口的镜头素质、小巧机身、精致做工——几乎都在，但少付一半的钱。不过手动控制的缺失需要通过 Manual Adapter 补上，买之前要确认配件是否齐全。
