# 帮我选相机

你不需要先记住几十个型号。先确定你会在哪里拍、愿意投入多少，以及希望相机替你完成多少操作。

<div class="buying-flow">
  <a href="./04_knowledge/选购决策/Step1-确定你的场景">
    <span>STEP 01 · SCENE</span>
    <div>
      <h3>确定拍摄场景</h3>
      <p>日常、旅行、街拍、人像和摄影学习，对体积、对焦速度和镜头的要求完全不同。</p>
    </div>
  </a>
  <a href="./04_knowledge/选购决策/Step2-确定你的预算">
    <span>STEP 02 · BUDGET</span>
    <div>
      <h3>计算完整预算</h3>
      <p>不要只看机身。镜头、胶卷、冲扫、电池和第一次维修都应该被算进去。</p>
    </div>
  </a>
  <a href="./04_knowledge/知识百科/01-相机类型详解">
    <span>STEP 03 · CONTROL</span>
    <div>
      <h3>选择操作方式</h3>
      <p>全自动适合记录生活，自动曝光适合温和入门，全机械适合认真学习摄影。</p>
    </div>
  </a>
  <a href="./04_knowledge/选购决策/Step3-推荐方案">
    <span>STEP 04 · SHORTLIST</span>
    <div>
      <h3>查看推荐方案</h3>
      <p>把前三步组合起来，得到两到三套更接近你真实使用方式的相机方案。</p>
    </div>
  </a>
</div>
