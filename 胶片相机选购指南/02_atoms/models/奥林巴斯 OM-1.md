# 奥林巴斯 OM-1

> [!image] 奥林巴斯 OM-1 外观参考
> 点击查看 [Olympus OM-1 SLR camera](https://www.google.com/search?tbm=isch&q=Olympus%20OM-1%20SLR%20camera)

## 一句话定位
豆瓣四大神机中最优雅的一台——工业艺术品级的全机械单反。

## 基本参数

| 参数 | 值 |
|------|-----|
| 发布时间 | 1972 |
| 快门类型 | 全机械快门 |
| 曝光模式 | M（全手动） |
| 卡口 | OM 卡口 |
| 对焦方式 | 手动（裂像+微棱镜） |
| 电池 | 不需要（或仅供电测光表） |
| 重量 | 约 510g |

## 优点
- **精致小巧**：机身小巧精致，和当时那些厚重如砖的单反截然不同，更像一件工业艺术品
- **全机械可靠性**：没电池也能正常拍摄
- **取景器大而明亮**：OM 系列招牌特征
- **Zuiko 镜头群**：镜头素质极高，尤其是 50mm f/1.8 廉价标头
- 豆瓣四大神机之一，保有量和社区支持好

## 缺点
- 纯手动操作，没有 A 档——每张照片都需要手动设置一切
- 价格已被推高（网红效应）
- 测光表老化和替代电池问题
- OM 卡口镜头不如尼康 F 卡口庞大且便宜

## 价格参考（2026）
- 二手市场：约 800-1,500 元

## 适合谁
- 欣赏 OM 工业设计、追求精致机械感的人
- 喜欢轻便手动单反、注重机身设计和手感的人

## 不适合谁
- 预算极度有限、想要 A 档便利的人（选 OM-10）
