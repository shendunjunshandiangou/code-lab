# 玛米亚 7

> [!image] 玛米亚 7 外观参考
> 点击查看 [Mamiya 7 camera](https://www.google.com/search?tbm=isch&q=Mamiya%207%20camera)

## 一句话定位
画质顶级——但张有伯说"没有把玩价值"。

## 基本参数

| 参数 | 值 |
|------|-----|
| 发布时间 | 1995 |
| 画幅 | 6×7 |
| 镜头卡口 | Mamiya 7 卡口（可换镜头） |
| 快门类型 | 电子 |
| 曝光模式 | A / M |
| 对焦方式 | 旁轴黄斑手动对焦 |
| 电池 | 需要 |
| 特点 | 旁轴设计，6×7 画幅里最轻便的存在 |

## 优点
- **画质可能是中画幅里的天花板级**——6×7 底片面积 + 顶级镜头
- 旁轴结构使其在 6×7 画幅中格外轻便——比玛米亚 RB67 和宾得 6×7 轻得多
- 镜头素质一流，可换镜头
- 现代人体工学设计，握持和操作都很舒服

## 缺点
- **张有伯的核心吐槽**："没有把玩价值，没有硕大腰平取景器，没有清脆快门声"——操作体验让老玩家失望
- 非常昂贵，luxury 价位
- 电子快门，依赖电池——在野外不放心
- 旁轴对焦不是所有人都喜欢——黄斑对焦对戴眼镜的人不友好
- 虽然轻于其他 6×7，但仍然是中画幅的体积和重量

## 价格参考（2026）
- 约 10,000-20,000 元

## 适合谁
- 把画质放在第一位的严肃创作者
- 想要 6×7 顶级画质但不想背 RB67 那种"健身器材"的人

## 不适合谁
- 追求"拍照过程本身就很爽"的人——张有伯认为它没有把玩感
- 预算有限的人——这个价格可以买一整套勃朗尼卡 ETR 还有找
- 需要全机械可靠性的人——这东西靠电池吃饭
