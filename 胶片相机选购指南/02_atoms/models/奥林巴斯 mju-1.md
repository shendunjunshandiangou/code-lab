# 奥林巴斯 mju-1

> [!image] 奥林巴斯 mju-1 外观参考
> 点击查看 [Olympus mju 35mm camera](https://www.google.com/search?tbm=isch&q=Olympus%20mju%2035mm%20camera)

## 一句话定位
mju-2 最具性价比的平价替代，几百块体验奥林巴斯 PS 机。

## 基本参数

| 参数 | 值 |
|------|-----|
| 发布时间 | 1991 |
| 镜头 | 35mm f/3.5 定焦 |
| 快门类型 | 电子 |
| 对焦方式 | 自动 |
| 曝光模式 | 全自动 |
| 闪光灯 | 内置，默认自动（难以完全关闭） |
| 电池 | 需要 |
| 开机方式 | 滑盖式（合盖 = 关机 + 锁快门） |

## 优点
- 滑盖开机 + 合盖锁定快门，塞口袋从不误触
- 全自动操作——装上电池和胶卷就能用
- 出片"锐度和胶片感都挺不错"
- 价格只有 mju-2 的 1/3 到 1/4

## 缺点
- 闪光灯默认自动——光线不够时强制闪，晴天也可能被误触发
- 无法完全关闭闪光灯（除非手动遮挡）
- 镜头 f/3.5 比 mju-2 的 f/2.8 小一档
- 20+ 年老电子设备，存在故障风险

## 价格参考（2026）
- 二手市场：约 460-800 元（对比 mju-2 的 2000+ 元）

## 适合谁
- 预算有限但想体验奥林巴斯 PS 机的人
- 想找一台"不心疼随便用"的便携口袋机

## 不适合谁
- 讨厌闪光灯直打效果的人（mju-1 闪光灯无法完全关闭）
- 追求 f/2.8 更大光圈的人（选 mju-2）
