## 关键参数

| 参数 | 信息 |
|------|-----|
| 发布时间 | 1991 |
| 镜头 | 35mm f/3.5 定焦 |
| 快门类型 | 电子 |
| 对焦方式 | 自动 |
| 曝光模式 | 全自动 |
| 闪光灯 | 内置，默认自动（难以完全关闭） |
| 电池 | 需要 |
| 开机方式 | 滑盖式（合盖 = 关机 + 锁快门） |

## 为什么值得买

- 滑盖开机 + 合盖锁定快门，塞口袋从不误触
- 全自动操作——装上电池和胶卷就能用
- 出片"锐度和胶片感都挺不错"
- 价格只有 mju-2 的 1/3 到 1/4

## 需要注意什么

- 闪光灯默认自动——光线不够时强制闪，晴天也可能被误触发
- 无法完全关闭闪光灯（除非手动遮挡）
- 镜头 f/3.5 比 mju-2 的 f/2.8 小一档
- 20+ 年老电子设备，存在故障风险

## 适合谁

- 预算有限但想体验奥林巴斯 PS 机的人
- 想找一台"不心疼随便用"的便携口袋机

## 不适合谁

- 讨厌闪光灯直打效果的人（mju-1 闪光灯无法完全关闭）
- 追求 f/2.8 更大光圈的人（选 mju-2）

## 购买检查

1. 检查快门、测光、过片、回卷和计数器是否正常。
2. 检查取景器、镜头和机身内部是否存在霉、雾、划痕或腐蚀。
3. 检查电池仓是否漏液，电子功能是否稳定。
4. 检查后盖密封海绵和反光镜缓冲材料是否老化。
5. 优先选择能够提供实拍测试、维修记录或退换保障的卖家。
