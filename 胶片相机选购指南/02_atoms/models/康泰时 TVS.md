# 康泰时 TVS

> [!image] 康泰时 TVS 外观参考
> 点击查看 [Contax TVS camera](https://www.google.com/search?tbm=isch&q=Contax%20TVS%20camera)

## 一句话定位
比 T2/T3 便宜得多的康泰时变焦 PS 机。

## 基本参数

| 参数 | 值 |
|------|-----|
| 发布时间 | 1994（TVS I），后续有 TVS II / TVS III |
| 镜头 | 变焦（约 28-56mm f/3.5-6.5） |
| 快门类型 | 电子 |
| 对焦方式 | 自动 |
| 曝光模式 | 全自动 |
| 电池 | 需要 |

## 优点
- 变焦覆盖多焦段——旅行场景中比定焦灵活
- 比康泰时 T2/T3 便宜很多——同为康泰时但不在炒作焦点上
- 自动对焦、自动测光，上手体验接近

## 缺点
- 变焦头光圈偏小，暗光表现受限
- 作为 30 年老的电子设备仍有故障风险

## 价格参考（2026）
- 二手市场：约 1,500-3,000 元（远低于 T2/T3）

## 适合谁
- 想要康泰时体验但不想付 T2/T3 溢价的人
- 旅行中需要变焦灵活性的人

- [[康泰时 T3]] — 旗舰定焦版
