# 尼康 F2

> [!image] 尼康 F2 外观参考
> 点击查看 [Nikon F2 camera](https://www.google.com/search?tbm=isch&q=Nikon%20F2%20camera)

## 一句话定位
纯机械巅峰——和 F3 二选一的终极手动单反。

## 基本参数

| 参数 | 值 |
|------|-----|
| 发布时间 | 1971 |
| 快门类型 | 全机械焦平面快门 |
| 最高快门 | 1/2000s |
| 曝光模式 | M（全手动） |
| 卡口 | 尼康 F 卡口 |
| 对焦方式 | 手动 |
| 电池 | 不需要（电池只给测光顶供电） |
| 重量 | 约 730g（仅机身） |

## 优点
- **纯机械巅峰**：所有快门速度由机械控制，无需电池即可正常拍摄——张有伯称其为 F3 唯一的真正对手
- **做工顶级**：日本制造最高水准，全金属机身，堪称艺术品
- **可换取景器**：DE-1 眼平最轻便、DW-1 腰平适合低角度、DP 系列测光顶内置测光表
- **镜头兼容性**：兼容尼康 F 卡口所有手动镜头（Non-AI 到 AI-S），可与数码机身共享资源
- **耐用性传奇**：机械结构极其坚固，几十年后仍能正常使用

## 缺点
- **比 F3 重**：F 卡口金属镜头也重，整套出门不便
- **无 A 档**：纯手动曝光，每次都要手动设置快门和光圈
- **比 F3 更老更贵**：好成色的 F2 可能比 F3 更贵
- **无内置测光**：需配合测光顶、手机 APP 或外置测光表

## 价格参考（2026）
- 二手市场：约 2,500-4,000 元（视成色和配件）

## 适合谁
- 追求纯机械仪式感、想要"永远不会因没电而停摆"的相机的人
- 已有尼康 F 卡口镜头，想在胶片和数码之间共享资源的人
- 注重做工和机械手感的人

## 不适合谁
- 想要 A 档便利的人（选 F3）
- 介意重量、需要便携的人（选 FM2）
