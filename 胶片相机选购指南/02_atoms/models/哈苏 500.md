# 哈苏 500 系列

> [!image] 哈苏 500 外观参考
> 点击查看 [Hasselblad 500C camera](https://www.google.com/search?tbm=isch&q=Hasselblad%20500C%20camera)

## 一句话定位
相机界爱马仕——登上月球的模块化中画幅。

## 基本参数

| 参数 | 值 |
|------|-----|
| 发布时间 | 500C 1957 / 500C/M 1970 / 503CX 1988 |
| 画幅 | 6×6 方形（120 胶卷，12 张/卷） |
| 快门类型 | 镜间快门（全机械） |
| 曝光模式 | M（全手动） |
| 卡口 | 哈苏 V 卡口（卡尔蔡司镜头） |
| 对焦方式 | 手动 |
| 电池 | 不需要 |
| 特点 | 模块化：可换后背、取景器、镜头；腰平取景 |
| 重量 | 约 1,500g（机身 + 80mm 镜头 + 后背） |

## 优点
- **传奇品牌**：第一台登上月球的相机，摄影史上里程碑式的存在
- **6×6 方形构图**：独特的正方形成像比例，与 135 的 3:2 完全不同
- **全模块化**：后背、取景器、镜头全部可拆卸互换，系统扩展性极强
- **全机械可靠**：无需电池，纯机械镜间快门
- **腰平取景体验**：低头看取景器的方式，和眼平取景是完全不同的拍摄体验

## 缺点
- **极重极大**：老王说"越沉越大就越不好带，所以我用得最多的反而是最便宜的 FX-3"
- **极贵**：张有伯警告入门低于 10,000 只能买到最老的机身 + 最差的 C80 镜头，不值得
- **涨价凶猛**：张有伯称两年内涨了 5,000 元
- **学习成本高**：120 胶卷装卷、腰平取景左右颠倒、方形构图都需要适应

## 价格参考（2026）
- 基础配置：8,000 元起（最老机身 + C80 镜头，不推荐）
- 好配置：15,000 元以上
- 注意：两年内价格上涨约 5,000 元

## 适合谁
- 预算充足且体力好、愿意扛重器材的深度玩家
- 追求方形构图和腰平取景独特体验的人
- 对相机历史有情怀的收藏者

## 不适合谁
- 新手（太贵太重太难上手）
- 预算有限的玩家（选勃朗尼卡 ETR）
- 追求便携随身拍的人
