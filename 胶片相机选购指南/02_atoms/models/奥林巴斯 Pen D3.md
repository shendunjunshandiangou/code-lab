## 关键参数

| 参数 | 信息 |
|------|-----|
| 镜头 | 固定镜头 |
| 快门类型 | 电子（硒光测光 — 免电池） |
| 对焦方式 | 固定焦距（泛焦） |
| 曝光模式 | 全自动 |
| 唯一手动设置 | ISO（由胶卷决定） |
| 画幅 | 半格（135 全格的 1/2，72 张/卷） |

## 为什么值得买

- 操作极简——只需要把镜头上的 ISO 拧到所用胶卷的值
- 光线不足时取景框弹出一面小红旗阻止拍摄（防翻车设计）
- 硒光测光免电池
- 72 张/卷，实验成本低

## 需要注意什么

- 半格画质打折扣
- 泛焦无法精确对焦
- 硒光测光可能随年龄老化失效

## 适合谁

- 第一次玩胶片、想体验半格 72 张的快感
- 需要一台完全不用动脑的随身机

## 不适合谁

- 待补充。

## 购买检查

1. 测试快门、测光、光圈和对焦机构，确认动作稳定。
2. 装入废卷检查半格过片步距、计数器和回卷是否正确。
3. 检查镜头和取景器是否有霉、雾、划痕，画幅遮框是否完整。
4. 检查电池仓、后盖锁扣和密封材料是否老化。
5. 了解冲扫店是否支持半格扫描及相邻画面拆分。
6. 优先选择能够提供实拍样片或退换保障的卖家。
