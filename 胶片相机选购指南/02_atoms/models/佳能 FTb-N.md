# 佳能 FTb-N

> [!image] 佳能 FTb-N 外观参考
> 点击查看 [Canon FTb camera](https://www.google.com/search?tbm=isch&q=Canon%20FTb%20camera)

## 一句话定位
佳能 FD 卡口最便宜的纯机械入门机。

## 基本参数

| 参数 | 值 |
|------|-----|
| 发布时间 | 1973 |
| 快门类型 | 全机械快门 |
| 曝光模式 | M（全手动） |
| 卡口 | 佳能 FD 卡口 |
| 对焦方式 | 手动 |
| 电池 | 不需要（或仅供电测光） |

## 优点
- **FD 卡口入门**：可以用佳能 FD 系统的所有镜头
- 全机械，无需电池
- 价格低，适合预算紧张的 FD 入门者

## 缺点
- 功能基础，只有全手动
- 比 AE-1 系列老旧，测光表可能已失效
- FD 卡口已停产

## 价格参考（2026）
- 二手市场：约 400-800 元

## 适合谁
- 想用 FD 卡口但预算有限的人

## 不适合谁
- 想要自动曝光便利的人（选 AE-1 或 A-1）
