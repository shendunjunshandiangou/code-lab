# 海鸥 4A

> [!image] 海鸥 4A 外观参考
> 点击查看 [Seagull 4A TLR camera](https://www.google.com/search?tbm=isch&q=Seagull%204A%20TLR%20camera)

## 一句话定位
百元体验腰平取景——120 画幅最低门槛

## 基本参数
| 参数 | 值 |
|------|-----|
| 发布时间 | 1960 年代 |
| 镜头 | 固定镜头（可升级 Tessar 型镜头） |
| 快门类型 | 机械 |
| 曝光模式 | M |
| 对焦方式 | 手动（腰平取景器 TLR） |
| 电池 | 不需要 |

## 优点 / 缺点

**优点**
- 最便宜的 120 入门选择，几百元即可体验中画幅
- 腰平取景器——135 用户前所未有的取景体验
- 张有伯推荐千元以下入手
- 可自行更换 Tessar 型镜头提升画质
- 全机械，无需电池，可靠性高
- 双反造型经典，社交话题度高

**缺点**
- 国产做工品控参差不齐
- 原厂镜头素质一般
- 无测光，需外置测光表或阳光十六法则
- 可靠性取决于个体（买到好品相靠运气）
- 对焦屏较暗，不如禄来明亮

## 价格参考（2026）
~500-1000 元

## 适合谁 / 不适合谁

**适合**：想用最低成本体验 120 画幅的人、腰平取景好奇者、喜欢折腾的 DIY 玩家（换镜头）

**不适合**：追求极致画质的人、不想折腾检修的用户、对国产品控不放心的用户
