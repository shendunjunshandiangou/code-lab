# 美能达 SRT-101

> [!image] 美能达 SRT-101 外观参考
> 点击查看 [Minolta SRT-101 camera](https://www.google.com/search?tbm=isch&q=Minolta%20SRT-101%20camera)

## 一句话定位
400 元全机械入门王，便宜皮实不怕坏。

## 基本参数

| 参数 | 值 |
|------|-----|
| 发布时间 | 1966 |
| 快门类型 | 全机械快门 |
| 曝光模式 | M（全手动） |
| 卡口 | 美能达 MC/MD 卡口 |
| 对焦方式 | 手动 |
| 电池 | 不需要（或仅供电测光，LR44/WeinCell 纽扣电池） |

## 优点
- **全机械快门**：无需电池即可工作，结构简单耐用
- **便宜**：300-500 元即可入手，胶片入门最低门槛
- **MC/MD 镜头群**：选择多且不贵
- **皮实耐用**：纯粹机械结构，不容易坏

## 缺点
- **测光可能不准或失效**：老机器的测光表大概率有问题，买前需确认电池仓有没有漏液腐蚀
- **无任何自动辅助**：没有 A 档、没有对焦辅助
- **机身较重**
- **翻车风险**：最初几卷可能需要摸索真实状态（测光偏差、各档快门精度、是否漏光）

## 价格参考（2026）
- 二手市场：300-500 元（机身）

## 适合谁
- 预算极度有限、想最低成本试水胶片的人
- 愿意"边拍边学"手动测光的新手

## 不适合谁
- 想要 A 档便利的人（选 X-700）
- 已经确定长期玩胶片、预算更高的人（建议提到 1000-2000 元价位）
