# 佳能 F-1

> [!image] 佳能 F-1 外观参考
> 点击查看 [Canon F-1 SLR camera](https://www.google.com/search?tbm=isch&q=Canon%20F-1%20SLR%20camera)

## 一句话定位
佳能 FD 系统专业旗舰，混合快门 + 可换取景器。

## 基本参数

| 参数 | 值 |
|------|-----|
| 发布时间 | 1971（原版）/ 1981（New F-1） |
| 快门类型 | 混合快门（机械 + 电子） |
| 曝光模式 | 视取景器配置 |
| 卡口 | 佳能 FD 卡口 |
| 对焦方式 | 手动 |
| 电池 | 需要（部分功能依赖） |

## 优点
- **FD 系统集大成者**：代表佳能手动时代的最高水准
- **混合快门**：兼具机械可靠性和电子便利
- **可换取景器**：支持腰平取景，专业级配置
- 扎实的做工和用料

## 缺点
- 重量不轻，不便携
- FD 卡口现代转接受限

## 价格参考（2026）
- 二手市场：约 1,500-3,000 元

## 适合谁
- 喜欢佳能系统、追求 FD 时代巅峰体验的人

## 不适合谁
- 日常轻便携带需求的人
