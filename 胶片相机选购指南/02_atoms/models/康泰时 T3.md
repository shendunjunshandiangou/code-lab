# 康泰时 T3

> [!image] 康泰时 T3 外观参考
> 点击查看 [Contax T3 camera](https://www.google.com/search?tbm=isch&q=Contax%20T3%20camera)

## 一句话定位
35mm 口袋机的画质标杆，除了贵没大缺点。

## 基本参数

| 参数 | 值 |
|------|-----|
| 发布时间 | 2001 |
| 镜头 | 35mm f/2.8 Sonnar + T* 镀膜 |
| 快门类型 | 电子 |
| 对焦方式 | 自动，最近 35cm |
| 电池 | 需要（CR2） |
| 重量 | 约 230g |

## 优点
- Sonnar 镜头 + T* 镀膜：解析力和抗眩光的优秀平衡，微反差和高光处理是"胶片味"的关键来源
- 对焦果断、闪光灯设置带记忆功能
- 体积小，钛金属机身
- 二手流动性极强——三进三出，每次不仅没赔钱还小赚
- 郝习惯评价："不想折腾相机，就想买个机器踏踏实实拍胶卷，T3 还是值得推荐的"

## 缺点
- 价格极高：二手 1 万人民币以上，溢价严重
- 镜头偏右——手稍大，一开机镜头就怼手指
- AFL 对焦锁定需要长按一秒（"简直就是个笑话"）
- 性价比严重不符——花同样的钱可以买到多台功能更完整的相机

## 价格参考（2026）
- 二手市场：10,000-15,000+ 元

## 适合谁
- 已经用过不少相机、清楚自己需要什么的人
- 追求便携 + 顶级画质、愿意为这种"不可替代的特质"付费的人

## 不适合谁
- 新手——溢价太高，入门阶段不需要这级别的相机
- 预算敏感的人——同样的钱可以买多台不同风格的相机 + 大量胶卷
