# 康泰时 G2

> [!image] 康泰时 G2 外观参考
> 点击查看 [Contax G2 camera](https://www.google.com/search?tbm=isch&q=Contax%20G2%20camera)

## 一句话定位
自动对焦旁轴——老王拍人像的抓拍利器。

## 基本参数

| 参数 | 值 |
|------|-----|
| 发布时间 | 1994 |
| 快门类型 | 电子快门 |
| 曝光模式 | A + M + P |
| 卡口 | 康泰时 G 卡口（可换镜头） |
| 对焦方式 | 自动对焦 |
| 电池 | 需要 |
| 机身 | 钛合金 |

## 优点
- **自动对焦快速**：对焦快、抓拍快，配合闪光灯拍"特别 fashion"的照片非常顺手
- **可换镜头**：蔡司 G 系列镜头（G28、G35、G45、G90），光学素质顶级
- **颜值逼格高**：钛合金机身 + 旁轴设计，权志龙带动 Contax 全品牌涨价
- **老王的肖像利器**：解决了手动对焦拍人效率低的问题

## 缺点
- **昂贵**：机身 10,000 元以上，镜头另算
- **电子元件老化风险**：30 年前的电子产品，维修难度高
- **涨价中**：Contax 品牌因明星效应持续升温

## 价格参考（2026）
- 二手市场：10,000 元以上（机身）

## 适合谁
- 预算充足、想要自动对焦旁轴体验的人
- 经常拍人像、需要抓拍速度的人
- 追求颜值和逼格的玩家

## 不适合谁
- 预算有限的新手
- 担心电子老化风险的人
