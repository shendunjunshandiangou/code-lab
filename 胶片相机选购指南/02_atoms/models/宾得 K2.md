# 宾得 K2

> [!image] 宾得 K2 外观参考
> 点击查看 [Pentax K2 camera](https://www.google.com/search?tbm=isch&q=Pentax%20K2%20camera)

## 一句话定位
宾得 K 卡口电子化进阶选择，介于 MX 和 LX 之间。

## 基本参数

| 参数 | 值 |
|------|-----|
| 发布时间 | 1975 |
| 快门类型 | 电子快门 |
| 曝光模式 | A + M |
| 卡口 | 宾得 K 卡口 |
| 对焦方式 | 手动 |
| 电池 | 需要 |

## 优点
- **K 卡口生态**：镜头资源丰富且便宜
- **A 档自动曝光**：有自动功能的全金属机身
- 做工扎实，宾得品质

## 缺点
- 电子快门依赖电池
- 不如 MX 知名，存量较少

## 价格参考（2026）
- 二手市场：约 600-1,000 元

## 适合谁
- 想要 K 卡口 + A 档、又不喜欢 ME Super 按钮操作的人

## 不适合谁
- 追求全机械的人（选 MX 或 K1000）
