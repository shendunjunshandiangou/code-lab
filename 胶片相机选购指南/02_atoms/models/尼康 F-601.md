# 尼康 F-601

> [!image] 尼康 F-601 外观参考
> 点击查看 [Nikon F-601 camera](https://www.google.com/search?tbm=isch&q=Nikon%20F-601%20camera)

## 一句话定位
被遗忘的自动单反，操作最接近现代数码相机。

## 基本参数

| 参数 | 值 |
|------|-----|
| 发布时间 | 1990 |
| 快门类型 | 电子快门 |
| 曝光模式 | P + A + S + M |
| 卡口 | 尼康 F 卡口 |
| 对焦方式 | 自动对焦 |
| 电池 | 需要 |

## 优点
- **自动对焦 + 自动过片**：操作逻辑和现代数码单反几乎一样，零学习成本
- **内置闪光灯**：日常拍摄方便
- **PASM 四档曝光全覆盖**
- **极便宜**：因为外观不够复古、不够"胶片感"，二手市场几乎无人问津

## 缺点
- 外观像 90 年代塑料相机，缺乏复古质感和仪式感
- 电子化程度高，电子故障风险大

## 价格参考（2026）
- 二手市场：约 200-500 元

## 适合谁
- 只想要一台能出片的 F 卡口相机、不在乎外观和仪式感的人
- 从数码过渡到胶片、希望操作逻辑接近的人

## 不适合谁
- 追求手动过片、机械质感和复古美学的人

## 相关机型
- [[尼康 FE]] — 手动对焦但有 A 档的平替
- [[尼康 FG]] — 手动对焦但有 P 档的平替
