# 奥林巴斯 XA

> [!image] 奥林巴斯 XA 外观参考
> 点击查看 [Olympus XA camera](https://www.google.com/search?tbm=isch&q=Olympus%20XA%20camera)

## 一句话定位
乐趣满分、画质拉胯——手动内对焦的口袋机异类。

## 基本参数

| 参数 | 值 |
|------|-----|
| 发布时间 | 1979 |
| 镜头 | 35mm f/2.8 带萤石玻璃 |
| 快门类型 | 电子 |
| 对焦方式 | 手动内对焦（估焦） |
| 电池 | 需要（仅供电测光） |
| 重量 | 约 225g |

## 优点
- **手动内对焦**：在 PS 机中极为罕见——过片回卷流程和徕卡 M7 一模一样
- 吉村义久设计的胶囊型塑料外壳和翻盖设计成为经典
- 有一段时期曾是郝习惯的主力机——"它有我对一台相机的全部需求，也刚好没有我不需要的东西"
- 拍摄乐趣和操控感在口袋机中独树一帜

## 缺点
- 镜头素质卡在郝习惯"成像最低标准线"上——对得起体积，但对不其他专业选手
- 手动估焦对新手的出片率有影响
- 后代的 XA2 简化了对焦操作

## 价格参考（2026）
- 二手市场：约 1,000-2,500 元

## 适合谁
- 想要口袋机的便携 + 手动对焦的操控感（极少数机器能做到）
- 把拍摄乐趣放在画质之上的人

## 不适合谁
- 追求画质的人——同价位的 35mm PS 机成像可能更好
- 不想学估焦的人——选 XA2 或其他自动对焦 PS 机
