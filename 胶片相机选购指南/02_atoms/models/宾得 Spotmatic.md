# 宾得 Spotmatic（SP）

> [!image] 宾得 Spotmatic 外观参考
> 点击查看 [Pentax Spotmatic camera](https://www.google.com/search?tbm=isch&q=Pentax%20Spotmatic%20camera)

## 一句话定位
M42 螺口经典，400 元买到海量 M42 镜头生态。

## 基本参数

| 参数 | 值 |
|------|-----|
| 发布时间 | 1964 |
| 快门类型 | 全机械快门 |
| 曝光模式 | M（全手动） |
| 卡口 | M42 螺口 |
| 对焦方式 | 手动 |
| 电池 | 不需要（或仅供电测光） |

## 优点
- **M42 螺口**：可以搭配海量的 M42 镜头——从东德蔡司到苏联 Jupiter，选择极广
- **全机械**：不依赖电池
- **便宜皮实**：300-600 元即可入手
- 测光可能还准（视保养情况）

## 缺点
- M42 螺口换镜头慢（需旋转拧入，不像卡口一拧到位）
- 测光表大概率已不准
- 无任何自动辅助

## 价格参考（2026）
- 二手市场：300-600 元

## 适合谁
- 喜欢玩法多样、愿意玩不同品牌 M42 老镜头的玩家
- 400 元入门三选之一

## 不适合谁
- 希望快速换镜头的人（M42 螺口慢）
