# 佳能 EOS 3

> [!image] 佳能 EOS 3 外观参考
> 点击查看 [Canon EOS 3 film camera](https://www.google.com/search?tbm=isch&q=Canon%20EOS%203%20film%20camera)

## 一句话定位
胶片机里的现代单反——和数码 EOS 一模一样

## 基本参数
| 参数 | 值 |
|------|-----|
| 发布时间 | 1998 |
| 镜头 | Canon EF 卡口（与数码单反共用） |
| 快门类型 | 电子 |
| 曝光模式 | P / A / S / M |
| 对焦方式 | 自动对焦（45 点眼控 AF） |
| 电池 | 需要 |

## 优点 / 缺点

**优点**
- EF 卡口，与 Canon 现代数码单反镜头完全通用
- 操作逻辑和数码 EOS 一模一样，零学习成本
- 眼控自动对焦（45 点），对焦快准
- P/A/S/M 全模式，当数码机用完全没问题
- Jie's Lens 自用机型，实战验证
- 张有伯在同类别推荐 EOS 7S

**缺点**
- 纯电子机身——电子元件坏了就是砖头
- 完全没有"胶片味"的操作仪式感
- 马达过片声音较大
- 机身塑料感重，不如老机械机扎实
- 1998 年的电子元器件，寿命有限

## 价格参考（2026）
~1000-2000 元

## 适合谁 / 不适合谁

**适合**：从数码 Canon 转胶片的人（镜头直接插）、想要自动对焦的胶片新手、把胶片当数码拍的实用主义者

**不适合**：追求机械手感和复古体验的人、想要小巧低调机身的人、不信任老旧电子产品的人
