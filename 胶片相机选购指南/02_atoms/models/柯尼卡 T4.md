# 柯尼卡 T4

> [!image] 柯尼卡 T4 外观参考
> 点击查看 [Konica Autoreflex T4 camera](https://www.google.com/search?tbm=isch&q=Konica%20Autoreflex%20T4%20camera)

## 一句话定位
T3N 的后续型号，加入了 TTL 测光和快门优先，但做工扎实度不如前代。

## 基本参数

| 参数 | 值 |
|------|-----|
| 发布时间 | 1970 年代 |
| 快门类型 | 电子快门 |
| 曝光模式 | S + M（快门优先 + 手动） |
| 卡口 | 柯尼卡 AR 卡口 |
| 对焦方式 | 手动 |
| 电池 | 需要 |

## 优点
- **TTL 测光 + 快门优先**：技术参数比 T3N 更先进
- 柯尼卡镜头素质好
- 小众个性选择

## 缺点
- **做工扎实度不如 T3N**：T3N 是 T 系列巅峰，T4 用料有所妥协
- AR 卡口生态小
- 电子快门依赖电池

## 价格参考（2026）
- 二手市场：约 400-800 元

## 适合谁
- 想要快门优先的小众玩家

## 不适合谁
- 追求做工扎实的人（选 T3N）
