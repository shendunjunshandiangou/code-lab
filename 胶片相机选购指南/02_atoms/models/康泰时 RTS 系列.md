# 康泰时 RTS 系列

> [!image] 康泰时 RTS 系列 外观参考
> 点击查看 [Contax RTS camera](https://www.google.com/search?tbm=isch&q=Contax%20RTS%20camera)

## 一句话定位
CY 卡口专业电子旗舰，保时捷设计 + 蔡司镜头。

## 基本参数

| 参数 | 值 |
|------|-----|
| 发布时间 | 1975（RTS）/ 1990（RTS III） |
| 快门类型 | 电子快门 |
| 曝光模式 | A + M |
| 卡口 | CY 卡口 |
| 对焦方式 | 手动 |
| 电池 | 需要 |

## 优点
- **CY 卡口专业级**：康泰时单反系统的旗舰线
- **蔡司镜头**：卡尔蔡司为 CY 卡口制作的高素质镜头群
- **保时捷设计**：外观由保时捷工作室操刀
- 延续多代，RTS III 功能最完整

## 缺点
- 电子化程度高，老机器故障风险大
- 维修难度大、配件少
- 价格不低

## 价格参考（2026）
- 二手市场：约 2,000-5,000 元

## 适合谁
- 康泰时/CY 系统的收藏者和深度玩家

## 不适合谁
- 预算有限或需要可靠维修支持的人
