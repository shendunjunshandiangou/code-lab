# 理光 GR21

> [!image] 理光 GR21 外观参考
> 点击查看 [Ricoh GR21 camera](https://www.google.com/search?tbm=isch&q=Ricoh%20GR21%20camera)

## 一句话定位
市面上唯一一台 21mm 便携胶片机——没得选，就它。

## 基本参数

| 参数 | 值 |
|------|-----|
| 发布时间 | 2001 |
| 镜头 | 21mm f/3.5 |
| 快门类型 | 电子 |
| 对焦方式 | 自动 |
| 电池 | 需要 |

## 优点
- **唯一性**：市面上唯一一台 21mm 的便携胶片机
- 成像扎实、抗眩光不错
- 和 GR1s 共享优秀的设计语言和握持手感

## 缺点
- 全开光圈边缘素质下降明显，四角失光明显
- **和 GR1s 一样的可靠性隐患**：排线问题、液晶问题

## 价格参考（2026）
- 二手市场：约 5,000-8,000 元（极为稀有）

## 适合谁
- 极端广角爱好者
- 想要 21mm 视角但不愿扛单反+超广角镜头的人
