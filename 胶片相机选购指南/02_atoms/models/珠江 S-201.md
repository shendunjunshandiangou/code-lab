# 珠江 S-201

> [!image] 珠江 S-201 外观参考
> 点击查看 [Pearl River S-201 camera](https://www.google.com/search?tbm=isch&q=Pearl%20River%20S-201%20camera)

## 一句话定位
国产唯一可换取景器的 135 单反，百元体验腰平取景。

## 基本参数

| 参数 | 值 |
|------|-----|
| 发布时间 | 1970 年代末 - 80 年代初 |
| 快门类型 | 全机械 |
| 取景器 | 可换（腰平 + 眼平双模式） |
| 曝光模式 | M（全手动） |
| 对焦方式 | 手动 |
| 电池 | 不需要 |
| 生产厂家 | 广州光学仪器厂 |

## 腰平取景
腰平取景是从相机顶部向下俯视取景，而非把眼睛凑到取景器后。中画幅相机（哈苏 500、禄来双反）上常见，135 单反中极少见。画面是左右镜像的——构图需要反向适应。但"低头俯视对焦屏"的拍摄姿态有独特的沉浸感。

## 优点
- **100 元左右**的腰平取景体验，门槛极低
- 可换取景器——腰平和眼平自由切换
- 全机械结构，无需电池

## 缺点
- 品控参差不齐——两台同年份同型号的机器状态可能天差地别
- 售后维修几乎不存在
- 测光系统（如果有）大概率已失效
- 腰平取景画面左右镜像，需要时间适应

## 价格参考（2026）
- 二手市场：约 100 元（带全套配件）

## 适合谁
- 对腰平取景感到好奇、想花极低代价体验的人
- 喜欢折腾老国产相机的收藏者

## 不适合谁
- 需要稳定可靠的主力机的人（当成实验品，不是主力机）
