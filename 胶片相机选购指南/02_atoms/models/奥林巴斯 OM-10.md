# 奥林巴斯 OM-10

> [!image] 奥林巴斯 OM-10 外观参考
> 点击查看 [Olympus OM-10 camera](https://www.google.com/search?tbm=isch&q=Olympus%20OM-10%20camera)

## 一句话定位
OM-1 的 A 档平价替代，300 元入门奥林巴斯。

## 基本参数

| 参数 | 值 |
|------|-----|
| 发布时间 | 1978 |
| 快门类型 | 电子快门 |
| 曝光模式 | A 档（默认），加 Manual Adapter 可全手动 |
| 卡口 | OM 卡口 |
| 对焦方式 | 手动 |
| 电池 | 需要 |

## 优点
- **A 档自动曝光**：对新手友好，设定光圈即可拍
- **机身小巧精致**：继承 OM 系列核心优点
- **Zuiko 镜头群**：可接入奥林巴斯高素质镜头，尤其是 50mm f/1.8 堪称廉价好标头
- **价格亲民**：约 OM-1 的一半，300 元起步

## 缺点
- **电子快门依赖电池**
- **默认无手动模式**：全手动需额外购买 Manual Adapter（配件越来越难找）
- **价格在涨**：随着胶片热度上升，二手价格已从两三百往上走

## 价格参考（2026）
- 二手市场：300-600 元（2024 年数据，持续上涨中）

## 适合谁
- 喜欢奥林巴斯 OM 颜值和做工、预算有限的新手
- 想要 A 档便利的 Zuiko 镜头入门者

## 不适合谁
- 一定要全手动控制的人（除非买到带 Manual Adapter 的）
- 想要全机械可靠性的人（选 OM-1）
