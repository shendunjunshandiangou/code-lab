# 富士 AX-5

> [!image] 富士 AX-5 外观参考
> 点击查看 [Fujica AX-5 camera](https://www.google.com/search?tbm=isch&q=Fujica%20AX-5%20camera)

## 一句话定位
富士 X 卡口功能最完整的手动单反，小众实用。

## 基本参数

| 参数 | 值 |
|------|-----|
| 发布时间 | 1979 |
| 快门类型 | 电子快门 |
| 曝光模式 | P + A + S + M |
| 卡口 | 富士 X 卡口（胶片时代） |
| 对焦方式 | 手动 |
| 电池 | 需要 |

## 优点
- **PASM 四档全覆盖**：富士 X 卡口功能最完整的型号
- **小众有特色**：和主流的尼康/佳能/宾得不同
- 富士的 EBC 镀膜镜头素质不错

## 缺点
- **富士 X 卡口（胶片时代）**：和现在的富士 X 卡口（数码 APS-C）不通用，是两套完全不同的系统
- 镜头不好找，生态小
- 维修困难

## 价格参考（2026）
- 二手市场：约 500-1,000 元

## 适合谁
- 喜欢富士品牌、想玩小众系统的人

## 不适合谁
- 想和数码富士共用镜头的人（卡口不兼容）
