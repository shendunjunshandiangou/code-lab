# 宾得 MX

> [!image] 宾得 MX 外观参考
> 点击查看 [Pentax MX camera](https://www.google.com/search?tbm=isch&q=Pentax%20MX%20camera)

## 一句话定位
极其小巧的全机械单反，手小者友好。

## 基本参数

| 参数 | 值 |
|------|-----|
| 发布时间 | 1976 |
| 快门类型 | 全机械快门 |
| 曝光模式 | M（全手动） |
| 卡口 | 宾得 K 卡口 |
| 对焦方式 | 手动 |
| 电池 | 不需要（或仅供电测光） |

## 优点
- **极其小巧**：全机械单反中体积最小的一档，手小的人会特别喜欢
- **全机械**：无需电池即可工作
- **K 卡口**：镜头生态好，至今仍在用
- 做工扎实

## 缺点
- 纯手动，没有 A 档
- 小而全金属意味着密度高，其实有一定分量

## 价格参考（2026）
- 二手市场：约 800-1,500 元

## 适合谁
- 喜欢小巧机身、手小的人
- 追求全机械仪式感但在意尺寸的人

## 不适合谁
- 手大、觉得小机身握持不舒服的人
