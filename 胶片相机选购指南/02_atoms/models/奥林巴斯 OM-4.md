# 奥林巴斯 OM-4

> [!image] 奥林巴斯 OM-4 外观参考
> 点击查看 [Olympus OM-4 camera](https://www.google.com/search?tbm=isch&q=Olympus%20OM-4%20camera)

## 一句话定位
OM 系列旗舰，多点测光 + A 档，OM 卡口的终点。

## 基本参数

| 参数 | 值 |
|------|-----|
| 发布时间 | 1983 |
| 快门类型 | 电子快门 |
| 曝光模式 | A + M |
| 卡口 | OM 卡口 |
| 对焦方式 | 手动 |
| 电池 | 需要 |

## 优点
- **多点测光**：OM 系列中测光最先进的型号
- **A 档自动** + 手动双模式
- 继承了 OM 系列精致小巧的设计
- OM 系统手动时代的旗舰终点

## 缺点
- 存量少，价格较高
- 电子快门依赖电池

## 价格参考（2026）
- 二手市场：约 1,500-3,000 元

## 适合谁
- 奥林巴斯粉丝、想要 OM 卡口最终体验的人

## 不适合谁
- 预算有限的人（选 OM-2 或 OM-10）
