# 柯尼卡 Hexar AF

> [!image] 柯尼卡 Hexar AF 外观参考
> 点击查看 [Konica Hexar AF camera](https://www.google.com/search?tbm=isch&q=Konica%20Hexar%20AF%20camera)

## 一句话定位
"穷人徕卡"——35mm f/2 大光圈自动旁轴，暗光街头利器，预算友好的高素质随身机。

## 基本参数

| 参数 | 值 |
|------|-----|
| 发布时间 | 1993 |
| 镜头 | 35mm f/2 Hexar（固定镜头，不可换） |
| 快门类型 | 电子 |
| 对焦方式 | 自动对焦（主动红外） |
| 曝光模式 | A（光圈优先）+ P（程序） |
| 电池 | 需要 |
| 重量 | 约 510g |

## 优点
- **35mm f/2 大光圈**：在自动机/口袋机中极为少见，暗光下有天然优势
- **被称为"穷人徕卡"**：镜头素质对标徕卡 35mm Summicron，但价格只有其零头
- **主动红外对焦**：暗光也能对焦，街拍响应快
- **安静**：旁轴结构，过片马达做了静音设计，偷拍不引人注意
- 程序 + 光圈优先两种模式，操作简单

## 缺点
- **不可换镜头**：固定 35mm f/2，焦段固定
- **体积偏大**：不算"口袋机"，更像"随身小旁轴"
- **不可手动对焦**：只有自动，对焦距离靠估
- **电池需求**：用专用锂电（CR-P2 等），不如常见电池方便
- **电子化**：1990 年代电子产品，电路老化风险存在

## 价格参考（2026）
- 二手市场：约 2,000-4,000 元

## 适合谁
- 喜欢 35mm 焦段、想要大光圈暗光街头拍摄的人
- 预算有限但想要接近徕卡镜头味道的人
- 觉得 PS 机光圈太小、又不想上沉重单反的人

## 不适合谁
- 想要可换镜头系统的人
- 追求极致便携（塞口袋）的人
- 必须手动控制一切的人
