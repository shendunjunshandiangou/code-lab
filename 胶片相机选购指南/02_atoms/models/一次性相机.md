## 关键参数

| 参数 | 信息 |
|------|-----|
| 镜头 | 塑料定焦，约 f/8-f/11 |
| 快门 | 固定 ~1/125s |
| 闪光灯 | 内置，需手动开启 |
| 张数 | 27 张（富士）/ 39 张（部分柯达型号） |
| 感光度 | 预装 ISO 400 或 800 胶卷 |
| 电池 | 内置（仅供电闪光灯） |

## 为什么值得买

- 全新机器，不怕二手故障
- 到手即拍，三步操作：过片 → 开闪光灯（如需）→ 按快门
- 机身极轻（塑料），随身携带零负担
- 自带闪光灯，室内/阴天可用
- 约 100-150 元，体验成本极低

## 需要注意什么

- 不可换卷，一次性使用
- 镜头素质低，画质一般
- 无任何参数调节，全固定
- 27 张容量少
- 长期使用成本高于可换卷的常规胶片机（每卷 = 一台新机器）

## 适合谁

- 完全没接触过胶片、想花最少的钱先试试
- 聚会/旅行/婚礼等一次性活动记录
- 已有胶片机但想多带一台当"备用机"

## 不适合谁

- 已经有数码相机使用经验、追求画质的人（太初级）
- 计划长期拍胶片的人（不如直接买可换卷相机）

## 购买检查

1. 检查快门、测光、过片、回卷和计数器是否正常。
2. 检查取景器、镜头和机身内部是否存在霉、雾、划痕或腐蚀。
3. 检查电池仓是否漏液，电子功能是否稳定。
4. 检查后盖密封海绵和反光镜缓冲材料是否老化。
5. 优先选择能够提供实拍测试、维修记录或退换保障的卖家。
