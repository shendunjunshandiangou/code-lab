# 富士 GA645Zi

> [!image] 富士 GA645Zi 外观参考
> 点击查看 [Fujifilm GA645Zi camera](https://www.google.com/search?tbm=isch&q=Fujifilm%20GA645Zi%20camera)

## 一句话定位
120 画幅的傻瓜机——拿起来就拍的中画幅。

## 基本参数

| 参数 | 值 |
|------|-----|
| 发布时间 | 1998 |
| 画幅 | 6×4.5 |
| 镜头 | 55-90mm f/4.5-6.9 变焦（等效 135 约 34-56mm） |
| 快门类型 | 电子 |
| 曝光模式 | P / A / M |
| 对焦方式 | 自动对焦 |
| 电池 | 需要 |
| 闪光灯 | 内置 |

## 优点
- 张有伯在 5000-10000 元价位段的 120 推荐——**"拿起来就拍"**
- **最大画幅的傻瓜机体验**：自动对焦 + 自动曝光 + 中画幅，独此一家
- 内置闪光灯，室内补光方便
- 在 120 相机里非常紧凑——竖拍设计，握持像一本厚书
- 自动对焦在光线好时可靠，出片率有保障

## 缺点
- 纯电子机器——如果电路板挂了就是一块砖，修无可修
- 固定变焦镜头，光圈不大（f/4.5-6.9），暗光吃力
- 只能拍 645 画幅，不能拍 6×6 或 6×7
- 90 年代末的电子产品，零件早已停产
- 快门声和操控感——不是每个人都会喜欢这种"大号傻瓜机"的体验

## 价格参考（2026）
- 约 6,000-9,000 元

## 适合谁
- 想要中画幅画质但不想学中画幅操作的人
- 旅行场景——想带 120 但又不想拖一堆配件
- 从 PS 机升级到中画幅的"懒人"

## 不适合谁
- 想要全机械可靠性的谨慎派——这台机器完全靠电池和电路板活着
- 追求大光圈虚化的人——f/4.5-6.9 对中画幅来说不算大
- 喜欢手动操控感的人——它是"大号傻瓜机"，不是"大号单反"
