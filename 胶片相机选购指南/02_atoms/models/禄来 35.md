## 关键参数

| 参数 | 信息 |
|------|-----|
| 发布时间 | 1966 |
| 相机类型 | 固定镜头 35mm 取景相机，不是单反也没有黄斑测距 |
| 镜头 | 原版固定 40mm f/3.5 Carl Zeiss Tessar；后续 35 S 为 40mm f/2.8 Sonnar |
| 对焦方式 | 估焦，需要目测距离并转动对焦环 |
| 曝光模式 | M（全手动） |
| 快门 | 机械镜间快门 |
| 测光 | 顶部 CdS 联动测光，指针显示不在取景器内 |
| 电池 | 只供测光；机械快门不需要电池 |
| 特殊结构 | 镜头可缩入机身；热靴位于机身底部 |

## 为什么值得买

- **体积非常小**：推出时是极具代表性的全画幅 35mm 口袋相机设计。
- **40mm Tessar 焦段实用**：介于 35mm 和 50mm 之间，适合旅行与日常记录。
- **全机械拍摄核心**：即使测光失效，仍可使用外置测光或阳光十六法继续拍摄。
- **伸缩镜头设计独特**：收纳后机身紧凑，机械操作感强。
- **版本与收藏体系丰富**：德国、后期新加坡生产及 35 T、35 S 等版本各有特点。

## 需要注意什么

- 禄来 35 没有自动对焦，也没有黄斑测距；估焦是使用门槛的核心。
- 镜头必须在快门上弦后按正确操作缩回，强行操作可能损坏机构。
- 原版、35 T、35 S、TE、SE、B、LED 等版本的镜头、测光和电池不同，不能混用参数。
- 热靴在底部，使用闪光灯时通常要改变持机方向或拆下相机套。
- 原用汞电池已停产，替代电池电压可能影响测光，需要校准、转接或外置测光。
- 小机身控制集中，快门和光圈拨盘的手感与普通单反差异很大。

## 适合谁

- 已经理解手动曝光，愿意练习距离估计的人。
- 需要小体积，但不追求自动化的旅行和街拍用户。
- 喜欢机械设计、40mm 焦段和独特操作方式的人。

## 不适合谁

- 第一次使用胶片相机，又不愿学习估焦与手动曝光的人。
- 需要快速近距离对焦、儿童或运动抓拍的人。
- 希望抬机即拍、在取景器中直接获得完整曝光信息的人。

## 购买检查

1. 先确认具体版本、产地和镜头铭牌：原版／35 T、35 S、TE、SE 等不能按同一规格估价。
2. 按正确顺序测试镜头伸出、锁定和缩回，确认没有卡顿、松旷或被强行操作的痕迹。
3. 逐档测试机械快门和 B 门，检查镜间快门叶片是否有油或粘滞。
4. 用可靠测光表对比机顶 CdS 测光，并确认替代电池方案与读数偏差。
5. 检查估焦环、镜头霉雾、取景器、底部热靴、过片、计数和回卷机构。
6. 优先选择能提供实拍样片、测光说明和具体版本照片的卖家。

## 同价位替代

- [[奥林巴斯 XA]] — 有黄斑联动测距和光圈优先，仍保持很小体积。
- [[奥林巴斯 mju-1]] — 自动对焦与自动曝光，更适合轻松随拍。
- [[康泰时 T2]] — 自动化与做工更高，但价格和维修风险也更高。
- Rollei 35 S — 40mm f/2.8 Sonnar 版本，价格通常更高。
