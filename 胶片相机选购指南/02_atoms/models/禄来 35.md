# 禄来 35

> [!image] 禄来 35 外观参考
> 点击查看 [Rollei 35 camera](https://www.google.com/search?tbm=isch&q=Rollei%2035%20camera)

## 一句话定位
经典口袋机械机——但估焦模式劝退多数人

## 基本参数
| 参数 | 值 |
|------|-----|
| 发布时间 | 1966 |
| 镜头 | 固定 40mm f/3.5（35S 为 f/2.8 Sonnar） |
| 快门类型 | 机械 |
| 曝光模式 | M |
| 对焦方式 | 估焦（必须目测距离） |
| 电池 | 测光需要（机械部分不需要） |

## 优点 / 缺点

**优点**
- 传奇工业设计，极致紧凑（曾是最小的 135 全幅相机）
- 35S / 35SE 版本搭载 Carl Zeiss Sonnar 镜头，画质优秀
- 全机械机身，无电池也能工作
- 复古造型极美，社交属性拉满

**缺点**
- **估焦（Zone Focus）——致命短板**：取景器无法对焦，必须目测距离再手动调焦环
- 张有伯明确警告："估焦相机，对大部分人来说很难用，不能快拍"
- 萌大统领也指出："需估焦，不适合快拍"
- 对新手极度不友好，废片率高
- 等效 40mm 焦段取景靠猜，街拍/抓拍几乎不可能

## 价格参考（2026）
~1000-2500 元，根据版本不同（35、35S、35SE、35T）

## 适合谁 / 不适合谁

**适合**：收藏玩家、估焦老手、追求极致便携的街头摄影师、想买"好看相机"的人

**不适合**：新手（废片率太高）、需要快速对焦的街拍/人文拍摄、对估焦没有概念的人、想要所见即所得取景体验的人
