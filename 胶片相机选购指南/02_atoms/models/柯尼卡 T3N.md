# 柯尼卡 T3N

> [!image] 柯尼卡 T3N 外观参考
> 点击查看 [Konica Autoreflex T3 camera](https://www.google.com/search?tbm=isch&q=Konica%20Autoreflex%20T3%20camera)

## 一句话定位
T 系列巅峰，全金属机身，小众但镜头素质被高度认可。

## 基本参数

| 参数 | 值 |
|------|-----|
| 发布时间 | 1970 年代 |
| 快门类型 | 全机械快门 |
| 曝光模式 | M（全手动） |
| 卡口 | 柯尼卡 AR 卡口 |
| 对焦方式 | 手动 |
| 电池 | 不需要（或仅供电测光） |

## 优点
- **T 系列巅峰之作**：全金属机身，操作丝滑，极度耐用
- **镜头素质好**：柯尼卡镜头被资深玩家高度认可
- 做工扎实——比后续的 T4 更可靠
- 小众有话题性

## 缺点
- 柯尼卡 AR 卡口生态小，镜头不好找
- 品牌知名度低，维修师傅少
- 纯手动，没有 A 档

## 价格参考（2026）
- 二手市场：约 400-800 元

## 适合谁
- 喜欢小众品牌、追求独特的人
- 不在乎自动功能、愿意玩转纯手动的玩家

## 不适合谁
- 需要丰富镜头选择和便利维修的普通用户
