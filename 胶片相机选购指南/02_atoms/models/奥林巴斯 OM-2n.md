# 奥林巴斯 OM-2n

> [!image] 奥林巴斯 OM-2n 外观参考
> 点击查看 [Olympus OM-2n camera](https://www.google.com/search?tbm=isch&q=Olympus%20OM-2n%20camera)

## 一句话定位
Jie's Lens 自用——小巧精致的日系电子单反

## 基本参数
| 参数 | 值 |
|------|-----|
| 发布时间 | 1979 |
| 镜头 | OM 卡口 |
| 快门类型 | 电子快门（机械备份） |
| 曝光模式 | A / M |
| 对焦方式 | 手动 |
| 电池 | 需要 |

## 优点 / 缺点

**优点**
- SLR 中极其紧凑（OM 系统以小型化闻名）
- A 档 OTF（Off-The-Film）实时测光，精度高
- Zuiko 镜头群素质优秀
- Jie's Lens 自用机型，实战检验
- 做工精致，手感极佳

**缺点**
- OM 卡口镜头不如 Nikon F 卡口普及
- 电子快门，没电无法工作
- 老电子元件有老化失效风险
- OM-2n 是 OM-2 的小改款，存量比 OM-2 少

## 价格参考（2026）
~800-1500 元

## 适合谁 / 不适合谁

**适合**：追求小巧精致 SLR 的用户、OM 镜头用户、想要 A 档便利的进阶玩家

**不适合**：需要大量便宜副厂镜头的用户（Nikon F 生态更大）、坚持全机械偏好者（选 OM-1）
