# 富士 ST-701

> [!image] 富士 ST-701 外观参考
> 点击查看 [Fujica ST701 camera](https://www.google.com/search?tbm=isch&q=Fujica%20ST701%20camera)

## 一句话定位
富士早期 M42 螺口入门单反，ST 系列的基础款。

## 基本参数

| 参数 | 值 |
|------|-----|
| 发布时间 | 1971 |
| 快门类型 | 全机械快门 |
| 曝光模式 | M（全手动） |
| 卡口 | M42 螺口 |
| 对焦方式 | 手动 |
| 电池 | 不需要 |

## 优点
- **M42 螺口**：镜头选择极广
- 全机械，无需电池
- 富士 EBC 镀膜

## 缺点
- 功能基础
- M42 螺口换镜头慢
- 存量不多

## 价格参考（2026）
- 二手市场：约 300-600 元

## 适合谁
- 想用最低成本体验 M42 系统的人

## 不适合谁
- 需要测光辅助的人（测光表大概率已失效）
