# 奥林巴斯 OM-2

> [!image] 奥林巴斯 OM-2 外观参考
> 点击查看 [Olympus OM-2 camera](https://www.google.com/search?tbm=isch&q=Olympus%20OM-2%20camera)

## 一句话定位
OM-1 的电子化升级版，A 档自动 + 精致做工。

## 基本参数

| 参数 | 值 |
|------|-----|
| 发布时间 | 1975 |
| 快门类型 | 电子快门 |
| 曝光模式 | A + M |
| 卡口 | OM 卡口 |
| 对焦方式 | 手动 |
| 电池 | 需要 |

## 优点
- **A 档自动曝光**：继承了 OM-1 的精致做工，多了光圈优先的便利
- **Zuiko 镜头群**：高素质镜头支撑
- 保留了 OM 系列小巧精致的设计语言

## 缺点
- 电子快门依赖电池
- 不如 OM-1 经典，但性价比可能更好

## 价格参考（2026）
- 二手市场：约 800-1,500 元

## 适合谁
- 喜欢 OM 精致做工 + 想要 A 档便利的人

## 不适合谁
- 追求全机械仪式感的人（选 OM-1）
