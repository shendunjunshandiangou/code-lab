# 美能达 X-1（XK/XM）

> [!image] 美能达 X-1 外观参考
> 点击查看 [Minolta X-1 camera](https://www.google.com/search?tbm=isch&q=Minolta%20X-1%20camera)

## 一句话定位
美能达手动时代旗舰，电子快门 + 取景器集成控制，超前于时代。

## 基本参数

| 参数 | 值 |
|------|-----|
| 发布时间 | 1973 |
| 快门类型 | 电子快门 |
| 曝光模式 | A + M |
| 卡口 | 美能达 MC/MD 卡口 |
| 对焦方式 | 手动 |
| 电池 | 需要 |

## 优点
- **美能达手动时代旗舰**：电子快门和取景器集成控制的设计在当时非常超前
- A 档自动曝光
- 专业级可换取景器系统
- MC/MD 镜头群支撑

## 缺点
- 存量极少，难以入手
- 40+ 年电子组件，故障风险高
- 价格不低

## 价格参考（2026）
- 二手市场：约 2,000-4,000 元（稀有品，价格波动大）

## 适合谁
- 美能达发烧友、收藏者

## 不适合谁
- 追求实用性的普通用户
