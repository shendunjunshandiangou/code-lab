# 尼康 F3

> [!image] 尼康 F3 外观参考
> 点击查看 [Nikon F3 camera](https://www.google.com/search?tbm=isch&q=Nikon%20F3%20camera)

## 一句话定位
手动单反的黄金标准——最多UP主共同推荐的胶片入门机。

## 基本参数

| 参数 | 值 |
|------|-----|
| 发布时间 | 1980 |
| 快门类型 | 电子快门（1/60s 机械备份） |
| 曝光模式 | A + M |
| 卡口 | 尼康 F 卡口 |
| 对焦方式 | 手动 |
| 电池 | 需要 |
| 重量 | 约 700g（F3HP），轻于 F2 |

## 优点
- **最多 UP 主共同推荐**：萌大统领将其列为第一台相机首选，张有伯将其作为 1000-5000 元区间的核心推荐
- **模块化设计**：可换取景器（F3HP 高眼点版本对戴眼镜者友好）、可换对焦屏、马达卷片器等
- **F 卡口镜头生态巨大**：兼容 Non-AI 到 AI-S 全部手动镜头，可和现代尼康数码共享资源
- **可靠耐用**：专业级做工，一代机皇，机身红线已成经典
- **测光精准**：中央重点测光系统，可靠性高

## 缺点
- **依赖电池**：电子快门，没电就只剩 1/60s 机械备份
- **比 FM2 重**：不算最便携的 SLR

## 价格参考（2026）
- 二手市场：约 2,500-4,000 元（机身 + 50mm 套机约 3,000 元）

## 适合谁
- 想要一台经典专业旗舰、有 A 档便利的人
- 戴眼镜的人（选 F3HP 版本，高眼点取景器）
- 已有尼康 F 卡口镜头的数码用户

## 不适合谁
- 追求纯机械、永不依赖电池的人（选 F2）
- 追求极致轻便的人（选 FM2）

## 相关机型
- [[尼康 F2]] — 纯机械巅峰，F3 的前代旗舰，全机械无需电池
- [[尼康 FM2]] — 更小更轻的纯机械选择，非旗舰级做工
- [[尼康 FE]] — 更便宜的电子快门替代，同样有 A 档
