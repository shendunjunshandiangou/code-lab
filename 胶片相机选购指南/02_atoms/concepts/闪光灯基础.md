# 闪光灯基础

## 一句话定义
胶片相机的闪光灯使用有一个硬约束——闪光同步速度，不同机型从 1/60s 到 1/250s 不等，超过这个速度用闪光灯会出现"半幅黑片"。这是买相机前就该知道的事。

## 闪光同步速度（最重要的概念）

闪光灯的发光时间极短（千分之一秒级），而相机的焦平面快门是两块帘幕先后扫过底片来曝光的。只有当**快门速度足够慢**，让两块帘幕之间露出完整的"缝隙"时，整个画面才能被闪光灯均匀照亮。

- 快门速度 ≤ 同步速度 → 画面完整被闪光照亮 ✅
- 快门速度 > 同步速度 → 第二帘在第一帘还没扫完时就开始追，画面被帘幕挡住一部分 → **半幅黑片** ❌

### 各类型相机的同步速度

| 相机类型 | 典型同步速度 | 备注 |
|----------|-------------|------|
| 老式全机械单反（如尼康 FM2 前期） | 1/250s | 较高，白天补光方便 |
| 多数电子单反（如尼康 FE、佳能 AE-1） | 1/125s | 主流水平 |
| 90 年代自动单反（如 F-601） | 1/125s - 1/250s | 部分支持高速同步 |
| 纯机械旁轴（如老徕卡） | 1/60s 甚至更低 | 限制较大 |
| 叶片快门（镜间快门，如部分中画幅、禄来） | 全速同步 | 任何快门速度都能闪，这是叶片快门的核心优势 |

> **选购时注意**：如果你打算白天户外用闪光灯补光，同步速度越高越好（1/250s 优于 1/125s）。纯机械旁轴和中画幅叶片快门是两个极端。

## GN 值（闪光指数）

外置闪光灯的功率用 GN 值表示。GN 值越大，闪光越强。用 GN 值可以反推该用什么光圈：

```
光圈 = GN ÷ 距离（米）
```

例：GN 28 的闪光灯，主体距离 4 米 → 用 f/7（约 f/8）光圈。

部分相机（如美能达 X-700 配自动闪光灯）能自动根据对焦距离控制闪光输出，省去手算的麻烦。

## 内置闪光 vs 外置闪光

| 维度 | 内置闪光灯 | 外置闪光灯 |
|------|-----------|-----------|
| 功率 | 弱（GN 7-12） | 强（GN 20-50+） |
| 方向 | 只能直打 | 可调角度，能跳闪（反射光更柔） |
| 耗电 | 耗相机电池 | 自带电池 |
| 红眼 | 容易出现 | 不易 |
| 代表机型 | 尼康 F-601、美能达 807Si、几乎所有 PS 机 | 通用热靴接口（各品牌都有各自的 TTL 自动闪） |

> 90 年代后的自动单反（F-601、807Si 等）的内置闪光灯对新手意外好用——功率虽小但能 TTL 自动控制输出，不需要你算 GN 值。

## 什么时候需要闪光灯

| 场景 | 用法 | 说明 |
|------|------|------|
| 室内光线不足 | 直打补光 | 最常见，但直打容易"惨白" |
| 白天逆光人像 | 补光（"压暗背景"） | 用闪光把逆光的人脸打亮，快门设到同步速度上限 |
| 阴天补眼神光 | 弱补光 | 让眼睛有光点 |
| 夜景人像 | 慢速同步 | 用慢快门拍亮背景 + 闪光定格人像 |

## 新手避雷

- **PS 机的闪光灯一般无法完全关闭**——尤其是老型号（如 mju-1 那种自动闪的），晴天可能误触发。介意闪光的人买前看清楚
- **不要隔着玻璃/镜面用闪光灯** —— 会反射回来一片白
- **注意 PS 机的"红色闪电"标志** —— 这是防红眼预闪功能，拍人像时打开
- **天黑用闪光灯时距离别超过 3-4 米** —— 内置闪的有效距离很有限，远了就是一片黑背景 + 苍白人脸
