# 装卷与回卷

## 一句话定义
手动胶片机上卷与回卷的完整操作流程，这是胶片摄影与数码摄影最根本的操作差异——装错毁一卷、回错毁一卷，必须熟练掌握。

## 手动上卷步骤（纯机械机身 / 手动过片机身）

1. **打开后盖**：通常通过提起回卷摇杆或拨动后盖释放拨杆
2. **放入胶卷暗盒**：将暗盒放入左侧胶卷仓（面向相机背面），回卷摇杆对准暗盒顶部凹槽后压入
3. **拉出片头**：从暗盒中拉出约 8-10cm 的片头，注意不要触摸胶卷的药膜面
4. **固定片头**：将片头插入右侧卷片轴的缝隙中，确保片头齿孔卡入卷片轴的挂钩
5. **咬合确认**（最关键步骤）：确认胶卷上下两排齿孔分别对准并咬合到过片齿轮的上下齿。关后盖前低头目视检查——如果只有单侧咬合，过片时胶卷会歪斜甚至脱落
6. **关后盖**：轻推过片扳手/拨轮一张，观察回卷摇杆是否随之转动——如果转动说明胶卷确实在走动
7. **空拍到计数器至 1**：通常前 1-2 张是装卷时已曝光的片头部分，需要空拍跳过

## PS 机 / 自动单反装卷——最简单

塞入胶卷暗盒 -> 拉出片头到机身上的标记线（通常是橙色或红色横线）-> 关后盖 -> 相机自动将片头卷入并识别 ISO（如有 DX 码）。全程不需要手动对齐齿孔。

## 手动回卷步骤

1. **确认拍完**：计数器到 36（或相机设定的张数），过片扳手/拨轮卡住无法继续扳动——说明胶卷已到底
2. **按下回卷释放按钮**：通常在机身底部（一个小按钮），按下后胶卷齿轮脱离锁止，允许倒转
3. **转动回卷摇杆**：按摇杆上箭头方向匀速转动，保持稳定节奏。如果突然感觉阻力增大——立即停止，可能是卷片轴某处卡住，强行回卷可能撕裂胶卷
4. **等待"落空感"**：回卷过程中阻力突然明显减小甚至消失——这表示胶卷已脱离另一侧卷片轴，完全收入暗盒
5. **多转 3-5 圈**：落空感后不要马上开盖！再转几圈确保片头也完全收回。如果片头留在外面，开盖瞬间整卷暴露
6. **打开后盖取出**：只有完成以上所有步骤后才能打开后盖，取出暗盒

## 各机型差异

- **尼康 FM / FE 系列**：回卷摇杆顶部折叠式，使用时翻开
- **宾得 K1000 / MX**：回卷摇杆在左侧正常位置，结构直观
- **奥林巴斯 OM 系列**：回卷释放按钮在机身左侧（非底部）
- **各机型具体步骤**：先搜索 `[你的相机型号] + load film` 或 `装卷教程`，不同相机设计差异大

## 常见翻车场景

| 翻车 | 原因 | 如何避免 |
|------|------|----------|
| 拍完一卷送去冲扫，回来全白 | 片头没钩牢，胶卷根本没走 | 装卷后观察回卷手柄是否跟着转动 |
| 全黑 | 后盖被打开了 | 拍完前绝不碰后盖 |
| 齿孔撕裂 | 拍到最后一张时用力过猛 | 感觉阻力变大就不要再过片了 |
| 半张空白 | 胶卷在相机内停留太久（几周以上） | 一卷尽快拍完，不要拖几个月 |

## 严禁事项

- 拍摄过程中**绝对不要**打开后盖
- 回卷完成前**绝对不要**打开后盖
- 不要用手指触摸胶卷的药膜面
- 装卷/回卷时避免阳光直射，最好在阴影或室内操作
