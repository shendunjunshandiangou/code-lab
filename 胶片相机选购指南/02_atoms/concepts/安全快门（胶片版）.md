# 安全快门（胶片版）

## 一句话定义
手持拍摄不因手抖导致画面模糊的最慢快门速度，约等于镜头焦距的倒数。在胶片相机上使用此法则时，需考虑老相机快门档位不连续以及低 ISO 胶卷带来的速度天花板。

## 基本法则

> 安全快门速度 >= 1 / 焦距

| 焦距 | 安全快门（不低于） | 实际可选档位 |
|------|-------------------|-------------|
| 28mm | 1/30s | 1/30s 或 1/60s |
| 35mm | 1/30s ~ 1/60s | 1/60s |
| 50mm | 1/50s | 1/60s 或 1/125s |
| 85mm | 1/85s | 1/125s |
| 135mm | 1/135s | 1/125s 或 1/250s |
| 200mm | 1/200s | 1/250s |

## 胶片相机的特殊考量

### 问题一：老相机快门档位不连续

现代数码相机可以设定任意快门速度（1/50s、1/80s 等），但老式机械相机只有固定的档位：B、1s、1/2s、1/4s、1/8s、1/15s、1/30s、1/60s、1/125s、1/250s、1/500s、1/1000s。

50mm 镜头的理论安全快门是 1/50s，但相机上没有 1/50s 档——只能选 1/60s 或 1/125s。原则是**选更快的，不选更慢的**。

### 问题二：低 ISO 胶卷的速度天花板

这是胶片和数码的关键差异——数码可以随时调高 ISO，胶片不行：

- ISO 200 的胶卷在室内晴天，测光结果可能是 1/30s（f/2.8）——低于 50mm 的安全快门
- ISO 400 在同样场景下能到 1/60s —— 刚好安全
- 这就是为什么新手首选 ISO 400 的胶卷——它给你更大的安全快门余量

选 ISO 400 而不是 ISO 100 的胶卷，本质上是在为自己的手持拍摄范围买保险——多一档快门速度，就多一个可拍的场景。

### 胶片机没有防抖

老相机没有 IS/VR 防抖，安全快门铁律比数码时代更严格。胶片不像数码可以立即回看——你拍完后要等冲扫出来才知道手有没有抖。新手最常见的废片原因之一就是快门太慢导致整张照片模糊。

## 低于安全快门怎么办

| 优先级 | 方案 | 适用场景 |
|--------|------|---------|
| 1 | 开大光圈 | 光圈还有余量时（如 f/8 -> f/4 多进 2 档光） |
| 2 | 找支撑 / 用三脚架 | 静态场景、风光 |
| 3 | 用闪光灯 | 有闪光灯且距离合适时 |
| 4 | 接受风险，多拍一张 | 实在没办法，拍两张提高成功率 |
| 5 | 不拍 | 如果画面不值得虚掉，省一张胶卷 |

**注意**：换更高 ISO 的胶卷不是"当场"能做的——你已经装了一卷胶卷，中途不能换。胶片的 ISO 选择是"开拍前"的决策。

## 低光环境下的 ISO 选择策略

- ISO 100-200：阳光充足的日子，快门速度充裕
- ISO 400：通用选择，阴天和室内都能应付
- ISO 800+：夜拍、室内无闪光灯、需要高速快门的场景（价格更贵、颗粒更粗）
