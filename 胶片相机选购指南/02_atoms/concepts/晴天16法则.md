# 晴天16法则

## 一句话
在没有测光表时，用天气状况快速估算曝光参数的口诀。

## 口诀

| 天气 | 光圈 | 快门 |
|------|------|------|
| 大晴天，阳光刺眼，影子清晰 | f/16 | 1/ISO（ISO 100→1/125s） |
| 白云朵朵，影子柔和 | f/11 | 同上 |
| 阴天，没有影子 | f/8 | 同上 |
| 乌云压顶，非常暗 | f/5.6 | 同上 |
| 日出日落、深阴影 | f/4 | 同上 |

## 什么时候用
- 纯机械相机没有测光表
- 老相机测光表已老化不准
- 光线稳定时做快速估光

## 实用技巧
- ISO 400 时：晴天 = f/16 + 1/500s，阴天 = f/8 + 1/500s
- 这个法则是估算，做不到精准——但对于负片的高宽容度来说，大致对了就能出片
- 不确定时，偏过曝半档至一档（胶片的"宁过勿欠"原则）
