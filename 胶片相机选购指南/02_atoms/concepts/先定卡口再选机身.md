# 先定卡口再选机身

## 一句话定义
选第一台胶片单反的正确顺序——定预算、定卡口生态、最后选具体机型，而不是反过来。

## 详细说明

### 为什么这个顺序重要

很多新手犯的错误是"先被一台好看的相机吸引，买了之后才发现能用的镜头又贵又少"。正确的顺序应该是先定卡口生态——这决定了你未来所有镜头的可用范围、价格区间和升级路径。

### 三步选机法

```
第一步：定预算 → 300-500 / 500-1000 / 1000-2000 / 2000-4000 / 4000+
第二步：定卡口 → 选一个镜头丰富、价格合理的生态系统
第三步：选机型 → 在选定卡口下找到预算内最好的机身
```

### 卡口生态优先级

按"镜头选择多 + 价格合理 + 未来可用"的维度排序：

1. **尼康 F 卡口** — 最高优先级。从 1959 年至今，胶片数码通用，镜头从几百到几万都有
2. **宾得 K 卡口** — 类似 F 卡口，但镜头规模稍小
3. **M42 螺口** — 镜头选择最广但换镜头慢，适合"玩"而非"用"
4. **佳能 FD 卡口** — 经典但已停产，镜头无法转接 EF，但镜头本身不贵
5. **美能达 MC/MD 卡口** — 镜头素质好、不贵，但生态已冻结
6. **OM 卡口（奥林巴斯）** — Zuiko 镜头小巧高质，但选择有限
7. **CY 卡口（康泰时/雅西卡）** — 蔡司镜头贵，雅西卡镜头便宜
8. **其他（柯尼卡 AR、徕卡 R、禄来 QBM）** — 生态极小或镜头极贵，纯小众/收藏路线

### 一个反直觉的事实

你买机身花的钱可能只是系统总成本的 20-50%。如果你花 500 买 FE 机身但配了 3 支尼康 F 卡口老镜头——每支几百块——你的总投入中镜头占了多数。卡口生态决定了这个"多数"能有多便宜或多贵。
