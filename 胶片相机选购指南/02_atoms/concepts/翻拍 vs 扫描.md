# 翻拍 vs 扫描

## 一句话定义
扫描 = 送店 → 收到数字文件；翻拍 = 自己用数码相机拍底片 → 信息更多但门槛更高。

## 差异对比

| 维度 | 店扫（SP3000 等） | 翻拍 |
|------|-----------------|------|
| 动态范围 | 受限于扫描仪引擎 | 通常更好 |
| 暗部细节 | 可能丢失 | 更多可恢复 |
| 色彩 | 扫描仪色彩引擎决定（风格化） | 自行后期，完全可控 |
| 便利性 | 极高（寄 → 等 → 收） | 低（需要设备和时间） |
| 设备门槛 | 无 | 全画幅数码机 + 1:1 微距镜头 + 翻拍台 |
| 成本 | 15-50 元/卷 | 设备初始投入约 2000-5000+ 元 |

## 翻拍的最低设备要求
- 全画幅数码相机（135 底片 = 全画幅大小）
- 1:1 放大倍率微距镜头（将底片细节完整映射到传感器）
- 光源（LED 观片器或翻拍台）
- 胶片夹/过片器（手动推片即可，不需要电动系统）
- Lightroom 或 Photoshop（去色罩 + 转正）

## 为什么重要

新手不需要考虑翻拍——先走店扫足够。等你拍了十几二十卷后，如果某些特别喜欢的照片想要更高质量的数字文件，再考虑翻拍。重点是先拍起来，技术细节的优化在后面。
