# 胶片ISO

## 一句话定义
胶片的感光度是固定的——装了一卷 ISO 400，整卷都按 400 来拍，中途不能更改。

## 和数码的根本区别

在数码相机中，ISO 是一个可以每张照片随意调整的电子参数。在胶片中，ISO 由胶卷本身决定——你买了一卷柯达金 200，它的 ISO 就是 200，一整卷 36 张都是 200。

这意味着：出门前必须根据当天的光线环境选择合适的胶卷。

## 选 ISO 的规则

| ISO | 适用场景 |
|-----|----------|
| 100-200 | 室外晴天、充足自然光 |
| 400 | 阴天、室内、黄昏——最通用的选择 |
| 800+ | 暗光、室内无闪光灯——颗粒明显增加 |

## 为什么选错怎么办

- 如果 ISO 太低：暗光下快门速度过慢 → 手抖糊片
- 如果 ISO 太高：晴天时即使最小光圈 + 最高快门也可能过曝 → 但可通过 ND 滤镜或收光圈解决
- 安全起见，新手第一选择是 ISO 400——最通用

## 「宁过勿欠」铁律

胶片的曝光铁律和数码相反：数码宁欠勿过（过曝拉不回来），胶片宁过勿欠（欠曝暗部死黑救不回来）。

负片过曝 2-3 档仍有细节，欠曝 1 档暗部就没有了。不确定曝光时，多给一档——把相机 ISO 设为比胶卷标称低半档到一档（如 400 的卷用 320 拍）。

此规则对反转片不适用（反转片对过曝零容忍）。
