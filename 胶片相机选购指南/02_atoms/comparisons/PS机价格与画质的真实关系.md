# PS机价格与画质的真实关系

## 一句话
几千块和几百块的 PS 机，盲测照片时大多数人分不出来。

## 盲测实验详情

一位 YouTuber 用 **Contax T2（约 1000 美元）** 和 **30 美元的二手傻瓜机** 做了对照实验：
- 两台相机使用完全相同的胶卷
- 在同一家冲扫店冲洗扫描
- 将照片标注为 A 和 B，让观众在不知道来源的情况下判断
- **结果：没有赢家——大多数观众分辨不出来**

## 盲测结论

猫吉屋翻译的一期视频中，盲测对比了"贵价 PS 机"和"便宜 PS 机"在同卷同冲扫下的出片。大多数参与者无法区分两者的照片。这个结论背后的逻辑是——在 PS 机上，成像差距的瓶颈不在机身，而在胶卷 + 冲扫 + 镜头。

这个实验验证了一个反复被提及的观点：**在胶片摄影中，胶卷和冲扫对成像的贡献远大于相机机身本身。**

## 贵的 PS 机到底贵在哪

| 贵的理由 | 是否影响出片质量 |
|----------|----------------|
| 蔡司/徕卡镜头 | ✅ 有影响（但普通观看下差异不易察觉） |
| 钛金属机身 | ❌ 不影响 |
| 品牌效应 | ❌ 不影响 |
| 名人用过/明星同款 | ❌ 不影响 |
| 更大的光圈 | ✅ 有影响（暗光表现更好） |

贵的 PS 机在做工、品牌、设计、手感上确实更胜一筹。但如果你只是看最终的照片，大多数人无法分辨哪张是贵相机拍的。

## 预算分配启示

与其把钱砸在高端机身上，不如：
1. 更多预算分配给**胶卷**——不同的卷带来不同的色彩风格
2. 配更好的**镜头**（如果是可换镜头系统）——镜头的成像差异远大于机身
3. 留预算给**冲扫**——选靠谱的冲扫店比选贵的相机更影响出片

## 但这不等于"买便宜的就行"

贵相机带来的价值在其他维度：
- 更好的**做工和手感**——让你更愿意带出门
- 更高的**可靠性**（对某些型号来说）
- 更好的**操作体验**（对焦速度、取景器大小等）
- **品牌效应**和持有乐趣

## 选购启示

如果你只是想在社交媒体上分享胶片照片——几百块的 PS 机配好胶卷，效果和几千块的几乎没有可见差距。贵的 PS 机买的是体验（做工、手感、品牌），不是画质。
