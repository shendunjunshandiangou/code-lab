# PS机高低价对比

## 一句话摘要
在相同胶卷和冲扫条件下，不同价位 PS 机的画质差异远小于价格差异——价格差异主要来自做工、手感、品牌和可靠性，而非成像质量。

## 盲测结果

Contax T2（约 1000 美元）vs 30 美元二手傻瓜机的盲测中，大多数观众**无法分辨**哪张是贵的拍的。

## 价格差异拆解

| 价格维度 | 低价机（100-500 元） | 中端机（500-2000 元） | 高端机（2000-10000+ 元） |
|----------|---------------------|----------------------|------------------------|
| 镜头镀膜 | 普通/无镀膜 | 有品牌镀膜 | 顶级镀膜（T\*等） |
| 对焦 | 估焦或基础 AF | 较准的 AF | 精确快速的 AF |
| 做工手感 | 塑料 | 混合 | 钛合金等高端材料 |
| 画质差异 | 在相同胶卷冲扫下，画质差异远小于价格差异 |
| 可靠性 | 不稳定 → 坏了只能扔 | 较稳定 | 不一定更可靠（排线断裂等通病） |
| 品牌溢价 | 无 | 少量 | 大量（名人效应+网络热度） |

## 理性选购框架

1. **先看缺点**（郝习惯逻辑）：列出各机型的缺点 → 哪些缺点你能接受 → 它就是你的最优解
2. **成像放在第二位**：135 胶片画质早就打不过现代数码了，日常记录不需要极致成像
3. **设计放在第一位**：一台你愿意每天带出门的相机，才是真正有用的相机
4. **冷门替代**：相同优秀镜头可能出现在不同价位——雅西卡 T3 Super（蔡司 T\*）千元 vs Contax T2（蔡司 T\*）大几千
