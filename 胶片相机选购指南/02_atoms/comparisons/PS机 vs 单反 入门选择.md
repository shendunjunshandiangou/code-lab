# PS机 vs 单反 入门选择

## 一句话
PS 机让你马上开始拍，单反让你学会怎么拍——两条路没有高下之分。

## 对比

| 维度 | PS 机（傻瓜机） | 手动单反 |
|------|----------------|---------|
| 上手速度 | 即开即拍 | 需要 30 分钟学会装卷和基本操作 |
| 每张拍摄时间 | 1-3 秒 | 10-30 秒（全手动） |
| 可换镜头 | ❌（固定镜头） | ✅ |
| 长期可玩性 | 低（拍久了会想升级） | 高（换镜头、换焦段） |
| 便携性 | 极高（口袋机） | 中（挂脖子或装包） |
| 学习价值 | 低 | 高 |
| 适合 | 纯粹想体验胶片感、只想记录日常 | 想深入学习摄影技术 |

## 不应该纠结的问题

- "哪个画质更好"——画质主要看胶卷和冲扫，不是机身类型
- "哪个更划算"——全自动单反（如 EOS 系列、F-601）比大部分 PS 机都便宜，画质可能更好
- "哪个更保值"——热门 PS 机在涨但泡沫风险大，冷门单反在跌但买入成本低

## 第三条路：电子单反 + A 档

如果你想同时拥有"可换镜头的扩展性"+"不操心的拍摄节奏"，电子快门单反（如尼康 FE/FE2、美能达 X-700、佳能 AE-1）是最好的折中选择。设定光圈，相机自动匹配快门——你只需要对焦和构图。
